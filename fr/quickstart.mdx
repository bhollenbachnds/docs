## Configurer votre environnement de développement

Apprenez à mettre à jour votre documentation localement et à la déployer en ligne.

### Modifier et prévisualiser

<AccordionGroup>
  <Accordion icon="github" title="Cloner votre documentation localement">
    Lors du processus d'intégration, nous avons créé un dépôt sur votre Github avec
    le contenu de votre documentation. Vous pouvez trouver ce dépôt sur notre
    [tableau de bord](https://dashboard.mintlify.com). Pour cloner le dépôt
    localement, suivez ces
    [instructions](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    dans votre terminal.
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Prévisualiser les modifications">
    La prévisualisation vous permet de vous assurer que vos modifications s'affichent comme prévu. Nous avons développé une interface en ligne de commande pour afficher ces modifications localement.

    1. Installer le
       [Mintlify CLI](https://www.npmjs.com/package/mintlify) pour prévisualiser les
       modifications de la documentation localement avec cette commande : `npm i -g mintlify`
    2. Exécuter la commande suivante à la racine de votre documentation (là où
       `docs.json` se trouve) : `mintlify dev`

    <Note>
      Si vous utilisez actuellement le fichier de configuration `mint.json` hérité, veuillez mettre à jour le Mintlify CLI :

      `npm i -g mintlify@latest`
      Et exécuter la nouvelle commande de mise à niveau dans votre dépôt de documentation :

      `mintlify upgrade`
      Vous devriez maintenant utiliser le nouveau fichier de configuration `docs.json`. N'hésitez pas à supprimer le fichier `mint.json` de votre dépôt.
    </Note>
  </Accordion>
</AccordionGroup>

### Déployer vos modifications

<AccordionGroup>
  <Accordion icon="message-bot" title="Installer notre application Github">
    Notre application Github déploie automatiquement vos modifications sur votre site de documentation, vous n'avez donc pas besoin de gérer les déploiements vous-même. Vous pouvez trouver le lien d'installation sur votre [tableau de bord](https://dashboard.mintlify.com). Une fois le bot installé avec succès, une coche devrait apparaître à côté du hash de commit du dépôt.
  </Accordion>

  <Accordion icon="rocket" title="Pousser vos modifications">
    [Valider et pousser vos modifications vers
    Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
    pour que vos modifications soient mises à jour sur votre site de documentation. Si vous poussez vos modifications et ne constatez pas que l'application Github les a déployées avec succès, vous pouvez également mettre à jour manuellement votre documentation via notre [tableau de bord](https://dashboard.mintlify.com).
  </Accordion>
</AccordionGroup>
