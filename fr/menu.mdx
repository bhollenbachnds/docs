## Enregistrer votre modèle

Lorsque votre mise en page est terminée :

1. Cliquez sur le bouton **Enregistrer** dans la barre d'outils supérieure.
   <img src="/images/digital-signage/cms/create-a-template15.png" alt="Create A Template15" title="Create A Template15" style={{ width:"56%" }} />
2. Votre modèle sera enregistré et disponible dans la section **Modèles**.
3. Il peut désormais être utilisé comme base pour créer des messages.
   ![Create A Template16](/images/digital-signage/cms/create-a-template16.png)
