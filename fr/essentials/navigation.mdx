Le menu de navigation est la liste de liens sur chaque site web.

Vous devrez probablement le mettre à jour`docs.json` chaque fois que vous ajoutez une nouvelle page. Les pages n'apparaissent pas automatiquement.

## Syntaxe de navigation

Notre syntaxe de navigation est récursive, ce qui signifie que vous pouvez créer des groupes de navigation imbriqués. Vous n'avez pas besoin d'inclure `.mdx` dans les noms de pages.

<CodeGroup>
  ```json Regular Navigation
  "navigation": {
    "tabs": [
      {
        "tab": "Docs",
        "groups": [
          {
            "group": "Getting Started",
            "pages": ["quickstart"]
          }
        ]
      }
    ]
  }
  ```

  ```json Nested Navigation
  "navigation": {
    "tabs": [
      {
        "tab": "Docs",
        "groups": [
          {
            "group": "Getting Started",
            "pages": [
              "quickstart",
              {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
              }
            ]
          }
        ]
      }
    ]
  }
  ```
</CodeGroup>

## Dossiers

Placez simplement vos fichiers MDX dans des dossiers et mettez à jour les chemins dans `docs.json`.

Par exemple, pour avoir une page à `https://yoursite.com/your-folder/your-page` vous créeriez un dossier appelé `your-folder` contenant un fichier MDX appelé `your-page.mdx`.

<Warning>
  Vous ne pouvez pas utiliser `api` pour le nom d'un dossier à moins de l'imbriquer dans un autre dossier. Mintlify utilise Next.js qui réserve le dossier de premier niveau `api` pour les appels de serveur internes. Un nom de dossier tel que `api-reference` serait accepté.
</Warning>

```json Navigation With Folder
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Group Name",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```

## Pages cachées

Les fichiers MDX non inclus dans `docs.json` n'apparaîtront pas dans la barre latérale mais sont accessibles via la barre de recherche et en les liant directement.
