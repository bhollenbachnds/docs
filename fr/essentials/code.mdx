## Basic

### Inline Code

Pour désigner un `word` ou `phrase` comme code, entourez-le de backticks (\`).

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### Code Block

Utilisez des [fenced code blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) en entourant le code de trois backticks et en faisant suivre les backticks de début par le langage de programmation de votre extrait pour obtenir la coloration syntaxique. Vous pouvez également, si vous le souhaitez, écrire le nom de votre code après le langage de programmation.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
