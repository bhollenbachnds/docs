## Basique

### Code en ligne

Pour désigner un `word` ou `phrase` en tant que code, entourez-le de guillemets obliques (\`).

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### Bloc de code

Utilisez [des blocs de code délimités](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) en entourant le code de trois backticks et en faisant suivre les backticks initiaux du langage de programmation de votre extrait pour obtenir la coloration syntaxique. Optionnellement, vous pouvez également écrire le nom de votre code après le langage de programmation.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
