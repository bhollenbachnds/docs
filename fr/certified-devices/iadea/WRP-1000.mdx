Ce guide fournit des instructions étape par étape pour **installing and configuring PADS4 Viewer** sur l'appareil **IAdea WRP-1000 (Android 9)**.

***

## **1. Première utilisation**

Avant l'installation, assurez-vous que le **PADS4 Server is set up and configured properly**.

<Note>
  La version minimale requise du **PADS4 Server version** est **PADS4 Foundation 2023.1**.
</Note>

***

### **1.1 Réinitialisation de l'appareil**

Si l'appareil a été utilisé précédemment, effectuez une **factory reset** pour effacer le stockage et éviter les conflits logiciels.

<Steps>
  <Step title="Première étape">
    Ouvrez **IAdea Basic Settings**.
  </Step>

  <Step title="Deuxième étape">
    Cliquez sur **Advanced**.
  </Step>

  <Step title="Troisième étape">
    Sélectionnez **System → Factory Default**.
  </Step>

  <Step title="Quatrième étape">
    Cliquez sur **OK** pour confirmer la réinitialisation d'usine.
  </Step>
</Steps>

Après la réinitialisation, l'écran **Basic Settings screen** apparaîtra.

***

### **1.2 Vérification du firmware de l'appareil**

Pour assurer une compatibilité complète avec **PADS4 Viewer compatibility**, vérifiez que l'appareil exécute la version de firmware **3.4.1 or higher**.

<Steps>
  <Step title="Première étape">
    Ouvrez **IAdea Basic Settings**.
  </Step>

  <Step title="Deuxième étape">
    Cliquez sur **Advanced**.
  </Step>

  <Step title="Troisième étape">
    Sélectionnez **Information**.
  </Step>

  <Step title="Quatrième étape">
    La **Firmware Version** est affichée sur le côté droit de l'écran.
  </Step>
</Steps>

***

### **1.3 Mise à jour du firmware**

Si le firmware est obsolète, mettez-le à jour en suivant ces étapes :

<Steps>
  <Step title="Première étape">
    Ouvrez **IAdea Basic Settings**.
  </Step>

  <Step title="Deuxième étape">
    Cliquez sur **Advanced**.
  </Step>

  <Step title="Troisième étape">
    Sélectionnez **System → Firmware Update**.
  </Step>

  <Step title="Quatrième étape">
    Cliquez sur **Check Latest Firmware from Internet**.
  </Step>

  <Step title="Cinquième étape">
    Si une nouvelle version du firmware est disponible, sélectionnez **Update from Internet**.
  </Step>
</Steps>

***

## **2. Installation de PADS4 Viewer**

Suivez les étapes ci-dessous pour installer et configurer **PADS4 Viewer on the IAdea WRP-1000**.

***

### **2.1 Installation de PADS4IAdea à partir d'une clé USB**

Le fichier **PADS4IAdea.apk** est disponible sur le **PADS4 Server** at:

```
https://[yourPADS4Server]/[domain]/apps  
```

Alternativement, téléchargez la dernière version depuis [**here**](https://github.com/PADS4/PADS4-Viewer-IAdea/releases/latest) et placez-la dans le **PADS4 Server apps directory**.

#### **Étapes d'installation**

<Steps>
  <Step title="Première étape">
    Copiez le fichier **PADS4IAdea.apk** sur une **USB stick**.
  </Step>

  <Step title="Deuxième étape">
    Insérez la **USB stick** dans l'appareil **IAdea WRP-1000**.
  </Step>

  <Step title="Troisième étape">
    Ouvrez **IAdea Basic Settings**.
  </Step>

  <Step title="Quatrième étape">
    Cliquez sur **Advanced**.
  </Step>

  <Step title="Cinquième étape">
    Cliquez **three times** dans le **lower right corner** du menu **Advanced menu** pour accéder à Android.
  </Step>

  <Step title="Sixième étape">
    L'**Android Home Screen** sera maintenant visible.
  </Step>

  <Step title="Septième étape">
    Ouvrez l'**Files app**.
  </Step>

  <Step title="Huitième étape">
    Sélectionnez **USB Disk** et cliquez sur **PADS4IAdea.apk**.
  </Step>

  <Step title="Neuvième étape">
    Cliquez sur **CONTINUE** dans la boîte de dialogue de confirmation.
  </Step>

  <Step title="Dixième étape">
    Cliquez sur **INSTALL** lorsque vous y êtes invité.
  </Step>

  <Step title="Onzième étape">
    Une fois installé, cliquez sur **DONE**.
  </Step>
</Steps>

***

## **3. Configuration de PADS4 Viewer**

### **3.1 Configuration du démarrage automatique**

<Steps>
  <Step title="Première étape">
    Ouvrez **IAdea Basic Settings**.
  </Step>

  <Step title="Deuxième étape">
    Cliquez sur **Advanced**.
  </Step>

  <Step title="Troisième étape">
    Sélectionnez **Content → AppStart**.
  </Step>

  <Step title="Étape Quatre">
    Cliquez sur **(+)Advanced**.
  </Step>

  <Step title="Étape Cinq">
    Dans **Package Name**, saisissez :

    ```
    com.pads4.viewer.iadea  
    ```
  </Step>

  <Step title="Étape Six">
    Cliquez sur **Set** pour enregistrer le paramètre.
  </Step>
</Steps>

***

### **3.2 Connexion au PADS4 Server**

**Format d'adresse du PADS4 Server :**

```
protocol://host:port/hostpath/  
```

* **Protocol:** `http` ou `https` (dépend du certificat SSL).
* **Host:** Nom DNS ou **adresse IP** du **PADS4 Server**.
* **Port:** *(Optionnel - à spécifier uniquement si vous utilisez des ports non standard).*
* **Chemin d'hôte :** Configuré dans **System Management** *`(default: PADS4)`*.

#### **Étapes de connexion**

<Steps>
  <Step title="Étape Un">
    Saisissez l'**emplacement du PADS4 Server**.
  </Step>

  <Step title="Étape Deux - *Optionnel*">
    Spécifiez un **Viewer Name** (si laissé vide, PADS4 en générera un).
  </Step>

  <Step title="Étape Trois">
    Appuyez sur **Start**.
  </Step>

  <Step title="Étape Quatre">
    Accordez la **permission d'accéder aux médias et fichiers** lorsque demandé.
  </Step>

  <Step title="Étape Cinq">
    Le **PADS4 Viewer** sera **téléchargé et installé**.
  </Step>
</Steps>

***

## **4. Mise à jour du PADS4 Viewer**

Le **PADS4 Viewer** se **mettra à jour automatiquement** lorsque le **PADS4 Server** est mis à jour.

***

## **5. Fonctionnalités et capacités prises en charge**

| Fonctionnalité | Description                                                                                                    |
| :------------: | -------------------------------------------------------------------------------------------------------------- |
|        ✅       | **Mises à jour automatiques du logiciel** – Se met à jour lorsque le **PADS4 Server** est mis à jour.          |
|        ✅       | **Mode hors ligne** – L'appareil continue d'exécuter les présentations pendant **7 jours** en mode hors ligne. |
|        ✅       | **Support HTTP & HTTPS**                                                                                       |
|        ✅       | **Streaming vidéo** – HTTPS & HLS pris en charge.                                                              |
|        ✅       | **Maximum 2 vidéos FHD simultanément**                                                                         |
|        ✅       | **Éléments PowerPoint & PDF** – *(PowerPoint nécessite une installation sur le PADS4 Server)*.                 |
|        ✅       | **Gestion des appareils** – Contrôle à distance via **PADS4 CTRL Center & PADS4 Agent**.                       |
|        ✅       | **Redémarrage & Relance du Viewer** – Pris en charge.                                                          |
|        ❌       | **Wake-on-LAN** – Non pris en charge.                                                                          |
|        ✅       | **Suivi Proof of Play** – Surveille l'historique de lecture du contenu.                                        |
