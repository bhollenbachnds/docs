Ce guide fournit des instructions pour **l'installation et la configuration de PADS4 Viewer** sur **Giada F108D fonctionnant sous Windows 10**, ainsi que les capacités du système et les fonctionnalités prises en charge.

***

## **1. Instructions d'installation**

Pour installer **PADS4 Viewer** sur un **Giada F108D**, suivez le **processus d'installation Windows standard**.

Pour des **instructions détaillées étape par étape**, consultez le guide [**Installing PADS4 on Windows**](/fr/certified-devices/windows/Windows-Viewer).

### **1.1 Amélioration des performances de lecture vidéo**

<Tip>
  Lorsque vous utilisez PADS4 version 2025.2 et supérieure, cela n'est plus nécessaire car le Windows Viewer utilise désormais les filtres LAV les plus récents disponibles.
</Tip>

Pour une **expérience de lecture vidéo plus fluide**, nous **recommandons vivement de mettre à jour les filtres LAV** vers la dernière version disponible, actuellement 0.80.

[**Télécharger LAV Filters 0.80**](https://github.com/Nevcairiel/LAVFilters/releases/download/0.80/LAVFilters-0.80-Installer.exe)

***

## **2. Capacités et fonctionnalités prises en charge**

### **2.1 Fonctionnalités générales**

| Fonctionnalité | Description                                                                                                   |
| :------------: | ------------------------------------------------------------------------------------------------------------- |
|        ✅       | **Automatic PADS4 Software Updates** – Géré via une tâche planifiée dans **PADS4 Agent**.                     |
|        ✅       | **Offline Mode** – L'appareil continue d'exécuter les présentations et les planifications en mode hors ligne. |

***

### **2.2 Capacités de l'appareil**

| Fonctionnalité | Description                                                                                                      |
| :------------: | ---------------------------------------------------------------------------------------------------------------- |
|        ✅       | **Portrait Mode** – Pris en charge.                                                                              |
|        ✅       | **4K UHD Support**                                                                                               |
|        ✅       | **Dual HDMI** – *La deuxième sortie HDMI prend en charge uniquement le mode miroir (pas le mode bureau étendu).* |
|        ✅       | **Transfer Protocols** – HTTP, HTTPS                                                                             |
|        ✅       | **Turn Off Display** – Pris en charge.                                                                           |
|        ✅       | **Cluster Viewer** – Cet appareil peut être utilisé comme **Cluster Viewer**.                                    |

***

### **2.3 Lecture et prise en charge du contenu**

| Fonctionnalité | Description                                                                                |
| :------------: | ------------------------------------------------------------------------------------------ |
|        ✅       | **Maximum Videos Supported** – Jusqu'à **4 vidéos Full HD (FHD)** ou **1 vidéo UHD (4K)**. |
|        ✅       | **Supported Video Codecs** – H.264, H.265                                                  |
|        ✅       | **4K Video Playback** – Pris en charge.                                                    |
|        ✅       | **Video Rotation, Seamless Videos, Reflection, Skew, Flip** – Pris en charge.              |
|        ❌       | **Streaming Video** – Non pris en charge.                                                  |
|        ✅       | **Image Formats** – JPG, PNG, BMP, GIF                                                     |
|        ✅       | **Text, Line, Shape, Scrolling Text, Formatted Text, Tables, Interactive Buttons**         |
|        ✅       | **Web Element** – Pris en charge.                                                          |
|        ✅       | **PowerPoint & PDF Elements** – Pris en charge.                                            |

***

### **2.4 Connexions de données**

| Fonctionnalité | Description                                                                                                                     |
| :------------: | ------------------------------------------------------------------------------------------------------------------------------- |

|        ✅       | **Viewer Variables** – Simplifie la gestion du contenu sans modifier les présentations.                                         |
|        ✅       | **Property Bindings** – Permet des propriétés d'éléments dynamiques utilisant des sources de données externes.                  |

### **2.6 Gestion des appareils**
