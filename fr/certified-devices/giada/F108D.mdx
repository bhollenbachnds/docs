Ce guide fournit des instructions pour **l'installation et la configuration de PADS4 Viewer** sur **Giada F108D fonctionnant sous Windows 10**, ainsi que les capacités du système et les fonctionnalités prises en charge.

## **Instructions d'installation**

Pour installer **PADS4 Viewer** sur un **Giada F108D**, suivez le **processus d'installation Windows standard**.

Pour des **instructions détaillées étape par étape**, consultez le guide [**Installation de PADS4 sur Windows**](/fr/certified-devices/windows/Windows-Viewer).

### **Amélioration des performances de lecture vidéo**

<Tip>
  Lorsque vous utilisez PADS4 version 2025.2 et supérieure, cela n'est plus nécessaire car le Windows Viewer utilise désormais les derniers LAV Filters disponibles.
</Tip>

Pour une **expérience de lecture vidéo plus fluide**, nous **recommandons vivement de mettre à jour les LAV Filters** vers la dernière version disponible, actuellement 0.80.

[**Télécharger LAV Filters 0.80**](https://github.com/Nevcairiel/LAVFilters/releases/download/0.80/LAVFilters-0.80-Installer.exe)

## **Capacités et fonctionnalités prises en charge**

### **Fonctionnalités générales**

| Fonctionnalité | Description                                                                                              |
| :------------: | -------------------------------------------------------------------------------------------------------- |
|        ✅       | **Mises à jour automatiques du logiciel PADS4** – Gérées via une tâche planifiée dans **PADS4 Agent**.   |
|        ✅       | **Mode hors ligne** – L'appareil continue d'exécuter les présentations et les planifications hors ligne. |

| Fonctionnalité | Description                                                                                                        |
| :------------: | ------------------------------------------------------------------------------------------------------------------ |
|        ✅       | **Mode Portrait** – Pris en charge.                                                                                |
|        ✅       | **Prise en charge 4K UHD**                                                                                         |
|        ✅       | **Double HDMI** – *La deuxième sortie HDMI prend en charge uniquement le mode miroir (pas le mode bureau étendu).* |
|        ✅       | **Protocoles de transfert** – HTTP, HTTPS                                                                          |
|        ✅       | **Désactivation de l'écran** – Pris en charge.                                                                     |
|        ✅       | **Cluster Viewer** – Cet appareil peut être utilisé comme **Cluster Viewer**.                                      |

### **Lecture & Prise en charge du contenu**

| Fonctionnalité | Description                                                                                                 |
| :------------: | ----------------------------------------------------------------------------------------------------------- |
|        ✅       | **Nombre maximum de vidéos prises en charge** – Jusqu'à **4 vidéos Full HD (FHD)** ou **1 vidéo UHD (4K)**. |
|        ✅       | **Codecs vidéo pris en charge** – H.264, H.265                                                              |
|        ✅       | **Lecture vidéo 4K** – Pris en charge.                                                                      |
|        ✅       | **Rotation vidéo, vidéos sans coupure, reflet, inclinaison, retournement** – Pris en charge.                |
|        ❌       | **Vidéo en streaming** – Non pris en charge.                                                                |
|        ✅       | **Formats d'image** – JPG, PNG, BMP, GIF                                                                    |
|        ✅       | **Text, Line, Shape, Scrolling Text, Formatted Text, Tables, Interactive Buttons**                          |
|        ✅       | **Web Element** – Pris en charge.                                                                           |
|        ✅       | **PowerPoint & PDF Elements** – Pris en charge.                                                             |
