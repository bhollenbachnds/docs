Ce guide fournit des instructions étape par étape pour l'installation et la configuration de **PADS4 Viewer** sur **Crestron TSS-1070-B-S** dispositifs.

***

## **1. Première utilisation**

Pendant l'installation, vous devrez configurer l'emplacement du **PADS4 Server location**. Il est recommandé de **terminer l'installation du PADS4 Server d'abord** avant de configurer le dispositif Crestron.

<Note>La version minimale de **PADS4 Server version** requise pour exécuter **PADS4 on Crestron** est **PADS4 Foundation 2022.3**.</Note>

***

### **1.1 Réinitialisation de l'appareil**

Si l'appareil a été utilisé précédemment, une **réinitialisation d'usine** est recommandée pour effacer le stockage et éviter les interférences avec le logiciel **PADS4 Viewer software**.

<Steps>
  <Step title="Première étape">
    Connectez-vous à l'**interface de configuration web** de l'appareil Crestron.
  </Step>

  <Step title="Deuxième étape">
    Cliquez sur le **bouton flèche** à côté du **bouton Action** en haut à droite de l'écran.
  </Step>

  <Step title="Troisième étape">
    Sélectionnez **Restore**.
  </Step>

  <Step title="Quatrième étape">
    Cliquez sur **Yes** dans la boîte de dialogue de confirmation.
  </Step>
</Steps>

***

### **1.2 Mise à jour du firmware**

**PADS4 est inclus avec le firmware Crestron** et ne peut pas être installé séparément.

<Note>La version minimale du firmware qui inclut **PADS4** est **2.003.1011**.</Note>

***

## **2. Installation de PADS4 Viewer**

Suivez les instructions ci-dessous pour exécuter **PADS4 Viewer** sur Crestron.

***

### **2.1 Définir le mode d'application Crestron sur PADS4**

**PADS4** est inclus dans le **Crestron firmware** et peut être installé en sélectionnant **PADS4 as the Application Mode** dans l'**interface de configuration web**.

<Steps>
  <Step title="Première étape">
    Connectez-vous à l'**interface de configuration web** de l'appareil Crestron.
  </Step>

  <Step title="Deuxième étape">
    Cliquez sur l'**onglet Paramètres**.
  </Step>

  <Step title="Troisième étape">
    Sélectionnez **Applications**.
  </Step>

  <Step title="Quatrième étape">
    Sélectionnez **PADS4** dans le menu déroulant **Application Mode dropdown**.
  </Step>

  <Step title="Cinquième étape">
    Cliquez sur **"Save Changes"** en haut à droite de l'interface web.
  </Step>

  <Step title="Sixième étape">
    Cliquez sur **"Yes, Reboot Now"** dans la boîte de dialogue de confirmation.
  </Step>
</Steps>

***

### **2.2 Connexion au PADS4 Server**

Après l'installation, **PADS4 Viewer** doit être connecté au **PADS4 Server**.

#### **Format d'adresse du PADS4 Server**

```
protocol://host:port/hostpath/  
```

* **Protocol:** Utilisez `HTTP` ou `HTTPS`, selon qu'un **certificat SSL valide** est utilisé.
* **Host:** Le **nom DNS** ou **l'adresse IP** du **PADS4 Server**.
* **Port:** Optionnel – si des **ports HTTP/HTTPS non standard** sont utilisés, spécifiez le **numéro de port**.
* **Chemin d'hôte :** Configuré dans **PADS4 System Management** (par défaut : `PADS4`).

**Example:**

```
http://localhost/pads4  
```

#### **Étapes de connexion**

<Steps>
  <Step title="Première étape">
    Entrez l'emplacement du **PADS4 Server Location**.
  </Step>

  <Step title="Deuxième étape">
    (Optionnel) Spécifiez un **Viewer Name**. Si laissé vide, le **PADS4 Server** en générera un.
  </Step>

  <Step title="Troisième étape">
    Appuyez sur le bouton **Start**.
  </Step>

  <Step title="Quatrième étape">
    Accordez la **permission de stocker définitivement du contenu** sur l'appareil lorsque vous y êtes invité.
  </Step>

  <Step title="Cinquième étape">
    Si le **Android System WebView** est obsolète, il sera automatiquement mis à jour.
  </Step>

  <Step title="Sixième étape">
    **PADS4 Viewer** sera téléchargé. Attendez que l'installation soit terminée, et le **Viewer démarrera** automatiquement.
  </Step>
</Steps>

***

### **2.3 Ajuster les paramètres de PADS4 Viewer**

Pour **ajuster les paramètres de PADS4 Viewer** directement sur l'**écran Crestron** :

* **Maintenez enfoncé le bouton gauche de la souris** ou **touchez l'écran pendant 3 secondes** pour ouvrir l'**écran d'options**.
* Pour **revenir à l'application PADS4**, appuyez sur le bouton **Launcher Settings**.

***

## **3. Configuration du capteur d'occupation Crestron**

**PADS4 prend en charge le capteur d'occupation Crestron.** Suivez les étapes ci-dessous pour **associer le capteur** et **l'enregistrer dans PADS4 Workspace**.

### **3.1 Appairage du capteur d'occupation avec l'écran**

<Steps>
  <Step title="Première étape">
    Connectez-vous à l'**interface de configuration web** de l'écran Crestron.
  </Step>

  <Step title="Étape Deux">
    Cliquez sur **Settings tab**.
  </Step>

  <Step title="Étape Trois">
    Ouvrez **Device Pairing**.
  </Step>

  <Step title="Étape Quatre">
    Sélectionnez **POE Occupancy Sensor** dans le **Device Model dropdown**.
  </Step>

  <Step title="Étape Cinq">
    Saisissez l'**IP address** ou **FQDN** du **sensor**.
  </Step>

  <Step title="Étape Six">
    Cliquez sur **Save Changes** en haut à droite de l'interface web.
  </Step>

  <Step title="Étape Sept">
    Après l'enregistrement, le **Pairing Status** devrait passer à **Online**.
  </Step>
</Steps>

***

### **3.2 Trouver le capteur d'occupation dans PADS4 Workspace**

Après l'appairage, le capteur sera **visible in PADS4 Workspace Admin**.

<Steps>
  <Step title="Étape Un">
    Connectez-vous à **PADS4 Workspace Admin**.
  </Step>

  <Step title="Étape Deux">
    Cliquez sur **Occupancy Sensors**.
  </Step>

  <Step title="Étape Trois">
    Cliquez sur l'**Sensors tab**.
  </Step>

  <Step title="Étape Quatre">
    Le **sensor** sera listé dans le **table**.
  </Step>
</Steps>

## **4. Fonctionnalités prises en charge et limitations**

### **Fonctionnalités générales**

| Fonctionnalité | Description                                                    |
| :------------: | -------------------------------------------------------------- |
|        ✅       | **Portrait Mode** – Pris en charge.                            |
|        ✅       | **HTTP/HTTPS** – Pris en charge.                               |
|        ✅       | **Offline Content Library Storage** – Mis en cache localement. |
|        ❌       | **Block Remote Control** – Non pris en charge.                 |
|        ✅       | **Remote Management via PADS4 CTRL Center & PADS4 Agent**      |

### **Contenu et lecture**

| Fonctionnalité | Description                                                                       |
| :------------: | --------------------------------------------------------------------------------- |
|        ✅       | **Supports JPG, PNG, BMP, GIF, Text, Shapes, Tables, Interactive Buttons**        |
|        ✅       | **Single Video Playback**                                                         |
|        ❌       | **Streaming Video** – Non pris en charge.                                         |
|        ✅       | **Content from PADS4 Content Library & URL**                                      |
|        ✅       | **Web Element** – Utilisation de **interactive Address Bar** non prise en charge. |

### **Gestion des appareils**

| Fonctionnalité | Description                                      |
| :------------: | ------------------------------------------------ |
|        ✅       | **Reboot & Restart Viewer** – Pris en charge.    |
|        ❌       | **Shutdown & Wake-on-LAN** – Non pris en charge. |
|        ✅       | **Retrieve Screenshots & System Logs**           |
|        ✅       | **Install Fonts & Proof of Play Tracking**       |
