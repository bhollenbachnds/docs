Ce guide fournit des instructions étape par étape pour installer et configurer **PADS4 Viewer** sur **Crestron TSS-1070-B-S** appareils.

## **1. Première utilisation**

Lors de l'installation, vous devrez configurer l'emplacement du **serveur PADS4**. Il est recommandé de **terminer l'installation du serveur PADS4 en premier** avant de configurer l'appareil Crestron.

<Note>
  La version minimale du **serveur PADS4** requise pour exécuter **PADS4 sur Crestron** est **PADS4 Foundation 2022.3**.
</Note>

### **1.1 Réinitialisation de l'appareil**

Si l'appareil a déjà été utilisé, une **réinitialisation d'usine** est recommandée pour effacer le stockage et éviter toute interférence avec le **logiciel PADS4 Viewer**.

<Steps>
  <Step title="Étape Un">
    Se connecter au **interface de configuration web** du périphérique Crestron.
  </Step>

  <Step title="Étape deux">
    Cliquez sur le **bouton fléché** à côté du **bouton Action** en haut à droite de l'écran.
  </Step>

  <Step title="Étape trois">
    Sélectionnez **Restore**.
  </Step>

  <Step title="Étape quatre">
    Cliquez sur **Yes** dans la boîte de dialogue de confirmation.
  </Step>
</Steps>

### **1.2 Mise à jour du firmware**

**PADS4 est inclus dans le firmware Crestron** et ne peut pas être installé séparément.

<Note>
  La version minimale du firmware qui inclut **PADS4** est **2.003.1011**.
</Note>

## **2. Installation de PADS4 Viewer**

Suivez les instructions ci-dessous pour exécuter **PADS4 Viewer** sur Crestron.

### **2.1 Définir le mode d'application Crestron sur PADS4**

**PADS4** est inclus dans le **firmware Crestron** et peut être installé en sélectionnant **PADS4 comme mode d'application** dans l'**interface de configuration web**.

<Steps>
  <Step title="Étape un">
    Connectez-vous à l'**interface de configuration web** du périphérique Crestron.
  </Step>

  <Step title="Étape deux">
    Cliquez sur l'**onglet Paramètres**.
  </Step>

  <Step title="Étape trois">
    Sélectionnez **Applications**.
  </Step>

  <Step title="Étape quatre">
    Sélectionnez **PADS4** dans le **menu déroulant Mode d'application**.
  </Step>

  <Step title="Étape cinq">
    Cliquez sur **"Enregistrer les modifications"** en haut à droite de l'interface web.
  </Step>

  <Step title="Étape six">
    Cliquez sur **"Yes, Reboot Now"** dans la boîte de dialogue de confirmation.
  </Step>
</Steps>

### **2.2 Connexion au PADS4 Server**

Après l'installation, **PADS4 Viewer** doit être connecté au **PADS4 Server**.

#### **Format d'adresse du PADS4 Server**

```
protocol://host:port/hostpath/  
```

* **Protocol:** Utilisez `HTTP` ou `HTTPS`, selon qu'un **certificat SSL valide** est utilisé.
* **Host:** Le **nom DNS** ou **adresse IP** du **PADS4 Server**.
* **Port:** Facultatif – si des **ports HTTP/HTTPS non par défaut** sont utilisés, spécifiez le **numéro de port**.
* **Chemin d'hôte :** Configuré dans **PADS4 System Management** (par défaut : `PADS4`).

**Example:**

```
http://localhost/pads4  
```

#### **Étapes de connexion**

<Steps>
  <Step title="Étape un">
    Entrez l'**emplacement du PADS4 Server**.
  </Step>

  <Step title="Étape deux">
    (Facultatif) Spécifiez un **nom de Viewer**. Si laissé vide, le **PADS4 Server** en générera un.
  </Step>

  <Step title="Étape trois">
    Appuyez sur le bouton **Start**.
  </Step>

  <Step title="Étape quatre">
    Accordez la **permission de stocker du contenu de façon permanente** sur le périphérique lorsque vous y êtes invité.
  </Step>

  <Step title="Étape cinq">
    Si l'**Android System WebView** est obsolète, il sera automatiquement mis à jour.
  </Step>

  <Step title="Étape six">
    **PADS4 Viewer** sera téléchargé. Attendez la fin de l'installation, et le **Viewer démarrera** automatiquement.
  </Step>
</Steps>

### **2.3 Ajuster les paramètres de PADS4 Viewer**

Pour **ajuster les paramètres de PADS4 Viewer** directement sur l'**écran Crestron** :

* **Maintenez le bouton gauche de la souris enfoncé** ou **touchez l'écran pendant 3 secondes** pour ouvrir l'**écran Options**.
* Pour **revenir à l'application PADS4**, appuyez sur le bouton **Launcher Settings**.

## **3. Configuration du capteur d'occupation Crestron**

**PADS4 prend en charge le capteur d'occupation Crestron.** Suivez les étapes ci-dessous pour **associer le capteur** et **l'enregistrer dans PADS4 Workspace**.

### **3.1 Association du capteur d'occupation avec l'écran**

<Steps>
  <Step title="Étape un">
    Connectez-vous à l'**interface de configuration web** de l'écran Crestron.
  </Step>

  <Step title="Étape deux">
    Cliquez sur l'**onglet Paramètres**.
  </Step>

  <Step title="Étape trois">
    Ouvrez **Device Pairing**.
  </Step>

  <Step title="Étape quatre">
    Sélectionnez **POE Occupancy Sensor** dans le **menu déroulant Modèle de périphérique**.
  </Step>

  <Step title="Étape cinq">
    Entrez l'**adresse IP** ou le **FQDN** du **capteur**.
  </Step>

  <Step title="Étape six">
    Cliquez sur **Save Changes** en haut à droite de l'interface web.
  </Step>

  <Step title="Étape sept">
    Après l'enregistrement, le **statut d'association** devrait passer à **En ligne**.
  </Step>
</Steps>

### **3.2 Recherche du capteur d'occupation dans PADS4 Workspace**

Après le couplage, le capteur sera **visible dans PADS4 Workspace Admin**.

<Steps>
  <Step title="Étape Un">
    Connectez-vous à **PADS4 Workspace Admin**.
  </Step>

  <Step title="Étape Deux">
    Cliquez sur **Occupancy Sensors**.
  </Step>

  <Step title="Étape Trois">
    Cliquez sur l'**onglet Sensors**.
  </Step>

  <Step title="Étape Quatre">
    Le **capteur** sera répertorié dans le **tableau**.
  </Step>
</Steps>

## **4. Fonctionnalités prises en charge & Limitations**

### **Fonctionnalités générales**

| Fonctionnalité | Description                                                    |
| :------------: | -------------------------------------------------------------- |
|        ✅       | **Portrait Mode** – Pris en charge.                            |
|        ✅       | **HTTP/HTTPS** – Pris en charge.                               |
|        ✅       | **Offline Content Library Storage** – Mis en cache localement. |
|        ❌       | **Block Remote Control** – Non pris en charge.                 |
|        ✅       | **Remote Management via PADS4 CTRL Center & PADS4 Agent**      |

### **Contenu & Lecture**

| Fonctionnalité | Description                                                                                      |
| :------------: | ------------------------------------------------------------------------------------------------ |
|        ✅       | **Prend en charge JPG, PNG, BMP, GIF, Texte, Formes, Tableaux, Boutons interactifs**             |
|        ✅       | **Lecture vidéo unique**                                                                         |
|        ❌       | **Streaming vidéo** – Non pris en charge.                                                        |
|        ✅       | **Contenu depuis PADS4 Content Library & URL**                                                   |
|        ✅       | **Web Element** – L'utilisation de la **barre d'adresse interactive** n'est pas prise en charge. |

### **Gestion des appareils**

| Fonctionnalité | Description                                               |
| :------------: | --------------------------------------------------------- |
|        ✅       | **Redémarrage & Redémarrage du Viewer** – Pris en charge. |
|        ❌       | **Arrêt & Wake-on-LAN** – Non pris en charge.             |
|        ✅       | **Récupération des captures d'écran & journaux système**  |
|        ✅       | **Installation de polices & suivi Proof of Play**         |

</Warning>

<Warning>
