## **Première utilisation**

Pendant l'installation, vous devrez configurer l'emplacement du **PADS4 Server location**. Il est recommandé de **terminer d'abord l'installation du PADS4 Server** avant de configurer l'appareil.

<Note>
  La version minimale requise du **PADS4 Server version** pour l'application **PADS4 AOpen App** est **2024.1**.
</Note>

## **Modèles pris en charge**

* AOpen AT1032TP
* AOpen eTILE-X1032TP

<Note>
  La prise en charge du X1032TP a été ajoutée dans la version 2025.1 de PADS4. Si vous avez besoin de prendre en charge ce modèle particulier sur des versions plus anciennes de PADS4, remplacez l'APK sur votre serveur par la dernière version disponible du téléchargement de PADS4AOpen.apk.
</Note>

## **Installation du PADS4 AOpen Viewer**

Le fichier **PADS4AOpen.apk** est disponible sur le **PADS4 Server** dans le dossier **Apps** du répertoire d'installation :

```
C:\Program Files\PADS4\Foundation\Apps  
```

Alternativement, téléchargez la dernière version [**here**](https://github.com/PADS4/PADS4-Viewer-AOpen/releases/download/v1.0/PADS4AOpen.apk).

<Note>
  Si vous téléchargez la dernière version, assurez-vous que l'APK est également placé dans le répertoire **Apps** sur le **PADS4 Server**. Si le serveur contient une version différente, il installera cette version lors de la connexion.
</Note>

### **Étapes d'installation**

<Steps>
  <Step title="Première étape">
    Copiez le fichier **PADS4AOpen.apk** sur une **clé USB**.
  </Step>

  <Step title="Deuxième étape">
    Insérez la **clé USB** dans l'appareil.
  </Step>

  <Step title="Troisième étape">
    Ouvrez l'application **File Manager app** sur l'appareil.
  </Step>

  <Step title="Quatrième étape">
    Naviguez jusqu'à la **clé USB** et ouvrez **PADS4AOpen.apk**.
  </Step>

  <Step title="Cinquième étape">
    Un avertissement apparaîtra indiquant que l'installation à partir de sources inconnues n'est pas autorisée.
  </Step>

  <Step title="Sixième étape">
    Allez dans **Settings** et autorisez l'installation à partir de cette source.
  </Step>

  <Step title="Septième étape">
    Appuyez sur le bouton **Back** et installez l'application.
  </Step>

  <Step title="Huitième étape">
    Après l'installation, ouvrez l'application **PADS4 AOpen**.
  </Step>
</Steps>

***

## **Connexion au PADS4 Server**

Une fois installé, le **PADS4 Viewer** doit être connecté au **PADS4 Server**.

### **Format d'adresse du PADS4 Server :**

```
protocol://host:port/hostpath/  
```

* **Protocol:** Utilisez `http` ou `https` selon qu'un certificat SSL est appliqué ou non.
* **Host:** Le nom DNS ou l'adresse IP du **PADS4 Server**.
* **Port:** À spécifier uniquement si le **PADS4 Server** n'utilise pas les ports HTTP/HTTPS par défaut.
* **Chemin d'hôte :** Configuré dans **System Management** (par défaut : `PADS4`).

**Example:**

```
http://localhost/pads4  
```

### **Étapes de connexion**

<Steps>
  <Step title="Première étape">
    Entrez l'emplacement du **PADS4 Server Location**.
  </Step>

  <Step title="Deuxième étape">
    (Optionnel) Spécifiez un **Viewer Name**. Si laissé vide, le **PADS4 Server** en générera un.
  </Step>

  <Step title="Troisième étape">
    Définissez **PADS4 AOpen Viewer** comme **Home app** et sélectionnez **Always**.
  </Step>

  <Step title="Quatrième étape">
    Appuyez sur **Start**.
  </Step>

  <Step title="Cinquième étape">
    Autorisez l'application **PADS4 AOpen application** à modifier les paramètres système.
  </Step>

  <Step title="Sixième étape">
    Activez **"Allow to modify system settings"** et appuyez sur le bouton **Back**.
  </Step>

  <Step title="Septième étape">
    Appuyez à nouveau sur **Start**.
  </Step>

  <Step title="Huitième étape">
    Accordez **l'autorisation de stocker du contenu** sur l'appareil lorsque vous y êtes invité.
  </Step>
</Steps>

Une fois terminé, le **PADS4 Viewer** devrait se connecter au **PADS4 Server**. Des paramètres supplémentaires tels que **l'orientation de l'écran** et **les restrictions de télécommande** peuvent être configurés en utilisant **PADS4 Agent** ou **Viewer Settings**.

***

## **Ajustement des options du PADS4 Viewer**

* **Appuyez longuement sur l'écran** pour ouvrir le menu.
* Alternativement, les options peuvent être modifiées à distance via **PADS4 CTRL Center** ou **PADS4 Agent**.

***

## **Accès au menu Android**

1. Ouvrez **Viewer Menu**.
2. Cliquez sur **Launcher Settings**.
3. Cliquez sur l'**Icône d'engrenage** dans le coin inférieur droit.
4. Le menu **Android Settings** s'ouvrira.

<Note>
  Si **"Hide Status Bar"** est activé, la seule façon de quitter Android Settings est de **redémarrer l'appareil** en déconnectant l'alimentation.
</Note>

***

## **Désactivation de la barre de navigation Android**

(Optionnel) La barre de navigation peut être désactivée pour empêcher toute interaction non autorisée.

### **Steps:**

1. Ouvrez **Viewer Options**.
2. Cliquez sur **Launcher Settings**.
3. Cliquez sur l'**Icône d'engrenage** dans le coin inférieur droit.
4. Sélectionnez **System Tools**.
5. Cliquez sur **Disable Navigation Bar**.

<Note>
  Si désactivé, la seule façon de quitter les paramètres est de **redémarrer l'appareil** en déconnectant l'alimentation.
</Note>

***

## **Effectuer une réinitialisation d'usine**

Pour restaurer l'appareil aux paramètres d'usine :

1. Ouvrez **Android Settings**.
2. Naviguez vers **Backup & Reset**.
3. Sélectionnez **Factory Data Reset**.
4. (Optionnel) Sélectionnez **Erase SD Card**.
5. Appuyez sur **Reset Tablet**.
6. Confirmez avec **Erase Everything**.

## **Remarques importantes**

<Warning>
  **Intégrité des fichiers de contenu** – La **PADS4 Content Library** vérifie l'intégrité des fichiers en utilisant la **somme de contrôle MD5** pendant la synchronisation.
</Warning>

<Warning>
  **Limitations de Proof of Play** – Un maximum de **50 000 enregistrements** peuvent être stockés localement pendant les périodes hors ligne. Lorsque la mémoire est pleine, les enregistrements les plus anciens sont supprimés en premier.
</Warning>
