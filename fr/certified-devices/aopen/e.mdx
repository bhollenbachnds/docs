Ce guide fournit des informations sur l'installation et la configuration de PADS4 Viewer sur les ordinateurs Aopen eTILE-Series fonctionnant sous Windows 10 ou Windows 11.

## Compatibilité

* Systèmes d'exploitation pris en charge : Windows 10 ou Windows 11
* Appareils pris en charge : ordinateurs AOPEN eTILE-Series

## Instructions d'installation

Pour installer PADS4 Viewer sur un appareil AOPEN eTILE-Series, suivez la procédure d'installation Windows standard.

Pour des instructions détaillées, étape par étape, consultez le guide « Installation de PADS4 sur Windows ».
