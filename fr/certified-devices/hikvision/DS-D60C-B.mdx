Ce guide fournit des instructions étape par étape pour **l'installation et la configuration du PADS4 Viewer** sur l'appareil **Hikvision DS-D60C-B**.

## **1. Première utilisation**

Avant l'installation, il est recommandé de **terminer la configuration du PADS4 Server**.

<Note>
  La version minimale du **PADS4 Server** requise est **PADS4 Foundation 2023.1**.
</Note>

### **1.1 Réinitialisation de l'appareil**

Si l'appareil a déjà été utilisé, effectuez une **réinitialisation d'usine** pour effacer le stockage et éviter les conflits logiciels.

<Steps>
  <Step title="Étape un">
    Ouvrez le **écran d'accueil** de l'appareil.
  </Step>

  <Step title="Étape deux">
    Ouvrir l'**application Paramètres**.
  </Step>

  <Step title="Étape trois">
    Cliquer sur **Système** dans le **menu Paramètres**.
  </Step>

  <Step title="Étape quatre">
    Faire défiler vers le bas et sélectionner **Restaurer les paramètres d'usine**.
  </Step>

  <Step title="Étape cinq">
    Cliquer sur **Confirmer** dans la boîte de dialogue qui apparaît.
  </Step>
</Steps>

### **1.2 Mise à jour du firmware**

S'assurer que le **firmware est mis à jour** vers au moins **V1.7.1 build 20230424** avant d'installer PADS4.

<Steps>
  <Step title="Étape un">
    Copier le **update.zip** fichier firmware dans le **répertoire racine** d'une clé USB.
  </Step>

  <Step title="Étape deux">
    Insérer la **clé USB** dans l'appareil.
  </Step>

  <Step title="Étape trois">
    L'appareil devrait détecter la **mise à jour du firmware USB** et demander l'installation.
  </Step>

  <Step title="Étape quatre">
    Suivre les instructions à l'écran pour installer la **mise à jour du firmware**.
  </Step>
</Steps>

## **2. Installation de PADS4 Viewer**

Suivre les étapes ci-dessous pour installer et configurer **PADS4 Viewer sur le Hikvision DS-D60C-B**.

### **2.1 Installation de PADS4 depuis une clé USB**

Le **PADS4Hikvision.apk** fichier est disponible sur le **PADS4 Server** in:

```
C:\Program Files\PADS4\Foundation\Apps  
```

Alternativement, télécharger la dernière version depuis [**ici**](https://github.com/PADS4/PADS4-Viewer-HikVision/releases/latest) et le placer dans le **répertoire des applications du PADS4 Server** pour s'assurer que la version correcte est installée.

#### **Étapes d'installation**

<Steps>
  <Step title="Étape un">
    Copier le **PADS4Hikvision.apk** fichier sur une **clé USB**.
  </Step>

  <Step title="Étape deux">
    Insérer la **clé USB** dans l'**appareil Hikvision DS-D60C-B**.
  </Step>

  <Step title="Étape trois">
    Ouvrir l'**écran d'accueil** de l'appareil.
  </Step>

  <Step title="Étape quatre">
    Ouvrir l'**application FileBrowser**.
  </Step>

  <Step title="Étape cinq">
    Sélectionner **USB DISK** dans la liste.
  </Step>

  <Step title="Étape six">
    Cliquer sur **PADS4Hikvision.apk**.
  </Step>

  <Step title="Étape sept">
    Lorsque la question **"Voulez-vous installer cette application ?"** s'affiche, cliquer sur **Installer**.
  </Step>

  <Step title="Étape huit">
    Une fois installé, cliquer sur **Ouvrir** pour lancer l'**application PADS4 Hikvision**.
  </Step>
</Steps>

### **2.2 Connexion au PADS4 Server**

**Format de l'adresse du PADS4 Server :**

```
protocol://host:port/hostpath/  
```

* **Protocol:** `http` ou `https` (dépend du certificat SSL).
* **Host:** Nom DNS ou **adresse IP** du **PADS4 Server**.
* **Port:** *(Optionnel - à spécifier uniquement si des ports non par défaut sont utilisés).*
* **Chemin d'hôte :** Configuré dans **System Management** *`(default: PADS4)`*.

#### **Étapes de connexion**

<Steps>
  <Step title="Étape un">
    Saisir l'**emplacement du PADS4 Server**.
  </Step>

  <Step title="Étape deux - *Optionnel*">
    Spécifier un **Nom de Viewer** (si laissé vide, PADS4 en générera un).
  </Step>

  <Step title="Étape trois">
    Appuyer sur **Démarrer**.
  </Step>

  <Step title="Étape quatre">
    Accorder **l'autorisation d'accéder aux photos, médias et fichiers** lorsque demandé.
  </Step>

  <Step title="Étape cinq">
    Le **PADS4 Viewer** sera **téléchargé et installé**.
  </Step>
</Steps>

### **2.3 Ajustement des paramètres du PADS4 Viewer**

* **Maintenir le bouton gauche de la souris enfoncé** ou **toucher l'écran pendant 3 secondes** pour accéder aux **Paramètres du Viewer**.

### **2.4 Mise à jour du PADS4 Viewer & de l'application PADS4 Hikvision**

Le **PADS4 Viewer & l'application PADS4 Hikvision** se **mettra à jour automatiquement** lorsque le **PADS4 Server** est mis à jour.

### **2.5 Accès au menu Android**

Si vous accédez au **menu Android** pendant que **PADS4 est en cours d'exécution**, l'appareil va **redémarrer dans 30 secondes** (car le **Viewer Monitor Service** détecte l'inactivité).

Pour éviter le redémarrage, **arrêter l'application PADS4 Hikvision dans les 30 secondes**.

## **3. Fonctionnalités prises en charge & Limitations**

### **3.1 Fonctionnalités générales**

| Fonctionnalité | Description                                                                                                                       |
| :------------: | --------------------------------------------------------------------------------------------------------------------------------- |
|        ✅       | **Mises à jour automatiques du logiciel PADS4** – Le Viewer se met à jour automatiquement lorsque le PADS4 Server est mis à jour. |
|        ✅       | **Mode hors ligne** – L'appareil continue d'exécuter les présentations pendant **7 jours** hors ligne.                            |

***

### **3.2 Capacités de l'appareil**

| Fonctionnalité | Description                                                                                                                    |
| :------------: | ------------------------------------------------------------------------------------------------------------------------------ |
|        ✅       | **Mode Portrait** – Pris en charge. L'appareil pivote automatiquement.                                                         |
|        ✅       | **Blocage de la télécommande** – Empêche l'interaction avec l'écran via la télécommande.                                       |
|        ✅       | **Prend en charge HTTP & HTTPS**                                                                                               |
|        ✅       | **Éteindre l'écran** – L'écran s'éteint lorsqu'aucune présentation n'est planifiée, réduisant ainsi la consommation d'énergie. |
|        ✅       | **Cluster Viewer** – Prend en charge le **Cluster Mode**.                                                                      |
|        ✅       | **Vérification de l'intégrité des fichiers MD5** – Garantit l'intégrité du contenu.                                            |
|        ❌       | **Contrôle du volume principal** – Le volume ne peut être ajusté qu'au niveau de l'élément.                                    |

***
