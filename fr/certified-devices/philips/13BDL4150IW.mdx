Ce guide fournit des instructions étape par étape pour **installer et configurer PADS4 Viewer** sur **les écrans Philips T-line (13BDL4150IW)**.

***

## **1. Première utilisation**

Avant l'installation, assurez-vous que le **PADS4 Server est configuré et paramétré correctement**.

📌 **Version minimale requise du PADS4 Server :** **PADS4 Foundation 2025.1**

Lors de la première mise en marche de l'appareil, l'**Assistant de configuration** s'affichera. Si ce n'est pas le cas, **réinitialisez l'appareil** comme décrit dans **1.2 Réinitialisation de l'appareil**.

***

### **1.1 Configuration de l'appareil**

<Steps>
  <Step title="Étape un">
    Sur l'**Écran d'accueil**, sélectionnez votre **langue** préférée et cliquez sur **Start**.
  </Step>

  <Step title="Étape deux">
    Sur l'écran **Security Settings**, assurez-vous que le **SICP Network Port** est **activé**. *(Les autres paramètres peuvent être activés ou désactivés selon les besoins.)*
  </Step>

  <Step title="Étape trois">
    Sur l'écran **Signage Display Settings**, appuyez sur **Skip**.
  </Step>

  <Step title="Étape quatre">
    Sur l'écran **Network Settings**, sélectionnez et configurez le **type de réseau**.
  </Step>

  <Step title="Étape cinq">
    Sur l'écran **Clone Media Files**, appuyez sur **Skip**.
  </Step>

  <Step title="Étape six">
    Appuyez sur **Finish** pour terminer la configuration.
  </Step>
</Steps>

L'appareil affichera **Philips Media Player App**.

***

### **1.2 Réinitialisation de l'appareil**

Si l'appareil a déjà été utilisé, effectuez une **réinitialisation d'usine** pour effacer le stockage et éviter les conflits logiciels.

<Steps>
  <Step title="Étape un">
    Ouvrez le **Android Menu** en appuyant sur **Home** puis sur le bouton **Volume Down** situé derrière l'**écran**.
  </Step>

  <Step title="Étape deux">
    Saisissez le **mot de passe administrateur** *(Par défaut : 1234)*.
  </Step>

  <Step title="Étape trois">
    Ouvrez **Settings**.
  </Step>

  <Step title="Étape quatre">
    Sélectionnez **Signage Display → System Tools → Factory Reset**.
  </Step>

  <Step title="Étape cinq">
    Cliquez sur **Factory Reset** et confirmez en appuyant sur **OK**.
  </Step>
</Steps>

Après la réinitialisation, l'**Écran d'accueil** s'affichera. Suivez les étapes de la section **1.1 Configuration de l'appareil**.

***

## **2. Installation de PADS4 Viewer**

Suivez ces étapes pour installer le **PADS4 Viewer** sur **Philips T-line (10BDL4551T)**.

***

### **2.1 Vérification du firmware de l'appareil**

Assurez-vous que la **version correcte du firmware** est installée pour bénéficier de toutes les fonctionnalités du **PADS4 Viewer**.

<Steps>
  <Step title="Étape un">
    Ouvrez le **Android Menu** en appuyant sur **Home + Volume Down**.
  </Step>

  <Step title="Étape deux">
    Saisissez le **mot de passe administrateur** *(Par défaut : 1234)*.
  </Step>

  <Step title="Étape trois">
    Ouvrez **Settings**.
  </Step>

  <Step title="Étape quatre">
    Sélectionnez **About**.
  </Step>

  <Step title="Étape cinq">
    Assurez-vous que le **Build Number** est **FB01.03T** *(requis pour PADS4)*.
  </Step>
</Steps>

***

### **2.2 Mise à jour du firmware de l'appareil**

Pour mettre à jour le **firmware**, suivez les **instructions Philips** :

📌 **Guide de mise à jour du firmware :** [Download Firmware Instructions](https://www.dropbox.com/s/isi1ksa6pyy9oig/10BDL4551TLatestReleaseFW.pdf?dl=0)\
📌 **Centre de téléchargement Philips :** [Download Center](https://www.dropbox.com/s/fje9ge0ao8at6rq/One_Link.xlsm?dl=0)

***

### **2.3 Installation de PADS4Philips depuis une clé USB**

Le **PADS4Philips.apk** fichier se trouve sur le **PADS4 Server** in:

```
C:\Program Files\PADS4\Foundation\Apps
```

Vous pouvez également télécharger la dernière version depuis [**ce lien**](http://go.pads4.com/?fwlink=1006) et le placer dans le **répertoire apps sur le PADS4 Server**.

#### **Étapes d'installation**

<Steps>
  <Step title="Étape un">
    Copiez le fichier **PADS4Philips.apk** sur une **clé USB**.
  </Step>

  <Step title="Étape deux">
    Insérez la **clé USB** dans l'**appareil Philips T-line**.
  </Step>

  <Step title="Étape trois">
    Ouvrez le **Android Menu** en appuyant sur **Home + Volume Down**.
  </Step>

  <Step title="Étape quatre">
    Saisissez le **mot de passe administrateur** *(Par défaut : 1234)*.
  </Step>

  <Step title="Étape cinq">
    Sélectionner **Storage → USB Drive** et ouvrir **PADS4Philips.apk**.
  </Step>

  <Step title="Étape Six">
    Sélectionner **Installer** puis **Ouvrir** l'application.
  </Step>
</Steps>

## **3. Fonctionnalités prises en charge & Limitations**

| Fonctionnalité | Description                                                                                            |
| :------------: | :----------------------------------------------------------------------------------------------------- |
|        ✅       | **Mises à jour automatiques du logiciel** – PADS4 Viewer se met à jour automatiquement.                |
|        ✅       | **Mode hors ligne** – L'appareil continue d'exécuter les présentations pendant **7 jours** hors ligne. |
|        ✅       | **Prise en charge HTTP & HTTPS**                                                                       |
|        ✅       | **Maximum 2 vidéos par page**                                                                          |
|        ✅       | **Prise en charge du streaming** – HTTP.                                                               |
|        ✅       | **Mode Cluster Viewer** – Pris en charge.                                                              |
|        ✅       | **Bouton interactif, Texte, Texte défilant, Image, Tableau**                                           |
|        ✅       | **Éléments PowerPoint & PDF** – *(PowerPoint nécessite une installation sur PADS4 Server)*.            |
|        ✅       | **Gestion des appareils à distance** – Contrôlé via **PADS4 Agent**.                                   |
|        ❌       | **Wake-on-LAN & Arrêt non pris en charge**                                                             |

## **4. Dépannage**

| **Problème**                                                                             | **Solution**                                                                     |
| :--------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------- |
| **Bloquer la télécommande, Éteindre l'écran, Les commandes d'arrêt ne fonctionnent pas** | S'assurer que **SICP over Network** est **activé** et hébergé sur **port 5000**. |
| **Le Viewer ne démarre pas en plein écran**                                              | Vérifier la **version du firmware** et **mettre à jour si nécessaire**.          |
| **L'appareil ne parvient pas à se connecter via Ethernet**                               | La mise à jour du **firmware** peut résoudre ce problème.                        |
| **La barre de navigation reste visible**                                                 | **Mise à jour du firmware** requise.                                             |
| **Le Viewer demande répétitivement l'autorisation de stockage de contenu**               | Vérifier si la **version correcte du firmware** est installée.                   |
