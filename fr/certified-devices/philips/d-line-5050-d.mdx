---
- title: 5050D-Series
---

Ce guide fournit des instructions étape par étape pour installer et configurer PADS4 Viewer sur les écrans Philips de la série 5050D.

## **Préparer le 5050D**

Avant l'installation, assurez-vous que le **PADS4 Server est correctement configuré**.

<Note>
  La version minimale du **PADS4 Server** requise est **PADS4 Foundation 2025.1**.
</Note>

Lors de la première mise sous tension de l'appareil, l'**Assistant de Configuration** s'affiche. Si ce n'est pas le cas, **réinitialisez l'appareil** comme décrit dans **Réinitialisation de l'appareil**.

### **Réinitialisation de l'appareil**

Si l'appareil a déjà été utilisé, effectuez une réinitialisation d'usine pour effacer le stockage et éviter les conflits logiciels.

1. Sur la télécommande Philips, appuyez sur **Home**, puis sur **1 8 8 8** pour ouvrir le menu Android.
2. Ouvrez **Settings**, puis sélectionnez **System Tools → Factory Reset**.
3. Confirmez en cliquant sur **Factory Reset** et en appuyant sur **OK**.

Après la réinitialisation, l'**Écran de bienvenue** s'affiche. Continuez avec les étapes décrites dans **Configuration de l'appareil**.

### **Configuration de l'appareil**

1. Sur l'**Écran de bienvenue**, sélectionnez votre **langue** préférée et cliquez sur **Start**.
2. Sur l'écran **Security Settings**, assurez-vous que le **SICP Network Port** est **activé**. *(Les autres paramètres peuvent être activés ou désactivés selon les besoins.)*
3. Sur l'écran **Network Settings**, sélectionnez et configurez le **type de réseau**.
4. Sur l'écran **Signage Display Settings**, appuyez sur **Skip**.
5. Sur l'écran **Clone Media Files**, appuyez sur **Skip**.
6. Appuyez sur **Finish** pour terminer la configuration.

### **Vérification du firmware de l'appareil**

Assurez-vous que la **version correcte du firmware** est installée pour bénéficier de toutes les fonctionnalités de **PADS4 Viewer**.

1. Ouvrez le menu Android en appuyant sur Home 1 8 8 8.
2. Ouvrez Settings → Sélectionnez About.
3. Assurez-vous que le numéro de build est FB01.05 *(requis pour PADS4)*.

## **Installation de PADS4 Viewer**

Suivez ces étapes pour installer PADS4 Viewer sur les écrans Philips D-line (5050D).

### **Installation de PADS4Philips depuis une clé USB**

Le fichier PADS4Philip**s.apk** se trouve sur le **PADS4 Server** in:

```text
https://[PADS4Server]/pads4/apps
```

### **Étapes d'installation**

1. Copiez le fichier  **PADS4Philips.apk** sur une **clé USB.**
2. Insérez la **clé USB** dans le **périphérique Philips D-line**.
3. Ouvrez le **menu Android** en appuyant sur **Home 1 8 8 8**.
4. Allez dans Apps → Settings → Security → More Security Setting
5. Activez Unknown Source
6. Retournez à l'écran des applications
7. Allez dans File Manager et sélectionnez votre clé USB
8. Ouvrez le **PADS4Philips.apk et cliquez sur install**
9. **Cliquez sur DONE**

### Pour configurer PADS4 afin qu'il démarre automatiquement sur l'écran :

1. Accédez à **Settings → Signage Display → Source Settings → Custom Apps**.
2. Sélectionnez **PADS4** dans la liste des applications disponibles.
3. Activez l'option **Force Restart Custom App**.
4. Redémarrez l'écran pour appliquer les modifications.

### Configurer l'écran

Lorsque l'appareil redémarre, vous verrez le PADS4 Viewer.

1. Saisissez l'**URL du serveur** au format [https://PADS4server/domain](https://servername/domain)
2. Renseignez le **ViewerName** selon les besoins.
3. Cliquez sur **Start** pour vous connecter.

Le PADS4 Viewer est maintenant connecté et prêt à l'emploi.

<Note>
  Un message peut s'afficher indiquant que PADS4 est incompatible avec cette version d'Android. Cela est dû au fait que l'écran utilise une version plus récente d'Android. Vous pouvez ignorer ce message en toute sécurité. PADS4 est compatible avec Android 14, et cet avertissement sera résolu dans une prochaine version.
</Note>
