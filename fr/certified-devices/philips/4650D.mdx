Ce guide fournit des instructions étape par étape pour **l'installation et la configuration de PADS4 Viewer** sur **les écrans Philips Q-line (4650Q)**.

***

## **1. Première utilisation**

Avant l'installation, assurez-vous que le **PADS4 Server est configuré et paramétré correctement**.

<Note>
  La version minimale du **PADS4 Server** requise est **PADS4 Foundation 2023.1**.
</Note>

Lors de la première mise en marche de l'appareil, l'**Assistant de configuration** apparaîtra. Si ce n'est pas le cas, **réinitialisez l'appareil** comme décrit dans **1.2 Réinitialisation de l'appareil**.

***

### **1.1 Configuration de l'appareil**

<Steps>
  <Step title="Étape Un">
    Sur l'**Écran d'accueil**, sélectionnez votre **langue** préférée et cliquez sur **Start**.
  </Step>

  <Step title="Étape Deux">
    Sur l'écran **Security Settings**, assurez-vous que le **SICP Network Port** est **activé**. *(Les autres paramètres peuvent être activés ou désactivés selon les besoins.)*
  </Step>

  <Step title="Étape Trois">
    Sur l'écran **Network Settings**, sélectionnez et configurez le **type de réseau**.
  </Step>

  <Step title="Étape Quatre">
    Sur l'écran **Signage Display Settings**, appuyez sur **Skip**.
  </Step>

  <Step title="Étape Cinq">
    Sur l'écran **Clone Media Files**, appuyez sur **Skip**.
  </Step>

  <Step title="Étape Six">
    Appuyez sur **Finish** pour terminer la configuration.
  </Step>
</Steps>

L'appareil affichera **CMND & Deploy**, une application fournie par **Philips**. **Aucune configuration n'est requise** pour cette application.

***

### **1.2 Réinitialisation de l'appareil**

Si l'appareil a déjà été utilisé, effectuez une **réinitialisation d'usine** pour effacer le stockage et éviter les conflits logiciels.

<Steps>
  <Step title="Étape Un">
    Ouvrez le **Menu Android** en appuyant sur **Home**, suivi de **1 8 8 8** sur la **télécommande Philips**.
  </Step>

  <Step title="Étape Deux">
    Ouvrez **Settings**.
  </Step>

  <Step title="Étape Trois">
    Sélectionnez **System Tools → Factory Reset**.
  </Step>

  <Step title="Étape Quatre">
    Cliquez sur **Factory Reset** et confirmez en appuyant sur **OK**.
  </Step>
</Steps>

Après la réinitialisation, l'**Écran d'accueil** apparaîtra. Suivez les étapes de **1.1 Configuration de l'appareil**.

***

## **2. Installation de PADS4 Viewer**

Suivez ces étapes pour installer le **PADS4 Viewer** sur **Philips D-line (4650D)**.

***

### **2.1 Vérification du firmware de l'appareil**

Assurez-vous que la **version correcte du firmware** est installée pour bénéficier de toutes les fonctionnalités du **PADS4 Viewer**.

<Steps>
  <Step title="Étape Un">
    Ouvrez le **Android Menu** en appuyant sur **Home 1 8 8 8**.
  </Step>

  <Step title="Étape Deux">
    Ouvrez **Settings**.
  </Step>

  <Step title="Étape Trois">
    Sélectionnez **About**.
  </Step>

  <Step title="Étape Quatre">
    Assurez-vous que le **Build Number** est **FB01.18** *(requis pour PADS4)*.
  </Step>
</Steps>

***

### **2.2 Mise à jour du firmware de l'appareil**

Pour mettre à jour le **firmware**, suivez les **instructions Philips**:

📌 **Téléchargez le firmware + les instructions depuis :**\
🔗 [PPDS Philips Download Center](https://www.ppds.com/download-center)

***

### **2.3 Installation de PADS4Philips depuis une clé USB**

Le **PADS4Philips.apk** fichier se trouve sur le **PADS4 Server** in:

```
C:\Program Files\PADS4\Foundation\Apps  
```

Vous pouvez également télécharger la dernière version depuis [**ce lien**](http://go.pads4.com/?fwlink=1006) et le placer dans le **répertoire apps sur le PADS4 Server**.

#### **Étapes d'installation**

<Steps>
  <Step title="Étape Un">
    Copiez le **PADS4Philips.apk** fichier sur une **clé USB**.
  </Step>

  <Step title="Étape Deux">
    Insérez la **clé USB** dans le **Philips D-line device**.
  </Step>

  <Step title="Étape Trois">
    Ouvrez le **Android Menu** en appuyant sur **Home 1 8 8 8**.
  </Step>

  <Step title="Étape Quatre">
    Sélectionnez **Apps → File Manager**.
  </Step>

  <Step title="Étape Cinq">
    Ouvrez **mnt → usb\_storage** et sélectionnez **PADS4Philips.apk**.
  </Step>

  <Step title="Étape Six">
    Sélectionnez **Install** puis **Open** l'application.
  </Step>
</Steps>

## **3. Fonctionnalités prises en charge & Limitations**

| Fonctionnalité | Description                                                                                            |
| :------------: | :----------------------------------------------------------------------------------------------------- |
|        ✅       | **Mises à jour automatiques du logiciel** – Le PADS4 Viewer se met à jour automatiquement.             |
|        ✅       | **Mode hors ligne** – L'appareil continue d'exécuter les présentations pendant **7 jours** hors ligne. |
|        ✅       | **Prend en charge HTTP & HTTPS**                                                                       |
|        ✅       | **Maximum 1 vidéo par page**                                                                           |
|        ❌       | **Rotation vidéo, vidéos sans coupure, reflet, inclinaison, retournement** – Non pris en charge.       |
|        ✅       | **Prise en charge du streaming** – HTTP.                                                               |
|        ✅       | **Mode Cluster Viewer** – Pris en charge.                                                              |
|        ✅       | **Bouton interactif, texte, texte défilant, image, tableau**                                           |
|        ✅       | **Éléments PowerPoint & PDF** – *(PowerPoint nécessite une installation sur le PADS4 Server)*.         |
|        ✅       | **Gestion à distance des appareils** – Contrôlé via **PADS4 Agent**.                                   |
|        ✅       | **Wake-on-LAN** – Démarrer l'appareil à distance.                                                      |
|        ❌       | **Éteindre l'écran** – Non pris en charge.                                                             |

***

## **4. Dépannage**

| **Problème**                                                                             | **Solution**                                                                           |
| :--------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------- |
| **Bloquer la télécommande, éteindre l'écran, les commandes d'arrêt ne fonctionnent pas** | Assurez-vous que **SICP over Network** est **activé** et hébergé sur le **port 5000**. |
| **Le Viewer ne démarre pas en plein écran**                                              | Vérifiez la **version du firmware** et **mettez-la à jour si nécessaire**.             |
| **L'appareil ne parvient pas à se connecter via Ethernet**                               | La mise à jour du **firmware** peut résoudre ce problème.                              |
| **La barre de navigation reste visible**                                                 | **Mise à jour du firmware** requise.                                                   |
| **Le Viewer demande répétitivement l'autorisation de stockage de contenu**               | Vérifiez si la **version correcte du firmware** est installée.                         |
