## Prérequis

* Windows 10 build 1909 ou supérieur
* PADS4 Server version 2025.1 ou supérieure

<Warning>
  PADS4 Windows HTML viewer version 1.4.0 + PADS4 version 2025.2 est nécessaire pour avoir l'option "Switch User" dans les paramètres du Launcher. Cela permet de changer facilement de compte utilisateur lors de l'utilisation du mode Kiosk de Windows.
</Warning>

## Aperçu

Le Windows HTML5 Viewer offre plusieurs avantages par rapport au Windows Viewer classique :

* Prend en charge la fonctionnalité de répétition synchronisée
* Permet des changements d'orientation sans modifier les paramètres Windows
* Fournit une expérience cohérente sur tous les HTML viewers
* Peut être utilisé comme économiseur d'écran
* Support plus facile pour les nouvelles fonctionnalités
* Performances améliorées
* Sécurité renforcée via le mode Kiosk
* Mises à jour automatiques depuis le PADS4 Server (pas de mises à jour manuelles requises)

> Note: This viewer shares the same limitations as other HTML viewers. Refer to the [HTML5 Viewer Known Limitations](https://support.pads4.com/hc/en-us/articles/207590696-Known-limitations-for-HTML5-Viewer) pour plus d'informations.

## Instructions d'installation

1. Exécutez `PADS4WindowsHTMLviewer.exe` en tant qu'**Administrator**.
2. Choisissez le répertoire d'installation (ou utilisez celui par défaut).
3. **Optionnel** : Décochez *"Configure Windows to launch the PADS4 HTML5 Viewer in Kiosk Mode"* if:
   * Vous souhaitez l'utiliser comme économiseur d'écran
   * Vous souhaitez l'exécuter sous un utilisateur Windows standard
4. Acceptez le contrat de licence et cliquez sur **Install**.

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/installer.png" />

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/installer-progress.png" />

5. Une fois l'installation terminée, cliquez sur **Reboot** pour redémarrer la machine.

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/installation-succesful.png" />

## Passage à un compte utilisateur standard

Vous pouvez quitter le mode Kiosk et passer à un compte Windows standard de trois façons :

* <u>Depuis la version PADS4 </u>\*\*<u>2025.2</u> \*\*vous pouvez changer d'utilisateur avec l'option "**Switch User**" en accédant aux \*\*Viewer settings \*\*-> **Launcher settings** -> en bas à droite "**Switch User**"
* Utilisez **Remote Desktop** (si configuré) pour vous connecter directement avec un compte utilisateur différent.
* Redémarrez la machine tout en maintenant la touche **Shift** enfoncée. Cela affiche l'écran de connexion Windows.

***

## Utilisation du Viewer comme économiseur d'écran

1. Pendant l'installation, assurez-vous que le mode Kiosk **n'est pas** activé.
2. Ouvrez les **Screensaver Settings** dans Windows.
3. Sélectionnez **Pads4viewer** comme économiseur d'écran.

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/screensaver.png" />

4. Cliquez sur **Apply** et **OK**.

Le viewer démarrera maintenant en fonction du minuteur d'économiseur d'écran configuré.

***

## Référence du script du mode Kiosk

Pour examiner la configuration du mode Kiosk du système, consultez le script PowerShell :

```plaintext
C:\Program Files\PADS4\Launcher Service\configure-kioskmode
```

## Fonctionnalités prises en charge et limitations

| Fonctionnalité                                 | Description                                                     |
| ---------------------------------------------- | --------------------------------------------------------------- |
| ✅ Mises à jour automatiques du logiciel        | Se met à jour automatiquement depuis le PADS4 Server            |
| ✅ Mode hors ligne                              | Exécute des présentations jusqu'à 7 jours sans connexion réseau |
| ✅ Support HTTP & HTTPS                         | Les deux protocoles sont pris en charge                         |
| ✅ Max 4 Vidéos par Page                        | Jusqu'à 4 vidéos peuvent être affichées sur une seule page      |
| ❌ Rotation Vidéo / Transparence / Retournement | Ces améliorations vidéo ne sont pas prises en charge            |
| ✅ Support de Streaming                         | Streaming via HTTP & HTTPS                                      |
| ✅ Mode Cluster Viewer                          | Pris en charge                                                  |
| ✅ Éléments Interactifs                         | Boutons, texte, texte défilant, image, tableaux                 |
| ✅ PowerPoint & PDF                             | PowerPoint nécessite une installation sur le serveur PADS4      |
| ✅ Gestion des Appareils à Distance             | Gérée via PADS4 Agent                                           |
| ✅ Wake-on-LAN                                  | Les appareils peuvent être allumés à distance                   |
| ❌ Éteindre l'Écran                             | Non pris en charge                                              |
