## Configuration requise

* Windows 10 build 1909 ou supérieur
* PADS4 Server version 2025.1 ou supérieur

<Warning>
  PADS4 Windows HTML viewer version 1.4.0 + PADS4 version 2025.2 est requis pour disposer de l'option « Switch User » dans les paramètres du Launcher. Ce qui permet de changer facilement de compte utilisateur lors de l'utilisation du mode Kiosk de Windows.
</Warning>

## Aperçu

Le Windows HTML5 Viewer offre plusieurs avantages par rapport au Windows Viewer classique :

* Prend en charge la fonctionnalité de répétition synchronisée
* Permet les changements d'orientation sans modifier les paramètres Windows
* Offre une expérience cohérente sur tous les HTML viewers
* Peut être utilisé comme économiseur d'écran
* Prise en charge plus facile des nouvelles fonctionnalités
* Performances améliorées
* Sécurité renforcée via le mode Kiosk
* Mises à jour automatiques depuis le PADS4 Server (aucune mise à jour manuelle requise)

> Note: This viewer shares the same limitations as other HTML viewers. Refer to the [Limitations connues du HTML5 Viewer](https://support.pads4.com/en/support/solutions/articles/204000036395-known-limitations-for-html5-viewer) pour plus d'informations.

## Instructions d'installation

1. Exécuter `PADS4WindowsHTMLviewer.exe` en tant que **Administrateur**.
2. Choisissez le répertoire d'installation (ou utilisez celui par défaut).
3. **Optionnel**: Décochez *"Configure Windows to launch the PADS4 HTML5 Viewer in Kiosk Mode"* if:
   * Vous souhaitez l'utiliser comme économiseur d'écran
   * Vous souhaitez l'exécuter sous un utilisateur Windows standard
4. Acceptez le Contrat de licence et cliquez sur **Installer**.

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/installer-progress.png" />

5. Une fois l'installation terminée, cliquez sur **Redémarrer** pour redémarrer la machine.

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/installation-succesful.png" />

## Passer à un compte utilisateur standard

Vous pouvez quitter le mode Kiosk et passer à un compte Windows standard de trois façons :

* <u>Depuis la version PADS4 </u>\*\*<u>2025.2</u> \*\*vous pouvez changer d'utilisateur avec l'option "**Switch User**" en accédant aux \*\*paramètres du Viewer \*\*-> **Launcher settings** -> en bas à droite "**Switch User**"
* Utilisez **Bureau à distance** (si configuré) pour vous connecter directement avec un autre compte utilisateur.
* Redémarrez la machine en maintenant la touche **Shift** enfoncée. Cela affiche l'écran de connexion Windows.

***

## Utiliser le Viewer comme économiseur d'écran

1. Lors de l'installation, assurez-vous que le mode Kiosk n'est **pas** activé.
2. Ouvrez **Paramètres de l'économiseur d'écran** dans Windows.
3. Sélectionnez **Pads4viewer** comme économiseur d'écran.

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/screensaver.png" />

4. Cliquez sur **Appliquer** et **OK**.

Le viewer démarrera désormais en fonction du minuteur de l'économiseur d'écran configuré.

***

## Référence du script du mode Kiosk

Pour examiner la configuration du mode Kiosk du système, référez-vous au script PowerShell :

```plaintext
C:\Program Files\PADS4\Launcher Service\configure-kioskmode
```

## Fonctionnalités prises en charge et limitations

| Fonctionnalité                                      | Description                                                     |
| --------------------------------------------------- | --------------------------------------------------------------- |
| ✅ Mises à jour automatiques du logiciel             | Mises à jour automatiques depuis le PADS4 Server                |
| ✅ Mode hors ligne                                   | Exécute des présentations jusqu'à 7 jours sans connexion réseau |
| ✅ Prise en charge HTTP & HTTPS                      | Les deux protocoles sont pris en charge                         |
| ✅ Maximum 4 vidéos par page                         | Jusqu'à 4 vidéos peuvent être affichées sur une seule page      |
| ❌ Rotation vidéo / Sans interruption / Retournement | Ces améliorations vidéo ne sont pas prises en charge            |
| ✅ Prise en charge du streaming                      | Streaming via HTTP & HTTPS                                      |
| ✅ Mode Cluster Viewer                               | Pris en charge                                                  |
| ✅ Éléments interactifs                              | Boutons, texte, texte défilant, image, tableaux                 |
| ✅ PowerPoint & PDF                                  | PowerPoint nécessite une installation sur le PADS4 Server       |
| ✅ Gestion des appareils à distance                  | Géré via PADS4 Agent                                            |
| ✅ Wake-on-LAN                                       | Les appareils peuvent être mis sous tension à distance          |
| ❌ Éteindre l'écran                                  | Non pris en charge                                              |
