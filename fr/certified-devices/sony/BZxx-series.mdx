Ce guide fournit **des instructions étape par étape** pour installer et configurer **PADS4 Viewer** sur **Sony BZxx Series Displays**.

***

## **Modèles Sony BZxx pris en charge**

Les **Sony BZxx Series** affichages suivants sont pris en charge :

* **BZ53L**
* **BZ50L**
* **BZ40L**
* **BZ35L**
* **BZ30L**
* **BZ30J**

**Assurez-vous que votre PADS4 Server est installé et configuré avant de continuer.**

***

## **Configuration minimale requise**

**Version du PADS4 Server :** **2024.2.6 ou version ultérieure**\
**Version du firmware :** **6.7612 ou version ultérieure**

**Assurez-vous que le firmware de votre appareil répond à la configuration minimale requise avant de procéder à l'installation.**

***

## **Configuration initiale**

**Avant d'installer PADS4, vous devez vérifier la version du firmware.**

**Comment vérifier la version actuelle du firmware :**\
1️. **Aller à** `Settings -> About -> Android TV OS Build`\
2️. **Vérifiez le numéro de version avant "release keys"**\
3️. **Si une connexion Internet est disponible**, vérifiez les mises à jour du firmware sous `About`\
4️. **Si aucune connexion Internet n'est disponible**, téléchargez le dernier firmware depuis **la page firmware de Sony**

***

## **Installation de PADS4Sony depuis une clé USB**

**Vous allez installer PADS4Sony depuis une clé USB.**

**Étapes pour installer PADS4Sony :**

<Steps>
  <Step title="Étape un">
    Téléchargez le **fichier PADS4Sony .pkg** depuis [**ici sur PADS4 Github.**](https://github.com/PADS4/PADS4-Viewer-Sony/releases/latest)
  </Step>

  <Step title="Étape deux">
    Copiez le **fichier .pkg** sur une **clé USB** et insérez la clé dans l'écran Sony.
  </Step>

  <Step title="Étape trois">
    Effectuez une **réinitialisation d'usine** sur l'écran :

    * **Aller à** `Settings -> System -> About -> Reset -> Factory Data Reset`
    * **Choisissez** "Tout effacer"
    * **(Ignorez cette étape si vous configurez un nouvel écran)**
  </Step>

  <Step title="Étape quatre">
    L'écran va **commencer une réinitialisation d'usine dans les 15 secondes** et **redémarrer**.
  </Step>

  <Step title="Étape cinq">
    **Lorsque l'écran de configuration initiale apparaît (sélection de la langue), ne démarrez pas le processus de configuration.**
  </Step>

  <Step title="Étape six">
    L'écran va **détecter** le `bpk` fichier sur la **clé USB** et afficher l'**écran d'installation du package**.
  </Step>

  <Step title="Étape sept">
    **Choisissez "Installer"** et suivez les instructions à l'écran.
  </Step>

  <Step title="Étape huit">
    Après l'installation, le téléviseur ouvrira un **écran d'activation**.
  </Step>

  <Step title="Étape neuf">
    Faites défiler vers le bas et **sélectionnez "Activer cette application d'administration de l'appareil".**
  </Step>

  <Step title="Étape dix">
    Un **message "Installation terminée"** apparaîtra. **Retirez la clé USB** et sélectionnez **OK**.
  </Step>

  <Step title="Étape onze">
    L'écran reviendra à l'**écran de configuration initiale**. **Terminez la configuration** mais **ignorez l'étape de connexion au compte Google**.
  </Step>
</Steps>

**Autorisez toutes les demandes d'autorisation pendant ou avant l'installation.**

***

## **Connexion au PADS4 Server**

**Vous devez configurer l'URL du PADS4 Server pour établir une connexion.**

**Format de l'URL du PADS4 Server :**

```
<protocol>://<host>:<port>/<hostpath>/
```

**Protocol:** `http` ou `https` (selon le certificat SSL)\
**Host:** `DNS name` ou `IP address` du PADS4 Server\
**Port:** **Optionnel** (à utiliser uniquement si le PADS4 Server n'est pas sur les ports HTTP/HTTPS par défaut)\
**Hostpath par défaut :** `PADS4`

**Exemple de configuration :**

```
http://localhost/pads4
```

***

### **Étapes pour se connecter au PADS4 Server**

<Steps>
  <Step title="Étape un">
    Entrez l'**emplacement du PADS4 Server**.
  </Step>

  <Step title="Étape deux">
    (Optionnel) Entrez un **Viewer Name** (ou laissez le PADS4 Server en générer un).
  </Step>

  <Step title="Étape trois">
    Appuyez sur **Start**.
  </Step>

  <Step title="Étape quatre">
    Attendez que le **téléchargement du Viewer soit terminé**, puis le Viewer démarrera.
  </Step>

  <Step title="Étape cinq">
    Lorsque vous y êtes invité, **accordez l'autorisation** de stocker du contenu de façon permanente sur l'appareil.
  </Step>
</Steps>

**Le PADS4 Viewer va maintenant démarrer et se connecter au PADS4 Server.**

***

## **Ajustement des paramètres du PADS4 Viewer**

**Vous pouvez ajuster les paramètres du PADS4 Viewer directement sur l'écran.**

**Étapes pour accéder aux paramètres :**\
1️. **Connectez une souris à l'appareil.**\
2️. **Faites un clic droit avec la souris** pour ouvrir le **menu Viewer Settings**.\
3️. **Effectuez des ajustements** sur l'orientation de l'écran, les paramètres d'affichage, etc.

**Vous pouvez également revenir à la configuration PADS4Sony en cliquant sur le bouton "Launcher Settings".**

***

## **Mise à jour de l'application PADS4Sony**

**L'application PADS4Sony se met à jour automatiquement lorsque le PADS4 Server est mis à jour.**

**Comment mettre à jour manuellement l'application :**\
1️. **Vérifiez la version du PADS4 Server.**\
2️. **Si une nouvelle version est disponible, redémarrez l'écran Sony.**\
3️. **L'application PADS4Sony se mettra à jour automatiquement.**

***

## **Accès au menu Android pendant que PADS4 est en cours d'exécution**

**Pour accéder aux paramètres rapides :**

1. **Appuyez sur le bouton "Options"** sur la télécommande.\
   2️. Ajustez divers **paramètres rapides** depuis ce menu.

**Pour accéder au menu des paramètres complets :**\
1️. **Appuyez sur le bouton "Home"** sur la télécommande.\
2️. L'**application PADS4 se fermera**.\
3️. **Pour redémarrer PADS4, accédez à**: `Settings -> System -> Restart`.

**L'application PADS4 démarrera maintenant automatiquement.**

|        ✅       | **Mode Portrait** – Pris en charge, mais **le menu de l'appareil n'est pas pris en charge**.          |

| Fonctionnalité | Description                                                                           |
| :------------: | ------------------------------------------------------------------------------------- |
|        ✅       | **2 vidéos par page** – Prend en charge la lecture fluide (**1920x1080**).            |

|        ✅       | **Streaming** – Prend en charge le streaming HTTP.                                    |
|        ✅       | **Mode 4K (3840x2160)**                                                               |

| Fonctionnalité | Description                                                       |
| :------------: | ----------------------------------------------------------------- |
