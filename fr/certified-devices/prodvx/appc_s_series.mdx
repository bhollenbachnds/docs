- title: APPC S-series
---

Ce manuel fournit des instructions étape par étape pour configurer l'appareil ProDVX APPC S-series et installer le PADS4 Viewer. Il couvre la configuration initiale, l'installation de l'APK, la configuration du démarrage automatique et la configuration du viewer.

**Appareils compatibles :**

* **APPC-10SFA**
* **APPC-10SFN**
* **APPC-10SF**
* **APPC-10S**

**Exigences minimales**

Version PADS4 requise : **2024.1**

Firmware ProDVX : **PPC-10N-S-series.R25.ProDVX\_013.A25\_02**

## Configuration initiale de l'appareil

Allumez l'appareil ProDVX APPC 10SFA pour commencer la configuration.

* Sur l'écran de bienvenue, **acceptez** les Conditions générales, puis cliquez sur Start​
* (Optionnel) Activez ou désactivez ProMGR et cliquez sur Continue.
* **Activez** l'API PRODVX, car PADS4 en dépend pour un bon fonctionnement, puis cliquez sur Continue.​
* **Désactivez** AutoStart et cliquez sur Continue.
* Réglez le **délai d'expiration** de l'écran sur Jamais et cliquez sur Continue.
* Cliquez sur **Reboot** Device pour appliquer les modifications.

## Installation du PADS4 Viewer APK

Téléchargez le dernier PADS4ProDVXv2.apk depuis votre serveur PADS4 (par ex., https\://\[your-server]/\[domain]/apps) ou des sources officielles, et copiez-le sur une clé USB.

1. Insérez la clé USB dans l'appareil ProDVX.
2. Ouvrez l'application Files et sélectionnez la clé USB.
3. Appuyez sur **PADS4ProDVXv2.apk**.
4. Cliquez sur **Continue** sur l'invite d'avertissement.
5. Cliquez sur **Install** pour continuer.
6. Une fois terminé, cliquez sur **Done**.

## Configuration du démarrage automatique

Lancez l'**application AutoStart** depuis le tiroir d'applications après l'installation. Cliquez sur Start App, sélectionnez PADS4ProDVXv2, puis fermez AutoStart et redémarrez l'appareil.

Cela garantit que le PADS4 Viewer se lance automatiquement au démarrage.

## Configuration du PADS4 Viewer

Après le redémarrage, le PADS4 Viewer s'ouvre. Entrez l'adresse de votre serveur PADS4 dans le format`https://<host>:<port>/<domain>/`

* Optionnellement, entrez un nom de viewer personnalisé.
* Cliquez sur Start pour vous connecter

Attendez la connexion ; une LED verte et le nom du viewer indiquent le succès.
