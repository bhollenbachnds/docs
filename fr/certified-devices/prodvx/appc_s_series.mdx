- title: APPC S-series
---

Ce manuel fournit des instructions étape par étape pour configurer l'appareil ProDVX APPC S-series et installer le PADS4 Viewer. Il couvre la configuration initiale, l'installation de l'APK, la configuration du démarrage automatique et la configuration du viewer.

**Appareils compatibles :**

* **APPC-10SFA**
* **APPC-10SFN**
* **APPC-10SF**
* **APPC-10S**

**Configuration minimale requise**

Version PADS4 requise : **2024.1**

Firmware ProDVX : **PPC-10N-S-series.R25.ProDVX\_013.A25\_02**

## Configuration initiale de l'appareil

Allumer l'appareil ProDVX APPC 10SFA pour commencer la configuration.

* Sur l'écran de bienvenue, **accepter** les Conditions générales, puis cliquer sur Démarrer​
* (Facultatif) Activer ou désactiver ProMGR et cliquer sur Continuer.
* **Activer** l'API PRODVX, car PADS4 en dépend pour fonctionner correctement, puis cliquer sur Continuer.​
* **Désactiver** AutoStart et cliquer sur Continuer.
* Définir le **délai d'expiration** de l'écran sur Jamais et cliquer sur Continuer.
* Cliquer sur **Redémarrer** l'appareil pour appliquer les modifications.

## Installation de l'APK PADS4 Viewer

Télécharger le dernier PADS4ProDVXv2.apk depuis votre serveur PADS4 (par ex., https\://\[your-server]/\[domain]/apps) ou depuis des sources officielles, et le copier sur une clé USB.

1. Insérer la clé USB dans l'appareil ProDVX.
2. Ouvrir l'application Fichiers et sélectionner la clé USB.
3. Appuyer sur **PADS4ProDVXv2.apk**.
4. Cliquer sur **Continuer** sur l'invite d'avertissement.
5. Cliquer sur **Installer** pour continuer.
6. Une fois terminé, cliquer sur **Terminé**.

## Configuration du démarrage automatique

Lancer l'**application AutoStart** depuis le tiroir d'applications après l'installation. Cliquer sur Démarrer l'application, sélectionner PADS4ProDVXv2, puis fermer AutoStart et redémarrer l'appareil.

Cela garantit que le PADS4 Viewer se lance automatiquement au démarrage.

## Configuration de PADS4 Viewer

Après le redémarrage, le PADS4 Viewer s'ouvre. Entrer l'adresse de votre serveur PADS4 dans le format`https://<host>:<port>/<domain>/`

* Facultativement, entrer un nom de viewer personnalisé.
* Cliquer sur Démarrer pour se connecter

Attendre la connexion ; une LED verte et le nom du viewer indiquent le succès.
