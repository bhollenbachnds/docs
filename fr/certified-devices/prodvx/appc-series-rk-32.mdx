Ce guide fournit des instructions étape par étape pour **installer et configurer PADS4 Viewer** sur **ProDVX APPC-Series (RK3288) devices** fonctionnant sous **Android 9**.

## **Avant de commencer**

### **Appareils ProDVX APPC-Series pris en charge**

Les **ProDVX APPC-Series (RK3288)** appareils suivants sont pris en charge :

* **APPC-7XPL**
* **APPC-7XPLN**
* **APPC-10X**
* **APPC-10XP**
* **APPC-10XPL**
* **APPC-10XPLN**
* **APPC-10SLB** (Nécessite **10X Firmware**)
* **APPC-10SLBN**
* **APPC-15XP**
* **APPC-22X**
* **APPC-22XP**
* **APPC-24**
* **APPC-27X**
* **APPC-32**

### Configuration minimale requise pour PADS4

* **PADS4 Server :** 2024.2 ou version ultérieure
* **Android:** 9

<Note>
  Les mises à jour du firmware peuvent être demandées auprès de votre revendeur ProDVX.
</Note>

## Réinitialiser l'appareil (recommandé)

Si l'appareil a déjà été utilisé, effectuez une **réinitialisation d'usine** pour effacer le stockage et éviter les conflits logiciels.

1. Ouvrez **Android Settings**.
2. Accédez à **System > Reset options**.
3. Sélectionnez **Erase all data (factory reset)**.
4. Sélectionnez **Reset tablet**, puis confirmez avec **Erase everything**.
5. Attendez que l'appareil redémarre automatiquement.

<Warning>
  Les paramètres réseau sont désactivés par défaut sur les nouveaux appareils. Avant l'installation, ouvrez **Android Settings** et activez **Wi-Fi** ou **Ethernet**.
</Warning>

## Installer le PADS4 Viewer

### Étape 1 : Télécharger et transférer l'APK

Téléchargez l'APK depuis l'un des emplacements suivants :

* **Répertoire PADS4 Server :** `C:\Program Files\PADS4\Clients\Apps`
* **Téléchargement officiel :** [PADS4ProDVX APK on GitHub](https://github.com/PADS4/PADS4-Viewer-ProDVX-x-series/releases/latest)

Puis transférez-le sur l'appareil :

1. Copiez **PADS4ProDVXv2.apk** sur une clé USB.
2. Insérez la clé USB dans l'appareil ProDVX APPC-Series.

### Étape 2 : Installer l'APK

1. Ouvrez l'application **APK Installer** sur l'appareil. Autorisez l'accès au stockage externe si vous y êtes invité.
2. Sélectionnez **Install** et parcourez la clé USB pour trouver **PADS4ProDVX.apk**.
3. Sélectionnez le fichier APK pour démarrer l'installation.
4. Si vous y êtes invité, activez **Allow install from unknown sources**, puis relancez l'installation.
5. Sélectionnez **Install**, et choisissez **Open** lorsque l'installation est terminée.
