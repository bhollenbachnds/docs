- title: Série APPC RK3288 (Android 9)

Ce guide fournit des instructions étape par étape pour **installer et configurer PADS4 Viewer** sur **les appareils ProDVX APPC-Series (RK3288)** fonctionnant sous **Android 9**.

## **Avant de commencer**

### **Appareils ProDVX APPC-Series pris en charge**

Les **ProDVX APPC-Series (RK3288)** appareils suivants sont pris en charge :

* **APPC-7XPL**
* **APPC-7XPLN**
* **APPC-10X**
* **APPC-10XP**
* **APPC-10XPL**
* **APPC-10XPLN**
* **APPC-10SLB** (Nécessite le **Firmware 10X**)
* **APPC-10SLBN**
* **APPC-15XP**
* **APPC-22X**
* **APPC-22XP**
* **APPC-24**
* **APPC-27X**
* **APPC-32**

### Configuration minimale requise pour PADS4

* **PADS4 Server :** 2024.2 ou version ultérieure
* **Android:** 9

<Note>
  Les mises à jour du firmware peuvent être demandées auprès de votre revendeur ProDVX.
</Note>

## Réinitialiser l'appareil (recommandé)

Si l'appareil a déjà été utilisé, effectuez une **réinitialisation d'usine** pour effacer le stockage et éviter les conflits logiciels.

1. Ouvrez **Paramètres Android**.
2. Accédez à **Système > Options de réinitialisation**.
3. Sélectionnez **Effacer toutes les données (réinitialisation d'usine)**.
4. Sélectionnez **Réinitialiser la tablette**, puis confirmez avec **Tout effacer**.
5. Attendez que l'appareil redémarre automatiquement.

<Warning>
  Les paramètres réseau sont désactivés par défaut sur les nouveaux appareils. Avant l'installation, ouvrez **Paramètres Android** et activez **Wi-Fi** ou **Ethernet**.
</Warning>

## Installer PADS4 Viewer

### Étape 1 : Télécharger et transférer l'APK

Téléchargez l'APK depuis l'un des emplacements suivants :

* **Répertoire PADS4 Server :** `C:\Program Files\PADS4\Clients\Apps`
* **Téléchargement officiel :** [PADS4ProDVX APK sur GitHub](https://github.com/PADS4/PADS4-Viewer-ProDVX-x-series/releases/latest)

Ensuite, transférez-le sur l'appareil :

1. Copiez **PADS4ProDVXv2.apk** sur une clé USB.
2. Insérez la clé USB dans l'appareil ProDVX APPC-Series.

### Étape 2 : Installer l'APK

1. Ouvrez l'application **APK Installer** sur l'appareil. Autorisez l'accès au stockage externe si vous y êtes invité.
2. Sélectionnez **Installer** et parcourez la clé USB pour trouver **PADS4ProDVX.apk**.
3. Sélectionnez le fichier APK pour démarrer l'installation.
4. Si vous y êtes invité, activez **Autoriser l'installation depuis des sources inconnues**, puis relancez l'installation.
5. Sélectionnez **Installer**, et choisissez **Ouvrir** une fois l'installation terminée.
