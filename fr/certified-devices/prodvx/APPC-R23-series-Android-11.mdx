Ce guide fournit des instructions étape par étape pour **installer et configurer PADS4 Viewer** sur **les appareils ProDVX APPC-R23 fonctionnant sous Android 11**.

**Appareils compatibles :**

* **APPC-15XP (R23-7222)**
* **APPC-22XP (R23-7222)**
* **APPC-24X (R23-7001)**
* **APPC-32X (R23-7001)**

**Version minimale requise de PADS4 :** **2024.1**

***

## **1. Avant de commencer**

### **1.1 Réinitialisation de l'appareil**

Si l'appareil a déjà été utilisé, effectuez une **réinitialisation d'usine** pour libérer de l'espace de stockage et éviter les conflits logiciels.

#### **Étapes de réinitialisation d'usine**

<Steps>
  <Step title="Étape Une">
    Ouvrez **Android Settings**.
  </Step>

  <Step title="Étape Deux">
    Cliquez sur **System**.
  </Step>

  <Step title="Étape Trois">
    Cliquez sur **Reset options**.
  </Step>

  <Step title="Étape Quatre">
    Sélectionnez **Erase all data (factory reset)**.
  </Step>

  <Step title="Étape Cinq">
    Cliquez sur **ERASE ALL DATA**.
  </Step>

  <Step title="Étape Six">
    Confirmez en cliquant sur **ERASE ALL DATA** à nouveau.
  </Step>
</Steps>

**Note:** La réinitialisation d'usine **supprime toutes les données**. Assurez-vous de **sauvegarder** les fichiers importants avant de continuer.

***

### **1.2 S'assurer que PADS4 Server est installé**

Pendant l'installation, vous devez **spécifier l'emplacement du PADS4 Server**. Il est **recommandé** d'**installer et configurer le PADS4 Server** avant de configurer l'appareil.

***

### **1.3 S'assurer que l'API ProDVX est installée**

**L'API ProDVX est requise** pour que PADS4 Viewer fonctionne sur les appareils ProDVX APPC-R23.

#### **Vérifier si l'API ProDVX est en cours d'exécution**

<Steps>
  <Step title="Étape Une">
    Recherchez l'**icône PA** dans la **barre d'état Android**.
  </Step>

  <Step title="Étape Deux">
    Si l'icône est absente, **faites glisser la barre d'état vers le bas** et vérifiez les notifications.
  </Step>

  <Step title="Étape Trois">
    Recherchez une notification indiquant **"ProDVX API (AOSP) is running."**
  </Step>

  <Step title="Étape Quatre">
    Si l'API n'est **pas installée**, téléchargez-la depuis [**here**](#) et installez-la avant de continuer.
  </Step>
</Steps>

***

### **1.4 Token API ProDVX**

PADS4 **utilise un token API par défaut** pour l'authentification de l'API ProDVX.

#### **Remplacer le token par défaut (si nécessaire)**

Si vous avez **un token API personnalisé**, suivez ces étapes :

<Steps>
  <Step title="Étape Une">
    Naviguez vers **C:\Program Files\PADS4\Foundation\Apps** sur le **PADS4 Server**.
  </Step>

  <Step title="Étape Deux">
    Localisez le fichier **prodvx-api-token.txt**.
  </Step>

  <Step title="Étape Trois">
    Ouvrez le fichier et **remplacez le token existant** par votre **token API personnalisé**.
  </Step>

  <Step title="Étape Quatre">
    Enregistrez le fichier.
  </Step>
</Steps>

***

## **2. Installation de PADS4 Viewer**

### **2.1 Télécharger le PADS4 Viewer**

**Vous pouvez télécharger le PADS4 Viewer depuis :**

**PADS4 Server :**

```
https://[yourPADS4Server]/[domain]/apps
```

*******`(Replace [yourPADS4Server] with your server's IP or name, and [domain] with your PADS4 domain – default is PADS4.)`*******

**Github:** [Télécharger la dernière version de PADS4ProDVXv2 APK](https://github.com/PADS4/PADS4-Viewer-ProDVX-r23/releases/latest)

**Important:** Si vous téléchargez une version plus récente, **placez-la également dans le répertoire des applications** sur le **PADS4 Server**. Sinon, le serveur installera l'ancienne version lorsque l'appareil se connectera.

***

### **2.2 Installer le PADS4 Viewer**

<Steps>
  <Step title="Étape Une">
    Copiez le **PADS4ProDVXv2.apk** sur une **clé USB**.
  </Step>

  <Step title="Étape Deux">
    Insérez la **clé USB** dans l'**appareil ProDVX**.
  </Step>

  <Step title="Étape Trois">
    Ouvrez l'**application Explorer** sur l'appareil.
  </Step>

  <Step title="Étape Quatre">
    Cliquez sur **USB** et localisez **PADS4ProDVXv2.apk**.
  </Step>

  <Step title="Étape Cinq">
    Cliquez sur **CONTINUE** lorsque vous y êtes invité.
  </Step>

  <Step title="Étape Six">
    Cliquez sur **INSTALL** lorsque vous y êtes invité.
  </Step>

  <Step title="Étape Sept">
    Attendez que l'installation soit terminée, puis cliquez sur **DONE**.
  </Step>
</Steps>

***

### **2.3 Configurer PADS4 Viewer pour démarrer automatiquement**

**Utilisez soit :**

**Logiciel ProDVX ProMGR** *(Contactez ProDVX pour plus de détails)*\
**Paramètres de l'appareil Android** *(Suivez les étapes ci-dessous)*

#### **Activer le démarrage automatique dans les paramètres Android**

<Steps>
  <Step title="Première étape">
    Ouvrir **Paramètres Android**.
  </Step>

  <Step title="Deuxième étape">
    Naviguez vers **Paramètres système → Options d'affichage dynamique**.
  </Step>

  <Step title="Troisième étape">
    Sélectionnez **Auto Start Application**.
  </Step>

  <Step title="Quatrième étape">
    Cliquez sur **Start App**, puis sélectionnez **PADS4ProDVXv2**.
  </Step>
</Steps>

***

## **3. Configurer PADS4 Viewer**

### **3.1 Configurer la connexion au serveur PADS4**

**Format:**

```
<protocol>://<host>:<port>/<hostpath>/
```

**Example:**

```
https://mydomain.pads365.com/pads4/
```

#### **Étapes pour se connecter au serveur PADS4**

<Steps>
  <Step title="Première étape">
    Saisissez l'**adresse du serveur PADS4**.
  </Step>

  <Step title="Deuxième étape">
    (Facultatif) Saisissez un **nom de Viewer personnalisé**.
  </Step>

  <Step title="Troisième étape">
    Cliquez sur **Start**.
  </Step>

  <Step title="Quatrième étape">
    Accordez les **autorisations d'accès** aux **photos, médias et fichiers**.
  </Step>

  <Step title="Cinquième étape">
    Le **PADS4 Viewer va télécharger** et démarrer automatiquement.
  </Step>
</Steps>

***

## **4. Ajustement et mise à jour de PADS4 Viewer**

### **4.1 Ajuster les paramètres du Viewer**

**Pour ouvrir l'écran des options :**

**Maintenez le bouton gauche de la souris enfoncé** ou **touchez l'écran pendant 3 secondes**.\
Cliquez sur **Launcher Settings** pour revenir à l'application **PADS4ProDVXv2**.

***

### **4.2 Mise à jour de PADS4 Viewer et PADS4ProDVXv2**

**Si vous mettez à jour le serveur PADS4**, à la fois **PADS4 Viewer** et **PADS4ProDVXv2** se **mettront à jour automatiquement** vers la dernière version.
