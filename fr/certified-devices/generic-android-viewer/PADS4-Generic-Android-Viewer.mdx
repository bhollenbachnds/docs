Ce guide explique comment installer le **PADS4 Launcher APK** pour démarrer le **PADS4 Viewer** sur **du matériel Android non certifié.**

<Note>
  Certains appareils sont livrés avec un **dedicated PADS4 Launcher**. Pour plus d'informations, consultez la page [**PADS4 Certified Devices**](#).
</Note>

***

</Warning>

## **Installation de PADS4 Launcher à partir d'une clé USB**

<Warning>
  Le PADS4 Generic Android Launcher est uniquement pris en charge sur la version du PADS4 Server **2025.1** et supérieure. Veuillez vous assurer que votre serveur est mis à jour en conséquence.
</Warning>

<Steps>
  <Step title="Première étape">
    Copiez le **PADS4 Launcher APK** sur une **clé USB**.
  </Step>

  <Step title="Deuxième étape">
    Insérez la **clé USB** dans l'appareil Android.
  </Step>

  <Step title="Troisième étape">
    Ouvrez **APK Installer** sur l'appareil.
  </Step>

  <Step title="Quatrième étape">
    Sélectionnez **Install** et naviguez jusqu'au fichier APK sur la **clé USB**.
  </Step>

  <Step title="Cinquième étape">
    Lorsque vous êtes invité avec :\
    *"Voulez-vous installer cette application ?"*

    * Appuyez sur **Next** jusqu'à ce que le bouton **Install** apparaisse.
    * Appuyez sur **Install** pour continuer.
  </Step>

  <Step title="Sixième étape">
    Après l'installation, appuyez sur **Open** pour lancer le **PADS4 Launcher**.
  </Step>

  <Step title="Septième étape">
    Entrez l'emplacement du **PADS4 Server location**, en précisant :

    * **Domain name**
    * **IP address**
    * **PC name** du serveur exécutant **PADS4 Server**
  </Step>

  <Step title="Huitième étape - *Optionnel*">
    Si les **server settings** par défaut ont été modifiés, allez dans **Advanced Settings** et entrez :

    * Le **port number**
    * Le **host/root path**
  </Step>

  <Step title="Neuvième étape">
    Appuyez sur **Start**.

    * La première fois que l'application s'exécute, elle vous demandera de sélectionner votre **home app**.
    * Sélectionnez **PADS4 Launcher** et choisissez **Always** pour éviter les futures invites.
  </Step>

  <Step title="Dixième étape">
    Assurez-vous d'avoir une **valid connection** au **PADS4 Server** lors du lancement du **PADS4 Launcher** pour la première fois.

    * Cela permet à l'appareil de **download and start PADS4 Viewer**.
    * Une fois téléchargé, **PADS4 Viewer** peut fonctionner **offline**.
  </Step>
</Steps>

## **Mise à jour de PADS4 Launcher à partir d'une clé USB**

<Steps>
  <Step title="Première étape">
    Copiez le dernier **PADS4 Launcher APK** sur une **clé USB**.
  </Step>

  <Step title="Deuxième étape">
    Insérez la **clé USB** dans l'appareil Android.
  </Step>

  <Step title="Troisième étape">
    Ouvrez **Viewer Settings**.
  </Step>

  <Step title="Quatrième étape">
    Naviguez vers **Launcher Settings**.
  </Step>

  <Step title="Cinquième étape">
    Appuyez sur l'**folder icon** dans le **bottom-right corner**.
  </Step>

  <Step title="Sixième étape">
    Naviguez vers :

    ```
    mnt -> usb_storage  
    ```

    Sélectionnez le **APK file**.
  </Step>

  <Step title="Septième étape">
    Lorsque vous êtes invité avec :\
    *"Voulez-vous installer une mise à jour de cette application existante ?"*

    * Appuyez sur **Install**.
  </Step>

  <Step title="Huitième étape">
    Une fois l'installation terminée, le **PADS4 Launcher** redémarrera automatiquement.
  </Step>
</Steps>
