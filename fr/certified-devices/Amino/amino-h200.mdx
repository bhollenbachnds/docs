Ce manuel fournit des instructions étape par étape pour configurer l'appareil Amino H200 et installer le PADS4 Viewer. Il couvre la configuration initiale, l'installation de l'APK, la configuration du démarrage automatique et la configuration du viewer.

**Appareils compatibles :**

* **Amino H200**

**Exigences minimales**

Version PADS4 requise : **2025.1** \
Firmware: **25.6.2540.10R**

## Installation du PADS4 Viewer dans Amino Orchestrate

Amino utilise une console centrale pour gérer les appareils. Assurez-vous d'avoir un compte et téléchargez le Generic Android Viewer

1. Dans votre navigateur, ouvrez l'URL : [https://orchestrate.amino.tv/](https://orchestrate.amino.tv/)
2. Sur la page principale, accédez à la section Application **sections**.
3. Cliquez sur **Add Application**.
4. Entrez PADS4 Viewer comme **Application Name**.
5. Téléchargez le **Generic Android Viewer APK**.

Le viewer est prêt à être déployé sur votre appareil.

## Configurer la configuration dans Orchestra

1. Accédez à **Configurations**
2. Sélectionnez **Add Configuration** et fournissez un nom pour vos Configurations
3. Cliquez sur **Add** pour enregistrer les Configurations
4. Ouvrez l'onglet **Parameters** .
5. Sélectionnez le **Target Version** du Firmware
6. Cliquez sur **Save Changes** pour appliquer.
7. Ouvrez l'**Applications** onglet dans la vue de configuration.
8. Cliquez sur **Add Action** et sélectionnez votre application PADS4.
9. Vérifiez que l'application est répertoriée, définissez **Action** sur **Install**, et sélectionnez la **Version** correcte si applicable.
10. Cliquez sur **Save Changes** pour appliquer.

## Définir le démarrage automatique

1. Ouvrez l'**Parameters** onglet.
2. Sous **Application Settings**, sélectionnez l'application PADS4 Viewer dans le menu déroulant **Home App**.
3. Cliquez sur **Save Changes**.

## Ajouter des PADS4 Viewers à Orchestra

1. Accédez à **Devices** et sélectionnez **Device list.**
2. Cliquez sur **Add Device**.
3. Entrez le **Serial Number** de votre appareil.
4. Sous **Configuration**, sélectionnez la configuration que vous souhaitez utiliser.
5. Une fois que le Viewer est en ligne, le lecteur s'installera.

## Configuration du PADS4 Viewer

Après l'installation, le PADS4 Viewer s'ouvre. Entrez l'adresse de votre serveur PADS4 au format`https://<host>:<port>/<domain>/`

* Optionnellement, entrez un nom de viewer personnalisé.
* Cliquez sur Start pour vous connecter

Attendez la connexion ; une LED verte et le nom du viewer indiquent le succès.
