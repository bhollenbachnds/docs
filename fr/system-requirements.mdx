- description: Configuration système requise pour PADS4 à partir de la version

## Configuration minimale requise pour PADS4 Server

* CPU: 4 cores
* RAM: 16 GB
* Taille d'installation complète 1,32 Go
* Système d'exploitation 64 bits : Windows 10, Windows 11, Windows Server 2016, Windows Server + édition Datacenter 2019, Windows Server + édition Datacenter 2022, Windows Server + édition Datacenter 2025

## Configuration recommandée pour PADS4 Server

* CPU: 8 cores
* RAM: 24 GB
* Taille d'installation complète 1,32 Go
* 64 bit OS: Windows 10, Windows 11, Windows Server 2016, Windows Server + Datacenter edition 2019, Windows Server + Datacenter edition 2022, Windows Server + Datacenter edition 2025
