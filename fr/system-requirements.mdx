---
- title: Configuration système requise pour PADS4 Server
- description: Configuration minimale et recommandée en termes de CPU, RAM, disque
    et systèmes d'exploitation Windows pris en charge pour PADS4 Server à partir
    de la version 2021.1.
---

## Configuration minimale requise PADS4 Server

* CPU: 4 cores
* RAM: 16 GB
* Taille d'installation complète 1,32 Go
* OS 64 bits : Windows 10, Windows 11, Windows Server 2016, Windows Server + Datacenter edition 2019, Windows Server + Datacenter edition 2022, Windows Server + Datacenter edition 2025

## Configuration recommandée PADS4 Server

* CPU: 8 cores
* RAM: 24 GB
* Taille d'installation complète 1,32 Go
* OS 64 bits : Windows 10, Windows 11, Windows Server 2016, Windows Server + Datacenter edition 2019, Windows Server + Datacenter edition 2022, Windows Server + Datacenter edition 2025
