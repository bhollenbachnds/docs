PADS4 prend en charge **les connexions de capteurs directes et indirectes**, permettant l'intégration avec divers **caméras et logiciels de mesure d'audience**. Les capteurs peuvent se connecter **directement** via des appareils IP ou **indirectement** via **des logiciels tiers** qui s'interfacent avec plusieurs capteurs.

***

## **Connexions de capteurs directes**

Les **appareils suivants ont été testés** et vérifiés par NDS pour **intégration directe** avec PADS4.

### **Caméras Hikvision**

| Fabricant     | Modèle                 | Fonctionnalité                                                | Description                                              |
| ------------- | ---------------------- | ------------------------------------------------------------- | -------------------------------------------------------- |
| **Hikvision** | DS-2TD1217B-6/PA       | Reconnaissance faciale, Détection de masque (firmware requis) | Caméra tourelle thermographique                          |
|               | DS-2TD2617B-3/PA       | Reconnaissance faciale, Détection de masque (firmware requis) | Caméra bullet thermographique                            |
|               | DS-2CD6365G0-IS        | Détection de ligne de franchissement                          | Caméra réseau fisheye                                    |
|               | DS-2CD6825G0/C-I(V)(S) | Comptage de personnes                                         | Caméra réseau de comptage de personnes à double objectif |

### **Configuration de l'API Hikvision**

Pour activer l'accès API pour PADS4 :

1. Ouvrez le **portail d'administration** de la caméra.
2. Accédez à **Configuration > Security > Verification**.
3. Définissez **l'authentification RTSP** sur **Digest/Basic**.
4. Définissez **la vérification Web** sur **Digest/Basic**.

## **Connexions de capteurs indirectes (logiciels tiers)**

PADS4 peut récupérer **des données et des événements** depuis divers **logiciels de reconnaissance faciale/d'audience**.

### **Logiciels tiers pris en charge**

| Fabricant     | Logiciel        | Fonctionnalité                                                              | Description                                                                                                                                                                                                |
| ------------- | --------------- | --------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Milestone** | XProtect        | Récupération d'alarmes                                                      | Se connecte avec **plus de 9 000 caméras**, permettant des alarmes que PADS4 peut utiliser pour informer ou guider le public. Voir **Configuration de Milestone XProtect** pour les détails d'intégration. |
| **Quividi**   | Vidireports 6/7 | Données de mesure d'audience                                                | Calcule **des analyses d'audience** à partir de flux vidéo de caméras. Fournit **le genre, l'âge, l'humeur et plus encore** en temps réel. Voir **le site web de Quividi** pour plus de détails.           |
| **SightCorp** | DeepSight       | Données de mesure d'audience, Reconnaissance faciale                        | Récupère en temps réel **le masque facial, le genre, l'âge, l'humeur et d'autres analyses**. Voir **le site web de SightCorp** pour plus de détails.                                                       |
| **Saimos**    | Saimos          | Données de mesure d'audience, Reconnaissance faciale, Comptage de personnes | Envoie en temps réel **des événements de détection de masque** lorsqu'une personne quitte le champ de la caméra. Voir **le site web de Saimos** pour plus de détails.                                      |

***

### **Points clés**

* PADS4 prend en charge les connexions de capteurs **directes** et **tierces**.
* Les intégrations directes incluent **les caméras Hikvision et Mobotix**.
* Les intégrations indirectes fonctionnent avec **Milestone, Quividi, SightCorp et Saimos**.
* Une configuration API peut être requise pour la connectivité directe des capteurs.

Pour plus de détails, consultez le **site web du fabricant** ou la documentation PADS4.
