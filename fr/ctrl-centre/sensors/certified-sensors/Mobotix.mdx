Ce guide explique comment connecter les **caméras Mobotix** avec **PADS4** pour récupérer des données de mesure d'audience en temps réel. Pour plus d'informations sur les caméras Mobotix, visitez : [www.Mobotix.com](https://www.mobotix.com).

***

## **Étape 1 : Configuration de l'appareil Mobotix M7**

Pour assurer une intégration correcte, les profils d'événements dans la caméra Mobotix doivent être configurés pour correspondre à l'application **AIFacedetect App**.

### **Accès aux paramètres de la caméra**

1. **Connectez-vous** à la caméra Mobotix.
2. Naviguez vers **Setup Menu > Event Control > Event Overview > Message Events**.

### **Vérifiez et appliquez les propriétés d'événement suivantes**

| **Propriété**          | **Valeur**          |
| ---------------------- | ------------------- |
| Event sensor type      | **MxMessageSystem** |
| Message name           | **AIFacedetect**    |
| Message range          | **Local**           |
| Filter message content | **No Filter**       |

***

## **Étape 2 : Configuration de l'application AIFacedetect**

Pour activer la **détection faciale** pour les visages masqués et non masqués :

1. **Naviguez vers :**\
   **Setup Menu > Certified App Settings > AIFacedetect Settings > Configurator**.
2. Appliquez les paramètres suivants :

| **Propriété**         | **Valeur**        |
| --------------------- | ----------------- |
| Detection algorithm   | **Deep**          |
| Detect unmasked faces | **Enable**        |
| Detect masked faces   | **Enable**        |
| Event management      | **Event on exit** |

3. **Cliquez sur "Set"** pour enregistrer les paramètres.

***

## **Étape 3 : Configuration de la connexion Mobotix dans PADS4**

1. **Ouvrez PADS4 CTRL Center**.
2. **Naviguez vers Sensors** (Si la catégorie **Sensors** n'est pas visible, vous pourriez avoir besoin d'une **licence de module IOT**. Contactez votre partenaire PADS4 pour obtenir de l'aide).
3. Cliquez sur **New Camera Software** et sélectionnez **Mobotix**.
4. **Saisissez les détails suivants** :
   * **Name** : Un nom descriptif pour ce capteur.
   * **Hostname/IP Address** : L'adresse IP de la caméra Mobotix.
   * **Port** : Le port utilisé par la caméra.
5. **Enregistrez la configuration**.

Une fois terminé, **PADS4 se connectera à la caméra Mobotix** et commencera à **recevoir des données d'audience**.

***

## **Notes supplémentaires**

* **Capteurs certifiés** : Pour plus de détails sur les données récupérées de Mobotix, consultez la page **Certified Sensors** dans PADS4.
* **Gestion des événements** : Les événements configurés déclencheront **la création d'incidents en temps réel** dans PADS4.
* **Utilisations de l'intégration** :
  * **Reconnaissance faciale pour la mesure d'audience**
  * **Détection du port de masque par les individus**
  * **Déclenchement de règles et d'automatisation PADS4 basé sur les données détectées**

En suivant cette configuration, **les caméras Mobotix s'intégreront parfaitement à PADS4**, améliorant l'analyse d'audience en temps réel et l'automatisation basée sur les événements.
