Ce guide explique comment connecter **les caméras Mobotix** avec **PADS4** pour récupérer des données de mesure d'audience en temps réel. Pour plus d'informations sur les caméras Mobotix, visitez : [www.Mobotix.com](https://www.mobotix.com).

***

## **Étape 1 : Configuration de l'appareil Mobotix M7**

Pour assurer une intégration correcte, les profils d'événements dans la caméra Mobotix doivent être configurés pour correspondre à l'**AIFacedetect App**.

### **Accès aux paramètres de la caméra**

1. **Connectez-vous** à la caméra Mobotix.
2. Accédez à **Setup Menu > Event Control > Event Overview > Message Events**.

### **Vérifier et appliquer les propriétés d'événement suivantes**

| **Propriété**                 | **Valeur**          |
| ----------------------------- | ------------------- |
| Type de capteur d'événement   | **MxMessageSystem** |
| Nom du message                | **AIFacedetect**    |
| Plage du message              | **Local**           |
| Filtrer le contenu du message | **Aucun filtre**    |

***

## **Étape 2 : Configuration de l'application AIFacedetect**

Pour activer **la détection faciale** pour les visages masqués et non masqués :

1. **Accéder à :**\
   **Setup Menu > Certified App Settings > AIFacedetect Settings > Configurator**.
2. Appliquer les paramètres suivants :

| **Propriété**                    | **Valeur**                |
| -------------------------------- | ------------------------- |
| Algorithme de détection          | **Deep**                  |
| Détecter les visages non masqués | **Activer**               |
| Détecter les visages masqués     | **Activer**               |
| Gestion des événements           | **Événement à la sortie** |

3. **Cliquer sur « Set »** pour enregistrer les paramètres.

***

## **Étape 3 : Configuration de la connexion Mobotix dans PADS4**

1. **Ouvrir PADS4 CTRL Center**.
2. **Accéder à Sensors** (Si la catégorie **Sensors** n'est pas visible, vous aurez peut-être besoin d'une **licence de module IOT**. Contactez votre partenaire PADS4 pour obtenir de l'aide).
3. Cliquer sur **New Camera Software** et sélectionner **Mobotix**.
4. **Saisir les informations suivantes** :
   * **Nom** : Un nom descriptif pour ce capteur.
   * **Nom d'hôte/Adresse IP** : L'adresse IP de la caméra Mobotix.
   * **Port** : Le port utilisé par la caméra.
5. **Enregistrer la configuration**.

Une fois terminé, **PADS4 se connectera à la caméra Mobotix** et commencera à **recevoir les données d'audience**.

***

## **Notes supplémentaires**

* **Capteurs certifiés** : Pour plus de détails sur les données récupérées depuis Mobotix, consultez la page **Certified Sensors** dans PADS4.
* **Gestion des événements** : Les événements configurés déclencheront **la création d'incidents en temps réel** dans PADS4.
* **Utilisations de l'intégration** :
  * **Reconnaissance faciale pour la mesure d'audience**
  * **Détection du port du masque par les individus**
  * **Déclenchement de règles et d'automatisations PADS4 basées sur les données détectées**

En suivant cette configuration, **les caméras Mobotix s'intégreront de manière transparente avec PADS4**, améliorant ainsi l'analyse d'audience en temps réel et l'automatisation pilotée par les événements.
