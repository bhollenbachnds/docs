---
- title: Paramètres des capteurs
---

Les paramètres de capteurs dans PADS4 vous permettent de gérer la façon dont **les données de capteurs, les images et les rapports d'incidents** sont stockés dans la base de données. Vous pouvez configurer **la durée de stockage**, **la gestion des images**, et **le type de base de données** pour des performances système optimales.

## **Configuration du Stockage des Données de Capteurs**

* **Définissez la période de rétention** des données de capteurs.
* Les données plus anciennes sont **automatiquement supprimées** après la période spécifiée.
* Si nécessaire, **des outils tiers** peuvent être utilisés pour **copier et exporter** les informations de la base de données.

## **Gestion des Images de Capteurs**

* Définissez la durée pendant laquelle **les images de capteurs** doivent être stockées.
* En raison des **réglementations GPRS**, le stockage des images peut être **désactivé** pour la conformité à la vie privée.

## **Paramètres de Stockage des Incidents**

* Spécifiez la durée pendant laquelle **les données d'incidents** (générées par les capteurs) sont conservées dans la base de données.
* Après la période définie, **les incidents seront automatiquement supprimés**.

## **Configuration de la Base de Données**

### **Base de Données par Défaut**

* **SQLite** est utilisé **par défaut** pour stocker les données de capteurs.

### **Modification des Paramètres de la Base de Données**

* La configuration de la base de données peut être modifiée en utilisant **NDS.DataStorage.ConfigurationTool.exe**.
* Location:\
  `C:\Program Files\NDS\PADS4 Foundation\NDS.DataStorage.ConfigurationTool`
* Vous pouvez également démarrer cet outil depuis la **Foundation Configuration Application** après l'installation.

## **Conclusion**

PADS4 offre une **gestion flexible des données de capteurs**, garantissant la conformité avec les **politiques de stockage et les réglementations** tout en optimisant les **performances et la sécurité du système**.
