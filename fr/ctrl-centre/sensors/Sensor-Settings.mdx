Les paramètres de capteur dans PADS4 vous permettent de gérer comment les **données de capteur, images et rapports d'incidents** sont stockés dans la base de données. Vous pouvez configurer la **durée de stockage**, **la gestion des images**, et le **type de base de données** pour des performances système optimales.

***

## **Configuration du stockage des données de capteur**

* **Définissez la période de conservation** pour les données de capteur.
* Les données plus anciennes sont **automatiquement supprimées** après la période spécifiée.
* Si nécessaire, **des outils tiers** peuvent être utilisés pour **copier et exporter** les informations de la base de données.

***

## **Gestion des images de capteur**

* Définissez combien de temps les **images de capteur** doivent être stockées.
* En raison des **réglementations GPRS**, le stockage d'images peut être **désactivé** pour la conformité à la confidentialité.

***

## **Paramètres de stockage des incidents**

* Spécifiez combien de temps les **données d'incident** (générées par les capteurs) sont conservées dans la base de données.
* Après la période définie, **les incidents seront automatiquement supprimés**.

***

## **Configuration de la base de données**

### **Base de données par défaut**

* **SQLite** est utilisé **par défaut** pour stocker les données de capteur.

### **Modification des paramètres de la base de données**

* La configuration de la base de données peut être modifiée à l'aide de **NDS.DataStorage.ConfigurationTool.exe**.
* Location:\
  `C:\Program Files\NDS\PADS4 Foundation\NDS.DataStorage.ConfigurationTool`
* Alternativement, vous pouvez démarrer cet outil à partir de l'**Foundation Configuration Application** après l'installation.

***

## **Conclusion**

PADS4 offre une **gestion flexible des données de capteur**, assurant la conformité avec les **politiques et réglementations de stockage** tout en optimisant les **performances et la sécurité** du système.
