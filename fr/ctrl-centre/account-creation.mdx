<Info>
  À partir de la version 2025.1 de PADS4 et au-delà, les utilisateurs et (la plupart) des permissions ont été déplacés vers le Portail Web PADS4. Alors qu'il était auparavant nécessaire d'avoir l'application System Management installée, vous pouvez désormais effectuer l'administration de la gestion des utilisateurs depuis votre navigateur web.
</Info>

La section utilisateurs et rôles dans PADS4 vous permet de créer des comptes utilisateurs avec la possibilité de définir exactement les accès dont ils ont besoin.

## **Comment créer votre premier utilisateur**

1. Connectez-vous au Portail Web PADS4 en naviguant vers l'emplacement où votre serveur PADS4 est situé / hébergé : http(s)://servername:portnumber
   * HTTPS est utilisé dans les environnements sécurisés, sinon utilisez http.
   * Servername spécifie sur quel serveur PADS4 CTRL Center est installé.
   * Portnumber est optionnel lorsqu'il est hébergé sur le port 80 ou 443
   * Connectez-vous avec le **su\root** compte créé lors de l'installation.

2. En haut à côté du logo PADS4, cliquez sur le **SU** menu déroulant

3. Dans la liste déroulante, sélectionnez l'option correspondant à votre domaine. Le domaine par défaut est **pads**.

   Si vous hébergez un environnement multi-domaines, cliquez sur le domaine pour lequel vous souhaitez créer un utilisateur.

   <img src="/images/ctrl-centre/accountcreation/Selectdomain.png" alt="Selectdomain Pn" style={{ width:"54%" }} />

4. Naviguez vers utilisateurs

   <img src="/images/ctrl-centre/accountcreation/Users.png" alt="Users Pn" style={{ width:"52%" }} />

5. Cliquez sur **+Nouveau** et sélectionnez Utilisateur.

   <img src="/images/ctrl-centre/accountcreation/Addusers.png" alt="Addusers Pn" style={{ width:"51%" }} />

6. Remplissez les **informations détaillées de l'utilisateur (nom d'utilisateur, email, etc.).**

   <img src="/images/ctrl-centre/accountcreation/Credentials.png" alt="Credentials Pn" style={{ width:"53%" }} />

7. Sélectionnez un **rôle** à attribuer à l'utilisateur ou créez une nouvelle catégorie de rôle en cliquant sur **+ Créer un rôle.**

   <img src="/images/ctrl-centre/accountcreation/Selectroles.png" alt="Selectroles Pn" style={{ width:"51%" }} />

8. Attribuez une ou plusieurs **licences** à l'utilisateur.

   <img src="/images/ctrl-centre/accountcreation/Selectlicenses.png" alt="Selectlicenses Pn" style={{ width:"53%" }} />
