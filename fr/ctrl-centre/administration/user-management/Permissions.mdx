Dans notre dernière version, 2025.2, nous introduisons un nouvel ajout à notre nouveau système de gestion des utilisateurs.

Un aperçu complet de toutes les permissions configurées pour un utilisateur ou des rôles pour les modules suivants :

* Content
* Viewers
* CMS 2.0

## Aperçu des permissions

Une fois que vous avez ouvert la page Permissions, vous pouvez choisir quelles permissions vous souhaitez examiner.

Vous avez la possibilité de choisir n'importe quel utilisateur spécifique ainsi que n'importe quel rôle spécifique.

* Type: **Utilisateur**

![Po Type User](/images/digital-signage/cms/po-type-user.png)

Type: **Rôle**

![Po Type Role](/images/digital-signage/cms/po-type-role.png)

Une fois que vous avez sélectionné et soumis votre choix, les permissions seront actualisées et afficheront les permissions qui ont été configurées pour cet utilisateur ou ce rôle comme indiqué ci-dessous :

* **Content** : Toutes les permissions pour tous les dossiers de contenu et les fichiers individuels sont affichées ici.

![Po Content](/images/digital-signage/cms/po-content.png)

* **Viewer** : Toutes les permissions pour tous les groupes de viewers et les viewers individuels sont affichées ici.

![Po Viewers](/images/digital-signage/cms/po-viewers.png)

* **CMS** : Toutes les permissions pour toutes les catégories et tous les modèles CMS 2.0 sont affichées ici.

![Po Cms](/images/digital-signage/cms/po-cms.png)
