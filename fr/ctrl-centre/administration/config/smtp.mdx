<Warning>
  Depuis PADS4 version 2025.2, la configuration SMTP est effectuée dans le PADS4 Web Portal au lieu de la page du plugin Admin CMS 1.0.
</Warning>

Ce document fournit des instructions étape par étape pour configurer SMTP avec une authentification de base, aussi bien pour le PADS4 Web Portal que pour CMS 1.0 / Workspace.

<Tip>
  Notre implémentation SMTP prend également en charge SMTP avec l'authentification OAuth2. Plus d'informations concernant la configuration de SMTP avec OAuth2 sont disponibles ici. [SMTP-Oauth2](https://docs.pads4.com/ctrl-centre/administration/smtp-oauth2/Microsoft)
</Tip>

## Comment configurer SMTP pour votre serveur PADS4

* Pour commencer :
  * Ouvrez votre **navigateur web**.
  * Accédez à l'**URL du serveur PADS4** (soit un nom DNS, soit une adresse IP).
  * Connectez-vous à l'aide d'un **compte Administrateur**.
  * Cliquez sur **config**

![SMTP Webportal](/images/ctrl-centre/administration/config/SMTP-Webportal.png)

* Ou, si vous êtes déjà connecté :
  * Utilisez le bouton **Toggle Sidebar** situé en **haut à gauche** de l'écran.
  * Accédez à **Config**.
  * Puis appuyez sur **Modifier.**

![Smtp Settings](/images/ctrl-centre/administration/config/smtp-settings.png)

* Renseignez les détails de votre serveur SMTP et de votre compte
* Vous pouvez ensuite tester la connexion en bas à gauche, ce qui affichera une coche verte si la connexion est correctement configurée.

![Smtp Test Connection](/images/ctrl-centre/administration/config/smtp-test-connection.png)
