<Warning>
  Depuis la version 2025.2 de PADS4, la configuration SMTP se fait dans le portail Web PADS4 au lieu de la page du plugin Admin CMS 1.0.
</Warning>

Ce document fournit des instructions étape par étape pour configurer SMTP avec authentification de base pour le portail Web PADS4 ainsi que pour CMS 1.0 / Workspace.

<Tip>
  Notre implémentation SMTP prend également en charge SMTP avec authentification OAuth2, plus d'informations concernant la configuration de SMTP avec OAuth2 peuvent être trouvées ici. [SMTP-Oauth2](https://docs.pads4.com/ctrl-centre/administration/smtp-oauth2/Microsoft)
</Tip>

## Comment configurer SMTP pour votre serveur PADS4

* Pour commencer :
  * Ouvrez votre **navigateur web**.
  * Naviguez vers l'**URL du serveur PADS4** (soit un nom DNS, soit une adresse IP).
  * Connectez-vous en utilisant un **compte Administrateur**.
  * Cliquez sur **config**

![SMTP Webportal](/images/ctrl-centre/administration/config/SMTP-Webportal.png)

* Ou, lorsque vous êtes déjà connecté :
  * Utilisez le bouton **Toggle Sidebar** en **haut à gauche** de l'écran.
  * Naviguez vers **Config**.
  * Puis appuyez sur **Edit.**

![Smtp Settings](/images/ctrl-centre/administration/config/smtp-settings.png)

* Remplissez les détails de votre serveur SMTP et de votre compte
* Vous pouvez ensuite tester la connexion en bas à gauche, ce qui affichera une coche verte si la connexion est correctement configurée.

![Smtp Test Connection](/images/ctrl-centre/administration/config/smtp-test-connection.png)
