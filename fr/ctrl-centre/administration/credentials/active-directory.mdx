---
- title: Active Directory
---

PADS4 permet l'intégration avec **Active Directory (AD)** pour simplifier **l'authentification et la gestion des utilisateurs**. Cela s'effectue via l'interface **Credentials** dans **PADS4 CTRL Center**.

Suivez les étapes ci-dessous pour connecter votre instance Active Directory :

***

## **1. Accès au PADS4 CTRL Center**

Pour commencer :

* Ouvrez votre **navigateur web**.
* Accédez à l'**URL du serveur PADS4** (un nom DNS ou une adresse IP).
* Connectez-vous à l'aide d'un **compte Administrateur**.

Une fois connecté :

* Cliquez sur **Dashboard**.
* Utilisez le bouton **Toggle Sidebar** situé en **haut à droite** de l'écran.
* Accédez à **Credentials**.
* Puis sélectionnez **Providers**.

***

## **2. Ajout d'une connexion Active Directory**

Pour ajouter Active Directory :

1. Cliquez sur **+ New**
2. Sélectionnez **Identity**
3. Choisissez **Active Directory**

Vous serez invité à renseigner les détails de configuration requis :

* **Host Name** – Le nom de domaine ou l'adresse IP de votre serveur AD.
* **Port** – Généralement `389` pour LDAP ou `636` pour LDAPS.
* **Use SSL** – Activez cette option si vous utilisez **Secure LDAP (LDAPS)**.
* **Username** – Un compte AD avec des autorisations de lecture (par exemple, `admin@domain.local`).
* **Password** – Le mot de passe associé au compte ci-dessus.

Une fois terminé, cliquez sur **Create** pour finaliser la connexion.

***

## **3. Gestion des utilisateurs via Active Directory**

Après une connexion réussie :

* Vous pouvez désormais **importer et gérer les utilisateurs** depuis votre Active Directory dans PADS4.
* Cela permet un contrôle centralisé des utilisateurs et une **attribution des rôles et authentification fluides**.

Pour des fonctionnalités de gestion plus avancées, consultez la **documentation PADS4 User Management**.
