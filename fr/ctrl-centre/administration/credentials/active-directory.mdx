PADS4 permet l'intégration avec **Active Directory (AD)** pour simplifier **user authentication and management**. Cela se fait via l'interface **Credentials** dans **PADS4 CTRL Center**.

Suivez les étapes ci-dessous pour connecter votre instance Active Directory :

***

## **1. Accéder au PADS4 CTRL Center**

Pour commencer :

* Ouvrez votre **web browser**.
* Naviguez vers l'**PADS4 server URL** (soit un nom DNS, soit une adresse IP).
* Connectez-vous en utilisant un **Administrator account**.

Une fois connecté :

* Cliquez sur **Dashboard**.
* Utilisez le bouton **Toggle Sidebar** en **top-right** de l'écran.
* Naviguez vers **Credentials**.
* Puis sélectionnez **Providers**.

***

## **2. Ajouter une connexion Active Directory**

Pour ajouter Active Directory :

1. Cliquez sur **+ New**
2. Sélectionnez **Identity**
3. Choisissez **Active Directory**

Vous serez invité à remplir les détails de configuration requis :

* **Host Name** – Le nom de domaine ou l'IP de votre serveur AD.
* **Port** – Généralement `389` pour LDAP ou `636` pour LDAPS.
* **Use SSL** – Activez cette option si vous utilisez **Secure LDAP (LDAPS)**.
* **Username** – Un compte AD avec des permissions de lecture (par exemple, `admin@domain.local`).
* **Password** – Le mot de passe associé au compte ci-dessus.

Une fois terminé, cliquez sur **Create** pour finaliser la connexion.

***

## **3. Gestion des utilisateurs via Active Directory**

Après une connexion réussie :

* Vous pouvez maintenant **importer et gérer les utilisateurs** de votre Active Directory dans PADS4.
* Cela permet un contrôle centralisé des utilisateurs et une **attribution des rôles et authentification**.

Pour des fonctionnalités de gestion plus avancées, consultez la **documentation de gestion des utilisateurs PADS4**.
