La section **Viewers** dans **PADS4** affiche tous les **Viewers et groupes de Viewers**. Un **Viewer** est le **point de terminaison** où les présentations sont affichées.

***

## **Aperçu**

* Affiche **tous les Viewers disponibles** et **les groupes de Viewers**.
* **Créer un nouveau Viewer** à l'aide du bouton **➕ Nouveau**.
* **Nom par défaut**:
  * Les Viewers utilisés pour la première fois **génèrent automatiquement** un nom basé sur le type d'appareil :
    * **Samsung** appareils → *Samsung000x*
    * **LG** appareils → *LG000x*
  * Les noms peuvent être modifiés via l'option **Modifier** (uniquement pour les Viewers en ligne).
* **Options d'affichage**:
  * Basculer entre **Vue en arborescence** et **Vue détaillée**.
  * **Vue en arborescence**

![Tree View](/images/ctrl-centre/viewers/tree-view.png)

* **Vue détaillée**

![Detail View](/images/ctrl-centre/viewers/detail-view.png)

* Utilisez l'option **Rechercher/Filtrer** pour localiser des Viewers spécifiques.

![Viewer Filter](/images/ctrl-centre/viewers/viewer-filter.png)

***

## **Créer un nouveau Viewer**

Suivez ces étapes pour **ajouter un nouveau Viewer** à PADS4 :

1️⃣ **Cliquez sur le bouton** ➕ **Nouveau** sur la page Viewers.

![New Viewer](/images/ctrl-centre/viewers/new-viewer.png)

2️⃣ **Sélectionnez le type de Viewer** (seuls les types de Viewers disposant de licences disponibles seront affichés) :

![New Viewer1](/images/ctrl-centre/viewers/new-viewer1.png)

* **Viewer Group** – Un groupe contenant plusieurs Viewers (simplifie la planification).
* **Viewer** – Un Viewer autonome prenant en charge les types **Windows et HTML5**.
* **Windows Viewer** – Un Viewer fonctionnant sous **Microsoft Windows**.
* **HTML5 Viewer** – Un Viewer fonctionnant sur des **systèmes non-Windows (compatibles HTML5)**.
* **Zone Viewer** – Un Viewer assigné à une **zone** spécifique dans une **présentation PADS4**.
* **Desktop Viewer** – Un Viewer fonctionnant sur des **PC de bureau Windows**.\
  3️⃣ **Attribuez un nom de Viewer unique** (optionnellement, ajoutez une description – par ex., l'emplacement du Viewer).\
  4️⃣ **Sélectionnez un groupe de Viewers** (le cas échéant).\
  5️⃣ **Attribuez une licence**:
* **Basic** – Prend en charge la gamme la plus restreinte d'éléments de présentation.
* **Xpress** – Offre une gamme **modérée** de fonctionnalités.
* **Xpert** – Fournit un **accès complet aux fonctionnalités** pour les présentations avancées.\
  6️⃣ **(Facultatif) Définir un mot de passe** – Le Viewer aura besoin de ce mot de passe pour s'authentifier lors de la connexion au **PADS4 Server**, garantissant ainsi la sécurité.

**Une fois toutes les étapes terminées, le nouveau Viewer est prêt à être utilisé !**

***

## **Informations sur le Viewer**

* Un simple clic sur une **ligne de Viewer** la développe pour afficher des **informations détaillées**.

![Viewer Detail](/images/ctrl-centre/viewers/viewer-detail.png)

* Un double-clic sur le Viewer vous amènera directement à la page de détails du Viewer décrite ci-dessous.
* En survolant l'un des Viewers avec votre souris, trois points apparaîtront à droite offrant plus d'options :

![Viewer Menu](/images/ctrl-centre/viewers/viewer-menu.png)

* **Voir les détails** – Inspecter et gérer le Viewer. ( Plus d'informations ci-dessous )
* **Planifications du Viewer** - Redirigera vers le module de planification pour vérifier les planifications de ce Viewer spécifique
* **Manage:**
  * **Redémarrer le périphérique Viewer** – Redémarrer le périphérique à distance.
  * **Redémarrer le Viewer** - Redémarrer l'application Viewer à distance.
  * **Récupérer la Blackbox** – Collecter des données de diagnostic.
* **Modifier** - Ici vous pouvez modifier les aspects suivants du Viewer :
  * Nom et description du Viewer
  * Attribuer ou modifier le groupe de Viewers
  * Attribuer une licence
  * Activer / désactiver l'exigence de mot de passe
* \*\*Supprimer - \*\*Supprimer le Viewer du système.
* \*\*Accès - \*\*Vous permet de définir des autorisations sur le Viewer / groupe de Viewers
  * [Vous pouvez en savoir plus à ce sujet ici.](https://docs.pads4.com/ctrl-centre/viewers/viewer-permissions)

## Informations détaillées sur le Viewer

![Viewer Details2](/images/ctrl-centre/viewers/viewer-details2.png)

* \*\*Informations système \*\*- Informations détaillées sur l'appareil, notamment les caractéristiques matérielles, l'espace de stockage, le firmware, la résolution, etc.
* **Viewer Log** – Consulter les journaux d'activité.
* **Viewer Settings** – Modifier les préférences du Viewer.
* **Viewer Screenshot** – Capturer l'affichage actuel.

## **Viewer Blackbox (Pour le dépannage)**

* Si un **problème de Viewer survient**, **NDS Support** peut demander le **Blackbox** pour le dépannage.
* Le **Blackbox** contains:
  * **Logs**
  * **Informations système et mémoire**
  * **Autres données de diagnostic**
* **Comment fournir le Blackbox**:
  * Soumettre via le portail PADS4 Support pour NDS Support.
  * Le fournir à votre **partenaire** pour une analyse approfondie.
  * Utiliser **PADS4 Agent** pour analyser les données.
