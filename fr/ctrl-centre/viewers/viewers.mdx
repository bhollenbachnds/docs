La **Viewers** section dans **PADS4** affiche tous les **Viewers et groupes de Viewers** disponibles. Un **Viewer** est le **point de terminaison** où les présentations sont affichées.

## **Aperçu**

* Affiche **tous les Viewers disponibles** et les **groupes de Viewers**.
* **Créer un nouveau Viewer** à l'aide du bouton **➕ New**.
* **Nommage par défaut**:
  * Les Viewers de première connexion **génèrent automatiquement** un nom basé sur le type d'appareil :
    * **Samsung** appareils → *Samsung000x*
    * **LG** appareils → *LG000x*
  * Les noms peuvent être modifiés via l'option **Edit** (uniquement pour les Viewers en ligne).
* **Options d'affichage**:
  * Basculer entre **Tree View** et **Detail View**.
  * **Vue arborescente**

![Tree View](/images/ctrl-centre/viewers/tree-view.png)

* **Vue détaillée**

![Detail View](/images/ctrl-centre/viewers/detail-view.png)

* Utilisez l'option **Recherche/Filtre** pour localiser des Viewers spécifiques.

![Viewer Filter](/images/ctrl-centre/viewers/viewer-filter.png)

## **Créer un nouveau Viewer**

Suivez ces étapes pour **ajouter un nouveau Viewer** à PADS4 :

1️⃣ **Cliquez sur le bouton** ➕ **New** sur la page Viewers.

![New Viewer](/images/ctrl-centre/viewers/new-viewer.png)

2️⃣ **Sélectionner le type de Viewer** (Seuls les types de Viewer disposant de licences disponibles seront affichés) :

![New Viewer1](/images/ctrl-centre/viewers/new-viewer1.png)

* **Viewer Group** – Un groupe contenant plusieurs Viewers (simplifie la planification).
* **Viewer** – Un Viewer autonome qui prend en charge les types **Windows et HTML5**.
* **Windows Viewer** – Un Viewer fonctionnant sous **Microsoft Windows**.
* **HTML5 Viewer** – Un Viewer fonctionnant sur **les systèmes non-Windows (compatibles HTML5)**.
* **Zone Viewer** – Un Viewer assigné à une **zone** spécifique dans une **présentation PADS4**.
* **Desktop Viewer** – Un Viewer fonctionnant sur **les PC de bureau Windows**.\
  3️⃣ **Attribuer un nom de Viewer unique** (optionnellement, ajouter une description – par ex., l'emplacement du Viewer).\
  4️⃣ **Sélectionner un Viewer Group** (le cas échéant).\
  5️⃣ **Attribuer une Licence** :
* **Basic** – Prend en charge la plus petite gamme d'éléments de présentation.
* **Xpress** – Offre une gamme **modérée** de fonctionnalités.
* **Xpert** – Fournit **un accès complet aux fonctionnalités** pour les présentations avancées.\
  6️⃣ **(Facultatif) Définir un mot de passe** – Le Viewer aura besoin de ce mot de passe pour s'authentifier lors de la connexion au **PADS4 Server**, garantissant ainsi la sécurité.

**Une fois toutes les étapes terminées, le nouveau Viewer est prêt à être utilisé !**

## **Informations sur le Viewer**

* Un simple clic sur une **ligne de Viewer** la développe pour afficher **des informations détaillées**.

![Viewer Detail](/images/ctrl-centre/viewers/viewer-detail.png)

* Un double-clic sur le viewer accède directement à la page de détails du viewer décrite ci-dessous.
* Lorsque vous survolez l'un des viewers avec votre souris, trois points apparaissent à droite offrant plus d'options :

![Viewer Menu](/images/ctrl-centre/viewers/viewer-menu.png)

* **Voir les détails** – Inspecter et gérer le Viewer. ( Plus d'informations ci-dessous )
* **Planifications du Viewer** - Redirige vers le module de planification pour vérifier les planifications de ce Viewer spécifique
* **Manage:**
  * **Redémarrer le périphérique** – Redémarrer le périphérique à distance.
  * **Redémarrer le viewer** - Redémarrer l'application viewer à distance.
  * **Récupérer la Blackbox** – Collecter les données de diagnostic.
* **Modifier** - Ici vous pouvez modifier les aspects suivants du viewer :
  * Nom et description du Viewer
  * Attribuer ou modifier le groupe de viewer
  * Attribuer une licence
  * Activer / désactiver l'exigence de mot de passe
* \*\*Supprimer - \*\*Supprimer le viewer du système.
* \*\*Accès - \*\*Vous permet de définir des autorisations sur le viewer / groupe de viewer
  * [Vous pouvez en savoir plus ici.](https://docs.pads4.com/ctrl-centre/viewers/viewer-permissions)

## Informations détaillées sur le viewer

![Viewer Details2](/images/ctrl-centre/viewers/viewer-details2.png)

* \*\*Informations système \*\*- Informations détaillées sur le périphérique, notamment les caractéristiques matérielles, l'espace de stockage, le firmware, la résolution, etc.
* **Journal du Viewer** – Consulter les journaux d'activité.
* **Paramètres du Viewer** – Modifier les préférences du Viewer.
* **Capture d'écran du Viewer** – Capturer l'affichage actuel.

## **Blackbox du Viewer (Pour le dépannage)**

* Si un **problème de Viewer survient**, **NDS Support** peut demander la **Blackbox** pour le dépannage.
* La **Blackbox** contains:
  * **Journaux**
  * **Informations système et mémoire**
  * **Autres données de diagnostic**
* **Comment fournir la Blackbox** :
  * Soumettre via le portail de support PADS4 pour NDS Support.
  * La fournir à votre **partenaire** pour une analyse approfondie.
  * Utiliser **PADS4 Agent** pour analyser les données.
