PADS4 CTRL Center est le hub central pour gérer et optimiser votre environnement PADS4, offrant une interface web intuitive pour surveiller, configurer et contrôler votre système.

## Présentation de PADS4 CTRL Center

PADS4 CTRL Center est une console de gestion web utilisée pour administrer et superviser l'ensemble de votre déploiement PADS4. Grâce à cette interface, vous pouvez surveiller l'état du système, ajuster les paramètres de configuration et contrôler les composants connectés depuis un seul emplacement.

## Démarrer PADS4 CTRL Center

Pour ouvrir PADS4 CTRL Center, entrez l'URL suivante dans votre navigateur :

`http(s)://servername:portnumber`

Utilisez `https` dans les environnements sécurisés ; dans tous les autres cas, vous pouvez utiliser `http`. Le `servername` spécifie le serveur sur lequel PADS4 CTRL Center est installé, et le `portnumber` est facultatif lorsque l'application est hébergée sur le port 80.

<Card title="Navigateurs web pris en charge" type="note">
  PADS4 CTRL Center prend officiellement en charge les navigateurs web suivants :

  * Google Chrome
  * Microsoft Edge

  Les autres navigateurs ne sont pas pris en charge et peuvent ne pas fonctionner comme prévu
</Card>

## **Se connecter à PADS4 CTRL Center**

Pour vous connecter à PADS4 CTRL Center, suivez ces étapes :

* Ouvrez un navigateur web pris en charge et accédez à l'URL de PADS4 CTRL Center.
* La page de connexion PADS4 s'affiche.
* Entrez votre Nom d'utilisateur et votre Mot de passe, puis cliquez sur Se connecter.

<img height="200" src="/images/ctrl-centre/login.png" />

* Lorsque des identifiants valides sont saisis, la page d'accueil de PADS4 CTRL Center s'affiche.
* Si vous disposez d'autorisations pour plusieurs applications, une liste des applications disponibles s'affiche. Sélectionnez l'application que vous souhaitez ouvrir.
