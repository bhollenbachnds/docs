## **Pourquoi des licences sont-elles nécessaires ?**

Les **utilisateurs et les Viewers** nécessitent des **licences** valides pour fonctionner correctement dans PADS4. Les licences doivent être **achetées auprès d'un revendeur agréé** puis activées dans le **PADS4 CTRL Center**.

Une fois les licences activées, elles peuvent être **attribuées aux utilisateurs** via **PADS4 System Management**, débloquant des fonctionnalités spécifiques pour chaque utilisateur.

***

## **Conditions d'activation de la licence**

✔ **Accès administrateur requis**\
Seuls les utilisateurs du **groupe Administrateurs** peuvent activer une licence.

✔ **Activation en ligne ou hors ligne**\
Les licences peuvent être activées à l'aide d'une **connexion Internet (en ligne)** ou d'un **téléchargement manuel de fichier (hors ligne)**.

***

## **Activation de licence en ligne (Internet requis)**

1. **Ouvrir PADS4 CTRL Center**.
2. **Accéder à** la **Licences** section depuis le menu de gauche.
3. Cliquer **Activer** to:
   * **Activer un nouveau système PADS4**.
   * **Vérifier les mises à jour de licence**.
4. Le système récupérera et appliquera automatiquement les mises à jour de licence.

***

## **Activation de licence hors ligne (Pas d'Internet disponible)**

Si une connexion Internet n'est **pas disponible**, les licences peuvent être activées manuellement.

1. **Ouvrir PADS4 CTRL Center**.
2. **Accéder à** la **Licences** section.
3. Cliquer **Activer** pour démarrer le processus.
4. Le système demandera une **Activation hors ligne**.
5. **Enregistrer le fichier d'activation** sur votre disque local.
6. Visiter [**Portail d'activation PADS4**](https://activation.pads4.com/portal) et suivre les instructions.
7. Une fois l'activation terminée, **téléverser le fichier de licence retourné** dans PADS4 CTRL Center.

***

## **Téléverser le fichier de licence**

* Si le **serveur n'a pas accès à Internet**, l'activation doit être effectuée manuellement.
* Une fois que NDS fournit le **fichier de licence**, le téléverser en utilisant le **Téléverser le fichier de licence** bouton dans **PADS4 CTRL Center**.

***

## **Attribution des licences aux utilisateurs**

Après l'activation, les licences doivent être attribuées aux utilisateurs :

1. Ouvrir **PADS4 System Management**.
2. Accéder à la **Gestion des utilisateurs** section.
3. Sélectionner l'utilisateur et attribuer la licence requise.

🔗 **[Cliquer ici pour des instructions détaillées sur la gestion des licences](#).**

***

## **Points clés**

✔ Les licences sont requises pour les **utilisateurs et les Viewers**.\
✔ **L'accès administrateur** est nécessaire pour l'activation.\
✔ Choisir entre **l'activation en ligne ou hors ligne** méthodes.\
✔ **L'attribution de licence** débloque les fonctionnalités pour les utilisateurs.

Pour plus de détails, se référer à **Aide de PADS4 System Management**.
