## **Pourquoi des licences sont-elles nécessaires ?**

Les **Users et Viewers** nécessitent des **licences** valides pour fonctionner correctement dans PADS4. Les licences doivent être **achetées auprès d'un revendeur agréé** puis activées dans le **PADS4 CTRL Center**.

Une fois les licences activées, elles peuvent être **attribuées aux utilisateurs** via **PADS4 System Management**, débloquant des fonctionnalités spécifiques pour chaque utilisateur.

***

## **Conditions requises pour l'activation des licences**

✔ **Accès administrateur requis**\
Seuls les utilisateurs du **groupe Administrators** peuvent activer une licence.

✔ **Activation en ligne ou hors ligne**\
Les licences peuvent être activées à l'aide d'une **connexion internet (en ligne)** ou d'un **téléchargement manuel de fichier (hors ligne)**.

***

## **Activation de licence en ligne (Internet requis)**

1. **Ouvrez PADS4 CTRL Center**.
2. **Accédez à** la section **Licenses** depuis le menu de gauche.
3. Cliquez sur **Activate** to:
   * **Activer un nouveau système PADS4**.
   * **Vérifier les mises à jour de licence**.
4. Le système récupérera et appliquera automatiquement les mises à jour de licence.

***

## **Activation de licence hors ligne (Pas d'Internet disponible)**

Si une connexion internet n'est **pas disponible**, les licences peuvent être activées manuellement.

1. **Ouvrez PADS4 CTRL Center**.
2. **Accédez à** la section **Licenses**.
3. Cliquez sur **Activate** pour démarrer le processus.
4. Le système vous invitera à effectuer une **Activation hors ligne**.
5. **Enregistrez le fichier d'activation** sur votre disque local.
6. Visitez le [**PADS4 Activation Portal**](https://activation.pads4.com/portal) et suivez les instructions.
7. Une fois l'activation terminée, **téléchargez le fichier de licence retourné** dans PADS4 CTRL Center.

***

## **Télécharger le fichier de licence**

* Si le **serveur n'a pas accès à Internet**, l'activation doit être effectuée manuellement.
* Une fois que NDS fournit le **fichier de licence**, téléchargez-le à l'aide du bouton **Upload License File** dans **PADS4 CTRL Center**.

***

## **Attribution des licences aux utilisateurs**

Après l'activation, les licences doivent être attribuées aux utilisateurs :

1. Ouvrez **PADS4 System Management**.
2. Accédez à la section **User Management**.
3. Sélectionnez l'utilisateur et attribuez la licence requise.

🔗 **[Cliquez ici pour des instructions détaillées sur la gestion des licences](#).**

***

## **Points clés à retenir**

✔ Les licences sont requises pour les **utilisateurs et les Viewers**.\
✔ **L'accès administrateur** est nécessaire pour l'activation.\
✔ Choisissez entre les méthodes d'activation **en ligne ou hors ligne**.\
✔ **L'attribution de licence** débloque les fonctionnalités pour les utilisateurs.

Pour plus de détails, consultez **PADS4 System Management Help**.
