Les variables Viewer, également connues sous le nom de **Filtrage Dynamique**, permet **la gestion de contenu pilotée par les données** dans PADS4. Cette fonctionnalité vous permet de créer une **présentation unique** qui s'ajuste dynamiquement pour plusieurs Viewers selon des valeurs prédéfinies.

Pour un exemple détaillé sur l'utilisation des Variables Viewer, consultez la **Page d'Introduction aux Variables Viewer**.

***

## **Variables Viewer Système**

En plus des **Variables Viewer personnalisées**, PADS4 fournit des **Variables Viewer Système** à l'exécution. Ces variables intégrées vous permettent d'accéder dynamiquement aux informations système essentielles.

### **Variables Viewer Système disponibles**

| **Nom de la Variable Viewer** | **Description**                                                                                               |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------- |
| **viewername**                | Le **nom d'identification** du Viewer.                                                                        |
| **monitorname**               | Identique à **viewername**. Utilisé dans les anciennes versions de PADS pour la **compatibilité ascendante**. |
| **pads4serverurl**            | Affiche l'**URL du Serveur PADS4**, formatée comme :**HTTP(s)://server/hostpath**                             |

***

## **Comment fonctionnent les Variables Viewer Système**

* **Mises à jour automatiques** – Les Variables Viewer Système sont générées par PADS4 à l'exécution et reflètent toujours l'état actuel du Viewer.
* **Utilisation dans les présentations** – Vous pouvez les inclure dans des **champs de texte**, **fournisseurs de données**, ou **autres éléments de contenu dynamique**.
* **Compatibilité** – Ces variables assurent une **intégration transparente** entre différentes versions de PADS4.

***

## **Conclusion**

Les Variables Viewer Système fournissent des informations essentielles sur le **Viewer et le Serveur PADS4** sans nécessiter de saisie manuelle. Elles simplifient la **personnalisation et l'automatisation du contenu**, faisant de PADS4 une solution d'**affichage dynamique** plus efficace.
