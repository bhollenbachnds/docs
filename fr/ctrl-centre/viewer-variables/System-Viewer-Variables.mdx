Les Variables de Viewer, également connues sous le nom de **Dynamic Filtering**, permettent une **gestion de contenu basée sur les données** dans PADS4. Cette fonctionnalité vous permet de créer une **présentation unique** qui s'ajuste dynamiquement pour plusieurs Viewers en fonction de valeurs prédéfinies.

Pour un exemple détaillé sur l'utilisation des Variables de Viewer, consultez la **Viewer Variables Introduction Page**.

***

## **Variables de Viewer système**

En plus des **Variables de Viewer personnalisées**, PADS4 fournit des **Variables de Viewer système** lors de l'exécution. Ces variables intégrées vous permettent d'accéder dynamiquement aux informations essentielles du système.

### **Variables de Viewer système disponibles**

| **Nom de la Variable de Viewer** | **Description**                                                                                               |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| **viewername**                   | Le **nom d'identification** du Viewer.                                                                        |
| **monitorname**                  | Identique à **viewername**. Utilisé dans les anciennes versions de PADS pour la **compatibilité ascendante**. |
| **pads4serverurl**               | Affiche l'**URL du Serveur PADS4**, formatée comme :**HTTP(s)://server/hostpath**                             |

***

## **Comment fonctionnent les Variables de Viewer système**

* **Mises à jour automatiques** – Les Variables de Viewer système sont générées par PADS4 lors de l'exécution et reflètent toujours l'état actuel du Viewer.
* **Utilisation dans les présentations** – Vous pouvez les inclure dans des **champs de texte**, **fournisseurs de données**, ou **autres éléments de contenu dynamique**.
* **Compatibilité** – Ces variables assurent une **intégration transparente** entre différentes versions de PADS4.

***

## **Conclusion**

Les Variables de Viewer système fournissent des informations essentielles sur le **Viewer et le Serveur PADS4** sans nécessiter de saisie manuelle. Elles simplifient la **personnalisation et l'automatisation du contenu**, faisant de PADS4 une solution **d'affichage dynamique** plus efficace.
