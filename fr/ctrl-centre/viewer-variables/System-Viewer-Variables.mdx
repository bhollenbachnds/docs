Les Viewer Variables, également connues sous le nom de **Dynamic Filtering**, permettent une **gestion de contenu basée sur les données** dans PADS4. Cette fonctionnalité vous permet de créer une **single presentation** qui s'ajuste dynamiquement pour plusieurs Viewers en fonction de valeurs prédéfinies.

Pour un exemple détaillé de l'utilisation des Viewer Variables, consultez la **Viewer Variables Introduction Page**.

***

## **System Viewer Variables**

En plus des **custom Viewer Variables**, PADS4 fournit des **System Viewer Variables** à l'exécution. Ces variables intégrées vous permettent d'accéder dynamiquement aux informations essentielles du système.

### **System Viewer Variables disponibles**

| **Viewer Variable Name** | **Description**                                                                                             |
| ------------------------ | ----------------------------------------------------------------------------------------------------------- |
| **viewername**           | Le **identification name** du Viewer.                                                                       |
| **monitorname**          | Identique à **viewername**. Utilisé dans les anciennes versions de PADS pour la **backward compatibility**. |
| **pads4serverurl**       | Affiche l'**URL of the PADS4 Server**, formatée comme :**HTTP(s)://server/hostpath**                        |

***

## **Comment fonctionnent les System Viewer Variables**

* **Automatic Updates** – Les System Viewer Variables sont générées par PADS4 à l'exécution et reflètent toujours l'état actuel du Viewer.
* **Use in Presentations** – Vous pouvez les inclure dans des **text fields**, **data providers**, ou **other dynamic content elements**.
* **Compatibility** – Ces variables assurent une **seamless integration** entre différentes versions de PADS4.

***

## **Conclusion**

Les System Viewer Variables fournissent des informations essentielles sur le **Viewer and PADS4 Server** sans nécessiter de saisie manuelle. Elles simplifient la **content customization and automation**, faisant de PADS4 une **digital signage solution** plus efficace.
