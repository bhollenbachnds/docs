Les Viewer Variables, également connues sous le nom de **Dynamic Filtering**, permettent une **gestion de contenu basée sur les données** dans PADS4. Cette fonctionnalité vous permet de créer une **présentation unique** qui s'ajuste dynamiquement pour plusieurs Viewers en fonction de valeurs prédéfinies.

Pour un exemple détaillé d'utilisation des Viewer Variables, consultez la **Viewer Variables Introduction Page**.

***

## **System Viewer Variables**

En plus des **custom Viewer Variables**, PADS4 fournit des **System Viewer Variables** à l'exécution. Ces variables intégrées vous permettent d'accéder dynamiquement aux informations essentielles du système.

### **System Viewer Variables disponibles**

| **Viewer Variable Name** | **Description**                                                                                               |
| ------------------------ | ------------------------------------------------------------------------------------------------------------- |
| **viewername**           | Le **nom d'identification** du Viewer.                                                                        |
| **monitorname**          | Identique à **viewername**. Utilisé dans les anciennes versions de PADS pour la **compatibilité ascendante**. |
| **pads4serverurl**       | Affiche l'**URL du PADS4 Server**, formatée comme :**HTTP(s)://server/hostpath**                              |

***

## **Comment fonctionnent les System Viewer Variables**

* **Mises à jour automatiques** – Les System Viewer Variables sont générées par PADS4 à l'exécution et reflètent toujours l'état actuel du Viewer.
* **Utilisation dans les présentations** – Vous pouvez les inclure dans des **champs de texte**, **fournisseurs de données**, ou **autres éléments de contenu dynamique**.
* **Compatibilité** – Ces variables assurent une **intégration transparente** entre différentes versions de PADS4.

***

## **Conclusion**

Les System Viewer Variables fournissent des informations essentielles sur le **Viewer et PADS4 Server** sans nécessiter de saisie manuelle. Elles simplifient la **personnalisation et l'automatisation du contenu**, faisant de PADS4 une **solution d'affichage dynamique** plus efficace.
