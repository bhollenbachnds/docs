---
- title: Installation de BONVOY VMS
---

Téléchargez le package d'installation BonVoy VMS depuis Finessetech ou recevez-le de la part du partenaire d'implémentation agréé. Exécutez le programme d'installation en tant qu'Administrateur.

### **Sélectionner l'emplacement de destination**

Cet écran permet à l'utilisateur de sélectionner le répertoire d'installation de BonVoy VMS.

<Frame>
  ![File Location](/images/Addons/Finessetech/BonvoyVMS/FileLocation.jpeg)
</Frame>

* Chemin d'installation – Spécifie l'emplacement où les fichiers BonVoy VMS seront installés.

<Check>
  Conservez le chemin d'installation par défaut, sauf si le client dispose d'un lecteur d'application dédié.
</Check>

Cliquez sur Suivant pour continuer l'installation.

### Configuration de SQL Server

**Cet écran configure la connexion SQL Server requise pour BonVoy VMS**

<Frame>
  ![Installer SQL Server](/images/Addons/Finessetech/BonvoyVMS/Installer-SQLServer.png)
</Frame>

* **SQL Server**: Spécifie le nom de l'instance SQL Server, par exemple `localhost`, `localhost\SQLEXPRESS`, ou `SERVERNAME\INSTANCE`
* **Nom d'utilisateur**: Spécifie le nom d'utilisateur pour l'authentification SQL.
* **Mot de passe**: Spécifie le mot de passe pour l'authentification SQL.\\

<Check>
  L'utilisateur SQL doit avoir la permission de créer ou de mettre à jour la base de données BonVoy.
</Check>

Cliquez sur Tester la connexion et assurez-vous que la connexion réussit avant de continuer.

Cliquez sur Suivant

### Paramètres PADS4 / VMS

Cet écran est utilisé pour configurer l'intégration entre BonVoy et le CMS PADS4.

<Frame>
  ![Installer PADS4 VMS Setting](/images/Addons/Finessetech/BonVoyVMS/Installer-PADS4-VMSSetting.png)
</Frame>

* **URL racine de l'API PADS4**: L'URL du serveur PADS4.
* **Nom d'utilisateur**: Un compte utilisateur PADS4 avec accès **Admin** et **VMS**.
* **Mot de passe**: Le mot de passe du compte utilisateur PADS4.
* **URL du serveur BonVoy**: L'URL où l'application web BonVoy est hébergée.
* **Port du serveur BonVoy**: Le port de l'application IIS utilisé par le serveur BonVoy.

Cliquez sur Suivant

### Prêt à installer

Cet écran affiche l'invite d'installation.

<Frame>
  ![Installer Ready](/images/AddOns/Finessetech/BonvoyVMS/Installer-Ready.png)
</Frame>

Cliquez sur Installer

### Progression de l'installation

Cet écran affiche la progression de l'installation.

<Frame>
  ![Installer Progress](/images/Addons/Finessetech/BonVoyVMS/Installer-Progress.png)
</Frame>

Au cours du processus d'installation, le programme d'installation effectue automatiquement les actions suivantes :

* Configure le site web IIS et le pool d'applications
* Crée la base de données BonVoy
* Restaure les objets de base de données requis
* Configure les paramètres de l'application
* Installe et démarre le service Windows

### Installation terminée

<Frame>
  ![Installer Complete](/images/Addons/Finessetech/BonVoyCMS/Installer-Complete.png)
</Frame>

Cochez la case et cliquez sur **Terminer** pour ouvrir l'application dans votre navigateur.

## Activation de la licence

**Après une installation réussie :**

1. Ouvrez l'application BonVoy dans votre navigateur.
2. Accédez à la page **Activation de la licence**.
3. Téléchargez le fichier XML de clé publique fourni.
4. Téléchargez le fichier de licence BonVoy.
5. Cliquez sur **Activer**.
