Utilisez **Réinitialiser MFA** lorsqu'un utilisateur ne peut plus accéder à son application d'authentification, a changé d'appareil ou doit enregistrer à nouveau MFA pour une autre raison.

La réinitialisation de MFA efface l'enregistrement MFA actuel de l'utilisateur, de sorte que l'utilisateur doit recommencer le processus d'enregistrement en scannant le code QR ou en saisissant le code manuel lors du flux de configuration.

## **Quand utiliser cette action**

Utilisez **Réinitialiser MFA** dans des situations telles que celles-ci :

* L'utilisateur a un nouveau téléphone.
* L'application d'authentification a été supprimée ou perdue.
* L'utilisateur ne peut pas effectuer MFA avec l'appareil précédemment enregistré.
* L'enregistrement MFA doit être redémarré depuis le début.

## **Comment réinitialiser MFA**

1. En tant qu'Administrateur PADS4, connectez-vous au PADS4 CTRL Center
2. Ouvrez Utilisateurs et localisez le compte utilisateur dont la MFA doit être réinitialisée.
3. Ouvrez le menu Plus d'actions de l'utilisateur pour ce compte.
4. Cliquez sur **Reset MFA** dans le menu.
   <Frame>
     <img src="/images/CTRLCenter/MFA/Reset-MFA.png" alt="Reset MFA" title="Reset MFA" style={{ width:"74%" }} />
   </Frame>
5. Confirmez l'action si PADS4 demande une confirmation.
