---
- title: Stratégies de groupe Microsoft Windows
---

De nombreuses organisations utilisent les stratégies de groupe Microsoft pour gérer de manière centralisée les paramètres des utilisateurs. Avec les stratégies de groupe de Microsoft, vous pouvez gérer de manière centralisée les paramètres du Desktop Viewer.

PADS4 a mis en œuvre des stratégies de groupe pour l'application Desktop Viewer afin de permettre à l'administrateur de spécifier les paramètres depuis un emplacement central et de les transmettre à un ou plusieurs utilisateurs ou groupes d'utilisateurs. Les stratégies définies par l'administrateur sont automatiquement détectées par la fenêtre d'options du Desktop Viewer et ces options sont désactivées dans l'interface utilisateur. De cette façon, il est possible de transmettre un ensemble de paramètres prédéfinis aux utilisateurs, tout en leur donnant la possibilité de modifier certains paramètres localement.

Les paramètres transmis sont stockés dans le registre sous :

HKEY\_CURRENT\_USER\SOFTWARE\Policies\Net Display Systems\PADS4

Les « paramètres normaux » sont stockés dans le registre sous :

HKEY\_CURRENT\_USER\SOFTWARE\Net Display Systems\PADS4

Pour configurer l'utilisation des stratégies de groupe pour l'installation des Desktop Viewers PADS4 – Serveur PADS4

Installez d'abord le serveur PADS4 (ou d'autres composants, mais les fichiers de stratégie sont installés avec l'installation du serveur).

Installer le fichier de modèle de stratégie de groupe

Copiez PADS4.admx depuis C:\Program Files\Net Display Systems\PADS4\PolicyDefinitions vers le répertoire de définitions de stratégie de domaine sur le serveur de domaine (généralement : \FQDN\SYSVOL\FQDN\Policies\PolicyDefinitions)

Fichier de traduction de stratégie

Copiez PADS4.adml depuis C:\Program Files\Net Display Systems\PADS4\PolicyDefinitions vers le répertoire de langue des définitions de stratégie de domaine sur le serveur de domaine (généralement : \FQDN\SYSVOL\<FQDN>\Policies\PolicyDefinitions\en-US)

Console de gestion des stratégies de groupe Microsoft (GPMC)

Ouvrez l'application Console de gestion des stratégies de groupe, puis ouvrez l'éditeur de stratégie de groupe.

Sélectionnez Configuration utilisateur / Stratégies / Modèles d'administration : Définition de stratégie / Net Display Systems / PADS4 / Desktop Viewer

REMARQUE

Cela permet à l'administrateur de spécifier les paramètres depuis un emplacement central et de les transmettre aux utilisateurs. Les utilisateurs ne peuvent pas remplacer ces paramètres via la fenêtre des options. Les options définies/transmises par l'administrateur sont désactivées dans la fenêtre des options du Desktop Viewer.
