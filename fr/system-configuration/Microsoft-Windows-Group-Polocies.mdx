De nombreuses organisations utilisent les stratégies de groupe Microsoft pour gérer de manière centralisée les paramètres des utilisateurs. Avec les stratégies de groupe de Microsoft, vous pouvez gérer de manière centralisée les paramètres du Desktop Viewer.

PADS4 a implémenté des stratégies de groupe pour l'application Desktop Viewer afin de permettre à l'administrateur de spécifier les paramètres depuis un emplacement central et de les transmettre à un ou plusieurs utilisateurs ou groupes d'utilisateurs. Les stratégies définies par l'administrateur sont automatiquement détectées par la fenêtre d'options du Desktop Viewer et ces options sont désactivées dans l'interface utilisateur. De cette façon, il est possible de transmettre un ensemble de paramètres prédéfinis aux utilisateurs, tout en leur donnant la possibilité de modifier certains paramètres localement.

Les paramètres transmis sont stockés dans le registre sous :

HKEY\_CURRENT\_USER\SOFTWARE\Policies\Net Display Systems\PADS4

Les 'paramètres normaux' sont stockés dans le registre sous :

HKEY\_CURRENT\_USER\SOFTWARE\Net Display Systems\PADS4

Pour configurer l'utilisation des stratégies de groupe pour les PADS4 Desktop Viewers
Installation PADS4 Server

Installez d'abord PADS4 Server (ou plus de composants, mais les fichiers de stratégie sont installés avec l'installation du Server).

Installer le fichier de modèle de stratégie de groupe

Copiez PADS4.admx depuis C:\Program Files\Net Display Systems\PADS4\PolicyDefinitions vers le répertoire Domain Policy Definitions sur le serveur de domaine (généralement : \FQDN\SYSVOL\FQDN\Policies\PolicyDefinitions)

Fichier de traduction de stratégie

Copiez PADS4.adml de C:\Program Files\Net Display Systems\PADS4\PolicyDefinitions vers le répertoire Domain Policy Definitions Language sur le serveur de domaine (généralement : \FQDN\SYSVOL\<FQDN>\Policies\PolicyDefinitions\en-US)

Microsoft Group Policy Management Console (GPMC)

Ouvrez l'application Group Policy Management Console puis ouvrez l'Éditeur de stratégie de groupe.

Sélectionnez Configuration utilisateur / Stratégies / Modèles d'administration : définition de stratégie / Net Display Systems / PADS4 / Desktop Viewer

REMARQUE

Cela permet à l'administrateur de spécifier les paramètres depuis un emplacement central et de les transmettre aux utilisateurs. Les utilisateurs ne peuvent pas remplacer ces paramètres à l'aide de la fenêtre Options. Les options définies/transmises par l'administrateur sont désactivées dans la fenêtre Options du Desktop Viewer.
