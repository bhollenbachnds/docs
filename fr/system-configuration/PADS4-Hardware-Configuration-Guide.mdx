## Conseils généraux de performance

Ces éléments sont nécessaires pour le bon fonctionnement et la configuration de votre environnement PADS4 respectif.

| Composant                     | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Disque                        | La synchronisation du contenu est l'un des processus qui a un impact important sur les performances du serveur. L'utilisation de disques SSD (Solid State Disks) améliore considérablement les performances du serveur.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Antivirus                     | Sachez que les antivirus peuvent avoir un impact important sur les performances liées à la synchronisation du contenu et des processus.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Serveur virtuel               | Lorsque vous utilisez un réseau virtuel, assurez-vous d'utiliser une adresse MAC statique. Une adresse MAC dynamique aura un impact négatif sur le mécanisme de licence PADS4 lors d'un redémarrage du serveur PADS4, ce qui peut nécessiter une réactivation.                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Réseau                        | Les meilleures performances seront atteintes lorsque le réseau est stable, 100Mbps ou plus rapide, réseau 1Gbit préféré                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Espace de stockage de contenu | La taille du stockage de contenu dépend des besoins du client et aura un impact sur la bande passante utilisée. L'utilisation de dispositifs de stockage SSD peut augmenter considérablement les performances tant pour le serveur que pour le Viewer.                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| Contenu                       | Veuillez convertir les vidéos 4K et les images 4K qui sont affichées sur des écrans Full HD aux spécifications des écrans pour améliorer les performances du réseau et de la visualisation.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Permissions de contenu        | Utilisez les permissions de contenu pour limiter la synchronisation du contenu envoyé à certains groupes de Viewers et/ou groupes d'utilisateurs. Cela empêche la synchronisation du contenu avec des appareils qui ne l'afficheront jamais.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| Data Server                   | Essayez de limiter la quantité de données récupérées à l'aide des fournisseurs de données pour éviter une utilisation excessive de la mémoire et des baisses de performance. Récupérez uniquement les informations de la source de données qui seront utilisées pour éviter une utilisation élevée de la bande passante. Limitez le nombre de lignes ou de colonnes utilisées pour n'obtenir que ce qui est utilisé par la présentation. Ne spécifiez pas de petits intervalles de lecture du fournisseur de données lorsque les données ne sont pas mises à jour en temps réel. Évitez les petits intervalles de lecture qui augmenteront l'utilisation du CPU lorsqu'il n'est pas nécessaire de lire les informations en continu. |
