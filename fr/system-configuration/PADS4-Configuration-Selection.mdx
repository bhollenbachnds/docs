## **Configurations d'Installation PADS4**

Vous pouvez choisir parmi **trois configurations différentes** selon vos besoins :

### Installation Autonome

**Page 13** – Adaptée aux **configurations mono-PC** où tous les **modules PADS4 et applications client** sont installés sur **un** seul appareil.

### Installation Réseau

**Page 14** – Conçue pour **environnements multi-PC**, où **PADS4 applications** sont installées sur **plusieurs appareils** pour créer une **configuration en réseau**.

### Installation de réseau global (large)

**Page 16** – Idéale pour **les réseaux mondiaux ou à grande échelle** avec **de nombreux écrans connectés**. Cette configuration **optimise l'utilisation de la bande passante** pour une distribution efficace du contenu.
