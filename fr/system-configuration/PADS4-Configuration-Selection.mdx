## **Configurations d'installation PADS4**

Vous pouvez choisir parmi **trois configurations différentes** selon vos besoins :

### Installation autonome

**Page 13** – Adaptée aux **configurations à PC unique** où tous les **PADS4 modules and client applications** sont installés sur **un seul** appareil.

### Installation réseau

**Page 14** – Conçue pour les **environnements multi-PC**, où les **PADS4 applications** sont installées sur **plusieurs appareils** pour créer une **configuration en réseau**.

### Installation réseau globale (grande échelle)

**Page 16** – Idéale pour les **réseaux mondiaux ou à grande échelle** avec **de nombreux écrans connectés**. Cette configuration **optimise l'utilisation de la bande passante** pour une distribution efficace du contenu.

## **Premiers pas avec PADS4**

Pour activer PADS4, vous aurez besoin d'une **clé de produit sous licence**.

**Contactez votre fournisseur local ou représentant commercial** pour obtenir votre **PADS4 license**.\
Vous recevrez un **manuel**, ainsi que les **Foundation & Clients installation files**.

Pour toute assistance supplémentaire, n'hésitez pas à **contacter votre représentant PADS4**!
