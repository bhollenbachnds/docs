## **Configurations d'Installation PADS4**

Vous pouvez choisir parmi **trois configurations différentes** selon vos besoins :

### Installation Autonome

**Page 13** – Adaptée aux **configurations à PC unique** où tous les **modules PADS4 et applications client** sont installés sur **un** seul appareil.

### Installation Réseau

**Page 14** – Conçue pour les **environnements multi-PC**, où les **applications PADS4** sont installées sur **plusieurs appareils** pour créer une **configuration en réseau**.

### Installation Réseau Global (Large)

**Page 16** – Idéale pour les **réseaux mondiaux ou à grande échelle** avec **de nombreux viewers connectés**. Cette configuration **optimise l'utilisation de la bande passante** pour une distribution efficace du contenu.

## **Démarrer avec PADS4**

Pour activer PADS4, vous aurez besoin d'une **clé de produit sous licence**.

**Contactez votre fournisseur local ou représentant commercial** pour obtenir votre **licence PADS4**.\
Vous recevrez un **manuel**, ainsi que le **Fichiers d'installation Foundation & Clients**.

Pour toute assistance supplémentaire, n'hésitez pas à **contacter votre représentant PADS4**!
