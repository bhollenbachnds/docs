## Présentation de l'interface

Une fois dans le Web Designer, l'interface est divisée en zones clés :

* **Zone de canevas** : L'espace de conception principal où vous placez et organisez les éléments.

<Frame>
  ![Create A Template13](/images/digital-signage/cms/create-a-template13.png)
</Frame>

* **Barre d'outils** : Donne accès aux outils de dessin, à l'insertion de médias, à l'alignement et à la gestion des calques.
  <Frame>
    <img src="/images/digital-signage/WebDesigner/WebDesignerElementToolBar.png" alt="Web Designer Element Tool Bar" title="Web Designer Element Tool Bar" style={{ width:"50%" }} />
  </Frame>

* **Panneau des propriétés** : Situé à droite, il affiche les options de configuration de l'élément sélectionné (par ex., taille, couleur, alignement).

<Frame>
  <img src="/images/digital-signage/cms/create-a-template5.png" alt="Create A Template5" title="Create A Template5" className="mr-auto" style={{ width:"64%" }} />
</Frame>

* **Panneau des calques** : Répertorie tous les objets du modèle, vous permettant de réorganiser, regrouper ou supprimer des éléments.\\
  <Frame>
    <img src="/images/digital-signage/cms/create-a-template4.png" alt="Create A Template4" title="Create A Template4" className="mr-auto" style={{ width:"72%" }} />
  </Frame>

## Agrandir la page principale

Pour zoomer ou agrandir l'espace de travail :

* Utilisez le curseur de zoom ou les icônes de zoom situés en bas à droite du canevas.
  <img src="/images/digital-signage/cms/create-a-template8.png" alt="Create A Template8" title="Create A Template8" style={{ width:"46%" }} className="mr-auto" />
* Vous pouvez également appuyer sur **Ctrl + Scroll** sur votre souris pour zoomer en avant et en arrière.

Cela permet un placement plus précis des éléments de conception.
