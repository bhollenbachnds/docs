L'installation de PADS4 comprend tous les modules spécifiés à l'exception des applications PADS4 Client (basées sur Windows) pour lesquelles une installation séparée est nécessaire lors d'une nouvelle installation.
Si vous mettez à jour votre système PADS4 qui contient des applications PADS4 Client, celles-ci seront également mises à niveau automatiquement.

## [PADS4 CTRL Center](/fr/ctrl-centre/introduction)

Portail PADS4 basé sur le web pour gérer et maintenir votre système PADS4. En plus des applications web, nous disposons également d'applications Windows pour gérer votre système PADS4.

## [PADS4 Digital Signage](/fr/modules/digital-signage/introduction)

Logiciel dédié appelé Viewers qui constitue le point final sur lequel le contenu est affiché. La présentation est créée à l'aide d'un puissant Designer.
Des serveurs secondaires et de sauvegarde peuvent être utilisés pour configurer des serveurs supplémentaires lorsque nécessaire, où PADS4 synchronisera automatiquement tout entre ces serveurs.

## [Applications client PADS4](https://docs.pads4.com/install-pads4/Downloads#pads4-client-applications)

Applications PADS4 basées sur Microsoft Windows qui peuvent être installées séparément sur les PC clients.
Ces applications peuvent se connecter à distance à n'importe quel serveur PADS4.

La plupart des fonctionnalités présentes dans l'application client peuvent également être disponibles dans le portail web PADS4 (\*actuellement, nous travaillons à la migration de toutes les fonctionnalités vers le portail web).

Chaque application dispose d'une aide distincte. Consultez cette page pour plus d'informations.

| Application             | Description                                                                      |
| ----------------------- | -------------------------------------------------------------------------------- |
| PADS4 Designer          | Application pour créer des designs de présentation à l'aide d'un éditeur WYSIWYG |
| PADS4 Scheduler         | Application pour planifier des présentations pour les Viewers                    |
| PADS4 Messenger         | Application pour planifier des messages pour les Desktop Viewers                 |
| PADS4 Agent             | Application pour surveiller le système PADS4                                     |
| PADS4 System Management | Application pour gérer le système PADS4                                          |
| PADS4 Desktop Viewer    | Application qui affiche les messages planifiés                                   |
| PADS4 Viewer            | Application qui affiche les présentations planifiées                             |

## [PADS4 FIDS](/fr/modules/fids/introduction)

Signalisation aéroportuaire numérique tout-en-un. PADS4 FIDS Operator est un système de gestion d'informations de vol pour gérer vos vols quotidiens pour les petits et moyens aéroports.
Les systèmes d'affichage d'informations de vol constituent le cœur de tout aéroport, opérant dans des fonctions cruciales tant pour les salles de contrôle que pour l'information des passagers.

## [PADS4 CMS](/fr/modules/digital-signage/cms/introduction)

Créez des messages à l'aide d'une application web et affichez-les sur les Viewers du module PADS4 Digital Signage.

## [PADS4 Workspace](/fr/modules/workspace/introduction)

Gestion des bureaux et des salles.

## PADS4 Wayfinding

Présentation de cartes interactives et guidage des personnes. Les cartes peuvent être affichées sur des bornes ou d'autres appareils interactifs.

## [PADS4 VMS](/fr/modules/visitor-management/introduction)

Préenregistrement, surveillance et rapports des visiteurs dans votre bâtiment numérique intelligent.
