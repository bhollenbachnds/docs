---
- title: Première connexion avec l'authentification multi-facteurs
---

Ce manuel explique le processus de configuration initiale de l'authentification multi-facteurs (MFA) dans PADS4. Lorsque la MFA est disponible ou requise pour un compte utilisateur, PADS4 affiche l'écran **Configurer l'authentification à deux facteurs** après la connexion afin que l'utilisateur puisse enregistrer une application d'authentification

Cet écran apparaît la première fois qu'un utilisateur se connecte à PADS4 et que la configuration de la MFA est nécessaire. Il permet à l'utilisateur de connecter une application d'authentification en scannant le code QR ou, comme alternative, en saisissant le code manuel affiché à l'écran.

## **Comment effectuer la configuration**

1. Se connecter à PADS4
2. Lorsque l'écran **Configurer l'authentification à deux facteurs** apparaît, ouvrez l'application d'authentification sur l'appareil mobile
3. Scannez le code QR affiché à l'écran avec l'application d'authentification .\\
   <Frame>
     ![QR CODE](/images/CTRLCenter/MFA/QR-CODE.png)
   </Frame>
4. Si le scan n'est pas possible, saisissez le **Code manuel** dans l'application d'authentification à la place.
5. Une fois l'application d'authentification configurée, cliquez sur **Continuer vers la vérification** pour passer à l'étape suivante

## **Option Ignorer**

Un bouton **Ignorer** peut être visible sur cet écran. Cette option n'est valable que jusqu'à ce que la MFA soit appliquée, il est donc recommandé de terminer la configuration dès que possible et de ne pas attendre le dernier moment.

Si la MFA a déjà été appliquée pour l'environnement ou l'utilisateur, il se peut que l'option d'ignorer la configuration ne soit plus autorisée. Dans ce cas, l'utilisateur doit terminer le processus d'enregistrement par code QR ou par code manuel avant de pouvoir continuer à accéder au système
