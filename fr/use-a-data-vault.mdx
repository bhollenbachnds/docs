---

Les Data Vaults stockent du contenu pouvant être partagé entre les présentations et maintenu synchronisé depuis une source unique. Ce guide vous montre comment associer un Data Vault existant à une présentation dans PADS4 Designer.

<Info>
  Les Data Vaults sont créés et gérés dans PADS4 System Management. Pour créer ou configurer un Data Vault, référez-vous au manuel PADS4 System Management.
</Info>

## Associer un Data Vault à une présentation

1. **Ouvrir le sélecteur de Data Vault.** Dans la **Elements** barre, sélectionnez l'icône **Data Vault**.
   <Frame>
     ![Data Vault icon in the Elements bar](images/DigitalSignage/Designer/IconDataVault.png)
   </Frame>
2. **Parcourir les Data Vaults disponibles.** Designer affiche les Data Vaults auxquels vous avez accès.
   <Frame>
     <img src="/images/DigitalSignage/Designer/OverviewDataVault.png" alt="Overview of available Data Vaults" title="Overview Data Vault" className="mr-auto" style={{ width: "58%" }} />
   </Frame>
3. **Ajouter le Data Vault à votre présentation.** Ouvrez le menu contextuel à côté du Data Vault que vous souhaitez utiliser, puis ajoutez-le à votre présentation. Son contenu devient immédiatement disponible et reste synchronisé avec la source définie dans System Management.
   <Frame>
     <img src="/images/DigitalSignage/Designer/DataVaultAddToPresentation.png" alt="Adding a Data Vault to a presentation" title="Data Vault Add To Presentation" style={{ width:"64%" }} />
   </Frame>
4. **Vérifier que le Data Vault est associé.** Ouvrez le **Properties** panneau et sélectionnez **Data Vaults** pour voir tous les Data Vaults associés à la présentation actuelle.
   <Frame>
     <img src="/images/DigitalSignage/Designer/DataVaultAdded.png" alt="Data Vault listed in the Properties pane" title="Data Vault Added" className="mr-auto" style={{ width:"62%" }} />
   </Frame>

<Tip>
  Vous ne trouvez pas le Data Vault dont vous avez besoin ? Vérifiez qu'il a été publié dans PADS4 System Management et que votre compte dispose des droits d'accès requis.
</Tip>
