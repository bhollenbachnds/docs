---
- title: Introduction à PADS4 System Management
- description: Présentation de PADS4 System Management, l'application Windows
    utilisée pour configurer et administrer les serveurs, les utilisateurs et
    les composants d'une installation PADS4.
---

Bienvenue dans PADS4, l'un des logiciels d'affichage dynamique les plus avancés et les plus complets disponibles aujourd'hui.

Ce manuel PADS4 System Management décrit les possibilités de PADS4 System Management, l'application permettant de configurer et d'administrer votre installation PADS4.
