Depuis la **destinations window,** vous pouvez définir de nouveaux viewer groups, viewers, zone viewers, HTML viewers et desktop viewers.

Un groupe de visionneuses est une unité logique qui se compose d'autres destinations. Les membres d'un groupe peuvent être des visionneuses ou d'autres groupes. Les groupes peuvent être imbriqués sans aucune limitation et les visionneuses ou groupes peuvent être membres d'un nombre illimité de groupes.

Les groupes sont généralement utilisés pour organiser vos destinations dans une structure logique et pour les contrôler par groupe plutôt que par visionneuse individuelle.

 

#### Pour ajouter un nouveau groupe de visionneuses

1. Sélectionnez **Destinations** dans le volet de navigation pour activer la fenêtre des destinations.
2. Cliquez sur **New Group** dans le menu **Home**.

   La fenêtre New Viewer Group apparaîtra.
3. Spécifiez les paramètres **Identification** dans l'onglet **General**.

   ![EN System Management Managing Destinations 01 Pn](/images/digital-signage/windows-applications/system-management/managing-destinations/EN-SystemManagement-Managing-Destinations-01.png)

   **Name**
   Un nom unique pour identifier le groupe de visionneuses.

   **Description**
   Une description pour donner des informations supplémentaires sur le groupe de visionneuses.
4. Spécifiez dans l'onglet **Member Of** à quel(s) groupe(s) de visionneuses ce groupe de visionneuses appartient en cochant les cases correspondantes.

   ![EN System Management Managing Destinations 02 Pn](/images/digital-signage/windows-applications/system-management/managing-destinations/EN-SystemManagement-Managing-Destinations-02.png)

   Utilisez éventuellement les options d'affichage, de tri et de filtrage dans la barre d'outils pour localiser vos destinations.
5. Spécifiez dans l'onglet **Members**quelles visionneuses ou quels groupes de visionneuses appartiennent à ce groupe de visionneuses en cochant les cases correspondantes.
   Lorsque vous sélectionnez un groupe de visionneuses, toutes les visionneuses qui font partie de ce groupe seront automatiquement incluses.

   ![EN System Management Managing Destinations 03 Pn](/images/digital-signage/windows-applications/system-management/managing-destinations/EN-SystemManagement-Managing-Destinations-03.png)

   Utilisez éventuellement les options d'affichage, de tri et de filtrage dans la barre d'outils pour localiser vos destinations.
6. Déterminez dans l'onglet **Permissions** quelles autorisations doivent s'appliquer à ce groupe de visionneuses.

   ![EN System Management Managing Destinations 04 Pn](/images/digital-signage/windows-applications/system-management/managing-destinations/EN-SystemManagement-Managing-Destinations-04.png)

   <Note>
     1. Avant de pouvoir attribuer une autorisation, vous devez d'abord la définir comme décrit dans **managing permissions**.
        Dans le cas des groupes de visionneuses, seules les autorisations de contenu peuvent être attribuées. Les autorisations de contenu sont utilisées pour filtrer le contenu téléchargé par une visionneuse.
   </Note>
7. Cliquez sur **OK** pour terminer la création du nouveau groupe de visionneuses.
