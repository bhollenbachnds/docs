Depuis la **fenêtre destinations,** vous pouvez définir de nouveaux groupes de viewers, viewers, zone viewers, HTML viewers et desktop viewers.

Un groupe de viewers est une unité logique qui se compose d'autres destinations. Les membres d'un groupe peuvent être des viewers ou d'autres groupes. Les groupes peuvent être imbriqués sans aucune limitation et les viewers ou les groupes peuvent être membres d'un nombre illimité de groupes.

Les groupes sont généralement utilisés pour organiser vos destinations dans une structure logique et pour les contrôler par groupe plutôt que par viewer individuel.

 

#### Pour ajouter un nouveau groupe de viewers

1. Sélectionnez **Destinations** dans le volet de navigation pour activer la fenêtre destinations.
2. Cliquez sur **New Group** dans le menu **Home**.

   La fenêtre New Viewer Group apparaîtra.
3. Spécifiez les paramètres **Identification** dans l'onglet **General**.

   ![EN System Management Managing Destinations 01 Pn](/images/digital-signage/windows-applications/system-management/managing-destinations/EN-SystemManagement-Managing-Destinations-01.png)

   **Name**
   Un nom unique pour identifier le groupe de viewers.

   **Description**
   Une chaîne descriptive pour donner des informations supplémentaires sur le groupe de viewers.
4. Spécifiez dans l'onglet **Member Of** à quel(s) groupe(s) de viewers ce groupe de viewers appartient en cochant les cases correspondantes.

   ![EN System Management Managing Destinations 02 Pn](/images/digital-signage/windows-applications/system-management/managing-destinations/EN-SystemManagement-Managing-Destinations-02.png)

   Utilisez éventuellement les options d'affichage, de tri et de filtrage dans la barre d'outils pour localiser vos destinations.
5. Spécifiez dans l'onglet **Members** quels viewers ou groupes de viewers appartiennent à ce groupe de viewers en cochant les cases correspondantes.
   Lorsque vous sélectionnez un groupe de viewers, tous les viewers qui font partie de ce groupe seront automatiquement inclus.

   ![EN System Management Managing Destinations 03 Pn](/images/digital-signage/windows-applications/system-management/managing-destinations/EN-SystemManagement-Managing-Destinations-03.png)

   Utilisez éventuellement les options d'affichage, de tri et de filtrage dans la barre d'outils pour localiser vos destinations.
6. Déterminez dans l'onglet **Permissions** quelles autorisations doivent s'appliquer à ce groupe de viewers.

   ![EN System Management Managing Destinations 04 Pn](/images/digital-signage/windows-applications/system-management/managing-destinations/EN-SystemManagement-Managing-Destinations-04.png)

   <Note>
     1. Avant de pouvoir attribuer une autorisation, vous devez d'abord la définir comme décrit dans **managing permissions**.
        Dans le cas des groupes de viewers, seules les autorisations de contenu peuvent être attribuées. Les autorisations de contenu sont utilisées pour filtrer le contenu téléchargé par un viewer.
   </Note>
7. Cliquez sur **OK** pour terminer la création du nouveau groupe de viewers.
