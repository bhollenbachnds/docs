Dans la fenêtre des destinations, vous pouvez voir toutes les destinations qui ont été définies pour votre installation PADS4. Il peut s'agir de viewers, de zone viewers, de HTML viewers ou de desktop viewers. En option, les viewers peuvent être organisés en groupes de viewers pour planifier vos présentations PADS4 par groupe. Les destinations peuvent être ajoutées, modifiées ou supprimées.

 

#### Pour afficher les destinations de votre système PADS4

1. Allez dans le volet de navigation dans la fenêtre principale.

   ![EN System Management Navigationpane Pn](/images/digital-signage/windows-applications/system-management/main-window/EN-SystemManagement-Navigationpane.png)
2. Cliquez sur Destinations dans la liste des catégories d'informations disponibles.

Après cela, le système récupérera automatiquement les informations demandées depuis le PADS4 Server connecté et affichera les résultats dans la fenêtre des destinations. Si votre installation PADS4 comprend des sous-serveurs, la fenêtre des destinations affichera toujours les résultats globaux de toute votre installation PADS4, quel que soit le serveur que vous avez sélectionné.

 

#### Fenêtre des destinations

La fenêtre des destinations affiche tous les viewers et groupes de viewers de votre installation PADS4. Les destinations répertoriées peuvent avoir été ajoutées manuellement au système ou automatiquement en connectant un nouveau PADS4 Viewer à votre PADS4 Server, en fonction de vos **paramètres de serveur**.

**Filtre** \
Dans la barre de ruban, il y a un groupe Filtre qui vous permet de filtrer par Type de Viewer ou de filtrer en fonction d'une valeur de sous-chaîne saisie par l'utilisateur, recherchée dans les colonnes Nom du Viewer et Description.

![EN System Management Destinationswindow Pn](/images/digital-signage/windows-applications/system-management/managing-destinations/EN-SystemManagement-Destinationswindow.png)

**Nom**
Le nom unique qui identifie la destination dans le système PADS4. Par défaut, les PADS4 Viewers nouvellement connectés seront automatiquement ajoutés au système en utilisant le nom de l'ordinateur comme nom de viewer.

**Description**
La chaîne descriptive qui peut avoir été spécifiée pour donner des informations supplémentaires sur le viewer.

**Type**
Le type défini par le système de votre destination. Les types possibles sont :

| Type de viewer | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| :------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Group          | Un groupe est une unité logique qui se compose d'autres destinations. Les membres d'un groupe peuvent être des viewers ou d'autres groupes. Les groupes peuvent être imbriqués sans aucune limitation et les viewers ou groupes peuvent être membres d'un nombre illimité de groupes. Les groupes sont généralement utilisés pour organiser vos destinations dans une structure logique et pour les contrôler par groupe plutôt que par viewer individuel. |
| Viewer         | Un Viewer standard représente une instance de l'application PADS4 Viewer standard pour la lecture ciblée de contenu planifié sur un dispositif d'affichage. Cette application PADS4 Viewer est basée sur Microsoft .NET Framework et fonctionne généralement sur une plateforme Microsoft Windows.                                                                                                                                                         |
| Zone Viewer    | Un Zone Viewer est un Viewer spécial. Il peut être accessible comme un élément dans PADS4 Designer pour visualiser le contenu planifié pour ce Viewer particulier comme une zone dans votre présentation.                                                                                                                                                                                                                                                  |
| HTML Viewer    | Un HTML Viewer représente une instance de l'application PADS4 HTML5 Viewer pour la lecture ciblée de contenu planifié dans un environnement de navigateur. L'application PADS4 HTML5 Viewer est conçue comme une solution indépendante de la plateforme.                                                                                                                                                                                                   |
| Desktop Viewer | Un Desktop Viewer représente une instance de l'application PADS4 Desktop Viewer pour la lecture ciblée de contenu planifié sur un PC utilisateur final. L'application PADS4 Desktop Viewer peut être utilisée comme économiseur d'écran, pour recevoir et visualiser manuellement des messages PADS4 ciblés ou pour l'affichage automatique de messages popup comme alerte.                                                                                |

 

**Licence**\
La licence qui a été attribuée à ce viewer. Les viewers ne peuvent être utilisés que lorsqu'ils ont une licence valide. Les licences peuvent être attribuées manuellement en **modifiant une destination** ou comme décrit dans la gestion des licences. Les groupes de viewers ne nécessitent jamais de licence.

<Note>
  Pour les PADS4 Desktop Viewers, vous ne pouvez pas attribuer de licence. Les PADS4 Desktop Viewers fonctionnent avec des licences simultanées, ce qui signifie que l'application réclamera une licence lorsqu'elle se connecte et libérera la licence à nouveau lorsqu'elle se déconnecte.
</Note>

**Type de licence** \
Le type de licence viewer qui a été attribué à ce viewer. Le type de licence attribué détermine le contenu pris en charge par l'application PADS4 Viewer. Les types possibles sont : BASIC (fonctionnalité très limitée), XPRESS (fonctionnalité limitée) et XPERT (aucune restriction).

**Statut de licence**\
Le statut de votre licence attribuée. Les valeurs possibles sont :

| **Statut**        | **Description**                                                                                                                                                                                                                                                                                                                                      |
| :---------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| \[Vide]           | La destination listée ne nécessite pas qu'une licence soit attribuée.                                                                                                                                                                                                                                                                                |
| Pas de licence    | Le viewer n'a pas de licence attribuée. En conséquence, l'application viewer ne fonctionnera pas correctement. Vous pouvez attribuer manuellement une licence en **modifiant la destination** ou comme décrit dans la gestion des licences.                                                                                                          |
| Licence attribuée | Le viewer a une licence attribuée, mais la licence n'a pas encore été activée. Lorsque l'application viewer se connecte avec succès au PADS4 Server, elle activera automatiquement la licence attribuée et liera ainsi la licence au matériel du PC Viewer. Lorsque le matériel du PC Viewer change, vous devez réattribuer manuellement la licence. |
| Licences activées | Le viewer a une licence et la licence attribuée a été activée avec succès. Tant que le matériel du PC Viewer ne changera pas, l'application viewer fonctionnera avec la licence attribuée.                                                                                                                                                           |
