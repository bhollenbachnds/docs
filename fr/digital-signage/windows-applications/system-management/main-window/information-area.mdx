Après avoir sélectionné une catégorie d'informations dans le volet de navigation, les informations seront récupérées et les résultats seront affichés dans une fenêtre dans la **information area**. La façon dont les résultats sont affichés dans la fenêtre d'information dépend de la catégorie choisie.

![EN System Management User Groupswindow Pn](/images/digital-signage/windows-applications/system-management/main-window/EN-SystemManagement-UserGroupswindow.png)

Pour la plupart des catégories, un certain nombre de fonctions contextuelles deviendront disponibles dans la barre de ruban. Les mêmes fonctions contextuelles peuvent également être appelées en faisant un clic droit sur la fenêtre d'information.

Par défaut, l'application PADS4 utilise une fenêtre d'information par catégorie d'information. Cela signifie que lorsque vous passez à une autre catégorie d'information, une autre fenêtre sera ouverte pour afficher les résultats. Si vous passez à une catégorie d'information qui a déjà une fenêtre ouverte, cette fenêtre deviendra active et sera réutilisée pour afficher les résultats. Lorsque vous travaillez avec plusieurs serveurs, le système utilisera une fenêtre d'information par catégorie d'information par connexion de serveur. En option, vous pouvez ouvrir manuellement des fenêtres d'information supplémentaires comme décrit dans **Working with multiple windows**.

 

#### Actualisation des données

Les informations dans la fenêtre d'information reflètent toujours la situation actuelle et sont automatiquement actualisées. Pour cette raison, il n'existe pas de fonction permettant d'actualiser manuellement les informations.
