En haut de la fenêtre principale, vous trouverez une barre de ruban - un ensemble de barres d'outils, placées sur plusieurs onglets - qui donne un accès rapide aux fonctions clés de l'application.

 

#### **Aperçu de la barre de ruban PADS4 System Management**

La barre de ruban PADS4 System Management a été divisée en 5 sections appelées groupes d'onglets. Vous pouvez y accéder en cliquant sur les en-têtes respectifs. Des informations détaillées seront fournies dans les sujets à suivre.

1. File

   ![EN System Management Ribbon Bar 00 Pn](/images/digital-signage/windows-applications/system-management/main-window/EN-SystemManagement-RibbonBar-00.png)

   Le groupe d'onglets File contient les fonctionnalités les plus basiques pour PADS4 System Management, y compris la gestion des connexions.
2. Home

   ![EN System Management Ribbon Bar 01 Pn](/images/digital-signage/windows-applications/system-management/main-window/EN-SystemManagement-RibbonBar-01.png)

   Le groupe d'onglets Home est le conteneur des fonctions clés de l'application. Celles-ci incluent la gestion des connexions au PADS4 Server et les fonctions les plus courantes pour la catégorie d'information choisie.
3. View

   ![EN System Management Ribbon Bar 02 Pn](/images/digital-signage/windows-applications/system-management/main-window/EN-SystemManagement-RibbonBar-02.png)

   Le groupe d'onglets View vous permet de déterminer la façon dont vous souhaitez afficher vos panneaux d'information.
4. Tools

   ![EN System Management Ribbonbar 05 Pn](/images/digital-signage/windows-applications/system-management/main-window/EN-SystemManagement-Ribbonbar-05.png)

   Le groupe d'onglets Tools donne accès aux options de l'application elle-même ainsi qu'aux fonctions à l'échelle du système.
5. Help

   ![EN System Management Ribbon Bar 04 Pn](/images/digital-signage/windows-applications/system-management/main-window/EN-SystemManagement-RibbonBar-04.png)

   Le groupe d'onglets Help donnera accès aux informations sur l'application elle-même, comme les instructions, la version et les détails de licence.
