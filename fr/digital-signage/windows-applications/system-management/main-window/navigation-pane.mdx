Après vous être connecté avec succès à votre PADS4 Server, vous pouvez utiliser le volet de navigation pour sélectionner une catégorie d'informations et accéder aux fonctions de gestion de cette catégorie d'informations.

 

#### Pour sélectionner une catégorie d'informations

1. Dans le volet de Navigation, cliquez sur une **Category** dans la liste des catégories d'informations disponibles.

   ![EN System Management Navigationpane Pn](/images/digital-signage/windows-applications/system-management/main-window/EN-SystemManagement-Navigationpane.png)

Après cela, le système récupérera automatiquement les informations demandées depuis le PADS4 Server connecté et affichera les résultats dans la zone d'information de la fenêtre principale.
