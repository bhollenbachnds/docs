Après vous être connecté avec succès à votre serveur PADS4, vous pouvez utiliser le volet de navigation pour sélectionner une catégorie d'information et accéder aux fonctions de gestion de cette catégorie d'information.

 

#### Pour sélectionner une catégorie d'information

1. Dans le volet de navigation, cliquez sur une **Catégorie** dans la liste des catégories d'information disponibles.

   ![EN System Management Navigationpane Pn](/images/digital-signage/windows-applications/system-management/main-window/EN-SystemManagement-Navigationpane.png)

Après cela, le système récupérera automatiquement les informations demandées depuis le serveur PADS4 connecté et affichera les résultats dans la \[zone d'information]\([https://pads4documentation.blob.core.windows.net/systemmanagement/Content/Managing](https://pads4documentation.blob.core.windows.net/systemmanagement/Content/Managing) Users/\_IntroPage.htm) de la fenêtre principale.
