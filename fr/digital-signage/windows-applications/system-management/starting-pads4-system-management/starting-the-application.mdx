PADS4 System Management est l'application utilisateur final pour configurer votre système PADS4.

 

#### **Pour démarrer l'application PADS4 System Management**

1. Cliquez sur le bouton **Start** dans la barre des tâches Windows.
2. Sélectionnez **(All) Programs** puis **PADS4**.
3. Sélectionnez **System Management**.

ou

1. Dans l'Explorateur Windows, accédez au dossier contenant les fichiers du programme PADS4 System Management.
2. Double-cliquez sur l'élément **PADSSystemManagement.exe**.

Après cela, l'application sera lancée.

<Tip>
  Si tous les paramètres de configuration de l'application sont corrects, le système se connectera automatiquement au logiciel PADS4 Server. Si vous démarrez l'application pour la toute première fois ou lorsqu'un ou plusieurs des paramètres de configuration sont incorrects (le système n'est pas en mesure de se connecter au logiciel PADS4 Server), la fenêtre de gestion de connexion PADS4 s'affichera automatiquement. Cela vous permet de spécifier les bons paramètres de configuration afin de vous connecter au logiciel PADS4 Server et ainsi démarrer l'application.
</Tip>
