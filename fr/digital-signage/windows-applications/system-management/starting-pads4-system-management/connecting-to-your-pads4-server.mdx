Lorsque PADS4 System Management a été démarré, il recherche les connexions PADS4 Server disponibles. Selon la façon dont vous avez configuré vos propriétés de connexion, l'application se connectera et ouvrira une session sur votre PADS4 Server automatiquement, ou vous devrez le faire manuellement.

<Tip>
  Vous pouvez également travailler avec plusieurs connexions PADS4 Server. Pour plus d'informations à ce sujet, voir [Connexion à plusieurs serveurs](/fr/digital-signage/windows-applications/agent/advanced-features/connecting-to-multiple-servers).
</Tip>

 

#### **Pour vous connecter à votre PADS4 Server pour la première fois**

Lorsque vous démarrez PADS4 System Management pour la toute première fois et que vous n'avez encore défini aucune propriété de connexion, vous obtiendrez le message suivant :

1. Cliquez sur **OK** pour continuer.
2. Dans la fenêtre PADS4 Server Connection (qui s'ouvre automatiquement), cliquez sur **Add...** pour définir une nouvelle connexion.
3. Spécifiez vos propriétés de connexion et cliquez sur **OK** pour les enregistrer. Pour plus d'informations, voir [Gestion de vos connexions](/fr/digital-signage/windows-applications/agent/starting-PADS4-Agent/managing-your-connections).
4. Cliquez sur **Connect** pour établir la connexion.

   ![EN System Management Connectionproperties 01 Pn](/images/digital-signage/windows-applications/system-management/starting-pads4-system-management/EN-SystemManagement-Connectionproperties-01.png)

   Même si vous avez configuré votre connexion pour vous connecter automatiquement, vous devrez vous connecter de cette façon pour la première fois.
5. Si vous devez ouvrir une session, entrez votre **Nom d'utilisateur** et **Mot de passe** et cliquez sur **Log On**.

   ![EN System Management Logon 01 Pn](/images/digital-signage/windows-applications/system-management/starting-pads4-system-management/EN-SystemManagement-Logon-01.png)

 

#### **Connexion à votre PADS4 Server**

Si vous avez spécifié dans vos propriétés de connexion de vous connecter automatiquement à votre PADS4 Server, PADS4 System Management tentera automatiquement d'établir une connexion une fois démarré. Dans toutes les autres situations, vous devrez établir une connexion manuellement :

1. Cliquez sur la partie inférieure du bouton **Connect** dans l'onglet **Home** de la barre de ruban.

   ![EN Agent Connecting 02 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Connecting-02.png)
2. Cliquez sur la connexion listée pour établir une connexion.

   ![EN Agent Connecting 03 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Connecting-03.png)
3. Si vous devez ouvrir une session, entrez votre **Nom d'utilisateur** et **Mot de passe** et cliquez sur **Log On**.

 

#### **Déconnexion de votre PADS4 Server**

Pour vous déconnecter de votre PADS4 Server, cliquez sur **Disconnect** dans l'onglet **Home** de la barre de ruban.

 

#### **Reconnexion à votre PADS4 Server**

Si vous avez été déconnecté de votre PADS4 Server, vous souhaiterez peut-être vous reconnecter. Pour vous reconnecter :

1. Cliquez sur la partie supérieure du bouton Connect dans l'onglet Home de la barre de ruban.

   ![EN Agent Connecting 01 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Connecting-01.png)

   Cela forcera l'application à se reconnecter automatiquement si la connexion a été configurée pour se connecter automatiquement.

Ou

1. Suivez les étapes de Connexion à votre PADS4 Server décrites précédemment.
