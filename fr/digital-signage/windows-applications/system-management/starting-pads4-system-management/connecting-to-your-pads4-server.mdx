Lorsque PADS4 System Management a été démarré, il recherchera les connexions PADS4 Server disponibles. Selon la façon dont vous avez configuré vos propriétés de connexion, l'application se connectera et s'identifiera automatiquement à votre PADS4 Server, ou vous devrez le faire manuellement.

<Tip>
  Vous pouvez éventuellement travailler avec plusieurs connexions PADS4 Server. Pour plus d'informations à ce sujet, voir [Connecting to multiple servers](/fr/digital-signage/windows-applications/agent/advanced-features/connecting-to-multiple-servers).
</Tip>

 

#### **Pour se connecter à votre PADS4 Server pour la première fois**

Lorsque c'est la toute première fois que vous démarrez PADS4 System Management et que vous n'avez pas encore défini de propriétés de connexion, vous obtiendrez le message suivant :

1. Cliquez sur **OK** pour continuer.
2. Dans la fenêtre PADS4 Server Connection (qui s'ouvre automatiquement), cliquez sur **Add...** pour définir une nouvelle connexion.
3. Spécifiez vos propriétés de connexion et cliquez sur **OK** pour les enregistrer. Pour plus d'informations, voir [Managing your connections](/fr/digital-signage/windows-applications/agent/starting-PADS4-Agent/managing-your-connections).
4. Cliquez sur **Connect** pour établir la connexion.

   ![EN System Management Connectionproperties 01 Pn](/images/digital-signage/windows-applications/system-management/starting-pads4-system-management/EN-SystemManagement-Connectionproperties-01.png)

   Même si vous avez configuré votre connexion pour se connecter automatiquement, vous devrez vous connecter de cette façon la première fois.
5. Si vous devez vous identifier, saisissez votre **User Name** et **Password** et cliquez sur **Log On**.

   ![EN System Management Logon 01 Pn](/images/digital-signage/windows-applications/system-management/starting-pads4-system-management/EN-SystemManagement-Logon-01.png)

 

#### **Connexion à votre PADS4 Server**

Si vous avez spécifié dans vos propriétés de connexion de vous connecter automatiquement à votre PADS4 Server, PADS4 System Management essaiera automatiquement d'établir une connexion une fois qu'il aura été démarré. Dans toutes les autres situations, vous devrez établir une connexion manuellement :

1. Cliquez sur la partie inférieure du bouton **Connect** dans l'onglet **Home** de la barre de ruban.

   ![EN Agent Connecting 02 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Connecting-02.png)
2. Cliquez sur la connexion listée pour établir une connexion.

   ![EN Agent Connecting 03 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Connecting-03.png)
3. Si vous devez vous connecter, saisissez votre **User Name** et **Password** et cliquez sur **Log On**.

 

#### **Déconnexion de votre PADS4 Server**

Pour vous déconnecter de votre PADS4 Server, cliquez sur **Disconnect** dans l'onglet **Home** de la barre de ruban.

 

#### **Reconnexion à votre PADS4 Server**

Si vous avez été déconnecté de votre PADS4 Server, vous voudrez peut-être vous reconnecter. Pour vous reconnecter :

1. Cliquez sur la partie supérieure du bouton Connect dans l'onglet Home de la barre de ruban.

   ![EN Agent Connecting 01 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Connecting-01.png)

   Cela forcera l'application à reconnecter automatiquement votre connexion si elle a été configurée pour se connecter automatiquement.

Ou

1. Suivez les étapes pour vous connecter à votre PADS4 Server comme décrit précédemment.
