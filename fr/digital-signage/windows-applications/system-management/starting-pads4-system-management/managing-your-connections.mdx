# **Gérer vos connexions**

Pour pouvoir vous connecter à votre serveur PADS4, vous devez d'abord configurer votre connexion. Une connexion décrit généralement comment se connecter et se connecter à votre serveur PADS4. Les connexions peuvent être gérées dans la fenêtre Connexions au serveur PADS4. Dans la plupart des situations, vous n'aurez qu'un seul serveur PADS4, mais vous pouvez éventuellement [vous connecter à plusieurs serveurs PADS4](/fr/digital-signage/windows-applications/agent/advanced-features/connecting-to-multiple-servers).

Les connexions que vous définissez dans la fenêtre Connexions au serveur PADS4 sont partagées par toutes les applications PADS4 installées sur votre ordinateur.

 

#### **Pour afficher vos connexions**

1. Cliquez sur **Gérer** dans la **Connexions** section de l'onglet **Accueil** dans la barre de ruban.

   ![EN Agent Manageconnections 01 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Manageconnections-01.png)

   Cela ouvre automatiquement la fenêtre Connexions au serveur PADS4, où vous pouvez afficher vos connexions.

 

#### **Fenêtre Connexions au serveur PADS**

La fenêtre Connexions au serveur PADS4 affiche la liste de toutes les connexions définies sur votre ordinateur avec leur statut actuel.

![EN Agent Manageconnections 02 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Manageconnections-02.png)

La fenêtre Connexions peut être utilisée pour se connecter manuellement à un serveur PADS4 ou s'en déconnecter. Si une connexion n'est pas encore établie, vous pouvez la sélectionner et cliquer sur **Connecter** pour établir manuellement une connexion. Pour vous déconnecter, cliquez simplement sur **Déconnecter**.

<Tip>
  Travailler avec plusieurs connexions et se connecter/déconnecter manuellement peut être particulièrement utile lorsque vous devez gérer plusieurs installations PADS4. Un exemple est une agence de design développant du contenu pour plusieurs clients utilisant PADS4.
</Tip>

 

#### **Pour ajouter une nouvelle connexion**

1. Cliquez sur **Ajouter...**.

   ![EN Agent Manageconnections Add Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Manageconnections-add.png)
2. Remplissez les paramètres dans l'onglet **Connexion**.

   ![EN Agent Manageconnections Details Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Manageconnections-details.png)

   **Nom de la connexion**\
   Un nom unique qui sera utilisé pour identifier votre connexion. Le nom que vous saisissez ici sera utilisé partout dans l'interface graphique pour indiquer votre connexion.

   **Démarrage de l'application**\
   Décidez si vous souhaitez que votre système se connecte automatiquement à votre serveur PADS4 au démarrage de votre application PADS4.

   **Connexion au serveur**\
   \
   Spécifiez l'emplacement de votre serveur PADS4. Les options possibles sont :

   <Note>
     Un serveur PADS4 ne peut être détecté automatiquement que si le processus du serveur PADS4 est en cours d'exécution et que le port UDP numéro 3702 n'est pas bloqué par un pare-feu sur le PC serveur ou ailleurs dans le réseau. La détection automatique ne fonctionne que dans un réseau local (LAN).
   </Note>

   | **Option de connexion** | **Description**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
   | :---------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
   | Automatique             | Votre système PADS4 détectera automatiquement l'emplacement de votre serveur PADS4. Si vous n'avez qu'un seul serveur PADS4 dans votre réseau, c'est peut-être la meilleure option à choisir. Lorsque votre serveur PADS4 sera déplacé vers un autre emplacement dans votre réseau ou obtiendra une adresse réseau différente, le système effectuera automatiquement les ajustements nécessaires. Si vous avez plusieurs serveurs PADS4 (comme un serveur de sauvegarde) en cours d'exécution dans votre réseau, vous ne pouvez pas utiliser cette option, car il n'est pas garanti que le système détecte le bon serveur.                                                                                                                                                |
   | Cet ordinateur          | Votre serveur PADS4 est censé fonctionner sur le même ordinateur que votre application. Si vous avez installé à la fois le serveur PADS4 et votre application PADS4 sur le même ordinateur, c'est l'option préférée. Cela garantit que le système continuera à fonctionner même si le nom de votre ordinateur change. Cela maintient également la communication en local, ce qui est beaucoup plus rapide. Enfin, cela évite que votre système passe par le réseau pour accéder à votre serveur PADS4 et soit potentiellement bloqué par un pare-feu.                                                                                                                                                                                                                     |
   | Autre                   | Utilisez cette option si aucune des options ci-dessus ne s'applique. Avec cette option, vous pouvez spécifier manuellement l'adresse réseau de votre **PADS4 Server**. Ici, vous pouvez saisir un nom d'ordinateur, une adresse IP ou un nom DNS. Vous pouvez également cliquer sur **Rechercher...** pour rechercher automatiquement les PADS4 Servers existants sur votre réseau. En cliquant sur **Afficher les options avancées** vous avez la possibilité de spécifier un **Numéro de port** alternatif pour communiquer avec votre PADS4 Server. Veuillez noter que le numéro de port que vous saisissez ici doit correspondre au numéro de port qui a été spécifié dans votre PADS4 Server via les paramètres du serveur de l'application PADS4 System Management. |

   **Paramètres Http**\
   Lorsqu'il est défini sur automatique, PADS4 identifie la connexion au serveur et configure les paramètres HTTP en fonction des paramètres de connexion par défaut. Cela peut être ajusté en le définissant sur **manuel** et en spécifiant un chemin d'hôte et un port.\
   \
   PADS4 utilise 2 protocoles réseau. Les connexions liées au Web sont utilisées dans le PADS4 Agent (Proof of Play et journal d'audit) et le PADS4 Desktop Viewer (synchronisation). Dans les situations où la connexion au serveur est établie via un routeur ou un proxy utilisant la redirection de port, il est possible que vous deviez spécifier un chemin d'hôte/port différent pour vous connecter au PADS4 Server.
3. Spécifiez les paramètres de connexion dans l'onglet **Identifiants**.

   ![EN Agent Logon Properties Creds Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-LogonProperties-creds.png)

   **Identifiants**\
   Ici, vous pouvez spécifier le **Nom d'utilisateur** et/ou le **Mot de passe** que vous souhaitez utiliser pour vous connecter à votre PADS4 Server lorsqu'une connexion est établie. Si vous ne spécifiez qu'un nom d'utilisateur, l'utilisateur sera invité à saisir le mot de passe lors de la connexion. Si vous choisissez **Connexion automatique** alors le système essaiera de se connecter automatiquement à votre PADS4 Server lors de la connexion. Si la connexion échoue, l'utilisateur sera invité à saisir le nom d'utilisateur et le mot de passe corrects.

   <Tip>
     1. Souvent, l'option **Connexion automatique** est utilisée en combinaison avec l'option **Démarrer automatiquement cette connexion** dans l'onglet **Connexion**. De cette façon, votre application se connectera et se connectera automatiquement à votre PADS4 Server au démarrage de votre application PADS4.
   </Tip>

   **Domaine**\
   Si votre PC PADS4 Server héberge plusieurs PADS4 Servers, vous avez ici la possibilité de spécifier manuellement le **Domaine** qui correspond à votre système PADS4. Une autre situation où l'utilisation des domaines peut être requise est lorsque vous avez plusieurs connexions à différents PADS4 Servers et que vous souhaitez garder ces connexions séparées sur votre PC client. Dans toutes les autres situations, vous devez choisir l'option **Automatique**.
4. Ajustez éventuellement les paramètres de synchronisation dans l'onglet **Synchronisation**.

   ![Agent Manageconnection Sync Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/Agent-manageconnection-sync.png)

   \
   **Activation de la synchronisation de la bibliothèque de contenu**\
   Spécifiez la méthode à utiliser pour la synchronisation.\
   **Automatique** synchronisera lorsque des modifications sont détectées, **Manuel** attendra que l'utilisateur clique sur un bouton de synchronisation immédiate et **jamais** ne synchronisera pas. Lorsque les trois options sont désactivées, il n'est pas permis d'utiliser la synchronisation. Par exemple, lorsque le serveur et le Designer sont sur le même PC, la synchronisation n'est pas nécessaire, car ils utilisent le même environnement.\
   \
   **Dossier de stockage de la bibliothèque de contenu PADS4**\
   Spécifiez où le contenu doit être stocké si l'emplacement généré automatiquement n'est pas possible.

   * **Domaine**\
     Si votre PC PADS4 Server héberge plusieurs PADS4 Servers, vous avez ici la possibilité de spécifier manuellement le **Domaine** qui correspond à votre système PADS4. Une autre situation où l'utilisation des domaines peut être requise est lorsque vous avez plusieurs connexions à différents PADS4 Servers et que vous souhaitez garder ces connexions séparées sur votre PC client. Dans toutes les autres situations, vous devez choisir l'option **Automatique**.
   * Cliquez sur **OK** pour enregistrer vos paramètres.

 

#### **Pour modifier une connexion**

Avant de pouvoir commencer à modifier une connexion, vous devez d'abord vous assurer que la connexion est déconnectée. Il n'est pas possible d'apporter des modifications à une connexion actuellement en cours d'utilisation.

1. Cliquez sur la connexion dans la fenêtre des connexions PADS4 Server pour la sélectionner.
2. Cliquez sur **Modifier...** pour ouvrir la fenêtre des propriétés de connexion.

   ![EN Agent Manageconnections Edit Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Manageconnections-edit.png)
3. Effectuez les ajustements nécessaires (voir la description ci-dessus).
4. Cliquez sur **OK** pour enregistrer vos paramètres.

 

#### **Pour supprimer une connexion**

1. Cliquez sur la connexion dans la fenêtre des connexions PADS4 Server pour la sélectionner.
2. Cliquez sur **Supprimer** pour supprimer la connexion.

Après cela, votre connexion sera définitivement supprimée pour toutes les applications PADS4 sur votre ordinateur.
