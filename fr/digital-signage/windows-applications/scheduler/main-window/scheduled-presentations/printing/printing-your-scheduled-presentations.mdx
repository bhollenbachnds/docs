PADS4 Scheduler offre une fonctionnalité intégrée pour **imprimer les plannings** tels qu'ils apparaissent dans votre vue actuelle. Ceci est utile pour partager les chronologies de présentation, l'archivage, ou pour afficher des références hors ligne pour les réunions et les opérations.

***

## Comment imprimer les présentations planifiées

Avant d'imprimer, assurez-vous que les éléments de planning que vous souhaitez imprimer sont **visibles** dans la fenêtre principale.

> **Important:** PADS4 Scheduler **n'imprimera que les plannings actuellement visibles** dans la vue sélectionnée (par exemple, Jour, Semaine, Tableau, etc.).

***

### Étapes pour imprimer :

1. Naviguez vers l'onglet **File** dans la barre de ruban.
2. Cliquez sur **Print**.

***

### Fenêtre des paramètres d'impression

Une **fenêtre d'impression** apparaîtra où vous pourrez :

* Choisir votre **imprimante**
* Ajuster les **paramètres de page** (par exemple, taille du papier, orientation)
* Définir **l'orientation d'impression** (portrait ou paysage)
* Voir un **aperçu** de ce qui sera imprimé sur le côté droit

***

### Finalisation de l'impression

Une fois vos paramètres confirmés :

* Cliquez sur **Print** pour imprimer la vue actuelle du calendrier.

Le système enverra les données visibles à l'imprimante sélectionnée.
