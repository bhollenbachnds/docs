PADS4 Scheduler offre une fonctionnalité intégrée pour **imprimer les plannings** tels qu'ils apparaissent dans votre vue actuelle. Cela est utile pour partager les chronologies de présentation, archiver ou afficher des références hors ligne pour les réunions et les opérations.

***

## Comment Imprimer les Présentations Planifiées

Avant d'imprimer, assurez-vous que les éléments de planning que vous souhaitez imprimer sont **visibles** dans la fenêtre principale.

> **Important:** PADS4 Scheduler n'imprimera **que les plannings actuellement visibles** dans la vue sélectionnée (par ex., Jour, Semaine, Tableau, etc.).

***

### Étapes pour Imprimer :

1. Accédez à l'onglet **Fichier** dans la barre de ruban.
2. Cliquez sur **Imprimer**.

***

### Fenêtre des Paramètres d'Impression

Une **fenêtre d'impression** apparaîtra où vous pourrez :

* Choisir votre **imprimante**
* Ajuster les **paramètres de page** (par ex., format du papier, orientation)
* Définir l'**orientation d'impression** (portrait ou paysage)
* Afficher un **aperçu** de ce qui sera imprimé sur le côté droit

***

### Finalisation de l'Impression

Une fois vos paramètres confirmés :

* Cliquez **Imprimer** pour imprimer la vue de planification actuelle.

Le système enverra les données visibles à l'imprimante sélectionnée.
