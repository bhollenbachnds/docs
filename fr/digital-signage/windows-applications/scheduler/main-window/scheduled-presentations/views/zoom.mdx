La **Zoom** fonctionnalité dans PADS4 Scheduler vous permet d'ajuster le niveau de détail affiché dans la **Calendar View**. Ceci est particulièrement utile lorsque vous travaillez avec des plannings chargés ou que vous examinez des présentations de longue durée.

> **Note:** Tous les modes de Calendar ne prennent pas en charge le zoom. La fonctionnalité de zoom est disponible en modes **Day**, **Work Week**, **Week**, et **Timeline** mais **not in Month view**.

***

## Zoom In

Pour augmenter le niveau de zoom (afficher des détails temporels plus précis) :

1. Allez dans l'onglet **View** dans la barre de ruban.
2. Dans le groupe **Zoom**, cliquez sur **Zoom In**.

Cela réduit l'échelle de temps, montrant des intervalles plus petits (par exemple, des blocs de 15 minutes au lieu de blocs de 30 minutes).

***

## Zoom Out

Pour diminuer le niveau de zoom (voir une vue d'ensemble plus large) :

1. Allez dans l'onglet **View** dans la barre de ruban.
2. Dans le groupe **Zoom**, cliquez sur **Zoom Out**.

Cela élargit la vue, affichant plus d'heures ou de jours à la fois.

***

## Select a Specific Zoom Level

Pour passer directement à un niveau de zoom prédéfini :

1. Dans le groupe **Zoom** de l'onglet **View**, cliquez sur **Zoom Level**.
2. Un menu déroulant apparaîtra avec les niveaux disponibles.
3. Cliquez sur le niveau de zoom souhaité pour l'appliquer.

***

## Zoom Levels by View Mode

Chaque mode de Calendar View prend en charge différents comportements de zoom :

| View Mode     | Zoom Supported | Notes                                                            |
| ------------- | -------------- | ---------------------------------------------------------------- |
| **Day**       | ✔              | Planification détaillée (horaire/minute)                         |
| **Work Week** | ✔              | Affiche uniquement les jours de semaine, prend en charge le zoom |
| **Week**      | ✔              | Semaine complète de 7 jours avec zoom ajustable                  |
| **Month**     | ✖              | Le zoom n'est **not** supported                                  |
| **Timeline**  | ✔              | Disposition horizontale avec mise à l'échelle ajustable          |
