La **Table View** affiche toutes les présentations programmées sous forme d'enregistrements structurés dans un format tabulaire. Cette vue est particulièrement utile pour examiner rapidement des données de planification détaillées et comparer plusieurs entrées.

***

## Accéder à la vue tableau

Pour passer à la vue tableau :

1. Naviguez vers l'onglet **View** dans la barre de ruban.
2. Dans la section **Schedules View**, cliquez sur **Table**.

Les planifications pour la destination et la/les date(s) sélectionnées apparaîtront maintenant dans une **disposition en tableau**, montrant des détails tels que :

* Nom de la présentation
* Heure de début et de fin
* Durée
* Destination assignée
* Propriétés supplémentaires (comme les intervalles de répétition ou la priorité)
