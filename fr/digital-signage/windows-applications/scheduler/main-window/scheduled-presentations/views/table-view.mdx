La **Vue Tableau** affiche toutes les présentations planifiées sous forme d'enregistrements structurés dans un format tabulaire. Cette vue est particulièrement utile pour examiner rapidement les données de planification détaillées et comparer plusieurs entrées.

***

## Accéder à la Vue Tableau

Pour passer à la Vue Tableau :

1. Accédez à l'onglet **View** dans la barre de ruban.
2. Dans la section **Schedules View**, cliquez sur **Table**.

Les planifications pour la destination et la ou les dates sélectionnées apparaîtront désormais dans une **disposition en tableau**, affichant des détails tels que :

* Nom de la présentation
* Heure de début et de fin
* Durée
* Destination assignée
* Propriétés supplémentaires (telles que les intervalles de répétition ou la priorité)
