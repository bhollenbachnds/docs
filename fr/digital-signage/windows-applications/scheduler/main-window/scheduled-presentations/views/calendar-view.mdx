La **Vue Calendrier** est la **vue par défaut** dans PADS4 Scheduler et s'affiche automatiquement au lancement de l'application. Il fournit un aperçu visuel, basé sur un calendrier, de toutes les présentations programmées pour la destination et la/les date(s) sélectionnée(s).

Cette vue est idéale pour planifier, examiner et modifier les calendriers de contenu avec une perspective orientée sur le temps.

***

## Accéder à la vue Calendrier

Pour passer à la vue Calendrier :

1. Allez dans l'onglet **View** dans la barre de ruban.
2. Dans la section **Schedules View**, cliquez sur **Calendar**.

L'affichage se mettra à jour pour montrer les présentations dans une disposition de calendrier traditionnelle.

***

## Modes de calendrier

PADS4 Scheduler prend en charge plusieurs **Calendar modes**, vous permettant de personnaliser l'affichage des planifications en fonction de vos besoins :

* **Day View** : Affiche le programme d'une seule journée, décomposé par heure.
* **Work Week View** : Affiche du lundi au vendredi, idéal pour la planification professionnelle.
* **Week View** : Affiche une semaine complète de sept jours.
* **Month View** : Présente un aperçu complet du mois pour une planification plus large.
* **Timeline View** : Affiche les planifications dans un format horizontal de type Gantt, particulièrement utile pour gérer le contenu qui se chevauche sur de plus longues périodes.

Pour changer le mode Calendrier :

1. Dans l'onglet **View**, dans la section **Calendar View**, cliquez sur le mode de votre choix :
   * **Day**
   * **Work Week**
   * **Week**
   * **Month**
   * **Timeline**

Le mode sélectionné mettra immédiatement à jour l'affichage dans la zone des planifications.
