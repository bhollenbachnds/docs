La **Calendar View** est la **vue par défaut** dans PADS4 Scheduler et s'affiche automatiquement au lancement de l'application. Elle fournit une vue d'ensemble visuelle, basée sur un calendrier, de toutes les présentations programmées pour la destination et la/les date(s) sélectionnée(s).

Cette vue est idéale pour planifier, examiner et modifier les calendriers de contenu avec une perspective orientée sur le temps.

***

## Accéder à la vue Calendrier

Pour passer à la vue Calendrier :

1. Allez dans l'onglet **View** dans la barre de ruban.
2. Dans la section **Schedules View**, cliquez sur **Calendar**.

L'affichage sera mis à jour pour montrer les présentations dans une disposition de calendrier traditionnelle.

***

## Modes de calendrier

PADS4 Scheduler prend en charge plusieurs **modes de Calendrier**, vous permettant de personnaliser l'affichage des planifications en fonction de vos besoins de planification :

* **Vue Jour** : Affiche le planning d'une seule journée, décomposé par heure.
* **Vue Semaine de travail** : Affiche du lundi au vendredi, idéal pour la planification professionnelle.
* **Vue Semaine** : Affiche une semaine complète de sept jours.
* **Vue Mois** : Présente une vue d'ensemble d'un mois complet pour une planification plus large.
* **Vue Chronologique** : Affiche les planifications dans un format horizontal de type Gantt, particulièrement utile pour gérer le contenu qui se chevauche sur de plus longues périodes.

Pour changer le mode Calendrier :

1. Dans l'onglet **View**, dans la section **Calendar View**, cliquez sur le mode de votre choix :
   * **Day**
   * **Work Week**
   * **Week**
   * **Month**
   * **Timeline**

Le mode sélectionné mettra immédiatement à jour l'affichage dans la zone des planifications.
