La **Vue Calendrier** est la **vue par défaut** dans PADS4 Scheduler et s'affiche automatiquement au lancement de l'application. Elle fournit un aperçu visuel, basé sur un calendrier, de toutes les présentations planifiées pour la destination et la ou les dates sélectionnées.

Cette vue est idéale pour planifier, examiner et modifier les plannings de contenu avec une perspective orientée dans le temps.

***

## Accéder à la Vue Calendrier

Pour passer à la Vue Calendrier :

1. Accédez à l'onglet **Vue** dans la barre de ruban.
2. Dans la section **Vue des plannings**, cliquez sur **Calendrier**.

L'affichage se mettra à jour pour présenter les présentations dans une mise en page de calendrier traditionnelle.

***

## Modes du calendrier

PADS4 Scheduler prend en charge plusieurs **modes de calendrier**, vous permettant de personnaliser l'affichage des plannings en fonction de vos besoins de planification :

* **Vue Jour**: Affiche le planning d'une seule journée, décomposé par heure.
* **Vue Semaine de travail**: Affiche du lundi au vendredi, idéal pour la planification professionnelle.
* **Vue Semaine**: Affiche une semaine complète de sept jours.
* **Vue Mois**: Présente un aperçu d'un mois complet pour une planification plus large.
* **Vue Chronologie**: Affiche les plannings dans un format horizontal de type Gantt, particulièrement utile pour gérer les contenus qui se chevauchent sur de longues périodes.

Pour changer le mode du calendrier :

1. Dans l'onglet **Vue**, dans la section **Vue Calendrier**, cliquez sur votre mode préféré :
   * **Jour**
   * **Semaine de travail**
   * **Semaine**
   * **Mois**
   * **Chronologie**

Le mode sélectionné mettra immédiatement à jour l'affichage dans la zone des plannings.
