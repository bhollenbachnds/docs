La **Calendar View** est la **vue par défaut** dans PADS4 Scheduler et s'affiche automatiquement au lancement de l'application. Elle fournit un aperçu visuel, basé sur un calendrier, de toutes les présentations planifiées pour la destination et la(les) date(s) sélectionnée(s).

Cette vue est idéale pour planifier, examiner et modifier les plannings de contenu avec une perspective orientée dans le temps.

***

## Accéder à la Calendar View

Pour passer à la Calendar View :

1. Aller dans l'onglet **View** dans la barre de ruban.
2. Dans la section **Schedules View**, cliquer sur **Calendar**.

L'affichage se mettra à jour pour présenter les présentations dans une mise en page de calendrier traditionnelle.

***

## Modes de calendrier

PADS4 Scheduler prend en charge plusieurs **modes de calendrier**, vous permettant de personnaliser l'affichage des plannings en fonction de vos besoins de planification :

* **Day View**: Affiche le planning d'une seule journée, décomposé par heure.
* **Work Week View**: Affiche du lundi au vendredi, idéal pour la planification professionnelle.
* **Week View**: Affiche une semaine complète de sept jours.
* **Month View**: Présente un aperçu d'un mois complet pour une planification plus large.
* **Timeline View**: Affiche les plannings dans un format horizontal de type Gantt, particulièrement utile pour gérer le contenu qui se chevauche sur de longues périodes.

Pour changer le mode Calendar :

1. Dans l'**View** onglet, dans la section **Calendar View**, cliquez sur votre mode préféré :
   * **Day**
   * **Work Week**
   * **Week**
   * **Month**
   * **Timeline**

Le mode sélectionné mettra immédiatement à jour l'affichage dans la zone Schedules.
