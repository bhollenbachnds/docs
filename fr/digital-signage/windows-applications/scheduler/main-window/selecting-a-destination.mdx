Avant de consulter ou de gérer les présentations planifiées, vous devez d'abord **sélectionner une Destination**. Dans PADS4, une *Destination* désigne un **Viewer** (un appareil d'affichage dynamique) ou un **Viewer Group** (un ensemble de Viewers). Ces appareils sont gérés de manière centralisée par le système PADS4.

PADS4 Scheduler offre plusieurs façons de sélectionner une Destination, selon vos préférences ou votre configuration.

***

## Sélectionner une Destination depuis la vue arborescente

1. Dans la **fenêtre principale**, localisez la **vue arborescente des Destinations** dans le panneau de gauche.
2. Cliquez sur un **Viewer** ou un **Viewer Group** pour le sélectionner.
   * Utilisez les symboles de flèche pour développer les éléments groupés si nécessaire.
3. Une fois sélectionnée, les vues de planning à droite se mettront automatiquement à jour pour refléter la Destination choisie.

> **Note:** La vue arborescente est généralement remplie automatiquement avec les Viewers définis dans votre système PADS4. Si votre compte utilisateur dispose d'autorisations limitées, seuls les Viewers auxquels vous êtes autorisé à accéder apparaîtront.

***

## Utiliser la vue alphabétique

Par défaut, les Destinations sont affichées dans une **Vue par catégorie**. Vous pouvez passer à la **Vue alphabétique** pour faciliter la recherche ou le défilement.

1. Déplacez votre curseur vers la zone **Destinations** dans le panneau de gauche.
2. Cliquez sur le bouton **Vue alphabétique**.
3. Les Destinations seront désormais listées par ordre alphabétique.

***

## Utiliser la fonction de recherche

Le **bouton Rechercher** vous permet de trouver des Destinations par nom.

1. Cliquez sur le bouton **Rechercher** situé à côté du bouton Vue alphabétique.
2. Saisissez votre **texte de recherche**.
3. Cliquez sur le **bouton flèche droite** pour trouver la première correspondance.
4. Cliquez à nouveau sur la flèche pour parcourir les résultats multiples.

> La fonction de recherche correspond à n'importe quelle partie du nom de la Destination, pas seulement au début.

***

## Utiliser l'option de filtre

Le filtrage permet de réduire la liste des Destinations visibles selon vos critères.

> **Important:** Le filtrage est uniquement disponible en **Vue alphabétique**.

1. Passer en **Vue alphabétique** si ce n'est pas déjà actif.
2. Cliquez sur le **Filter** bouton.
3. Saisissez votre **texte de filtre**.
   * Les résultats se mettent à jour en temps réel au fur et à mesure que vous tapez.
4. Pour affiner les résultats par type :
   * Cliquez sur le **Viewer Type selection box**.
   * Choisissez **Viewers Only** ou **All Destinations**.

### Effacer le filtre :

* Cliquez sur le **Clear Filter** bouton pour réinitialiser la liste et afficher à nouveau toutes les Destinations.

***

## Utiliser Search et Filter ensemble

Vous pouvez combiner **Search** et **Filter** pour affiner davantage votre liste de Destinations.

1. Cliquez sur le **Search** bouton.
2. Cliquez sur le **Filter** bouton.
3. Saisissez vos termes de recherche ou de filtre.
4. Naviguez dans les résultats selon vos besoins.

***

## Sélectionner une Destination via la Ribbon Bar

Les Destinations peuvent également être sélectionnées à l'aide de l'onglet **View tab** dans la ribbon bar.

1. Dans le **View** groupe d'onglets, cliquez sur **Select Destination**.
2. La **Browse for a Destination** fenêtre apparaît.
   * Cela inclut les vues Categorized et Alphabetical.
   * Les fonctions Search et Filter sont également disponibles ici.
3. Sélectionnez la Destination souhaitée.
4. Cliquez sur **OK** pour confirmer.

La Destination sélectionnée sera désormais active dans la fenêtre principale.
