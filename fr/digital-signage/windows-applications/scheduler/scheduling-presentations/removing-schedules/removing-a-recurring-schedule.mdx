PADS4 Scheduler vous permet de supprimer **plannings récurrents** de deux manières distinctes :

* **Supprimer une occurrence** – Supprime uniquement une seule instance du planning récurrent.
* **Supprimer la série** – Supprime la **définition de récurrence entière** et toutes les occurrences associées.

> **Distinction importante**
>
> * **Supprimer la récurrence** : Supprime le modèle de récurrence et convertit l'occurrence sélectionnée en un **planning autonome**. Toutes les autres occurrences sont supprimées.
> * **Supprimer la série**: Supprime le **planning récurrent entier**, y compris les paramètres de récurrence **et toutes les occurrences**.

***

## Pour supprimer un planning récurrent

### Steps:

1. Dans la **arborescence des destinations**, sélectionnez le **Viewer** ou **Viewer Group** où le planning récurrent a été créé.

2. Dans la **vue Calendrier** ou **Tableau**, **sélectionnez le planning récurrent** que vous souhaitez modifier.

3. Dans l'onglet **Home** de la barre de ruban, sous le groupe **Clipboard**, cliquez sur le bouton **Delete**.

4. Une boîte de dialogue apparaîtra vous demandant de choisir entre :
   * **Delete Occurrence**
   * **Delete Series**

5. Cliquez sur **OK** pour confirmer l'action sélectionnée.

***

## Méthodes alternatives

### Utilisation du menu contextuel

1. **Clic droit** sur le planning récurrent.

2. Survolez l'option **Delete** dans le menu contextuel.
   * Choisissez **Occurrence** pour supprimer une seule instance.
   * Choisissez **Series** pour supprimer toutes les occurrences et la règle de récurrence.

3. Vous pouvez également survoler **Recurrence**, puis sélectionner **Remove** pour supprimer la récurrence et ne conserver que l'instance sélectionnée en tant que planning **unique, non récurrent**.

***

### Utilisation de la barre de ruban

1. Accédez à l'onglet **Home**.
2. Dans le **Manage Schedule** group:
   * Cliquez sur **Remove Recurrence** pour supprimer le modèle mais conserver l'occurrence sélectionnée en tant que planning ponctuel.
   * Ou cliquez sur la **flèche** à côté de **Delete** et sélectionnez :
     * **Delete Occurrence**
     * **Delete Series**

***

Cette flexibilité vous permet de contrôler comment et quand les plannings récurrents sont modifiés ou supprimés — partiellement ou entièrement.
