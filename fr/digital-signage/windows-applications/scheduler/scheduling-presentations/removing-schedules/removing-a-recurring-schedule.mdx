PADS4 Scheduler vous permet de supprimer des **planifications récurrentes** de deux manières distinctes :

* **Supprimer une occurrence** – Supprime uniquement une seule instance de la planification récurrente.
* **Supprimer la série** – Supprime la **définition de récurrence entière** et toutes les occurrences associées.

> **Distinction importante**
>
> * **Supprimer la récurrence** : Supprime le modèle de récurrence et convertit l'occurrence sélectionnée en **planification autonome**. Toutes les autres occurrences sont supprimées.
> * **Supprimer la série** : Supprime la **planification récurrente entière**, y compris les paramètres de récurrence **et toutes les occurrences**.

***

## Pour supprimer une planification récurrente

### Steps:

1. Dans la **vue arborescente des destinations**, sélectionnez le **Viewer** ou le **Viewer Group** où la planification récurrente a été créée.

2. Dans la **Calendrier** ou **Table** vue, **sélectionnez le planning récurrent** que vous souhaitez modifier.

3. Dans l'onglet **Home** du ruban, sous le groupe **Clipboard**, cliquez sur le bouton **Delete**.

4. Une boîte de dialogue apparaîtra vous demandant de choisir entre :
   * **Delete Occurrence**
   * **Delete Series**

5. Cliquez sur **OK** pour confirmer l'action sélectionnée.

***

## Méthodes alternatives

### Utilisation du menu contextuel

1. **Faites un clic droit** sur le planning récurrent.

2. Passez la souris sur l'option **Delete** dans le menu contextuel.
   * Choisissez **Occurrence** pour supprimer une seule instance.
   * Choisissez **Series** pour supprimer toutes les occurrences et la règle de récurrence.

3. Alternativement, passez la souris sur **Recurrence**, puis sélectionnez **Remove** pour supprimer la récurrence et ne conserver que l'instance sélectionnée comme un planning **unique, non récurrent**.

***

### Utilisation du ruban

1. Naviguez vers l'onglet **Home**.
2. Dans le **Manage Schedule** group:
   * Cliquez sur **Remove Recurrence** pour supprimer le modèle mais conserver l'occurrence sélectionnée comme un planning unique.
   * Ou cliquez sur la **flèche** à côté de **Delete** et sélectionnez :
     * **Delete Occurrence**
     * **Delete Series**

***

Cette flexibilité vous permet de contrôler comment et quand les plannings récurrents sont modifiés ou supprimés—partiellement ou entièrement.
