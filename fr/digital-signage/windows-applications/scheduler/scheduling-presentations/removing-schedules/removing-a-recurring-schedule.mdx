PADS4 Scheduler vous permet de supprimer **des programmations récurrentes** de deux manières distinctes :

* **Supprimer une occurrence** – Supprime uniquement une seule instance de la programmation récurrente.
* **Supprimer la série** – Supprime la **définition de récurrence complète** et toutes les occurrences associées.

> **Distinction importante**
>
> * **Supprimer la récurrence** : Supprime le modèle de récurrence et convertit l'occurrence sélectionnée en une **programmation autonome**. Toutes les autres occurrences sont supprimées.
> * **Supprimer la série** : Supprime la **programmation récurrente complète**, y compris les paramètres de récurrence **et toutes les occurrences**.

***

## Pour supprimer une programmation récurrente

### Steps:

1. Dans la **vue arborescente des destinations**, sélectionnez le **Viewer** ou **Viewer Group** où la programmation récurrente a été créée.

2. Dans la vue **Calendrier** ou **Tableau**, **sélectionnez la programmation récurrente** que vous souhaitez modifier.

3. Dans l'onglet **Home** du ruban, sous le groupe **Clipboard**, cliquez sur le bouton **Delete**.

4. Une boîte de dialogue apparaîtra vous demandant de choisir entre :
   * **Delete Occurrence**
   * **Delete Series**

5. Cliquez sur **OK** pour confirmer l'action sélectionnée.

***

## Méthodes alternatives

### Utilisation du menu contextuel

1. **Cliquez avec le bouton droit** sur la programmation récurrente.

2. Passez la souris sur l'option **Delete** dans le menu contextuel.
   * Choisissez **Occurrence** pour supprimer une seule instance.
   * Choisissez **Series** pour supprimer toutes les occurrences et la règle de récurrence.

3. Alternativement, passez la souris sur **Recurrence**, puis sélectionnez **Remove** pour supprimer la récurrence et ne conserver que l'instance sélectionnée comme **programmation unique, non récurrente**.

***

### Utilisation du ruban

1. Naviguez vers l'onglet **Home**.
2. Dans le **Manage Schedule** group:
   * Cliquez sur **Remove Recurrence** pour supprimer le modèle mais conserver l'occurrence sélectionnée comme programmation unique.
   * Ou cliquez sur la **flèche** à côté de **Delete** et sélectionnez :
     * **Delete Occurrence**
     * **Delete Series**

***

Cette flexibilité vous permet de contrôler comment et quand les programmations récurrentes sont modifiées ou supprimées—partiellement ou entièrement.
