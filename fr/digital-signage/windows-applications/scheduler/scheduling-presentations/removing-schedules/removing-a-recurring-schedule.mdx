PADS4 Scheduler vous permet de supprimer des **recurring schedules** de deux manières distinctes :

* **Delete an Occurrence** – Supprime uniquement une seule instance du planning récurrent.
* **Delete the Series** – Supprime la **entire recurrence definition** et toutes les occurrences associées.

> **Distinction importante**
>
> * **Remove Recurrence** : Supprime le modèle de récurrence et convertit l'occurrence sélectionnée en un **standalone schedule**. Toutes les autres occurrences sont supprimées.
> * **Delete Series** : Supprime le **entire recurring schedule**, y compris les paramètres de récurrence **and all occurrences**.

***

## Pour supprimer un planning récurrent

### Steps:

1. Dans la **Destinations tree view**, sélectionnez le **Viewer** ou le **Viewer Group** où le planning récurrent a été créé.

2. Dans la vue **Calendar** ou **Table**, **select the recurring schedule** que vous souhaitez modifier.

3. Dans l'onglet **Home** du ruban, sous le groupe **Clipboard**, cliquez sur **Delete** bouton.

4. Une boîte de dialogue apparaîtra vous demandant de choisir entre :
   * **Supprimer l'occurrence**
   * **Supprimer la série**

5. Cliquez sur **OK** pour confirmer votre action sélectionnée.

***

## Méthodes alternatives

### Utilisation du menu contextuel

1. **Cliquez avec le bouton droit** sur le planning récurrent.

2. Passez la souris sur l'option **Supprimer** dans le menu contextuel.
   * Choisissez **Occurrence** pour supprimer une seule instance.
   * Choisissez **Série** pour supprimer toutes les occurrences et la règle de récurrence.

3. Alternativement, passez la souris sur **Récurrence**, puis sélectionnez **Supprimer** pour supprimer la récurrence et ne conserver que l'instance sélectionnée comme un planning **unique, non récurrent**.

***

### Utilisation de la barre de ruban

1. Naviguez vers l'onglet **Accueil**.
2. Dans la section **Gérer le planning** group:
   * Cliquez sur **Supprimer la récurrence** pour supprimer le modèle mais conserver l'occurrence sélectionnée comme un planning unique.
   * Ou cliquez sur la **flèche** à côté de **Supprimer** et sélectionnez :
     * **Supprimer l'occurrence**
     * **Supprimer la série**

***

Cette flexibilité vous permet de contrôler comment et quand les plannings récurrents sont modifiés ou supprimés — partiellement ou entièrement.
