Vous pouvez **supprimer manuellement** tout type de présentation planifiée — qu'il s'agisse d'une planification **Simple**, **Playlist** ou **Advanced** — directement depuis la **Calendar View** ou la **Table View**.

***

## Comment supprimer une planification

### Steps:

1. Dans la **Destinations tree view** (panneau gauche), sélectionnez le **Viewer** ou le **Viewer Group** où la planification a été créée.

2. Dans la vue **Calendar** ou **Table**, **cliquez** pour sélectionner la présentation planifiée que vous souhaitez supprimer.

3. Accédez à l'onglet **Home** dans la barre de ruban.

4. Dans le groupe **Clipboard**, cliquez sur **Delete**.

***

### Méthodes de suppression alternatives

Vous pouvez également supprimer une planification en utilisant l'une des méthodes suivantes :

* Cliquez sur le bouton **Delete** dans le groupe **Manage Schedule** de l'onglet **Home**.
* **Faites un clic droit** sur la planification et sélectionnez **Delete** dans le menu contextuel.
* Appuyez sur la touche **Del** de votre clavier après avoir sélectionné la planification.

***

### Confirmation de suppression

Un message de confirmation apparaîtra :

> "Êtes-vous sûr de vouloir supprimer cette planification ?"

* Cliquez sur **Yes** pour supprimer définitivement la planification.
* Cliquez sur **No** pour annuler l'action.

Une fois confirmée, la planification sera supprimée de la vue et ne sera plus active pour la ou les destinations sélectionnées.
