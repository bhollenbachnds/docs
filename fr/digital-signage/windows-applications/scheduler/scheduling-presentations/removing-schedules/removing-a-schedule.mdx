Vous pouvez **supprimer manuellement** tout type de présentation programmée—qu'il s'agisse d'un planning **Simple**, **Playlist**, ou **Advanced**—directement depuis la **Calendar View** ou la **Table View**.

***

## Comment supprimer un planning

### Steps:

1. Dans la **Destinations tree view** (panneau de gauche), sélectionnez le **Viewer** ou le **Viewer Group** où le planning a été créé.

2. Dans la vue **Calendar** ou **Table**, **cliquez** pour sélectionner la présentation programmée que vous souhaitez supprimer.

3. Naviguez vers l'onglet **Home** dans la barre de ruban.

4. Dans le groupe **Clipboard**, cliquez sur **Delete**.

***

### Méthodes alternatives de suppression

Vous pouvez également supprimer un planning en utilisant l'une des méthodes suivantes :

* Cliquez sur le bouton **Delete** dans le groupe **Manage Schedule** de l'onglet **Home**.
* **Faites un clic droit** sur le planning et sélectionnez **Delete** dans le menu contextuel.
* Appuyez sur la touche **Del** de votre clavier après avoir sélectionné le planning.

***

### Invite de confirmation

Un message de confirmation apparaîtra :

> "Êtes-vous sûr de vouloir supprimer ce planning ?"

* Cliquez sur **Yes** pour supprimer définitivement le planning.
* Cliquez sur **No** pour annuler l'action.

Une fois confirmé, le planning sera supprimé de la vue et ne sera plus actif pour la ou les destination(s) sélectionnée(s).
