Vous pouvez **supprimer manuellement** tout type de présentation planifiée—qu'il s'agisse d'une planification **Simple**, **Playlist** ou **Advanced**—directement depuis la **Calendar View** ou la **Table View**.

***

## Comment supprimer une planification

### Steps:

1. Dans la **vue arborescente des destinations** (panneau gauche), sélectionnez le **Viewer** ou le **Viewer Group** où la planification a été créée.

2. Dans la vue **Calendar** ou **Table**, **cliquez** pour sélectionner la présentation planifiée que vous souhaitez supprimer.

3. Accédez à l'onglet **Home** dans la barre de ruban.

4. Dans le groupe **Clipboard**, cliquez sur **Delete**.

***

### Autres méthodes de suppression

Vous pouvez également supprimer un planning en utilisant l'une des méthodes suivantes :

* Cliquez sur le **Delete** dans le groupe **Manage Schedule** de l'onglet **Home**.
* **Faites un clic droit** sur le planning et sélectionnez **Delete** dans le menu contextuel.
* Appuyez sur la touche **Suppr** de votre clavier après avoir sélectionné le planning.

***

### Invite de confirmation

Un message de confirmation apparaîtra :

> "Êtes-vous sûr de vouloir supprimer ce planning ?"

* Cliquez sur **Oui** pour supprimer définitivement le planning.
* Cliquez sur **Non** pour annuler l'action.

Une fois confirmé, le planning sera supprimé de la vue et ne sera plus actif pour la ou les destination(s) sélectionnée(s).
