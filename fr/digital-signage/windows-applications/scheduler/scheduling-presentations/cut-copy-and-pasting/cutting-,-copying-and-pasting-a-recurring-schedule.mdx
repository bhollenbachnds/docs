PADS4 Scheduler vous permet d'utiliser **cut**, **copy**, et **paste** opérations sur **recurring schedules**, ce qui facilite leur déplacement ou leur duplication entre différents **Viewers** ou **Viewer Groups**. Le comportement de ces actions diffère légèrement des planifications régulières (non récurrentes).

***

## Couper et coller un planning récurrent

Lorsque vous **cut and paste** un planning récurrent, un **new schedule** est créé à la destination, et l'original est supprimé.

### Steps:

1. Dans la **Destinations tree view**, sélectionnez le **Viewer** ou **Viewer Group** qui contient le planning récurrent.

2. Dans la vue **Calendar** ou **Table**, sélectionnez le planning récurrent.

3. Naviguez vers l'onglet **Home** dans la barre de ruban et cliquez sur **Cut** dans le groupe **Clipboard**.

4. Une boîte de dialogue de confirmation apparaîtra. Confirmez que vous souhaitez supprimer le planning récurrent.

   * Le planning est supprimé de l'emplacement d'origine.
   * Une copie reste dans le presse-papiers, prête à être collée.

5. Naviguez vers la nouvelle destination et sélectionnez la position souhaitée.

6. Cliquez sur **Paste** dans le groupe **Clipboard**.

Le planning récurrent est maintenant créé comme un **new, independent schedule** à la destination.

Pour confirmer :

* Faites un clic droit sur le planning collé
* Sélectionnez **Edit > Schedule**
* La fenêtre **Edit Schedule** affichera une nouvelle définition de récurrence

***

## Copier et coller un planning récurrent

Lorsque vous **copy and paste** un planning récurrent, cela crée une **new timetable entry** dans l'objet de planning original, à condition que le type de planning prenne en charge plusieurs destinations ou créneaux horaires (par exemple, un Advanced Schedule).

### Steps:

1. Dans la **Destinations tree view**, sélectionnez le **Viewer** ou **Viewer Group** qui contient le planning récurrent.

2. Dans la vue **Calendar** ou **Table**, sélectionnez le planning récurrent.

3. Cliquez sur **Copy** dans le groupe **Clipboard** de la barre de ruban.

4. Naviguez vers la nouvelle date ou le Viewer où le planning doit être dupliqué.

5. Cliquez sur **Paste**.

Le planning récurrent est maintenant ajouté comme **new entry in the schedule's timetable**.

Pour confirmer :

* Faites un clic droit sur le planning collé
* Sélectionnez **Edit > Schedule**
* La fenêtre **Edit Schedule** affichera plusieurs entrées, y compris la nouvelle

***

## Résumé

| Action       | Résultat                                                    |
| ------------ | ----------------------------------------------------------- |
| Cut + Paste  | Supprime l'original et crée un **new recurring schedule**   |
| Copy + Paste | Ajoute une **new recurring entry** au **existing schedule** |
