Si vous avez créé des présentations avec **PADS4 Designer** et les avez enregistrées en tant que `.PADSX` fichiers sur votre disque local ou votre stockage réseau, vous pouvez les planifier directement en les **faisant glisser-déposer** dans **PADS4 Scheduler**.

Cette fonctionnalité offre un moyen rapide et intuitif de planifier des présentations existantes sans avoir à parcourir manuellement les fichiers.

***

## Comment glisser-déposer une présentation dans Scheduler

### Prerequisites:

* Assurez-vous que **PADS4 Scheduler** et **Windows Explorer** sont ouverts et positionnés de manière à ce que les deux fenêtres soient visibles.

### Steps:

1. **Localiser le fichier de présentation**
   * Dans **Windows Explorer**, accédez au `.PADSX` fichier que vous souhaitez planifier.

2. **Cliquez et maintenez** le fichier avec votre souris.

3. **Faites glisser le fichier** vers la **fenêtre PADS4 Scheduler**.
   * Le curseur de votre souris changera pour indiquer qu'un fichier peut être déposé (généralement une petite icône de document apparaît).

4. **Relâchez le bouton de la souris** pour déposer le fichier dans l'interface Scheduler.

***

### Ce qui se passe ensuite

* Le **Schedule Wizard** se lancera automatiquement.
* Le `.PADSX` fichier sélectionné sera déjà chargé dans l'assistant.
* À partir de ce moment, le processus de planification suit les mêmes étapes qu'un **Simple Schedule** :
  * Saisissez un nom
  * Choisir la destination
  * Définir la période d'affichage et la priorité
  * (Facultativement) appliquer une récurrence
  * Cliquer sur **Terminer** pour compléter la planification
