Si vous avez créé des présentations à l'aide de **PADS4 Designer** et les avez enregistrées comme `.PADSX` fichiers sur votre disque local ou stockage réseau, vous pouvez les programmer directement en les **glissant-déposant** dans **PADS4 Scheduler**.

Cette fonctionnalité offre un moyen rapide et intuitif de programmer des présentations existantes sans avoir à parcourir manuellement les fichiers.

***

## Comment glisser-déposer une présentation dans Scheduler

### Prerequisites:

* Assurez-vous que **PADS4 Scheduler** et **Windows Explorer** sont ouverts et positionnés de manière à ce que les deux fenêtres soient visibles.

### Steps:

1. **Localisez le fichier de présentation**
   * Dans **Windows Explorer**, naviguez jusqu'au `.PADSX` fichier que vous souhaitez programmer.

2. **Cliquez et maintenez** le fichier avec votre souris.

3. **Faites glisser le fichier** vers la fenêtre **PADS4 Scheduler**.
   * Le curseur de votre souris changera pour indiquer qu'un fichier peut être déposé (généralement une petite icône de document apparaît).

4. **Relâchez le bouton de la souris** pour déposer le fichier sur l'interface Scheduler.

***

### Ce qui se passe ensuite

* L'**Schedule Wizard** se lancera automatiquement.
* Le `.PADSX` fichier sélectionné sera déjà chargé dans l'assistant.
* À partir de ce point, le processus de programmation suit les mêmes étapes qu'un **Simple Schedule**:
  * Entrez un nom
  * Choisir la destination
  * Définir la période d'affichage et la priorité
  * (Optionnellement) appliquer la récurrence
  * Cliquez **Terminer** pour compléter la planification
