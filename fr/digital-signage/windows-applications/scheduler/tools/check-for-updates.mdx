# Vérifier les mises à jour

PADS4 Scheduler inclut une fonctionnalité intégrée qui permet aux utilisateurs de vérifier manuellement les mises à jour logicielles. Cela garantit que votre installation reste à jour avec les dernières fonctionnalités, améliorations et mises à jour de sécurité.

***

## Effectuer une vérification manuelle des mises à jour

Pour vérifier manuellement les mises à jour disponibles :

1. Naviguez vers l'onglet **Tools** dans la barre de ruban.
2. Dans le groupe **Software**, cliquez sur **Check for Updates**.

Cette action lance l'**PADS4 Software Update Wizard**.

> L'assistant vous guidera tout au long du processus de mise à jour. Pour des instructions détaillées étape par étape, consultez le **PADS4 Installation Manual**, en particulier la section intitulée **Maintaining the Installation**.

***

Faites-moi savoir si vous souhaitez inclure une présentation détaillée de l'assistant de mise à jour lui-même ou lier ceci à une stratégie de mise à jour à l'échelle du réseau pour les environnements multi-clients.
