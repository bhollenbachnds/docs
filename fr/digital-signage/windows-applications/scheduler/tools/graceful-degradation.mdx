Lors de l'installation initiale de PADS4, l'activation est complétée à l'aide du **PADS4 License Wizard**. Ce processus génère une **hardware-bound product key**, qui est vérifiée par rapport au profil matériel de votre système.

***

## Période de dégradation progressive : 7 jours

Si **hardware changes** sont détectés (par exemple, remplacement de l'adaptateur réseau ou modifications des identifiants système), PADS4 entre dans une **graceful degradation period**.

* **Durée** : 7 jours
* Pendant cette période, le système **continue de fonctionner normalement**
* Un **warning message** s'affiche lors de la connexion pour les applications suivantes :
  * PADS4 Agent
  * PADS4 Messenger
  * System Management
  * Scheduler
  * Designer

> Vous pouvez configurer une alerte dans le **PADS4 Agent** pour notifier les administrateurs par e-mail lorsque des changements matériels sont détectés et qu'une réactivation est nécessaire.

***

## Réactivation du PADS4 Server

Pour éviter que PADS4 ne s'arrête après la période de grâce de 7 jours, suivez ces étapes :

### Processus de réactivation étape par étape :

1. **Contactez votre revendeur** pour signaler le changement matériel et demander une réactivation.
   * Fournissez votre **PADS4 product key**.
2. Le revendeur contactera **Net Display Systems** pour vérifier votre licence et autoriser la réactivation.
3. Vous recevrez une **email confirmation** une fois que votre licence aura été approuvée pour la réactivation.

### Réactivation via License Wizard :

1. Lancez le **License Wizard** sur le PADS4 Server.
2. Sélectionnez le **Server** et cliquez sur **Next**.
3. Vérifiez les détails de la licence actuelle et cliquez à nouveau sur **Next**.
4. Dans la fenêtre **Select an Action**, choisissez :
   * **Reactivate current installation**
5. L'assistant se connectera au PADS4 License Server pour vérifier votre clé.
6. Après validation réussie, le profil matériel est réinitialisé et la période de dégradation prend fin.

> Note: An **internet connection** est requise pour la réactivation en ligne.

### Activation hors ligne :

Si votre serveur n'a pas accès à Internet :

* Sélectionnez **Activate installation offline** dans le License Wizard
* Suivez les instructions guidées pour terminer l'activation manuellement

## Bonnes pratiques et recommandations

* **PADS4 will stop working** après 7 jours si la réactivation n'est pas effectuée.
* Si vous utilisez une **virtual machine**, assurez-vous que l'**MAC address is fixed** pour éviter des changements matériels involontaires.
* Évitez d'activer ou de partager des **Bluetooth devices** sur les machines virtuelles, car cela peut déclencher une détection de changement matériel.
* Utilisez des **email alerts** via le PADS4 Agent pour rester informé des exigences de réactivation à venir.

***

## Qu'est-ce qui déclenche une détection de changement matériel ?

PADS4 compare plusieurs identifiants matériels aux données d'activation d'origine. Si l'un des éléments suivants change, une réactivation peut être nécessaire :

* Operating System
* Processor ID
* OS Installation Date
* OS Serial Number
* BIOS Serial Number and Version
* Network Adapter and MAC Address
* Windows Activation Code
