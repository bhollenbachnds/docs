Le **À propos** dans PADS4 Scheduler fournit des informations sur la version de l'application que vous utilisez actuellement, ainsi que les détails de licence — si disponibles.

***

## Comment accéder à la fenêtre À propos

1. Ouvrez l'onglet **Aide** dans la barre de ruban.
2. Dans la **Info** section, cliquez sur **À propos**.

La fenêtre **À propos** s'ouvrira et affichera :

* Nom et version de l'application
* Numéro de build
* Informations sur les droits d'auteur et les marques déposées
* Détails de la licence (si connecté à un serveur)

***

## Remarques

> **Important:**\
> Les informations de licence affichées dans la fenêtre À propos sont récupérées depuis le **PADS4 Server** connecté. Si l'application n'est **pas connectée** à un serveur, les détails de la licence ne seront pas affichés.

Pour des informations de licence plus détaillées, utilisez l'un des éléments suivants :

* **PADS4 System Management** application
* **PADS4 License Wizard**
