PADS4 Scheduler prend en charge le travail avec plusieurs fenêtres, permettant aux utilisateurs de **comparer les plannings entre différentes Destinations** côte à côte. Cette fonctionnalité est utile pour gérer des plannings complexes sur différents écrans ou groupes de Viewers.

***

## Création d'une nouvelle fenêtre

Pour ouvrir une fenêtre de planification supplémentaire :

1. Naviguez vers l'onglet **View** dans la barre de ruban.
2. Dans le groupe **Window**, cliquez sur **New Window**.

Une nouvelle fenêtre s'ouvrira dans la zone de planification, affichant initialement la **même Destination** que la fenêtre d'origine.

Pour changer de Destination :

* Utilisez le bouton **Select Destination** dans la barre de ruban, ou
* Utilisez la **vue arborescente des Destinations** dans le volet gauche de l'application.

Chaque fenêtre peut maintenant afficher le planning d'une Destination différente pour une comparaison facile.

***

## Création d'un nouveau groupe d'onglets

Une fois que vous avez plus d'une fenêtre ouverte, vous pouvez les positionner **côte à côte** pour comparaison :

1. Allez dans l'onglet **View**.
2. Dans le groupe **Window**, cliquez sur **New Tab Group**.
3. Dans le menu déroulant, sélectionnez **New Vertical Tab Group**.

Les fenêtres de planification apparaîtront maintenant côte à côte dans une vue divisée.

***

## Basculer entre les fenêtres

Lorsque plusieurs fenêtres sont ouvertes, vous pouvez basculer entre elles de deux façons :

* Cliquez sur l'onglet correspondant à la fenêtre souhaitée.
* Ou, depuis le groupe **Window**, cliquez sur **Switch Windows** et sélectionnez dans la liste des fenêtres ouvertes.

La fenêtre sélectionnée devient la fenêtre active.

***

## Déplacer une fenêtre entre les groupes

Pour repositionner une fenêtre d'un groupe d'onglets à un autre :

1. Dans le groupe **Window**, cliquez sur **Move To**.
2. Choisissez **Move to Previous Tab Group** dans le menu déroulant.

La fenêtre actuellement active sera déplacée vers le groupe d'onglets spécifié.

***

## Utilisation du menu contextuel

Vous pouvez également faire un clic droit sur un onglet de fenêtre pour accéder à un menu contextuel avec des options supplémentaires de mise en page et de gestion des fenêtres.
