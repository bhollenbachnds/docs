PADS4 Scheduler prend en charge les connexions à **plusieurs PADS4 Servers** simultanément. Cette fonctionnalité vous permet de visualiser et de gérer les planifications de différents serveurs au sein d'une seule instance de l'application — idéale pour les environnements avec des déploiements distribués ou segmentés.

***

## Connexion à un PADS4 Server supplémentaire

En supposant que vous ayez déjà une connexion active (par ex., **MyPADSServer**), et qu'une autre connexion serveur ait été configurée (par ex., **AnotherPADSServer**) via les **Connection Properties** settings:

Pour établir une connexion à un second serveur :

1. Accédez à l'onglet **Home** dans la barre de ruban.
2. Dans le **Connections** groupe, cliquez sur le **Connect** bouton partagé.
3. Dans la liste déroulante, sélectionnez la connexion serveur supplémentaire (par ex., **AnotherPADSServer**).

Une fois connecté, les deux connexions serveur apparaîtront dans le **volet de navigation** en bas à gauche de l'interface. Le serveur nouvellement sélectionné devient le serveur **actif**, indiqué par :

* Nom du serveur mis en surbrillance dans le volet de navigation
* Affichage du nom du serveur au-dessus du calendrier
* Informations de la barre d'état mises à jour

> **Note:** Les volets de navigation ne sont affichés que lorsque deux connexions serveur ou plus sont actives. Si un seul serveur est connecté, le volet reste masqué.

***

## Affichage des planifications depuis plusieurs serveurs

Vous pouvez maintenant ouvrir **des fenêtres séparées** pour différents PADS4 Servers et comparer les planifications côte à côte.

Pour ce faire :

* Utilisez le **New Window** bouton sous l'onglet **View** (voir la *Working with Multiple Windows* section).
* Assignez chaque fenêtre à une **Destination** différente sur un serveur distinct.

Cette disposition permet une visibilité simultanée sur les planifications des deux serveurs.

***

## Déconnexion des PADS4 Servers

Pour vous déconnecter d'une ou plusieurs connexions serveur :

### Déconnecter un seul serveur

1. Accédez à l'onglet **Home**.
2. Dans le **Connections** groupe, cliquez sur le **Disconnect** bouton partagé.
3. Sélectionnez la connexion serveur spécifique que vous souhaitez déconnecter.

Cela va :

* Fermer la connexion sélectionnée
* Supprimer son volet de navigation
* Fermer toutes les fenêtres associées à ce serveur

### Déconnecter tous les serveurs

Pour fermer toutes les connexions serveur actives en même temps :

* Dans l'onglet **Home**, sous **Connections**, cliquez sur **Disconnect All**.

Tous les serveurs connectés et les fenêtres associées seront fermés.
