La **Propriétés de connexion** vous permet de configurer et de gérer les connexions entre PADS4 Scheduler et votre PADS4 Server. Cela inclut les méthodes de connexion, les identifiants de connexion, les paramètres de synchronisation et les options HTTP.

***

## Ouverture de la fenêtre Connexions au PADS4 Server

Pour gérer les connexions au serveur :

1. Accédez à l'onglet **Home** dans la barre de ruban.
2. Dans la section **Connexions**, cliquez sur le bouton **Gérer**.\
   Cela ouvre la fenêtre **Connexions au PADS4 Server**.

***

## Gestion des connexions

Depuis la fenêtre Connexions au PADS4 Server, vous pouvez :

* **Ajouter...** une nouvelle connexion au PADS4 Server
* **Modifier...** une connexion existante
* **Supprimer...** une connexion dont vous n'avez plus besoin

***

## Ajout ou modification d'une connexion au serveur

Que vous créiez une nouvelle connexion ou que vous modifiiez une connexion existante, la fenêtre de configuration est la même.

### Paramètres de l'onglet Connexion

#### **Nom de la connexion**

Un nom unique utilisé pour identifier cette connexion au serveur dans l'interface Scheduler.

#### **Option de démarrage de l'application**

Choisissez si vous souhaitez **vous connecter automatiquement** à ce serveur au démarrage de PADS4 Scheduler.

#### **Types de connexion au serveur**

| Option             | Description                                                                                                                                                             |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Automatique**    | Détecte automatiquement le serveur. Recommandé uniquement lorsqu'un seul PADS4 Server existe dans votre réseau local. Nécessite que le port UDP 3702 soit ouvert.       |
| **Cet ordinateur** | Suppose que le PADS4 Server s'exécute sur la même machine que le Scheduler. Préférable pour les configurations locales.                                                 |
| **Autre**          | Spécifiez manuellement le nom d'hôte, l'adresse IP ou l'entrée DNS du serveur. Optionnellement, cliquez sur **Rechercher...** pour détecter les serveurs sur le réseau. |

> **Options avancées** : Vous pouvez définir un numéro de port personnalisé si le serveur utilise des ports non standard. Le numéro de port doit correspondre à la configuration dans **PADS4 System Management**.

***

### Paramètres HTTP

Utilisé pour la communication web telle que la journalisation d'audit ou la synchronisation.

* **Automatique** : Paramètres par défaut basés sur la configuration du serveur.
* **Manuel** : Spécifiez un **Chemin d'hôte** et un **Port** personnalisés, utile lors de l'utilisation de routeurs, de proxies ou de la redirection de ports.

***

### Onglet Identifiants

Spécifiez les informations de connexion pour votre connexion au serveur.

* **Nom d'utilisateur** : Le nom d'utilisateur pour l'authentification au serveur.
* **Mot de passe** : (Facultatif) Mot de passe enregistré. S'il est laissé vide, l'utilisateur sera invité à le saisir.
* **Connexion automatique** : Tente automatiquement de se connecter lors de la connexion. En cas d'échec, l'utilisateur est invité à saisir ses identifiants.

> **Tip:** Combinez **Connexion automatique** avec **Démarrer automatiquement cette connexion** pour vous connecter de manière transparente au lancement de l'application.

***

### Onglet Synchronisation

Contrôlez la façon dont le contenu de présentation est synchronisé entre le Scheduler et le Server.

* **Activation de la synchronisation**:
  * **Automatique**: Synchronise lors des modifications
  * **Manuel**: Synchronise uniquement lorsque l'utilisateur en prend l'initiative
  * **Jamais**: Désactive la synchronisation

> Si le server et le Scheduler se trouvent sur la même machine, la synchronisation peut ne pas être nécessaire.

* **Dossier de stockage de la bibliothèque de contenu PADS4**:\
  Définissez un dossier local pour le stockage du contenu si l'emplacement par défaut ne convient pas.

* **Domaine**:\
  Utilisez ce champ si le server héberge plusieurs domaines PADS4, ou si vous gérez plusieurs connexions au server et devez les garder séparées. Sinon, choisissez **Automatique**.

***

### Enregistrement de la configuration

Cliquez sur **OK** pour enregistrer vos paramètres de connexion et revenir à la fenêtre principale de l'application.
