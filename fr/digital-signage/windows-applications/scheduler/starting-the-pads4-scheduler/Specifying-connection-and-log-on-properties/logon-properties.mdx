Dans les **Connection Properties**, vous pouvez définir comment vous souhaitez **vous connecter à votre PADS4 Server**. Ces paramètres se trouvent dans l'**onglet Credentials** lors de l'ajout ou de la modification d'une connexion au serveur.

***

## Spécification des détails de connexion

Pour définir vos préférences de connexion :

1. Ouvrez la fenêtre **PADS4 Server Connections**.
2. Choisissez d'**Ajouter** ou de **Modifier** une connexion au serveur.
3. Naviguez jusqu'à l'onglet **Credentials**.

### Champs obligatoires :

* **User Name** :\
  Entrez le nom du compte utilisé pour accéder au PADS4 Server.

* **Password** (optionnel):\
  Vous pouvez laisser ce champ vide si vous préférez être invité à vous connecter à chaque connexion.

### Option de connexion automatique:

* Cochez **Connexion automatique** pour que l'application tente de vous connecter automatiquement lors de la connexion.
* Lorsque cette option est activée, vous devez également saisir votre **Mot de passe**.

> Si la connexion automatique échoue, vous serez invité à saisir vos identifiants manuellement.

***

## Travailler avec les Domains

> **Note:** Si votre PADS4 Server utilise des **Domains**, vous pouvez spécifier les détails de connexion spécifiques au domaine dans la section **Domain** des paramètres de connexion.

Ceci est considéré comme une **configuration avancée**, et des conseils supplémentaires sont disponibles dans le **PADS4 Installation Manual**.

Dans la plupart des scénarios, laisser le **Domain setting to Automatic** sera suffisant.
