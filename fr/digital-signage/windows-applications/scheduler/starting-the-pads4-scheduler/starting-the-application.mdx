**PADS4 Scheduler** est l'application destinée aux utilisateurs finaux pour planifier et gérer les présentations dans votre environnement PADS4.

***

## Comment lancer PADS4 Scheduler

Vous pouvez démarrer l'application en utilisant l'une des méthodes suivantes :

### Option 1 : Via le menu Démarrer

1. Cliquez sur le bouton **Démarrer** dans la barre des tâches Windows.
2. Accédez à **Tous les programmes** ou **PADS4** dans la liste des programmes.
3. Cliquez sur **Scheduler** pour lancer l'application.

### Option 2 : Via l'Explorateur de fichiers

1. Ouvrez **l'Explorateur Windows**.
2. Accédez au dossier où les **PADS4 Scheduler** fichiers du programme sont installés.
3. Double-cliquez sur le fichier nommé :
   ```
   PADSScheduler.exe
   ```

Une fois exécutée, l'application commencera à se charger.

***

## Premier démarrage ou problèmes de connexion

> **Note:** Si tous les paramètres de configuration sont corrects, PADS4 Scheduler se connectera automatiquement au **logiciel PADS4 Server** au lancement.

Si vous démarrez l'application pour la **première fois**, ou si des paramètres de configuration sont incorrects (par exemple, une adresse de serveur ou des identifiants invalides), le système affichera la **fenêtre de gestion des connexions**.

Cette fenêtre vous permet de :

* Saisir l'adresse du serveur et le port corrects
* Tester la connexion au PADS4 Server
* Enregistrer votre configuration

Une fois la connexion établie avec succès, l'interface principale de l'application sera lancée.
