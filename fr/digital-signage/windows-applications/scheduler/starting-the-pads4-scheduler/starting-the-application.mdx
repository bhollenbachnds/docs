**PADS4 Scheduler** est l'application utilisateur finale utilisée pour planifier et gérer les présentations dans votre environnement PADS4.

***

## Comment Lancer PADS4 Scheduler

Vous pouvez démarrer l'application en utilisant l'une des méthodes suivantes :

### Option 1 : Via le Menu Démarrer

1. Cliquez sur le bouton **Démarrer** dans la barre des tâches Windows.
2. Naviguez vers **Tous les Programmes** ou **PADS4** dans la liste des programmes.
3. Cliquez sur **Scheduler** pour lancer l'application.

### Option 2 : Via l'Explorateur de Fichiers

1. Ouvrez **l'Explorateur Windows**.
2. Naviguez vers le dossier où les fichiers du programme **PADS4 Scheduler** sont installés.
3. Double-cliquez sur le fichier nommé :
   ```
   PADSScheduler.exe
   ```

Une fois exécuté, l'application commencera à se charger.

***

## Premier Démarrage ou Problèmes de Connexion

> **Note:** Si tous les paramètres de configuration sont corrects, PADS4 Scheduler se connectera automatiquement au **logiciel PADS4 Server** au démarrage.

Si vous démarrez l'application pour la **première fois**, ou si des paramètres de configuration sont incorrects (par exemple, adresse de serveur ou identifiants invalides), le système affichera la **fenêtre de Gestion de Connexion**.

Cette fenêtre vous permet de :

* Saisir l'adresse et le port corrects du serveur
* Tester la connexion au PADS4 Server
* Enregistrer votre configuration

Une fois connecté avec succès, l'interface principale de l'application sera lancée.
