L'application PADS4 Agent vous permet d'ouvrir plusieurs fenêtres dans la zone d'information simultanément. De cette façon, vous pouvez rapidement basculer entre les informations ou visualiser les informations côte à côte.

 

#### **Pour ouvrir une fenêtre supplémentaire**

1. Cliquez sur **New Window** dans la section **Windows** du menu **View**.

   ![EN Agent Ribbon Newwindow Pn](/images/digital-signage/windows-applications/agent/advanced-features/EN-Agent-Ribbon-Newwindow.png)

#### **Pour basculer entre vos fenêtres**

1. Cliquez sur **Switch Windows** dans la section **Windows** du menu **View**.
2. Dans la liste des fenêtres, sélectionnez la fenêtre vers laquelle vous souhaitez basculer.

   ![EN Agent Ribbon Switchwindows Pn](/images/digital-signage/windows-applications/agent/advanced-features/EN-Agent-Ribbon-Switchwindows.png)

 

#### **Organisation de vos fenêtres**

Après avoir ouvert plusieurs fenêtres, vous pouvez éventuellement organiser vos fenêtres en groupes d'onglets horizontaux ou verticaux. Les groupes d'onglets vous permettent de visualiser vos fenêtres côte à côte.

<Tip>
  Avoir plusieurs fenêtres ouvertes côte à côte est très utile lorsque vous souhaitez comparer des informations entre les PADS4 Servers ou Viewers.
</Tip>

 

#### **Pour placer une fenêtre dans un nouveau groupe d'onglets**

1. Cliquez sur **New Tab Group** dans la section **Windows** du menu **View**.
2. Cliquez sur **New Horizontal Tab Group** ou **New Vertical Tab Group**.

   ![EN Agent Ribbon Newtabgroup Pn](/images/digital-signage/windows-applications/agent/advanced-features/EN-Agent-Ribbon-Newtabgroup.png)

#### **Pour déplacer une fenêtre vers un autre groupe d'onglets**

1. Cliquez sur **Move to** dans la **Windows** section du **View** menu.
2. Cliquez sur **Move to Previous Tab Group** ou **Move to Next Tab Group**.

   ![EN Agent Movetabgroup Pn](/images/digital-signage/windows-applications/agent/advanced-features/EN-Agent-Movetabgroup.png)

<Tip>
  Les fonctions pour organiser vos fenêtres d'information peuvent également être appelées en faisant un clic droit sur la zone d'onglet d'une fenêtre d'information et en cliquant sur l'un des éléments du menu contextuel.

  Pour fermer toutes les fenêtres sauf celle sur laquelle vous travaillez, faites un clic droit sur la zone d'onglet de votre fenêtre actuelle et sélectionnez **Close All But This** dans le menu contextuel.
</Tip>

<Tip>
  Lorsque vous souhaitez que l'application ouvre automatiquement des fenêtres supplémentaires dans certaines situations, vous pouvez ajuster les [Windows options](https://pads4documentation.blob.core.windows.net/agent/Content/Tools/Options/View%20options.htm) dans la fenêtre d'options.
</Tip>
