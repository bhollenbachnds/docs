L'application PADS4 Agent vous permet d'ouvrir plusieurs fenêtres simultanément dans la zone d'information. Vous pouvez ainsi basculer rapidement entre les informations ou les afficher côte à côte.

 

#### **Pour ouvrir une fenêtre supplémentaire**

1. Cliquez sur **New Window** dans la **Windows** section du **View** menu.

   ![EN Agent Ribbon Newwindow Pn](/images/digital-signage/windows-applications/agent/advanced-features/EN-Agent-Ribbon-Newwindow.png)

#### **Pour basculer entre vos fenêtres**

1. Cliquez sur **Switch Windows** dans la **Windows** section du **View** menu.
2. Dans la liste des fenêtres, sélectionnez la fenêtre vers laquelle vous souhaitez basculer.

   ![EN Agent Ribbon Switchwindows Pn](/images/digital-signage/windows-applications/agent/advanced-features/EN-Agent-Ribbon-Switchwindows.png)

 

#### **Organiser vos fenêtres**

Après avoir ouvert plusieurs fenêtres, vous pouvez éventuellement les organiser en groupes d'onglets horizontaux ou verticaux. Les groupes d'onglets vous permettent d'afficher vos fenêtres côte à côte.

<Tip>
  Avoir plusieurs fenêtres ouvertes côte à côte est très utile lorsque vous souhaitez comparer des informations entre des PADS4 Servers ou des Viewers.
</Tip>

 

#### **Pour placer une fenêtre dans un nouveau groupe d'onglets**

1. Cliquez sur **New Tab Group** dans la **Windows** section du **View** menu.
2. Cliquez sur **New Horizontal Tab Group** ou **New Vertical Tab Group**.

   ![EN Agent Ribbon Newtabgroup Pn](/images/digital-signage/windows-applications/agent/advanced-features/EN-Agent-Ribbon-Newtabgroup.png)

#### **Pour déplacer une fenêtre vers un autre groupe d'onglets**

1. Cliquez sur **Move to** dans la **Windows** section du **View** menu.
2. Cliquez sur **Move to Previous Tab Group** ou **Move to Next Tab Group**.

   ![EN Agent Movetabgroup Pn](/images/digital-signage/windows-applications/agent/advanced-features/EN-Agent-Movetabgroup.png)

<Tip>
  Les fonctions d'organisation de vos fenêtres d'information peuvent également être appelées en faisant un clic droit sur la zone d'onglets d'une fenêtre d'information et en cliquant sur l'un des éléments du menu contextuel.

  Pour fermer toutes les fenêtres sauf celle sur laquelle vous travaillez, faites un clic droit sur la zone d'onglets de votre fenêtre actuelle et sélectionnez **Close All But This** dans le menu contextuel.
</Tip>

<Tip>
  Lorsque vous souhaitez que l'application ouvre automatiquement des fenêtres supplémentaires dans certaines situations, vous pouvez ajuster les [Windows options](https://pads4documentation.blob.core.windows.net/agent/Content/Tools/Options/View%20options.htm) dans la fenêtre des options.
</Tip>
