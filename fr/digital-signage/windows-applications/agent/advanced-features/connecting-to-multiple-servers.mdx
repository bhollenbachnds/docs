Avec l'application PADS4 Agent, vous pouvez être connecté simultanément à plusieurs PADS4 Servers. Cela vous permet de surveiller et de gérer à distance plusieurs installations PADS4 en même temps ou de visualiser les informations de plusieurs installations PADS4 côte à côte.

<Tip>
  Veuillez noter qu'il existe une différence entre la connexion à plusieurs serveurs et le travail avec une installation PADS4 composée de plusieurs serveurs. La méthode de travail décrite dans cette rubrique est destinée à travailler avec plusieurs PADS4 Servers de <u>différentes</u> installations PADS4. Pour travailler avec plusieurs PADS4 Servers au sein de la <u>même</u> installation PADS4, il est préférable de suivre la procédure décrite dans [Travailler avec des sous-serveurs](/fr/digital-signage/windows-applications/agent/advanced-features/working-with-sub-servers).
</Tip>

 

#### **Ajout de plusieurs PADS4 Servers**

Avant de pouvoir commencer à utiliser plusieurs PADS4 Servers, vous devez vous assurer que tous les PADS4 Servers concernés ont été définis dans le système. Pour plus d'informations sur l'ajout d'une connexion à un PADS4 Server, voir [Gérer vos connexions.](/fr/digital-signage/windows-applications/agent/starting-PADS4-Agent/managing-your-connections)

 

#### **Connexion à plusieurs PADS4 Servers**

Lors de l'utilisation de l'application PADS4 Agent, vous pouvez décider vous-même à quel(s) PADS4 Server(s) défini(s) vous souhaitez vous connecter. Pour chaque PADS4 Server auquel vous souhaitez vous connecter, vous devez suivre la procédure décrite dans [Connexion à votre PADS4 Server](/fr/digital-signage/windows-applications/agent/starting-PADS4-Agent/Connecting-to-your-PADS4-Server).

Lorsque vous avez spécifié pour une ou plusieurs de vos connexions PADS4 Server que vous souhaitez vous connecter automatiquement, PADS4 Agent se connectera automatiquement à ces PADS4 Servers au démarrage de l'application.

 

#### **Utilisation du volet de navigation avec plusieurs PADS4 Servers connectés**

Après vous être connecté avec succès à plusieurs PADS4 Servers, le bas du volet de navigation affichera une liste de tous vos PADS4 Servers connectés. Pour chacun de vos PADS4 Servers connectés, vous pouvez utiliser le volet de navigation pour sélectionner une catégorie d'informations et accéder aux nombreuses [fonctions serveur](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/server-functions) ou [fonctions viewer](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/viewer-functions) intégrées de l'application.

Pour sélectionner une catégorie d'informations d'un PADS4 Server :

1. Cliquez sur le PADS4 Server en bas du volet de navigation.

   ![EN Agent Multipleservers Navigationpane Pn](/images/digital-signage/windows-applications/agent/advanced-features/EN-Agent-Multipleservers-Navigationpane.png)
2. Suivez la procédure de sélection d'une catégorie d'informations, telle que décrite dans [utilisation du volet de navigation](/fr/digital-signage/windows-applications/agent/main-window/navigation-pane).

<Tip>
  Pour tirer le meilleur parti des fonctionnalités de travail avec plusieurs PADS4 Servers, vous pouvez les utiliser en combinaison avec [le travail avec plusieurs fenêtres](/fr/digital-signage/windows-applications/scheduler/advanced-features/working-with-mulitple-windows). Cela vous permet de comparer les mêmes catégories d'informations de deux PADS4 Servers ou plus côte à côte.
</Tip>

Lorsque vous travaillez avec plusieurs PADS4 Servers, le titre du volet de navigation et les informations dans la barre d'état affichent toujours le nom de votre serveur actuel. Ainsi, vous savez toujours à quel serveur appartient la fenêtre d'informations active affichée.

 

#### **Déconnexion des PADS4 Servers**

Pour déconnecter l'un de vos PADS4 Servers connectés :

1. Cliquez sur **Disconnect** dans le menu **Home**.

   ![EN Agent Ribbon Disconnect Pn](/images/digital-signage/windows-applications/agent/advanced-features/EN-Agent-Ribbon-Disconnect.png)
2. Dans la liste déroulante des serveurs connectés, sélectionnez le serveur à déconnecter.

Après cela, vous serez automatiquement déconnecté du serveur sélectionné.

 

Pour déconnecter tous les PADS4 Servers connectés en une seule fois, cliquez sur **Disconnect All** dans le menu **Home**.
