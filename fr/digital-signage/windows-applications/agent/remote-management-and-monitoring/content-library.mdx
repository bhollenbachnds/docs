Avec les fonctions de bibliothèque de contenu, vous pouvez parcourir la bibliothèque de contenu de votre PADS4 Server ou PADS4 Viewer.

 

#### **Pour récupérer le contenu de la bibliothèque de contenu de votre PADS4 Server**

1. Cliquez sur l'onglet **Server** dans le volet de navigation.

   ![EN Agent Navigationpane Servertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Servertab.png)
2. Cliquez sur **Content library** dans la liste des catégories d'informations de serveur disponibles.

Après cela, le système récupérera automatiquement les informations demandées depuis le PADS4 Server connecté et affichera les résultats dans la fenêtre de bibliothèque de contenu. Si votre installation PADS4 inclut des sous-serveurs, vous devez d'abord sélectionner un serveur avant de sélectionner la catégorie d'informations.

 

#### **Pour récupérer le contenu de la bibliothèque de contenu d'un PADS4 Viewer**

1. Cliquez sur l'onglet **Viewer** dans le volet de navigation.

   ![EN Agent Navigationpane Viewertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Viewertab.png)
2. Sélectionnez un Viewer dans la liste des destinations disponibles.
3. Cliquez sur **Content library** dans la liste des catégories d'informations de viewer disponibles.

Après cela, le système se connectera automatiquement au PADS4 Server connecté pour récupérer les informations demandées depuis le Viewer sélectionné et affichera les résultats dans la fenêtre de bibliothèque de contenu.

<Warning>
  L'application PADS4 ne pourra commencer à récupérer les informations de viewer demandées que lorsque vous aurez sélectionné <u>à la fois</u> un Viewer et une Catégorie dans le volet de navigation.
</Warning>

 

#### **Fenêtre de bibliothèque de contenu**

La fenêtre de bibliothèque de contenu affiche le contenu de la bibliothèque de contenu PADS4 sur votre serveur ou viewer et vous permet de la parcourir en lecture seule. Cette information est destinée particulièrement à vérifier si le contenu de votre bibliothèque de contenu est correct et si tous les fichiers de la bibliothèque de contenu ont été synchronisés.

![EN Agent Contentlibrary Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Contentlibrary.png)

<Tip>
  Au bas de la fenêtre de bibliothèque de contenu, vous pouvez voir le nombre de fichiers, le nombre de dossiers et la taille totale des fichiers de votre bibliothèque de contenu. En comparant ces informations entre deux bibliothèques de contenu, vous pouvez rapidement obtenir une indication si les deux bibliothèques de contenu sont identiques.
</Tip>
