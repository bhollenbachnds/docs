Avec les fonctions serveur de PADS4 Agent, vous pouvez récupérer toutes sortes d'informations depuis votre PADS4 Server connecté et effectuer des ajustements à distance.

Pour accéder aux fonctions serveur, vous devez [sélectionner une catégorie d'informations serveur](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm) dans le volet de navigation de la fenêtre principale. Si vous avez une installation PADS4 qui, en plus d'un serveur principal, comprend également des sous-serveurs, vous devez suivre la procédure décrite dans [travailler avec des sous-serveurs](https://pads4documentation.blob.core.windows.net/agent/Content/Advanced%20features/Working%20with%20sub%20servers.htm).

| [Bibliothèque de contenu](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/content-library)                                                           | Pour parcourir la bibliothèque de contenu de votre PADS4 Server.                                                                                       |
