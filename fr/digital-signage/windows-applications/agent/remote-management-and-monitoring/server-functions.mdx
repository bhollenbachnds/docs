Avec les fonctions serveur de PADS4 Agent, vous pouvez récupérer toutes sortes d'informations de votre serveur PADS4 connecté et effectuer des ajustements à distance.

Pour accéder aux fonctions du serveur, vous devez [sélectionner une catégorie d'informations du serveur](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm) dans le volet de navigation de la fenêtre principale. Si vous avez une installation PADS4 qui comprend, outre un serveur principal, également des sous-serveurs, vous devez suivre la procédure décrite dans [working with sub servers](https://pads4documentation.blob.core.windows.net/agent/Content/Advanced%20features/Working%20with%20sub%20servers.htm).

 

#### **Aperçu des fonctions du serveur**

| **Fonction**                                                                                                                                                                     | **Description**                                                                                                                                         |
| :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [Server connections](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/server-connections)                                                         | (Applicable uniquement aux installations PADS4 avec sous-serveurs) Pour afficher l'état de connexion de vos serveurs PADS4.                             |
| [Viewer connections](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/viewer-connections)                                                         | Pour afficher l'état de connexion de vos PADS4 Viewers.<br />Les Viewers peuvent être redémarrés, éteints ou allumés (wake-on-LAN).                     |
| [User connections](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/user-connections)                                                             | Pour voir quel utilisateur est connecté à quelles applications PADS4.<br />L'utilisateur peut être forcé à quitter l'application                        |
| [Scheduled tasks](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/scheduled-tasks)                                                               | Pour planifier certaines tâches pour votre serveur PADS4 ou vos PADS4 Viewers.                                                                          |
| [Monitoring & alerts](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Monitoring%20&%20alerts/Monitoring%20&%20alerts.htm) | Pour surveiller certains aspects de votre serveur PADS4 ou de vos PADS4 Viewers et recevoir automatiquement des notifications concernant les problèmes. |
| [Synchronization queue](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/synchronization-queue)                                                   | Pour afficher l'état actuel du processus de synchronisation du contenu.                                                                                 |
| [Content library](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/content-library)                                                               | Pour parcourir la bibliothèque de contenu de votre serveur PADS4.                                                                                       |
| [Data server](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/data-server)                                                                       | Pour inspecter les paramètres du fournisseur de données et les données en temps réel fournies par votre serveur PADS4.                                  |
| [PADS software](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/pads4-software)                                                                  | Pour récupérer les informations de version des fichiers PADS4 sur votre serveur PADS4 et pour mettre à jour le logiciel PADS4.                          |
| [System info](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/system-info)                                                                       | Pour récupérer des informations système détaillées de votre serveur PADS4.                                                                              |
| [Processes](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/processes)                                                                           | Pour afficher les processus en cours d'exécution sur votre serveur PADS4.<br />Les processus peuvent être démarrés, redémarrés ou arrêtés.              |
| [Services](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/services)                                                                             | Pour afficher les services Windows sur votre serveur PADS4.<br />Les services peuvent être démarrés, mis en pause ou redémarrés.                        |
| [Windows logs](/fr/digital-signage/windows-applications/agent/remote-management-and-monitoring/windows-logs)                                                                     | Pour afficher les journaux d'événements Windows de votre serveur PADS4.                                                                                 |
| [Audit Logs](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Audit%20logs.htm)                                             | Pour afficher les journaux d'audit enregistrés                                                                                                          |
| [Proof of play](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Proof%20of%20Play.htm)                                     | Pour afficher les entrées du journal de lecture qui sont stockées dans les bases de données de preuve de lecture sur votre serveur PADS4.               |
| [Viewer variables](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Viewer%20Variables.htm)                                 | Pour afficher et modifier les valeurs des variables Viewer pour chaque Viewer.                                                                          |
| [Television Channels](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Television%20channels.htm)                           | Pour inspecter et modifier les chaînes de télévision.                                                                                                   |
