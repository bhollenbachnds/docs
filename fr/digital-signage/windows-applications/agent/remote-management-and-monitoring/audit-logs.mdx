Avec les fonctions de journal d'audit, vous pouvez consulter les journaux d'audit qui sont stockés sur votre PADS4 Server.

Avec les journaux d'audit, l'utilisateur peut, par exemple, identifier le comportement de connexion/déconnexion des utilisateurs. En outre, il est possible de savoir quel utilisateur a programmé une certaine présentation ou modifié certains paramètres. Si un utilisateur tente d'utiliser une application non autorisée, cette activité inappropriée est enregistrée dans la piste d'audit.

 

#### **Pour consulter les journaux d'audit stockés sur votre PADS4 Server**

1. Cliquez sur l'onglet **Server** dans le volet de navigation.

   ![EN Agent Navigationpane Servertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Servertab.png)
2. Cliquez sur **Audit Logs** dans la liste des catégories d'informations de serveur disponibles.

Après cela, le système récupérera automatiquement les informations demandées depuis le PADS4 Server connecté et affichera les résultats dans la fenêtre du journal d'audit. Par défaut, la période sélectionnée commence le 1er du mois en cours.

<Warning>
  Par défaut, les journaux d'audit sont désactivés. Utilisez l'application System Management pour modifier les paramètres du serveur afin de permettre l'enregistrement des journaux d'audit.
</Warning>

 

#### **Fenêtre des journaux d'audit**

La fenêtre Audit Logs affiche les entrées du journal d'audit qui ont été stockées sur votre PADS4 Server.

![EN Agent Audit Window Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-AuditWindow.png)

Dans la fenêtre des journaux d'audit de votre PADS4 Server, vous pouvez consulter toutes les entrées de journal qui ont été enregistrées sur votre PADS4 Server.

La **période** peut être spécifiée. En outre, il est possible de **filtrer les données** en cliquant sur l'icône de filtre dans l'en-tête de colonne. Cliquez sur **Export View** pour exporter les données, y compris les filtres définis.

 

**Timestamp**\
Date et heure auxquelles l'action a été enregistrée.

**Username**\
Quel utilisateur a effectué cette action.

**Application**\
Quelle application a été utilisée par cet utilisateur.

**Description**\
Une description qui décrit l'action effectuée par l'utilisateur. ("Updated Schedule x", "Updated Server Settings", "Created new Schedule y")

**Details**\
Lorsque les modifications (mise à jour) affectent un ou plusieurs changements de propriété (exemple "Priorité à Élevée", ou "DateDébut à 01/01/2017"), ceux-ci seront enregistrés dans la colonne détails.\
Chaque paire Propriété et nouvelle valeur est séparée par un saut de ligne.

**Type d'utilisateur**\
Utilisateur PADS ou LDAP de l'Active Directory.

**Nom d'affichage**\
Nom d'utilisateur saisi dans l'application System management pour identifier un utilisateur ou récupéré par LDAP.

**Type d'événement**\
Type d'action effectuée.

| Connexion     | L'utilisateur essaie de se connecter                                               |
| :------------ | :--------------------------------------------------------------------------------- |
| Déconnexion   | L'utilisateur essaie de se déconnecter                                             |
| Créer         | L'utilisateur crée une entrée (l'entrée est spécifique à l'action effectuée)       |
| Mettre à jour | L'utilisateur met à jour une entrée (l'entrée est spécifique à l'action effectuée) |
| Supprimer     | L'utilisateur supprime une entrée (l'entrée est spécifique à l'action effectuée)   |
| Exécuter      | L'utilisateur exécute une tâche Agent                                              |

**Adresse IP**\
Adresse IP à partir de laquelle l'utilisateur a démarré l'application.\
La colonne contient \[local] lorsque l'application s'exécute sur le même ordinateur que le serveur PADS4.

**Nom de l'ordinateur**\
Nom de l'ordinateur sur lequel l'application s'exécute.

**ID historique**\
ID interne pouvant être utilisé pour regrouper afin que les actions basées sur la même destination puissent être examinées (par exemple, modification de planning).

**Nom du serveur**\
Sur quel serveur cette action a été effectuée.

 

#### **Période**

Spécifiez la période à partir de laquelle les données doivent être récupérées. Lorsqu'aucune donnée n'est disponible, un message s'affichera.

 

#### **Exportation des informations des journaux d'audit**

Pour traiter les informations des journaux d'audit dans une application externe (comme Microsoft Excel), les informations peuvent être exportées dans différents formats. Cela inclut XML et le format Microsoft Excel.

Pour exporter les informations des journaux d'audit :

1. Cliquez sur **Exporter** dans le menu **Accueil**.
2. Saisissez le **Début** et la **Fin** de la période pour laquelle vous souhaitez exporter les informations de preuve de diffusion.

   ![EN Agent Audit Export Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-AuditExport.png)
3. Sélectionnez le type de sortie, **XML** ou **Excel**.
4. Cliquez sur **Parcourir** pour spécifier le **Nom du fichier** dans lequel vous souhaitez enregistrer les résultats de l'exportation.
5. Dans la boîte de dialogue Enregistrer qui apparaît, spécifiez le nom du fichier et sélectionnez le format de fichier dans lequel vous souhaitez que les informations soient enregistrées, puis cliquez sur **Enregistrer**.
6. Cliquez sur **Exporter** pour commencer à enregistrer toutes les informations de preuve de diffusion pour la période spécifiée dans le fichier spécifié.

 

#### **Quelles informations utilisateur sont enregistrées ?**

* Tentatives de connexion / déconnexion pour chaque application PADS4
* Actions CRUD (Créer/Lire/Mettre à jour/Supprimer) du planificateur pour la programmation des présentations dans l'application PADS4 Scheduler\
  Les mises à jour sont enregistrées en détail. (niveau de propriété, quels sont les changements entre l'ancienne et la nouvelle situation)
* Actions CRUD (Créer/Lire/Mettre à jour/Supprimer) de Messenger pour la programmation des présentations dans l'application PADS4 Messenger\
  Les mises à jour sont enregistrées en détail. (niveau de propriété, quels sont les changements entre l'ancienne et la nouvelle situation)
* Récupération des informations du serveur par l'Agent dans PADS4 Agent\
  Connexions Viewer, Connexions utilisateur, Tâches planifiées, Surveillance et alertes, File d'attente de synchronisation, Bibliothèque de contenu, Serveur de données, Logiciel PADS4, Informations système, Processus, Services, Journaux Windows, Variables Viewer, Chaînes de télévision.
* Récupération des informations Viewer par l'Agent dans PADS4 Agent\
  Capture d'écran, Journal Viewer, Utilisation de la mémoire, Bibliothèque de contenu, Journal de synchronisation, Présentations planifiées, Fournisseurs de données, Paramètres Viewer, Logiciel PADS4, Add-Ons, Tâches planifiées, Surveillance et alertes, Informations système, Processus, Services, Journaux Windows, Polices, Preuve de diffusion, télévision.
* Actions de connexion Viewer de l'Agent (serveur) : **redémarrer/réinitialiser/arrêter/wake-on-lan**
* Actions de connexion utilisateur de l'Agent (serveur) : **fermer la connexion**
* Actions de tâches planifiées de l'Agent (serveur) : **Nouveau/Modifier/Dupliquer/Supprimer/Activer/Désactiver**
* Actions de surveillance et d'alerte de l'Agent (serveur) : **Ajouter/Modifier/Dupliquer/Supprimer/Activer/Désactiver/En attente/Résolu**
* Action logiciel PADS4 de l'Agent (serveur) : **Mettre à jour**
* Actions de processus de l'Agent (serveur) : **Démarrer/Arrêter/Redémarrer**
* Actions de services de l'Agent (serveur) : **Démarrer/Mettre en pause/Redémarrer**
* Actions de journaux Windows de l'Agent (serveur) : **Effacer le journal**
* Actions de journal d'audit de l'Agent (serveur) : **Exporter**
* Actions de preuve de diffusion de l'Agent (serveur) : **Exporter**
* Actions de variables Viewer de l'Agent (serveur) : **Modifier/Réinitialiser/Dupliquer**
* Actions de chaînes de télévision de l'Agent (serveur) : **Ajouter/Modifier/Supprimer/Fusionner**
* Actions de journal Viewer de l'Agent (viewer) : **Effacer le journal/Enregistrer la boîte noire**
* Actions d'utilisation de la mémoire de l'Agent (viewer) :**Effacer le journal/Enregistrer la boîte noire** 
* Actions de journal de synchronisation de l'Agent (viewer) : **Effacer le journal/Enregistrer la boîte noire**
* Actions de paramètres Viewer de l'Agent (viewer) : **Modifier**
* Actions logiciel PADS4 de l'Agent (viewer) : **Mettre à jour**
* Actions Add-Ons de l'Agent (viewer) : **Assistant d'installation**
* Actions de processus de l'Agent (viewer) : **Démarrer/Redémarrer/Arrêter**
* Actions de services de l'Agent (viewer) : **Démarrer/Mettre en pause/Redémarrer**
* Actions de journaux Windows de l'Agent (viewer) : **Effacer le journal**
* Actions de polices de l'Agent (viewer) : **Installer/Installer à partir d'un fichier/Désinstaller**
* Actions de télévision de l'Agent (viewer) : **Analyse automatique des chaînes/copier vers les destinations/créer un nom de chaîne**

* Actions de gestion des licences du système : **Assistant d'attribution de licence/Retrait de licence**

</Tip>
