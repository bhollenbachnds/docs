Grâce aux fonctions de journal de piste d'audit, vous pouvez consulter les journaux d'audit stockés sur votre PADS4 Server.

Grâce aux journaux d'audit, l'utilisateur peut, par exemple, identifier le comportement de connexion/déconnexion des utilisateurs. De plus, il est possible de savoir quel utilisateur a planifié une certaine présentation ou modifié certains paramètres. Si un utilisateur tente d'utiliser une application non autorisée, cette activité inappropriée est enregistrée dans la piste d'audit.

 

#### **Pour consulter les journaux d'audit stockés sur votre PADS4 Server**

1. Cliquez sur l'onglet **Server** dans le volet de navigation.

   ![EN Agent Navigationpane Servertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Servertab.png)
2. Cliquez sur **Audit Logs** dans la liste des catégories d'informations serveur disponibles.

Après cela, le système récupérera automatiquement les informations demandées depuis le PADS4 Server connecté et affichera les résultats dans la fenêtre du journal d'audit. Par défaut, la période sélectionnée correspond au mois en cours à partir du 1er.

<Warning>
  Par défaut, les journaux d'audit sont désactivés. Utilisez l'application System Management pour modifier les paramètres du serveur afin de permettre l'enregistrement des journaux d'audit.
</Warning>

 

#### **Fenêtre des journaux d'audit**

La fenêtre Audit Logs affiche les entrées de journal d'audit qui ont été stockées sur votre PADS4 Server.

![EN Agent Audit Window Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-AuditWindow.png)

Dans la fenêtre des journaux d'audit de votre PADS4 Server, vous pouvez consulter toutes les entrées de journal qui ont été enregistrées sur votre PADS4 Server.

La **période** peut être spécifiée. De plus, il est possible de **filtrer les données** en cliquant sur l'icône de filtre dans l'en-tête de colonne. Cliquez sur **Export View** pour exporter les données, y compris les filtres définis.

 

**Horodatage**\
Date et heure auxquelles l'action a été enregistrée.

**Nom d'utilisateur**\
Quel utilisateur a effectué cette action.

**Application**\
Quelle application a été utilisée par cet utilisateur.

**Description**\
Une description qui décrit l'action effectuée par l'utilisateur. ("Mise à jour du Schedule x", "Mise à jour des paramètres du Server", "Création d'un nouveau Schedule y")

**Détails**\
Lorsque les modifications (mise à jour) affectent un ou plusieurs changements de propriété (par exemple "Priority à High", ou "StartDate au 01/01/2017"), ceux-ci seront enregistrés dans la colonne des détails.\
Chaque paire Propriété et nouvelle valeur est séparée par un saut de ligne.

**Usertype**\
Utilisateur PADS ou LDAP provenant d'Active Directory.

**Displayname**\
Nom d'utilisateur saisi dans l'application de gestion du système pour identifier un utilisateur ou récupéré par LDAP.

**EventType**\
Type d'action effectuée.

| Logon   | L'utilisateur tente de se connecter                                                |
| :------ | :--------------------------------------------------------------------------------- |
| Logoff  | L'utilisateur tente de se déconnecter                                              |
| Create  | L'utilisateur crée une entrée (l'entrée est spécifique à l'action effectuée)       |
| Update  | L'utilisateur met à jour une entrée (l'entrée est spécifique à l'action effectuée) |
| Delete  | L'utilisateur supprime une entrée (l'entrée est spécifique à l'action effectuée)   |
| Execute | L'utilisateur exécute une tâche Agent                                              |

**IPAddress**\
Adresse IP depuis laquelle l'utilisateur a démarré l'application.\
La colonne contient \[local] lorsque l'application s'exécute sur le même ordinateur que le PADS4 Server.

**Computername**\
Nom de l'ordinateur sur lequel l'application s'exécute.

**HistoricalID**\
L'ID interne peut être utilisé pour regrouper les actions basées sur la même destination afin de les analyser (par exemple, modifier un schedule).

**Servername**\
Sur quel server cette action a été effectuée.

 

#### **Période**

Spécifiez la période à partir de laquelle les données doivent être récupérées. Lorsqu'il n'y a pas de données disponibles, un message s'affichera.

 

#### **Exportation des informations des journaux d'audit**

Pour traiter les informations des journaux d'audit dans une application externe (comme Microsoft Excel), les informations peuvent être exportées dans différents formats. Cela inclut XML et le format Microsoft Excel.

Pour exporter les informations des journaux d'audit :

1. Cliquez sur **Export** dans le **Home** menu.
2. Saisissez le **Start** et **End** de la période pour laquelle vous souhaitez exporter les informations de preuve de diffusion.

   ![EN Agent Audit Export Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-AuditExport.png)
3. Sélectionnez le type de sortie, **XML** ou **Excel**.
4. Cliquez sur **Browse** pour spécifier le **File Name** du fichier dans lequel vous souhaitez enregistrer les résultats de l'exportation.
5. Dans la boîte de dialogue Enregistrer qui apparaît, spécifiez le nom du fichier et sélectionnez le format de fichier dans lequel vous souhaitez que les informations soient enregistrées, puis cliquez sur **Save**.
6. Cliquez sur **Export** pour commencer à enregistrer toutes les informations de preuve de diffusion pour la période spécifiée dans le fichier spécifié.

 

#### **Quelles informations utilisateur sont enregistrées ?**

* Tentatives de Logon / Logoff pour chaque application PADS4
* Actions CRUD du Scheduler (Create/Read/Update/Delete) pour la planification des présentations dans l'application PADS4 Scheduler\
  Les mises à jour sont enregistrées en détail. (niveau propriété, quelles sont les modifications entre l'ancienne et la nouvelle situation)
* Actions CRUD du Messenger (Create/Read/Update/Delete) pour la planification des présentations dans l'application PADS4 Messenger\
  Les mises à jour sont enregistrées en détail. (niveau propriété, quelles sont les modifications entre l'ancienne et la nouvelle situation)
* Récupération des informations du Server par l'Agent dans le PADS4 Agent\
  Viewer Connections, User Connections, Scheduled Tasks, Monitoring & Alerts, Synchronization Queue, Content Library, Data Server, PADS4 Software, System Info, Processes, Services, Windows logs, Viewer Variables, Television channels.
* Récupération des informations du Viewer par l'Agent dans le PADS4 Agent\
  Screenshot, Viewer log, Memory usage, Content library, Synchronization log, Scheduled presentations, Data providers, Viewer settings, PADS4 software, Add-Ons, Scheduled tasks, Monitoring & alerts, System info, Processes, Services, Windows logs, Fonts, Proof of play, television.
* Actions de connexion du Viewer par l'Agent (server) : **restart/reboot/shutdown/wake-on-lan**
* Actions de connexion utilisateur par l'Agent (server) : **close connection**
* Actions de tâches planifiées par l'Agent (server) : **New/Edit/Duplicate/Delete/Enable/Disable**
* Actions de Monitoring & Alert par l'Agent (server) : **Add/Edit/Duplicate/Delete/Enable/Disable/Pending/Resolved**
* Action du logiciel PADS4 par l'Agent (server) : **Update**
* Actions de Processes par l'Agent (server) : **Start/Stop/Restart**
* Actions de Services par l'Agent (server) : **Start/Pauze/Restart**
* Actions des journaux Windows par l'Agent (server) : **Clear log**
* Actions du journal d'audit par l'Agent (server) : **Export**
* Actions de preuve de diffusion par l'Agent (server) : **Export**
* Actions des variables du Viewer par l'Agent (server) : **Edit/Reset/Duplicate**
* Actions des chaînes de télévision par l'Agent (server) : **Add/Edit/Delete/Merge**
* Actions du journal du Viewer par l'Agent (viewer) : **Clear log/Save blackbox**
* Actions d'utilisation de la mémoire par l'Agent (viewer) :**Clear log/Save blackbox** 
* Actions du journal de synchronisation par l'Agent (viewer) : **Clear log/Save blackbox**
* Actions des paramètres du Viewer par l'Agent (viewer) : **Edit**
* Actions du logiciel PADS4 par l'Agent (viewer) : **Update**
* Actions des Add-Ons par l'Agent (viewer) : **Installation Wizard**
* Actions de Processes par l'Agent (viewer) : **Start/Restart/Stop**
* Actions de Services par l'Agent (viewer) : **Start/Pauze/Restart**
* Actions des journaux Windows par l'Agent (viewer) : **Clear log**
* Actions des polices par l'Agent (viewer) : **Install/Install from file/Uninstall**
* Actions de télévision par l'Agent (viewer) : **Auto channel scan/copy to destinations/create channel name**
* Actions de gestion du système - Destinations : **Ajouter/Modifier/Supprimer/Réactivation du Viewer**
* Actions de gestion du système - Utilisateurs : **Ajouter/Modifier/Supprimer/Réinitialiser le mot de passe**
* Actions de gestion du système - Groupes d'utilisateurs : **Nouveau/Modifier/Supprimer**
* Actions de gestion du système - Permissions : **Nouveau/Modifier/Supprimer**
* Actions de gestion du système - Priorités : **Ajouter/Modifier/Supprimer/Augmenter le niveau/Diminuer le niveau**
* Gestion du système - Catégories de contenu : **Nouveau/Modifier/Supprimer**
* Actions de gestion du système - Serveurs : **Nouveau sous-serveur/Nouveau serveur de sauvegarde/Modifier/Supprimer**
* Actions de gestion du système - Licences : **Attribuer une licence/Assistant de retrait de licence**
* Actions de gestion du système - Paramètres du serveur : **Modifier**
* Actions de gestion du système - Paramètres de télévision : **Ajouter/Modifier/Supprimer/Fusionner**
* Actions de gestion du système - Variables du Viewer : **Ajouter/Modifier/Supprimer**
* Content Organizer : actions sur les dossiers et fichiers de contenu : **Ajouter/Modifier/Supprimer/Déplacer**

 

<Tip>
  Lors de l'exportation, vous recevrez deux colonnes supplémentaires EventTypeID et ApplicationID qui contiendront la valeur numérique de EventType et Application.\
  \
  En outre, des métadonnées sont également stockées, contenant les informations suivantes :\
  Utilisateur, Période, Date de génération et éventuellement le filtre utilisé sur la grille.
</Tip>
