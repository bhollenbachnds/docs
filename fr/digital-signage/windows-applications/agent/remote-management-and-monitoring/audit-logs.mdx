Grâce aux fonctions de journal de piste d'audit, vous pouvez consulter les journaux d'audit stockés sur votre PADS4 Server.

Grâce aux journaux d'audit, l'utilisateur peut, par exemple, identifier le comportement de connexion/déconnexion des utilisateurs. De plus, il est possible de savoir quel utilisateur a planifié une certaine présentation ou modifié certains paramètres. Si un utilisateur tente d'utiliser une application non autorisée, cette activité inappropriée est enregistrée dans la piste d'audit.

 

#### **Pour consulter les journaux d'audit stockés sur votre PADS4 Server**

1. Cliquez sur l'onglet **Serveur** dans le volet de navigation.

   ![EN Agent Navigationpane Servertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Servertab.png)
2. Cliquez sur **Journaux d'audit** dans la liste des catégories d'informations serveur disponibles.

Après cela, le système récupérera automatiquement les informations demandées depuis le PADS4 Server connecté et affichera les résultats dans la fenêtre du journal d'audit. Par défaut, la période sélectionnée correspond au mois en cours à partir du 1er.

<Warning>
  Par défaut, les journaux d'audit sont désactivés. Utilisez l'application de gestion du système pour modifier les paramètres du serveur afin de permettre l'enregistrement des journaux d'audit.
</Warning>

 

#### **Fenêtre des journaux d'audit**

La fenêtre des journaux d'audit affiche les entrées de journal d'audit qui ont été stockées sur votre PADS4 Server.

![EN Agent Audit Window Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-AuditWindow.png)

Dans la fenêtre des journaux d'audit de votre PADS4 Server, vous pouvez consulter toutes les entrées de journal qui ont été enregistrées sur votre PADS4 Server.

La **période** peut être spécifiée. De plus, il est possible de **filtrer les données** en cliquant sur l'icône de filtre dans l'en-tête de colonne. Cliquez sur **Exporter la vue** pour exporter les données, y compris les filtres définis.

 

**Horodatage**\
Date et heure à laquelle l'action a été enregistrée.

**Nom d'utilisateur**\
Quel utilisateur a effectué cette action.

**Application**\
Quelle application a été utilisée par cet utilisateur.

**Description**\
Une description qui décrit l'action effectuée par l'utilisateur. ("Updated Schedule x", "Updated Server Settings", "Created new Schedule y")

**Détails**\
Lorsque les modifications (mise à jour) affectent un ou plusieurs changements de propriété (par exemple "Priority to High" ou "StartDate to 01/01/2017"), ceux-ci seront enregistrés dans la colonne des détails.\
Chaque paire Propriété et nouvelle valeur est séparée par un saut de ligne.

**Type d'utilisateur**\
Utilisateur PADS ou LDAP provenant d'Active Directory.

**Nom d'affichage**\
Nom d'utilisateur saisi dans l'application de gestion du système pour identifier un utilisateur ou récupéré par LDAP.

**Type d'événement**\
Type d'action effectuée.

| Connexion     | L'utilisateur tente de se connecter                                                |
| :------------ | :--------------------------------------------------------------------------------- |
| Déconnexion   | L'utilisateur tente de se déconnecter                                              |
| Créer         | L'utilisateur crée une entrée (l'entrée est spécifique à l'action effectuée)       |
| Mettre à jour | L'utilisateur met à jour une entrée (l'entrée est spécifique à l'action effectuée) |
| Supprimer     | L'utilisateur supprime une entrée (l'entrée est spécifique à l'action effectuée)   |
| Exécuter      | L'utilisateur exécute une tâche Agent                                              |

**Adresse IP**\
Adresse IP depuis laquelle l'utilisateur a démarré l'application.\
La colonne contient \[local] lorsque l'application s'exécute sur le même ordinateur que le serveur PADS4.

**Nom de l'ordinateur**\
Nom de l'ordinateur sur lequel l'application s'exécute.

**ID historique**\
L'ID interne peut être utilisé pour regrouper les actions basées sur la même destination afin de les analyser (par exemple, modifier un planning).

**Nom du serveur**\
Sur quel serveur cette action a-t-elle été effectuée.

 

#### **Période**

Spécifiez la période à partir de laquelle les données doivent être récupérées. Lorsqu'aucune donnée n'est disponible, un message s'affichera.

 

#### **Exportation des informations des journaux d'audit**

Pour traiter les informations des journaux d'audit dans une application externe (comme Microsoft Excel), les informations peuvent être exportées dans différents formats. Cela inclut XML et le format Microsoft Excel.

Pour exporter les informations des journaux d'audit :

1. Cliquez sur **Exporter** dans le **Accueil** menu.
2. Entrez le **Début** et la **Fin** de la période pour laquelle vous souhaitez exporter les informations de preuve de diffusion.

   ![EN Agent Audit Export Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-AuditExport.png)
3. Sélectionnez le type de sortie, **XML** ou **Excel**.
4. Cliquez sur **Parcourir** pour spécifier le **Nom du fichier** du fichier dans lequel vous souhaitez enregistrer les résultats de l'exportation.
5. Dans la boîte de dialogue Enregistrer qui apparaît, spécifiez le nom du fichier et sélectionnez le format de fichier dans lequel vous souhaitez que les informations soient enregistrées, puis cliquez sur **Enregistrer**.
6. Cliquez sur **Exporter** pour commencer à enregistrer toutes les informations de preuve de diffusion pour la période spécifiée dans le fichier spécifié.

 

#### **Quelles informations utilisateur sont enregistrées ?**

* Tentatives de connexion / déconnexion pour chaque application PADS4
* Actions CRUD (Créer/Lire/Mettre à jour/Supprimer) du Scheduler pour la planification des présentations dans l'application PADS4 Scheduler\
  Les mises à jour sont enregistrées en détail. (niveau propriété, quelles sont les modifications entre l'ancienne et la nouvelle situation)
* Actions CRUD (Créer/Lire/Mettre à jour/Supprimer) du Messenger pour la planification des présentations dans l'application PADS4 Messenger\
  Les mises à jour sont enregistrées en détail. (niveau propriété, quelles sont les modifications entre l'ancienne et la nouvelle situation)
* Récupération des informations serveur par l'Agent dans le PADS4 Agent\
  Connexions des Viewers, Connexions utilisateur, Tâches planifiées, Surveillance et alertes, File de synchronisation, Bibliothèque de contenu, Serveur de données, Logiciel PADS4, Informations système, Processus, Services, Journaux Windows, Variables du Viewer, Chaînes de télévision.
* Récupération des informations du Viewer par l'Agent dans le PADS4 Agent\
  Capture d'écran, Journal du Viewer, Utilisation de la mémoire, Bibliothèque de contenu, Journal de synchronisation, Présentations planifiées, Fournisseurs de données, Paramètres du Viewer, Logiciel PADS4, Add-Ons, Tâches planifiées, Surveillance et alertes, Informations système, Processus, Services, Journaux Windows, Polices, Preuve de diffusion, Télévision.
* Actions de connexion du Viewer par l'Agent (serveur) : **restart/reboot/shutdown/wake-on-lan**
* Actions de connexion utilisateur par l'Agent (serveur) : **fermer la connexion**
* Actions sur les tâches planifiées par l'Agent (serveur) : **Nouveau/Modifier/Dupliquer/Supprimer/Activer/Désactiver**
* Actions de surveillance et d'alerte par l'Agent (serveur) : **Ajouter/Modifier/Dupliquer/Supprimer/Activer/Désactiver/En attente/Résolu**
* Action sur le logiciel PADS4 par l'Agent (serveur) : **Mettre à jour**
* Actions sur les processus par l'Agent (serveur) : **Démarrer/Arrêter/Redémarrer**
* Actions sur les services par l'Agent (serveur) : **Démarrer/Mettre en pause/Redémarrer**
* Actions sur les journaux Windows par l'Agent (serveur) : **Effacer le journal**
* Actions sur le journal d'audit par l'Agent (serveur) : **Exporter**
* Actions sur la preuve de diffusion par l'Agent (serveur) : **Exporter**
* Actions sur les variables du Viewer par l'Agent (serveur) : **Modifier/Réinitialiser/Dupliquer**
* Actions sur les chaînes de télévision par l'Agent (serveur) : **Ajouter/Modifier/Supprimer/Fusionner**
* Actions sur le journal du Viewer par l'Agent (viewer) : **Effacer le journal/Enregistrer la boîte noire**
* Actions sur l'utilisation de la mémoire par l'Agent (viewer) :**Effacer le journal/Enregistrer la boîte noire** 
* Actions sur le journal de synchronisation par l'Agent (viewer) : **Effacer le journal/Enregistrer la boîte noire**
* Actions sur les paramètres du Viewer par l'Agent (viewer) : **Modifier**
* Actions sur le logiciel PADS4 par l'Agent (viewer) : **Mettre à jour**
* Actions sur les Add-Ons par l'Agent (viewer) : **Assistant d'installation**
* Actions sur les processus par l'Agent (viewer) : **Démarrer/Redémarrer/Arrêter**
* Actions sur les services par l'Agent (viewer) : **Démarrer/Mettre en pause/Redémarrer**
* Actions sur les journaux Windows par l'Agent (viewer) : **Effacer le journal**
* Actions sur les polices par l'Agent (viewer) : **Installer/Installer depuis un fichier/Désinstaller**
* Actions sur la télévision par l'Agent (viewer) : **Balayage automatique des chaînes/copier vers les destinations/créer un nom de chaîne**
* Actions de gestion du système - Destinations : **Ajouter/Modifier/Supprimer/Réactivation du Viewer**
* Actions de gestion du système - Utilisateurs : **Ajouter/Modifier/Supprimer/Réinitialiser le mot de passe**
* Actions de gestion du système - Groupes d'utilisateurs : **Nouveau/Modifier/Supprimer**
* Actions de gestion du système - Permissions : **Nouveau/Modifier/Supprimer**
* Actions de gestion du système - Priorités : **Ajouter/Modifier/Supprimer/Augmenter le niveau/Diminuer le niveau**
* Gestion du système - Catégories de contenu : **Nouveau/Modifier/Supprimer**
* Actions de gestion du système - Serveurs : **Nouveau serveur secondaire/Nouveau serveur de sauvegarde/Modifier/Supprimer**
* Actions de gestion du système - Licences : **Attribuer une licence/Assistant de retrait de licence**
* Actions de gestion du système - Paramètres du serveur : **Modifier**
* Actions de gestion du système - Paramètres de télévision : **Ajouter/Modifier/Supprimer/Fusionner**
* Actions de gestion du système - Variables du Viewer : **Ajouter/Modifier/Supprimer**
* Content Organizer : actions sur les dossiers et fichiers de contenu : **Ajouter/Modifier/Supprimer/Déplacer**

 

<Tip>
  Lors de l'exportation, vous recevrez deux colonnes supplémentaires EventTypeID et ApplicationID qui contiendront la valeur numérique de EventType et Application.\
  \
  En outre, des métadonnées sont également stockées, contenant les informations suivantes :\
  Utilisateur, Période, Date de génération et éventuellement le filtre utilisé sur la grille.
</Tip>
