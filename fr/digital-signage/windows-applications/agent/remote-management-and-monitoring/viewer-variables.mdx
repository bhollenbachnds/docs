Avec les fonctions de variables Viewer, vous pouvez afficher et modifier les variables Viewer actuellement disponibles dans le système PADS4.

 

#### **Pour afficher les informations sur les variables Viewer stockées sur votre PADS4 Server**

1. Cliquez sur l'onglet **Server** dans le volet de navigation.

   ![EN Agent Navigationpane Servertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Servertab.png)
2. Cliquez sur **Viewer variables** dans la liste des catégories d'informations serveur disponibles.

Après cela, le système récupérera automatiquement les informations demandées depuis le PADS4 Server connecté et affichera les résultats dans la fenêtre Viewer variables.

 

#### **Fenêtre Viewer variables**

La fenêtre Viewer variables affiche toutes les entrées qui ont été stockées sur votre PADS4 Server.\
Les Viewer variables sont créées à l'aide de l'application PADS4 System Management.

 

**Modifier**\
Modifier la valeur d'une Viewer variable. Cliquez sur le bouton **Edit** dans la barre de ruban.

![En Agent Viewervariablesdocedit Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/en_agent_viewervariablesdocedit.png)

\
Cliquez sur la colonne Valeur de la Viewer variable que vous souhaitez modifier manuellement.

**Réinitialiser**\
Cliquez sur le bouton Reset dans la barre de ruban pour réinitialiser les valeurs à la valeur par défaut pour ce Viewer.

 

**Dupliquer**\
Cliquez sur le bouton **Duplicate** dans la barre de ruban pour dupliquer cette valeur vers un ou plusieurs Viewers.\
Il vous sera demandé de spécifier les destinations.\
\
Cela vous permet de définir la valeur pour plusieurs Viewers en même temps.

 

**Exporter**\
Exportez toutes les Viewer variables vers Microsoft Excel. Lorsque vous souhaitez dupliquer des Viewer variables vers un autre système PADS4, ou lorsque vous souhaitez modifier les valeurs des Viewer variables dans Excel, ce qui permet des actions de copier/coller plus avancées, l'exportation crée une feuille Excel avec toutes les Viewer variables. Seules les cellules modifiables sont déverrouillées.

 

**Importer**\
Importez des Viewer variables depuis Microsoft Excel.

Lors de l'importation, une fenêtre d'aperçu s'affiche montrant les résultats après l'importation. Il existe deux options d'importation. L'une est un remplacement où toutes les Viewer variables sont remplacées, et l'autre est une fusion où les Viewer variables existantes sont mises à jour et les nouvelles Viewer variables sont ajoutées.

* Il est possible d'importer de nouvelles Viewer variables dans le document Excel
* Il n'est pas possible d'importer de nouveaux noms de Viewer dans le document Excel
* Les cellules non modifiables sont verrouillées
* Utilisez le modèle de document Excel issu de l'exportation
* Les documents Excel ne correspondant pas au modèle seront ignorés

 

<Warning>
  L'ajout/la suppression de Viewer variables s'effectue avec l'application PADS4 System Management.
</Warning>
