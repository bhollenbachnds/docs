Avec les fonctions de variables Viewer, vous pouvez consulter et modifier les variables Viewer actuellement disponibles dans le système PADS4.

 

#### **Pour consulter les informations des variables Viewer stockées sur votre serveur PADS4**

1. Cliquez sur l'onglet **Server** dans le volet de navigation.

   ![EN Agent Navigationpane Servertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Servertab.png)
2. Cliquez sur **Viewer variables** dans la liste des catégories d'informations de serveur disponibles.

Après cela, le système récupérera automatiquement les informations demandées depuis le serveur PADS4 connecté et affichera les résultats dans la fenêtre des variables Viewer.

 

#### **Fenêtre des variables Viewer**

La fenêtre des variables Viewer affiche toutes les entrées qui ont été stockées sur votre serveur PADS4.\
Les variables Viewer sont créées en utilisant l'application PADS4 System Management.

 

**Modifier**\
Modifier la valeur d'une variable Viewer. Cliquez sur le bouton **Edit** dans la barre de ruban.

![En Agent Viewervariablesdocedit Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/en_agent_viewervariablesdocedit.png)

\
Cliquez sur la colonne Valeur pour la variable Viewer que vous souhaitez modifier manuellement.

**Réinitialiser**\
Cliquez sur le bouton Réinitialiser dans la barre de ruban pour rétablir les valeurs par défaut pour ce Viewer.

 

**Dupliquer**\
Cliquez sur le bouton **Duplicate** dans la barre de ruban pour dupliquer cette valeur vers un ou plusieurs Viewers.\
Il vous sera demandé de spécifier les destinations.\
\
Cela vous permet de définir la valeur pour plusieurs viewers en même temps.

 

**Exporter**\
Exportez toutes les variables viewer vers Microsoft Excel. Lorsque vous souhaitez dupliquer des variables viewer vers un autre système PADS4, ou lorsque vous souhaitez modifier les valeurs des variables viewer dans Excel, ce qui permet des actions copier/coller plus avancées, l'exportation crée une feuille Excel avec toutes les variables viewer. Seules les cellules modifiables sont déverrouillées.

 

**Importer**\
Importez des variables viewer depuis Microsoft Excel.

Pendant l'importation, une fenêtre d'aperçu s'affiche montrant les résultats après importation. Il existe deux options pour l'importation. L'une est un remplacement où toutes les variables viewer sont remplacées, et une fusion où les variables viewer existantes sont mises à jour et de nouvelles variables viewer sont ajoutées.

* Il est possible d'importer de nouvelles variables viewer dans le document Excel
* Il n'est pas possible d'importer de nouveaux noms de viewer dans le document Excel
* Les cellules non modifiables sont verrouillées
* Utilisez le modèle de document Excel de l'exportation
* Les documents Excel qui ne correspondent pas au modèle seront ignorés

 

<Warning>
  L'ajout/suppression de variables Viewer s'effectue avec l'application PADS4 System Management.
</Warning>
