Avec les fonctions de variables Viewer, vous pouvez afficher et modifier les variables Viewer actuellement disponibles dans le système PADS4.

 

#### **Pour afficher les informations sur les variables Viewer stockées sur votre serveur PADS4**

1. Cliquez sur l'onglet **Server** dans le volet de navigation.

   ![EN Agent Navigationpane Servertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Servertab.png)
2. Cliquez sur **Viewer variables** dans la liste des catégories d'informations serveur disponibles.

Après cela, le système récupère automatiquement les informations demandées depuis le serveur PADS4 connecté et affiche les résultats dans la fenêtre Viewer variables.

 

#### **Fenêtre Viewer variables**

La fenêtre Viewer variables affiche toutes les entrées qui ont été stockées sur votre serveur PADS4.\
Les Viewer variables sont créées à l'aide de l'application PADS4 System Management.

 

**Edit**\
Modifier une valeur de Viewer variable. Cliquez sur le bouton **Edit** dans la barre de ruban.

![En Agent Viewervariablesdocedit Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/en_agent_viewervariablesdocedit.png)

\
Cliquez sur la colonne Valeur pour la Viewer variable que vous souhaitez modifier manuellement.

**Reset**\
Cliquez sur le bouton Reset dans la barre de ruban pour réinitialiser les valeurs à la valeur par défaut pour ce Viewer.

 

**Duplicate**\
Cliquez sur le bouton **Duplicate** dans la barre de ruban pour dupliquer cette valeur vers un ou plusieurs Viewers.\
Il vous sera demandé de spécifier les destinations.\
\
Cela vous permet de définir la valeur pour plusieurs viewers en même temps.

 

**Export**\
Exporter toutes les viewer variables vers Microsoft Excel. Lorsque vous souhaitez dupliquer des viewer variables vers un autre système PADS4, ou lorsque vous souhaitez modifier des valeurs de viewer variables dans Excel, ce qui permet des actions de copier/coller plus avancées, l'export crée une feuille Excel avec toutes les viewer variables. Seules les cellules modifiables sont déverrouillées.

 

**Import**\
Importer des viewer variables depuis Microsoft Excel.

Lors de l'import, une fenêtre de prévisualisation s'affiche montrant les résultats après l'import. Il existe deux options d'import. L'une est un remplacement où toutes les viewer variables sont remplacées, et une fusion où les viewer variables existantes sont mises à jour et les nouvelles viewer variables sont ajoutées.

* Il est possible d'importer de nouvelles viewer variables dans le document Excel
* Il n'est pas possible d'importer de nouveaux noms de viewer dans le document Excel
* Les cellules non modifiables sont verrouillées
* Utilisez le modèle de document Excel issu de l'export
* Les documents Excel ne correspondant pas au modèle seront ignorés

 

<Warning>
  L'ajout/la suppression de Viewer variables s'effectue avec l'application PADS4 System Management.
</Warning>
