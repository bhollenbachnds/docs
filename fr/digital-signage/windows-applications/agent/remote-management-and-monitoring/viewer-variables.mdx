Avec les fonctions de variables Viewer, vous pouvez afficher et modifier les variables Viewer actuellement disponibles dans le système PADS4.

 

#### **Pour afficher les informations de variables Viewer stockées sur votre serveur PADS4**

1. Cliquez sur l'onglet **Server** dans le volet de navigation.

   ![EN Agent Navigationpane Servertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Servertab.png)
2. Cliquez sur **Viewer variables** dans la liste des catégories d'informations de serveur disponibles.

Après cela, le système récupérera automatiquement les informations demandées depuis le serveur PADS4 connecté et affichera les résultats dans la fenêtre Viewer variables.

 

#### **Fenêtre Viewer variables**

La fenêtre Viewer variables affiche toutes les entrées qui ont été stockées sur votre serveur PADS4.\
Les Viewer variables sont créées en utilisant l'application PADS4 System Management.

 

**Edit**\
Modifier la valeur d'une Viewer variable. Cliquez sur le bouton **Edit** dans la barre de ruban.

![En Agent Viewervariablesdocedit Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/en_agent_viewervariablesdocedit.png)

\
Cliquez sur la colonne Value pour la Viewer variable que vous souhaitez modifier manuellement.

**Reset**\
Cliquez sur le bouton Reset dans la barre de ruban pour réinitialiser les valeurs à la valeur par défaut pour ce Viewer.

 

**Duplicate**\
Cliquez sur le bouton **Duplicate** dans la barre de ruban pour dupliquer cette valeur vers un ou plusieurs Viewers.\
Il vous sera demandé de spécifier les destinations.\
\
Cela vous permet de définir la valeur pour plusieurs viewers en même temps.

 

**Export**\
Exporter toutes les viewer variables vers Microsoft Excel. Lorsque vous souhaitez dupliquer des viewer variables vers un autre système PADS4, ou lorsque vous souhaitez modifier les valeurs des viewer variables dans Excel, ce qui permet des actions copier/coller plus avancées, l'exportation crée une feuille Excel avec toutes les viewer variables. Seules les cellules modifiables sont déverrouillées.

 

**Import**\
Importer des viewer variables depuis Microsoft Excel.

Pendant l'importation, une fenêtre d'aperçu s'affiche montrant les résultats après importation. Il existe deux options pour l'importation. L'une est un remplacement où toutes les viewer variables sont remplacées, et une fusion où les viewer variables existantes sont mises à jour et les nouvelles viewer variables sont ajoutées.

* Il est possible d'importer de nouvelles viewer variables dans le document Excel
* Il n'est pas possible d'importer de nouveaux noms de viewer dans le document Excel
* Les cellules non modifiables sont verrouillées
* Utilisez le modèle de document Excel de l'exportation
* Les documents Excel qui ne correspondent pas au modèle seront ignorés

 

<Warning>
  L'ajout/suppression de Viewer variables s'effectue avec l'application PADS4 System Management.
</Warning>
