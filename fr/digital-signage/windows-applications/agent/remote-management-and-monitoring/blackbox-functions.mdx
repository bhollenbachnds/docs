Avec les fonctions de boîte noire de PADS4 Agent, vous pouvez consulter toutes les informations stockées dans le fichier de boîte noire.

Pour accéder aux fonctions de la boîte noire, vous devez [sélectionner une catégorie d'informations de boîte noire](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm) dans le volet de navigation de la fenêtre principale.

 

#### **Aperçu des fonctions de la boîte noire**

| **Fonction**                                                                                                                                           | **Description**                                                                                                         |
| :----------------------------------------------------------------------------------------------------------------------------------------------------- | :---------------------------------------------------------------------------------------------------------------------- |
| [Viewer log](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Viewer%20log.htm)                   | Pour consulter les messages de journal stockés dans votre boîte noire PADS4.                                            |
| [Viewer settings](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Viewer%20settings.htm)         | Pour consulter les paramètres du visualiseur stockés dans votre boîte noire PADS4.                                      |
| [Memory usage](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Memory%20usage.htm)               | Pour analyser les informations d'utilisation de la mémoire stockées dans votre boîte noire PADS4.                       |
| [Synchronization log](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Synchronization%20log.htm) | Pour consulter les messages de journal du processus de synchronisation de contenu stockés dans votre boîte noire PADS4. |
| [System info](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/System%20info.htm)                 | Pour récupérer les informations système stockées dans votre boîte noire PADS4.                                          |
| [PADS software](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/PADS%20software.htm)             | Pour récupérer les informations de version PADS4 stockées dans votre boîte noire PADS4.                                 |
