Avec les fonctions de boîte noire de PADS4 Agent, vous pouvez consulter toutes les informations stockées dans le fichier de boîte noire.

Pour accéder aux fonctions de boîte noire, vous devez [sélectionner une catégorie d'informations de boîte noire](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm) dans le volet de navigation de la fenêtre principale.

 

#### **Aperçu des fonctions de boîte noire**

| **Fonction**                                                                                                                                                  | **Description**                                                                                                         |
| :------------------------------------------------------------------------------------------------------------------------------------------------------------ | :---------------------------------------------------------------------------------------------------------------------- |
| [Journal du Viewer](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Viewer%20log.htm)                   | Pour consulter les messages de journal stockés dans votre boîte noire PADS4.                                            |
| [Paramètres du Viewer](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Viewer%20settings.htm)           | Pour consulter les paramètres du Viewer stockés dans votre boîte noire PADS4.                                           |
| [Utilisation de la mémoire](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Memory%20usage.htm)         | Pour analyser les informations d'utilisation de la mémoire stockées dans votre boîte noire PADS4.                       |
| [Journal de synchronisation](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Synchronization%20log.htm) | Pour consulter les messages de journal du processus de synchronisation de contenu stockés dans votre boîte noire PADS4. |
| [Informations système](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/System%20info.htm)               | Pour récupérer les informations système stockées dans votre boîte noire PADS4.                                          |
| [Logiciel PADS](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/PADS%20software.htm)                    | Pour récupérer les informations de version PADS4 stockées dans votre boîte noire PADS4.                                 |
