Avec les fonctions de polices, vous pouvez récupérer des informations sur les polices installées sur votre PADS4 Viewer. Lorsque vous utilisez certaines polices dans vos présentations PADS4, ces polices doivent être installées sur un PADS4 Viewer pour une lecture correcte de votre présentation.

 

#### **Pour récupérer des informations sur les polices installées sur un PADS4 Viewer**

1. Cliquez sur l'onglet **Viewer** dans le volet de navigation.

   ![EN Agent Navigationpane Viewertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Viewertab.png)
2. Sélectionnez un Viewer dans la liste des destinations disponibles.
3. Cliquez sur **Fonts** dans la liste des catégories d'informations disponibles pour le viewer.

Après cela, le système se connectera automatiquement au PADS4 Server connecté pour récupérer les informations demandées du Viewer sélectionné et afficher les résultats dans la fenêtre des polices.

<Warning>
  L'application PADS4 ne pourra commencer à récupérer les informations demandées sur le viewer que lorsque vous aurez sélectionné <u>à la fois</u> un Viewer et une Catégorie dans le volet de navigation.
</Warning>

 

#### **Fenêtre des polices**

La fenêtre des polices donne un aperçu des polices qui ont été installées sur le PADS4 Viewer, y compris les polices existantes et les polices ajoutées à l'aide de PADS4 System Management.

![EN Agent Fonts Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Fonts.png)

**Installé par PADS4**\
Si cette case est cochée, cette police a été ajoutée à l'aide de l'application PADS4 System Management. Consultez la documentation de System Management pour plus d'informations sur l'ajout de polices à PADS4.

**Nom de la police**\
Nom de la famille de polices. Il peut y avoir plusieurs fichiers sur un ordinateur qui appartiennent à la même famille de polices.

**Aperçu de la police**\
Donne un aperçu de la police pour avoir une idée de son apparence. Les styles de police comme le gras ou l'italique ne sont pas inclus dans l'aperçu.

**Nom du fichier**\
Affiche le nom du fichier physique qui contient les définitions de police.

 

#### **Installation d'une nouvelle police**

Ouvrez l'application PADS4 System Management et sélectionnez la catégorie Font pour ajouter de nouvelles polices, qui seront automatiquement installées sur tous les Viewers (HTML5).
