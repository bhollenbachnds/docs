Avec les fonctions de connexion utilisateur, vous pouvez vérifier à distance tous les utilisateurs du système PADS4, les applications auxquelles ils sont connectés et éventuellement forcer la fermeture d'une connexion sur une ou toutes les applications.

 

#### **Pour vérifier l'état de connexion de vos utilisateurs PADS4**

1. Cliquez sur l'onglet **Server** dans le volet de navigation.

   ![EN Agent Navigationpane Servertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Servertab.png)
2. Cliquez sur **User Connections** dans la liste des catégories d'informations serveur disponibles.

Après cela, le système récupérera automatiquement les informations demandées depuis le serveur PADS4 connecté et affichera les résultats dans la fenêtre des connexions utilisateur. Si votre installation PADS4 inclut des sous-serveurs, la fenêtre des connexions utilisateur affichera toujours les résultats globaux de l'ensemble de votre installation PADS4, quel que soit le serveur que vous avez sélectionné.

 

#### **Fenêtre des connexions utilisateur**

Dans la fenêtre des connexions utilisateur, vous pouvez voir en un coup d'œil quel utilisateur est connecté à quelle application PADS4 et depuis combien de temps.

![EN Agent Userconnections Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Userconnections.png)

* La première colonne affiche les utilisateurs, les colonnes suivantes contiennent les applications PADS4. Les durées affichées indiquent depuis combien de temps l'utilisateur est connecté à une certaine application. Si vous survolez la durée avec le curseur de la souris, une fenêtre contextuelle avec des informations plus détaillées s'affiche.
* Si vous souhaitez (forcer) la fermeture d'une connexion, sélectionnez la cellule avec la durée que vous souhaitez fermer en cliquant sur la durée et sélectionnez l'option **Close Connection** dans la barre de ruban en haut dans le groupe Manage.
* Si vous souhaitez (forcer) la fermeture de toutes les connexions, sélectionnez la cellule utilisateur que vous souhaitez fermer en cliquant sur le nom d'utilisateur et sélectionnez l'option **Close Connection** dans la barre de ruban en haut dans le groupe Manage.
* Si vous souhaitez afficher uniquement les utilisateurs ayant 1 ou plusieurs connexions ouvertes, sélectionnez l'option **Connected** dans le groupe Show de la barre de ruban.
