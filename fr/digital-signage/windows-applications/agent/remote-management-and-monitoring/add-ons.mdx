Avec les fonctions Add-On, vous pouvez récupérer des informations sur les Add-Ons installés sur votre PADS4 Viewer et installer des Add-Ons supplémentaires. Les Add-Ons sont des éléments de page développés sur mesure qui peuvent être utilisés dans les présentations PADS4. Pour pouvoir utiliser un Add-On, il doit être installé sur chaque PC PADS4 Viewer pour la lecture de la présentation.

 

#### **Pour récupérer des informations sur les Add-Ons installés sur un PADS4 Viewer**

1. Cliquez sur l'onglet **Viewer** dans le volet de navigation.

   ![EN Agent Navigationpane Viewertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Viewertab.png)
2. Sélectionnez un Viewer dans la liste des destinations disponibles.
3. Cliquez sur **Add-Ons** dans la liste des catégories d'informations disponibles pour le viewer.

Après cela, le système se connectera automatiquement au PADS4 Server connecté pour récupérer les informations demandées du Viewer sélectionné et afficher les résultats dans la fenêtre Add-Ons.

<Warning>
  L'application PADS4 ne pourra commencer à récupérer les informations du viewer demandées que lorsque vous aurez sélectionné <u>à la fois</u> un Viewer et une Catégorie dans le volet de navigation.
</Warning>

 

#### **Fenêtre Add-Ons**

La fenêtre Add-Ons affiche les informations de version de tous les Add-Ons installés et de tous les fichiers associés. Cela vous permet de vérifier si tous les Add-Ons ont été installés et si vous utilisez la bonne version des fichiers.

![EN Agent Add Ons Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Add-Ons.png)

**Nom du fichier**\
Nom du fichier qui a été installé dans le dossier \AddOn Elements de PADS4.

**Add-Ons**\
Nom de l'Add-On lorsque le fichier est reconnu comme étant un fichier Add-On.

**Version**\
Numéro de version du fichier installé.

**Taille**\
\
Taille du fichier installé.

**Créé**\
Date et heure auxquelles le fichier a été initialement installé.

**Dernière modification**\
Date et heure auxquelles le fichier a été mis à jour pour la dernière fois.

 

#### **Installation d'Add-Ons supplémentaires**

Depuis la fenêtre Add-Ons, vous pouvez installer à distance un nouvel Add-On sur votre PADS4 Viewer. Pour installer un nouvel Add-On :

1. Cliquez sur **Install** dans le menu **Home**.

   ![EN Agent Ribbon Install Add On Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-InstallAdd-On.png)
2. Parcourez pour trouver le fichier Add-On que vous souhaitez installer ou saisissez manuellement le **Nom du fichier**.

   ![EN Agent Install Add On Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-InstallAdd-On.png)

   Vous pouvez également sélectionner plusieurs fichiers en maintenant enfoncée la touche \[Ctrl] ou \[Shift] tout en cliquant sur les fichiers.
3. Cliquez sur **Open** pour commencer l'installation des fichiers sélectionnés.

Après cela, le système commencera automatiquement à transférer les fichiers sélectionnés vers votre PADS4 Viewer. Pour voir les résultats de cette action, vous devez consulter la fenêtre [tasks](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Tasks.htm).
