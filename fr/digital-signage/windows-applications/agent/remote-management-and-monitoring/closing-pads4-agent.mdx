Lorsque vous avez terminé la gestion ou la surveillance à distance de votre système PADS4, vous pouvez fermer l'application PADS4 Agent.

 

#### **Pour fermer l'application PADS4 Agent**

1. Sélectionnez **Fichier** dans la barre de ruban.
2. Cliquez sur **Quitter**.

Ou appuyez sur le bouton de fermeture pour fermer l'application

![EN Agent Close Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-close.png)

Alternativement, vous pouvez utiliser la combinaison de touches \[Alt] + F4 pour quitter l'application. Si vous avez ouvert des fenêtres supplémentaires, vous devez utiliser cette combinaison de touches plusieurs fois pour quitter.

**REMARQUE**

Lors de la fermeture de l'application, vous serez automatiquement déconnecté de toutes les connexions que vous avez actives.
