Avec les fonctions d'information système, vous pouvez récupérer des informations système détaillées de votre PADS4 Server, PADS4 Viewer ou d'un fichier boîte noire.

 

#### **Pour récupérer les informations système de votre PADS4 Server**

1. Cliquez sur l'onglet **Server** dans le volet de navigation.

   ![EN Agent Navigationpane Servertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Servertab.png)
2. Cliquez sur **System info** dans la liste des catégories d'informations de serveur disponibles.

Après cela, le système récupérera automatiquement les informations demandées du PADS4 Server connecté et affichera les résultats dans la fenêtre d'informations système. Si votre installation PADS4 comprend des sous-serveurs, vous devez d'abord sélectionner un serveur avant de sélectionner la catégorie d'informations.

 

#### **Pour récupérer les informations système d'un PADS4 Viewer**

1. Cliquez sur l'onglet **Viewer** dans le volet de navigation.

   ![EN Agent Navigationpane Viewertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Viewertab.png)
2. Sélectionnez un Viewer dans la liste des destinations disponibles.
3. Cliquez **System info** dans la liste des catégories d'informations disponibles pour le viewer.

Après cela, le système se connectera automatiquement au PADS4 Server connecté pour récupérer les informations demandées du Viewer sélectionné et afficher les résultats dans la fenêtre d'informations système.

<Warning>
  L'application PADS4 ne pourra commencer à récupérer les informations du viewer demandées que lorsque vous aurez sélectionné <u>à la fois</u> un Viewer et une Catégorie dans le volet de navigation.
</Warning>

 

#### **Pour afficher les informations système d'un fichier black box**

1. Assurez-vous que la black box a été [ajoutée](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm) à l'application.
2. Cliquez sur l'onglet **Black Box** dans le volet de navigation.

   ![EN Agent Navigationpane Blackboxtab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Blackboxtab.png)
3. Sélectionnez l'élément black box dans la liste des black boxes disponibles.
4. Cliquez **System info** dans la liste des catégories d'informations disponibles pour la black box.

Après cela, le système ouvrira automatiquement le fichier black box pour récupérer les informations demandées et afficher les résultats dans la fenêtre d'informations système.

 

#### **Fenêtre d'informations système**

La fenêtre d'informations système affiche une grande variété d'informations système. Cela vous permet de vérifier à distance des éléments comme la résolution d'écran, le fuseau horaire ou l'espace disque disponible.

![EN Agent Systeminfo Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Systeminfo.png)

 

#### **Vues**

Par défaut, un résumé des informations système clés est affiché. Pour changer la vue afin de vérifier certaines catégories d'informations système, cliquez sur l'un des boutons suivants dans le menu **Home** du ruban :

* ![EN Agent Ribbon System Info Summary Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-SystemInfoSummary.png)
* Affiche un résumé des informations système clés. Cela comprend des informations sur le système d'exploitation, le processeur et la mémoire.
* ![EN Agent Ribbon System Info Monitors Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-SystemInfoMonitors.png)
* Affiche des informations sur les moniteurs connectés. Pour chaque moniteur, des informations sont données sur la résolution, l'orientation et divers autres éléments.

  Cette vue n'est PAS disponible pour votre PADS4 Server.
* ![EN Agent Ribbon System Info Display Drivers Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-SystemInfoDisplayDrivers.png)
* Affiche des informations sur les cartes graphiques installées. Pour chaque adaptateur, des informations sont données sur le modèle, la RAM, la version du pilote et divers autres éléments.
* ![EN Agent Ribbon System Info Network Adapters Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-SystemInfoNetworkAdapters.png)
* Affiche des informations sur les adaptateurs réseau installés. Les informations listées comprennent des éléments comme l'adresse MAC, l'adresse IP et les paramètres DHCP.
* ![EN Agent Ribbon System Info Disks Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-SystemInfoDisks.png)
* Affiche des informations sur les disques durs existants et les lecteurs réseau. Pour chaque disque ou lecteur, des informations sont données sur la taille et l'espace libre.
* ![EN Agent Ribbon System Info Codecs Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-SystemInfoCodecs.png)
* Affiche des informations sur les pilotes de codec audio et vidéo installés. Pour chaque codec, des informations sont affichées sur le fabricant et le numéro de version.
