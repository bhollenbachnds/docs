Avec les fonctions de capture d'écran, vous pouvez récupérer une capture d'écran de la situation réelle sur votre PADS4 Viewer. Les fonctions de zoom intégrées vous permettent de visualiser les aspects détaillés des captures d'écran.

 

#### **Pour récupérer la capture d'écran d'un PADS4 Viewer**

1. Cliquez sur l'onglet **Viewer** dans le volet de navigation.

   ![EN Agent Navigationpane Viewertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Viewertab.png)
2. Sélectionnez un Viewer dans la liste des destinations disponibles.
3. Cliquez sur **Capture d'écran** dans la liste des catégories d'informations disponibles pour le viewer.

Après cela, le système se connectera automatiquement au PADS4 Server connecté pour récupérer les informations demandées du Viewer sélectionné et afficher les résultats dans la fenêtre de capture d'écran.

<Warning>
  L'application PADS4 ne pourra commencer à récupérer les informations du viewer demandées que lorsque vous aurez sélectionné <u>les deux</u> un Viewer et une Catégorie dans le volet de navigation.
</Warning>

 

#### **Fenêtre de capture d'écran**

La fenêtre de capture d'écran montre un instantané de la situation sur votre PC PADS4 Viewer au moment où la capture d'écran a été prise. Cela vous permet de vérifier rapidement la situation réelle de votre PADS4 Viewer.

![EN Agent Screenshot Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Screenshot.png)

<Tip>
  Les captures d'écran sont toujours prises en taille réelle. Bien que les informations soient compressées lors de leur envoi sur le réseau, la récupération d'une capture d'écran peut entraîner une augmentation du trafic réseau. Sur les réseaux à faible bande passante, la récupération d'une capture d'écran peut prendre un temps considérable.
</Tip>

Lorsque l'état de votre PADS4 Viewer est 'connecté', la fenêtre de capture d'écran affichera une capture d'écran de la fenêtre PADS4 Viewer. Si le PADS4 Viewer est déconnecté, mais que votre PADS4 Controller est toujours connecté, une capture d'écran de l'ensemble du bureau Windows sera renvoyée à la place.

 

#### **Zoom**

Plusieurs fonctions de zoom vous permettent de visualiser la capture d'écran à votre niveau de zoom préféré. Comme les captures d'écran sont toujours récupérées en taille réelle, vous pouvez zoomer sur tous les détails que vous souhaitez.

Les fonctions de zoom disponibles sont accessibles via la barre de ruban de l'application ou vous pouvez utiliser le contrôle de zoom qui s'affiche en haut de la fenêtre de capture d'écran.

![EN Agent Ribbon Zoom Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-Zoom.png)

 

#### **Enregistrement de la capture d'écran**

Pour enregistrer la capture d'écran affichée dans un fichier pour une utilisation ultérieure :

1. Cliquez sur **Save** dans le menu **Home**.

   ![EN Agent Ribbon Save Screenshot Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-SaveScreenshot.png)
2. Saisissez un **File name** et cliquez sur **Save** pour enregistrer la capture d'écran.

Après cela, le fichier de capture d'écran sera enregistré au format .PNG sur votre ordinateur.
