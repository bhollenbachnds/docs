Avec les fonctions de visualisation de PADS4 Agent, vous pouvez récupérer toutes sortes d'informations de chaque Viewer de votre PADS4 Server connecté et effectuer des ajustements à distance.

Pour accéder aux fonctions du viewer, vous devez [sélectionner une catégorie d'informations du viewer](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm) dans le volet de navigation de la fenêtre principale.
