Avec les fonctions de visualisation de PADS4 Agent, vous pouvez récupérer toutes sortes d'informations de chaque Viewer de votre PADS4 Server connecté et effectuer des ajustements à distance.

Pour accéder aux fonctions du viewer, vous devez [sélectionner une catégorie d'informations du viewer](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm) dans le volet de navigation de la fenêtre principale.

| [Screenshot](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Screenshot.htm)                                               | Pour récupérer une capture d'écran de la situation actuelle sur votre PADS4 Viewer.                                                        |
| [Viewer log](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Viewer%20log.htm)                                             | Pour consulter les messages du journal de votre PADS4 Viewer.                                                                              |
| [Memory usage](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Memory%20usage.htm)                                         | Pour analyser l'utilisation de la mémoire des processus PADS4 sur votre PADS4 Viewer ou l'ensemble de votre système PADS4 Viewer.          |
| [Content library](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Content%20library.htm)                                   | Pour parcourir la bibliothèque de contenu de votre PADS4 Viewer.                                                                           |
| [Synchronization log](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Synchronization%20log.htm)                           | Pour consulter les messages du journal du processus de synchronisation du contenu sur votre PADS4 Viewer.                                  |
