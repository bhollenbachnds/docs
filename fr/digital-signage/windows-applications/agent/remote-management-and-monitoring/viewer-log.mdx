Avec les fonctions de journal du viewer, vous pouvez consulter les messages de journal de votre PADS4 (HTML5) Viewer ou fichier black box. Le journal du viewer d'un PADS4 Viewer peut être effacé ou enregistré dans un fichier, avec d'autres informations de black box.

 

#### **Pour consulter les messages de journal d'un PADS4 (HTML5) Viewer**

1. Cliquez sur l'onglet **Viewer** dans le volet de navigation.

   ![EN Agent Navigationpane Viewertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Viewertab.png)
2. Sélectionnez un Viewer dans la liste des destinations disponibles.
3. Cliquez sur **Viewer log** dans la liste des catégories d'informations de viewer disponibles.

Après cela, le système se connectera automatiquement au PADS4 Server connecté pour récupérer les informations demandées du Viewer sélectionné et afficher les résultats dans la fenêtre du journal du viewer.

<Warning>
  L'application PADS4 ne pourra commencer à récupérer les informations du viewer demandées que lorsque vous aurez sélectionné <u>à la fois</u> un Viewer et une Catégorie dans le volet de navigation.
</Warning>

#### **Pour consulter les messages de journal d'un fichier black box**

1. Assurez-vous que la black box a été [ajoutée](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm) à l'application.
2. Cliquez sur l'onglet **Black Box** dans le volet de navigation.

   ![EN Agent Navigationpane Blackboxtab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Blackboxtab.png)
3. Sélectionnez l'élément black box dans la liste des black boxes disponibles.
4. Cliquez sur **Viewer log** dans la liste des catégories d'informations de black box disponibles.

Après cela, le système ouvrira automatiquement le fichier black box pour récupérer les informations demandées et afficher les résultats dans la fenêtre du journal du viewer.

 

#### **Fenêtre du journal du viewer**

La fenêtre du journal du viewer affiche tous les messages de journal qui ont été générés par l'application PADS4 Viewer ou PADS4 Controller. Lorsque vous rencontrez un problème avec un PADS4 Viewer, le journal du Viewer est généralement le point de départ pour trouver la cause de votre problème.

![EN Agent Viewerlog Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Viewerlog.png)

Facultativement, vous pouvez utiliser les boutons à bascule au bas de la fenêtre du journal du viewer pour filtrer les messages affichés. Par exemple, lorsque vous souhaitez voir uniquement les avertissements et les messages d'erreur.\
Le bouton ci-dessous à droite copie l'élément sélectionné dans le **presse-papiers**. Une autre façon est d'utiliser le bouton droit de la souris au-dessus de l'élément et de sélectionner copier dans le presse-papiers dans le menu.

**Niveau**\
Une icône pour indiquer si l'élément journalisé est un message d'information, un avertissement ou une erreur.

**Horodatage**\
Date et heure auxquelles l'élément a été journalisé.

**Source**\
Nom du processus qui a journalisé l'élément. Si le processus est PADS4 Viewer, le nom du viewer qui identifie le PADS4 Viewer est répertorié.

**Message**\
Le message réel qui est enregistré.

 

#### **Effacer le journal du Viewer**

En effaçant le journal du Viewer, vous pouvez redémarrer le processus d'enregistrement de votre PADS4 Viewer. Pour effacer le journal du Viewer :

1. Cliquez sur **Clear Log** dans le menu **Home**.

   ![EN Agent Ribbon Clear Log Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-ClearLog.png)
2. Cliquez sur **Yes** pour confirmer que vous souhaitez effacer le journal du Viewer.

Après cela, tous les messages du journal seront supprimés du journal du Viewer sur le PC PADS4 Viewer.

 

#### **Enregistrer le journal du Viewer (HTML5)**

Pour enregistrer le journal du Viewer afin de le consulter hors ligne ultérieurement, vous pouvez l'enregistrer avec d'autres informations [black box](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm) de votre Viewer. Pour enregistrer le journal du Viewer :

1. Cliquez sur **Save Black Box** dans le menu **Home**.
2. Saisissez un **File name** et cliquez sur **Save** pour enregistrer les informations.

Après cela, le fichier black box sera enregistré sur votre ordinateur.

<Tip>
  Chaque PADS4 Viewer crée automatiquement ce qu'on appelle une black box dans laquelle des informations précieuses sur PADS4 sont stockées. Cela comprend des informations système et divers journaux. Lorsque vous rencontrez un problème avec un PADS4 (HTML5) Viewer, la black box est la source d'information idéale pour analyser la situation et tracer le problème. Ainsi, lorsque vous contactez le support pour un problème avec un PADS4 (HTML5) Viewer, il est toujours suggéré d'envoyer le fichier black box.
</Tip>
