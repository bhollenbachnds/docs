Avec les fonctions du journal de synchronisation, vous pouvez consulter les messages du journal du processus de synchronisation de contenu sur votre PADS4 Viewer ou les messages du journal stockés dans un fichier boîte noire. Le journal de synchronisation d'un PADS4 Viewer peut être effacé ou enregistré dans un fichier, avec d'autres informations de la boîte noire.

 

#### **Pour consulter le journal de synchronisation d'un PADS4 Viewer**

1. Cliquez sur l'onglet **Viewer** dans le volet de navigation.

   ![EN Agent Navigationpane Viewertab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Viewertab.png)
2. Sélectionnez un Viewer dans la liste des destinations disponibles.
3. Cliquez sur **Journal de synchronisation** dans la liste des catégories d'informations viewer disponibles.

Après cela, le système se connectera automatiquement au PADS4 Server connecté pour récupérer les informations demandées depuis le Viewer sélectionné et afficher les résultats dans la fenêtre du journal de synchronisation.

<Warning>
  L'application PADS4 ne pourra commencer à récupérer les informations du viewer demandées que lorsque vous aurez sélectionné <u>les deux</u> un Viewer et une Catégorie dans le volet de navigation.
</Warning>

 

#### **Pour consulter le journal de synchronisation d'un fichier boîte noire**

1. Assurez-vous que la boîte noire a été [ajoutée](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm) à l'application.
2. Cliquez sur l'onglet **Black Box** dans le volet de navigation.

   ![EN Agent Navigationpane Blackboxtab Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Navigationpane-Blackboxtab.png)
3. Sélectionnez l'élément boîte noire dans la liste des boîtes noires disponibles.
4. Cliquez sur **Journal de synchronisation** dans la liste des catégories d'informations boîte noire disponibles.

Après cela, le système ouvrira automatiquement le fichier boîte noire pour récupérer les informations demandées et afficher les résultats dans la fenêtre du journal de synchronisation.

 

#### **Fenêtre du journal de synchronisation**

La fenêtre du journal de synchronisation affiche tous les messages du journal qui ont été générés par le processus de synchronisation de contenu sur un PC Viewer. Grâce à ces informations, vous pouvez vérifier quand le processus de synchronisation a été exécuté et obtenir des détails sur les éléments en cours de synchronisation.

![EN Agent Synchronizationlog Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Synchronizationlog.png)

**Horodatage**\
Date et heure auxquelles l'élément a été enregistré.

**Message**\
L'élément réel qui a été enregistré. En général, chaque session de synchronisation commence par le message « Requesting new synchronization session » et se termine par « Synchronization session finished ».

**Taille**\
Si le processus de synchronisation ajoute de nouveaux éléments à votre bibliothèque de contenu, la taille du fichier ajouté est indiquée ici.

 

#### **Effacement du journal de synchronisation**

En effaçant le journal de synchronisation, vous pouvez redémarrer le processus de journalisation de votre PADS4 Viewer. Pour effacer le journal de synchronisation :

1. Cliquez sur **Clear Log** dans le **Home** menu.

   ![EN Agent Ribbon Clear Log Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-ClearLog.png)
2. Cliquez sur **Yes** pour confirmer que vous souhaitez effacer le journal de synchronisation.

Après cela, tous les messages du journal seront supprimés du journal de synchronisation sur le PC PADS4 Viewer.

 

#### **Enregistrement du journal de synchronisation**

Pour enregistrer le journal de synchronisation afin de le consulter hors ligne ultérieurement, vous pouvez l'enregistrer avec d'autres informations de type [boîte noire](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm) de votre Viewer. Pour enregistrer le journal de synchronisation :

1. Cliquez sur **Save Black Box** dans le **Home** menu.
2. Saisissez un **File name** et cliquez sur **Save** pour enregistrer les informations.

Après cela, le fichier boîte noire sera enregistré sur votre ordinateur.

<Tip>
  Chaque PADS4 Viewer crée automatiquement une soi-disant boîte noire dans laquelle des informations précieuses sur PADS4 sont stockées. Cela inclut des informations système et divers journaux. Lorsque vous rencontrez un problème avec un PADS4 Viewer, la boîte noire est la source d'information idéale pour analyser la situation et identifier le problème. Ainsi, lorsque vous contactez le support pour un problème avec un PADS4 Viewer, il est conseillé d'envoyer toujours le fichier boîte noire.
</Tip>
