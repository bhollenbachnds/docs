En haut de la fenêtre principale, vous trouverez une barre de ruban - un ensemble de barres d'outils, placées sur plusieurs onglets - qui donne un accès rapide aux fonctions clés de l'application.

 

#### **Aperçu de la barre de ruban PADS4 Agent**

La barre de ruban PADS4 Agent a été divisée en 5 sections appelées groupes d'onglets. Vous pouvez y accéder en cliquant sur les en-têtes respectifs. Des informations détaillées seront fournies dans les rubriques à suivre.

1. Fichier

   ![EN Agent Ribbon File Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-File.png)

   Le groupe d'onglets Fichier contient les fonctionnalités les plus basiques pour PADS4 Agent, y compris la gestion des connexions.
2. Accueil

   ![EN Agent Ribbon Home Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-Home.png)

   Le groupe d'onglets Accueil est le conteneur des fonctions clés de l'application. Celles-ci incluent la gestion des connexions au serveur PADS4, et les fonctions les plus courantes pour la catégorie d'information choisie.
3. Affichage

   ![EN Agent Ribbon View Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-View.png)

   Le groupe d'onglets Affichage vous permet de déterminer la façon dont vous souhaitez visualiser vos panneaux d'information.
4. Outils

   ![EN Agent Ribbon Tools Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-Tools.png)

   Le groupe d'onglets Outils donne accès aux options de l'application elle-même ainsi qu'aux fonctions système.
5. Aide

   ![EN Agent Ribbon Help Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-Help.png)

   Le groupe d'onglets Aide fournit des informations sur l'application elle-même comme les instructions, la version et les détails de licence.
