En haut de la fenêtre principale se trouve une barre de ruban — un ensemble de barres d'outils, placées sur plusieurs onglets — qui donne un accès rapide aux fonctions clés de l'application.

 

#### **Aperçu de la barre de ruban de PADS4 Agent**

La barre de ruban de PADS4 Agent est divisée en 5 sections appelées groupes d'onglets. Vous pouvez y accéder en cliquant sur les en-têtes correspondants. Des informations détaillées seront fournies dans les rubriques suivantes.

1. File

   ![EN Agent Ribbon File Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-File.png)

   Le groupe d'onglets File contient les fonctionnalités les plus basiques de PADS4 Agent, notamment la gestion des connexions.
2. Home

   ![EN Agent Ribbon Home Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-Home.png)

   Le groupe d'onglets Home est le conteneur des fonctions clés de l'application. Celles-ci comprennent la gestion des connexions au PADS4 Server et les fonctions les plus courantes pour la catégorie d'informations choisie.
3. View

   ![EN Agent Ribbon View Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-View.png)

   Le groupe d'onglets View vous permet de déterminer la façon dont vous souhaitez afficher vos panneaux d'information.
4. Tools

   ![EN Agent Ribbon Tools Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-Tools.png)

   Le groupe d'onglets Tools donne accès aux options de l'application elle-même ainsi qu'aux fonctions à l'échelle du système.
5. Help

   ![EN Agent Ribbon Help Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-Help.png)

   Le groupe d'onglets Help donne accès aux informations sur l'application elle-même, telles que les instructions, la version et les détails de la licence.
