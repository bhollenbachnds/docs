En haut de la fenêtre principale se trouve une barre de ruban - un ensemble de barres d'outils, placées sur plusieurs onglets - qui donne un accès rapide aux fonctions clés de l'application.

 

#### **Aperçu de la barre de ruban PADS4 Agent**

La barre de ruban PADS4 Agent est divisée en 5 sections appelées groupes d'onglets. Vous pouvez y accéder en cliquant sur les en-têtes correspondants. Des informations détaillées seront fournies dans les rubriques suivantes.

1. Fichier

   ![EN Agent Ribbon File Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-File.png)

   Le groupe d'onglets Fichier contient les fonctionnalités les plus basiques de PADS4 Agent, notamment la gestion des connexions.
2. Accueil

   ![EN Agent Ribbon Home Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-Home.png)

   Le groupe d'onglets Accueil est le conteneur des fonctions clés de l'application. Celles-ci comprennent la gestion de la connexion au PADS4 Server, ainsi que les fonctions les plus courantes pour la catégorie d'informations choisie.
3. Affichage

   ![EN Agent Ribbon View Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-View.png)

   Le groupe d'onglets Affichage vous permet de déterminer la façon dont vous souhaitez visualiser vos panneaux d'informations.
4. Outils

   ![EN Agent Ribbon Tools Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-Tools.png)

   Le groupe d'onglets Outils donne accès aux options de l'application elle-même ainsi qu'aux fonctions à l'échelle du système.
5. Aide

   ![EN Agent Ribbon Help Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Ribbon-Help.png)

   Le groupe d'onglets Aide donne accès aux informations sur l'application elle-même, telles que les instructions, la version et les détails de la licence.
