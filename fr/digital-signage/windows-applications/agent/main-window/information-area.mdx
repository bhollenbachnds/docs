Après avoir sélectionné une catégorie d'informations d'un serveur, d'un viewer ou d'une black box dans le [volet de navigation](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm), les informations seront récupérées et les résultats seront affichés dans une fenêtre de la zone d'informations. La façon dont les résultats sont affichés dans la fenêtre d'informations dépend de la catégorie choisie.

![EN Agent Viewerconnections Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Viewerconnections.png)

Pour la plupart des catégories, un certain nombre de fonctions contextuelles seront disponibles dans la ribbon bar. Ces mêmes fonctions contextuelles peuvent également être appelées en faisant un clic droit sur la fenêtre d'informations.

Par défaut, l'application PADS4 utilise une seule fenêtre d'informations pour afficher tous les résultats. Cela signifie que lorsque vous passez à une autre catégorie d'informations, la même fenêtre sera réutilisée pour afficher les résultats. Vous pouvez éventuellement ouvrir manuellement des fenêtres d'informations supplémentaires comme décrit dans [Travailler avec plusieurs fenêtres](https://pads4documentation.blob.core.windows.net/agent/Content/Advanced%20features/Working%20with%20multiple%20windows.htm) ou utiliser les [options d'affichage](https://pads4documentation.blob.core.windows.net/agent/Content/Tools/Options/View%20options.htm) de l'application pour que le système ouvre automatiquement de nouvelles fenêtres d'informations.

 

#### **Actualisation des données**

La première fois que vous sélectionnez une certaine catégorie d'informations, les informations sont toujours récupérées depuis le serveur ou le viewer distant. Lorsque vous sélectionnez à nouveau la même catégorie d'informations dans le volet de navigation, la réponse du système dépend de la catégorie sélectionnée. Pour des raisons de performances, les catégories dont les informations changent rarement afficheront les résultats précédents mis en cache localement. Les catégories avec des informations dynamiques récupèreront toujours leurs informations depuis le serveur ou le viewer distant. Si les informations sont récupérées depuis le serveur ou le viewer distant, vous verrez une nouvelle tâche apparaître dans la [fenêtre des tâches.](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Tasks.htm) Si aucune nouvelle tâche n'apparaît dans la fenêtre des tâches, cela signifie que les informations sont récupérées depuis le cache local.

<Note>
  Pour forcer l'application PADS4 Agent à récupérer les informations demandées depuis le serveur ou le viewer connecté, vous devez cliquer sur **Refresh** dans l'onglet **Home** de la ribbon bar ou utiliser la touche de raccourci F5.
</Note>

Pour certaines catégories d'informations (comme toutes les catégories de black box), la fonction **Refresh** n'est pas disponible, car les informations de la catégorie sont toujours disponibles et il n'est pas nécessaire d'actualiser les informations.
