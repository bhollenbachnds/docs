Après avoir sélectionné une catégorie d'informations d'un serveur, visualiseur ou boîtier noir dans le [volet de navigation](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Navigation%20pane.htm), les informations seront récupérées et les résultats seront affichés dans une fenêtre dans la zone d'information. La façon dont les résultats sont affichés dans la fenêtre d'information dépend de la catégorie choisie.

![EN Agent Viewerconnections Pn](/images/digital-signage/windows-applications/agent/main-window/EN-Agent-Viewerconnections.png)

Pour la plupart des catégories, un certain nombre de fonctions contextuelles deviendront disponibles dans le ruban. Les mêmes fonctions contextuelles peuvent également être appelées en faisant un clic droit sur la fenêtre d'information.

Par défaut, l'application PADS4 utilise une fenêtre d'information pour afficher tous les résultats. Cela signifie que lorsque vous passez à une autre catégorie d'informations, la même fenêtre sera réutilisée pour afficher les résultats. Vous pouvez éventuellement ouvrir manuellement des fenêtres d'information supplémentaires comme décrit dans [Travailler avec plusieurs fenêtres](https://pads4documentation.blob.core.windows.net/agent/Content/Advanced%20features/Working%20with%20multiple%20windows.htm) ou utiliser les [options d'affichage](https://pads4documentation.blob.core.windows.net/agent/Content/Tools/Options/View%20options.htm) de l'application pour que le système ouvre automatiquement de nouvelles fenêtres d'information.

 

#### **Actualisation des données**

La première fois que vous sélectionnez une certaine catégorie d'informations, les informations sont toujours récupérées à partir du serveur ou du visualiseur distant. Lorsque vous sélectionnez ultérieurement la même catégorie d'informations dans le volet de navigation, la réponse du système dépend de la catégorie sélectionnée. Pour des raisons de performance, les catégories contenant des informations qui ne changeront presque jamais afficheront les résultats précédents mis en cache localement. Les catégories avec des informations dynamiques récupéreront toujours leurs informations à partir du serveur ou du visualiseur distant. Si les informations sont récupérées à partir du serveur ou du visualiseur distant, vous verrez une nouvelle tâche apparaître dans la [fenêtre des tâches.](https://pads4documentation.blob.core.windows.net/agent/Content/Main%20Window/Tasks.htm)Si aucune nouvelle tâche n'apparaît dans la fenêtre des tâches, cela signifie que les informations sont récupérées depuis le cache local.

<Note>
  Pour forcer l'application PADS4 Agent à récupérer les informations demandées depuis le serveur connecté ou le visualiseur, vous devez cliquer sur **Refresh** dans l'onglet **Home** du ruban ou utiliser le raccourci clavier F5.
</Note>

Pour certaines catégories d'informations (comme toutes les catégories de boîte noire), la fonction **Refresh** n'est pas disponible, car les informations de la catégorie sont toujours disponibles et il n'est pas nécessaire de les actualiser.
