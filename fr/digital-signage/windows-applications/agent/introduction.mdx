---
- title: Introduction à PADS4 Agent
- description: Présentation de PADS4 Agent, l'application Windows utilisée pour
    gérer et surveiller à distance les serveurs, les viewers et les processus
    dans une installation PADS4.
---

Bienvenue dans PADS4, l'un des logiciels d'affichage dynamique les plus avancés et les plus complets disponibles aujourd'hui.

Ce manuel PADS4 Agent décrit les possibilités de PADS4 Agent, l'application destinée aux utilisateurs finaux pour gérer et surveiller à distance votre installation PADS4.
