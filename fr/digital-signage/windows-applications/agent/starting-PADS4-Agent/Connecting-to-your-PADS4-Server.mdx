Lorsque PADS4 Agent a été démarré, il recherchera les connexions PADS4 Server disponibles. Selon la façon dont vous avez configuré vos propriétés de connexion, l'application se connectera et s'identifiera automatiquement à votre PADS4 Server ou vous devrez le faire manuellement.

<Note>
  Vous pouvez éventuellement travailler avec plusieurs connexions PADS4 Server. Pour plus d'informations à ce sujet, voir [Connexion à plusieurs serveurs](https://pads4documentation.blob.core.windows.net/agent/Content/Advanced%20features/Connecting%20to%20multiple%20servers.htm).
</Note>

### **Connexion à votre PADS4 Server pour la première fois**

Lorsque c'est la toute première fois que vous démarrez PADS4 Agent et que vous n'avez pas encore défini de propriétés de connexion, vous obtiendrez le message suivant :

![EN Agent Connecting Firsttime Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Connecting-Firsttime.png)

1. Cliquez sur **OK** pour continuer.
2. Dans la fenêtre de connexion PADS4 Server (qui s'ouvre automatiquement), cliquez sur **Add...** pour définir une nouvelle connexion.
3. Spécifiez vos propriétés de connexion et cliquez sur **OK** pour les enregistrer. Pour plus d'informations, voir [Gestion de vos connexions](https://pads4documentation.blob.core.windows.net/agent/Content/Starting%20PADS%20Agent/Managing%20your%20connections.htm).
4. Cliquez sur **Connect** pour établir la connexion.

   ![EN Agent Manageconnections 02 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Manageconnections-02.png)

   \
   Même si vous avez configuré votre connexion pour se connecter automatiquement, vous devrez vous connecter de cette façon la première fois.
5. Si vous devez vous identifier, saisissez votre **User Name** et **Password** et cliquez sur **Log On**.

   ![EN Agent Loggingon 01 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Loggingon-01.png)

### **Connexion à votre PADS4 Server**

Si vous avez spécifié dans vos propriétés de connexion de vous connecter automatiquement à votre PADS4 Server, PADS4 Agent essaiera automatiquement d'établir une connexion une fois qu'il aura été démarré. Dans toutes les autres situations, vous devrez établir une connexion manuellement :

1. Cliquez sur la partie inférieure du bouton **Connect** dans l'onglet **Home** de la barre de ruban.

   ![EN Agent Connecting 02 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Connecting-02.png)
2. Cliquez sur la connexion listée pour établir une connexion.

   ![EN Agent Connecting 03 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Connecting-03.png)
3. Si vous devez vous identifier, saisissez votre **User Name** et **Password** et cliquez sur **Log On**.

### **Déconnexion de votre PADS4 Server**

Pour vous déconnecter de votre PADS4 Server, cliquez sur **Disconnect** dans l'onglet **Home** de la barre de ruban.

### **Reconnexion à votre PADS4 Server**

Si vous avez été déconnecté de votre PADS4 Server, vous voudrez peut-être vous reconnecter. Pour vous reconnecter :

1. Cliquez sur la partie supérieure du bouton **Connect** dans l'onglet **Home** de la barre de ruban.

   ![EN Agent Connecting 01 Pn](/images/digital-signage/windows-applications/agent/starting-PADS4-Agent/EN-Agent-Connecting-01.png)

   Cela forcera l'application à reconnecter automatiquement votre connexion si elle a été configurée pour se connecter automatiquement.

Ou

1. Suivez les étapes de connexion à votre PADS4 Server comme décrit précédemment.
