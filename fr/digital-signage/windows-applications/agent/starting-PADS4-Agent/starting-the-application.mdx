**PADS4 Agent** est l'application utilisateur finale utilisée pour gérer et surveiller à distance votre système PADS4. Il fournit des outils pour la santé du système, les diagnostics, les alertes et le contrôle opérationnel.

***

## Comment lancer PADS4 Agent

Vous pouvez démarrer l'application PADS4 Agent de l'une des manières suivantes :

### Méthode 1 : Utilisation du menu Démarrer

1. Cliquez sur le bouton **Start** dans la barre des tâches Windows.
2. Naviguez vers **All Programs** > **PADS4**.
3. Cliquez sur **Agent**.

### Méthode 2 : Utilisation de l'Explorateur Windows

1. Ouvrez **Windows Explorer**.
2. Naviguez vers le dossier où les fichiers du programme **PADS4 Agent** sont installés.
3. Double-cliquez `PADSAgent.exe` pour lancer l'application.

***

## Configuration initiale

> **Note:**\
> Si tous les paramètres de configuration sont corrects, l'application se connectera automatiquement au **PADS4 Server** au démarrage.

Si c'est votre **première utilisation** de l'application ou si les **paramètres de connexion sont incorrects**, la fenêtre **PADS4 Connection Management** apparaîtra. Cela vous permet de configurer manuellement ou de corriger les paramètres de connexion au serveur pour démarrer l'application avec succès.
