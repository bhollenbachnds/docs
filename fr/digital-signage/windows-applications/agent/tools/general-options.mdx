Ces options représentent les paramètres globaux de l'application PADS4 Agent.

#### **Pour ouvrir les paramètres généraux de l'application Agent**

1. Cliquez sur l'onglet **Tools** dans la barre de ruban.

   ![EN Agent Toolstab Pn](/images/digital-signage/windows-applications/agent/tools/EN-Agent-Toolstab.png)
2. Cliquez sur **Options** dans la liste des options disponibles.
3. Le menu des options s'ouvrira, et la page **General** sera affichée.

![EN Agent Options General Pn](/images/digital-signage/windows-applications/agent/tools/EN-Agent-Options-General.png)

 

#### **Langue**

**Language**\
\
La langue dans laquelle l'interface utilisateur de l'application PADS4 Agent sera affichée.

Lorsque le paramètre de langue est modifié, le nouveau paramètre ne prendra effet qu'après un redémarrage de l'application.
