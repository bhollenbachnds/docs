La fenêtre À propos de PADS4 Agent vous affiche des informations sur la version de l'application que vous utilisez actuellement ainsi que des informations sur votre licence actuelle.

 

Pour afficher la fenêtre À propos

1. Cliquez sur **About** dans la **Info** section du **Help** menu.

   ![EN Agent Ribbon About Pn](/images/digital-signage/windows-applications/agent/tools/EN-Agent-Ribbon-About.png)
2. La fenêtre À propos apparaîtra.

   ![EN Agent About Pn](/images/digital-signage/windows-applications/agent/tools/EN-Agent-About.png)

<Warning>
  La fenêtre À propos collecte ses informations de licence auprès du PADS4 Server de votre système. Cela signifie que les informations de licence ne sont disponibles que lorsque vous êtes connecté à un PADS4 Server.
</Warning>

<Note>
  Pour des informations détaillées sur votre licence actuelle, veuillez consulter l'application PADS4 System Management ou le PADS4 License Wizard.
</Note>
