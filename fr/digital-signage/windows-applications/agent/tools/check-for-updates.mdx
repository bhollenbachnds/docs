PADS4 Agent, comme toutes les applications clés de PADS4, offre la possibilité de vérifier manuellement les mises à jour logicielles.

 

#### **Pour vérifier les mises à jour**

1. Dans le **Software** groupe d'onglets de l'onglet **Tools** dans la barre de ruban, cliquez sur le bouton **Check for Updates**.

   ![EN Agent Ribbon Updatesoftware Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-Updatesoftware.png)
2. L'assistant de mise à jour logicielle PADS4 Software Update Wizard est lancé. Les étapes suivantes sont décrites dans le manuel d'installation PADS4 Installation Manual, dans la section Maintaining the installation. Veuillez consulter ce manuel pour obtenir des instructions supplémentaires.
