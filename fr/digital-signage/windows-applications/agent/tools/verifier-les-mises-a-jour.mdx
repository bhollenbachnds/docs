PADS Agent, comme toutes les applications clés de PADS4, offre la possibilité de vérifier manuellement les mises à jour logicielles.

 

#### **Pour vérifier les mises à jour**

1. Dans le groupe d'onglets **Software** de l'onglet **Tools** dans la barre de ruban, cliquez sur le bouton **Check for Updates**.

   ![EN Agent Ribbon Updatesoftware Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-Ribbon-Updatesoftware.png)
2. L'Assistant de mise à jour du logiciel PADS4 est lancé. Les étapes qui suivent sont décrites dans le Manuel d'installation PADS4, dans la section Maintenance de l'installation. Veuillez consulter ce manuel pour plus d'instructions.
