Ces options déterminent comment les alertes qui ont été définies dans la [fenêtre de surveillance et d'alertes](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Monitoring%20&%20alerts/Monitoring%20&%20alerts.htm) seront déclenchées par votre application.

#### **Pour ouvrir les paramètres des alertes de l'application Agent**

1. Cliquez sur l'onglet **Tools** dans la barre de ruban.

   ![EN Agent Toolstab Pn](/images/digital-signage/windows-applications/agent/tools/EN-Agent-Toolstab.png)
2. Cliquez sur **Options** dans la liste des options disponibles.
3. Le menu des options s'ouvrira, cliquez sur **Alerts**

![EN Agent Options Alerts Pn](/images/digital-signage/windows-applications/agent/tools/EN-Agent-Options-Alerts.png)

 

#### **Sons**

Ici, vous pouvez spécifier le son qui doit être joué pour les alertes devant émettre un son lorsqu'elles sont déclenchées.

**Utiliser les sons d'alerte Windows par défaut**\
Jouer le son de notification par défaut de Microsoft Windows, tel qu'il a été spécifié sous **Son** dans le **Panneau de configuration** de Windows.

**Utiliser un son d'alerte personnalisé**\
Spécifiez le **nom de fichier** de votre propre fichier son qui sera joué en cas d'alerte. Utilisez éventuellement le bouton **Parcourir...** pour localiser les fichiers son existants sur votre ordinateur.
