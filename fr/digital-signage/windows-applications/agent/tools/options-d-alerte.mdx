Ces options déterminent comment les alertes qui ont été définies dans la [monitoring & alerts window](https://pads4documentation.blob.core.windows.net/agent/Content/Remote%20management%20and%20monitoring/Monitoring%20&%20alerts/Monitoring%20&%20alerts.htm) seront déclenchées par votre application.

#### **Pour ouvrir les paramètres d'Alertes de l'application Agent**

1. Cliquez sur l'onglet **Tools** dans la barre de ruban.

   ![EN Agent Toolstab Pn](/images/digital-signage/windows-applications/agent/tools/EN-Agent-Toolstab.png)
2. Cliquez sur **Options** dans la liste des options disponibles.
3. Le menu des options s'ouvrira, cliquez sur **Alerts**

![EN Agent Options Alerts Pn](/images/digital-signage/windows-applications/agent/tools/EN-Agent-Options-Alerts.png)

 

#### **Sons**

Ici, vous pouvez spécifier le son qui doit être joué pour les alertes qui doivent émettre un son d'alerte lorsque l'alerte est déclenchée.

**Utiliser les sons d'alerte par défaut de Windows**\
Jouer le son de notification par défaut de Microsoft Windows, tel qu'il a été spécifié sous **Sound** dans le **Control Panel** de Windows.

**Utiliser un son d'alerte personnalisé**\
Spécifiez le **file name** de votre propre fichier sonore qui sera joué en cas d'alerte. Utilisez éventuellement le bouton **Browse...** pour localiser des fichiers sonores existants sur votre ordinateur.
