Ces options déterminent la vue de votre application, comme l'ouverture de fenêtres supplémentaires pour afficher des informations.

#### **Pour ouvrir les paramètres de Vue de l'application Agent**

1. Cliquez sur l'onglet **Tools** dans la barre de ruban.

   ![EN Agent Toolstab Pn](/images/digital-signage/windows-applications/agent/tools/EN-Agent-Toolstab.png)
2. Cliquez sur **Options** dans la liste des options disponibles.
3. Le menu des options s'ouvrira, cliquez sur **View**

![EN Agent Options View Pn](/images/digital-signage/windows-applications/agent/tools/EN-Agent-Options-View.png)

 

#### **Navigation Pane**

**Expand all destinations by default**\
Si cette option est cochée, tous les éléments de groupe de visualisation dans le volet de navigation seront développés, afin que vous puissiez voir tous les éléments de visualisation individuels. Si l'option est décochée, tous les éléments de destination seront réduits, et vous ne verrez que les éléments de premier niveau.

 

#### **Information Windows**

**Automatically open a new window**\
Pour décider comment l'application traitera les fenêtres d'information, vous disposez des options suivantes :

| **Option**                    | **Description**                                                                                                                                                                                                                                                                                                                                                                                                 |
| :---------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Never                         | La même fenêtre d'information est utilisée pour afficher tout type d'information.<br />Lorsque vous sélectionnez une autre destination ou catégorie d'information dans le volet de navigation, les informations de la fenêtre active seront remplacées par les nouvelles informations.<br />Pour ouvrir des fenêtres supplémentaires, vous devez cliquer manuellement sur **New Window** dans le menu **View**. |
| For each destination          | Uniquement lorsque vous sélectionnez une nouvelle destination dans le volet de navigation, l'application ouvrira une nouvelle fenêtre d'information.<br />Dans toutes les autres situations, l'une des fenêtres d'information existantes sera utilisée pour afficher les résultats.                                                                                                                             |
| For each information category | Uniquement lorsque vous sélectionnez une nouvelle catégorie d'information dans le volet de navigation, l'application ouvrira une nouvelle fenêtre d'information.<br />Dans toutes les autres situations, l'une des fenêtres d'information existantes sera utilisée pour afficher les résultats.                                                                                                                 |
| Always                        | Chaque fois que vous sélectionnez une nouvelle destination ou catégorie d'information dans le volet de navigation, l'application ouvrira une nouvelle fenêtre d'information.                                                                                                                                                                                                                                    |
