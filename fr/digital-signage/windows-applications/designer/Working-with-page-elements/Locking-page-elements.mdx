Vous pouvez **verrouiller tous les éléments de la page** sur une page pour **éviter les modifications accidentelles**. Lorsqu'une page est verrouillée, les éléments **ne peuvent pas être déplacés ou redimensionnés avec la souris**, ce qui aide à "geler" la conception et à protéger la mise en page.

***

## Comment verrouiller/déverrouiller les éléments de page

* Cliquez sur **Lock Elements** dans le menu **Home**.
* Si les éléments étaient **déverrouillés**, ils seront maintenant **verrouillés**.
* Si les éléments étaient **verrouillés**, ils seront maintenant **déverrouillés**.

***

## CONSEIL

* **Verrouillez toujours votre page** après avoir finalisé sa conception pour éviter les modifications accidentelles.

***

## Remarques

* Même lorsque les éléments sont verrouillés, vous pouvez toujours les **sélectionner** et les **déplacer/redimensionner** en utilisant le **clavier** :
  * Pour **déplacer** des éléments verrouillés :\
    Appuyez et maintenez **\[Ctrl]** et utilisez les **touches fléchées**.
  * Pour **redimensionner** des éléments verrouillés :\
    Appuyez et maintenez **\[Shift]** et utilisez les **touches fléchées**.
