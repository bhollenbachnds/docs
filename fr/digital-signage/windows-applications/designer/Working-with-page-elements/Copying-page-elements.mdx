Les éléments de page peuvent être copiés **within the same page**, **to another page** dans la même présentation, ou **to another presentation**.

***

## Pour copier des éléments de page dans la même page

1. **Sélectionnez** les éléments de page que vous souhaitez copier (voir *Selecting page elements* pour plus de détails).
2. Cliquez sur **Copy** dans le menu **Home**.
3. Cliquez sur **Paste** dans le menu **Home**.

* Les éléments collés apparaîtront **below the original elements**, tant qu'il reste suffisamment d'espace.

### Note:

Si tous les éléments sélectionnés sont connectés à un data provider et partagent le même **Row** property:

* Lors du collage, la propriété **Row** sera automatiquement **increment by 1**.
* Cependant, **repeated pasting** ne continuera pas à incrémenter (par exemple, pas 2, 3, etc.).
* Pour obtenir des lignes consécutives, vous devez répéter **both Copy and Paste** pour chaque étape.

***

## Pour copier des éléments de page vers une autre page dans la même présentation

1. **Sélectionnez** les éléments de page à copier.
2. Cliquez sur **Copy** dans le menu **Home**.
3. **Naviguez** vers une autre page en utilisant les boutons de navigation dans le menu **View**.
4. Cliquez sur **Paste** dans le menu **Home**.

* Les éléments seront collés à la **same position** que les originaux.

***

## Pour copier des éléments de page vers une autre présentation

1. **Sélectionnez** les éléments à copier.
2. Cliquez sur **Copy** dans le menu **Home**.
3. **Ouvrez** la présentation cible (si elle n'est pas encore ouverte), ou cliquez sur son onglet pour l'activer.
4. **Naviguez** vers la page cible en utilisant le menu **View**.
5. Cliquez sur **Paste** dans le menu **Home**.

* Les éléments seront insérés à la **same position** que dans la présentation source.
* Si un élément est connecté à un **data provider**, ce data provider sera **also be copied** vers la nouvelle présentation (s'il n'y est pas déjà).

***

## Remarques

* Les éléments collés seront **only** placés à la position choisie s'ils **fit within the page boundaries**.
* S'ils ne s'adaptent pas, le système **auto-adjust** la position pour s'assurer que tous les éléments sont dans les limites de la page.
* Pour coller des éléments à une **specific position** :
  * **Faites un clic droit** sur la position cible sur la page.
  * Sélectionnez **Paste** dans le menu contextuel — les éléments seront collés à cet endroit si l'espace le permet.

***

## Astuce

* **Raccourci pour copier/coller** :
  * **\[Ctrl] + C** pour copier
  * **\[Ctrl] + V** pour coller

* Si vous souhaitez **copy an entire page** d'une présentation à une autre, utilisez le **Import Page Wizard** :
  * Cliquez sur **Importer la page** dans le **Page** menu.
  * Cela copiera la page entière **y compris ses propriétés**.
