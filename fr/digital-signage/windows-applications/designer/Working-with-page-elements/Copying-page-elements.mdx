Les éléments de page peuvent être copiés **dans la même page**, **vers une autre page** dans la même présentation, ou **vers une autre présentation**.

***

## Pour copier des éléments de page dans la même page

1. **Sélectionner** les éléments de page que vous souhaitez copier (voir *Sélection des éléments de page* pour plus de détails).
2. Cliquez sur **Copy** dans le menu **Home**.
3. Cliquez sur **Paste** dans le menu **Home**.

* Les éléments collés apparaîtront **en dessous des éléments originaux**, à condition qu'il reste suffisamment d'espace.

### Note:

Si tous les éléments sélectionnés sont connectés à un fournisseur de données et partagent le même **Row** property:

* Lors du collage, la propriété **Row** sera automatiquement **incrémentée de 1**.
* Cependant, **les collages répétés** ne continueront pas à s'incrémenter (par exemple, pas 2, 3, etc.).
* Pour obtenir des lignes consécutives, vous devez répéter **à la fois Copy et Paste** pour chaque étape.

***

## Pour copier des éléments de page vers une autre page dans la même présentation

1. **Sélectionner** les éléments de page à copier.
2. Cliquez sur **Copy** dans le menu **Home**.
3. **Naviguer** vers une autre page à l'aide des boutons de navigation dans le menu **View**.
4. Cliquez sur **Paste** dans le menu **Home**.

* Les éléments seront collés à la **même position** que les originaux.

***

## Pour copier des éléments de page vers une autre présentation

1. **Sélectionner** les éléments à copier.
2. Cliquez sur **Copy** dans le menu **Home**.
3. **Ouvrir** la présentation cible (si elle n'est pas encore ouverte), ou cliquez sur son onglet pour la rendre active.
4. **Naviguer** vers la page cible à l'aide du menu **View**.
5. Cliquez sur **Paste** dans le menu **Home**.

* Les éléments seront insérés à la **même position** que dans la présentation source.
* Si un élément est connecté à un **fournisseur de données**, ce fournisseur de données sera **également copié** dans la nouvelle présentation (s'il n'y est pas déjà).

***

## Remarques

* Les éléments collés seront **uniquement** placés à la position choisie s'ils **s'inscrivent dans les limites de la page**.
* S'ils ne s'y inscrivent pas, le système **ajustera automatiquement** la position pour s'assurer que tous les éléments se trouvent dans la page.
* Pour coller des éléments à une **position spécifique** :
  * **Clic droit** sur la position cible sur la page.
  * Sélectionner **Paste** dans le menu contextuel — les éléments y seront collés si l'espace le permet.

***

## Conseil

* **Raccourci pour copier/coller** :
  * **\[Ctrl] + C** pour copier
  * **\[Ctrl] + V** pour coller

* Si vous souhaitez **copier une page entière** d'une présentation à une autre, utilisez l'assistant **Import Page Wizard** :
  * Cliquez sur **Import Page** dans le menu **Page**.
  * Cela copiera la page entière **y compris ses propriétés**.
