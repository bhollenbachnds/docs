Les éléments de page peuvent être **placés devant ou derrière** d'autres éléments sur la page. Pour contrôler quel élément apparaît au-dessus, vous devez ajuster **l'ordre d'affichage**.

### Conseil

Utilisez la **fenêtre Elements** pour vérifier et gérer l'ordre d'affichage actuel des éléments de page. La liste reflète l'ordre d'empilement de l'arrière-plan au premier plan.

***

## Comment modifier l'ordre d'affichage des éléments de page

### Étape 1 : Sélectionner les éléments

Sélectionnez les éléments de page dont vous souhaitez modifier l'ordre d'affichage. (Référez-vous à *Selecting Page Elements* pour obtenir de l'aide.)

### Étape 2 : Ajuster l'ordre

Allez à **Order** dans le **menu Elements** et choisissez l'une des options suivantes :

| Élément de menu             | Fonction                                                               |
| --------------------------- | ---------------------------------------------------------------------- |
| **Mettre au premier plan**  | Placer les éléments sélectionnés au premier plan (avant-plan).         |
| **Avancer**                 | Déplacer les éléments sélectionnés d'une couche vers l'avant.          |
| **Mettre à l'arrière-plan** | Placer les éléments sélectionnés au niveau le plus bas (arrière-plan). |
| **Reculer**                 | Déplacer les éléments sélectionnés d'une couche vers l'arrière.        |

***

## Méthode alternative : Utilisation de la fenêtre Éléments

1. Assurez-vous que la **fenêtre Éléments** est visible.
2. Cliquez et maintenez l'élément dans la liste Éléments.
3. **Glissez-déposez** l'élément vers une nouvelle position dans la liste pour ajuster son ordre d'affichage.
4. Relâchez le bouton de la souris pour confirmer le nouvel ordre.

***

### Remarques importantes

* **Les éléments Flash** et **les éléments de page web** (sauf ceux fonctionnant en mode capture d'écran) sont **toujours placés au premier plan** des autres éléments de page, indépendamment de l'ordre d'affichage spécifié.
* Si un élément de page ne **chevauche pas** d'autres éléments, l'ajustement de son ordre (par exemple, "Avancer") peut n'avoir **aucun effet visible**.
* Si vous sélectionnez **plusieurs éléments**, les ajustements d'ordre d'affichage seront appliqués **individuellement** à chaque élément sélectionné.
