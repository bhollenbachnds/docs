Les éléments de page peuvent être **placés devant ou derrière** d'autres éléments sur la page. Pour contrôler quel élément apparaît au premier plan, vous devez ajuster l'**ordre d'affichage**.

### Conseil

Utilisez la **fenêtre Éléments** pour vérifier et gérer l'ordre d'affichage actuel des éléments de page. La liste reflète l'ordre d'empilement de l'arrière-plan au premier plan.

***

## Comment modifier l'ordre d'affichage des éléments de page

### Étape 1 : Sélectionner les éléments

Sélectionnez les éléments de page pour lesquels vous souhaitez modifier l'ordre d'affichage. (Consultez *Sélectionner des éléments de page* pour obtenir de l'aide.)

### Étape 2 : Ajuster l'ordre

Accédez à **Ordre** dans le **menu Éléments** et choisissez l'une des options suivantes :

| Élément de menu              | Fonction                                                        |
| ---------------------------- | --------------------------------------------------------------- |
| **Mettre au premier plan**   | Placer les éléments sélectionnés en haut (premier plan).        |
| **Avancer**                  | Déplacer les éléments sélectionnés d'une couche vers l'avant.   |
| **Envoyer à l'arrière-plan** | Placer les éléments sélectionnés en bas (arrière-plan).         |
| **Reculer**                  | Déplacer les éléments sélectionnés d'une couche vers l'arrière. |

***

## Méthode alternative : Utiliser la fenêtre Éléments

1. Assurez-vous que la **fenêtre Éléments** est visible.
2. Cliquez et maintenez l'élément dans la liste Éléments.
3. **Glissez-déposez** l'élément vers une nouvelle position dans la liste pour ajuster son ordre d'affichage.
4. Relâchez le bouton de la souris pour confirmer le nouvel ordre.

***

### Remarques importantes

* **Les éléments Flash** et **les éléments de page web** (sauf ceux fonctionnant en mode capture d'écran) sont **toujours placés au premier plan** par rapport aux autres éléments de page, quel que soit l'ordre d'affichage spécifié.
* Si un élément de page ne **se chevauche pas** avec d'autres éléments, l'ajustement de son ordre (par exemple, « Avancer ») peut n'avoir **aucun effet visible**.
* Si vous sélectionnez **plusieurs éléments**, les ajustements de l'ordre d'affichage seront appliqués **individuellement** à chaque élément sélectionné.
