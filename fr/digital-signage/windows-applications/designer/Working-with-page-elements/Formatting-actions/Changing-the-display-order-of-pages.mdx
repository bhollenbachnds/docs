Les éléments de page peuvent être **placés devant ou derrière** d'autres éléments sur la page. Pour contrôler quel élément apparaît au-dessus, vous devez ajuster **l'ordre d'affichage**.

### Conseil

Utilisez la **fenêtre Éléments** pour vérifier et gérer l'ordre d'affichage actuel des éléments de page. La liste reflète l'ordre d'empilement de l'arrière-plan au premier plan.

***

## Comment modifier l'ordre d'affichage des éléments de page

### Étape 1 : Sélectionnez les éléments

Sélectionnez les éléments de page pour lesquels vous souhaitez modifier l'ordre d'affichage. (Référez-vous à *Selecting Page Elements* pour obtenir de l'aide.)

### Étape 2 : Ajustez l'ordre

Allez à **Ordre** sur le **Elements menu** et choisissez l'une des options suivantes :

| Option de menu              | Fonction                                                        |
| --------------------------- | --------------------------------------------------------------- |
| **Mettre au premier plan**  | Placer les éléments sélectionnés au-dessus (premier plan).      |
| **Avancer**                 | Déplacer les éléments sélectionnés d'une couche vers l'avant.   |
| **Mettre à l'arrière-plan** | Placer les éléments sélectionnés au fond (arrière-plan).        |
| **Reculer**                 | Déplacer les éléments sélectionnés d'une couche vers l'arrière. |

***

## Méthode alternative : Utilisation de la fenêtre Elements

1. Assurez-vous que la **Elements window** est visible.
2. Cliquez et maintenez l'élément dans la liste Elements.
3. **Glisser-déposer** l'élément vers une nouvelle position dans la liste pour ajuster son ordre d'affichage.
4. Relâchez le bouton de la souris pour confirmer le nouvel ordre.

***

### Remarques importantes

* **Flash elements** et **web page elements** (sauf ceux fonctionnant en mode capture d'écran) sont **toujours placés au premier plan** des autres éléments de la page, indépendamment de l'ordre d'affichage spécifié.
* Si un élément de page ne **chevauche pas** d'autres éléments, l'ajustement de son ordre (par exemple, "Avancer") peut n'avoir **aucun effet visible**.
* Si vous sélectionnez **plusieurs éléments**, les ajustements d'ordre d'affichage seront appliqués **individuellement** à chaque élément sélectionné.
