**Page elements** dans une page PADS4 Designer peuvent être alignés **les uns par rapport aux autres** ou alignés **sur la grille** pour une mise en page précise et cohérente.

***

## Alignement des éléments de page les uns par rapport aux autres

1. **Sélectionnez au moins deux éléments de page.**\
   (Voir *Sélection des éléments de page* pour des instructions détaillées.)

2. **Faites d'un élément l'élément principal sélectionné** (la référence pour l'alignement).

3. Cliquez sur **Align** dans le menu **Elements**.

4. Choisissez l'une des options d'alignement suivantes :

| Élément de menu | Fonction                                                                                                    |
| --------------- | ----------------------------------------------------------------------------------------------------------- |
| **Lefts**       | Aligne horizontalement tous les éléments sélectionnés sur la **gauche** de l'élément principal sélectionné. |
| **Centers**     | Aligne horizontalement tous les éléments sélectionnés sur le **centre** de l'élément principal sélectionné. |
| **Rights**      | Aligne horizontalement tous les éléments sélectionnés sur la **droite** de l'élément principal sélectionné. |
| **Tops**        | Aligne verticalement tous les éléments sélectionnés sur le **haut** de l'élément principal sélectionné.     |
| **Middles**     | Aligne verticalement tous les éléments sélectionnés sur le **milieu** de l'élément principal sélectionné.   |
| **Bottoms**     | Aligne verticalement tous les éléments sélectionnés sur le **bas** de l'élément principal sélectionné.      |

***

## Alignement des éléments de page sur la grille

1. **Sélectionnez** les éléments de page que vous souhaitez aligner sur la grille.

2. Cliquez sur **Align** dans le menu **Elements**, puis sélectionnez **To Grid**.

***

## Remarques

* **Au moins deux éléments** doivent être sélectionnés pour activer les fonctions d'alignement relatif entre eux.
* Si **Snap to Grid** est déjà activé, l'utilisation de **Align to Grid** n'**aura pas d'effet supplémentaire**, car les éléments s'aligneront déjà sur les points de grille les plus proches lorsqu'ils sont déplacés ou redimensionnés.
