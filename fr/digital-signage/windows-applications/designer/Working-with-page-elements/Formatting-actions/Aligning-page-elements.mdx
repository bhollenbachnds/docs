**Éléments de page** dans une page PADS4 Designer peuvent être alignés **les uns par rapport aux autres** ou alignés **sur la grille** pour une mise en page précise et cohérente.

***

## Aligner les éléments de page les uns par rapport aux autres

1. **Sélectionnez au moins deux éléments de page.**\
   (Voir *Selecting Page Elements* pour des instructions détaillées.)

2. **Définissez un élément comme élément sélectionné principal** (la référence pour l'alignement).

3. Cliquez sur **Align** dans le menu **Elements**.

4. Choisissez l'une des options d'alignement suivantes :

| Élément de menu | Fonction                                                                                                  |
| --------------- | --------------------------------------------------------------------------------------------------------- |
| **Lefts**       | Aligne tous les éléments sélectionnés horizontalement à **gauche** de l'élément sélectionné principal.    |
| **Centers**     | Aligne tous les éléments sélectionnés horizontalement au **centre** de l'élément sélectionné principal.   |
| **Rights**      | Aligne tous les éléments sélectionnés horizontalement à **droite** de l'élément sélectionné principal.    |
| **Tops**        | Aligne tous les éléments sélectionnés verticalement sur le **haut** de l'élément sélectionné principal.   |
| **Milieux**     | Aligne tous les éléments sélectionnés verticalement sur le **milieu** de l'élément sélectionné principal. |
| **Bas**         | Aligne tous les éléments sélectionnés verticalement sur le **bas** de l'élément sélectionné principal.    |

***

## Alignement des éléments de page sur la grille

1. **Sélectionnez** les éléments de page que vous souhaitez aligner sur la grille.

2. Cliquez sur **Aligner** dans le **menu Éléments**, puis sélectionnez **Sur la grille**.

***

## Remarques

* **Au moins deux éléments** doivent être sélectionnés pour activer les fonctions d'alignement les uns par rapport aux autres.
* Si **Aligner sur la grille** est déjà activé, l'utilisation de **Aligner sur la grille** n'aura **aucun effet supplémentaire**, car les éléments s'aligneront déjà sur les points de grille les plus proches lors de leur déplacement ou redimensionnement.
