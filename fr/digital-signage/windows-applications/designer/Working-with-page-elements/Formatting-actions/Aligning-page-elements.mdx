**Éléments de page** dans une page PADS4 Designer peuvent être alignés **les uns par rapport aux autres** ou alignés **sur la grille** pour une mise en page précise et cohérente.

***

## Aligner les éléments de page les uns par rapport aux autres

1. **Sélectionnez au moins deux éléments de page.**\
   (Voir *Selecting Page Elements* pour des instructions détaillées.)

2. **Définissez un élément comme élément principal sélectionné** (la référence pour l'alignement).

3. Cliquez sur **Align** dans le menu **Elements**.

4. Choisissez l'une des options d'alignement suivantes :

| Élément de menu | Fonction                                                                                                         |
| --------------- | ---------------------------------------------------------------------------------------------------------------- |
| **Lefts**       | Aligne tous les éléments sélectionnés horizontalement sur le **bord gauche** de l'élément principal sélectionné. |
| **Centers**     | Aligne tous les éléments sélectionnés horizontalement sur le **centre** de l'élément principal sélectionné.      |
| **Rights**      | Aligne tous les éléments sélectionnés horizontalement sur le **bord droit** de l'élément principal sélectionné.  |
| **Tops**        | Aligne tous les éléments sélectionnés verticalement sur le **haut** de l'élément principal sélectionné.          |
| **Middles**     | Aligne tous les éléments sélectionnés verticalement sur le **milieu** de l'élément principal sélectionné.        |
| **Bottoms**     | Aligne tous les éléments sélectionnés verticalement sur le **bas** de l'élément principal sélectionné.           |

***

## Aligner les éléments de page sur la grille

1. **Sélectionnez** les éléments de page que vous souhaitez aligner sur la grille.

2. Cliquez sur **Align** dans le menu **Elements**, puis sélectionnez **To Grid**.

***

## Remarques

* **Au moins deux éléments** doivent être sélectionnés pour activer les fonctions d'alignement les uns par rapport aux autres.
* Si **Snap to Grid** est déjà activé, l'utilisation de **Align to Grid** **n'aura aucun effet supplémentaire**, car les éléments s'aligneront déjà sur les points de grille les plus proches lors de leur déplacement ou redimensionnement.
