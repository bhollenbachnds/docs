Grâce aux fonctions de rotation, vous pouvez **placer les éléments de page à n'importe quel angle souhaité** pour améliorer la conception de votre présentation.

***

## Comment faire pivoter un ou plusieurs éléments de page

### Option 1 : Utilisation du menu Éléments

1. **Sélectionnez** les éléments de page que vous souhaitez faire pivoter. (Voir *Sélection des éléments de page* pour obtenir des instructions.)
2. Accédez au **menu Éléments** et cliquez sur **Rotation**.
3. Choisissez l'une des options suivantes :

| Élément de menu                 | Fonction                                                                                            |
| ------------------------------- | --------------------------------------------------------------------------------------------------- |
| **Rotation droite 90°**         | Fait pivoter tous les éléments sélectionnés de 90° dans le sens des aiguilles d'une montre.         |
| **Rotation gauche 90°**         | Fait pivoter tous les éléments sélectionnés de 90° dans le sens inverse des aiguilles d'une montre. |
| **Rotation 180°**               | Fait pivoter tous les éléments sélectionnés de 180°.                                                |
| **Plus d'options de rotation…** | Ouvre une boîte de dialogue d'options de rotation détaillées pour l'élément principal sélectionné.  |

***

### Option 2 : Utilisation de la poignée de rotation

1. **Sélectionnez** les éléments de page que vous souhaitez faire pivoter.
2. Placez le pointeur de la souris sur la **poignée de rotation** (généralement au-dessus ou près de l'élément).
3. **Appuyez et maintenez** le bouton gauche de la souris.
4. **Faites glisser** la souris pour définir l'angle de rotation souhaité.
5. **Relâchez** le bouton de la souris pour appliquer la rotation.

***

### Conseil

* Pour **annuler** la rotation pendant le glissement, appuyez sur la touche **Échap** de votre clavier.

***

## Remarques

* Si **Verrouiller les éléments** est activé dans le **menu Accueil**, la rotation via la poignée est **désactivée**.
* **Flash** et **les éléments de page Web** **ne peuvent pas être pivotés**.
* Utilisez **Plus d'options de rotation…** pour définir un angle exact ou pour affiner la rotation au-delà des incréments de 90°.
