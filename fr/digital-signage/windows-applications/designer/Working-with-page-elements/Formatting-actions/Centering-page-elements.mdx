Les éléments de page dans PADS4 Designer peuvent être **centrés horizontalement, verticalement**, ou **les deux** dans une page pour un meilleur alignement visuel.

***

## Comment centrer les éléments de page dans la page actuelle

1. **Sélectionnez** les éléments de page que vous souhaitez centrer.\
   (Voir *Sélection des éléments de page* pour plus de détails.)

2. Cliquez sur **Centrer** dans le menu **Éléments**.

3. Choisissez l'une des options de centrage suivantes :

| Élément de menu          | Fonction                                                                                      |
| ------------------------ | --------------------------------------------------------------------------------------------- |
| **Horizontal**           | Centre les éléments sélectionnés **horizontalement** dans la page.                            |
| **Vertical**             | Centre les éléments sélectionnés **verticalement** dans la page.                              |
| **Centrer dans la page** | Centre les éléments sélectionnés **à la fois horizontalement et verticalement** dans la page. |

***

## Remarques

* Lorsque **plusieurs éléments** sont sélectionnés, ils sont traités **comme un groupe** et centrés **dans leur ensemble**, pas individuellement.
* Le centrage aide à garantir que les éléments sont visuellement équilibrés dans l'espace de conception.
