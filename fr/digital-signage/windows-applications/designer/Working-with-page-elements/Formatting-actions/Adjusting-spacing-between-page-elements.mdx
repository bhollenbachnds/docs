L'**espacement horizontal** ou **vertical** entre les éléments de la page peut être facilement rendu **égal**, **augmenté**, **diminué**, ou **supprimé** pour obtenir une conception bien alignée.

***

## Étapes pour ajuster l'espacement

### 1. Sélectionner les éléments

* **Sélectionnez au moins deux éléments de page** que vous souhaitez ajuster. (Voir *Sélection des éléments de page* pour plus de détails.)
* **Marquez un élément** comme **élément principal sélectionné**.

### 2. Ouvrir les options d'espacement

* Cliquez sur **Espacement** dans le menu **Éléments**.
* Choisissez **Horizontal** ou **Vertical**, selon la direction dans laquelle vous souhaitez ajuster l'espacement.

### 3. Choisir une option d'espacement

| Élément de menu | Fonction                                                                          |
| --------------- | --------------------------------------------------------------------------------- |
| **Égal**        | Rendre l'espacement **égal** entre tous les éléments sélectionnés.                |
| **Augmenter**   | **Augmenter** l'espacement entre tous les éléments sélectionnés.                  |
| **Diminuer**    | **Diminuer** l'espacement entre tous les éléments sélectionnés.                   |
| **Supprimer**   | **Supprimer** l'espacement pour que les éléments soient côte à côte sans espaces. |

***

## Remarques

* **Au moins deux éléments** doivent être sélectionnés pour activer les ajustements d'espacement.
* Les ajustements sont effectués **par rapport à l'élément principal sélectionné**.\
  Example: When **lors de la suppression de l'espacement**, **tous les autres éléments se déplacent vers l'élément principal sélectionné**.
* Si **Aligner sur la grille** est activé :
  * **Les ajustements d'espacement** déplaceront les éléments de **un point de grille**.
* Si **Aligner sur la grille** est désactivé :
  * **Les ajustements d'espacement** déplaceront les éléments de **un pixel**.
