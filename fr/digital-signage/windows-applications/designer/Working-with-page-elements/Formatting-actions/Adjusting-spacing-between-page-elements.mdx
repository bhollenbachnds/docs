Le **horizontal** ou **espacement vertical** entre les éléments de page peut être facilement rendu **égal**, **augmenté**, **diminué**, ou **supprimé** pour obtenir une conception bien alignée.

***

## Étapes pour ajuster l'espacement

### 1. Sélectionner les éléments

* **Sélectionnez au moins deux éléments de page** que vous souhaitez ajuster. (Voir *Selecting Page Elements* pour plus de détails.)
* **Marquez un élément** comme **primary selected item**.

### 2. Ouvrir les options d'espacement

* Cliquez sur **Spacing** dans le menu **Elements**.
* Choisissez **Horizontal** ou **Vertical**, selon la direction dans laquelle vous souhaitez ajuster l'espacement.

### 3. Choisir une option d'espacement

| Élément de menu | Fonction                                                                          |
| --------------- | --------------------------------------------------------------------------------- |
| **Equal**       | Rendre l'espacement **égal** entre tous les éléments sélectionnés.                |
| **Increase**    | **Augmenter** l'espacement entre tous les éléments sélectionnés.                  |
| **Decrease**    | **Diminuer** l'espacement entre tous les éléments sélectionnés.                   |
| **Remove**      | **Supprimer** l'espacement pour que les éléments soient côte à côte sans espaces. |

***

## Remarques

* **Au moins deux éléments** doivent être sélectionnés pour activer les ajustements d'espacement.
* Les ajustements sont effectués **par rapport à l'élément sélectionné principal**.\
  Example: When **lors de la suppression de l'espacement**, **tous les autres éléments se déplacent vers l'élément sélectionné principal**.
* Si **Snap to Grid** est activé :
  * **Les ajustements d'espacement** déplaceront les éléments de **un point de grille**.
* Si **Snap to Grid** est désactivé :
  * **Les ajustements d'espacement** déplaceront les éléments de **un pixel**.
