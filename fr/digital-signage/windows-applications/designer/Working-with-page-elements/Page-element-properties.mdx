Chaque élément de page possède un ensemble de **propriétés** qui déterminent son comportement et son apparence dans la page.

***

## Pour modifier les propriétés d'un seul élément de page

1. **Sélectionnez** l'élément (par exemple, en cliquant dessus).
2. Cliquez sur **Element Properties** dans le **menu Elements**.
3. Ajustez un ou plusieurs paramètres dans la fenêtre des propriétés.
4. Cliquez sur **OK** pour enregistrer les modifications et revenir à votre présentation, ou cliquez sur **Apply** pour enregistrer les modifications sans fermer la fenêtre des propriétés.

***

## Pour modifier les propriétés de plusieurs éléments de page

1. **Sélectionnez plusieurs éléments** (voir *Selecting multiple page elements*).
2. Assurez-vous que l'élément correct est marqué comme **primary selected item**.
3. Cliquez sur **Element Properties** dans le **menu Elements**.
4. Ajustez les propriétés dans la fenêtre. Notez que cette fenêtre affiche les propriétés de l'**primary selected item**.
5. Cliquez sur **OK** ou **Apply** pour appliquer les modifications à tous les éléments sélectionnés.

***

### Astuce

La modification des propriétés pour plusieurs éléments en une seule action **fait gagner du temps** et **réduit les erreurs**.

Si vous sélectionnez différents types d'éléments (comme du texte, des images, des vidéos), lorsque vous appliquez des modifications :

* Le système vérifie si chaque élément prend en charge la propriété.
* Si elle est prise en charge, la propriété est mise à jour.
* Si elle n'est pas prise en charge, cet élément est ignoré.

***

## Remarques

* Si l'option **show preview** est activée dans la fenêtre des propriétés, une **fenêtre d'aperçu** apparaîtra pour montrer immédiatement l'effet de vos modifications.
* Si vous modifiez quelque chose par accident, utilisez **Undo** dans le menu Home pour annuler les modifications.
* **Remarque** : Cliquer sur **OK** enregistre automatiquement les modifications. Vous **n'avez pas besoin de cliquer sur Apply** avant de cliquer sur OK — les deux actions ont le même effet lorsqu'elles sont combinées.
