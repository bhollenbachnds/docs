Chaque élément de page possède un ensemble de **propriétés** qui déterminent son comportement et son apparence dans la page.

***

## Pour modifier les propriétés d'un seul élément de page

1. **Sélectionnez** l'élément (par exemple, en cliquant dessus).
2. Cliquez sur **Propriétés de l'élément** dans le **menu Éléments**.
3. Ajustez un ou plusieurs paramètres dans la fenêtre des propriétés.
4. Cliquez sur **OK** pour enregistrer les modifications et revenir à votre présentation, ou cliquez sur **Appliquer** pour enregistrer les modifications sans fermer la fenêtre des propriétés.

***

## Pour modifier les propriétés de plusieurs éléments de page

1. **Sélectionnez plusieurs éléments** (voir *Sélection de plusieurs éléments de page*).
2. Assurez-vous que l'élément correct est marqué comme **élément principal sélectionné**.
3. Cliquez sur **Propriétés de l'élément** dans le **menu Éléments**.
4. Ajustez les propriétés dans la fenêtre. Notez que cette fenêtre affiche les propriétés de **l'élément principal sélectionné**.
5. Cliquez sur **OK** ou **Appliquer** pour appliquer les modifications à tous les éléments sélectionnés.

***

### Astuce

Modifier les propriétés de plusieurs éléments en une seule action **fait gagner du temps** et **réduit les erreurs**.

Si vous sélectionnez différents types d'éléments (comme du texte, des images, des vidéos), lorsque vous appliquez des modifications :

* Le système vérifie si chaque élément prend en charge la propriété.
* Si elle est prise en charge, la propriété est mise à jour.
* Si elle n'est pas prise en charge, cet élément est ignoré.

***

## Remarques

* Si l'option **afficher l'aperçu** est activée dans la fenêtre des propriétés, une **fenêtre d'aperçu** apparaîtra pour montrer immédiatement l'effet de vos modifications.
* Si vous modifiez quelque chose par accident, utilisez **Annuler** dans le menu Accueil pour annuler les modifications.
* **Remarque** : Cliquer sur **OK** enregistre automatiquement les modifications. Vous **n'avez pas besoin de cliquer sur Appliquer** avant de cliquer sur OK — les deux actions ont le même effet lorsqu'elles sont combinées.
