Les éléments de page qui ne sont plus nécessaires peuvent simplement être supprimés.

***

## Pour supprimer un ou plusieurs éléments de page de la page actuelle :

1. **Sélectionner** les éléments de page que vous souhaitez supprimer (voir *Sélection des éléments de page* pour plus de détails).
2. Cliquez sur **Supprimer les éléments** dans le **Éléments** menu.

***

## Tip:

* Si la **Zone de conception** ou la **fenêtre Éléments** est active, vous pouvez également appuyer sur la **touche \[Suppr]** de votre clavier pour supprimer les éléments de page sélectionnés.

***

## Remarks:

* Si vous avez **supprimé accidentellement** des éléments de page, vous pouvez toujours cliquer sur **Annuler** dans le **Accueil** menu pour restaurer la situation précédente.
