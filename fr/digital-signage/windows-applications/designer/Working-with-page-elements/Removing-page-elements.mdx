Les éléments de page qui ne sont plus nécessaires peuvent simplement être supprimés.

***

## Pour supprimer un ou plusieurs éléments de page de la page actuelle :

1. **Select** les éléments de page que vous souhaitez supprimer (voir *Selecting page elements* pour plus de détails).
2. Cliquez sur **Remove Elements** dans le menu **Elements**.

***

## Tip:

* Si la **Design area** ou la **Elements window** est active, vous pouvez également appuyer sur la touche **\[Del] key** de votre clavier pour supprimer les éléments de page sélectionnés.

***

## Remarks:

* Si vous avez **accidentally removed** des éléments de page, vous pouvez toujours cliquer sur **Undo** dans le menu **Home** pour restaurer la situation précédente.
