Les éléments de page qui ne sont plus nécessaires peuvent simplement être supprimés.

***

## Pour supprimer un ou plusieurs éléments de page de la page actuelle :

1. **Sélectionnez** les éléments de page que vous souhaitez supprimer (voir *Sélection des éléments de page* pour plus de détails).
2. Cliquez sur **Remove Elements** dans le **Elements** menu.

***

## Tip:

* Si la **Design area** ou la **Elements window** est active, vous pouvez également appuyer sur la touche **\[Del]** de votre clavier pour supprimer les éléments de page sélectionnés.

***

## Remarks:

* Si vous avez **supprimé accidentellement** des éléments de page, vous pouvez toujours cliquer sur **Undo** dans le **Home** menu pour restaurer la situation précédente.
