Les éléments de page qui ne sont plus nécessaires peuvent être simplement supprimés.

***

## Pour supprimer un ou plusieurs éléments de page de la page actuelle :

1. **Sélectionnez** les éléments de page que vous souhaitez supprimer (voir *Sélection des éléments de page* pour plus de détails).
2. Cliquez sur **Supprimer les éléments** dans le menu **Éléments**.

***

## Tip:

* Si la **zone de conception** ou la **fenêtre Éléments** est active, vous pouvez également appuyer sur la touche **\[Suppr]** de votre clavier pour supprimer les éléments de page sélectionnés.

***

## Remarks:

* Si vous avez **supprimé accidentellement** des éléments de page, vous pouvez toujours cliquer sur **Annuler** dans le menu **Accueil** pour restaurer la situation précédente.
