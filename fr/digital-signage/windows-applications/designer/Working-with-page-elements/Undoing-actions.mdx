Chaque fois que vous effectuez une tâche qui modifie la conception de votre page, PADS4 Designer prend un instantané de la situation actuelle. Cela vous permet de **revenir** aux états précédents lorsque vous faites une erreur. PADS4 Designer offre des fonctions intégrées pour se déplacer **en arrière** et **en avant** dans vos étapes de conception.

***

## Pour revenir d'une étape en arrière dans la conception de votre page :

* Cliquez sur **Annuler** dans le menu **Accueil**.

Vous pouvez utiliser **Annuler** plusieurs fois pour revenir plusieurs étapes en arrière. Après être revenu en arrière, vous pouvez également avancer à nouveau si nécessaire.

***

## Pour avancer d'une étape dans la conception de votre page :

* Cliquez sur **Rétablir** dans le menu **Accueil**.

***

## Pour accéder directement à une étape spécifique dans la conception de votre page :

1. Cliquez sur le menu déroulant **Liste d'annulation** dans le menu **Accueil**.
2. Sélectionnez l'étape que vous souhaitez restaurer dans la liste.

***

## Remarks:

* **Annuler** et **Rétablir** ne s'appliquent que **dans la même page**.
* Chaque page conserve son **propre historique des étapes de conception**.
* Lorsque vous passez d'une page à l'autre, vous verrez la liste d'annulation correspondant à cette page spécifique.
