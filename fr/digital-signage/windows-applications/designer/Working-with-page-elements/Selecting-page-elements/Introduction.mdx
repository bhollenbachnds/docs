Pour effectuer **n'importe quelle action** (comme déplacer, redimensionner ou supprimer) sur les éléments de la page, vous devez **sélectionnez-les d'abord**.\
Parfois les éléments sont **sélectionnés automatiquement** (par exemple, juste après leur création), mais dans d'autres cas, **une sélection manuelle** est nécessaire.

Lorsqu'un élément de page est sélectionné, il affiche **huit poignées de préhension** — une à chaque coin et une au milieu de chaque côté.

***

## **Dans cette section**

### 1. Sélection d'un élément de page

Explique comment **sélectionner un seul élément** sur une page.

### 2. Sélection de plusieurs éléments de page

Montre comment **sélectionner plusieurs éléments** pour effectuer des actions de groupe comme le déplacement ou le redimensionnement.

### 3. Sélection de tous les éléments de page

Décrit comment **sélectionner tous les éléments d'une page à la fois** pour des opérations en masse.

### 4. Inversion de la sélection

Explique comment **inverser la sélection actuelle** — désélectionner les éléments sélectionnés et sélectionner les éléments non sélectionnés.

### 5. Effacement de la sélection

Décrit comment **désélectionner** tous les éléments de page actuellement sélectionnés.

### 6. Élément principal sélectionné

Couvre le concept d'**élément principal sélectionné**, qui a une fonction spéciale lors du travail avec plusieurs éléments sélectionnés (par exemple, pour l'alignement ou les références de taille).
