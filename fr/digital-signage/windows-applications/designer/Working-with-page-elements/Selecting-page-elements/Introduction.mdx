Pour effectuer **toute action** (telle que déplacer, redimensionner ou supprimer) sur des éléments de page, vous devez **d'abord les sélectionner**.\
Parfois, les éléments sont **sélectionnés automatiquement** (par exemple, juste après leur création), mais dans d'autres cas, une **sélection manuelle** est requise.

Lorsqu'un élément de page est sélectionné, il affiche **huit poignées de saisie** — une à chaque coin et une au milieu de chaque côté.

### 2. Sélectionner plusieurs éléments de page

Décrit comment **sélectionner tous les éléments d'une page en une seule fois** pour des opérations en masse.

### 4. Inverser la sélection

Explique comment **inverser la sélection actuelle** — désélectionner les éléments sélectionnés et sélectionner les éléments non sélectionnés.

### 5. Effacer la sélection

Décrit comment **désélectionner** tous les éléments de page actuellement sélectionnés.

### 6. Élément sélectionné principal

Couvre le concept d'un **élément sélectionné principal**, qui a une fonction spéciale lors de l'utilisation de plusieurs éléments sélectionnés (par exemple, références d'alignement ou de taille).
