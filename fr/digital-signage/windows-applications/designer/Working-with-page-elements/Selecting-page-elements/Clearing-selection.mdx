Si vous ne souhaitez plus qu'aucun élément de page soit sélectionné, vous pouvez **facilement effacer la sélection**.

***

## Pour Effacer la Sélection sur la Page Actuelle

* Allez au menu **Elements**.
* Cliquez sur **Clear Selection**.

***

### Conseil

Vous pouvez également **cliquer sur l'arrière-plan** de votre page pour effacer rapidement toute sélection actuelle.
