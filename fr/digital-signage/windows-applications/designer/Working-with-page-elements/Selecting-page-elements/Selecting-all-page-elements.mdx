Tous les éléments de page sur la **page actuelle** peuvent être rapidement sélectionnés en une **seule opération**.

***

## Pour sélectionner tous les éléments de page sur la page actuelle

* Allez dans le menu **Elements**.
* Cliquez sur **Select All**.

***

Une fois sélectionnés, **tous les éléments** de la page peuvent être manipulés ensemble — y compris le déplacement, le redimensionnement, la copie ou l'application de formatage.
