L'inversion de la sélection vous permet de **permuter rapidement** les éléments de page sélectionnés et non sélectionnés sur la **page actuelle**.\
Tous les **éléments sélectionnés** deviendront **non sélectionnés**, et tous les **éléments non sélectionnés** deviendront **sélectionnés**.

***

## Pour inverser la sélection sur la page actuelle

* Allez au menu **Elements**.
* Cliquez sur **Invert Selection**.

***

### Astuce

Si vous souhaitez sélectionner **presque tous** les éléments de la page, il est souvent plus rapide de :

1. **Sélectionner** uniquement les quelques éléments que vous **ne voulez pas**.
2. Puis utiliser **Invert Selection** pour sélectionner tout le reste en une seule étape.
