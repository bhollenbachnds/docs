Les éléments de page peuvent être déplacés&#x20;
&#x20; **dans la même page**,&#x20;
&#x20; **vers une autre page dans la même présentation**, ou&#x20;
&#x20; **vers une autre présentation**.
&#x20;&#x20;

***

## Déplacer des éléments de page dans la même page

1. **Sélectionner** les éléments de page que vous souhaitez déplacer (voir *Sélection des éléments de page* pour plus de détails).
2. Déplacer le pointeur de la souris **sur l'un des éléments sélectionnés**, puis appuyer et maintenir le **bouton gauche de la souris**.
3. **Faire glisser** le pointeur de la souris vers la position souhaitée.
4. **Relâcher** le bouton de la souris pour placer les éléments.

### Tips:

* **Annuler** l'opération de déplacement à tout moment en appuyant sur **\[Esc]** sur votre clavier.
* Si **Lock Elements** dans le menu **Home** est actif, le déplacement des éléments de cette façon **ne sera pas possible**.
* Pour un **déplacement précis**, utilisez le clavier :
  * Maintenir **\[Ctrl]** et utiliser les **touches fléchées** pour déplacer les éléments.
  * Si **Snap to Grid** est activé, les éléments se déplacent d'**un point de grille** à la fois.
  * Si **Snap to Grid** est désactivé, les éléments se déplacent d'**un pixel** à la fois.
* Pour ajuster Snap to Grid, accédez à la fenêtre **Options**.

***

## Déplacement des éléments de page vers une autre page dans la même présentation

1. **Sélectionner** les éléments à déplacer.
2. Cliquer sur **Cut** dans le menu **Home**.
3. Naviguer vers la page cible à l'aide du menu **View**.
4. Cliquer sur **Paste** dans le menu **Home**.

* Les éléments seront collés à la **même position** que dans la page d'origine.

***

## Déplacement des éléments de page vers une autre présentation

1. **Sélectionner** les éléments à déplacer.
2. Cliquer sur **Cut** dans le menu **Home**.
3. Ouvrir la présentation cible ou basculer vers son onglet.
4. Naviguer vers la page cible à l'aide du menu **View**.
5. Cliquer sur **Paste** dans le menu **Home**.

* Les éléments seront insérés à la **même position** que dans la présentation d'origine.
* Si un élément est connecté à un **fournisseur de données**, ce fournisseur sera **copié dans la nouvelle présentation** (mais restera également dans l'original).

***

## Remarques

* Les éléments ne seront collés à la **position choisie que s'ils s'inscrivent dans les limites de la page**.
* S'ils ne s'y inscrivent pas, le système **ajustera automatiquement** la position pour s'assurer qu'ils sont visibles.
* Pour coller des éléments à une **position spécifique**, **faire un clic droit** sur cette position et choisir **Paste** dans le menu contextuel. Les éléments y seront insérés si l'espace le permet.
* **Undo** est disponible dans le menu **Home** si vous devez annuler des modifications.

***

## Conseil

Pour déplacer rapidement des éléments vers une autre page ou présentation :

* Utiliser **\[Ctrl] + X** pour couper.
* Utiliser **\[Ctrl] + V** pour coller.
