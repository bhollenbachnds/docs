Les éléments de page peuvent être déplacés **au sein de la même page**, **vers une autre page de la même présentation**, ou **vers une autre présentation**.

***

## Déplacement d'éléments de page au sein de la même page

1. **Sélectionnez** les éléments de page que vous souhaitez déplacer (voir *Sélection d'éléments de page* pour plus de détails).
2. Déplacez le pointeur de la souris **sur l'un des éléments sélectionnés**, puis appuyez et maintenez le **bouton gauche de la souris**.
3. **Faites glisser** le pointeur de la souris vers la position souhaitée.
4. **Relâchez** le bouton de la souris pour placer les éléments.

### Tips:

* **Annulez** l'opération de déplacement à tout moment en appuyant sur **\[Esc]** sur votre clavier.
* Si **Verrouiller les éléments** dans le menu **Accueil** est actif, le déplacement des éléments de cette façon **ne sera pas possible**.
* Pour un **mouvement précis**, utilisez le clavier :
  * Maintenez **\[Ctrl]** et utilisez les **touches fléchées** pour déplacer les éléments.
  * Si **Aligner sur la grille** est activé, les éléments se déplacent d'**un point de grille** à la fois.
  * Si **Aligner sur la grille** est désactivé, les éléments se déplacent d'**un pixel** à la fois.
* Pour ajuster Aligner sur la grille, accédez à la fenêtre **Options**.

***

## Déplacement d'éléments de page vers une autre page dans la même présentation

1. **Sélectionnez** les éléments à déplacer.
2. Cliquez sur **Couper** dans le menu **Accueil**.
3. Naviguez vers la page cible en utilisant le menu **Affichage**.
4. Cliquez sur **Coller** dans le menu **Accueil**.

* Les éléments seront collés à la **même position** que dans la page d'origine.

***

## Déplacement d'éléments de page vers une autre présentation

1. **Sélectionnez** les éléments à déplacer.
2. Cliquez sur **Couper** dans le menu **Accueil**.
3. Ouvrez la présentation cible ou passez à son onglet.
4. Naviguez vers la page cible en utilisant le menu **Affichage**.
5. Cliquez sur **Coller** dans le menu **Accueil**.

* Les éléments seront insérés à la **même position** que dans la présentation d'origine.
* Si un élément est connecté à un **fournisseur de données**, ce fournisseur sera **copié dans la nouvelle présentation** (mais aussi rester dans l'original).

***

## Remarques

* Les éléments ne seront collés à la **position choisie que s'ils s'adaptent aux limites de la page**.
* S'ils ne s'adaptent pas, le système va **ajuster automatiquement** la position pour s'assurer qu'ils sont visibles.
* Pour coller des éléments à une **position spécifique**, **faites un clic droit** sur cette position et choisissez **Coller** dans le menu contextuel. Les éléments seront insérés à cet endroit si l'espace le permet.
* **Annuler** est disponible dans le menu **Home** si vous devez revenir en arrière.

***

## Conseil

Pour déplacer rapidement des éléments vers une autre page ou présentation :

* Utilisez **\[Ctrl] + X** pour couper.
* Utilisez **\[Ctrl] + V** pour coller.
