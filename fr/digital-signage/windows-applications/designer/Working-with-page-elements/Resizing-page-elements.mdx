Les éléments d'une page peuvent être facilement redimensionnés.

***

## Pour redimensionner un ou plusieurs éléments de page sur la page actuelle :

1. **Sélectionnez** les éléments de page que vous souhaitez redimensionner (voir *Sélection des éléments de page* pour plus de détails).
2. Positionnez le pointeur de la souris **sur l'une des poignées** de n'importe quel élément sélectionné.
3. **Appuyez et maintenez** le bouton gauche de la souris.
4. **Faites glisser** le pointeur de la souris pour ajuster la taille des éléments sélectionnés.
5. **Relâchez** le bouton de la souris pour appliquer la nouvelle taille.

***

## Remarks:

* Si **Lock Elements** est activé dans le menu **Home**, le redimensionnement des éléments **ne sera pas** possible.
* Si la taille d'un élément de page est définie sur **Automatic size** (par exemple, les éléments de texte), le redimensionnement **ne sera pas** possible. Cela est indiqué par des **poignées grises** au lieu de noires ou blanches.

***

## Tips:

* Vous pouvez **annuler le redimensionnement** en appuyant sur **\[Esc]** pendant le processus.
* Pour un **redimensionnement précis**, utilisez le clavier :
  * Maintenez **\[Shift]** et utilisez les **touches fléchées** pour redimensionner.
  * Si **Snap to Grid** est activé, les éléments seront redimensionnés d'une unité de grille.
  * Si **Snap to Grid** est désactivé, les éléments seront redimensionnés d'un pixel.
* Pour ajuster les paramètres de Snap to Grid, accédez à la fenêtre **Options**.
* Si vous avez **redimensionné accidentellement** des éléments, vous pouvez toujours cliquer sur **Undo** dans le menu **Home** pour restaurer la situation précédente.
