L'application **PADS4 Designer** vous permet de **prévisualiser les présentations** pour voir exactement comment elles apparaîtront et se comporteront lorsqu'elles seront diffusées dans l'application **PADS4 Viewer**. Cela permet de vérifier la mise en page, le contenu, les connexions de données et l'interactivité avant la publication.

***

## **Comment démarrer une prévisualisation**

### **Prévisualiser la présentation entière**

1. **Cliquez** sur l'onglet **View** dans la barre de ruban.
2. Dans la **section Preview**, cliquez sur **Presentation**.
3. La fenêtre **Presentation Preview** s'ouvrira et **démarrera automatiquement la présentation**.

**Tip:**

* Démarrez rapidement une prévisualisation en appuyant sur **F2**.
* Appuyez sur **F12** pendant la prévisualisation pour le **débogage des éléments web** (lors de l'utilisation du moteur de navigateur Chromium).

***

### **Prévisualiser une page spécifique**

1. Naviguez vers la page que vous souhaitez prévisualiser en utilisant la section **Go To** dans le menu **View**.
2. Suivez les étapes ci-dessus pour démarrer la prévisualisation de la présentation.
3. Alternativement, dans la fenêtre **Presentation Preview** :
   * Utilisez le **menu déroulant de sélection de page** pour choisir une page de départ.
   * Cliquez sur **Start** pour commencer la prévisualisation à partir de cette page.

***

## **Aperçu de la fenêtre de prévisualisation de présentation**

### **Groupe Présentation**

* **Close** : Ferme la fenêtre de prévisualisation.
* **Start** : Démarre la prévisualisation à partir de la page sélectionnée.
* **Stop** : Arrête la présentation en cours.
* **Page & Total** : Affiche le **numéro de page actuel**, le **temps sur la page actuelle** et le **temps d'exécution total**.

***

### **Groupe Zoom**

* **Options d'étirement/zoom** : Ajustez le niveau de zoom pour adapter la fenêtre de prévisualisation.\
  Note: Some content like **web elements** peuvent ne pas s'adapter parfaitement.

***

### **Groupe Options**

* **Viewer Name** : Entrez un nom de Viewer existant pour **simuler** comment ce Viewer spécifique exécuterait la présentation. Utile pour tester les **Viewer Variables** comme `#ViewerName#`.
* **Page** : Sélectionnez la page de départ pour la prévisualisation.
* **Viewer Type** : Choisissez un type de Viewer à **simuler** et supprimez les éléments **non pris en charge** par ce type de Viewer.

***

### **Groupe Source de données**

(Ces options peuvent être ajustées uniquement **avant** de démarrer la prévisualisation)

* **Use Data Cache** :
  * Utilise les **données mises en cache** pour une prévisualisation plus rapide.
  * Évite les appels chronophages aux fournisseurs de données.
  * Essentiel pour travailler hors ligne ou concevoir avec des sources de données lentes/distantes.
  * En savoir plus : **Data Caching in PADS4 Designer**.
* **Use Backup** :
  * Force PADS4 à utiliser des **Data Providers de sauvegarde** (si définis), permettant de simuler des scénarios de repli.

***

### **Groupe Affichage**

* **Viewer Variables** : Ouvre une fenêtre listant **toutes les Viewer Variables** pour cette session de prévisualisation. Voir la page **Viewer Variables** pour plus de détails.
* **Data Preview** : Affiche tous les **Data Providers** connectés et leurs **données récupérées** pour cette présentation.
* **Log Information** :
  * Affiche les journaux avec **erreurs/avertissements** pendant la prévisualisation.
  * Se met à jour automatiquement en temps réel.
  * Le bouton devient **rouge** si un problème est détecté.
  * Affiche directement les 4 derniers messages du journal.

***

## **Notes importantes**

* **Les pages masquées** peuvent toujours être forcées à s'afficher pour la prévisualisation si elles sont choisies comme page de départ.
* Si vous utilisez des **web elements** avec **l'interactivité activée**, appuyez sur **F12** pour ouvrir les **outils de débogage du navigateur Chromium**.\
  (Remarque : Cette fonction n'est pas prise en charge pour le moteur Internet Explorer).

***

| Action                                                   | Touche de raccourci |
| -------------------------------------------------------- | ------------------- |
| Démarrer la prévisualisation complète de la présentation | F2                  |
