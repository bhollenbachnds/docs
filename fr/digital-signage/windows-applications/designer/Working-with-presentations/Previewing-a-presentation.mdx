L'**application PADS4 Designer** vous permet de **prévisualiser les présentations** pour voir exactement comment elles apparaîtront et se comporteront lors de la lecture dans l'**application PADS4 Viewer**. Cela permet de vérifier la mise en page, le contenu, les connexions de données et l'interactivité avant la publication.

***

## **Comment démarrer un aperçu**

### **Prévisualiser la présentation entière**

1. **Cliquez** sur l'onglet **View** dans la barre de ruban.
2. Dans la **section Preview**, cliquez sur **Presentation**.
3. La **fenêtre Presentation Preview** s'ouvrira et **démarrera automatiquement la présentation**.

**Tip:**

* Démarrez rapidement un aperçu en appuyant sur **F2**.
* Appuyez sur **F12** pendant l'aperçu pour le **débogage des éléments web** (lors de l'utilisation du moteur de navigateur Chromium).

***

### **Prévisualiser une page spécifique**

1. Accédez à la page que vous souhaitez prévisualiser en utilisant la section **Go To** dans le menu **View**.
2. Suivez les étapes ci-dessus pour démarrer l'aperçu de la présentation.
3. Alternativement, dans la **fenêtre Presentation Preview** :
   * Utilisez le **menu déroulant de sélection de page** pour choisir une page de départ.
   * Cliquez sur **Start** pour commencer l'aperçu à partir de cette page.

***

## **Aperçu de la fenêtre Presentation Preview**

### **Groupe Presentation**

* **Close**: Ferme la fenêtre d'aperçu.
* **Start**: Démarre l'aperçu à partir de la page sélectionnée.
* **Stop**: Arrête la présentation en cours.
* **Page & Total**: Affiche le **numéro de page actuel**, **le temps sur la page actuelle**, et le **temps de fonctionnement total**.

***

### **Groupe Zoom**

* **Options Stretch/Zoom**: Ajustez le niveau de zoom pour adapter la fenêtre d'aperçu.\
  Note: Some content like **éléments web** peuvent ne pas s'adapter parfaitement.

***

### **Groupe Options**

* **Viewer Name**: Entrez un nom de Viewer existant pour **simuler** la façon dont ce Viewer spécifique exécuterait la présentation. Utile pour tester les **Viewer Variables** comme `#ViewerName#`.
* **Page**: Sélectionnez la page de départ pour l'aperçu.
* **Viewer Type**: Choisissez un Viewer Type pour **simuler** et supprimer les éléments **non pris en charge** par ce Viewer Type.

***

### **Groupe Data Source**

(Ceux-ci ne peuvent être ajustés que **avant** de démarrer l'aperçu)

* **Use Data Cache**:
  * Utilise les **données en cache** pour un aperçu plus rapide.
  * Évite les appels chronophages aux fournisseurs de données.
  * Indispensable pour travailler hors ligne ou concevoir avec des sources de données lentes/distantes.
  * En savoir plus : **Mise en cache des données dans PADS4 Designer**.

* **Use Backup**:
  * Force PADS4 à utiliser les **Data Providers de sauvegarde** (si définis), permettant la simulation de scénarios de repli.

***

### **Groupe Show**

* **Viewer Variables**: Ouvre une fenêtre listant **toutes les Viewer Variables** pour cette session d'aperçu. Voir la page **Viewer Variables** pour plus de détails.
* **Data Preview**: Affiche tous les **Data Providers** connectés et leurs **données récupérées** pour cette présentation.
* **Log Information**:
  * Affiche les journaux avec les **erreurs/avertissements** pendant l'aperçu.
  * Se met à jour automatiquement en temps réel.
  * Le bouton devient **rouge** si un problème est détecté.
  * Afficher les 4 derniers messages du journal directement.

***

## **Remarques importantes**

* **Pages masquées** peuvent toujours être forcées à s'afficher pour la prévisualisation si elles sont choisies comme page de démarrage.
* Si vous utilisez **éléments web** avec **l'interactivité activée**, appuyez sur **F12** pour ouvrir les **outils de débogage du navigateur Chromium**.\
  (Remarque : Ceci n'est pas pris en charge pour le moteur Internet Explorer).

***

## **Récapitulatif des touches de raccourci**

| Action                                                   | Touche de raccourci |
| -------------------------------------------------------- | ------------------- |
| Démarrer la prévisualisation complète de la présentation | F2                  |
| Ouvrir le débogueur web Chromium (pour les éléments web) | F12                 |
