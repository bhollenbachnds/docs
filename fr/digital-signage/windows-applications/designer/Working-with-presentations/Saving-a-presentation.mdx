PADS4 Designer vous permet de **sauvegarder votre présentation** sous son nom actuel ou de créer une **copie sous un nom différent**. Lors de l'enregistrement en tant que copie, la présentation originale reste inchangée, et vous continuez à travailler sur la nouvelle copie sauvegardée.

Par défaut, les présentations sont enregistrées dans la **PADS4 Content Library**, mais vous pouvez également choisir de les enregistrer à d'autres emplacements.

***

## **Pour sauvegarder votre présentation**

1. **Cliquez** `Save` sur le menu **File**.
2. Si la présentation n'a pas été sauvegardée auparavant, **entrez un nom** lorsque vous y êtes invité.
3. Cliquez sur **Save** pour stocker la présentation.

***

## **Pour sauvegarder une copie de votre présentation**

1. **Cliquez** `Save As` sur le menu **File**.
2. Entrez un **nouveau nom** pour la présentation.
3. Cliquez sur **Save** pour créer une copie et continuer à travailler dessus.

***

## **Pour sauvegarder votre présentation dans la Content Library**

1. **Cliquez** sur le menu **File**.
2. **Survolez** l'option `Save As` pour afficher le sous-menu.
3. Cliquez sur **To Content Library** dans le sous-menu.
4. Dans la bibliothèque de contenu, **entrez un nom** et cliquez sur **Save**.

***

## **Pour sauvegarder votre présentation dans un emplacement de fichier**

1. **Cliquez** sur le menu **File**.
2. **Survolez** l'option `Save As` pour afficher le sous-menu.
3. Cliquez sur **To File** dans le sous-menu.
4. Dans la liste des dossiers, **entrez un nom** pour votre présentation.
5. Cliquez sur **Save** pour l'enregistrer à l'emplacement choisi.

***

## **Pour exporter votre présentation avec tout le contenu utilisé (PADS4 Show)**

1. **Cliquez** sur le menu **File**.
2. **Survolez** l'option `Save As` pour afficher le sous-menu.
3. Cliquez sur **Export PADS4 Show** dans le sous-menu.
4. Suivez les instructions dans l'**Export Wizard** pour créer un package qui inclut la présentation et tous ses fichiers de contenu.

***

## **Notes importantes et conseils**

* Un `*` (astérisque) dans l'en-tête de l'onglet de la présentation indique qu'il y a des **unsaved changes**.
* Si PADS4 Designer est ouvert via **PADS4 Scheduler**, vous ne verrez **not** de bouton `Save`. À la place :
  * Utilisez **Schedule...** pour sauvegarder et planifier la présentation directement dans le système PADS4.
  * Utilisez **Update** pour modifier une présentation déjà planifiée.
  * Pour sauvegarder une **local copy**, utilisez `Save As` depuis le menu File.
* Si vous ouvrez une **PADS4 3.1 SR1 presentation**, elle sera **automatically converted** au nouveau format PADS4 (.padsx), garantissant que le fichier original n'est **not overwritten**.
