Les **Interactivity Properties** contrôlent comment les présentations interactives répondent à **l'inactivité de l'utilisateur** et définissent comment **la navigation au clavier** est gérée pendant la lecture.

***

## **Navigation au clavier entre les pages**

### **Activer la navigation à l'aide des touches du clavier**

Lorsque cette option est activée, les utilisateurs peuvent **naviguer entre les pages** de votre présentation en utilisant le clavier sur le **PADS4 Viewer PC**.\
Voici les touches de navigation disponibles :

| Touche du clavier                             | Action                          |
| --------------------------------------------- | ------------------------------- |
| Flèche gauche / Flèche haut / Page précédente | Passer à la **page précédente** |
| Flèche droite / Flèche bas / Page suivante    | Passer à la **page suivante**   |
| Début                                         | Passer à la **première page**   |
| Fin                                           | Passer à la **dernière page**   |

***

## **Expiration (Expiration automatique de page)**

### **Expiration automatique de page**

Lorsqu'elle est activée, la présentation exécutera automatiquement une action spécifiée si **aucune interaction utilisateur** (clic de souris ou appui sur une touche) ne se produit dans un délai défini.

### **Après x seconde(s)**

Définissez le **nombre de secondes** d'inactivité avant qu'une action ne soit déclenchée.

### **Retour à la Page**

Spécifiez quelle **page à laquelle retourner** lorsque le temps d'expiration a été atteint.\
Cela est généralement utilisé pour revenir à une page **home** ou **welcome** après qu'un utilisateur a quitté la présentation interactive.

***

### **Conseil**

La configuration des propriétés d'expiration est particulièrement utile pour les **interactive kiosks** afin de s'assurer que la présentation **revienne à une page par défaut** lorsqu'aucun utilisateur n'interagit.

***

## **Remarques**

* Si **Enable Navigation by Using Keyboard Keys** est activé, **page durations** s'appliquent toujours.\
  Par exemple, si une page est configurée pour s'afficher pendant 30 secondes et qu'aucune touche n'est pressée, le système va automatiquement **passer à la page suivante** lorsque la durée se termine.
* Si **Synchronize Viewers** est activé pour une présentation, **interactivity functions** telles que la navigation au clavier et l'expiration automatique de page **ne fonctionneront plus**, car la présentation doit rester synchronisée sur tous les écrans.
