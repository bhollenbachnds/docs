Les **Interactivity Properties** contrôlent comment les présentations interactives répondent à **user inactivity** et définissent comment **keyboard navigation** est gérée pendant la lecture.

***

## **Page Keyboard Navigation**

### **Enable Navigation by Using Keyboard Keys**

Lorsque cette option est activée, les utilisateurs peuvent **navigate between pages** de votre présentation en utilisant le clavier sur le **PADS4 Viewer PC**.\
Voici les touches de navigation disponibles :

| Keyboard Key                         | Action                        |
| ------------------------------------ | ----------------------------- |
| Left arrow / Up arrow / Page Up      | Passer à la **previous page** |
| Right arrow / Down arrow / Page Down | Passer à la **next page**     |
| Home                                 | Passer à la **first page**    |
| End                                  | Passer à la **last page**     |

***

## **Expiration (Automatic Page Expiration)**

### **Automatic Page Expiration**

Lorsque cette option est activée, la présentation exécutera automatiquement une action spécifiée si **no user interaction** (clic de souris ou appui sur une touche) ne se produit dans un délai défini.

### **After x Second(s)**

Définir le **number of seconds** d'inactivité avant qu'une action ne soit déclenchée.

### **Return to Page**

Spécifier à quelle **page to return to** lorsque le délai d'expiration est atteint.\
Ceci est généralement utilisé pour revenir à une page **home** ou **welcome** après qu'un utilisateur ait quitté la présentation interactive.

***

### **Tip**

La configuration des propriétés d'expiration est particulièrement utile pour les **interactive kiosks** afin de s'assurer que la présentation **resets to a default page** lorsqu'aucun utilisateur n'interagit.

***

## **Remarks**

* Si **Enable Navigation by Using Keyboard Keys** est activé, **page durations** s'appliquent toujours.\
  Par exemple, si une page est configurée pour s'afficher pendant 30 secondes et qu'aucune touche n'est pressée, le système va automatiquement **switch to the next page** lorsque la durée se termine.
* Si **Synchronize Viewers** est activé pour une présentation, **interactivity functions** telles que la navigation au clavier et l'expiration automatique de page **will no longer work**, car la présentation doit rester synchronisée sur tous les écrans.
