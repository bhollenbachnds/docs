La **Data Preview Window** en mode **Presentation Preview mode** vous permet de **visualiser et vérifier** toutes les données récupérées à partir des **Data Providers** utilisés dans votre présentation PADS4. Ceci est particulièrement utile pour confirmer que les **connexions de données en direct** fonctionnent correctement et pour inspecter les données pendant la phase de conception.

***

## **Ouverture de la Data Preview Window**

### Step-by-step:

1. **Démarrez** un **presentation preview** :
   * Allez dans l'onglet **View** dans la barre de ruban.
   * Cliquez sur **Presentation** dans la section **Preview**.
   * La **Presentation Preview window** s'ouvrira et la présentation démarrera automatiquement.
2. Dans la **Presentation Preview window**, cliquez sur **Data Preview** dans la barre de ruban pour ouvrir la **Data Preview Window**.

***

## **Aperçu de la Data Preview Window**

### Ce que vous voyez :

* **Onglets pour Chaque Data Provider** :\
  Chaque **Data Provider** utilisé dans la présentation aura **son propre onglet**. Cliquez sur les onglets pour basculer entre les Data Providers et afficher leurs données.

* **Données en Direct** :\
  Vous voyez les **données réelles** renvoyées par chaque Data Provider, vous montrant ce qui sera affiché dans les éléments de votre présentation connectés à ces données.

***

## **Fonctions de la Data Preview Window**

| **Fonction** | **Description**                                                                                                                             |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------- |
| **Refresh**  | Recharge et met à jour les données pour **all Data Providers**. Cela vous permet de voir les données nouvellement récupérées en temps réel. |
| **Close**    | Ferme la fenêtre Data Preview.                                                                                                              |

***

## **Cas d'Utilisation pour Data Preview**

* **Vérifier la Connexion de Données**: S'assurer que les données provenant de sources externes (par exemple, bases de données, services web, flux RSS) sont correctement récupérées.
* **Test du Contenu Dynamique**: Vérifier comment les données dynamiques apparaîtront dans vos éléments de présentation.
* **Débogage**: Si des données sont manquantes ou incorrectes dans la présentation, utilisez cette fenêtre pour vérifier si la source de données renvoie les résultats attendus.
* **Mises à jour en direct**: Confirmer que les mises à jour dans votre source de données sont reflétées lors de l'actualisation des données.
