La **Fenêtre d'aperçu des données** en mode **Aperçu de présentation** vous permet de **visualiser et vérifier** toutes les données récupérées à partir des **Fournisseurs de données** utilisés dans votre présentation PADS4. Ceci est particulièrement utile pour confirmer que les **connexions de données en direct** fonctionnent correctement et pour inspecter les données pendant la phase de conception.

***

## **Ouverture de la fenêtre d'aperçu des données**

### Step-by-step:

1. **Démarrez** un **aperçu de présentation** :
   * Allez dans l'onglet **Affichage** dans la barre de ruban.
   * Cliquez sur **Présentation** dans la **Aperçu** section.
   * La **Presentation Preview window** s'ouvrira et la présentation démarrera automatiquement.
2. Dans la **Presentation Preview window**, cliquez sur **Data Preview** dans la barre de ruban pour ouvrir la **Data Preview Window**.

***

## **Aperçu de la Data Preview Window**

### Ce que vous voyez :

* **Onglets pour chaque Data Provider**:\
  Chaque **Data Provider** utilisé dans la présentation aura son **propre onglet**. Cliquez sur les onglets pour basculer entre les Data Providers et visualiser leurs données.

* **Données en direct**:\
  Vous voyez les **données réelles** renvoyées par chaque Data Provider, vous montrant ce qui sera affiché dans les éléments de votre présentation connectés à ces données.

***

## **Fonctions de la Data Preview Window**

| **Fonction** | **Description**                                                                                                                                  |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Refresh**  | Recharge et met à jour les données pour **tous les Data Providers**. Cela vous permet de voir les données nouvellement récupérées en temps réel. |
| **Close**    | Ferme la fenêtre Data Preview.                                                                                                                   |

***

## **Cas d'utilisation pour Data Preview**

* **Vérifier la connexion des données** : Assurez-vous que les données provenant de sources externes (par exemple, bases de données, services web, flux RSS) sont correctement récupérées.
* **Tester le contenu dynamique** : Vérifiez comment les données dynamiques apparaîtront dans les éléments de votre présentation.
* **Débogage** : Si des données sont manquantes ou incorrectes dans la présentation, utilisez cette fenêtre pour vérifier si la source de données renvoie les résultats attendus.
* **Mises à jour en direct** : Confirmez que les mises à jour dans votre source de données sont reflétées lors de l'actualisation des données.
