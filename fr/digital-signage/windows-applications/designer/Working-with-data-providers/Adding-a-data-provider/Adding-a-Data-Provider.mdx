Avant de connecter un **presentation element** à une **real-time data source**, un **Data Provider** doit être créé. Le **Data Provider Wizard** simplifie ce processus en guidant les utilisateurs à travers les étapes de configuration essentielles.

***

## **Étapes pour ajouter un fournisseur de données**

### **1. Ouvrir les paramètres du fournisseur de données**

* Naviguez vers le **Home Menu**
* Cliquez sur **Data Providers**
* Cliquez sur **Add…** pour démarrer le **Data Provider Wizard**

### **2. Suivez les étapes de l'assistant**

Le **Data Provider Wizard** comprend les étapes suivantes :

#### **Choisir un type de fournisseur**

* Sélectionnez le **source type** pour vos données en temps réel
* Les options incluent :
  * **Databases** (Microsoft SQL Server, MySQL, Oracle, etc.)
  * **XML & RSS Feeds**
  * **Microsoft Excel**
  * **Text & CSV Files**
  * **Web-based APIs & JSON Files**

#### **Paramètres généraux du fournisseur**

* Attribuez un **unique name** au fournisseur
* Choisissez d'utiliser ou non le **PADS4 Data Server** pour la récupération en temps réel

#### **Paramètres de connexion du fournisseur**

* Entrez les paramètres de connexion spécifiques en fonction de la **data source**
* Configurez les **authentication credentials** (si nécessaire)

#### **Réorganiser les données**

* Échangez les **rows and columns** selon les besoins

#### **Ajout d'expressions régulières**

* Appliquez la **text pattern matching** et les **optional text replacements**

#### **Ajout de colonnes personnalisées**

* Définissez des **calculated fields** pour étendre l'ensemble de données

#### **Application de filtres et de tri**

* Configurez des **conditions** pour affiner la **data selection**
* Définissez des **sorting rules**

#### **Paramètres avancés du fournisseur**

* Configurez des **backup data sources**
* Spécifiez des **advanced retrieval options**

***

## **Finalisation de la configuration du fournisseur de données**

* Cliquez sur **Finish** pour terminer la configuration
* Le fournisseur de données est maintenant **available for use in the presentation**

***

## **Notes supplémentaires**

* **Data Vault Providers** ne prennent pas en charge toutes les étapes de l'assistant (**only expressions, sorting, and filtering are available**)
* **Preview** votre fournisseur de données pour **verify real-time data retrieval**

### **Conseil**

Pour accélérer la configuration, cliquez sur **Finish** après avoir terminé les étapes **essential**.
