Avant de connecter un **presentation element** à une **source de données en temps réel**, un **Data Provider** doit être créé. L'**Data Provider Wizard** simplifie ce processus en guidant les utilisateurs à travers les étapes de configuration essentielles.

***

## **Étapes pour ajouter un fournisseur de données**

### **1. Ouvrir les paramètres du fournisseur de données**

* Naviguez vers le **Home Menu**
* Cliquez sur **Data Providers**
* Cliquez sur **Add…** pour démarrer l'**Data Provider Wizard**

### **2. Suivre les étapes de l'assistant**

L'**Data Provider Wizard** comprend les étapes suivantes:

#### **Choisir un type de fournisseur**

* Sélectionnez le **source type** pour vos données en temps réel
* Les options incluent:
  * **Databases** (Microsoft SQL Server, MySQL, Oracle, etc.)
  * **XML & RSS Feeds**
  * **Microsoft Excel**
  * **Text & CSV Files**
  * **Web-based APIs & JSON Files**

#### **Paramètres généraux du fournisseur**

* Attribuez un **nom unique** au fournisseur
* Choisissez d'utiliser ou non le **PADS4 Data Server** pour la récupération en temps réel

#### **Paramètres de connexion du fournisseur**

* Saisissez les paramètres de connexion spécifiques en fonction du **source de données**
* Configurer **les identifiants d'authentification** (si nécessaire)

#### **Réorganiser les données**

* Permuter **lignes et colonnes** si nécessaire

#### **Ajout d'expressions régulières**

* Appliquer **la correspondance de motifs de texte** et **les remplacements de texte optionnels**

#### **Ajout de colonnes personnalisées**

* Définir **des champs calculés** pour étendre l'ensemble de données

#### **Application de filtres et de tri**

* Configurer **des conditions** pour affiner **la sélection des données**
* Définir **des règles de tri**

#### **Paramètres avancés du fournisseur**

* Configurer **des sources de données de secours**
* Spécifier **des options de récupération avancées**

***

## **Finalisation de la configuration du fournisseur de données**

* Cliquez sur **Terminer** pour compléter la configuration
* Le fournisseur de données est maintenant **disponible pour utilisation dans la présentation**

***

## **Notes supplémentaires**

* **Data Vault Providers** ne prennent pas en charge toutes les étapes de l'assistant (**seules les expressions, le tri et le filtrage sont disponibles**)
* **Aperçu** de votre fournisseur de données pour **vérifier la récupération des données en temps réel**

### **Conseil**

Pour accélérer la configuration, cliquez sur **Terminer** après avoir complété les étapes **essentielles**.
