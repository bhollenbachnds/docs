Avant de connecter un **élément de présentation** à une **source de données en temps réel**, un **Data Provider** doit être créé. Le **Data Provider Wizard** simplifie ce processus en guidant les utilisateurs à travers les étapes de configuration essentielles.

***

## **Étapes pour ajouter un Data Provider**

### **1. Ouvrir les paramètres du Data Provider**

* Accédez au **Home Menu**
* Cliquez sur **Data Providers**
* Cliquez sur **Add…** pour démarrer le **Data Provider Wizard**

### **2. Suivre les étapes de l'assistant**

Le **Data Provider Wizard** comprend les étapes suivantes :

#### **Choisir un type de Provider**

* Sélectionnez le **type de source** pour vos données en temps réel
* Les options incluent :
  * **Bases de données** (Microsoft SQL Server, MySQL, Oracle, etc.)
  * **Flux XML & RSS**
  * **Microsoft Excel**
  * **Fichiers texte & CSV**
  * **APIs web & fichiers JSON**

#### **Paramètres généraux du Provider**

* Attribuez un **nom unique** au provider
* Choisissez d'utiliser ou non le **PADS4 Data Server** pour la récupération en temps réel

#### **Paramètres de connexion du Provider**

* Entrez les paramètres de connexion spécifiques en fonction de la **source de données**
* Configurez les **identifiants d'authentification** (si requis)

#### **Réorganiser les données**

* Permutez les **lignes et colonnes** selon les besoins

#### **Ajout d'expressions régulières**

* Appliquez la **correspondance de motifs de texte** et **des remplacements de texte optionnels**

#### **Ajout de colonnes personnalisées**

* Définissez des **champs calculés** pour étendre le jeu de données

#### **Application de filtres & tri**

* Configurez des **conditions** pour affiner la **sélection des données**
* Définissez des **règles de tri**

#### **Paramètres avancés du Provider**

* Configurez des **sources de données de secours**
* Spécifiez des **options de récupération avancées**

***

## **Finalisation de la configuration du Data Provider**

* Cliquez sur **Finish** pour terminer la configuration
* Le Data Provider est maintenant **disponible pour utilisation dans la présentation**

***

## **Notes supplémentaires**

* **Data Vault Providers** ne prennent pas en charge toutes les étapes de l'assistant (**seules les expressions, le tri et le filtrage sont disponibles**)
* **Aperçu** de votre Data Provider pour **vérifier la récupération des données en temps réel**

### **Conseil**

Pour accélérer la configuration, cliquez sur **Finish** après avoir complété les étapes **essentielles**.
