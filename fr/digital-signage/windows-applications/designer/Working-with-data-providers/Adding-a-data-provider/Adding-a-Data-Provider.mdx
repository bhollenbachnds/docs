Avant de connecter un **élément de présentation** à une **source de données en temps réel**, un **Data Provider** doit être créé. L'assistant **Data Provider Wizard** simplifie ce processus en guidant les utilisateurs à travers les étapes essentielles de configuration.

***

## **Étapes pour ajouter un Data Provider**

### **1. Ouvrir les paramètres du Data Provider**

* Naviguez vers le **Home Menu**
* Cliquez sur **Data Providers**
* Cliquez sur **Add…** pour démarrer le **Data Provider Wizard**

### **2. Suivre les étapes de l'assistant**

Le **Data Provider Wizard** comprend les étapes suivantes :

#### **Choisir un type de fournisseur**

* Sélectionnez le **type de source** pour vos données en temps réel
* Les options incluent :
  * **Bases de données** (Microsoft SQL Server, MySQL, Oracle, etc.)
  * **Flux XML et RSS**
  * **Microsoft Excel**
  * **Fichiers texte et CSV**
  * **APIs Web et fichiers JSON**

#### **Paramètres généraux du fournisseur**

* Attribuez un **nom unique** au fournisseur
* Choisissez d'utiliser ou non le **PADS4 Data Server** pour la récupération en temps réel

#### **Paramètres de connexion du fournisseur**

* Saisissez les paramètres de connexion spécifiques en fonction de la **source de données**
* Configurez les **identifiants d'authentification** (si nécessaire)

#### **Réorganiser les données**

* Échangez les **lignes et colonnes** selon les besoins

#### **Ajout d'expressions régulières**

* Appliquez la **correspondance de motifs de texte** et les **remplacements de texte optionnels**

#### **Ajout de colonnes personnalisées**

* Définissez des **champs calculés** pour étendre l'ensemble de données

#### **Application de filtres et de tri**

* Configurez des **conditions** pour affiner la **sélection des données**
* Définissez des **règles de tri**

#### **Paramètres avancés du fournisseur**

* Configurez des **sources de données de sauvegarde**
* Spécifiez des **options de récupération avancées**

***

## **Finalisation de la configuration du Data Provider**

* Cliquez sur **Finish** pour terminer la configuration
* Le Data Provider est maintenant **disponible pour utilisation dans la présentation**

***

## **Notes supplémentaires**

* **Fournisseurs Data Vault** ne prennent pas en charge toutes les étapes de l'assistant (**seules les expressions, le tri et le filtrage sont disponibles**)
* **Aperçu** votre Fournisseur de Données pour **vérifier la récupération des données en temps réel**

### **Conseil**

Pour accélérer la configuration, cliquez sur **Terminer** après avoir complété les **essentielles** étapes.
