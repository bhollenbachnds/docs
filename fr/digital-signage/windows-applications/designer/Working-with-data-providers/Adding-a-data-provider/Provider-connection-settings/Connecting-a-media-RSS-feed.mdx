Media RSS (**Rich Site Summary**) étend le format standard **RSS web feed format** pour prendre en charge le contenu multimédia, comme les **vidéos et images**.

***

### **Étapes pour ajouter ou modifier une connexion Media RSS Feed**

#### **1. Démarrer le Data Provider Wizard**

* Ouvrez le **Data Provider Wizard** en suivant les instructions dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Media RSS Feed** comme type de fournisseur.

***

#### **2. Spécifier l'URL du Media RSS Feed**

* Saisissez l'**URL (URI)** du flux Media RSS.
* Si nécessaire, cliquez sur **Preview** pour vérifier la sortie du flux.

***

#### **3. Configurer l'authentification (si nécessaire)**

* Si le flux Media RSS **nécessite des identifiants de connexion**, activez **Authorization Required**.
* Saisissez l'**User ID, Password, and (if necessary) a Domain**.

***

#### **4. Configurer l'User-Agent (si nécessaire)**

* Certains serveurs peuvent exiger une **specific User-Agent string** pour autoriser l'accès.
* Si nécessaire, saisissez la **User-Agent identification string**.

***

#### **5. Finaliser et enregistrer la connexion**

* Cliquez sur **Next** pour continuer.
* Complétez la configuration en terminant le **Data Provider Wizard**.

***

### **Considérations importantes**

* **Utilisation d'un PADS4 Data Server**
  * Si un **PADS4 Data Server** est utilisé, assurez-vous que le **Media RSS feed is accessible** depuis le **point de vue du serveur**.
  * Si **vous n'utilisez pas de Data Server**, confirmez que **PADS4 Designer and PADS4 Viewers** ont un accès direct au flux Media RSS via internet.

En suivant ces étapes, PADS4 récupérera et affichera avec succès **du contenu multimédia à partir d'un flux Media RSS**.
