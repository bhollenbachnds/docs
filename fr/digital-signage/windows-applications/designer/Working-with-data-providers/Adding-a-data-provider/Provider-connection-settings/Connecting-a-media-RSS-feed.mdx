Media RSS (**Rich Site Summary**) étend le format standard **RSS web feed format** pour prendre en charge le contenu multimédia, comme les **videos and images**.

***

### **Étapes pour Ajouter ou Modifier une Connexion de Flux Media RSS**

#### **1. Démarrer l'Assistant Data Provider**

* Ouvrez l'**Data Provider Wizard** en suivant les instructions dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Media RSS Feed** comme type de fournisseur.

***

#### **2. Spécifier l'URL du Flux Media RSS**

* Entrez l'**URL (URI)** du flux Media RSS.
* Si nécessaire, cliquez sur **Preview** pour vérifier la sortie du flux.

***

#### **3. Configurer l'Authentification (Si Nécessaire)**

* Si le flux Media RSS **requires login credentials**, activez **Authorization Required**.
* Entrez l'**User ID, Password, and (if necessary) a Domain**.

***

#### **4. Configurer l'User-Agent (Si Nécessaire)**

* Certains serveurs peuvent nécessiter une **specific User-Agent string** pour autoriser l'accès.
* Si nécessaire, entrez la **User-Agent identification string** requise.

***

#### **5. Finaliser et Enregistrer la Connexion**

* Cliquez sur **Next** pour continuer.
* Complétez la configuration en terminant l'**Data Provider Wizard**.

***

### **Considérations Importantes**

* **Utilisation d'un PADS4 Data Server**
  * Si un **PADS4 Data Server** est utilisé, assurez-vous que le **Media RSS feed is accessible** depuis le **server's point of view**.
  * Si **not using a Data Server**, confirmez que **PADS4 Designer and PADS4 Viewers** avoir un accès direct au flux Media RSS via Internet.

En suivant ces étapes, PADS4 récupérera et affichera avec succès **du contenu multimédia à partir d'un flux Media RSS**.
