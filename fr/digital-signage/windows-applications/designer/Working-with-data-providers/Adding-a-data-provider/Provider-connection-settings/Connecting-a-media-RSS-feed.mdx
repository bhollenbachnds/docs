Media RSS (**Rich Site Summary**) étend le format standard **de flux web RSS** pour prendre en charge les contenus multimédias, tels que **les vidéos et les images**.

***

### **Étapes pour ajouter ou modifier une connexion de flux Media RSS**

#### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez le **Data Provider Wizard** en suivant les instructions dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Media RSS Feed** comme type de fournisseur.

***

#### **2. Spécifier l'URL du flux Media RSS**

* Entrez l'**URL (URI)** du flux Media RSS.
* Si nécessaire, cliquez sur **Preview** pour vérifier la sortie du flux.

***

#### **3. Configurer l'authentification (si nécessaire)**

* Si le flux Media RSS **nécessite des identifiants de connexion**, activez **Authorization Required**.
* Entrez l'**ID utilisateur, le mot de passe et (si nécessaire) un domaine**.

***

#### **4. Configurer le User-Agent (si nécessaire)**

* Certains serveurs peuvent nécessiter une **chaîne User-Agent spécifique** pour autoriser l'accès.
* Si nécessaire, entrez la **chaîne d'identification User-Agent** requise.

***

#### **5. Finaliser et enregistrer la connexion**

* Cliquez sur **Next** pour continuer.
* Terminez la configuration en finalisant le **Data Provider Wizard**.

***

### **Considérations importantes**

* **Utilisation d'un PADS4 Data Server**
  * Si un **PADS4 Data Server** est utilisé, assurez-vous que le **flux Media RSS est accessible** depuis le **point de vue du serveur**.
  * Si **vous n'utilisez pas de Data Server**, confirmez que **PADS4 Designer et PADS4 Viewers** ont un accès direct au flux Media RSS via Internet.

En suivant ces étapes, PADS4 récupérera et affichera avec succès **le contenu multimédia d'un flux Media RSS**.
