Le **Cognimatics feed provider** permet l'intégration avec les **Cognimatics devices** pour les **données de mesure d'audience**, telles que le **comptage de personnes**, la **surveillance des files d'attente** et l'**analyse démographique**.

***

## **Étapes pour ajouter ou modifier un Cognimatics Data Provider**

### **1. Démarrer l'Assistant Data Provider**

* Suivez les étapes dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Choisissez **Cognimatics** comme type de fournisseur.

### **2. Configurer les paramètres de connexion Cognimatics**

* Entrez l'**HTTP Address (IP Address)** de l'**application Cognimatics**.
* Alternativement, spécifiez une **URL manuelle** avec tous les paramètres.

### **3. Sélectionner la source active**

* Choisissez la **source installée et active** dans la liste.
* Seulement **une source** peut être active à la fois.
* La sélection d'une **source inactive** entraînera une **erreur 404** lors du test.

### **4. Sélectionner le type de données : Temps réel ou Période**

#### **Données en temps réel**

* Affiche le **nombre actuel de personnes** entrant ou sortant d'un lieu.

#### **Données basées sur une période**

* Récupère les **résultats statistiques** pour une période spécifique :

| Option de période | Description                                   |
| ----------------- | --------------------------------------------- |
| **None**          | Aucune plage de temps n'est appliquée.        |
| **Today**         | Données pour **aujourd'hui** uniquement.      |
| **Yesterday**     | Données pour **hier** uniquement.             |
| **Manual**        | Sélection personnalisée de la plage de dates. |

## **Considérations importantes**

* **Assurez-vous que la source correcte est sélectionnée :**
  * Si une **inactive source** est sélectionnée, le bouton **Preview** renverra une **404 error**.

* **Vérifiez l'accès réseau :**
  * Si vous utilisez **PADS4 Data Server**, assurez-vous que le **Cognimatics feed** est **accessible** depuis le **server's network**.
  * Si **not** vous n'utilisez pas de **Data Server**, chaque **PADS4 Viewer** doit avoir **direct access** au flux Cognimatics.

* **Consultez la documentation Cognimatics :**
  * Pour les **detailed field descriptions** et les **device configurations**, consultez la **Cognimatics official documentation**.

***

## **Dépannage**

* **Aucune donnée n'apparaît ?**
  * Vérifiez que l'**HTTP Address/IP** est correcte.
  * Assurez-vous que l'**Cognimatics device** est allumé et actif.
  * Vérifiez si la **correct source** est sélectionnée.

* **Valeurs de données inattendues ?**
  * Confirmez que la **selected time range** correspond aux données que vous attendez.
  * Si vous utilisez le **Real-Time Mode**, assurez-vous que le flux de données est mis à jour.

***
