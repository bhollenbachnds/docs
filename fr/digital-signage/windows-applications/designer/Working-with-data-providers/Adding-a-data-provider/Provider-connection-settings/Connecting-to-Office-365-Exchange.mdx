Le **Office 365 Microsoft Exchange** fournisseur dans **PADS4** permet les connexions à **Exchange Online** (basé sur le cloud). Cela permet la récupération de données telles que **les calendriers, les messages de la boîte de réception et la disponibilité des utilisateurs**.

***

### **Étapes pour ajouter ou modifier une connexion Office 365 Microsoft Exchange**

#### **1. Démarrer l'Assistant Fournisseur de données**

* Ouvrez le **Data Provider Wizard** dans **PADS4**.
* Suivez les étapes pour **Ajouter un fournisseur de données** ou **Modifier un fournisseur de données**.
* Sélectionnez **Office 365 Microsoft Exchange** comme **type de fournisseur**.

***

#### **2. Configurer les paramètres de connexion**

* **Spécifier les paramètres de proxy (si nécessaire)**
  * Si votre organisation utilise un **serveur proxy**, saisissez les paramètres requis.
  * Contactez votre **administrateur système** pour plus de détails.

* **S'authentifier auprès de Microsoft Exchange**
  * Cliquez sur le bouton **Authenticate**.
  * Une **page de connexion Microsoft** s'ouvrira.
  * Connectez-vous avec vos **identifiants Exchange Online** et autorisez **PADS4** à accéder à votre environnement.
  * Une fois l'authentification réussie, le **statut** affichera **réussi**.

* **Réauthentification**
  * Chaque fois que vous modifiez ce **fournisseur de données**, vous devrez vous réauthentifier.

Cliquez sur **Next** pour continuer.

***

#### **3. Choisir la source de données à récupérer**

Sélectionnez le type de **données Exchange** à récupérer :

| **Option**                         | **Description**                                                                                                                                                           |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **My Calendar**                    | Récupère les événements du calendrier depuis votre calendrier Exchange **personnel**.                                                                                     |
| **Shared Calendar**                | Récupère **partagé** les calendriers en spécifiant des adresses e-mail. Utile pour **les salles de réunion ou les événements de groupe**.                                 |
| **Disponibilité des utilisateurs** | Récupère **le statut libre/occupé** pour les utilisateurs spécifiés. Utile pour **les applications de planification**.                                                    |
| **Ma boîte de réception**          | Récupère **les e-mails** depuis votre **boîte de réception personnelle**. Spécifiez éventuellement un dossier et si vous souhaitez inclure les **sous-dossiers**.         |
| **Boîte de réception partagée**    | Récupère les e-mails depuis une **boîte de réception partagée** en spécifiant des adresses e-mail. Fusionnez éventuellement les données de plusieurs boîtes de réception. |

Cliquez sur **Suivant** pour continuer.

***

#### **4. Configurer les options de filtrage des données**

* **Inclure le texte du message**
  * Cochez cette option pour **récupérer le corps de l'e-mail** ainsi que les autres métadonnées.

* **Spécifier une plage de dates (De / Jusqu'à)**
  * Utilisez une période **statique** ou **dynamique** pour filtrer les résultats.
  * Réduit **l'utilisation de la mémoire** et **améliore les performances**.

**Note:**\
Si vous récupérez une **grande période de données**, cela peut entraîner une **utilisation élevée de la mémoire**. Utilisez des filtres ou définissez un **nombre maximum de lignes** pour éviter des chargements de données excessifs.

Cliquez sur **Suivant** pour continuer.

***

#### **5. Finaliser la configuration**

* Complétez les étapes restantes dans l'**Assistant Fournisseur de données**.
* Cliquez sur **Terminer** pour finaliser la configuration.

***

### **Notes supplémentaires**

* Si **l'authentification échoue**, assurez-vous que votre **compte Microsoft 365** dispose des **autorisations** nécessaires pour accéder à **Exchange Online**.
* Si **de grands ensembles de données** causent des problèmes de performances, **limitez la plage de temps ou le nombre de lignes récupérées**.
* Si vous utilisez un **serveur proxy**, assurez-vous que les **paramètres du proxy** sont correctement configurés.

Cette configuration permet à **PADS4** de récupérer et d'afficher en temps réel les **données Exchange** pour les **e-mails, les calendriers et la disponibilité**.
