Le fournisseur **Office 365 Microsoft Exchange** dans **PADS4** permet de se connecter à **Exchange Online** (basé sur le cloud). Cela permet de récupérer des données telles que les **calendriers, les messages de la boîte de réception et la disponibilité des utilisateurs**.

***

### **Étapes pour ajouter ou modifier une connexion Office 365 Microsoft Exchange**

#### **1. Démarrer l'Assistant de fournisseur de données**

* Ouvrez l'**Assistant de fournisseur de données** dans **PADS4**.
* Suivez les étapes pour **Ajouter un fournisseur de données** ou **Modifier un fournisseur de données**.
* Sélectionnez **Office 365 Microsoft Exchange** comme **type de fournisseur**.

***

#### **2. Configurer les paramètres de connexion**

* **Spécifier les paramètres du proxy (si nécessaire)**
  * Si votre organisation utilise un **serveur proxy**, saisissez les paramètres requis.
  * Contactez votre **administrateur système** pour plus de détails.

* **S'authentifier auprès de Microsoft Exchange**
  * Cliquez sur le bouton **Authenticate**.
  * Une **page de connexion Microsoft** s'ouvrira.
  * Connectez-vous avec vos **identifiants Exchange Online** et autorisez **PADS4** à accéder à votre environnement.
  * Une fois l'authentification réussie, le **statut** affichera **successful**.

* **Réauthentification**
  * Chaque fois que vous modifiez ce **fournisseur de données**, vous devrez vous réauthentifier.

Cliquez **Next** pour continuer.

***

#### **3. Choisir la Source de Données à Récupérer**

Sélectionnez le type de **Exchange data** à récupérer :

| **Option**            | **Description**                                                                                                                                                      |
| --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **My Calendar**       | Récupère les événements du calendrier de votre **personnel** Exchange calendar.                                                                                      |
| **Shared Calendar**   | Récupère les **partagés** calendriers en spécifiant les adresses e-mail. Utile pour les **salles de réunion ou événements de groupe**.                               |
| **User Availability** | Récupère le **statut libre/occupé** pour les utilisateurs spécifiés. Utile pour les **applications de planification**.                                               |
| **My Inbox**          | Récupère les **e-mails** de votre **personnel** boîte de réception. Spécifiez éventuellement un dossier et si vous souhaitez inclure les **sous-dossiers**.          |
| **Shared Inbox**      | Récupère les e-mails d'une **partagée** boîte de réception en spécifiant les adresses e-mail. Fusionnez éventuellement les données de plusieurs boîtes de réception. |

Cliquez **Next** pour continuer.

***

#### **4. Configurer les Options de Filtrage des Données**

* **Include Message Text**
  * Cochez cette option pour **récupérer le corps de l'e-mail** ainsi que d'autres métadonnées.

* **Specify Date Range (From / Until)**
  * Utilisez une période **statique** ou **dynamique** pour filtrer les résultats.
  * Réduit **l'utilisation de la mémoire** et **améliore les performances**.

**Note:**\
Si vous récupérez **une grande période de données**, cela peut entraîner **une utilisation élevée de la mémoire**. Utilisez des filtres ou définissez un **nombre maximum de lignes** pour éviter des chargements de données excessifs.

Cliquez **Next** pour continuer.

***

#### **5. Finaliser la Configuration**

* Complétez les étapes restantes dans l'**Data Provider Wizard**.
* Cliquez **Finish** pour finaliser la configuration.

***

### **Notes Supplémentaires**

* Si **l'authentification échoue**, assurez-vous que votre **compte Microsoft 365** dispose des **autorisations** nécessaires pour accéder à **Exchange Online**.
* Si **des ensembles de données volumineux** causent des problèmes de performance, **limitez la plage de temps ou le nombre de lignes récupérées**.
* Si vous utilisez un **serveur proxy**, assurez-vous que les **paramètres du proxy** sont correctement configurés.

Cette configuration permet à **PADS4** de récupérer et d'afficher en temps réel les **Exchange data** pour les **e-mails, calendriers et disponibilités**.
