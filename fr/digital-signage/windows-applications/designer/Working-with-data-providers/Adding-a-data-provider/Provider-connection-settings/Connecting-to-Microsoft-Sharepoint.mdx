Le fournisseur **Microsoft SharePoint** dans **PADS4** vous permet de vous connecter et de récupérer des données à partir d'un **site SharePoint, d'une bibliothèque de documents ou d'une liste**.

***

### **Étapes pour ajouter ou modifier une connexion Microsoft SharePoint**

#### **1. Démarrer l'Assistant Fournisseur de données**

* Ouvrez **PADS4** et naviguez vers l'**Assistant Fournisseur de données**.
* Sélectionnez **Microsoft SharePoint** comme **type de fournisseur**.
* Cliquez sur **Suivant** pour continuer.

#### **2. Spécifier les paramètres de connexion SharePoint**

* Entrez l'**URL SharePoint** pour vous connecter à votre environnement SharePoint.

* Sélectionnez la **méthode d'authentification** en fonction de la configuration de votre organisation :

  | **Méthode d'authentification**               | **Description**                                            |
  | -------------------------------------------- | ---------------------------------------------------------- |
  | **Identifiants SharePoint Online**           | Nécessite un **nom d'utilisateur** et un **mot de passe**. |
  | **Authentification NTLM**                    | Nécessite **nom d'utilisateur, mot de passe et domaine**.  |
  | **Authentification de base**                 | Aucun identifiant requis.                                  |
  | **Authentification Active Directory (ADAL)** | Authentification via **Microsoft Active Directory**.       |

* Si votre système utilise un **Serveur Proxy**, spécifiez les paramètres du proxy.

* Cliquez sur **Suivant** pour continuer.

#### **3. Sélectionner la source de données**

* La **source de données** doit être un **dossier** dans SharePoint.
* Cliquez sur le bouton **Parcourir** pour rechercher et sélectionner le **dossier SharePoint** correct. *(Ce processus peut prendre du temps en fonction de la taille de l'environnement SharePoint.)*

#### **4. Appliquer des filtres optionnels**

* **Activer la date de publication** : Filtrer les fichiers en fonction de leur date de publication.
* **Sélectionner une période** : Choisir la plage de fichiers que vous souhaitez récupérer.

#### **5. Définir les options de récupération de données**

* **Colonnes couramment utilisées** *(Par défaut)* : Limite la quantité de données récupérées en ne renvoyant que les colonnes essentielles.
* **Récupérer toutes les colonnes** : Renvoie **toutes les colonnes disponibles** (peut augmenter la charge de données).
* **Ajouter des colonnes "NoHTML"** : Convertit les **colonnes basées sur des chaînes** en **texte brut** (supprime le contenu HTML).

#### **6. Terminer la configuration**

* Cliquez sur **Suivant** pour continuer.
* **Terminer** l'assistant pour enregistrer le fournisseur de données.

***

### **Remarques importantes**

* **Versions SharePoint prises en charge** : Seules **SharePoint 2013 et versions ultérieures** sont prises en charge. *(Les versions plus anciennes ne sont pas implémentées.)*
* **Considérations de performance** : Sélectionner **toutes les colonnes disponibles** peut **augmenter le temps de chargement** et la **taille des données**.
* **Paramètres du proxy** : Si votre système utilise un **serveur proxy**, assurez-vous qu'il autorise les connexions SharePoint.

Cette configuration permet à **PADS4** de récupérer et d'utiliser des **données en temps réel de SharePoint** pour l'affichage dynamique, les rapports ou l'automatisation.
