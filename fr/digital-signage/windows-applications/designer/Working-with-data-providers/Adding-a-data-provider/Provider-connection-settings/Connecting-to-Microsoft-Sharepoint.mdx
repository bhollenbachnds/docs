Le **Microsoft SharePoint** fournisseur dans **PADS4** vous permet de vous connecter et de récupérer des données depuis un **site SharePoint, une bibliothèque de documents ou une liste**.

***

### **Étapes pour ajouter ou modifier une connexion Microsoft SharePoint**

#### **1. Démarrer l'Assistant Fournisseur de données**

* Ouvrez **PADS4** et accédez à l'**Assistant Fournisseur de données**.
* Sélectionnez **Microsoft SharePoint** comme **type de fournisseur**.
* Cliquez sur **Suivant** pour continuer.

#### **2. Spécifier les paramètres de connexion SharePoint**

* Entrez l'**URL SharePoint** pour vous connecter à votre environnement SharePoint.

  <Info>
    Assurez-vous d'utiliser l'URL du site et non l'URL complète d'un emplacement de dossier spécifique.

    À l'étape 3 « Sélectionner la source de données », vous pouvez spécifier le répertoire exact que vous souhaitez utiliser.
  </Info>
* Sélectionnez la **méthode d'authentification** en fonction de la configuration de votre organisation :

  | **Méthode d'authentification**               | **Description**                                                    |
  | -------------------------------------------- | ------------------------------------------------------------------ |
  | **Identifiants SharePoint Online**           | Nécessite un **nom d'utilisateur** et un **mot de passe**.         |
  | **Authentification NTLM**                    | Nécessite **un nom d'utilisateur, un mot de passe et un domaine**. |
  | **Authentification de base**                 | Aucun identifiant requis.                                          |
  | **Authentification Active Directory (ADAL)** | Authentifiez-vous via **Microsoft Active Directory**.              |
* Si votre système utilise un **Serveur Proxy**, spécifiez les paramètres du proxy.
* Cliquez sur **Suivant** pour continuer.

#### **3. Sélectionner la source de données**

* La **source de données** doit être un **dossier** dans SharePoint.
* Cliquez sur le bouton **Parcourir** pour rechercher et sélectionner le bon **dossier SharePoint**. *(Ce processus peut prendre du temps selon la taille de l'environnement SharePoint.)*

#### **4. Appliquer des filtres optionnels**

* **Activer la date de publication**: Filtrez les fichiers en fonction de la date à laquelle ils ont été publiés.
* **Sélectionner une période**: Choisissez la plage de fichiers que vous souhaitez récupérer.

#### **5. Définir les options de récupération des données**

* **Colonnes couramment utilisées** *(Par défaut)*: Limite la quantité de données récupérées en ne renvoyant que les colonnes essentielles.
* **Récupérer toutes les colonnes**: Renvoie **toutes les colonnes disponibles** (peut augmenter la charge de données).
* **Ajouter des colonnes « NoHTML »**: Convertit **colonnes basées sur des chaînes** en **texte brut** (supprime le contenu HTML).

#### **6. Terminer la configuration**

* Cliquez sur **Suivant** pour continuer.
* **Terminer** l'assistant pour enregistrer le fournisseur de données.

***

### **Remarques importantes**

* **Versions SharePoint prises en charge** : Seuls **SharePoint 2013 et versions ultérieures** sont pris en charge. *(Les versions plus anciennes ne sont pas implémentées.)*
* **Considérations de performance** : La sélection de **toutes les colonnes disponibles** peut **augmenter le temps de chargement** et **la taille des données**.
* **Paramètres du proxy** : Si votre système utilise un **serveur proxy**, assurez-vous qu'il autorise les connexions SharePoint.

Cette configuration permet à **PADS4** de récupérer et d'utiliser **des données en temps réel depuis SharePoint** pour l'affichage dynamique, les rapports ou l'automatisation.
