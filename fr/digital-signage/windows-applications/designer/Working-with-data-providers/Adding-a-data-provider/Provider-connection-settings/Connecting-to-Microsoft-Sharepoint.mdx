Le **Microsoft SharePoint** fournisseur dans **PADS4** vous permet de vous connecter et de récupérer des données à partir d'un **SharePoint site, document library, or list**.

***

### **Étapes pour ajouter ou modifier une connexion Microsoft SharePoint**

#### **1. Démarrer l'Assistant Fournisseur de Données**

* Ouvrez **PADS4** et naviguez vers l'**Data Provider Wizard**.
* Sélectionnez **Microsoft SharePoint** comme **type de fournisseur**.
* Cliquez sur **Next** pour continuer.

#### **2. Spécifier les paramètres de connexion SharePoint**

* Saisissez l'**SharePoint URL** pour vous connecter à votre environnement SharePoint.

  <Info>
    Assurez-vous d'utiliser l'URL du site et non l'URL complète d'un emplacement de dossier spécifique.

    À l'étape 3 "Select the Data Source", vous pouvez spécifier le répertoire exact que vous souhaitez utiliser.
  </Info>
* Sélectionnez la **méthode d'authentification** en fonction de la configuration de votre organisation :

  | **Méthode d'authentification**             | **Description**                                      |
  | ------------------------------------------ | ---------------------------------------------------- |
  | **SharePoint Online Credentials**          | Nécessite un **username** et un **password**.        |
  | **NTLM Authentication**                    | Nécessite **username, password, and domain**.        |
  | **Basic Authentication**                   | Aucune information d'identification requise.         |
  | **Active Directory Authentication (ADAL)** | Authentification via **Microsoft Active Directory**. |
* Si votre système utilise un **Proxy Server**, spécifiez les paramètres du proxy.
* Cliquez sur **Next** pour continuer.

#### **3. Sélectionner la source de données**

* La **source de données** doit être un **dossier** dans SharePoint.
* Cliquez sur le bouton **Browse** pour rechercher et sélectionner le **dossier SharePoint** correct. *(Ce processus peut prendre du temps selon la taille de l'environnement SharePoint.)*

#### **4. Appliquer des filtres optionnels**

* **Enable Published Date** : Filtrer les fichiers en fonction de leur date de publication.
* **Sélectionner une période** : Choisissez la plage de fichiers que vous souhaitez récupérer.

#### **5. Définir les options de récupération de données**

* **Commonly Used Columns** *(Default)* : Limite la quantité de données récupérées en ne renvoyant que les colonnes essentielles.
* **Retrieve All Columns** : Renvoie **toutes les colonnes disponibles** (peut augmenter la charge de données).
* **Add "NoHTML" Columns** : Convertit les **colonnes basées sur des chaînes** en **texte brut** (supprime le contenu HTML).

#### **6. Terminer la configuration**

* Cliquez sur **Next** pour continuer.
* **Finish** l'assistant pour enregistrer le fournisseur de données.

***

### **Remarques importantes**

* **Versions SharePoint prises en charge** : Seules **SharePoint 2013 et versions ultérieures** sont prises en charge. *(Les versions plus anciennes ne sont pas implémentées.)*
* **Considérations de performance** : La sélection de **toutes les colonnes disponibles** peut **augmenter le temps de chargement** et la **taille des données**.
* **Paramètres du proxy** : Si votre système utilise un **proxy server**, assurez-vous qu'il autorise les connexions SharePoint.

Cette configuration permet à **PADS4** de récupérer et d'utiliser des **données en temps réel depuis SharePoint** pour l'affichage dynamique, les rapports ou l'automatisation.
