Le **Microsoft SharePoint** provider dans **PADS4** vous permet de vous connecter et de récupérer des données à partir d'un **SharePoint site, document library, or list**.

***

### **Étapes pour ajouter ou modifier une connexion Microsoft SharePoint**

#### **1. Démarrer l'Assistant Fournisseur de Données**

* Ouvrez **PADS4** et naviguez vers l'**Data Provider Wizard**.
* Sélectionnez **Microsoft SharePoint** comme **provider type**.
* Cliquez sur **Next** pour continuer.

#### **2. Spécifier les paramètres de connexion SharePoint**

* Entrez l'**SharePoint URL** pour vous connecter à votre environnement SharePoint.

* Sélectionnez la **authentication method** en fonction de la configuration de votre organisation :

  | **Authentication Method**                  | **Description**                                      |
  | ------------------------------------------ | ---------------------------------------------------- |
  | **SharePoint Online Credentials**          | Nécessite un **username** et un **password**.        |
  | **NTLM Authentication**                    | Nécessite **username, password, and domain**.        |
  | **Basic Authentication**                   | Aucune information d'identification requise.         |
  | **Active Directory Authentication (ADAL)** | Authentification via **Microsoft Active Directory**. |

* Si votre système utilise un **Proxy Server**, spécifiez les paramètres du proxy.

* Cliquez sur **Next** pour continuer.

#### **3. Sélectionner la source de données**

* La **data source** doit être un **folder** dans SharePoint.
* Cliquez sur le bouton **Browse** pour rechercher et sélectionner le **SharePoint folder** correct. *(Ce processus peut prendre du temps selon la taille de l'environnement SharePoint.)*

#### **4. Appliquer des filtres optionnels**

* **Enable Published Date** : Filtrer les fichiers en fonction de leur date de publication.
* **Select a time period** : Choisissez la plage de fichiers que vous souhaitez récupérer.

#### **5. Définir les options de récupération de données**

* **Commonly Used Columns** *(Default)* : Limite la quantité de données récupérées en ne renvoyant que les colonnes essentielles.
* **Retrieve All Columns** : Renvoie **all available columns** (peut augmenter la charge de données).
* **Add "NoHTML" Columns** : Convertit les **string-based columns** en **plain text** (supprime le contenu HTML).

#### **6. Terminer la configuration**

* Cliquez sur **Next** pour continuer.
* **Finish** l'assistant pour enregistrer le fournisseur de données.

***

### **Remarques importantes**

* **Supported SharePoint Versions** : Seules les versions **SharePoint 2013 and newer** sont prises en charge. *(Les versions plus anciennes ne sont pas implémentées.)*
* **Performance Considerations** : La sélection de **all available columns** peut **increase load time** et **data size**.
* **Proxy Settings** : Si votre système utilise un **proxy server**, assurez-vous qu'il autorise les connexions SharePoint.

Cette configuration permet à **PADS4** de récupérer et d'utiliser des **real-time data from SharePoint** pour l'affichage dynamique, les rapports ou l'automatisation.
