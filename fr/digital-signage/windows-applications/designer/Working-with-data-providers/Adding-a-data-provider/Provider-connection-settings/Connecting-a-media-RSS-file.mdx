Media RSS (**Rich Site Summary**) étend le **format de flux web RSS** pour prendre en charge **le contenu multimédia** tel que **des vidéos et des images** stockés dans un fichier local.

***

### **Étapes pour ajouter ou modifier une connexion de fichier Media RSS**

#### **1. Démarrer l'assistant Fournisseur de données**

* Ouvrez l'**assistant Fournisseur de données** en suivant les instructions dans **Ajouter un fournisseur de données** ou **Modifier un fournisseur de données**.
* Sélectionnez **Fichier Media RSS** comme type de fournisseur.

***

#### **2. Spécifier l'emplacement du fichier Media RSS**

* Choisissez entre :
  * **Bibliothèque de contenu** (Recommandé) → Utilise un fichier stocké dans la **Bibliothèque de contenu PADS4**.
  * **Nom de fichier** → Utilise un fichier **en dehors de la Bibliothèque de contenu** (le chemin complet du fichier doit être spécifié).
* Cliquez sur **Parcourir** pour localiser et sélectionner le fichier Media RSS.

***

#### **3. Finaliser la connexion**

* Cliquez sur **Suivant** pour continuer.
* Terminez la configuration en finalisant l'**assistant Fournisseur de données**.

***

### **Considérations importantes**

* **Utilisation d'un serveur de données PADS4**
  * Si un **serveur de données PADS4** est utilisé, assurez-vous que le **fichier Media RSS est accessible** depuis le **point de vue du serveur**.
  * Si **vous n'utilisez pas de serveur de données**, confirmez que **PADS4 Designer et les PADS4 Viewers** peuvent localiser le fichier Media RSS.
* **Utilisation d'un fichier Media RSS téléchargé**
  * Pour enregistrer un flux Media RSS depuis Internet :
    1. Ouvrez **Microsoft Internet Explorer**.
    2. Accédez au **flux Media RSS**.
    3. **Cliquez avec le bouton droit** sur la fenêtre du navigateur et sélectionnez **"Afficher la source"**.
    4. Enregistrez le contenu en tant que **fichier .xml** pour une utilisation locale.

En suivant ces étapes, PADS4 récupérera et affichera avec succès **le contenu multimédia d'un fichier Media RSS**.
