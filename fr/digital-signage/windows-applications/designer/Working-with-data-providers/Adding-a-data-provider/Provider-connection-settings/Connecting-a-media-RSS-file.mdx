Media RSS (**Rich Site Summary**) étend le **format de flux web RSS** pour prendre en charge **le contenu multimédia** tel que **les vidéos et les images** stockés dans un fichier local.

***

### **Étapes pour ajouter ou modifier une connexion de fichier Media RSS**

#### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez le **Data Provider Wizard** en suivant les instructions dans **Ajout d'un fournisseur de données** ou **Modification d'un fournisseur de données**.
* Sélectionnez **Media RSS File** comme type de fournisseur.

***

#### **2. Spécifier l'emplacement du fichier Media RSS**

* Choisissez entre :
  * **Content Library** (Recommandé) → Utilise un fichier stocké dans la **PADS4 Content Library**.
  * **File Name** → Utilise un fichier **en dehors de la Content Library** (le chemin complet du fichier doit être spécifié).
* Cliquez sur **Browse** pour localiser et sélectionner le fichier Media RSS.

***

#### **3. Finaliser la connexion**

* Cliquez sur **Next** pour continuer.
* Terminez la configuration en finalisant le **Data Provider Wizard**.

***

### **Considérations importantes**

* **Utilisation d'un PADS4 Data Server**
  * Si un **PADS4 Data Server** est utilisé, assurez-vous que le **fichier Media RSS est accessible** depuis le **point de vue du serveur**.
  * Si **vous n'utilisez pas de Data Server**, confirmez que **PADS4 Designer et les PADS4 Viewers** peuvent localiser le fichier Media RSS.
* **Utilisation d'un fichier Media RSS téléchargé**
  * Pour enregistrer un flux Media RSS depuis Internet :
    1. Ouvrez **Microsoft Internet Explorer**.
    2. Accédez au **flux Media RSS**.
    3. **Faites un clic droit** sur la fenêtre du navigateur et sélectionnez **"Afficher la source"**.
    4. Enregistrez le contenu en tant que **fichier .xml** pour une utilisation locale.

En suivant ces étapes, PADS4 récupérera et affichera avec succès **le contenu multimédia d'un fichier Media RSS**.
