Le **Microsoft Dynamics CRM provider type** permet à **PADS4** de se connecter à Microsoft Dynamics CRM, spécifiquement à la version **Dynamics 365 (online)**.

***

### **Étapes pour se connecter à Microsoft Dynamics CRM**

#### **1. Démarrer l'Assistant Fournisseur de données**

* Ouvrez **Data Provider Wizard** en suivant les instructions dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Microsoft Dynamics CRM** comme **provider type**.

***

#### **2. Spécifier les paramètres de connexion**

* Entrez **l'URL complète** de votre **organisation Microsoft Dynamics CRM**.
* Cliquez sur **Authenticate** (si ce n'est pas déjà fait).
* Une fois authentifié, le champ **Authenticated As** affichera vos identifiants.

***

#### **3. Spécifier la région en ligne**

* Choisissez la **région** où se trouve votre **serveur Microsoft Dynamics CRM**.
* Si vous n'êtes pas sûr, utilisez la fonction **Search** pour trouver la région correcte.

**Pour les serveurs proxy :**

* Si votre connexion passe par un **serveur proxy**, entrez les **paramètres d'authentification du serveur proxy**.

***

#### **4. Sélectionner l'organisation CRM**

* Choisissez **l'organisation Microsoft Dynamics CRM** dans la liste des options disponibles.

***

#### **5. Sélectionner la vue de données**

* Choisissez la **vue** qui contient les données dont vous avez besoin.
* Utilisez le bouton **Preview** pour voir un échantillon des données.
* Facultativement, **filtrez la liste des vues** en saisissant une **sous-chaîne** pour réduire les résultats.

***

#### **6. Terminer la configuration**

* Cliquez sur **Next** pour continuer.
* Terminez la configuration en complétant **Data Provider Wizard**.

***

### **Notes supplémentaires**

* **Seul Microsoft Dynamics 365 (online) est pris en charge.**
* Assurez-vous que votre **URL d'organisation CRM** est correct avant l'authentification.
* Si vous utilisez un **serveur proxy**, assurez-vous que les paramètres corrects sont appliqués.

Ce processus permet à **PADS4** de récupérer des **données en temps réel** depuis **Microsoft Dynamics CRM** et de les intégrer dans les présentations.
