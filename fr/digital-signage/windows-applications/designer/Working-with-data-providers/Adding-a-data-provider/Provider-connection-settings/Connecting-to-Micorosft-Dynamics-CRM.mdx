Le **Microsoft Dynamics CRM provider type** permet à **PADS4** de se connecter à Microsoft Dynamics CRM, spécifiquement la version **Dynamics 365 (online) version**.

***

### **Étapes pour se connecter à Microsoft Dynamics CRM**

#### **1. Démarrer l'Assistant de fournisseur de données**

* Ouvrez le **Data Provider Wizard** en suivant les instructions dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Microsoft Dynamics CRM** comme **provider type**.

***

#### **2. Spécifier les paramètres de connexion**

* Saisissez l'**full URL** de votre **Microsoft Dynamics CRM organization**.
* Cliquez sur **Authenticate** (si ce n'est pas déjà fait).
* Une fois authentifié, le champ **Authenticated As** affichera vos identifiants.

***

#### **3. Spécifier la région en ligne**

* Choisissez la **region** où se trouve votre **Microsoft Dynamics CRM server**.
* Si vous n'êtes pas sûr, utilisez la fonction **Search** pour trouver la région correcte.

**Pour les serveurs proxy :**

* Si votre connexion passe par un **proxy server**, saisissez les **proxy server authentication settings**.

***

#### **4. Sélectionner l'organisation CRM**

* Choisissez l'**Microsoft Dynamics CRM organization** dans la liste des options disponibles.

***

#### **5. Sélectionner la vue de données**

* Choisissez la **view** qui contient les données dont vous avez besoin.
* Utilisez le bouton **Preview** pour voir un échantillon des données.
* Optionnellement, **filter the view list** en saisissant une **substring** pour réduire les résultats.

***

#### **6. Terminer la configuration**

* Cliquez sur **Next** pour continuer.
* Terminez la configuration en complétant le **Data Provider Wizard**.

***

### **Notes supplémentaires**

* **Seul Microsoft Dynamics 365 (online) est pris en charge.**
* Assurez-vous que votre **CRM organization URL** est correcte avant l'authentification.
* Si vous utilisez un **proxy server**, assurez-vous que les paramètres corrects sont appliqués.

Ce processus permet à **PADS4** de récupérer des **real-time data** depuis **Microsoft Dynamics CRM** et de les intégrer dans des présentations.
