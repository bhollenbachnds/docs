---
- title: Connexion à Microsoft LinkedIn
---

Le **type de fournisseur Microsoft LinkedIn** permet à **PADS4** de se connecter à **LinkedIn** et de récupérer les publications d'une organisation spécifiée.

***

### **Étapes pour se connecter à Microsoft LinkedIn**

#### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez le **Data Provider Wizard** en suivant les instructions dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Microsoft LinkedIn** comme **type de fournisseur**.

***

#### **2. S'authentifier avec LinkedIn**

* Cliquez sur **Authenticate with LinkedIn credentials** pour ouvrir une fenêtre de connexion.
* Entrez vos **identifiants LinkedIn** et suivez le processus d'authentification.
* Une fois l'authentification **réussie**, vous pouvez continuer.

***

#### **3. Configurer les paramètres du proxy (Optionnel)**

* Si votre système utilise un **serveur proxy**, entrez les **informations d'authentification du serveur proxy**.

***

#### **4. Sélectionner l'organisation LinkedIn**

* Choisissez l'**organisation** à partir de laquelle vous souhaitez récupérer les **publications**.
* Si vous voyez le message :\
  *"Vous ne disposez pas des droits d'accès suffisants pour accéder aux publications d'une organisation,"*\
  assurez-vous que votre compte LinkedIn dispose d'un accès **ADMINISTRATOR** à la page de l'organisation.

***

#### **5. Récupérer les données d'engagement des publications (Optionnel)**

* Cochez **Get number of likes, comments, clicks, and shares per post** pour récupérer les **statistiques d'engagement** pour chaque publication LinkedIn.

***

#### **6. Finaliser la configuration**

* Cliquez sur **Next** pour continuer.
* Terminez la configuration en complétant le **Data Provider Wizard**.

***

### **Notes supplémentaires**

* Vous devez disposer d'un accès **Administrateur** à la page de l'organisation LinkedIn pour récupérer les publications.
* Si vous utilisez un **serveur proxy**, assurez-vous que les paramètres corrects sont appliqués.

Cette configuration permet à **PADS4** de récupérer **les publications LinkedIn en temps réel et les données d'engagement** pour les afficher dans les présentations.
