Le **Microsoft SQL Server Compact Edition (CE)** fournisseur dans **PADS4** vous permet de vous connecter à un **fichier de base de données SQL Server CE** et de récupérer des données pour les utiliser dans vos présentations.

***

### **Étapes pour ajouter ou modifier une connexion Microsoft SQL Server CE**

#### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrir **PADS4** et accéder à l'**Assistant de fournisseur de données**.
* Sélectionner **Microsoft SQL Server CE** comme **type de fournisseur**.
* Cliquer sur **Next** pour continuer.

#### **2. Spécifier l'emplacement de la base de données et les paramètres d'accès**

* Choisir l'une des options suivantes :
  * **Content Library** : Utiliser une **base de données SQL Server CE** stockée dans la **PADS4 Content Library** *(recommandé pour la synchronisation automatique)*.
  * **File Name** : Utiliser une **base de données SQL Server CE** stockée **en dehors** de la bibliothèque de contenu *(distribution manuelle des fichiers requise)*.

* Saisir le **chemin du fichier** ou utiliser le bouton **Browse** pour localiser le fichier.

* Si la base de données est **protégée par un mot de passe**, saisir le **mot de passe**.

* Cliquer sur **Next** pour continuer.

#### **3. Sélectionner la table ou la requête SQL**

* Le système détectera automatiquement **les tables disponibles** dans la base de données.

* Sélectionner une table dans la **liste déroulante**.

* Alternativement, sélectionner **SQL Statement** et saisir une **requête SQL personnalisée** pour récupérer des données spécifiques.

* Cliquer sur **Next** pour continuer.

#### **4. Terminer la configuration**

* **Vérifier** les paramètres sélectionnés.
* Cliquer sur **Finish** pour enregistrer le fournisseur de données.

***

### **Remarques importantes**

* **Considérations relatives au PADS4 Data Server** :
  * Si vous utilisez un **PADS4 Data Server**, assurez-vous que le **fichier de base de données est accessible depuis le serveur**.
  * Si vous n'utilisez **pas** de **PADS4 Data Server**, assurez-vous que **PADS4 Designer et les Viewers** peuvent accéder au **fichier de base de données**.
* **Recommandations de stockage des fichiers** :
  * L'utilisation de la **PADS4 Content Library** garantit **la synchronisation automatique des fichiers**.
  * Le stockage de la base de données **en dehors** de la bibliothèque de contenu nécessite **une distribution manuelle des fichiers**.
* **Requêtes SQL** :
  * Les **instructions SQL** saisies manuellement doivent être optimisées pour **les performances et l'efficacité**.

Cette configuration permet à **PADS4** d'extraire des données en temps réel depuis une **base de données SQL Server CE**, en les intégrant de manière transparente dans l'affichage dynamique, les rapports ou les flux de travail d'automatisation.
