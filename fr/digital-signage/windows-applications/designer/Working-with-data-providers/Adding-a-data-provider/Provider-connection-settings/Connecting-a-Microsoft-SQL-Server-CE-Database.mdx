Le fournisseur **Microsoft SQL Server Compact Edition (CE)** dans **PADS4** vous permet de vous connecter à un fichier de base de données **SQL Server CE database file** et de récupérer des données pour les utiliser dans vos présentations.

***

### **Étapes pour ajouter ou modifier une connexion Microsoft SQL Server CE**

#### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez **PADS4** et naviguez vers le **Data Provider Wizard**.
* Sélectionnez **Microsoft SQL Server CE** comme **provider type**.
* Cliquez sur **Next** pour continuer.

#### **2. Spécifier l'emplacement de la base de données et les paramètres d'accès**

* Choisissez l'une des options suivantes :
  * **Content Library** : Utilisez une base de données **SQL Server CE database** stockée dans la **PADS4 Content Library** *(recommandé pour la synchronisation automatique)*.
  * **Nom du fichier**: Utilisez une **SQL Server CE database** stockée **en dehors** de la bibliothèque de contenu *(distribution manuelle des fichiers requise)*.

* Saisissez le **chemin du fichier** ou utilisez le bouton **Parcourir** pour localiser le fichier.

* Si la base de données est **protégée par mot de passe**, saisissez le **mot de passe**.

* Cliquez sur **Suivant** pour continuer.

#### **3. Sélectionnez la Table ou la Requête SQL**

* Le système détectera automatiquement les **tables disponibles** dans la base de données.

* Sélectionnez une table dans la **liste déroulante**.

* Alternativement, sélectionnez **SQL Statement** et saisissez une **requête SQL personnalisée** pour récupérer des données spécifiques.

* Cliquez sur **Suivant** pour continuer.

#### **4. Terminez la configuration**

* **Vérifiez** les paramètres sélectionnés.
* Cliquez sur **Terminer** pour enregistrer le fournisseur de données.

***

### **Remarques importantes**

* **Considérations relatives au PADS4 Data Server**:
  * Si vous utilisez un **PADS4 Data Server**, assurez-vous que le **fichier de base de données est accessible depuis le serveur**.
  * Si **vous n'utilisez pas** de **PADS4 Data Server**, assurez-vous que **PADS4 Designer and Viewers** peuvent accéder au **fichier de base de données**.
* **Recommandations pour le stockage des fichiers**:
  * L'utilisation de la **PADS4 Content Library** assure une **synchronisation automatique des fichiers**.
  * Le stockage de la base de données **en dehors** de la bibliothèque de contenu nécessite une **distribution manuelle des fichiers**.
* **Requêtes SQL**:
  * Les **instructions SQL** saisies manuellement doivent être optimisées pour la **performance et l'efficacité**.

Cette configuration permet à **PADS4** d'extraire des données en temps réel d'une **SQL Server CE database**, en les intégrant de manière transparente dans l'affichage dynamique, les rapports ou les flux de travail d'automatisation.
