Le **Microsoft SQL Server Compact Edition (CE)** fournisseur dans **PADS4** vous permet de vous connecter à un **fichier de base de données SQL Server CE** et de récupérer des données pour les utiliser dans vos présentations.

***

### **Étapes pour ajouter ou modifier une connexion Microsoft SQL Server CE**

#### **1. Démarrer le Data Provider Wizard**

* Ouvrez **PADS4** et accédez au **Data Provider Wizard**.
* Sélectionnez **Microsoft SQL Server CE** comme **type de fournisseur**.
* Cliquez sur **Next** pour continuer.

#### **2. Spécifier l'emplacement de la base de données et les paramètres d'accès**

* Choisissez l'une des options suivantes :
  * **Content Library** : Utilisez une **base de données SQL Server CE** stockée dans la **PADS4 Content Library** *(recommandé pour la synchronisation automatique)*.
  * **File Name** : Utilisez une **base de données SQL Server CE** stockée **en dehors** de la bibliothèque de contenu *(distribution manuelle des fichiers requise)*.

* Entrez le **chemin du fichier** ou utilisez le bouton **Browse** pour localiser le fichier.

* Si la base de données est **protégée par un mot de passe**, entrez le **mot de passe**.

* Cliquez sur **Next** pour continuer.

#### **3. Sélectionner la table ou la requête SQL**

* Le système détectera automatiquement les **tables disponibles** dans la base de données.

* Sélectionnez une table dans la **liste déroulante**.

* Vous pouvez également sélectionner **SQL Statement** et entrer une **requête SQL personnalisée** pour récupérer des données spécifiques.

* Cliquez sur **Next** pour continuer.

#### **4. Terminer la configuration**

* **Vérifiez** les paramètres sélectionnés.
* Cliquez sur **Finish** pour enregistrer le fournisseur de données.

***

### **Notes importantes**

* **Considérations relatives au PADS4 Data Server** :
  * Si vous utilisez un **PADS4 Data Server**, assurez-vous que le **fichier de base de données est accessible depuis le serveur**.
  * Si vous n'utilisez **pas** de **PADS4 Data Server**, assurez-vous que **PADS4 Designer et les Viewers** peuvent accéder au **fichier de base de données**.
* **Recommandations de stockage des fichiers** :
  * L'utilisation de la **PADS4 Content Library** garantit la **synchronisation automatique des fichiers**.
  * Le stockage de la base de données **en dehors** de la bibliothèque de contenu nécessite une **distribution manuelle des fichiers**.
* **Requêtes SQL**:
  * Saisie manuelle **Instructions SQL** doivent être optimisées pour **la performance et l'efficacité**.

Cette configuration permet à **PADS4** d'extraire des données en temps réel depuis une **base de données SQL Server CE**, en les intégrant de manière transparente dans la signalétique numérique, les rapports ou les flux de travail d'automatisation.
