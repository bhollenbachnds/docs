---
- title: Connexion d'une feuille Microsoft Excel
---

Le **fournisseur Microsoft Excel** dans **PADS4** vous permet de vous connecter à des **fichiers Excel** (Office 2010 et versions antérieures) pour récupérer des données pour des présentations en temps réel.

***

### **Étapes pour ajouter ou modifier une connexion Microsoft Excel**

#### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez le **Data Provider Wizard** dans **PADS4**.
* Suivez les étapes pour **Ajouter un fournisseur de données** ou **Modifier un fournisseur de données**.
* Sélectionnez **Microsoft Excel** comme **type de fournisseur**.

***

#### **2. Configurer les paramètres de connexion**

* **Spécifier l'emplacement du fichier Excel**
  * **Option 1 : Utiliser la bibliothèque de contenu PADS4 (Recommandé)**
    * La bibliothèque de contenu **synchronise automatiquement les fichiers sur tous les ordinateurs** du système PADS4.
  * **Option 2 : Utiliser un fichier en dehors de la bibliothèque de contenu**
    * Si vous sélectionnez cette option, vous devez manuellement **distribuer le fichier Excel** pour vous assurer qu'il est disponible pour tous les PADS4 Viewers.
* **Sélectionner le fichier**
  * Cliquez sur **Parcourir** pour localiser le **fichier Excel**.
* **Activer « La première ligne contient les informations de colonne »**
  * Si votre feuille Excel **inclut des en-têtes de colonnes**, activez cette option pour supprimer la première ligne de votre jeu de données.

Cliquez sur **Suivant** pour continuer.

***

#### **3. Sélectionner la feuille Excel à connecter**

* PADS4 **détecte automatiquement** toutes les **feuilles** disponibles dans le fichier Excel.
* Cliquez sur la **liste déroulante** et **sélectionnez** la feuille souhaitée.

Cliquez sur **Suivant** pour continuer.

***

#### **4. Terminer la configuration**

* Suivez les étapes restantes dans le **Data Provider Wizard**.
* Cliquez sur **Terminer** pour finaliser la configuration.

***

### **Notes supplémentaires**

* **Vous utilisez un PADS4 Data Server ?**
  * Assurez-vous que les **pilotes Office System** sont installés sur le **PADS4 Server**.
  * L'**emplacement du fichier Excel doit être accessible** depuis le **point de vue du Data Server**.
* **Vous n'utilisez pas de PADS4 Data Server ?**
  * Assurez-vous que **PADS4 Designer** (pour la prévisualisation) et tous les **PADS4 Viewers** have:
    * Le **pilotes Office System** installés.
    * **Accès** à l'emplacement du **fichier Excel** spécifié.
* **Éviter les dossiers réseau partagés**
  * Référencer un **fichier Excel dans un dossier réseau partagé** peut entraîner :
    * **Un trafic réseau élevé** lors de l'actualisation des données.
    * **Des problèmes de verrouillage de fichier** si plusieurs utilisateurs accèdent au fichier simultanément.
* **Copie automatique du fichier**
  * PADS4 **crée une copie interne** du fichier Excel avant de lire les données.
  * Cela évite les **conflits de verrouillage de fichier** et garantit **une récupération fluide des données**.
