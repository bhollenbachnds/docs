---
- title: Connexion d'une feuille Microsoft Excel
---

Le **fournisseur Microsoft Excel** dans **PADS4** vous permet de vous connecter à des **fichiers Excel** (Office 2010 et versions antérieures) pour récupérer des données pour des présentations en temps réel.

***

### **Étapes pour ajouter ou modifier une connexion Microsoft Excel**

#### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrir l'**assistant de fournisseur de données** dans **PADS4**.
* Suivre les étapes pour **ajouter un fournisseur de données** ou **modifier un fournisseur de données**.
* Sélectionner **Microsoft Excel** comme **type de fournisseur**.

***

#### **2. Configurer les paramètres de connexion**

* **Spécifier l'emplacement du fichier Excel**
  * **Option 1 : Utiliser la bibliothèque de contenu PADS4 (recommandé)**
    * La bibliothèque de contenu **synchronise automatiquement les fichiers sur tous les ordinateurs** du système PADS4.
  * **Option 2 : Utiliser un fichier en dehors de la bibliothèque de contenu**
    * Si vous sélectionnez cette option, vous devez manuellement **distribuer le fichier Excel** pour vous assurer qu'il est disponible pour toutes les visionneuses PADS4.
* **Sélectionner le fichier**
  * Cliquer sur **Parcourir** pour localiser le **fichier Excel**.
* **Activer « La première ligne contient les informations de colonne »**
  * Si votre feuille Excel **inclut des en-têtes de colonne**, activez cette option pour supprimer la première ligne de votre jeu de données.

Cliquer sur **Suivant** pour continuer.

***

#### **3. Sélectionner la feuille Excel à laquelle se connecter**

* PADS4 **détecte automatiquement** toutes les **feuilles** disponibles dans le fichier Excel.
* Cliquer sur la **liste déroulante** et **sélectionner** la feuille souhaitée.

Cliquer sur **Suivant** pour continuer.

***

#### **4. Finaliser la configuration**

* Suivre les étapes restantes dans l'**assistant de fournisseur de données**.
* Cliquer sur **Terminer** pour finaliser la configuration.

***

### **Notes supplémentaires**

* **Vous utilisez un serveur de données PADS4 ?**
  * S'assurer que les **pilotes Office System** sont installés sur le **serveur PADS4**.
  * L'**emplacement du fichier Excel doit être accessible** depuis le **point de vue du serveur de données**.
* **Vous n'utilisez pas de serveur de données PADS4 ?**
  * S'assurer que **PADS4 Designer** (pour la prévisualisation) et tous les **PADS4 Viewers** have:
    * Le **Office System drivers** installé.
    * **Accès** à l'emplacement **fichier Excel** spécifié.
* **Éviter les dossiers réseau partagés**
  * Référencer un **fichier Excel dans un dossier réseau partagé** peut entraîner :
    * **Trafic réseau élevé** lors de l'actualisation des données.
    * **Problèmes de verrouillage de fichier** si plusieurs utilisateurs accèdent au fichier simultanément.
* **Copie automatique de fichier**
  * PADS4 **crée une copie interne** du fichier Excel avant de lire les données.
  * Cela évite les **conflits de verrouillage de fichier** et garantit **une récupération fluide des données**.
