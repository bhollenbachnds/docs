Le fournisseur **Bluefox Digital Signage Content Feed** permet **l'intégration de contenu en temps réel** avec **Bluefox**, une entreprise spécialisée dans **digital signage feeds**.

***

## **Étapes pour connecter un flux de contenu Bluefox**

### **1. Démarrer l'Assistant Fournisseur de Données**

* Suivez les étapes dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Bluefox** comme type de fournisseur.

### **2. Configurer les paramètres de connexion Bluefox**

* Saisissez l'**Bluefox Feed URL** à utiliser.
* Cliquez sur **Preview** pour visualiser la sortie du flux sous forme de **data table**.

### **3. Récupérer le flux depuis votre compte Bluefox**

* Cliquez sur **Open Bluefox Content Selector** pour ouvrir le **Bluefox website**.
* Sur la **feeds page**, sélectionnez le **feed type** et la **size**.
* Copiez l'**feed URL** et collez-le dans le champ **Bluefox feed field** dans PADS4.

### **4. (Optionnel) Configurer un serveur proxy**

* Si un **proxy** est requis, saisissez les **proxy details**.

### **5. Terminer l'Assistant**

* Cliquez sur **Next** pour continuer.
* Terminez l'assistant comme décrit dans **Adding a Data Provider**.

***

## **Considérations importantes**

* **Si vous utilisez PADS4 Data Server :**
  * Assurez-vous que le **Bluefox feed** est **accessible** depuis le réseau du **PADS4 Data Server's network**.
  * Le **Data Server** va **retrieve the feed** et le distribuer à tous les **PADS4 Viewers** connectés.

* **Si vous n'utilisez pas PADS4 Data Server :**
  * Assurez-vous que **each PADS4 Viewer** et **PADS4 Designer** a un accès direct au **Bluefox feed**.

***

## **Dépannage**

* **Le flux ne se charge pas ?**
  * Vérifiez l'**feed URL** pour son exactitude.
  * Assurez-vous que votre **internet connection** est active.
  * Si vous utilisez un **proxy**, vérifiez les **proxy settings**.

* **L'aperçu n'affiche aucune donnée ?**
  * Assurez-vous que le **feed is active** dans votre **Bluefox account**.
  * Essayez de rafraîchir l'**feed URL** sur le site web Bluefox.

***

Une fois configuré, le **Bluefox content feed** se mettra à jour automatiquement, garantissant un **dynamic and engaging content** pour votre **digital signage display**.
