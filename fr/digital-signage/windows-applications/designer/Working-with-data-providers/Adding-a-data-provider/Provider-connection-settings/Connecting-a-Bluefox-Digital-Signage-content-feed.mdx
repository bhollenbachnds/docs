Le fournisseur **Bluefox Digital Signage Content Feed** permet **l'intégration de contenu en temps réel** avec **Bluefox**, une entreprise spécialisée dans les **flux d'affichage dynamique**.

***

## **Étapes pour connecter un flux de contenu Bluefox**

### **1. Démarrer l'Assistant de fournisseur de données**

* Suivez les étapes dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Bluefox** comme type de fournisseur.

### **2. Configurer les paramètres de connexion Bluefox**

* Saisissez l'**URL du flux Bluefox** à utiliser.
* Cliquez sur **Preview** pour visualiser la sortie du flux sous forme de **tableau de données**.

### **3. Récupérer le flux depuis votre compte Bluefox**

* Cliquez sur **Open Bluefox Content Selector** pour ouvrir le **site web Bluefox**.
* Sur la **page des flux**, sélectionnez le **type de flux** et la **taille**.
* Copiez l'**URL du flux** et collez-la dans le **champ de flux Bluefox** dans PADS4.

### **4. (Facultatif) Configurer un serveur proxy**

* Si un **proxy** est requis, saisissez les **détails du proxy**.

### **5. Terminer l'Assistant**

* Cliquez sur **Next** pour continuer.
* Terminez l'assistant comme décrit dans **Adding a Data Provider**.

***

## **Considérations importantes**

* **Si vous utilisez PADS4 Data Server :**
  * Assurez-vous que le **flux Bluefox** est **accessible** depuis le **réseau du PADS4 Data Server**.
  * Le **Data Server** va **récupérer le flux** et le distribuer à tous les **PADS4 Viewers** connectés.

* **Si vous n'utilisez pas PADS4 Data Server :**
  * Assurez-vous que **chaque PADS4 Viewer** et **PADS4 Designer** a un accès direct au **flux Bluefox**.

***

## **Dépannage**

* **Le flux ne se charge pas ?**
  * Vérifiez l'**URL du flux** pour son exactitude.
  * Assurez-vous que votre **connexion internet** est active.
  * Si vous utilisez un **proxy**, vérifiez les **paramètres du proxy**.

* **L'aperçu n'affiche aucune donnée ?**
  * Assurez-vous que le **flux est actif** dans votre **compte Bluefox**.
  * Essayez d'actualiser l'**URL du flux** sur le site web Bluefox.

***

Une fois configuré, le **flux de contenu Bluefox** se mettra à jour automatiquement, garantissant un **contenu dynamique et attrayant** pour votre **affichage dynamique**.
