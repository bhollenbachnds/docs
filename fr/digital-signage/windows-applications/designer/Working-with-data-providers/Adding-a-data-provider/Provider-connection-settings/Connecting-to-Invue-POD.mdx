Le fournisseur **Invue POD** dans **PADS4** vous permet de vous connecter à un **Invue Merchandising DIB (Device Interface Box)**, qui gère les **POD (Point of Display) security devices**. Cette configuration aide à intégrer les **solutions de sécurité pour le merchandising** avec les **systèmes d'affichage dynamique**.

***

## **Étapes pour ajouter/modifier une connexion Invue dans PADS4**

### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez l'**Data Provider Wizard** en suivant les étapes dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Invue POD** comme **type de fournisseur**.

***

### **2. Configurer les paramètres de connexion Invue POD**

* **Sélectionnez le port COM :**
  * Choisissez le **port COM** auquel le **DIB (Device Interface Box)** est connecté.
  * En cas de doute, vérifiez le **Gestionnaire de périphériques Windows** pour trouver le bon **port série**.

Cliquez sur **Next** pour continuer.

***

### **3. Terminer la configuration**

* Cliquez sur **Finish** pour finaliser la connexion.
* Le **PADS4 Viewer** sera maintenant capable de récupérer en temps réel les **données Invue POD**.

***

## **Considérations importantes**

### **Configuration du PADS4 Data Server**

* Si vous utilisez le **PADS4 Data Server**, assurez-vous que le **périphérique Invue DIB est connecté** au **Data Server**.
* Si vous **n'utilisez pas de Data Server**, le **périphérique DIB doit être connecté** au **PC Viewer** à la place.

### **Connectivité des périphériques**

* Assurez-vous que le **périphérique DIB** est correctement installé et reconnu par le système.
* Les **pilotes logiciels Invue** doivent être installés sur le **serveur ou le PC Viewer**.

***

## **Cas d'utilisation**

* **Sécurité en magasin :** Surveiller l'état de **sécurité des marchandises**.
* **Engagement client :** Intégrer les **données d'interaction avec les produits** dans l'**affichage dynamique**.
* **Prévention des pertes :** Alerter le personnel si les **conditions de sécurité changent**.

En suivant ces étapes, **PADS4** intègre avec succès les **périphériques Invue POD** pour **surveillance de sécurité en temps réel et analyses** dans **les environnements de vente au détail et de marchandisage**.
