Le **Folder** type de fournisseur vous permet de vous connecter à un **dossier spécifique** pour récupérer et afficher son **contenu de fichiers**. Vous pouvez inclure **tous les fichiers** ou les filtrer en fonction des **extensions**. **Les sous-dossiers** peuvent également être inclus si nécessaire.

***

## **Étapes pour ajouter/modifier une connexion de dossier dans PADS4**

### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez le **Data Provider Wizard** en suivant les instructions dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Folder** comme **type de fournisseur**.

### **2. Spécifier les paramètres de connexion au dossier**

* Choisissez l'une des **options de source** suivantes :
  * **Content Library :** Utilise un dossier de la **PADS4 Content Library** (Recommandé).
  * **Folder:** Utilise un **dossier externe** à la bibliothèque de contenu.

* **Entrez le chemin du dossier** :
  * Si vous utilisez la **Content Library**, fournissez le **chemin relatif**.
  * Si vous utilisez un **dossier externe**, fournissez le **chemin absolu** (par ex., `C:\MyFolder`).
  * Utilisez le bouton **Browse** pour localiser le dossier.

* **Inclure les sous-dossiers** *(Optionnel)*
  * Activez cette option pour inclure les fichiers **des sous-dossiers** du dossier sélectionné.

* **Définir des filtres de fichiers**
  * Utilisez **Search Extension Restriction** pour **filtrer des types de fichiers spécifiques** (par ex., `.jpg`, `.png`, `.mp4`).

* **Trier les fichiers**
  * Utilisez les **Sorting File List** paramètres pour **définir l'ordre de tri**.

### **3. Finaliser la connexion**

* Cliquez sur **Next** pour continuer.
* Complétez l'assistant comme décrit dans **Adding a Data Provider**.

***

## **Meilleures pratiques**

* **Utiliser la PADS4 Content Library**
  * Les fichiers de la **Content Library** sont **automatiquement synchronisés** sur **tous les systèmes PADS4**.
  * Si vous utilisez un **dossier externe**, vous devez **distribuer manuellement** les fichiers sur les systèmes.

* **Éviter les dossiers réseau partagés**
  * **Le trafic réseau peut augmenter** lors de l'utilisation d'un **lecteur mappé** ou d'un **chemin UNC** (`\\server\folder`).
  * Cela peut **ralentir** les **performances du Viewer** lors de la récupération des listes de fichiers ou du chargement du contenu.

* **Règles de tri**
  * **Tri spécifique au dossier** s'applique en premier.
  * **Tri générique** (défini dans les étapes ultérieures de l'assistant) **remplace le tri par dossier**.

***

## **Considérations importantes**

* **Accès au serveur de données**
  * Si vous utilisez un **PADS4 Data Server**, assurez-vous que le **dossier est accessible** depuis le **point de vue du serveur**.
  * Si **vous n'utilisez pas de Data Server**, chaque **PADS4 Designer et Viewer** doit avoir un **accès direct** au dossier.

En suivant ces étapes, vous pouvez **connecter un dossier** et récupérer son contenu pour l'utiliser dans des **présentations d'affichage dynamique**.
