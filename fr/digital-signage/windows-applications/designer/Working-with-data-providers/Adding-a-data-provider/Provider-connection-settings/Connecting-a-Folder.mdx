Le **Dossier** le type de fournisseur vous permet de vous connecter à un **dossier spécifique** pour récupérer et afficher son **contenu de fichiers**. Vous pouvez inclure **tous les fichiers** ou les filtrer en fonction des **extensions**. **Sous-dossiers** peuvent également être inclus si nécessaire.

***

## **Étapes pour ajouter/modifier une connexion de dossier dans PADS4**

### **1. Démarrer l'Assistant de fournisseur de données**

* Ouvrez l'**Assistant de fournisseur de données** en suivant les instructions dans **Ajout d'un fournisseur de données** ou **Modification d'un fournisseur de données**.
* Sélectionnez **Dossier** comme **type de fournisseur**.

### **2. Spécifier les paramètres de connexion du dossier**

* Choisissez l'une des **options de source** suivantes :
  * **Bibliothèque de contenu :** Utilise un dossier de la **Bibliothèque de contenu PADS4** (Recommandé).
  * **Folder:** Utilise un **dossier externe** à la bibliothèque de contenu.

* **Saisir le chemin du dossier** :
  * Si vous utilisez la **Bibliothèque de contenu**, fournissez le **chemin relatif**.
  * Si vous utilisez un **dossier externe**, fournissez le **chemin absolu** (par exemple, `C:\MyFolder`).
  * Utilisez le bouton **Parcourir** pour localiser le dossier.

* **Inclure les sous-dossiers** *(Optionnel)*
  * Activez cette option pour inclure les fichiers **des sous-dossiers** du dossier sélectionné.

* **Définir les filtres de fichiers**
  * Utilisez **Restriction de recherche par extension** pour **filtrer des types de fichiers spécifiques** (par exemple, `.jpg`, `.png`, `.mp4`).

* **Trier les fichiers**
  * Utilisez les paramètres **Tri de la liste de fichiers** pour **définir l'ordre de tri**.

### **3. Finaliser la connexion**

* Cliquez sur **Suivant** pour continuer.
* Complétez l'assistant comme décrit dans **Ajout d'un fournisseur de données**.

***

## **Meilleures pratiques**

* **Utiliser la bibliothèque de contenu PADS4**
  * Les fichiers dans la **Bibliothèque de contenu** sont **automatiquement synchronisés** sur **tous les systèmes PADS4**.
  * Si vous utilisez un **dossier externe**, vous devez **distribuer manuellement** les fichiers sur tous les systèmes.

* **Éviter les dossiers réseau partagés**
  * **Le trafic réseau peut augmenter** lors de l'utilisation d'un **lecteur mappé** ou **chemin UNC** (`\\server\folder`).
  * Cela peut **ralentir** les **performances du Viewer** lors de la récupération des listes de fichiers ou du chargement du contenu.

* **Règles de tri**
  * **Le tri spécifique au dossier** s'applique en premier.
  * **Le tri générique** (défini dans les étapes ultérieures de l'assistant) **remplace le tri du dossier**.

***

## **Considérations importantes**

* **Accès au serveur de données**
  * Si vous utilisez un **PADS4 Data Server**, assurez-vous que le **dossier est accessible** du **point de vue du serveur**.
  * Si vous **n'utilisez pas de Data Server**, chaque **PADS4 Designer et Viewer** doit avoir **un accès direct** au dossier.

En suivant ces étapes, vous pouvez **connecter un dossier** et récupérer son contenu pour l'utiliser dans des **présentations d'affichage dynamique**.
