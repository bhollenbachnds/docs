Le fournisseur **Instagram feed** dans **PADS4** vous permet de vous connecter à **Instagram** et de récupérer des publications, des commentaires ou du contenu spécifique à l'utilisateur pour l'affichage dynamique.

***

## **Étapes pour ajouter/modifier une connexion Instagram Feed dans PADS4**

### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez le **Data Provider Wizard** en suivant les étapes dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Instagram Feed** comme **provider type**.

***

### **2. S'authentifier avec Instagram**

* Cliquez sur le bouton **Authenticate** pour ouvrir la page de connexion d'Instagram.
* Entrez vos **Instagram credentials** et complétez le **authentication process**.
* Accordez à **PADS4 access** pour récupérer les données Instagram.
* Une fois l'authentification réussie, **PADS4 will generate an access token**.

Fermez le navigateur une fois l'authentification terminée.

***

### **3. Configurer les paramètres d'Instagram Feed**

Choisissez l'une des options suivantes :

* **Popular on Instagram:** Récupère les **most popular posts** d'Instagram.
* **Search:** Récupère les publications liées à un **specific keyword or hashtag**.
* **User Name:** Affiche les publications d'un **specific Instagram user** (doit être le nom d'utilisateur exact).
* **Get Image Comments:** Récupère les **comments** pour un **image ID** spécifié.

Cliquez sur **Next** pour continuer.

***

### **4. Terminer la configuration**

* Cliquez sur **Finish** pour finaliser la connexion Instagram feed.
* Le **PADS4 Viewer** affichera maintenant du **real-time Instagram content**.

***

## **Considérations importantes**

* **Internet Access Required:**
  * Si vous utilisez un **PADS4 Data Server**, assurez-vous qu'il a **internet access** pour récupérer les données Instagram.
  * Si **not using a Data Server**, chaque **PADS4 Designer and Viewer** doit pouvoir se connecter à Instagram.
* **Authentication Token Expiry:**
  * Les jetons d'authentification Instagram **expire periodically** et nécessitent une **re-authentication**.
* **Username Matching:**
  * Lors du filtrage par **user**, vous devez saisir le **exact username** (les noms d'utilisateur partiels ne sont pas pris en charge).
* **Content Limitations:**
  * L'API d'Instagram restreint l'accès aux **private accounts**, assurez-vous donc que le compte à partir duquel vous récupérez des données est **public**.

En suivant ces étapes, **PADS4** peut récupérer avec succès du **Instagram content** pour un affichage en temps réel dans les **digital signage presentations**.
