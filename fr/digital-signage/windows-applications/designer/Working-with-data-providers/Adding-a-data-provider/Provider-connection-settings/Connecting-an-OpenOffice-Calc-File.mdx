Le fournisseur **OpenOffice** dans **PADS4** vous permet de vous connecter aux **OpenOffice Calc files (.ods)** pour récupérer et afficher des données de feuilles de calcul.

***

### **Étapes pour ajouter ou modifier une connexion de fichier OpenOffice**

#### **1. Démarrer l'Assistant de fournisseur de données**

* Ouvrez **PADS4** et naviguez vers **Data Provider Wizard**.
* Sélectionnez **OpenOffice File** comme **provider type**.
* Cliquez sur **Next** pour continuer.

#### **2. Spécifier l'emplacement du fichier**

* Choisissez où le fichier est stocké :
  * **Content Library** – Recommandé, car il **se synchronise automatiquement** sur tous les appareils connectés à PADS4.
  * **File Name** – Sélectionnez un fichier OpenOffice en dehors de la bibliothèque de contenu *(vous devrez gérer manuellement la distribution)*.
* Cliquez sur **Browse** pour localiser le fichier, ou entrez manuellement le chemin complet.
* Cliquez sur **Next** pour continuer.

#### **3. Sélectionner la feuille**

* Choisissez la **feuille spécifique** du fichier OpenOffice Calc.
* Cliquez sur **Preview** pour vérifier si les données sont correctement chargées.
* Cliquez sur **Next** pour continuer.

#### **4. Terminer la configuration**

* Vérifiez les paramètres de connexion.
* Cliquez sur **Finish** pour enregistrer le fournisseur de données.

***

### **Remarques importantes**

* **Considérations de performance** :
  * L'utilisation d'un **fichier OpenOffice externe** *(en dehors de la Content Library)* nécessite une **distribution manuelle des fichiers**.
  * Si le fichier OpenOffice est volumineux, assurez-vous que **PADS4 dispose de ressources suffisantes** pour le traitement.
* **Sécurité et accessibilité** :
  * Si vous utilisez un **PADS4 Data Server**, le fichier **doit être accessible** depuis l'emplacement du serveur.
  * Sans PADS4 Data Server, assurez-vous que **chaque PADS4 Viewer** peut localiser et accéder au fichier OpenOffice spécifié.
* **Utilisation hors ligne** :
  * Vous pouvez **télécharger** des données depuis un **flux RSS** et les enregistrer comme **fichier OpenOffice Calc** pour un accès hors ligne.
  * Pour ce faire, ouvrez le **flux RSS dans un navigateur**, sélectionnez **View Source**, et enregistrez-le comme **fichier .txt ou .csv** avant de l'importer dans OpenOffice.

***

Cette configuration assure **un accès en temps réel ou programmé** aux **données OpenOffice Calc** dans **PADS4**, ce qui est utile pour **les présentations dynamiques, la visualisation de données et les rapports**.
