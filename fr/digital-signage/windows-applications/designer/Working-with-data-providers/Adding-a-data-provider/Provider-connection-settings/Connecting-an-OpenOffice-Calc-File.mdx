Le **OpenOffice** fournisseur dans **PADS4** vous permet de vous connecter à des **fichiers OpenOffice Calc (.ods)** pour récupérer et afficher des données de feuille de calcul.

***

### **Étapes pour ajouter ou modifier une connexion à un fichier OpenOffice**

#### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez **PADS4** et accédez à l'assistant **Data Provider Wizard**.
* Sélectionnez **Fichier OpenOffice** comme **type de fournisseur**.
* Cliquez sur **Suivant** pour continuer.

#### **2. Spécifier l'emplacement du fichier**

* Choisissez l'emplacement de stockage du fichier :
  * **Bibliothèque de contenu** – Recommandé, car elle **synchronise automatiquement** sur tous les appareils connectés à PADS4.
  * **Nom de fichier** – Sélectionnez un fichier OpenOffice en dehors de la bibliothèque de contenu *(vous devrez gérer la distribution manuellement)*.
* Cliquez sur **Parcourir** pour localiser le fichier, ou saisissez manuellement le chemin complet.
* Cliquez sur **Suivant** pour continuer.

#### **3. Sélectionner la feuille**

* Choisissez la **feuille spécifique** dans le fichier OpenOffice Calc.
* Cliquez sur **Aperçu** pour vérifier si les données sont correctement chargées.
* Cliquez sur **Suivant** pour continuer.

#### **4. Finaliser la configuration**

* Vérifiez les paramètres de connexion.
* Cliquez sur **Terminer** pour enregistrer le fournisseur de données.

***

### **Remarques importantes**

* **Considérations de performance** :
  * L'utilisation d'un **fichier OpenOffice externe** *(en dehors de la bibliothèque de contenu)* nécessite une **distribution manuelle des fichiers**.
  * Si le fichier OpenOffice est volumineux, assurez-vous que **PADS4 dispose de ressources suffisantes** pour le traitement.
* **Sécurité et accessibilité** :
  * Si vous utilisez un **PADS4 Data Server**, le fichier **doit être accessible** depuis l'emplacement du serveur.
  * Sans PADS4 Data Server, assurez-vous que **chaque PADS4 Viewer** peut localiser et accéder au fichier OpenOffice spécifié.
* **Utilisation hors ligne** :
  * Vous pouvez **télécharger** des données depuis un **flux RSS** et les enregistrer sous forme de **fichier OpenOffice Calc** pour un accès hors ligne.
  * Pour ce faire, ouvrez le **flux RSS dans un navigateur**, sélectionnez **Afficher la source**, et enregistrez-le en tant que **fichier .txt ou .csv** avant de l'importer dans OpenOffice.

***

Cette configuration garantit **un accès en temps réel ou planifié** aux **données OpenOffice Calc** dans **PADS4**, ce qui le rend utile pour **les présentations dynamiques, la visualisation de données et les rapports**.
