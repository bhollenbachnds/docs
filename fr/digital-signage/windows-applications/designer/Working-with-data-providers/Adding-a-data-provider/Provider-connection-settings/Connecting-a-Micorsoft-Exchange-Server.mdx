---
- title: Connexion à un serveur Microsoft Exchange
---

Le **Microsoft Exchange Server** fournisseur dans **PADS4** vous permet de vous connecter aux **Microsoft Exchange Servers** (Office 2010 ou 2007 SP1) pour récupérer des données de **calendrier, boîte de réception ou contacts**.

***

### **Étapes pour ajouter ou modifier une connexion Microsoft Exchange Server**

#### **1. Démarrer l'Assistant Fournisseur de Données**

* Ouvrez l'**Assistant Fournisseur de Données** dans **PADS4**.
* Suivez les étapes pour **Ajouter un Fournisseur de Données** ou **Modifier un Fournisseur de Données**.
* Sélectionnez **Microsoft Exchange Server** comme **type de fournisseur**.

***

#### **2. Configurer les paramètres de connexion**

* **Sélectionner la version de Microsoft Exchange Server**
  * Choisissez la **version d'Exchange Server** appropriée parmi les options disponibles.
* **Choisir la configuration automatique ou manuelle**

  * **Automatique**:
    * Saisissez l'**Adresse e-mail** d'un **compte Exchange** valide.
  * **Manuel**:
    * Saisissez l'**URL du serveur Exchange** pour établir une connexion.

  **Exemple d'URL:**

  ```
  https://YourSmallBusinessServer/EWX/Exchange.asmx
  ```
* **Saisir les identifiants de connexion**
  * Fournissez un **Nom d'utilisateur, Mot de passe et Domaine** d'un compte Exchange Server valide.
* **Accéder à la boîte aux lettres d'un autre utilisateur (Optionnel)**
  * Saisissez une **Adresse e-mail** pour récupérer les données de la boîte aux lettres d'un autre utilisateur.
* **Usurper l'identité d'un autre utilisateur (Optionnel)**
  * Saisissez une **Adresse e-mail** pour usurper l'identité d'un utilisateur différent.

Cliquez sur **Suivant** pour continuer.

***

#### **3. Sélectionner la source de données à laquelle se connecter**

* **Calendrier**
  * Récupère les données du calendrier.
  * En option, activez **Inclure le texte du message** pour récupérer les détails des événements.
  * *Note:* La récupération du corps du message **peut causer des problèmes de performance**.
* **Boîte de réception**
  * Récupère les e-mails de la **Boîte de réception**.
  * En option, spécifiez:
    * Un **(sous-)dossier** spécifique pour récupérer les e-mails.
    * **Inclure le texte du message** (pour le corps de l'e-mail).
* **Contacts**
  * Récupère la liste des contacts du compte Exchange.
* **Dossiers publics**
  * Accédez aux **dossiers publics** utilisés pour le partage d'informations au sein d'une organisation.
  * *Note:* Les dossiers publics n'ont pas d'**adresse SMTP** et sont accessibles via l'**API EWS**.
* **Calendrier de ressources**
  * Récupère les données de ressources **salles et réunions**.
  * Spécifiez **une ou plusieurs adresses e-mail** pour accéder à plusieurs calendriers de ressources.
  * En option, activez **Inclure le texte du message** pour récupérer les détails.
* **Disponibilité des utilisateurs**
  * Récupère le **statut libre/occupé** des utilisateurs.
  * Spécifiez **une ou plusieurs adresses e-mail** pour obtenir des données de disponibilité fusionnées.

***

#### **4. Définir la plage de temps pour la récupération des données**

* **Aujourd'hui**
  * Récupère les données du planning **d'aujourd'hui**.
  * En option, appliquez un **décalage** en jours.
* **Maintenant**
  * Récupère les données à partir de la **date et l'heure actuelles**.
  * En option, spécifiez un **décalage avant ou après** l'heure actuelle.
* **Date et heure**
  * Récupère les données pour une **période fixe**.
* **Prédéfini**
  * Sélectionnez une période prédéfinie:
    * **Cette semaine**
    * **Semaine prochaine**
    * **Ce mois-ci**

**Note importante:**

* La récupération de données pour des **longues périodes** peut causer des **problèmes de performance** en raison de grandes quantités de données stockées.
* Utilisez des **filtres** ou **limitez le nombre de lignes** pour éviter une récupération excessive de données.

Cliquez sur **Suivant** pour continuer.

***

#### **5. Terminer la configuration**

* Suivez les étapes restantes dans l'**Assistant Fournisseur de Données**.
* Cliquez sur **Terminer** pour finaliser la configuration.

***

### **Notes supplémentaires**

* **Utilisation d'un serveur de données PADS4?**
  * Assurez-vous que l'**URL du serveur Exchange** est **accessible depuis le PADS4 Server**.
* **Vous n'utilisez pas de PADS4 Data Server ?**
  * Assurez-vous que **PADS4 Designer** (pour la prévisualisation) et tous les **PADS4 Viewers** peuvent accéder au serveur Exchange.
* **Évitez les charges de données importantes**
  * Lors de la récupération de **longues périodes de données**, utilisez des filtres pour limiter l'utilisation de la mémoire.
