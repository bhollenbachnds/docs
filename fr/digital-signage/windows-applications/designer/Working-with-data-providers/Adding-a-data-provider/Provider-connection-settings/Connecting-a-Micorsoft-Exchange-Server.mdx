---
- title: Connexion d'un serveur Microsoft Exchange
---

Le fournisseur **Microsoft Exchange Server** dans **PADS4** vous permet de vous connecter aux **Microsoft Exchange Servers** (Office 2010 ou 2007 SP1) pour récupérer des données de **calendar, inbox, or contacts**.

***

### **Étapes pour ajouter ou modifier une connexion Microsoft Exchange Server**

#### **1. Démarrer l'Assistant Fournisseur de Données**

* Ouvrez l'**Data Provider Wizard** dans **PADS4**.
* Suivez les étapes pour **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Microsoft Exchange Server** comme **provider type**.

***

#### **2. Configurer les paramètres de connexion**

* **Sélectionner la version de Microsoft Exchange Server**
  * Choisissez la **Exchange Server version** appropriée parmi les options disponibles.
* **Choisir la configuration automatique ou manuelle**

  * **Automatic**:
    * Entrez l'**E-mail Address** d'un **Exchange account** valide.
  * **Manual**:
    * Entrez l'**Exchange Server URL** pour établir une connexion.

  **Exemple d'URL:**

  ```
  https://YourSmallBusinessServer/EWX/Exchange.asmx
  ```
* **Saisir les identifiants de connexion**
  * Fournissez un **User Name, Password, and Domain** d'un compte Exchange Server valide.
* **Accéder à la boîte aux lettres d'un autre utilisateur (Optionnel)**
  * Entrez une **E-mail Address** pour récupérer les données de la boîte aux lettres d'un autre utilisateur.
* **Usurper l'identité d'un autre utilisateur (Optionnel)**
  * Entrez une **E-mail Address** pour usurper l'identité d'un utilisateur différent.

Cliquez sur **Next** pour continuer.

***

#### **3. Sélectionner la source de données à laquelle se connecter**

* **Calendar**
  * Récupère les données du calendrier.
  * Optionnellement, activez **Include Message Text** pour récupérer les détails des événements.
  * *Note:* La récupération du corps du message **may cause performance issues**.
* **Inbox**
  * Récupère les emails de la **Inbox**.
  * Optionnellement, spécifiez:
    * Un **(sub)folder** spécifique pour récupérer les emails.
    * **Include Message Text** (pour le corps de l'email).
* **Contacts**
  * Récupère la liste des contacts du compte Exchange.
* **Public Folders**
  * Accédez aux **public folders** utilisés pour le partage d'informations au sein d'une organisation.
  * *Note:* Les dossiers publics n'ont pas d'**SMTP address** et sont accessibles via l'**EWS API**.
* **Resource Calendar**
  * Récupère les données de ressources **room and meeting**.
  * Spécifiez **one or more e-mail addresses** pour accéder à plusieurs calendriers de ressources.
  * Optionnellement, activez **Include Message Text** pour récupérer les détails.
* **User Availability**
  * Récupère le statut **free/busy status** des utilisateurs.
  * Spécifiez **one or more e-mail addresses** pour obtenir des données de disponibilité fusionnées.

***

#### **4. Définir la plage de temps pour la récupération des données**

* **Today**
  * Récupère les données de l'emploi du temps **today's**.
  * Optionnellement, appliquez un **offset** en jours.
* **Now**
  * Récupère les données à partir de la **current date and time**.
  * Optionnellement, spécifiez un **offset before or after** l'heure actuelle.
* **Date and Time**
  * Récupère les données pour une **fixed period**.
* **Predefined**
  * Sélectionnez une période prédéfinie:
    * **This Week**
    * **Next Week**
    * **This Month**

**Note importante:**

* La récupération de données pour des **long periods** peut causer des **performance issues** en raison de grandes quantités de données stockées.
* Utilisez des **filters** ou **limit the number of rows** pour éviter une récupération excessive de données.

Cliquez sur **Suivant** pour continuer.

***

#### **5. Terminer la configuration**

* Suivez les étapes restantes dans l'**Assistant de fournisseur de données**.
* Cliquez sur **Terminer** pour finaliser la configuration.

***

### **Notes supplémentaires**

* **Utilisation d'un serveur de données PADS4 ?**
  * Assurez-vous que l'**URL du serveur Exchange** est **accessible depuis le serveur PADS4**.
* **Pas d'utilisation d'un serveur de données PADS4 ?**
  * Assurez-vous que **PADS4 Designer** (pour la prévisualisation) et tous les **PADS4 Viewers** peuvent accéder au serveur Exchange.
* **Éviter les charges importantes de données**
  * Lors de la récupération de **longues périodes de données**, utilisez des filtres pour limiter l'utilisation de la mémoire.
