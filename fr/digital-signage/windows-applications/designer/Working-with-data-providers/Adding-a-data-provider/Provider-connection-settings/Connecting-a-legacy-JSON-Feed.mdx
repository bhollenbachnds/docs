**JSON (JavaScript Object Notation)** est un format léger largement utilisé pour **l'échange de données**. PADS4 prend en charge **l'intégration de flux JSON hérités**, permettant la récupération de données en temps réel pour les présentations d'affichage dynamique.

***

## **Étapes pour ajouter ou modifier une connexion de flux JSON**

### **1. Démarrer l'Assistant fournisseur de données**

* Ouvrez l'**Assistant fournisseur de données** en suivant les instructions dans **Ajouter un fournisseur de données** ou **Modifier un fournisseur de données**.
* Sélectionnez **Flux JSON** comme type de fournisseur.

***

### **2. Spécifier l'URL du flux JSON**

* Entrez l'**URI du flux JSON** (URL) pointant vers la source de données.
* Cliquez sur **Aperçu** pour voir un échantillon des données récupérées.

Cliquez sur **Suivant** pour continuer.

***

### **3. Choisir la méthode de requête**

* **GET (par défaut)** → Récupère des données sans les modifier.
* **POST** → Utilisé pour envoyer des paramètres supplémentaires.
  * Si **POST** est sélectionné, entrez le **corps de la requête** avec les paramètres requis.

***

### **4. Configurer le formatage des données**

* **Aplatir les données hiérarchiques** → Convertit les **structures JSON imbriquées** en un format de tableau plat.
* **Inclure les noms de colonnes des niveaux parents** → Ajoute des **identifiants de niveau parent** pour les relations hiérarchiques.
* **Créer des colonnes supplémentaires sans balises HTML** → Supprime le **formatage HTML** et extrait le **texte brut**.

Cliquez sur **Suivant** pour continuer.

***

### **5. Configurer la sécurité et l'authentification**

* **Autorisation (si nécessaire)**
  * Activez **Autorisation requise**.
  * Entrez **ID utilisateur, Mot de passe et Domaine**.

* **Serveur proxy (si applicable)**
  * Configurez les paramètres du proxy si votre réseau nécessite une authentification.

* **Identification de l'agent utilisateur**
  * Si nécessaire, spécifiez une **chaîne d'agent utilisateur personnalisée** (utile pour les restrictions d'accès API).

Cliquez sur **Suivant** pour continuer.

***

### **6. Finalisation de la configuration du flux JSON**

* Cliquez sur **Terminer** pour compléter la configuration.

***

## **Considérations importantes**

* **PADS4 Data Server**
  * Si vous utilisez un **PADS4 Data Server**, assurez-vous que le **flux JSON est accessible depuis le serveur**.
  * Si vous n'utilisez pas de Data Server, assurez-vous que **PADS4 Designer (pour l'aperçu) et les Viewers** peuvent accéder au flux JSON.

* **Gestion des données JSON hiérarchiques**
  * Si votre structure JSON comporte **plusieurs niveaux**, assurez-vous que **Aplatir les données hiérarchiques** est activé pour une sortie structurée.

En suivant ces étapes, **PADS4 peut récupérer et afficher efficacement des données en temps réel basées sur JSON** dans les présentations d'affichage dynamique.
