Le **Facebook provider** dans **PADS4** vous permet de **vous connecter à une Page Facebook Business** et de récupérer du contenu en utilisant l'**Facebook Graph API (v15.0)**.

### **Prérequis**

* Une **Facebook/Meta Business Page**.
* Un **Facebook account** valide avec accès à la page.
* **Accès Internet** pour l'authentification et la récupération des données.

***

## **Étapes pour ajouter/modifier une connexion Facebook dans PADS4**

### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez le **Data Provider Wizard** en suivant les instructions dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Facebook** comme **provider type**.

### **2. Authentifier votre compte Facebook**

* Cliquez sur **Authenticate** pour ouvrir la **page de connexion Facebook**.
* Entrez vos **credentials** et cliquez sur **OK**.
* PADS4 générera un **token** et fermera automatiquement la fenêtre.
* La **date d'expiration** du token sera affichée.

**Notes importantes sur l'authentification :**

* Chaque fois que vous modifiez le **Facebook Data Provider**, vous devez **vous ré-authentifier**.
* Si les **identifiants sont enregistrés** dans le navigateur, le processus sera **automatique**.
* Le **token est valide pendant 60 jours** et est **automatiquement prolongé** tant qu'il est utilisé pendant cette période.
* Si le **token expire**, Facebook **ne renverra plus de données**, et vous devrez **vous ré-authentifier manuellement**.

### **3. Sélectionner le compte et la page Facebook**

* **Choisissez le type de compte**:
  * **My account** (récupère les données de votre propre compte Facebook).
  * **Page** (récupère des données depuis un **Facebook Business Page**).
* Si vous sélectionnez une **Page**, saisissez l'**Page ID** (recommandé) ou sélectionnez-le dans la liste.

**Important:**

* Le **Page ID number** se trouve sous le **Page Name** dans les **Facebook settings**.
* **Les noms d'albums dans Facebook ne sont pas uniques**—PADS4 sélectionnera le **first matching album name**.

### **4. Charger les détails de la page et spécifier la source de contenu**

* Cliquez sur **Load** pour récupérer les **page-specific details**.
* Choisissez le type de **content to retrieve** :
  * **Photos** (vous pouvez spécifier un album).
  * **Other Facebook content** (comme les publications ou les événements).
* **Limitez le nombre de lignes retournées** pour de **better performance**.

### **5. Finaliser la connexion**

* Cliquez sur **Next** pour continuer.
* Complétez l'assistant comme décrit dans **Adding a Data Provider**.

***

## **Bonnes pratiques**

* **Utilisez le Page ID** au lieu du **Page Name** pour garantir que la page correcte est sélectionnée.
* **Maintenez l'authentification active** en vous assurant que le **data provider is used at least once every 60 days**.
* **Limitez le nombre de lignes** pour une **faster data retrieval** et de **better performance**.

***

## **Considérations importantes**

* **Expiration du jeton :**
  * Les jetons d'authentification Facebook **expire after 60 days**.
  * Si une **presentation** n'est **not used for more than 60 days**, une **re-authentication** est nécessaire.

* **Accès au serveur de données :**
  * Si vous utilisez un **PADS4 Data Server**, assurez-vous que **Facebook is accessible** depuis le **server's point of view**.
  * Si vous **not using a Data Server**, vérifiez que chaque **PADS4 Designer and Viewer** dispose d'une **active internet connection**.

En suivant ces étapes, vous pouvez **connect PADS4 to Facebook** et **display dynamic social media content** dans vos **digital signage presentations**.
