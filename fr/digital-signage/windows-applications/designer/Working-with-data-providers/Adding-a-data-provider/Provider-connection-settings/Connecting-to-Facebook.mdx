Le **fournisseur Facebook** dans **PADS4** vous permet de **vous connecter à une page Facebook Business** et de récupérer du contenu via l'**API Facebook Graph (v15.0)**.

### **Prérequis**

* Une **page Facebook/Meta Business**.
* Un **compte Facebook** valide avec accès à la page.
* **Un accès Internet** pour l'authentification et la récupération des données.

***

## **Étapes pour ajouter/modifier une connexion Facebook dans PADS4**

### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez l'**assistant de fournisseur de données** en suivant les instructions dans **Ajouter un fournisseur de données** ou **Modifier un fournisseur de données**.
* Sélectionnez **Facebook** comme **type de fournisseur**.

### **2. Authentifier votre compte Facebook**

* Cliquez sur **Authenticate** pour ouvrir la **page de connexion Facebook**.
* Entrez vos **identifiants** et cliquez sur **OK**.
* PADS4 générera un **token** et fermera automatiquement la fenêtre.
* La **date d'expiration** du token sera affichée.

**Remarques importantes sur l'authentification :**

* Chaque fois que vous modifiez le **fournisseur de données Facebook**, vous devez **vous ré-authentifier**.
* Si les **identifiants sont enregistrés** dans le navigateur, le processus sera **automatique**.
* Le **token est valide pendant 60 jours** et est **automatiquement prolongé** tant qu'il est utilisé dans ce délai.
* Si le **token expire**, Facebook ne **retournera plus de données**, et vous devez **vous ré-authentifier manuellement**.

### **3. Sélectionner le compte Facebook et la page**

* **Choisissez le type de compte** :
  * **Mon compte** (récupère les données de votre propre compte Facebook).
  * **Page** (récupère les données d'une **page Facebook Business**).
* Si vous sélectionnez une **Page**, entrez l'**ID de la page** (recommandé) ou sélectionnez-la dans la liste.

**Important:**

* Le **numéro d'ID de la page** se trouve sous le **nom de la page** dans les **paramètres Facebook**.
* **Les noms d'albums sur Facebook ne sont pas uniques**—PADS4 sélectionnera le **premier nom d'album correspondant**.

### **4. Charger les détails de la page et spécifier la source de contenu**

* Cliquez sur **Load** pour récupérer les **détails spécifiques à la page**.
* Choisissez le type de **contenu à récupérer** :
  * **Photos** (vous pouvez spécifier un album).
  * **Autre contenu Facebook** (comme des publications ou des événements).
* **Limiter le nombre de lignes retournées** pour de **meilleures performances**.

### **5. Finaliser la connexion**

* Cliquez sur **Next** pour continuer.
* Complétez l'assistant comme décrit dans **Ajouter un fournisseur de données**.

***

## **Bonnes pratiques**

* **Utilisez l'ID de la page** plutôt que le **nom de la page** pour vous assurer que la bonne page est sélectionnée.
* **Maintenez l'authentification active** en vous assurant que le **fournisseur de données est utilisé au moins une fois tous les 60 jours**.
* **Limitez le nombre de lignes** pour une **récupération des données plus rapide** et de **meilleures performances**.

***

## **Considérations importantes**

* **Expiration du token :**
  * Les tokens d'authentification Facebook **expirent après 60 jours**.
  * Si une **présentation** n'est **pas utilisée pendant plus de 60 jours**, **une ré-authentification** est requise.

* **Accès au serveur de données :**
  * Si vous utilisez un **PADS4 Data Server**, assurez-vous que **Facebook est accessible** depuis le **point de vue du serveur**.
  * Si vous **n'utilisez pas de Data Server**, vérifiez que chaque **PADS4 Designer et Viewer** dispose d'une **connexion Internet active**.

En suivant ces étapes, vous pouvez **connecter PADS4 à Facebook** et **afficher du contenu dynamique sur les réseaux sociaux** dans votre **présentations d'affichage dynamique**.
