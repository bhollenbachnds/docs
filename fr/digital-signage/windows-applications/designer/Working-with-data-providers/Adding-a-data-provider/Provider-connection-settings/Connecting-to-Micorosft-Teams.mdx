Le **Microsoft Teams** fournisseur de données permet **PADS4** de se connecter à un **Microsoft Teams Azure environment** et de récupérer les **Teams meetings** programmées pour une utilisation dans les **PADS4 presentations**.

***

### **Prérequis**

Avant de configurer le **Microsoft Teams Data Provider**, assurez-vous que les exigences suivantes sont satisfaites :

1. **Configuration du certificat SSL**
   * Votre **PADS4 server** doit avoir un **SSL certificate** configuré.
   * La communication doit être établie via **HTTPS**.
   * Plus de détails sont disponibles dans le **PADS4 Partner Portal** sous *PADS4 and SSL Certificates*.

2. **Accès Administrateur Azure IT**
   * Vous avez besoin d'un **Azure IT Administrator** pour lier le **PADS4 server** à l'environnement **Microsoft Teams Azure environment**.

3. **Utilisateur Microsoft Teams avec les autorisations appropriées**
   * L'**Teams user** sélectionné doit avoir les **permissions** et **licenses** nécessaires pour accéder aux **calendar information**.

4. **Configuration de Microsoft Teams dans PADS4**
   * Configurez les **Microsoft Teams credentials** dans **PADS4 CTRL Center** sous la **Credentials section**.
   * Assurez-vous qu'un **Azure Administrator** est disponible pour la configuration initiale.

***

### **Étapes pour ajouter/modifier un Microsoft Teams Data Provider**

#### **1. Démarrer l'Assistant Fournisseur de Données**

* Ouvrez l'**Data Provider Wizard** dans PADS4.
* Suivez les étapes pour **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Microsoft Teams** comme **provider type**.

***

#### **2. Configurer les paramètres de connexion Microsoft Teams**

* **Paramètres Teams – User Token**
  * Récupérez le **user token** depuis **PADS4 CTRL Center** sous l'onglet **Credentials tab**.
  * Si vous n'avez pas configuré l'**Azure environment** et l'**Teams user**, suivez le **PADS4 tutorial** pour l'**Azure integration**.
  * Copiez et collez le **user/refresh token** dans le champ approprié.

* **Utiliser un autre PADS4 Server (Optionnel)**
  * Si vous avez plusieurs **PADS4 servers**, vous pouvez en sélectionner un autre avec une **Teams connection** existante.

* **Aperçu des données**
  * Cliquez sur le bouton **Preview** pour voir les **current data** récupérées depuis l'**Teams user** configuré.

***

#### **3. Finaliser la configuration**

* Cliquez sur **Next** pour continuer.
* Complétez la configuration en suivant les étapes restantes dans l'**Data Provider Wizard**.

***

### **Notes supplémentaires**

* Si vous rencontrez des **connection issues**, vérifiez que votre **PADS4 server**, **Teams user** et **Azure configuration** sont correctement configurés.
* Assurez-vous que l'**Teams user** dispose des **permissions and licenses** appropriées pour accéder aux **calendar data**.
* Pour plus d'informations, consultez le **PADS4 Partner Portal** ou contactez le **Azure IT support**.

Cette configuration permet à **PADS4** de **retrieve and display Microsoft Teams meeting data** en **real-time** dans vos **presentations**.
