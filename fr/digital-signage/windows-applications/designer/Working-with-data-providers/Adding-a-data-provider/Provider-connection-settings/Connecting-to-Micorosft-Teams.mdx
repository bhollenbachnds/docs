---
- title: Connexion à Microsoft Teams
---

Le fournisseur de données **Microsoft Teams** permet à **PADS4** de se connecter à un **Microsoft Teams Azure environment** et de récupérer les **Teams meetings** programmées pour une utilisation dans les **PADS4 presentations**.

***

### **Prérequis**

Avant de configurer le **Microsoft Teams Data Provider**, assurez-vous que les exigences suivantes sont satisfaites :

1. **Configuration du Certificat SSL**
   * Votre **PADS4 server** doit avoir un **certificat SSL** configuré.
   * La communication doit être établie via **HTTPS**.
   * Plus de détails peuvent être trouvés dans le **PADS4 Partner Portal** sous *PADS4 and SSL Certificates*.
2. **Accès Administrateur Azure IT**
   * Vous avez besoin d'un **Azure IT Administrator** pour lier le **PADS4 server** au **Microsoft Teams Azure environment** correct.
3. **Utilisateur Microsoft Teams avec les Autorisations Appropriées**
   * L'**Teams user** sélectionné doit avoir les **permissions** et **licenses** nécessaires pour accéder aux **informations de calendrier**.
4. **Configuration de Microsoft Teams dans PADS4**
   * Configurez les **Microsoft Teams credentials** dans le **PADS4 CTRL Center** sous la **section Credentials**.
   * Assurez-vous qu'un **Azure Administrator** est disponible pour la configuration initiale.

***

### **Étapes pour Ajouter/Modifier un Microsoft Teams Data Provider**

#### **1. Démarrer l'Assistant de Fournisseur de Données**

* Ouvrez l'**Data Provider Wizard** dans PADS4.
* Suivez les étapes pour **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Microsoft Teams** comme **type de fournisseur**.

***

#### **2. Configurer les Paramètres de Connexion Microsoft Teams**

* **Paramètres Teams – Jeton Utilisateur**
  * Récupérez le **jeton utilisateur** depuis le **PADS4 CTRL Center** sous l'**onglet Credentials**.
  * Si vous n'avez pas configuré l'**Azure environment** et l'**Teams user**, suivez le **tutoriel PADS4** pour l'**intégration Azure**.
  * Copiez et collez le **jeton utilisateur/d'actualisation** dans le champ approprié.
* **Utiliser un Serveur PADS4 Différent (Optionnel)**
  * Si vous avez plusieurs **serveurs PADS4**, vous pouvez en sélectionner un différent avec une **connexion Teams** existante.
* **Aperçu des Données**
  * Cliquez sur le bouton **Preview** bouton pour afficher le **données actuelles** récupérées à partir de l'utilisateur **Teams user**.

***

#### **3. Finaliser la configuration**

* Cliquez sur **Next** pour continuer.
* Complétez la configuration en suivant les étapes restantes dans le **Data Provider Wizard**.

***

### **Notes supplémentaires**

* Si vous rencontrez des **problèmes de connexion**, vérifiez que votre **PADS4 server**, **Teams user**, et **configuration Azure** sont correctement configurés.
* Assurez-vous que l'**Teams user** dispose des **permissions et licences** appropriées pour accéder aux **données du calendrier**.
* Pour plus d'informations, consultez le **PADS4 Partner Portal** ou contactez le **support informatique Azure**.

Cette configuration permet à **PADS4** de **récupérer et afficher les données de réunion Microsoft Teams** en **temps réel** dans vos **présentations**.
