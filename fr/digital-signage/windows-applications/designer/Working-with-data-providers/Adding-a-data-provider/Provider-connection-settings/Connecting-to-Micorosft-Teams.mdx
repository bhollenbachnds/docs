---
- title: Connexion à Microsoft Teams
---

Le **Microsoft Teams** fournisseur de données permet à **PADS4** de se connecter à un **environnement Azure Microsoft Teams** et de récupérer les **réunions Teams** planifiées pour une utilisation dans les **présentations PADS4**.

***

### **Prérequis**

Avant de configurer le **fournisseur de données Microsoft Teams**, assurez-vous que les exigences suivantes sont satisfaites :

1. **Configuration du certificat SSL**
   * Votre **serveur PADS4** doit avoir un **certificat SSL** configuré.
   * La communication doit être établie via **HTTPS**.
   * Plus de détails sont disponibles sur le **PADS4 Partner Portal** sous *PADS4 et les certificats SSL*.
2. **Accès administrateur Azure IT**
   * Vous avez besoin d'un **administrateur Azure IT** pour lier le **serveur PADS4** au bon **environnement Azure Microsoft Teams**.
3. **Utilisateur Microsoft Teams avec les autorisations appropriées**
   * L'**utilisateur Teams** sélectionné doit disposer des **autorisations** et des **licences** nécessaires pour accéder aux **informations du calendrier**.
4. **Configuration de Microsoft Teams dans PADS4**
   * Configurez les **identifiants Microsoft Teams** dans **PADS4 CTRL Center** sous la **section Credentials**.
   * Assurez-vous qu'un **Azure Administrator** est disponible pour la configuration initiale.

***

### **Étapes pour ajouter/modifier un fournisseur de données Microsoft Teams**

#### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez le **Data Provider Wizard** dans PADS4.
* Suivez les étapes pour **Ajouter un fournisseur de données** ou **Modifier un fournisseur de données**.
* Sélectionnez **Microsoft Teams** comme **type de fournisseur**.

***

#### **2. Configurer les paramètres de connexion Microsoft Teams**

* **Paramètres Teams – Jeton utilisateur**
  * Récupérez le **jeton utilisateur** depuis **PADS4 CTRL Center** sous l'onglet **Credentials**.
  * Si vous n'avez pas configuré l' **environnement Azure** et l'**utilisateur Teams**, suivez le **tutoriel PADS4** pour l'**intégration Azure**.
  * Copiez et collez le **jeton utilisateur/de rafraîchissement** dans le champ approprié.
* **Utiliser un autre serveur PADS4 (Optionnel)**
  * Si vous avez plusieurs **serveurs PADS4**, vous pouvez en sélectionner un autre avec une **connexion Teams** existante.
* **Aperçu des données**
  * Cliquez sur le bouton **Aperçu** pour afficher les **données actuelles** récupérées depuis l' **utilisateur Teams** configuré.

***

#### **3. Finaliser la configuration**

* Cliquez sur **Suivant** pour continuer.
* Terminez la configuration en suivant les étapes restantes dans le **Data Provider Wizard**.

***

### **Notes supplémentaires**

* Si vous rencontrez des **problèmes de connexion**, vérifiez que votre **serveur PADS4**, **utilisateur Teams** et la **configuration Azure** sont correctement configurés.
* Assurez-vous que l' **utilisateur Teams** dispose des **autorisations et licences appropriées** pour accéder aux **données du calendrier**.
* Pour plus d'informations, consultez le **PADS4 Partner Portal** ou contactez le **support informatique Azure**.

Cette configuration permet à **PADS4** de **récupérer et afficher les données des réunions Microsoft Teams** en **temps réel** dans vos **présentations**.
