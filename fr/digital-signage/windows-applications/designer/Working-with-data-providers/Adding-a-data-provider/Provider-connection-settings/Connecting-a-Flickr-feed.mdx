Le **fournisseur de flux Flickr** dans **PADS4** vous permet de **récupérer des images** depuis **Flickr**, une plateforme de partage de photos en ligne.

***

## **Étapes pour ajouter/modifier une connexion de flux Flickr dans PADS4**

### **1. Démarrer le Data Provider Wizard**

* Ouvrez le **Data Provider Wizard** en suivant les instructions dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Flickr Feed** comme **type de fournisseur**.

### **2. Spécifier les arguments de recherche Flickr**

* Saisissez les **termes de recherche Flickr** pour **récupérer des images** correspondant à des mots-clés spécifiques.
* Vous pouvez filtrer les images en fonction de :
  * **Tags** (mots-clés pour trouver des photos pertinentes).
  * **User ID** (récupérer les images d'un utilisateur Flickr spécifique).
  * **Plage de dates** (filtrer les photos par date de téléchargement).
  * **Ordre de tri** (les plus récentes, les plus pertinentes, etc.).
* Cliquez sur **Next** pour continuer.

### **3. Terminer l'assistant**

* Cliquez sur **Finish** pour finaliser le **fournisseur de données de flux Flickr**.
* Le **PADS4 Viewer** récupérera et affichera les **images Flickr** en temps réel.

***

## **Considérations importantes**

* **Accès au Data Server :**
  * Si vous utilisez un **PADS4 Data Server**, assurez-vous que **Flickr est accessible** du **point de vue du serveur**.
  * Si **vous n'utilisez pas de Data Server**, vérifiez que chaque **PADS4 Designer et Viewer** dispose d'une **connexion internet active**.

En suivant ces étapes, vous pouvez **connecter PADS4 à Flickr** et **afficher du contenu d'images dynamiques** dans vos **présentations d'affichage dynamique**.
