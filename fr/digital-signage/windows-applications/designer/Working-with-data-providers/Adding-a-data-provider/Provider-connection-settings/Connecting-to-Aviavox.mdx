Le **fournisseur Aviavox** dans **PADS4** permet l'intégration avec le **Aviavox Artificial Voice System**, permettant des annonces vocales automatisées en temps réel.

***

### **Étapes pour ajouter ou modifier une connexion Aviavox**

#### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrez **PADS4** et accédez à l'**assistant de fournisseur de données**.
* Sélectionnez **Aviavox** comme **type de fournisseur**.
* Cliquez sur **Suivant** pour continuer.

#### **2. Configurer les paramètres de connexion Aviavox**

* Entrez l'**adresse IP du serveur** où le **serveur Aviavox** est en cours d'exécution.
* Entrez le **numéro de port** pour la communication.
* Sélectionnez le **format d'encodage du texte** approprié.
* Cliquez sur **Suivant** pour continuer.

#### **3. Finaliser la configuration**

* Vérifiez les paramètres de connexion.
* Cliquez sur **Terminer** pour enregistrer le fournisseur de données.

***

### **Fonctionnement d'Aviavox dans PADS4**

* Le **système Aviavox** envoie des annonces au **fournisseur de données Aviavox de PADS4**.
* Chaque message inclut la **durée**, **la langue**, **les zones** et le **contenu textuel**.
* Une fois le message annoncé, il est **supprimé de la table du fournisseur de données**.
* Plusieurs messages peuvent être annoncés **simultanément** dans **différentes zones**.

***

### **Structure des données Aviavox**

| **Nom de la colonne**       | **Description**                                                     |
| --------------------------- | ------------------------------------------------------------------- |
| **ID**                      | Identifiant unique du message                                       |
| **Langue**                  | Langue du message (par ex., `En`, `Fr`)                             |
| **Texte**                   | Message d'annonce                                                   |
| **Durée**                   | Temps nécessaire pour annoncer le message (en secondes)             |
| **Zones**                   | Liste de zones séparées par des virgules où le message sera annoncé |
| **Description**             | Détails supplémentaires sur l'annonce                               |
| **TextPlaylistitem**        | Annonce textuelle dans la structure de playlist PADS4               |
| **DescriptionPlaylistitem** | Description dans la structure de playlist PADS4                     |

***

### **Exemple de structure d'un message Aviavox**

```xml
<Announcements>
  <Announcement>
    <ID>G01En,DuDBA 8420EE02131649476614</ID>
    <Text>
      <LanguageText Language='En' Text='We invite business class passengers to come forward for boarding'/>
      <LanguageText Language='Du' Text='We nodigen business class passagiers uit naar voren te komen voor het instappen'/>
    </Text>
    <Duration>13</Duration>
    <Zones>A,B</Zones>
    <Description>Business class boarding</Description>
  </Announcement>
</Announcements>
```

***

### **Considérations clés**

* **Mises à jour en temps réel** : Les messages arrivent dynamiquement depuis **Aviavox** et sont supprimés après la lecture.
* **Prise en charge multi-zones** : Les messages peuvent être lus **simultanément** dans différentes zones.
* **Prise en charge des langues** : Chaque message peut être **multilingue**, avec différentes traductions.

Cette configuration garantit une **intégration transparente** des **annonces vocales Aviavox** dans votre **système d'affichage dynamique PADS4**.
