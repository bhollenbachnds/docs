---
- title: Connexion à une table de calendrier Microsoft Exchange Server
---

La table **Calendar table** dans Microsoft Exchange Server contient des informations détaillées sur les **scheduled events, meetings, and appointments**. Ci-dessous se trouve une ventilation des **columns and data types** de la table.

***

### **Colonnes et descriptions de la table Calendar**

| **Column Name**            | **Data Type**        | **Description**                                                                           |
| -------------------------- | -------------------- | ----------------------------------------------------------------------------------------- |
| **Subject**                | String               | Titre de l'événement.                                                                     |
| **Corps**                  | Chaîne de caractères | Description de l'événement (*Inclus en option*).                                          |
| **UniqueBody**             | Chaîne de caractères | Affiche la partie du corps qui est unique à la conversation dont cet élément fait partie. |
| **Importance**             | Chaîne de caractères | Priorité de l'événement (**Haute, Normale, Basse**).                                      |
| **Start**                  | Date                 | Date et heure de début de l'événement.                                                    |
| **Start\_DayOfWeek**       | Entier               | Représentation numérique du jour de la semaine de début.                                  |
| **End**                    | Date                 | Date et heure de fin de l'événement.                                                      |
| **End\_DayOfWeek**         | Entier               | Représentation numérique du jour de la semaine de fin.                                    |
| **Location**               | Chaîne de caractères | Emplacement physique ou virtuel où se déroule l'événement.                                |
| **Duration**               | Entier               | Durée de l'événement (unité par défaut : minutes).                                        |
| **DurationInSeconds**      | Entier               | Durée de l'événement en secondes.                                                         |
| **DurationInMinutes**      | Entier               | Durée de l'événement en minutes.                                                          |
| **DurationInHours**        | Entier               | Durée de l'événement en heures.                                                           |
| **DurationInDays**         | Entier               | Durée de l'événement en jours.                                                            |
| **MinutesLeftBeforeStart** | Entier               | Nombre de minutes restantes avant le début de l'événement.                                |
| **MinutesLeftBeforeEnd**   | Entier               | Nombre de minutes restantes avant la fin de l'événement.                                  |
| **Organizer**              | Chaîne de caractères | Nom de la personne qui a organisé l'événement.                                            |
| **IsAllDayEvent**          | Booléen              | Indique si l'événement dure toute la journée (**Vrai/Faux**).                             |
| **IsAssociated**           | Booléen              | Indique si l'événement est associé à un autre élément (**Vrai/Faux**).                    |
| **IsMeeting**              | Booléen              | Indique si l'événement est une réunion (**Vrai/Faux**).                                   |
| **IsOnlineMeeting**        | Booléen              | Indique si l'événement est une réunion en ligne (**Vrai/Faux**).                          |
| **IsRecurring**            | Booléen              | Indique si l'événement est un événement récurrent (**Vrai/Faux**).                        |
| **RecurrentStartDate**     | Date                 | Date de début de la **série d'événements récurrents**.                                    |
| **RecurrentEndDate**       | Date                 | Date de fin de la **série d'événements récurrents**.                                      |
| **IsReminderSet**          | Booléen              | Indique si un rappel est défini pour cet événement (**Vrai/Faux**).                       |
| **NetShowUrl**             | Chaîne de caractères | URL de la **Microsoft NetShow** réunion en ligne.                                         |
| **OriginalStart**          | Date                 | La date de début originale avant que des modifications ne soient apportées.               |
| **DateTimeCreated**        | Date                 | La date et l'heure de création de l'événement.                                            |
| **DateTimeSent**           | Date                 | La date et l'heure d'envoi de l'invitation à l'événement.                                 |
| **DateTimeReceived**       | Date                 | La date et l'heure de réception de l'invitation à l'événement.                            |
| **AppointmentType**        | Chaîne de caractères | Type de rendez-vous (**Single, RecurringMaster, Occurrence, Exception**).                 |
| **HasAttachments**         | Booléen              | Indique si l'événement a des pièces jointes (**Vrai/Faux**).                              |
| **Sensitivity**            | Chaîne de caractères | Visibilité de l'événement (**Normal, Personal, Private, Confidential**).                  |
| **Status**                 | Chaîne de caractères | **Statut Libre/Occupé hérité** de l'événement.                                            |
| **Categories**             | Chaîne de caractères | Catégories associées à cet événement.                                                     |
| **IsUnmodified**           | Booléen              | Indique si l'événement a été modifié après sa création (**Vrai/Faux**).                   |
| **LastModifiedTime**       | Date                 | La date et l'heure de la dernière modification de l'événement.                            |
| **LastModifiedName**       | Chaîne de caractères | Le nom de la personne qui a modifié l'événement en dernier.                               |
| **RequiredAttendees**      | Chaîne de caractères | Liste des **participants obligatoires**.                                                  |
| **OptionalAttendees**      | Chaîne de caractères | Liste des **participants facultatifs**.                                                   |
| **Resources**              | Chaîne de caractères | Liste des **ressources** (salles, équipements) réservées pour cette réunion.              |

***

### **Notes:**

* Cette structure de table est utilisée dans **PADS4** lors de la récupération des données de calendrier depuis **Microsoft Exchange Server**.
* Certains champs (comme **Body, UniqueBody**) peuvent être **facultatifs** selon vos paramètres Exchange.
* **Événements récurrents** ont des propriétés supplémentaires comme **RecurrentStartDate, RecurrentEndDate**.
* **Réunions en ligne** ont des attributs spécifiques comme **IsOnlineMeeting et NetShowUrl**.
