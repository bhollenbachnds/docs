Le **fournisseur de dossier FTP** dans **PADS4** vous permet de vous connecter à un **serveur FTP distant** et d'accéder à ses fichiers pour les utiliser dans vos **présentations d'affichage dynamique**.

***

## **Étapes pour ajouter/modifier une connexion de dossier FTP dans PADS4**

### **1. Démarrer l'Assistant Fournisseur de données**

* Ouvrez l'**Assistant Fournisseur de données** en suivant les étapes décrites dans **Ajout d'un fournisseur de données** ou **Modification d'un fournisseur de données**.
* Sélectionnez **FTP Folder** comme **type de fournisseur**.

### **2. Spécifier le serveur FTP et les paramètres de connexion**

* **Entrez l'URL FTP** (par ex., `ftp://yourserver.com/folder/`).
* **Authentification (si requise) :**
  * Entrez le **nom d'utilisateur** et le **mot de passe** si le serveur FTP requiert des identifiants de connexion.
* **Paramètres du proxy (si nécessaire) :**
  * Configurez un serveur proxy si nécessaire pour l'accès à Internet.
* **Options de sécurité :**
  * Activez **Utiliser SSL** pour les connexions FTP sécurisées (**FTPS**).
  * Activez **Mode passif** si le client doit utiliser un port supérieur à **1024** lors de la connexion au port **21**.

Cliquez sur **Suivant** pour continuer.

***

### **3. Configurer les paramètres de récupération des dossiers et fichiers**

* **Inclure les sous-dossiers** si vous souhaitez récupérer des fichiers depuis des sous-répertoires.
* **Filtrage des fichiers :**
  * Utilisez **Restriction par extension de recherche** pour filtrer les fichiers par type (par ex., `.jpg`, `.pdf`).
* **Options de tri :**
  * Choisissez les préférences de tri pour l'**ordre de récupération des fichiers**.

Cliquez sur **Suivant** pour continuer.

***

### **4. Terminer l'assistant**

* Cliquez sur **Terminer** pour finaliser la connexion FTP.
* Le **PADS4 Viewer** récupérera désormais les **fichiers** depuis le dossier FTP en temps réel.

***

## **Considérations importantes**

* **Accès au PADS4 Data Server :**
  * Si vous utilisez un **PADS4 Data Server**, le dossier FTP **doit être accessible** depuis l'**emplacement du serveur**.
  * Si **vous n'utilisez pas de Data Server**, assurez-vous que chaque **PADS4 Designer et Viewer** dispose d'un **accès Internet** vers le **dossier FTP**.
* **Comportement du tri :**
  * Les **options de tri** génériques dans les **dernières étapes** de l'assistant **remplaceront** les paramètres de tri spécifiques au dossier.

En suivant ces étapes, vous pouvez **intégrer avec succès le contenu du dossier FTP** dans vos **présentations d'affichage dynamique PADS4**.
