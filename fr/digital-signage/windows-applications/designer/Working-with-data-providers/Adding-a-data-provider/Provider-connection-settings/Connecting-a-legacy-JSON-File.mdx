JSON (**JavaScript Object Notation**) est un format largement utilisé pour **stocker et échanger des données structurées**. Dans **PADS4**, les fichiers JSON peuvent être utilisés comme **sources de données** pour le contenu d'affichage dynamique en temps réel.

***

### **Étapes pour ajouter ou modifier une connexion à un fichier JSON**

#### **1. Démarrer le Data Provider Wizard**

* Ouvrir le **Data Provider Wizard** en suivant les instructions dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionner **JSON File** comme type de fournisseur.

***

#### **2. Spécifier l'emplacement du fichier JSON**

* **Content Library** → Sélectionner cette option pour utiliser un fichier JSON stocké dans la **PADS4 Content Library**.
* **Nom du fichier** → Choisissez cette option pour utiliser un fichier JSON situé **en dehors de la bibliothèque de contenu**.
* Utilisez la fonction de navigation pour localiser et sélectionner le fichier JSON souhaité.

**Recommandation**:\
Si **vous n'utilisez pas un PADS4 Data Server**, il est préférable de stocker les fichiers JSON dans la **PADS4 Content Library**. Cela garantit **une synchronisation automatique** sur tous les ordinateurs connectés. Si un fichier JSON est **en dehors de la bibliothèque**, la distribution des fichiers doit être gérée manuellement.

***

#### **3. Configurer l'encodage du texte**

* Si le fichier JSON **nécessite un format d'encodage de texte spécifique**, sélectionnez l'option appropriée.

***

#### **4. Configurer le formatage des données**

* **Aplatir les données hiérarchiques** → Si la structure JSON contient plusieurs niveaux, cette option **aplatit les données imbriquées** dans un format tabulaire.
* **Inclure les noms de colonnes des niveaux parents** → Ajoute les noms des niveaux parents pour une **représentation hiérarchique plus claire**.
* **Créer des colonnes supplémentaires sans balises HTML** → Supprime les balises HTML des champs de texte et **les convertit en texte brut**.

Passez à l'étape suivante.

***

#### **5. Finalisation de la configuration du fichier JSON**

* Complétez la configuration du fournisseur de données.

***

## **Considérations importantes**

* **Utilisation du PADS4 Data Server**
  * Si un **PADS4 Data Server** est utilisé, assurez-vous que le fichier JSON **est accessible** depuis le **point de vue du serveur**.
  * Si **vous n'utilisez pas de Data Server**, confirmez que **PADS4 Designer et les PADS4 Viewers** peuvent accéder à l'emplacement du fichier JSON spécifié.

* **Gestion des données JSON hiérarchiques**
  * Si la structure JSON contient plusieurs niveaux, l'activation de **Aplatir les données hiérarchiques** garantit une meilleure compatibilité.

En suivant ces étapes, **PADS4 peut intégrer et afficher de manière transparente des données au format JSON** dans des présentations d'affichage dynamique.
