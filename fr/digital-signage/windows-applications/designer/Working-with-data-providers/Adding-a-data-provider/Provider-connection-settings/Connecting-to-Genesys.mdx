Le **Genesys provider** dans **PADS4** vous permet de vous connecter à un **Genesys WSDL web service** pour récupérer des données pour **real-time digital signage integration**.

***

## **Étapes pour ajouter/modifier une connexion Genesys dans PADS4**

### **1. Démarrer l'Assistant de fournisseur de données**

* Ouvrez le **Data Provider Wizard** en suivant les étapes dans **Adding a Data Provider** ou **Editing a Data Provider**.
* Sélectionnez **Genesys** comme **provider type**.

### **2. Configurer la connexion Genesys**

* **Saisissez l'URL du serveur Genesys** (par exemple, `https://yourgenesysserver.com/wsdl`).
* **Authentification (si nécessaire) :**
  * Saisissez le **username** et le **password** si le service web Genesys nécessite des identifiants de connexion.
* **Paramètres du proxy (si nécessaire) :**
  * Configurez un **proxy server** si votre configuration réseau l'exige.

Cliquez sur **Next** pour continuer.

***

### **3. Configurer les paramètres du service Genesys**

* Vérifiez les **service settings** en fonction de votre **Genesys documentation**.
* Sélectionnez les **relevant data endpoints** pour récupérer les informations.

Cliquez sur **Next** pour continuer.

***

### **4. Terminer l'assistant**

* Cliquez sur **Finish** pour finaliser la connexion Genesys.
* Le **PADS4 Viewer** va maintenant récupérer des **real-time data** depuis le service web Genesys.

***

## **Considérations importantes**

* **Assurer une authentification correcte :**
  * Si des identifiants sont requis, assurez-vous que vous **have the correct access rights** pour l'API Genesys.
* **Accès au PADS4 Data Server :**
  * Si vous utilisez un **PADS4 Data Server**, le service Genesys **must be accessible** depuis **server's location**.
  * Si **not using a Data Server**, assurez-vous que chaque **PADS4 Designer and Viewer** a **internet access** au **Genesys web service**.
* **Suivre la documentation de l'API Genesys :**
  * Certains champs de données peuvent nécessiter des **custom query parameters** ou des **specific API calls**.
  * Consultez la **Genesys documentation** pour des instructions d'intégration détaillées.

En suivant ces étapes, vous pouvez **successfully integrate Genesys data** dans vos **PADS4 digital signage presentations**.
