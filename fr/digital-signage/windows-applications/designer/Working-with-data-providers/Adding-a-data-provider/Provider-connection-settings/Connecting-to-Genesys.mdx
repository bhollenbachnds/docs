Le **Genesys provider** dans **PADS4** vous permet de vous connecter à un **service web WSDL Genesys** pour récupérer des données pour **l'intégration d'affichage dynamique en temps réel**.

***

## **Étapes pour ajouter/modifier une connexion Genesys dans PADS4**

### **1. Démarrer l'Assistant de fournisseur de données**

* Ouvrez l'**Assistant de fournisseur de données** en suivant les étapes dans **Ajouter un fournisseur de données** ou **Modifier un fournisseur de données**.
* Sélectionnez **Genesys** comme **type de fournisseur**.

### **2. Configurer la connexion Genesys**

* **Saisissez l'URL du serveur Genesys** (par exemple, `https://yourgenesysserver.com/wsdl`).
* **Authentification (si nécessaire) :**
  * Saisissez le **nom d'utilisateur** et le **mot de passe** si le service web Genesys nécessite des identifiants de connexion.
* **Paramètres de proxy (si nécessaire) :**
  * Configurez un **serveur proxy** si votre configuration réseau le nécessite.

Cliquez sur **Next** pour continuer.

***

### **3. Configurer les paramètres du service Genesys**

* Vérifiez les **paramètres du service** en fonction de votre **documentation Genesys**.
* Sélectionnez les **points de terminaison de données pertinents** pour récupérer les informations.

Cliquez sur **Next** pour continuer.

***

### **4. Terminer l'assistant**

* Cliquez sur **Finish** pour finaliser la connexion Genesys.
* Le **PADS4 Viewer** va maintenant récupérer **des données en temps réel** depuis le service web Genesys.

***

## **Considérations importantes**

* **Assurer une authentification correcte :**
  * Si des identifiants sont requis, assurez-vous que vous **disposez des droits d'accès appropriés** pour l'API Genesys.
* **Accès au PADS4 Data Server :**
  * Si vous utilisez un **PADS4 Data Server**, le service Genesys **doit être accessible** depuis **l'emplacement du serveur**.
  * Si **vous n'utilisez pas de Data Server**, assurez-vous que chaque **PADS4 Designer et Viewer** a **accès à internet** pour le **service web Genesys**.
* **Suivre la documentation de l'API Genesys :**
  * Certains champs de données peuvent nécessiter **des paramètres de requête personnalisés** ou **des appels API spécifiques**.
  * Consultez la **documentation Genesys** pour des instructions d'intégration détaillées.

En suivant ces étapes, vous pouvez **intégrer avec succès les données Genesys** dans vos **présentations d'affichage dynamique PADS4**.
