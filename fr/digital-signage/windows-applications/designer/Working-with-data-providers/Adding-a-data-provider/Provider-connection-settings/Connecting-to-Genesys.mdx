Le **fournisseur Genesys** dans **PADS4** vous permet de vous connecter à un **service web Genesys WSDL** pour récupérer des données pour **l'intégration d'affichage dynamique en temps réel**.

***

## **Étapes pour ajouter/modifier une connexion Genesys dans PADS4**

### **1. Démarrer l'assistant de fournisseur de données**

* Ouvrir le **Data Provider Wizard** en suivant les étapes dans **Ajout d'un fournisseur de données** ou **Modification d'un fournisseur de données**.
* Sélectionner **Genesys** comme **type de fournisseur**.

### **2. Configurer la connexion Genesys**

* **Saisir l'URL du serveur Genesys** (par ex., `https://yourgenesysserver.com/wsdl`).
* **Authentification (si requise) :**
  * Saisir le **nom d'utilisateur** et le **mot de passe** si le service web Genesys requiert des identifiants de connexion.
* **Paramètres du proxy (si nécessaire) :**
  * Configurer un **serveur proxy** si votre configuration réseau l'exige.

Cliquer sur **Suivant** pour continuer.

***

### **3. Configurer les paramètres du service Genesys**

* Consulter les **paramètres du service** en fonction de votre **documentation Genesys**.
* Sélectionner les **points de terminaison de données pertinents** pour récupérer les informations.

Cliquer sur **Suivant** pour continuer.

***

### **4. Terminer l'assistant**

* Cliquer sur **Terminer** pour finaliser la connexion Genesys.
* Le **PADS4 Viewer** va maintenant récupérer les **données en temps réel** depuis le service web Genesys.

***

## **Considérations importantes**

* **Assurer une authentification correcte :**
  * Si des identifiants sont requis, assurez-vous de **disposer des droits d'accès corrects** pour l'API Genesys.
* **Accès au PADS4 Data Server :**
  * Si vous utilisez un **PADS4 Data Server**, le service Genesys **doit être accessible** depuis **l'emplacement du serveur**.
  * Si **vous n'utilisez pas de Data Server**, assurez-vous que chaque **PADS4 Designer et Viewer** dispose d'un **accès internet** au **service web Genesys**.
* **Suivre la documentation de l'API Genesys :**
  * Certains champs de données peuvent nécessiter des **paramètres de requête personnalisés** ou des **appels API spécifiques**.
  * Se référer à la **documentation Genesys** pour des instructions d'intégration détaillées.

En suivant ces étapes, vous pouvez **intégrer avec succès les données Genesys** dans vos **présentations d'affichage dynamique PADS4**.
