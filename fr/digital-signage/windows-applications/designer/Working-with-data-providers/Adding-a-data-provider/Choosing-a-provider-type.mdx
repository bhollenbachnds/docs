La **première étape** dans l'**Data Provider Wizard** consiste à sélectionner le type de fournisseur qui fournira **des données en temps réel** à votre présentation.

***

## **Étapes pour sélectionner un type de fournisseur**

1. Ouvrez le **Data Provider Wizard**
   * Suivez les instructions dans **Adding a Data Provider**
2. Choisissez un **Provider Type**
3. Cliquez sur **Next** pour procéder à la configuration
4. Complétez le **Data Provider Wizard**

***

## **Types de fournisseurs disponibles**

Chaque type de fournisseur se connecte à une **source de données** spécifique.

| **Provider Type**             | **Description**                                                               |
| ----------------------------- | ----------------------------------------------------------------------------- |

| **Google+ Feed**              | Récupère des données **Google+**.                                             |
| **Intel AIM**                 | Récupère **des données de reconnaissance faciale** depuis **Intel AIM**.      |

| **Microsoft Excel**           | Récupère des données depuis une **feuille de calcul Excel**.                  |

* Cliquez sur **Next** pour continuer le **Data Provider Wizard**

## **Additional Notes**

* Le **provider type** que vous sélectionnez détermine les **next configuration steps**
* **Certains types de fournisseurs nécessitent des pilotes supplémentaires** (par exemple, ODBC, OLEDB, OData)
