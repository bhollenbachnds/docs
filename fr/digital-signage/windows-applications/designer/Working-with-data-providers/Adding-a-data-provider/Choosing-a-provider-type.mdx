La **première étape** dans le **Data Provider Wizard** consiste à sélectionner le type de fournisseur qui alimentera **les données en temps réel** de votre présentation.

***

## **Étapes pour sélectionner un type de fournisseur**

1. Ouvrir le **Data Provider Wizard**
   * Suivez les instructions dans **Ajout d'un fournisseur de données**
2. Choisissez un **Type de fournisseur**
3. Cliquez sur **Suivant** pour continuer la configuration
4. Complétez le **Data Provider Wizard**

***

## **Étapes suivantes**

* Cliquez sur **Suivant** pour continuer l'**Assistant Fournisseur de données**

***

## **Notes supplémentaires**

* Le **type de fournisseur** que vous sélectionnez détermine les **étapes de configuration suivantes**
* **Certains types de fournisseurs nécessitent des pilotes supplémentaires** (par ex., ODBC, OLEDB, OData)
