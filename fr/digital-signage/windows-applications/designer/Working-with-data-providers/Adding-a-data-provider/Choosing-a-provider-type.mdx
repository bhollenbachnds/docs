La **première étape** dans l'**Assistant Fournisseur de Données** consiste à sélectionner le type de fournisseur qui fournira **des données en temps réel** à votre présentation.

***

## **Étapes pour sélectionner un type de fournisseur**

1. Ouvrez l'**Assistant Fournisseur de Données**
   * Suivez les instructions dans **Ajouter un Fournisseur de Données**
2. Choisissez un **Type de Fournisseur**
3. Cliquez sur **Suivant** pour procéder à la configuration
4. Complétez l'**Assistant Fournisseur de Données**

***

## **Types de fournisseurs disponibles**

Chaque type de fournisseur se connecte à une **source de données**.

| **Type de Fournisseur**       | **Description**                                                             |
| ----------------------------- | --------------------------------------------------------------------------- |
| **.NET Framework**            | Se connecte aux sources de données en utilisant **Microsoft .NET**.         |
| **Bluefox**                   | Récupère du contenu depuis **Bluefox**.                                     |
| **Cognimatics**               | Récupère **des données de mesure d'audience** depuis **Cognimatics**.       |
| **Facebook**                  | Se connecte aux **Facebook** flux de données.                               |
| **Flux Flickr**               | Récupère **des images** depuis **Flickr**.                                  |
| **Contenu de dossier**        | Récupère des fichiers depuis un **dossier local**.                          |
| **Dossier FTP**               | Se connecte à un **répertoire FTP** pour récupérer des fichiers.            |
| **ODBC Générique**            | Se connecte à une **source de données** en utilisant un **pilote ODBC**.    |
| **OLEDB Générique**           | Se connecte à une **source de données** en utilisant un **pilote OLEDB**.   |
| **Flux Google Calendar**      | Se connecte aux **Google Calendar** événements.                             |
| **Flux Google+**              | Récupère **Google+** données.                                               |
| **Intel AIM**                 | Récupère **des données de reconnaissance faciale** depuis **Intel AIM**.    |
| **Flux Instagram**            | Récupère **des images** depuis **Instagram**.                               |
| **Flux JSON**                 | Se connecte à un **flux JSON en ligne**.                                    |
| **Fichier JSON**              | Récupère **des données JSON** depuis un fichier local.                      |
| **Microsoft Access**          | Se connecte à une **base de données Microsoft Access**.                     |
| **Microsoft Excel**           | Récupère des données depuis une **feuille de calcul Excel**.                |
| **Microsoft Exchange Server** | Se connecte à **Microsoft Exchange**.                                       |
| **Microsoft SQL Server**      | Se connecte à **Bases de données, vues ou procédures stockées SQL Server**. |
| **Microsoft SQL Server CE**   | Se connecte à **des bases de données SQL Server Compact Edition**.          |
| **Media RSS Feed**            | Récupère des données depuis une source **Media RSS**.                       |
| **Media RSS File**            | Récupère du contenu **Media RSS** à partir d'un fichier local.              |
| **Newmarket Delphi Feed**     | Se connecte à **Newmarket Delphi**.                                         |
| **OData**                     | Se connecte à une source de données **Open Data Protocol**.                 |
| **Office 365 Exchange**       | Se connecte à **Microsoft Exchange Online**.                                |
| **Office 365 Excel**          | Se connecte à un **classeur Excel stocké dans Office 365**.                 |
| **Open Office**               | Se connecte à un **fichier Open Office Calc**.                              |
| **PADS4 Viewer Connections**  | Récupère le **statut de tous les PADS4 Viewers**.                           |
| **PADS4FIDS**                 | Se connecte au **PADS4 Flight Information Display System (FIDS)**.          |
| **Picasa Feed**               | Récupère des **images** depuis **Picasa**.                                  |
| **PostgreSQL**                | Se connecte à une **base de données PostgreSQL**.                           |
| **Quividi Feed**              | Récupère des **données de mesure d'audience** depuis **Quividi**.           |
| **Seenspire**                 | Récupère du contenu depuis **Seenspire**.                                   |
| **Static Table**              | Utilise un **tableau intégré** dans votre présentation.                     |
| **SQL Server Database**       | Se connecte à une **base de données SQL Server**.                           |
| **SQL Server CE Database**    | Se connecte à une **base de données SQL Server Compact Edition**.           |
| **Text Feed**                 | Récupère des données depuis un **flux web basé sur du texte**.              |
| **Text File**                 | Récupère des **données texte brut** depuis un fichier local.                |
| **Twitter Feed**              | Se connecte à **Twitter** pour des publications en temps réel.              |
| **RSS Feed**                  | Récupère du **contenu RSS** depuis le web.                                  |
| **RSS File**                  | Récupère des **données RSS** depuis un fichier local.                       |
| **SOAP Web Service**          | Se connecte à un **service web basé sur SOAP**.                             |
| **WMI Class**                 | Récupère des **données Windows Management Instrumentation (WMI)**.          |
| **XML Feed**                  | Récupère des **données XML** depuis internet.                               |
| **XML File**                  | Récupère des **données XML** depuis un fichier local.                       |
| **YouTube Feed**              | Se connecte à **YouTube** pour la récupération de vidéos.                   |

## **Étapes suivantes**

* Cliquez sur **Suivant** pour continuer l'**Assistant de fournisseur de données**
* Suivez les étapes dans **Configuration des paramètres de connexion**

***

## **Notes supplémentaires**

* Le **type de fournisseur** que vous sélectionnez détermine les **prochaines étapes de configuration**
* **Certains types de fournisseurs nécessitent des pilotes supplémentaires** (par exemple, ODBC, OLEDB, OData)
