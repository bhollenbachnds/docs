La **première étape** dans l'**Assistant de fournisseur de données** consiste à sélectionner le type de fournisseur qui fournira des **données en temps réel** à votre présentation.

***

## **Étapes pour sélectionner un type de fournisseur**

1. Ouvrez l'**Assistant de fournisseur de données**
   * Suivez les instructions dans **Ajout d'un fournisseur de données**
2. Choisissez un **Type de fournisseur**
3. Cliquez sur **Suivant** pour procéder à la configuration
4. Complétez l'**Assistant de fournisseur de données**

***

## **Types de fournisseurs disponibles**

Chaque type de fournisseur se connecte à une **source de données** spécifique.

| **Type de fournisseur**           | **Description**                                                               |
| --------------------------------- | ----------------------------------------------------------------------------- |
| **.NET Framework**                | Se connecte aux sources de données en utilisant **Microsoft .NET**.           |
| **Bluefox**                       | Récupère le contenu de **Bluefox**.                                           |
| **Cognimatics**                   | Récupère les **données de mesure d'audience** de **Cognimatics**.             |
| **Facebook**                      | Se connecte aux flux de données **Facebook**.                                 |
| **Flickr Feed**                   | Récupère des **images** de **Flickr**.                                        |
| **Folder Content**                | Récupère des fichiers d'un **dossier local**.                                 |
| **FTP Folder**                    | Se connecte à un **répertoire FTP** pour récupérer des fichiers.              |
| **Generic ODBC**                  | Se connecte à une **source de données** en utilisant un **pilote ODBC**.      |
| **Generic OLEDB**                 | Se connecte à une **source de données** en utilisant un **pilote OLEDB**.     |
| **Google Calendar Feed**          | Se connecte aux événements **Google Calendar**.                               |
| **Google+ Feed**                  | Récupère les données **Google+**.                                             |
| **Intel AIM**                     | Récupère les **données de reconnaissance faciale** d'**Intel AIM**.           |
| **Instagram Feed**                | Récupère des **images** d'**Instagram**.                                      |
| **JSON Feed**                     | Se connecte à un **flux JSON en ligne**.                                      |
| **JSON File**                     | Récupère des **données JSON** à partir d'un fichier local.                    |
| **Microsoft Access**              | Se connecte à une **base de données Microsoft Access**.                       |
| **Microsoft Excel**               | Récupère des données d'une **feuille de calcul Excel**.                       |
| **Microsoft Exchange Server**     | Se connecte à **Microsoft Exchange**.                                         |
| **Microsoft SQL Server**          | Se connecte aux **bases de données, vues ou procédures stockées SQL Server**. |
| **Microsoft SQL Server CE**       | Se connecte aux **bases de données SQL Server Compact Edition**.              |
| **Flux RSS multimédia**           | Récupère des données à partir d'une **source Media RSS**.                     |
| **Fichier Media RSS**             | Récupère **Media RSS** contenu à partir d'un fichier local.                   |
| **Flux Newmarket Delphi**         | Se connecte à **Newmarket Delphi**.                                           |
| **OData**                         | Se connecte à une source de données **Open Data Protocol**.                   |
| **Office 365 Exchange**           | Se connecte à **Microsoft Exchange Online**.                                  |
| **Office 365 Excel**              | Se connecte à un **Excel spreadsheet stored in Office 365**.                  |
| **Open Office**                   | Se connecte à un **Open Office Calc file**.                                   |
| **PADS4 Viewer Connections**      | Récupère le **status of all PADS4 Viewers**.                                  |
| **PADS4FIDS**                     | Se connecte au **PADS4 Flight Information Display System (FIDS)**.            |
| **Flux Picasa**                   | Récupère des **images** depuis **Picasa**.                                    |
| **PostgreSQL**                    | Se connecte à une **base de données PostgreSQL**.                             |
| **Flux Quividi**                  | Récupère des **données de mesure d'audience** depuis **Quividi**.             |
| **Seenspire**                     | Récupère du contenu depuis **Seenspire**.                                     |
| **Tableau statique**              | Utilise un **tableau intégré** dans votre présentation.                       |
| **Base de données SQL Server**    | Se connecte à une **base de données SQL Server**.                             |
| **Base de données SQL Server CE** | Se connecte à une **base de données SQL Server Compact Edition**.             |
| **Flux texte**                    | Récupère des données à partir d'un **flux web basé sur du texte**.            |
| **Fichier texte**                 | Récupère des **données texte brut** à partir d'un fichier local.              |
| **Flux Twitter**                  | Se connecte à **Twitter** pour des publications en temps réel.                |
| **Flux RSS**                      | Récupère du **contenu RSS** depuis le web.                                    |
| **Fichier RSS**                   | Récupère des **données RSS** à partir d'un fichier local.                     |
| **Service Web SOAP**              | Se connecte à un **service web basé sur SOAP**.                               |
| **Classe WMI**                    | Récupère des **données Windows Management Instrumentation (WMI)**.            |
| **Flux XML**                      | Récupère des **données XML** depuis internet.                                 |
| **Fichier XML**                   | Récupère des **données XML** à partir d'un fichier local.                     |
| **Flux YouTube**                  | Se connecte à **YouTube** pour la récupération de vidéos.                     |

## **Notes supplémentaires**

* Le **type de fournisseur** que vous sélectionnez détermine les **prochaines étapes de configuration**
* **Certains types de fournisseurs nécessitent des pilotes supplémentaires** (par exemple, ODBC, OLEDB, OData)
