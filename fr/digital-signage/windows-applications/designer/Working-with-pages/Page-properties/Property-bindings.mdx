Les liaisons de propriétés vous permettent de **lier dynamiquement des propriétés de page** à des sources de données externes. Lorsque la source de données est mise à jour, la propriété de page correspondante sera automatiquement mise à jour lors de la lecture.

Par exemple, si vous **liez la couleur d'arrière-plan** d'une page à un fichier XML, **la couleur d'arrière-plan changera automatiquement** lorsque la valeur dans le fichier XML change.

***

## **Aperçu des liaisons de propriétés**

Cette section contient une **liste des propriétés de page** pouvant être liées à une source de données externe.\
Pour créer une liaison, vous devez définir au minimum :

* **Fournisseur de données**
* **Colonne (Champ)**
* **Ligne (Enregistrement)**

***

### **Propriété**

* La **propriété de page spécifique** à lier (par ex., **Couleur d'arrière-plan**, **Visible**, etc.).
* **Défini par le système** — ne peut pas être modifié.

### **Type**

* Le **type de données** de la propriété (par ex., **Couleur**, **Booléen**, **Entier**).
* **Défini par le système** — ne peut pas être modifié.

### **Fournisseur de données**

* Choisissez le **fournisseur de données** qui fournit les données (par ex., XML, Excel, SQL, JSON).
* Si aucun fournisseur de données n'est disponible, **créez-en un d'abord** dans la configuration du fournisseur de données.

### **Colonne**

* Sélectionnez le **champ de données** (colonne) dans le fournisseur de données qui contient la valeur de la propriété.

### **Ligne**

* Sélectionnez l'**enregistrement (ligne)** dans le fournisseur de données qui contient la valeur souhaitée.

### **Défilement**

* **Activez cette option** si le défilement des données doit être appliqué.
* Lorsqu'elle est activée, **les propriétés de défilement des données** détermineront la façon dont les données défilent pendant la lecture.

***

## **Exemple d'utilisation : Visibilité dynamique des pages**

* Liez la **propriété Visible** d'une page à une **cellule dans un fichier Excel**.
* Si la valeur Excel est « 1 », la page sera affichée.
* Si la valeur Excel est « 0 », la page sera masquée.
* Cela permet un **contrôle dynamique** sur les pages affichées, simplement en modifiant le fichier Excel — utile pour le contenu basé sur des planifications.

***

> **Conseil** :\
> Les liaisons de propriétés sont puissantes pour le **contenu adaptatif en temps réel**. Par exemple :

* Modifier la couleur d'arrière-plan en fonction des données de capteurs en temps réel.
* Afficher ou masquer des pages en fonction de fichiers de planification externes.
* Ajuster dynamiquement les effets de transition.
