Les **Data Scrolling Properties** définissent comment les **real-time data** défilent dans une page pendant la lecture.

***

## **General Data Scrolling**

### **Interval**

* Spécifie l'**time interval (in seconds)** utilisé pour faire défiler les données pour tous les éléments de page qui ont **Data Scrolling** activé.
* Cet intervalle contrôle **how fast** les nouvelles lignes de données sont affichées dans des éléments comme les tableaux, les listes ou les tickers.

***

### **Hold Scroll Positions after Page Change**

* **When enabled**, la **current row position** de tous les fournisseurs de données sur la page sera **remembered** pendant la lecture lors du passage à une autre page.
* En revenant à cette page, le défilement **continue from where it left off**, au lieu de recommencer depuis la première ligne.
* Par exemple :
  * Si la page affichait la **row 4** lorsqu'elle a changé, en y revenant, elle continuera avec la **row 5**.

> **Astuce**: C'est **particulièrement utile pour les listes de lecture de publicités ou les longues listes de données**, garantissant que le contenu est diffusé **séquentiellement** à travers les boucles, plutôt que de recommencer depuis le début chaque fois que la page apparaît.

***

### **Exemple de cas d'utilisation**

* Vous affichez une liste rotative de publicités, et vous voulez vous assurer que **chaque publicité est affichée** sans redémarrer la liste lorsque vous revenez à cette page.
* En conservant la position de défilement, **la publicité suivante dans la séquence** sera diffusée lorsque la page sera affichée à nouveau.
