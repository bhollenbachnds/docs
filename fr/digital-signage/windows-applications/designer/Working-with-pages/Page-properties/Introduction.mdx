Chaque page de votre présentation possède un certain nombre de propriétés qui déterminent son **apparence, comportement et interaction** pendant la lecture.

***

## **Comment Modifier les Propriétés de Page**

1. Cliquez sur **Page Properties** dans le menu **Page**.
2. Dans la **fenêtre Page Properties**, ajustez un ou plusieurs paramètres selon vos besoins.
3. Cliquez sur **OK** pour enregistrer et revenir à votre présentation, ou cliquez sur **Apply** pour enregistrer les modifications sans fermer la fenêtre.

***

## **Aperçu des Propriétés de Page**

| **Section de Propriété** | **Description**                                                                                              |
| ------------------------ | ------------------------------------------------------------------------------------------------------------ |
| **General**              | Propriétés pour **identifier votre page** et définir **combien de temps elle est affichée**.                 |
| **Style**                | Propriétés pour déterminer **comment la page sera lue**.                                                     |
| **Data Scrolling**       | Propriétés pour définir **le comportement de défilement pour les données en temps réel**.                    |
| **Visibility**           | Propriétés pour contrôler **si la page est affichée pendant la lecture**.                                    |
| **Transition Effect**    | Propriétés pour définir l'**effet de transition** lorsque la page devient visible.                           |
| **Property Bindings**    | Définir les **property bindings** pour connecter les propriétés de page aux **sources de données externes**. |
| **Rules**                | Définir les **règles spécifiques à la page** pour que la page réponde aux **conditions prédéfinies**.        |

***

## **Remarques**

* Les propriétés de page permettent une **personnalisation avancée** des pages individuelles, permettant des présentations dynamiques et réactives.
* Certaines propriétés, comme les **effets de transition et le défilement des données**, améliorent l'expérience utilisateur et la visualisation des données.
* **Les règles et les property bindings** offrent une automatisation et une adaptabilité en temps réel.
