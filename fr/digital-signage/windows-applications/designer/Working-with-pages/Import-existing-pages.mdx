Une façon rapide et efficace de composer une nouvelle présentation est en **important des pages** à partir d'autres présentations existantes. Cela aide à **recycler du contenu précédemment conçu**, économisant du temps et des efforts.

***

## **Étapes pour importer une ou plusieurs pages d'une autre présentation**

### 1. Démarrer l'assistant d'importation de page

* Allez dans le menu **Page**.
* Cliquez sur **Import Page** pour ouvrir l'assistant d'importation de page.

***

### 2. Sélectionner la présentation à partir de laquelle importer

* **Choisissez** si vous souhaitez importer à partir de :
  * **Content Library** — pour sélectionner une présentation stockée dans la PADS4 Content Library.
  * **File Name** — pour importer une présentation à partir d'un emplacement de fichier spécifique.
* **Spécifiez le nom du fichier** (y compris le chemin complet) de la présentation PADS4 (fichier .padsx).
* Vous pouvez utiliser le bouton **browse** pour localiser le fichier.
* Cliquez sur **Next** pour continuer.

***

### 3. Sélectionner les pages à importer

* Cliquez sur **la/les vignette(s)** de la/des page(s) que vous souhaitez importer.
* **Important** : Pour sélectionner plusieurs pages, maintenez la touche **Ctrl** ou **Shift** enfoncée tout en cliquant sur les vignettes des pages.
* Cliquez sur **Next** pour continuer.

***

### 4. Finaliser l'importation

* Cliquez sur **Finish** pour importer les pages sélectionnées et quitter l'assistant.

***

## **Remarques**

* Si un **conflit de nom de page** se produit (c'est-à-dire si une page importée a le même nom qu'une page existante), PADS4 **générera automatiquement un nouveau nom** pour la page importée.
* **Les pages importées** seront ajoutées **directement après la page actuellement active** dans votre présentation.
* Si les pages importées **utilisent des fournisseurs de données**, ces fournisseurs de données seront également **importés automatiquement** dans votre présentation.
