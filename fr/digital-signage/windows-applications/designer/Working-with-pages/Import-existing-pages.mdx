Une façon rapide et efficace de composer une nouvelle présentation est en **important des pages** d'autres présentations existantes. Cela aide à **recycler du contenu précédemment conçu**, économisant du temps et des efforts.

***

## **Étapes pour importer une ou plusieurs pages d'une autre présentation**

### 1. Démarrer l'Assistant d'importation de page

* Allez dans le menu **Page**.
* Cliquez sur **Importer une page** pour ouvrir l'Assistant d'importation de page.

***

### 2. Sélectionner la présentation à partir de laquelle importer

* **Choisissez** si vous souhaitez importer depuis :
  * **Bibliothèque de contenu** — pour sélectionner une présentation stockée dans la Bibliothèque de contenu PADS4.
  * **Nom de fichier** — pour importer une présentation depuis un emplacement de fichier spécifique.
* **Spécifiez le nom du fichier** (incluant le chemin complet) de la présentation PADS4 (fichier .padsx).
* Vous pouvez utiliser le bouton **parcourir** pour localiser le fichier.
* Cliquez sur **Suivant** pour continuer.

***

### 3. Sélectionner les pages à importer

* Cliquez sur la/les **miniature(s)** de la/des page(s) que vous souhaitez importer.
* **Important** : Pour sélectionner plusieurs pages, maintenez la touche **Ctrl** ou **Shift** enfoncée tout en cliquant sur les miniatures des pages.
* Cliquez sur **Suivant** pour continuer.

***

### 4. Finaliser l'importation

* Cliquez sur **Terminer** pour importer les pages sélectionnées et quitter l'assistant.

***

## **Remarques**

* Si un **conflit de nom de page** se produit (c'est-à-dire si une page importée a le même nom qu'une page existante), PADS4 **générera automatiquement un nouveau nom** pour la page importée.
* **Les pages importées** seront ajoutées **directement après la page actuellement active** dans votre présentation.
* Si les pages importées **utilisent des fournisseurs de données**, ces fournisseurs de données seront également **importés automatiquement** dans votre présentation.
