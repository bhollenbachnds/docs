Au lieu de créer une page maître à partir de zéro, vous pouvez **importer une page existante** d'une autre présentation et la désigner comme votre page maître. Cela permet de gagner du temps et assure la cohérence de conception si vous disposez déjà d'une mise en page préconstruite.

***

## Comment importer une page maître à partir d'une autre présentation

1. Allez dans le **menu Page**.
2. Sélectionnez **Import Master Page** pour démarrer l'**Import Master Page Wizard**.

### Étape 1 : Sélectionner la présentation à importer

* Choisissez **Content Library** pour importer une présentation stockée dans la PADS4 Content Library.
* Choisissez **File Name** pour importer une présentation depuis un autre emplacement sur votre système.
* Spécifiez le chemin complet vers le fichier, ou utilisez l'option **browse** pour le localiser.
* Cliquez sur **Next** pour continuer.

### Étape 2 : Sélectionner la page à importer comme page maître

* Cliquez sur la vignette de la page que vous souhaitez utiliser comme page maître.
* Vous pouvez sélectionner **n'importe quelle page** de la présentation choisie.
* Si la page sélectionnée **n'est pas déjà une page maître**, elle sera **automatiquement convertie** en page maître lors de l'importation.
* Cliquez sur **Next** pour continuer.

***

## Remarques importantes

* **Écrasement d'une page maître existante**:\
  Si votre présentation actuelle **contient déjà une page maître**, **elle sera remplacée** par la page maître importée.

* **Data Providers**:\
  Si la page importée **utilise des data providers**, ceux-ci seront également **importés dans votre présentation** pour garantir que tout le contenu et les données liés fonctionnent correctement.
