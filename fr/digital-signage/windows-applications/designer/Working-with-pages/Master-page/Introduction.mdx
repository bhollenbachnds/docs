Une **page maître** est utilisée pour créer une mise en page cohérente dans toute votre présentation. En ajoutant une page maître, vous pouvez définir une mise en page qui sera affichée comme arrière-plan pour toutes les autres pages où la page maître est activée.

Lors de l'utilisation d'une page maître, **elle restera active et visible pendant la lecture**, même lors du passage entre les pages normales. Cela signifie que si votre page maître contient du contenu dynamique comme des vidéos ou du texte défilant, ceux-ci **continueront à fonctionner** lorsque vous passez d'une page à une autre.

Si vous avez besoin que certaines pages aient une mise en page différente, vous pouvez désactiver la page maître pour ces pages spécifiques.

## Remarques importantes

* Lors de l'utilisation d'une page maître, **ses éléments restent actifs et fonctionnent pendant la lecture**. Par exemple, si la page maître contient une vidéo, celle-ci continuera à être lue même lors du passage à d'autres pages.
* Les pages maîtres sont idéales pour des éléments tels que :
  * Images ou couleurs d'arrière-plan
  * Logos d'entreprise
  * Messages défilants ou bandeaux d'actualités
  * Horloges ou affichages dynamiques de l'heure
  * Boucles vidéo utilisées comme arrière-plan décoratif
* Si vous **ne souhaitez pas que la page maître soit affichée sur une page spécifique**, vous pouvez désactiver l'option "Afficher la page maître en arrière-plan de cette page" dans les **Propriétés de Style** de cette page.
