PADS4 Designer offre plusieurs façons de **naviguer entre les pages** de votre présentation, ce qui facilite la révision et l'ajustement efficace du contenu.

***

## **Comment naviguer vers une page spécifique**

### Option 1 : Utiliser les boutons de navigation

* Allez dans le menu **View**.
* Dans la section **Go To**, utilisez les boutons suivants :

| Bouton                 | Fonction                                                                                                                           |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| Go to Master Page      | Navigue vers la **master page** de votre présentation. *(Remarque : Vous devez d'abord créer une master page si aucune n'existe.)* |
| Go to First Page       | Ouvre la **first page** de votre présentation.                                                                                     |
| Go to Previous Page    | Passe à la **previous page**.                                                                                                      |
| Go to Next Page        | Passe à la **next page**.                                                                                                          |
| Go to Last Page        | Ouvre la **last page** de votre présentation.                                                                                      |
| Page Selector Dropdown | Sélectionnez et naviguez vers une **specific page** à partir de la liste déroulante.                                               |

***

### Option 2 : Utiliser la fenêtre des miniatures

* Cliquez directement sur la **thumbnail** de la page que vous souhaitez ouvrir.

***

### Option 3 : Utiliser les raccourcis clavier

| Touche clavier | Action                        |
| -------------- | ----------------------------- |
| **Home**       | Aller à la **first page**.    |
| **Page Up**    | Aller à la **previous page**. |
| **Page Down**  | Aller à la **next page**.     |
| **End**        | Aller à la **last page**.     |

***

## **Astuce**

L'utilisation des **thumbnails** ou des **keyboard shortcuts** est un moyen rapide de naviguer tout en travaillant sur votre design — idéal pour réviser des présentations longues.
