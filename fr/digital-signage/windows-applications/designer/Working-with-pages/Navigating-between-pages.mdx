PADS4 Designer offre plusieurs façons de **naviguer entre les pages** de votre présentation, facilitant la révision et l'ajustement du contenu de manière efficace.

***

## **Comment naviguer vers une page spécifique**

### Option 1 : Utiliser les boutons de navigation

* Allez dans le menu **View**.
* Dans la section **Go To**, utilisez les boutons suivants :

| Bouton                 | Fonction                                                                                                                           |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| Go to Master Page      | Navigue vers la **master page** de votre présentation. *(Remarque : Vous devez d'abord créer une master page si aucune n'existe.)* |
| Go to First Page       | Ouvre la **première page** de votre présentation.                                                                                  |
| Go to Previous Page    | Passe à la **page précédente**.                                                                                                    |
| Go to Next Page        | Passe à la **page suivante**.                                                                                                      |
| Go to Last Page        | Ouvre la **dernière page** de votre présentation.                                                                                  |
| Page Selector Dropdown | Sélectionnez et naviguez vers une **page spécifique** à partir de la liste déroulante.                                             |

***

### Option 2 : Utiliser la fenêtre des miniatures

* Cliquez directement sur la **miniature** de la page que vous souhaitez ouvrir.

***

### Option 3 : Utiliser les raccourcis clavier

| Touche clavier | Action                          |
| -------------- | ------------------------------- |
| **Home**       | Aller à la **première page**.   |
| **Page Up**    | Aller à la **page précédente**. |
| **Page Down**  | Aller à la **page suivante**.   |
| **End**        | Aller à la **dernière page**.   |

***

## **Conseil**

L'utilisation des **miniatures** ou des **raccourcis clavier** est un moyen rapide de naviguer tout en travaillant sur votre conception — idéal pour réviser des présentations longues.
