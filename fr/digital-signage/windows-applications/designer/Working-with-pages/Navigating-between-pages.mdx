PADS4 Designer offre plusieurs façons de **naviguer entre les pages** de votre presentation, facilitant la révision et l'ajustement du contenu de manière efficace.

***

## **Comment naviguer vers une page spécifique**

### Option 1 : Utiliser les boutons de navigation

* Accédez au menu **View**.
* Dans la section **Go To**, utilisez les boutons suivants :

| Bouton                 | Fonction                                                                                                                           |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| Go to Master Page      | Navigue vers la **master page** de votre presentation. *(Remarque : Vous devez d'abord créer une master page si aucune n'existe.)* |
| Go to First Page       | Ouvre la **première page** de votre presentation.                                                                                  |
| Go to Previous Page    | Passe à la **page précédente**.                                                                                                    |
| Go to Next Page        | Passe à la **page suivante**.                                                                                                      |
| Go to Last Page        | Ouvre la **dernière page** de votre presentation.                                                                                  |
| Page Selector Dropdown | Sélectionnez et naviguez vers une **page spécifique** depuis la liste déroulante.                                                  |

***

### Option 2 : Utiliser la fenêtre Thumbnails

* Cliquez directement sur la **miniature** de la page que vous souhaitez ouvrir.

***

### Option 3 : Utiliser les raccourcis clavier

| Touche du clavier | Action                          |
| ----------------- | ------------------------------- |
| **Home**          | Aller à la **première page**.   |
| **Page Up**       | Aller à la **page précédente**. |
| **Page Down**     | Aller à la **page suivante**.   |
| **End**           | Aller à la **dernière page**.   |

***

## **Conseil**

Utiliser les **miniatures** ou **les raccourcis clavier** est un moyen rapide de naviguer tout en travaillant sur votre design — idéal pour réviser de longues presentations.
