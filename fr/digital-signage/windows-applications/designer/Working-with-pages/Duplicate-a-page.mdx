La duplication d'une page est utile lorsque vous souhaitez **réutiliser la mise en page ou le contenu** d'une page existante comme point de départ pour une nouvelle. Cela peut **faire gagner du temps** et garantir la cohérence.

***

## **Pour créer une copie exacte de la page actuelle**

### Option 1 : Utiliser le menu Ruban

1. Accédez au menu **Page**.
2. Cliquez sur **Duplicate Page**.

### Option 2 : Utiliser la fenêtre Thumbnails

1. **Faites un clic droit** sur la miniature de la page que vous souhaitez dupliquer dans la **fenêtre Thumbnails**.
2. Sélectionnez **Duplicate Page** dans le menu contextuel.

***

## **Remarques**

* **Pages dupliquées** recevra automatiquement un nom basé sur le **nom de page original**, avec un suffixe ajouté (par ex., "Page1 (Copy)").
* La **page copiée** est toujours insérée **directement après la page originale** dans la présentation.
