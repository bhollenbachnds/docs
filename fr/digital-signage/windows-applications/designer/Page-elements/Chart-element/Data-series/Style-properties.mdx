Les **Style properties** dans le **Data Series Wizard** déterminent l'**appearance and visual formatting** de votre série de données, y compris les **lines, markers, and custom images**.

***

## **Options de marqueur**

Selon le **Chart Type** sélectionné, vous pouvez configurer divers paramètres de style liés aux marqueurs :

### **Line Thickness (pour Line Charts uniquement)**

| **Propriété**      | **Description**                                                                     |
| ------------------ | ----------------------------------------------------------------------------------- |
| **Line Thickness** | Définit la **width of the line** (en pixels) utilisée pour représenter les données. |

***

## **Paramètres de marqueur**

### **Predefined Marker**

| **Propriété**             | **Description**                                                                                                                        |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| **Use Predefined Marker** | Activez cette option pour utiliser des marqueurs standard configurés ci-dessous.                                                       |
| **Marker**                | Choisissez une **marker shape** (par exemple, carré, cercle, triangle, diamant) parmi l'ensemble disponible.                           |
| **Marker Size**           | Définissez la **size of the markers** (en pixels).                                                                                     |
| **Border Width**          | Définissez l'**border thickness** (en pixels) autour des marqueurs.                                                                    |
| **Marker Color**          | Choisissez une couleur pour les marqueurs. Vous pouvez utiliser des couleurs unies, des dégradés, des motifs ou des textures.          |
| **Marker Border Color**   | Choisissez une couleur pour la bordure du marqueur. Les options incluent des couleurs unies, des dégradés, des motifs ou des textures. |

***

### **Image as Marker**

| **Propriété**           | **Description**                                                                                                                                                |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Use Image as Marker** | Activez cette option pour **use an image as a marker** au lieu d'une forme.                                                                                    |
| **Image Source**        | Spécifiez le fichier image. Vous pouvez **retrieve images from the PADS4 Content Library** ou spécifier un **local disk file** (fournissez le chemin complet). |

***

## **Remarques**

* **Markers** aident à **highlight individual data points** dans la série.
* **Only references** aux fichiers image sont stockées dans la présentation — **not the actual image content**.\
  ➤ **Si l'image source est mise à jour**, la version mise à jour sera utilisée automatiquement pendant la lecture.
* Le **sélecteur de couleurs** vous permet de choisir parmi **des couleurs fixes, des dégradés, des textures ou des motifs** pour les marqueurs et leurs bordures.
* Si vous utilisez **image markers**, assurez-vous que le fichier image source reste disponible et synchronisé pour une lecture cohérente de la présentation.
