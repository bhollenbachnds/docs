Les **propriétés de couleur** dans l'**Assistant de séries de données** définissent comment les données de votre série seront visuellement colorées dans le graphique. Vous pouvez choisir des **couleurs automatiques**, des **couleurs de série personnalisées**, ou même des **couleurs individuelles par point de données**.

***

## **Couleurs**

| **Option**                                     | **Description**                                                                                                                                                                                                                    |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Couleur prédéfinie par série**               | Le système **attribue automatiquement une couleur** à votre série. Lorsque plusieurs séries de données sont présentes, chacune recevra une **couleur différente automatiquement** pour assurer la distinction.                     |
| **Couleur personnalisée par série**            | **Définissez manuellement** la couleur pour toute votre série. Vous devez **attribuer différentes couleurs pour chaque série** vous-même pour maintenir la clarté visuelle.                                                        |
| **Couleur personnalisée par point de données** | Attribuez des **couleurs individuelles à chaque point de données** dans la série. Cela permet un contrôle des couleurs plus détaillé, utile pour mettre en évidence des éléments de données spécifiques ou pour le codage couleur. |

***

## **Options de sélection de couleur**

* Cliquez sur le **bouton déroulant** pour ouvrir le **sélecteur de couleur** et choisissez votre couleur désirée.
* Vous pouvez sélectionner:
  * **Couleurs solides (fixes)**.
  * **Dégradés** (deux ou trois couleurs).
  * **Textures**.
  * **Motifs**.

> **CONSEIL**: Voir la section **"Utilisation du sélecteur de couleur"** pour une explication complète de toutes les options de sélection de couleur disponibles.

***

## **Remarques**

* **Couleur prédéfinie par série** est le moyen le plus rapide de différencier plusieurs séries sans effort manuel.
* **Couleur personnalisée par série** donne un **contrôle total** sur l'apparence visuelle mais nécessite une planification minutieuse dans les graphiques multi-séries pour éviter des couleurs similaires.
* **Couleur personnalisée par point de données** est idéale pour mettre en évidence des **valeurs spécifiques** ou créer des **effets de type carte thermique** au sein d'une même série.
* Choisir soigneusement les couleurs aide à **clarifier les tendances des données**, **mettre en évidence les points importants**, et maintenir une **cohérence esthétique** dans vos présentations.
