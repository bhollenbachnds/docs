Les **Color properties** dans le **Data Series Wizard** définissent comment les données de votre série seront visuellement colorées dans le graphique. Vous pouvez choisir **automatic colors**, **custom series colors**, ou même **individual colors per data point**.

***

## **Colors**

| **Option**                     | **Description**                                                                                                                                                                                                    |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Predefined Color by Series** | Le système **automatically assigns a color** à votre série. Lorsque plusieurs séries de données sont présentes, chacune recevra une **different color automatically** pour assurer la distinction.                 |
| **Custom Color by Series**     | **Définir manuellement** la couleur pour toute votre série. Vous devez **assign different colors for each series** vous-même pour maintenir la clarté visuelle.                                                    |
| **Custom Color by Datapoint**  | Attribuer **individual colors to each data point** dans la série. Cela permet un contrôle des couleurs plus détaillé, utile pour mettre en évidence des éléments de données spécifiques ou pour le codage couleur. |

***

## **Color Selection Options**

* Cliquez sur le **drop-down button** pour ouvrir le **color picker** et choisir la couleur souhaitée.
* Vous pouvez sélectionner :
  * **Solid (fixed) colors**.
  * **Gradients** (deux ou trois couleurs).
  * **Textures**.
  * **Patterns**.

> **TIP** : Voir la section **"Using the color picker"** pour une explication complète de toutes les options de sélection de couleurs disponibles.

***

## **Remarks**

* **Predefined Color by Series** est le moyen le plus rapide de différencier plusieurs séries sans effort manuel.
* **Custom Color by Series** donne **full control** sur l'apparence visuelle mais nécessite une planification minutieuse dans les graphiques multi-séries pour éviter des couleurs similaires.
* **Custom Color by Datapoint** est idéal pour mettre en évidence **specific values** ou créer **heatmap-like effects** au sein d'une même série.
* Choisir soigneusement les couleurs aide à **clarify data trends**, **highlight important points**, et maintenir **aesthetic consistency** dans vos présentations.
