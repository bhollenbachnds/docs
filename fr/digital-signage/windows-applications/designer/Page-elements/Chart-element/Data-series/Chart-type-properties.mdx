Les **propriétés du type de graphique** dans le **Data Series Wizard** définissent **comment votre série de données sera représentée visuellement** dans le graphique. Ces propriétés vous permettent de sélectionner le **type de graphique**, **nommer la série**, contrôler sa **visibilité**, et **l'affecter à un axe secondaire** si nécessaire.

***

## **Aperçu des propriétés**

| **Propriété**                             | **Description**                                                                                                                         |
| ----------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| **Type de graphique**                     | Sélectionnez le **type de graphique** pour afficher votre série de données. Les exemples incluent **barre, ligne, secteur, aire**, etc. |
|                                           | Lorsque vous choisissez un type, une **fenêtre d'aperçu** affiche un exemple visuel du graphique pour vous aider à décider.             |
| **Série visible**                         | Activez ou désactivez cette option **on/off** pour afficher ou masquer la série de données dans le graphique final.                     |
| **Nom de la série**                       | Saisissez un **nom unique** pour identifier la série de données dans la **liste des séries** au sein de la configuration du graphique.  |
| **Titre de légende de la série**          | Ajoutez un **titre descriptif** pour la série à afficher dans la **légende du graphique**.                                              |
| **Ajouter la série à l'axe Y secondaire** | Affectez cette série à l'**axe Y secondaire**, utile pour comparer des séries de données avec des échelles différentes.                 |
|                                           | **Note:** Cette option n'est disponible que si le paramètre général du graphique **"Afficher l'axe Y secondaire"** est activé.          |

***

## **Points clés**

* **Choisir le bon type de graphique** est essentiel pour une visualisation des données claire et efficace.
* **L'axe Y secondaire** aide à visualiser des ensembles de données ayant des plages de valeurs très différentes sur le même graphique.
* Vous pouvez gérer la **visibilité** des séries de manière dynamique, ce qui est utile lorsque vous souhaitez comparer uniquement certaines séries de données sans en supprimer d'autres.
* **Les titres de légende** facilitent la compréhension des graphiques pour les spectateurs, en particulier lorsque plusieurs séries sont affichées.
