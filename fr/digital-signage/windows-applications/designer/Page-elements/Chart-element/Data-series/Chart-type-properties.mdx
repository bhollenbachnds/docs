Les **Chart Type properties** dans le **Data Series Wizard** définissent **comment vos séries de données seront représentées visuellement** dans le graphique. Ces propriétés vous permettent de sélectionner le **type de graphique**, **nommer la série**, contrôler sa **visibilité**, et **l'assigner à un axe secondaire** si nécessaire.

***

## **Aperçu des propriétés**

| **Propriété**                     | **Description**                                                                                                                  |
| --------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| **Chart Type**                    | Sélectionnez le **type de graphique** pour afficher votre série de données. Les exemples incluent **bar, line, pie, area**, etc. |
|                                   | Lorsque vous choisissez un type, une **fenêtre d'aperçu** montre un exemple visuel du graphique pour vous aider à décider.       |
| **Visible Serie**                 | Activez ou désactivez cette option **on/off** pour afficher ou masquer la série de données dans le graphique final.              |
| **Serie Name**                    | Entrez un **nom unique** pour identifier la série de données dans la **liste des séries** dans la configuration du graphique.    |
| **Serie Legend Title**            | Ajoutez un **titre descriptif** pour la série à afficher dans la **légende du graphique**.                                       |
| **Add serie to secondary Y-Axis** | Assignez cette série à l'**axe Y secondaire**, utile lors de la comparaison de séries de données avec différentes échelles.      |
|                                   | **Note:** Cette option n'est disponible que si le paramètre général du graphique **"Show Secondary Y-Axis"** est activé.         |

***

## **Notes importantes**

* **Choisir le type de graphique approprié** est crucial pour une visualisation claire et efficace des données.
* **Secondary Y-Axis** aide à visualiser des ensembles de données qui ont des plages de valeurs significativement différentes sur le même graphique.
* Vous pouvez gérer la **visibilité** des séries de façon dynamique, ce qui est utile lorsque vous souhaitez comparer uniquement certaines séries de données sans supprimer les autres.
* **Legend titles** rendent les graphiques plus faciles à comprendre pour les spectateurs, surtout lorsque plusieurs séries sont affichées.
