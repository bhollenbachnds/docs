Les **propriétés du type de graphique** dans le **Assistant de séries de données** définissent **comment votre série de données sera représentée visuellement** dans le graphique. Ces propriétés vous permettent de sélectionner le **type de graphique**, **nommer la série**, contrôler sa **visibilité**, et **l'affecter à un axe secondaire** si nécessaire.

***

## **Aperçu des propriétés**

| **Propriété**                              | **Description**                                                                                                                              |
| ------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------- |
| **Type de graphique**                      | Sélectionnez le **type de graphique** pour afficher votre série de données. Les exemples incluent **barres, courbes, secteurs, aires**, etc. |
|                                            | Lorsque vous choisissez un type, une **fenêtre d'aperçu** affiche un exemple visuel du graphique pour vous aider à décider.                  |
| **Série visible**                          | Activez ou désactivez cette option **activé/désactivé** pour afficher ou masquer la série de données dans le graphique final.                |
| **Nom de la série**                        | Saisissez un **nom unique** pour identifier la série de données dans la **liste des séries** au sein de la configuration du graphique.       |
| **Titre de la légende de la série**        | Ajouter un **titre descriptif** pour que la série soit affichée dans la **légende du graphique**.                                            |
| **Ajouter une série à l'axe Y secondaire** | Assigner cette série à l'**axe Y secondaire**, utile pour comparer des séries de données avec des échelles différentes.                      |
|                                            | **Note:** Cette option n'est disponible que si le paramètre général du graphique **"Afficher l'axe Y secondaire"** est activé.               |

***

## **Notes clés**

* **Choisir le bon type de graphique** est crucial pour une visualisation des données claire et efficace.
* **Axe Y secondaire** aide à visualiser des ensembles de données ayant des plages de valeurs significativement différentes sur le même graphique.
* Vous pouvez gérer la **visibilité** des séries de manière dynamique, ce qui est utile lorsque vous souhaitez comparer uniquement certaines séries de données sans en supprimer d'autres.
* **Les titres de légende** rendent les graphiques plus faciles à comprendre pour les spectateurs, surtout lorsque plusieurs séries sont affichées.
