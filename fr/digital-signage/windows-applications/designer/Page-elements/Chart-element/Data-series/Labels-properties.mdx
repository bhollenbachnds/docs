Les **propriétés des étiquettes** dans le **Data Series Wizard** définissent comment les **étiquettes des éléments de données** sont affichées dans votre graphique, notamment **la visibilité, le style, le contenu et le positionnement**.

***

## **Étiquettes**

| **Propriété**                                     | **Description**                                                                              |
| ------------------------------------------------- | -------------------------------------------------------------------------------------------- |
| **Afficher les étiquettes des points de données** | **Activer/désactiver** l'affichage des étiquettes pour les points de données de votre série. |

***

## **Style**

| **Propriété**              | **Description**                                                                                                                               |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| **Police**                 | Affiche la **police et le style actuels** des étiquettes. Cliquez sur le bouton à côté de la zone de texte pour l'ajuster.                    |
| **Couleur de l'étiquette** | Choisissez la couleur du texte de l'étiquette. Les options incluent **couleurs fixes, dégradés, textures et motifs**.                         |
| **Couleur d'arrière-plan** | Choisissez la **couleur d'arrière-plan** derrière le texte de l'étiquette. Prend en charge les couleurs unies, les dégradés, les motifs, etc. |

***

## **Position**

| **Propriété**              | **Description**                                                                                                                                                   |
| -------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Position**               | Sélectionnez la façon dont les étiquettes sont positionnées dans le graphique. Utilisez **Automatique** pour laisser le système décider de l'emplacement optimal. |
| **Intervalle d'étiquette** | Définissez l'intervalle (en pixels) entre deux étiquettes consécutives.                                                                                           |

***

## **Contenu**

| **Propriété**          | **Description**                                                                                                     |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Contenu**            | Choisissez d'afficher uniquement les **étiquettes** ou **étiquettes et valeurs** ensemble.                          |
| **Format de valeur**   | Si des valeurs sont incluses, choisissez un **format** pour afficher ces valeurs (par ex., devise, décimal).        |
| **Format d'étiquette** | Choisissez un **format** pour le texte de l'étiquette (par ex., texte brut, mise en forme de chaîne personnalisée). |

***

## **Bordure**

| **Propriété**                          | **Description**                                                                                                          |
| -------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| **Afficher la bordure de l'étiquette** | **Activer/désactiver** une bordure autour des étiquettes.                                                                |
| **Largeur**                            | Définissez l'**épaisseur (en pixels)** de la bordure de l'étiquette.                                                     |
| **Couleur de bordure**                 | Choisissez une couleur pour la bordure de l'étiquette. Prend en charge **couleurs fixes, dégradés, textures et motifs**. |

***

## **Remarques**

* Les étiquettes peuvent être entièrement personnalisées pour correspondre à vos **besoins en matière de design et d'image de marque**, notamment **polices, couleurs, arrière-plans et bordures**.
* L'**Intervalle d'étiquette** contrôle la **densité des étiquettes**, ce qui peut être utile pour les graphiques comportant de nombreux points de données.
* Si **des valeurs sont affichées**, les options de mise en forme garantissent que les données numériques sont présentées clairement (par ex., en pourcentages, en devise).
* Le positionnement des étiquettes peut être laissé à **l'ajustement automatique**, garantissant que le graphique reste lisible sans réglage manuel.
