Le **Labels properties** dans le **Data Series Wizard** définissent comment **data item labels** sont affichés dans votre graphique, y compris **visibility, style, content, and positioning**.

***

## **Labels**

| **Property**              | **Description**                                                                                |
| ------------------------- | ---------------------------------------------------------------------------------------------- |
| **Show Datapoint Labels** | **Activer/désactiver** l'affichage des étiquettes pour les points de données dans votre série. |

***

## **Style**

| **Property**         | **Description**                                                                                                                         |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| **Font**             | Affiche la **current font and style** des étiquettes. Cliquez sur le bouton à côté de la zone de texte pour l'ajuster.                  |
| **Label Color**      | Choisissez la couleur du texte de l'étiquette. Les options incluent **fixed colors, gradients, textures, and patterns**.                |
| **Background Color** | Choisissez la **background color** derrière le texte de l'étiquette. Prend en charge les couleurs unies, les dégradés, les motifs, etc. |

***

## **Position**

| **Property**       | **Description**                                                                                                                                           |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Position**       | Sélectionnez comment les étiquettes sont positionnées dans le graphique. Utilisez **Automatic** pour laisser le système décider de l'emplacement optimal. |
| **Label Interval** | Définissez l'intervalle (en pixels) entre deux étiquettes consécutives.                                                                                   |

***

## **Content**

| **Property**     | **Description**                                                                                                  |
| ---------------- | ---------------------------------------------------------------------------------------------------------------- |
| **Content**      | Choisissez d'afficher uniquement les **labels** ou **labels and values** ensemble.                               |
| **Value Format** | Si les valeurs sont incluses, choisissez un **format** pour afficher ces valeurs (par exemple, devise, décimal). |
| **Label Format** | Choisissez un **format** pour le texte de l'étiquette (par exemple, simple, formatage de chaîne personnalisé).   |

***

## **Border**

| **Property**          | **Description**                                                                                                                 |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **Show Label Border** | **Activer/désactiver** une bordure autour des étiquettes.                                                                       |
| **Width**             | Définissez l'**thickness (in pixels)** de la bordure de l'étiquette.                                                            |
| **Border Color**      | Choisissez une couleur pour la bordure de l'étiquette. Prend en charge les **fixed colors, gradients, textures, and patterns**. |

***

## **Remarks**

* Les étiquettes peuvent être entièrement personnalisées pour correspondre à vos **design and branding needs**, y compris **fonts, colors, backgrounds, and borders**.
* Le **Label Interval** contrôle la **density of labels**, ce qui peut être utile pour les graphiques avec de nombreux points de données.
* Si **values are displayed**, les options de formatage garantissent que les données numériques sont présentées clairement (par exemple, en pourcentages, en devise).
* Le positionnement des étiquettes peut être laissé à l'**automatic adjustment**, garantissant que le graphique reste lisible sans réglage manuel fin.
