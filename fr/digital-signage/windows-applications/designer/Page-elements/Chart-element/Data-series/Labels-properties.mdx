Les **Labels properties** dans le **Data Series Wizard** définissent comment **data item labels** sont affichés dans votre graphique, y compris **la visibilité, le style, le contenu et le positionnement**.

***

## **Labels**

| **Propriété**             | **Description**                                                                                |
| ------------------------- | ---------------------------------------------------------------------------------------------- |
| **Show Datapoint Labels** | **Activer/désactiver** l'affichage des étiquettes pour les points de données dans votre série. |

***

## **Style**

| **Propriété**        | **Description**                                                                                                                               |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| **Font**             | Affiche la **police et le style actuels** des étiquettes. Cliquez sur le bouton à côté de la zone de texte pour l'ajuster.                    |
| **Label Color**      | Choisissez la couleur du texte de l'étiquette. Les options incluent **des couleurs fixes, des dégradés, des textures et des motifs**.         |
| **Background Color** | Choisissez la **couleur d'arrière-plan** derrière le texte de l'étiquette. Prend en charge les couleurs unies, les dégradés, les motifs, etc. |

***

## **Position**

| **Propriété**      | **Description**                                                                                                                                           |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Position**       | Sélectionnez comment les étiquettes sont positionnées dans le graphique. Utilisez **Automatic** pour laisser le système décider de l'emplacement optimal. |
| **Label Interval** | Définissez l'intervalle (en pixels) entre deux étiquettes consécutives.                                                                                   |

***

## **Content**

| **Propriété**    | **Description**                                                                                                  |
| ---------------- | ---------------------------------------------------------------------------------------------------------------- |
| **Content**      | Choisissez d'afficher uniquement les **labels** ou **labels and values** ensemble.                               |
| **Value Format** | Si des valeurs sont incluses, choisissez un **format** pour afficher ces valeurs (par exemple, devise, décimal). |
| **Label Format** | Choisissez un **format** pour le texte de l'étiquette (par exemple, simple, formatage de chaîne personnalisé).   |

***

## **Border**

| **Propriété**         | **Description**                                                                                                                          |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| **Show Label Border** | **Activer/désactiver** une bordure autour des étiquettes.                                                                                |
| **Width**             | Définissez **l'épaisseur (en pixels)** de la bordure de l'étiquette.                                                                     |
| **Border Color**      | Choisissez une couleur pour la bordure de l'étiquette. Prend en charge **des couleurs fixes, des dégradés, des textures et des motifs**. |

***

## **Remarques**

* Les étiquettes peuvent être entièrement personnalisées pour correspondre à vos **besoins de conception et d'image de marque**, y compris **les polices, les couleurs, les arrière-plans et les bordures**.
* Le **Label Interval** contrôle la **densité des étiquettes**, ce qui peut être utile pour les graphiques avec de nombreux points de données.
* Si **des valeurs sont affichées**, les options de formatage garantissent que les données numériques sont présentées clairement (par exemple, en pourcentages, en devise).
* Le positionnement des étiquettes peut être laissé à **l'ajustement automatique**, garantissant que le graphique reste lisible sans réglage manuel fin.
