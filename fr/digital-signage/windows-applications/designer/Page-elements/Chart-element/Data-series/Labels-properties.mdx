Les **propriétés Labels** dans le **Data Series Wizard** définissent la façon dont les **étiquettes des éléments de données** sont affichées dans votre graphique, notamment la **visibilité, le style, le contenu et le positionnement**.

***

## **Labels**

| **Propriété**             | **Description**                                                                              |
| ------------------------- | -------------------------------------------------------------------------------------------- |
| **Show Datapoint Labels** | **Activer/désactiver** l'affichage des étiquettes pour les points de données de votre série. |

***

## **Style**

| **Propriété**        | **Description**                                                                                                                            |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| **Font**             | Affiche la **police et le style actuels** des étiquettes. Cliquer sur le bouton à côté de la zone de texte pour l'ajuster.                 |
| **Label Color**      | Choisir la couleur du texte de l'étiquette. Les options incluent les **couleurs fixes, les dégradés, les textures et les motifs**.         |
| **Background Color** | Choisir la **couleur d'arrière-plan** derrière le texte de l'étiquette. Prend en charge les couleurs unies, les dégradés, les motifs, etc. |

***

## **Position**

| **Propriété**      | **Description**                                                                                                                                                 |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Position**       | Sélectionner la façon dont les étiquettes sont positionnées dans le graphique. Utiliser **Automatic** pour laisser le système décider de l'emplacement optimal. |
| **Label Interval** | Définir l'intervalle (en pixels) entre deux étiquettes consécutives.                                                                                            |

***

## **Content**

| **Propriété**    | **Description**                                                                                                  |
| ---------------- | ---------------------------------------------------------------------------------------------------------------- |
| **Content**      | Choisir d'afficher uniquement les **étiquettes** ou les **étiquettes et valeurs** ensemble.                      |
| **Value Format** | Si des valeurs sont incluses, choisir un **format** pour afficher ces valeurs (par ex., devise, décimal).        |
| **Label Format** | Choisir un **format** pour le texte de l'étiquette (par ex., texte brut, mise en forme de chaîne personnalisée). |

***

## **Border**

| **Propriété**                          | **Description**                                                                                                                          |
| -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| **Afficher la bordure de l'étiquette** | **Activer/désactiver** une bordure autour des étiquettes.                                                                                |
| **Largeur**                            | Définir l' **épaisseur (en pixels)** de la bordure de l'étiquette.                                                                       |
| **Couleur de la bordure**              | Choisissez une couleur pour la bordure de l'étiquette. Prend en charge les **couleurs fixes, les dégradés, les textures et les motifs**. |

***

## **Remarques**

* Les étiquettes peuvent être entièrement personnalisées pour correspondre à vos **besoins en matière de design et d'image de marque**, notamment **les polices, les couleurs, les arrière-plans et les bordures**.
* Le **Label Interval** contrôle la **densité des étiquettes**, ce qui peut être utile pour les graphiques comportant de nombreux points de données.
* Si les **valeurs sont affichées**, les options de mise en forme garantissent que les données numériques sont présentées clairement (par exemple, en pourcentages, en devise).
* Le positionnement des étiquettes peut être laissé à un **ajustement automatique**, garantissant que le graphique reste lisible sans réglage manuel.
