Les **Propriétés du fournisseur de données** dans l'**Assistant de séries de données** définissent **comment la série de données se connecte à une source de données externe**, garantissant que votre graphique est alimenté avec des **données en temps réel**.

***

## **Aperçu des propriétés**

| **Propriété**                              | **Description**                                                                                                                                                                                        |
| ------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Sélectionner un fournisseur de données** | Choisissez un **fournisseur de données** existant (table, requête, procédure stockée, fichier XML, flux RSS, etc.) qui alimente la série de données.                                                   |
|                                            | Les fournisseurs de données sont définis **au niveau de la présentation** et sont disponibles pour tous les éléments. Vous pouvez créer un **nouveau fournisseur à la volée** via le bouton déroulant. |
| **Colonne de données**                     | Sélectionnez la **colonne** du fournisseur de données qui contient les **valeurs de données principales** pour le graphique.                                                                           |
| **Colonne d'étiquette** (Optionnel)        | Sélectionnez une **colonne** qui contient des **étiquettes** associées aux valeurs de données (par exemple, noms de produits, dates).                                                                  |

***

## **Colonnes spéciales pour types de graphiques spécifiques**

### **Colonnes de graphique à bulles**

Si **Graphique à bulles** est sélectionné comme type de graphique :

| **Colonne de bulle**           | **Description**                                                                                                                         |
| ------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------- |
| **Colonne de taille de bulle** | Sélectionnez la colonne qui fournit la **taille de chaque bulle**, utilisée pour représenter une troisième dimension dans le graphique. |

***

### **Colonnes de graphique boursier**

Si **Graphique boursier** est sélectionné comme type de graphique :

| **Colonne boursière** | **Description**                                           |
| --------------------- | --------------------------------------------------------- |
| **Colonne Haut**      | Colonne contenant la **valeur boursière la plus élevée**. |
| **Colonne Bas**       | Colonne contenant la **valeur boursière la plus basse**.  |
| **Colonne Ouverture** | Colonne contenant la **valeur boursière d'ouverture**.    |
| **Colonne Clôture**   | Colonne contenant la **valeur boursière de clôture**.     |

***

## **Notes clés**

* **Les fournisseurs de données** permettent une alimentation dynamique en données en temps réel pour des mises à jour en direct dans vos graphiques.
* La sélection de colonnes **lie des champs spécifiques** de vos données source au graphique pour la visualisation.
* Différents types de graphiques (par exemple, Bulle, Boursier) nécessitent des **colonnes spécialisées** pour représenter correctement les données.
* Vous pouvez définir **de nouveaux fournisseurs de données directement** lors de la configuration de votre série de données pour plus de commodité.
