Les **propriétés du Data Provider** dans le **Data Series Wizard** définissent **comment la série de données se connecte à une source de données externe**, garantissant que votre graphique est alimenté par des **données en temps réel**.

***

## **Aperçu des propriétés**

| **Propriété**                     | **Description**                                                                                                                                                                            |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Sélectionner un Data Provider** | Choisissez un **data provider** existant (table, requête, procédure stockée, fichier XML, flux RSS, etc.) qui alimente la série de données.                                                |
|                                   | Les data providers sont définis **au niveau de la présentation** et sont disponibles pour tous les éléments. Vous pouvez créer un **nouveau provider à la volée** via le bouton déroulant. |
| **Data Column**                   | Sélectionnez la **colonne** du data provider qui contient les **valeurs de données principales** pour le graphique.                                                                        |
| **Label Column** (Optionnel)      | Sélectionnez une **colonne** qui contient les **labels** associés aux valeurs de données (par ex., noms de produits, dates).                                                               |

***

## **Colonnes spéciales pour des types de graphiques spécifiques**

### **Colonnes du Bubble Chart**

Si **Bubble Chart** est sélectionné comme type de graphique :

| **Colonne Bubble**     | **Description**                                                                                                                         |
| ---------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| **Bubble Size Column** | Sélectionnez la colonne qui fournit la **taille de chaque bulle**, utilisée pour représenter une troisième dimension dans le graphique. |

***

### **Colonnes du Stock Chart**

Si **Stock Chart** est sélectionné comme type de graphique :

| **Colonne Stock** | **Description**                                          |
| ----------------- | -------------------------------------------------------- |
| **High Column**   | Colonne contenant la **valeur boursière la plus haute**. |
| **Low Column**    | Colonne contenant la **valeur boursière la plus basse**. |
| **Open Column**   | Colonne contenant la **valeur boursière d'ouverture**.   |
| **Close Column**  | Colonne contenant la **valeur boursière de clôture**.    |

***

## **Points clés**

* **Data Providers** permettent une alimentation dynamique en données en temps réel pour des mises à jour en direct dans vos graphiques.
* La sélection de colonnes **lie des champs spécifiques** de vos données source au graphique pour la visualisation.
* Différents types de graphiques (par ex., Bubble, Stock) nécessitent des **colonnes spécialisées** pour représenter correctement les données.
* Vous pouvez définir **de nouveaux data providers directement** lors de la configuration de votre série de données pour plus de commodité.
