Les séries de données définissent **quelles données sont affichées** dans le graphique et **comment elles sont représentées**. Chaque série de données se connecte à une source de données (comme une base de données ou un fichier XML) et contient les paramètres qui contrôlent **le type de graphique**, **le style**, **les étiquettes**, **les couleurs**, et **d'autres options d'affichage**.

Bien que la plupart des graphiques fonctionnent avec une **série de données unique**, vous pouvez ajouter **plusieurs séries de données** pour afficher différents ensembles de données dans le même graphique.

***

## **Gestion des séries de données**

### **Pour ajouter une nouvelle série de données**

1. Allez à la section **Data Series** dans la **fenêtre des propriétés du graphique**.
2. Cliquez sur **Add...**.
3. Suivez l'assistant pour définir les propriétés de la série de données.
4. Cliquez sur **Finish** pour ajouter la série de données.

### **Pour modifier une série de données existante**

* Sélectionnez la série de données souhaitée dans la liste et cliquez sur **Edit...**.

### **Pour supprimer une série de données**

* Sélectionnez la série de données et cliquez sur **Remove**.

### **Pour dupliquer une série de données**

* Sélectionnez la série de données et cliquez sur **Duplicate** pour créer une copie exacte.

### **Pour ajuster l'ordre d'affichage**

* Utilisez **Move Up** ou **Move Down** pour modifier la séquence des séries de données.
* **Note** : La séquence détermine **comment les séries de données apparaissent dans le graphique**.

***

## **Aperçu des propriétés**

| **Section de propriété**   | **Description**                                                                                                   |
| -------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| **Type de graphique**      | Définit le **type de graphique** (2D ou 3D) pour afficher vos données (par ex., barre, ligne, secteur).           |
| **Fournisseur de données** | Connecte la série de données à une **source de données externe** pour des données en temps réel.                  |
| **Options**                | Divers paramètres liés au **comportement et aux options d'affichage** de la série de données.                     |
| **Style**                  | Contrôle l'**apparence**, comme la largeur de ligne, l'espacement des barres ou le style de marqueur.             |
| **Étiquettes**             | Permet d'ajouter des **étiquettes de données** à votre série pour une meilleure visualisation et explication.     |
| **Couleurs**               | Définit les **couleurs** pour les éléments comme les barres, les lignes ou les segments dans la série de données. |

***

## **Notes clés**

* Vous pouvez **mélanger différents types de graphiques** (par ex., barres et lignes) dans le même graphique en ajoutant plusieurs séries de données.
* La gestion de plusieurs séries de données permet de créer des **visualisations complexes et dynamiques** à partir de divers ensembles de données.
* L'ajustement de l'**ordre des séries de données** a un impact sur la façon dont elles sont superposées ou empilées dans le graphique.
