Ces propriétés définissent comment la légende d'un graphique est affichée. Une légende explique, pour chaque série de données, comment elle peut être identifiée dans le graphique.

***

#### **Légende**

**Afficher la légende**\
Activer ou désactiver l'affichage d'une légende dans le graphique.

**Position**\
Sélectionnez une position prédéfinie pour la légende dans la zone du graphique.

**Afficher l'arrière-plan**\
Activer ou désactiver un cadre d'arrière-plan pour la légende.

**Couleur d'arrière-plan**\
Sélectionnez une couleur d'arrière-plan pour le cadre de la légende. Cela peut être une couleur fixe, un dégradé, une texture ou un motif. (Voir *Utilisation du sélecteur de couleur* pour plus de détails.)

**Couleur de bordure**\
Sélectionnez une couleur de bordure pour le cadre de la légende. Prend en charge les couleurs fixes, les dégradés, les textures ou les motifs.

**Largeur de bordure**\
Définissez l'épaisseur de la bordure en pixels.

***

#### **Police (Éléments de légende)**

**Style**\
Affiche la police actuelle et le style de police utilisés pour les éléments de légende. Cliquez pour ouvrir la boîte de dialogue de police et changer la police.

**Couleur**\
Sélectionnez une couleur de police pour les éléments de légende. Prend en charge les couleurs fixes, les dégradés, les textures ou les motifs.

**Ombre**\
Sélectionnez une couleur d'ombre pour les éléments de légende. Prend en charge les couleurs fixes, les dégradés, les textures ou les motifs.

**Profondeur de l'ombre**\
Définissez la distance (en pixels) entre les éléments de légende et leurs ombres.

***

#### **En-tête**

**Texte**\
Le titre affiché dans le cadre de la légende.

**Police**\
Affiche la police actuelle et le style de police utilisés pour le titre de la légende. Cliquez pour ouvrir la boîte de dialogue de police et changer la police.

**Couleur**\
Sélectionnez une couleur de police pour le titre de la légende. Prend en charge les couleurs fixes, les dégradés, les textures ou les motifs.
