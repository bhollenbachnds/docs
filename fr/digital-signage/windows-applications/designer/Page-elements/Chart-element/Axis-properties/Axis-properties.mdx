Ces propriétés déterminent la **disposition, l'apparence et la visibilité** de l'axe en général, et peuvent être définies pour les **axes X, Y et Z** individuellement.

***

## **Propriétés de l'axe X**

| **Propriété**             | **Description**                                                                                                                         |
| ------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| **Title**                 | Le libellé textuel affiché le long de l'axe, décrivant son objectif (par exemple, « Ventes », « Temps », « Catégories »).               |
| **Color**                 | Définit la couleur de la ligne d'axe pour une personnalisation visuelle.                                                                |
| **Show X-Axis**           | Active ou désactive la visibilité de l'axe X.                                                                                           |
| **Show Secondary X-Axis** | Active ou désactive un second axe X du côté opposé du graphique, utile pour les comparaisons de données complexes.                      |
| **Show X-Axis Margins**   | Lorsqu'elle est activée, ajoute des marges (rembourrage) entre les valeurs minimales/maximales des données et le début/la fin de l'axe. |

***

## **Remarques**

* **Toutes ces propriétés** sont également disponibles pour les **axes Y et Z**, permettant une personnalisation cohérente et flexible de tous les axes du graphique.
* **Les axes secondaires** sont particulièrement utiles pour les graphiques qui doivent comparer des jeux de données avec des échelles différentes.
* **Les marges d'axe** peuvent améliorer la lisibilité en empêchant les points de données de toucher les bords du graphique.
