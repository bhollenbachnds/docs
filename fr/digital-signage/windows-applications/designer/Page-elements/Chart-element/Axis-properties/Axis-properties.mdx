Ces propriétés déterminent la **disposition, l'apparence et la visibilité** de l'axe en général, et peuvent être définies pour **X-axis, Y-axis, et Z-axis** individuellement.

***

## **Propriétés de l'axe X**

| **Propriété**             | **Description**                                                                                                           |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| **Title**                 | L'étiquette de texte affichée le long de l'axe, décrivant son objectif (par ex., "Ventes", "Temps", "Catégories").        |
| **Color**                 | Définit la couleur de la ligne d'axe pour la personnalisation visuelle.                                                   |
| **Show X-Axis**           | Active ou désactive la visibilité de l'axe X.                                                                             |
| **Show Secondary X-Axis** | Active ou désactive un second axe X du côté opposé du graphique, utile pour les comparaisons de données complexes.        |
| **Show X-Axis Margins**   | Lorsqu'activé, ajoute des marges (rembourrage) entre les valeurs de données minimales/maximales et le début/fin de l'axe. |

***

## **Remarques**

* **Toutes ces propriétés** sont également disponibles pour le **Axe Y et axe Z**, permettant une personnalisation cohérente et flexible de tous les axes du graphique.
* **Axes secondaires** sont particulièrement utiles pour les graphiques qui doivent comparer des ensembles de données avec différentes échelles.
* **Marges d'axe** peuvent améliorer la lisibilité en empêchant les points de données de toucher les bords du graphique.
