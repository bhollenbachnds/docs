Ces propriétés déterminent la **mise en page, l'apparence et la visibilité** de l'axe en général, et peut être défini pour **Axe X, Axe Y et Axe Z** individuellement.

***

## **Propriétés de l'axe X**

| **Propriété**                      | **Description**                                                                                                                        |
| ---------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| **Titre**                          | L'étiquette de texte affichée le long de l'axe, décrivant son objectif (par ex., « Ventes », « Temps », « Catégories »).               |
| **Couleur**                        | Définit la couleur de la ligne d'axe pour une personnalisation visuelle.                                                               |
| **Afficher l'axe X**               | Active ou désactive la visibilité de l'axe X.                                                                                          |
| **Afficher l'axe X secondaire**    | Active ou désactive un second axe X du côté opposé du graphique, utile pour les comparaisons de données complexes.                     |
| **Afficher les marges de l'axe X** | Lorsqu'elle est activée, ajoute des marges (rembourrage) entre les valeurs de données minimales/maximales et le début/la fin de l'axe. |

***

## **Remarques**

* **Toutes ces propriétés** sont également disponibles pour l'**axe Y et l'axe Z**, permettant une personnalisation cohérente et flexible de tous les axes du graphique.
* **Les axes secondaires** sont particulièrement utiles pour les graphiques qui doivent comparer des ensembles de données avec des échelles différentes.
* **Les marges d'axe** peuvent améliorer la lisibilité en empêchant les points de données de toucher les bords du graphique.
