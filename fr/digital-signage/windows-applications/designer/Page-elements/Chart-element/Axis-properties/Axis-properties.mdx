Ces propriétés déterminent la **disposition, l'apparence et la visibilité** de l'axe en général, et peuvent être définies pour **l'axe X, l'axe Y et l'axe Z** individuellement.

***

## **Propriétés de l'axe X**

| **Propriété**                      | **Description**                                                                                                                              |
| ---------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| **Titre**                          | Le texte affiché le long de l'axe, décrivant son objectif (par ex., "Ventes", "Temps", "Catégories").                                        |
| **Couleur**                        | Définit la couleur de la ligne d'axe pour la personnalisation visuelle.                                                                      |
| **Afficher l'axe X**               | Active ou désactive la visibilité de l'axe X.                                                                                                |
| **Afficher l'axe X secondaire**    | Active ou désactive un second axe X du côté opposé du graphique, utile pour les comparaisons de données complexes.                           |
| **Afficher les marges de l'axe X** | Lorsque cette option est activée, ajoute des marges (rembourrage) entre les valeurs de données minimales/maximales et le début/fin de l'axe. |

***

## **Remarques**

* **Toutes ces propriétés** sont également disponibles pour **l'axe Y et l'axe Z**, permettant une personnalisation cohérente et flexible de tous les axes du graphique.
* **Les axes secondaires** sont particulièrement utiles pour les graphiques qui doivent comparer des ensembles de données avec différentes échelles.
* **Les marges d'axe** peuvent améliorer la lisibilité en empêchant les points de données de toucher les bords du graphique.
