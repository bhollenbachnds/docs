Ces propriétés définissent comment les **labels** sont affichés le long des **X-axis, Y-axis, et Z-axis** d'un élément graphique. Les labels aident les utilisateurs à comprendre ce que représente chaque axe et peuvent être entièrement personnalisés pour la clarté visuelle et la précision des données.

***

## **Paramètres des Labels**

| **Propriété**                | **Description**                                                                                                                              |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| **Police**                   | Spécifie la police et le style des labels d'axe. Cliquez sur le bouton à côté du champ de police pour ouvrir la boîte de dialogue de police. |
| **Couleur**                  | Définit la couleur des labels d'axe pour un meilleur contraste et une meilleure lisibilité.                                                  |
| **Mode Label Automatique**   | Détermine automatiquement le nombre de labels qui s'adapteront à l'axe et ajuste dynamiquement leur placement.                               |
| **Mode Label Manuel**        | Définissez manuellement le nombre et le positionnement des labels pour un contrôle précis.                                                   |
| **Angle**                    | Définit l'angle de rotation (en degrés) des labels par rapport à l'axe. Utile pour adapter les textes longs.                                 |
| **Intervalle**               | Définit l'espace (en pixels) entre les labels consécutifs en mode manuel.                                                                    |
| **Zones**                    | Lorsqu'activé, inclut des labels pour les valeurs minimales et maximales de l'axe.                                                           |
| **Valeur Minimale de l'Axe** | Spécifie la valeur minimale de l'axe à laquelle un label doit être attribué.                                                                 |
| **Valeur Maximale de l'Axe** | Spécifie la valeur maximale de l'axe à laquelle un label doit être attribué.                                                                 |

***

## **Intégration du Fournisseur de Données**

| **Propriété**                                               | **Description**                                                                                                                       |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| **Afficher le Contenu du Fournisseur de Données sur l'Axe** | Active ou désactive l'utilisation d'un fournisseur de données pour fournir dynamiquement les labels.                                  |
| **Fournisseur de Données**                                  | Sélectionnez un fournisseur de données qui représente une connexion à une source de données (par exemple, base de données, XML, RSS). |
| **Colonne**                                                 | Choisissez la colonne dans le fournisseur de données sélectionné qui contient les valeurs des labels.                                 |

> **Remarque** : Les fournisseurs de données sont définis au niveau de la **presentation level** et sont disponibles pour tous les éléments de votre présentation. Vous pouvez créer un nouveau fournisseur de données directement à partir de cette fenêtre en utilisant le bouton adjacent. Référez-vous à **Working with Data Providers** pour plus de détails.

***

## **Remarques**

* Toutes les propriétés décrites ci-dessus pour l'**X-axis** sont également applicables à l'**Y-axis and Z-axis**.
* Lors de l'utilisation du **Automatic Label Mode**, le système garantit que les labels sont lisibles et bien espacés, s'ajustant automatiquement en fonction de l'espace disponible.
* **Manual Label Mode** est idéal lorsqu'un contrôle précis des positions des labels est nécessaire, comme pour des intervalles fixes ou des catégories prédéfinies.
