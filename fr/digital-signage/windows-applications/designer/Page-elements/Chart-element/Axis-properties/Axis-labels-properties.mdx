Ces propriétés définissent la façon dont les **étiquettes** sont affichées le long de l'**axe X, de l'axe Y et de l'axe Z** d'un élément de graphique. Les étiquettes aident les utilisateurs à comprendre ce que représente chaque axe et peuvent être entièrement personnalisées pour une meilleure clarté visuelle et une meilleure précision des données.

***

## **Paramètres des étiquettes**

| **Propriété**                     | **Description**                                                                                                                                        |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Police**                        | Spécifie la police et le style des étiquettes d'axe. Cliquez sur le bouton situé à côté du champ de police pour ouvrir la boîte de dialogue de police. |
| **Couleur**                       | Définit la couleur des étiquettes d'axe pour un meilleur contraste et une meilleure lisibilité.                                                        |
| **Mode d'étiquetage automatique** | Détermine automatiquement le nombre d'étiquettes pouvant tenir sur l'axe et ajuste leur placement de manière dynamique.                                |
| **Mode d'étiquetage manuel**      | Définissez manuellement le nombre et le positionnement des étiquettes pour un contrôle précis.                                                         |
| **Angle**                         | Définit l'angle de rotation (en degrés) des étiquettes par rapport à l'axe. Utile pour les textes longs.                                               |
| **Intervalle**                    | Définit l'espace (en pixels) entre les étiquettes consécutives en mode manuel.                                                                         |
| **Zones**                         | Lorsqu'il est activé, inclut des étiquettes pour les valeurs minimale et maximale de l'axe.                                                            |
| **Valeur minimale de l'axe**      | Spécifie la valeur minimale de l'axe à laquelle une étiquette doit être attribuée.                                                                     |
| **Valeur maximale de l'axe**      | Spécifie la valeur maximale de l'axe à laquelle une étiquette doit être attribuée.                                                                     |

***

## **Intégration du Data Provider**

| **Propriété**                                      | **Description**                                                                                                        |
| -------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| **Afficher le contenu du Data Provider sur l'axe** | Active ou désactive l'utilisation d'un data provider pour fournir les étiquettes de manière dynamique.                 |
| **Data Provider**                                  | Sélectionnez un data provider représentant une connexion à une source de données (par ex., base de données, XML, RSS). |
| **Colonne**                                        | Choisissez la colonne dans le data provider sélectionné qui contient les valeurs des étiquettes.                       |

> **Remarque**: Les data providers sont définis au niveau de la **présentation** et sont disponibles pour tous les éléments de votre présentation. Vous pouvez créer un nouveau data provider directement depuis cette fenêtre à l'aide du bouton adjacent. Consultez **Utilisation des Data Providers** pour plus de détails.

***

## **Remarques**

* Toutes les propriétés décrites ci-dessus pour l'**axe X** s'appliquent également à l'**axe Y et à l'axe Z**.
* Lors de l'utilisation du **mode d'étiquetage automatique**, le système s'assure que les étiquettes sont lisibles et bien espacées, en s'ajustant automatiquement en fonction de l'espace disponible.
* **Mode d'étiquetage manuel** est idéal lorsqu'un contrôle précis des positions des étiquettes est nécessaire, par exemple pour des intervalles fixes ou des catégories prédéfinies.
