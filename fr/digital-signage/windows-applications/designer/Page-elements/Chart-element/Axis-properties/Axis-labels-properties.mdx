Ces propriétés définissent comment les **étiquettes** sont affichées le long de l'**axe X, de l'axe Y et de l'axe Z** d'un élément de graphique. Les étiquettes aident les utilisateurs à comprendre ce que représente chaque axe et peuvent être entièrement personnalisées pour une clarté visuelle et une précision des données.

***

## **Paramètres des étiquettes**

| **Propriété**                     | **Description**                                                                                                                                  |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Police**                        | Spécifie la police et le style des étiquettes d'axe. Cliquez sur le bouton à côté du champ de police pour ouvrir la boîte de dialogue de police. |
| **Couleur**                       | Définit la couleur des étiquettes d'axe pour un meilleur contraste et une meilleure lisibilité.                                                  |
| **Mode d'étiquetage automatique** | Détermine automatiquement le nombre d'étiquettes qui tiendront sur l'axe et ajuste leur placement de manière dynamique.                          |
| **Mode d'étiquetage manuel**      | Définissez manuellement le nombre et le positionnement des étiquettes pour un contrôle précis.                                                   |
| **Angle**                         | Définit l'angle de rotation (en degrés) des étiquettes par rapport à l'axe. Utile pour les textes longs.                                         |
| **Intervalle**                    | Définit l'espace (en pixels) entre les étiquettes consécutives en mode manuel.                                                                   |
| **Zones**                         | Lorsqu'activé, inclut des étiquettes pour les valeurs minimale et maximale de l'axe.                                                             |
| **Valeur minimale de l'axe**      | Spécifie la valeur minimale de l'axe à laquelle une étiquette doit être attribuée.                                                               |
| **Valeur maximale de l'axe**      | Spécifie la valeur maximale de l'axe à laquelle une étiquette doit être attribuée.                                                               |

***

## **Intégration du fournisseur de données**

| **Propriété**                                               | **Description**                                                                                                                 |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **Afficher le contenu du fournisseur de données sur l'axe** | Active ou désactive l'utilisation d'un fournisseur de données pour fournir les étiquettes de manière dynamique.                 |
| **Fournisseur de données**                                  | Sélectionnez un fournisseur de données représentant une connexion à une source de données (par ex., base de données, XML, RSS). |
| **Colonne**                                                 | Choisissez la colonne dans le fournisseur de données sélectionné qui contient les valeurs des étiquettes.                       |

> **Remarque**: Les fournisseurs de données sont définis au **niveau de la présentation** et sont disponibles pour une utilisation dans tous les éléments de votre présentation. Vous pouvez créer un nouveau fournisseur de données directement depuis cette fenêtre en utilisant le bouton adjacent. Référez-vous à **Travailler avec les fournisseurs de données** pour plus de détails.

***

## **Remarques**

* Toutes les propriétés décrites ci-dessus pour l'**axe X** sont également applicables à l'**axe Y et à l'axe Z**.
* Lors de l'utilisation du **mode d'étiquetage automatique**, le système s'assure que les étiquettes sont lisibles et bien espacées, en s'ajustant automatiquement en fonction de l'espace disponible.
* **Mode d'étiquetage manuel** est idéal lorsqu'un contrôle précis des positions des étiquettes est nécessaire, par exemple pour des intervalles fixes ou des catégories prédéfinies.
