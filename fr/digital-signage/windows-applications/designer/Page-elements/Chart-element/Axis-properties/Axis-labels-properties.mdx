Ces propriétés définissent comment les **labels** sont affichés le long des **X-axis, Y-axis, et Z-axis** d'un élément graphique. Les labels aident les utilisateurs à comprendre ce que représente chaque axe et peuvent être entièrement personnalisés pour la clarté visuelle et la précision des données.

***

## **Paramètres des labels**

| **Propriété**                    | **Description**                                                                                                                                  |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Police**                       | Spécifie la police et le style des étiquettes d'axe. Cliquez sur le bouton à côté du champ de police pour ouvrir la boîte de dialogue de police. |
| **Couleur**                      | Définit la couleur des étiquettes d'axe pour un meilleur contraste et une meilleure lisibilité.                                                  |
| **Mode d'étiquette automatique** | Détermine automatiquement le nombre d'étiquettes qui s'adapteront à l'axe et ajuste dynamiquement leur placement.                                |
| **Mode d'étiquette manuel**      | Définissez manuellement le nombre et le positionnement des étiquettes pour un contrôle précis.                                                   |
| **Angle**                        | Définit l'angle de rotation (en degrés) des étiquettes par rapport à l'axe. Utile pour adapter les textes longs.                                 |
| **Intervalle**                   | Définit l'espace (en pixels) entre les étiquettes consécutives en mode manuel.                                                                   |
| **Zones**                        | Lorsqu'activé, inclut des étiquettes pour les valeurs minimales et maximales de l'axe.                                                           |
| **Valeur minimale de l'axe**     | Spécifie la valeur minimale de l'axe à laquelle une étiquette doit être attribuée.                                                               |
| **Valeur maximale de l'axe**     | Spécifie la valeur maximale de l'axe à laquelle une étiquette doit être attribuée.                                                               |

***

## **Intégration du fournisseur de données**

| **Propriété**                                               | **Description**                                                                                                                       |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| **Afficher le contenu du fournisseur de données sur l'axe** | Active ou désactive l'utilisation d'un fournisseur de données pour fournir dynamiquement les étiquettes.                              |
| **Fournisseur de données**                                  | Sélectionnez un fournisseur de données qui représente une connexion à une source de données (par exemple, base de données, XML, RSS). |
| **Colonne**                                                 | Choisissez la colonne dans le fournisseur de données sélectionné qui contient les valeurs d'étiquette.                                |

> **Remarque**: Les fournisseurs de données sont définis au niveau de la **présentation** et sont disponibles pour une utilisation dans tous les éléments de votre présentation. Vous pouvez créer un nouveau fournisseur de données directement à partir de cette fenêtre en utilisant le bouton adjacent. Référez-vous à **Travailler avec les fournisseurs de données** pour plus de détails.

***

## **Remarques**

* Toutes les propriétés décrites ci-dessus pour l'**axe X** sont également applicables à l'**axe Y et l'axe Z**.
* Lors de l'utilisation du **Mode d'étiquette automatique**, le système garantit que les étiquettes sont lisibles et bien espacées, s'ajustant automatiquement en fonction de l'espace disponible.
* **Mode d'étiquette manuel** est idéal lorsqu'un contrôle précis des positions d'étiquettes est nécessaire, comme pour des intervalles fixes ou des catégories prédéfinies.
