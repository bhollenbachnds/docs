PADS4 Designer fournit une **large gamme d'éléments de page intégrés** pour créer des présentations professionnelles. Chaque type d'élément possède des **propriétés spécifiques**, ainsi que des **propriétés communes** partagées par tous les éléments.

En plus des éléments intégrés, les **Add-On elements** peuvent être installés et utilisés comme des éléments standard, offrant des fonctionnalités étendues.

***

## **Aperçu des éléments de page**

| **Page Element**              | **Description**                                                               |
| ----------------------------- | ----------------------------------------------------------------------------- |
| **Picture**                   | Affiche des logos, photos ou autres images.                                   |
| **Mixed Media**               | Affiche une playlist de vidéos et d'images.                                   |
| **Text**                      | Ajoute du texte brut.                                                         |
| **PowerPoint**                | Intègre des présentations Microsoft PowerPoint.                               |
| **Rich Text**                 | Ajoute du texte formaté.                                                      |
| **Video/Sound**               | Lecture de fichiers vidéo et/ou audio.                                        |
| **Web Page**                  | Affiche du contenu web ou des pages Internet.                                 |
| **Scrolling Text**            | Ajoute des bandeaux défilants ou du texte en mouvement continu.               |
| **Formatted Text**            | Affiche des valeurs dynamiques comme la date, l'heure ou les numéros de page. |
| **Shape**                     | Ajoute des formes comme des carrés, rectangles, cercles ou ovales.            |
| **Line**                      | Dessine des lignes.                                                           |
| **Chart**                     | Affiche des graphiques et diagrammes 2D ou 3D en temps réel.                  |
| **Table**                     | Présente des données en temps réel dans un format tabulaire.                  |
| **TV Element**                | Affiche des chaînes TV.                                                       |
| **Flash Player**              | Lit du contenu Flash.                                                         |
| **PDF Document Reader**       | Affiche des fichiers PDF directement sur la page.                             |
| **Zone Viewer**               | Créer une zone de contenu distincte contrôlée par son propre planificateur.   |
| **Présentation dans la Page** | Intégrer une autre présentation PADS4 dans la page actuelle.                  |
| **Bouton Interactif**         | Ajouter des boutons pour la navigation interactive dans les présentations.    |
| **Application Externe**       | Lancer un logiciel externe directement depuis la présentation.                |
| **Communication RS-232**      | Contrôler des appareils externes via des commandes RS-232.                    |

***

## **Aperçu des Éléments Additionnels**

Les modules complémentaires suivants sont inclus dans une **installation standard de PADS4**:

| **Élément Additionnel**    | **Description**                                                                                                     |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Barre de Progression**   | Affiche une barre indiquant le temps restant avant l'affichage de la page suivante.                                 |
| **Lecteur Multimédia DVD** | Lit des DVD et d'autres contenus multimédias.                                                                       |
| **TV Avancée**             | Affiche des chaînes de télévision en direct avec un contrôle avancé.                                                |
| **Code QR**                | Génère et affiche des codes QR (pour la numérisation avec des smartphones).                                         |
| **Sélecteur de Page**      | Affiche une page spécifique en fonction du contenu d'un fichier texte (par exemple, "10" pour afficher la page 10). |
| **Personnalisé (SDK)**     | Créer des éléments personnalisés via le **PADS4 SDK** pour des fonctionnalités et apparences spécifiques.           |

***

## **Points Clés**

* **Les éléments additionnels** fonctionnent comme des éléments intégrés — les utilisateurs ne voient aucune différence.
* Vous pouvez **vérifier les éléments additionnels installés** en naviguant vers **Éléments additionnels** dans l'application.
* En utilisant une combinaison d'**éléments standard** et d'**éléments additionnels**, vous pouvez concevoir des présentations hautement flexibles et dynamiques.
