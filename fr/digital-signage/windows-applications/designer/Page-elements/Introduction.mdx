PADS4 Designer fournit une **large gamme d'éléments de page intégrés** pour créer des présentations professionnelles. Chaque type d'élément possède des **propriétés spécifiques**, ainsi que des **propriétés communes** partagées par tous les éléments.

En plus des éléments intégrés, **Add-On elements** peuvent être installés et utilisés comme des éléments standard, offrant des fonctionnalités étendues.

***

## **Aperçu des éléments de page**

| **Élément de page**      | **Description**                                                                |
| ------------------------ | ------------------------------------------------------------------------------ |
| **Image**                | Afficher des logos, photos ou autres images.                                   |
| **Mixed Media**          | Afficher une liste de lecture de vidéos et d'images.                           |
| **Texte**                | Ajouter du texte brut.                                                         |
| **PowerPoint**           | Intégrer des présentations Microsoft PowerPoint.                               |
| **Rich Text**            | Ajouter du texte formaté.                                                      |
| **Vidéo/Son**            | Lecture de fichiers vidéo et/ou audio.                                         |
| **Page Web**             | Afficher du contenu web ou des pages Internet.                                 |
| **Scrolling Text**       | Ajouter des bandeaux défilants ou du texte en mouvement continu.               |
| **Formatted Text**       | Afficher des valeurs dynamiques comme la date, l'heure ou les numéros de page. |
| **Forme**                | Ajouter des formes comme des carrés, rectangles, cercles ou ovales.            |
| **Ligne**                | Dessiner des lignes.                                                           |
| **Graphique**            | Afficher des graphiques et diagrammes 2D ou 3D en temps réel.                  |
| **Tableau**              | Présenter des données en temps réel sous forme tabulaire.                      |
| **TV Element**           | Afficher des chaînes de télévision.                                            |
| **Flash Player**         | Lire du contenu Flash.                                                         |
| **PDF Document Reader**  | Afficher des fichiers PDF directement sur la page.                             |
| **Zone Viewer**          | Créer une zone de contenu séparée contrôlée par son propre planificateur.      |
| **Presentation in Page** | Intégrer une autre présentation PADS4 dans la page actuelle.                   |
| **Interactive Button**   | Ajouter des boutons pour la navigation interactive dans les présentations.     |
| **External Application** | Lancer des logiciels externes directement depuis la présentation.              |
| **RS-232 Communication** | Contrôler des appareils externes via des commandes RS-232.                     |

***

## **Aperçu des éléments Add-On**

Les add-ons suivants sont inclus dans une **installation standard PADS4** :

| **Élément Add-On**   | **Description**                                                                                                     |
| -------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Progress Bar**     | Affiche une barre indiquant le temps restant avant l'affichage de la page suivante.                                 |
| **DVD Media Player** | Lit des DVD et autres contenus multimédias.                                                                         |
| **Advanced TV**      | Affiche des chaînes de télévision en direct avec un contrôle avancé.                                                |
| **QR Code**          | Génère et affiche des codes QR (pour la numérisation avec des smartphones).                                         |
| **Page Selector**    | Affiche une page spécifique en fonction du contenu d'un fichier texte (par exemple, "10" pour afficher la page 10). |
| **Custom (SDK)**     | Créer des éléments personnalisés via le **PADS4 SDK** pour des fonctionnalités et apparences spécifiques.           |

***

## **Points clés**

* **Add-On elements** fonctionnent comme des éléments intégrés — les utilisateurs ne voient aucune différence.
* Vous pouvez **vérifier les éléments Add-On installés** en naviguant vers **Add-on elements** dans l'application.
* En utilisant une combinaison d'éléments **standard** et **add-on**, vous pouvez concevoir des présentations hautement flexibles et dynamiques.
