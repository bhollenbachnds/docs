PADS4 Designer fournit **une large gamme d'éléments de page intégrés** pour créer des présentations professionnelles. Chaque type d'élément possède **des propriétés spécifiques**, ainsi que **des propriétés communes** partagées par tous les éléments.

En plus des éléments intégrés, **des éléments Add-On** peuvent être installés et utilisés comme des éléments standard, offrant des fonctionnalités étendues.

***

## **Aperçu des éléments de page**

| **Élément de page**      | **Description**                                                                      |
| ------------------------ | ------------------------------------------------------------------------------------ |
| **Picture**              | Afficher des logos, des photos ou d'autres images.                                   |
| **Mixed Media**          | Afficher une liste de lecture de vidéos et d'images.                                 |
| **Text**                 | Ajouter du texte brut.                                                               |
| **PowerPoint**           | Intégrer des présentations Microsoft PowerPoint.                                     |
| **Rich Text**            | Ajouter du texte formaté.                                                            |
| **Video/Sound**          | Lecture de fichiers vidéo et/ou audio.                                               |
| **Web Page**             | Afficher du contenu web ou des pages Internet.                                       |
| **Scrolling Text**       | Ajouter des bandeaux défilants ou du texte en mouvement continu.                     |
| **Formatted Text**       | Afficher des valeurs dynamiques telles que la date, l'heure ou les numéros de page.  |
| **Shape**                | Ajouter des formes telles que des carrés, des rectangles, des cercles ou des ovales. |
| **Line**                 | Tracer des lignes.                                                                   |
| **Chart**                | Afficher des graphiques et des diagrammes 2D ou 3D en temps réel.                    |
| **Table**                | Présenter des données en temps réel dans un format tabulaire.                        |
| **TV Element**           | Afficher des chaînes de télévision.                                                  |
| **Flash Player**         | Lire du contenu Flash.                                                               |
| **PDF Document Reader**  | Afficher des fichiers PDF directement sur la page.                                   |
| **Zone Viewer**          | Créer une zone de contenu distincte contrôlée par son propre planificateur.          |
| **Presentation in Page** | Intégrer une autre présentation PADS4 dans la page actuelle.                         |
| **Interactive Button**   | Ajouter des boutons pour la navigation interactive dans les présentations.           |
| **External Application** | Lancer des logiciels externes directement depuis la présentation.                    |
| **RS-232 Communication** | Contrôler des appareils externes via des commandes RS-232.                           |

***

## **Aperçu des éléments Add-On**

Les add-ons suivants sont inclus dans une **installation PADS4 standard**:

| **Élément Add-On**   | **Description**                                                                                                    |
| -------------------- | ------------------------------------------------------------------------------------------------------------------ |
| **Progress Bar**     | Affiche une barre indiquant le temps restant avant l'affichage de la page suivante.                                |
| **DVD Media Player** | Lit des DVD et d'autres contenus multimédias.                                                                      |
| **Advanced TV**      | Afficher des chaînes de télévision en direct avec un contrôle avancé.                                              |
| **QR Code**          | Générer et afficher des codes QR (pour la lecture avec des smartphones).                                           |
| **Page Selector**    | Afficher une page spécifique en fonction du contenu d'un fichier texte (par ex., « 10 » pour afficher la page 10). |
| **Custom (SDK)**     | Créer des éléments personnalisés via le **PADS4 SDK** pour des fonctionnalités et une apparence spécifiques.       |

***

## **Points clés**

* **Les éléments Add-On** fonctionnent comme des éléments intégrés — les utilisateurs ne voient aucune différence.
* Vous pouvez **vérifier les éléments Add-On installés** en accédant à **Add-on elements** dans l'application.
* En utilisant une combinaison d'**éléments standard** et d'**éléments add-on**, vous pouvez concevoir des présentations très flexibles et dynamiques.
