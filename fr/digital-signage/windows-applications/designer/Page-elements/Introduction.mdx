PADS4 Designer fournit une **large gamme d'éléments de page intégrés** pour créer des présentations professionnelles. Chaque type d'élément possède des **propriétés spécifiques**, ainsi que des **propriétés communes** partagées par tous les éléments.

En plus des éléments intégrés, les **Add-On elements** peuvent être installés et utilisés comme des éléments standard, offrant des fonctionnalités étendues.

***

## **Aperçu des éléments de page**

| **Page Element**              | **Description**                                                                |
| ----------------------------- | ------------------------------------------------------------------------------ |
| **Picture**                   | Afficher des logos, photos ou autres images.                                   |
| **Mixed Media**               | Afficher une playlist de vidéos et d'images.                                   |
| **Text**                      | Ajouter du texte brut.                                                         |
| **PowerPoint**                | Intégrer des présentations Microsoft PowerPoint.                               |
| **Rich Text**                 | Ajouter du texte formaté.                                                      |
| **Video/Sound**               | Lecture de fichiers vidéo et/ou audio.                                         |
| **Web Page**                  | Afficher du contenu web ou des pages Internet.                                 |
| **Scrolling Text**            | Ajouter des bandeaux défilants ou du texte en mouvement continu.               |
| **Formatted Text**            | Afficher des valeurs dynamiques comme la date, l'heure ou les numéros de page. |
| **Shape**                     | Ajouter des formes comme des carrés, rectangles, cercles ou ovales.            |
| **Line**                      | Dessiner des lignes.                                                           |
| **Chart**                     | Afficher des graphiques et diagrammes 2D ou 3D en temps réel.                  |
| **Table**                     | Présenter des données en temps réel dans un format tabulaire.                  |
| **TV Element**                | Afficher des chaînes TV.                                                       |
| **Flash Player**              | Lire du contenu Flash.                                                         |
| **Lecteur de documents PDF**  | Affiche les fichiers PDF directement sur la page.                              |
| **Zone Viewer**               | Créez une zone de contenu séparée contrôlée par son propre planificateur.      |
| **Présentation dans la Page** | Intégrez une autre présentation PADS4 dans la page actuelle.                   |
| **Interactive Button**        | Ajoutez des boutons pour la navigation interactive dans les présentations.     |
| **Application Externe**       | Lancez des logiciels externes directement depuis la présentation.              |
| **Communication RS-232**      | Contrôlez des appareils externes via des commandes RS-232.                     |

***

## **Aperçu des éléments additionnels**

Les modules complémentaires suivants sont inclus dans une **installation standard PADS4**:

| **Élément additionnel** | **Description**                                                                                                     |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Progress Bar**        | Affiche une barre indiquant le temps restant avant l'affichage de la page suivante.                                 |
| **DVD Media Player**    | Lit des DVD et d'autres contenus multimédias.                                                                       |
| **Advanced TV**         | Affiche des chaînes de télévision en direct avec un contrôle avancé.                                                |
| **QR Code**             | Génère et affiche des codes QR (pour la numérisation avec des smartphones).                                         |
| **Page Selector**       | Affiche une page spécifique en fonction du contenu d'un fichier texte (par exemple, "10" pour afficher la page 10). |
| **Custom (SDK)**        | Créez des éléments personnalisés via le **PADS4 SDK** pour des fonctionnalités et apparences spécifiques.           |

***

## **Points clés**

* **Add-On elements** fonctionnent comme des éléments intégrés — les utilisateurs ne voient aucune différence.
* Vous pouvez **vérifier les éléments additionnels installés** en naviguant vers **Add-on elements** dans l'application.
* En utilisant une combinaison d'**éléments standard** et d'**éléments additionnels**, vous pouvez concevoir des présentations hautement flexibles et dynamiques.
