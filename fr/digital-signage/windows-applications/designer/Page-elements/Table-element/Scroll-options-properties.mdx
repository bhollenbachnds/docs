Le **scroll options** définissent comment **data scrolling** est géré dans l'élément tableau, y compris **non-scrolling rows, grouped data display, and automatic page transitions**.

***

## **Scroll Data Once and Show the Next Page**

* **Enabled:** Le tableau fait défiler toutes les **data rows once**. Après l'affichage de la dernière ligne, la présentation **automatically switches to the next page**.
* **Disabled:** Le tableau continue de défiler **indefinitely**, en bouclant à travers les données disponibles.

> **Best Use Case:**
>
> * Lorsque le nombre de **data rows is unknown**, mais que vous devez vous assurer que **each row is displayed exactly once** avant de passer à la page suivante.

***

## **Don't Scroll the First X Rows**

* **Define a fixed number of rows** qui **always remain visible** pendant que les autres lignes défilent.
* Utile pour **displaying critical or high-priority information** en haut du tableau.

> **Example Use Case:**
>
> * **Airports**: Les **first five rows** restent statiques pour afficher les vols qui partent bientôt, tandis que les autres vols défilent en dessous.
> * **Stock Markets**: Les **top companies** restent visibles, avec **less critical stocks scrolling**.

***

## **Scroll Data, Group By**

* Affiche **grouped data sets** un à la fois avant de passer au groupe suivant.
* **Select a column** qui contient les informations de regroupement.
* Le **Data Provider must already be sorted** par la colonne de groupe sélectionnée.

> **Example Use Case:**
>
> * **Hotel Room Reservations**: Affiche d'abord **Room 101 bookings**, puis passe à **Room 102**, puis **Room 103**, etc.
> * **Company Sales Reports**: Affiche les données de vente **grouped by department**, en s'assurant que les données de chaque département sont affichées séparément.

***

## **Important Notes**

* **Each element in a row must have data scrolling enabled** pour qu'il défile dans le tableau.
* **Sorting must be configured within the Data Provider settings**—l'élément tableau ne **not** trie pas les données automatiquement.
