Les **options de défilement** définissent comment **le défilement des données** est géré dans l'élément de tableau, y compris **les lignes non défilantes, l'affichage des données groupées et les transitions automatiques de pages**.

***

## **Faire défiler les données une fois et afficher la page suivante**

* **Enabled:** Le tableau fait défiler toutes les **lignes de données une fois**. Après l'affichage de la dernière ligne, la présentation **passe automatiquement à la page suivante**.
* **Disabled:** Le tableau continue de défiler **indéfiniment**, en bouclant à travers les données disponibles.

> **Meilleur cas d'utilisation :**
>
> * Lorsque le nombre de **lignes de données est inconnu**, mais vous devez vous assurer **que chaque ligne est affichée exactement une fois** avant de passer à la page suivante.

***

## **Ne pas faire défiler les X premières lignes**

* **Définir un nombre fixe de lignes** qui **resteront toujours visibles** pendant que les autres lignes défilent.
* Utile pour **afficher des informations critiques ou de haute priorité** en haut du tableau.

> **Exemple de cas d'utilisation :**
>
> * **Aéroports** : Les **cinq premières lignes** restent statiques pour afficher les vols qui partent bientôt, tandis que les autres vols défilent en dessous.
> * **Marchés boursiers** : Les **principales entreprises** restent visibles, avec **les actions moins critiques qui défilent**.

***

## **Défilement des données, Regroupement par**

* Affiche **des ensembles de données groupées** un à la fois avant de passer au groupe suivant.
* **Sélectionnez une colonne** qui contient les informations de regroupement.
* Le **Fournisseur de données doit déjà être trié** par la colonne de groupe sélectionnée.

> **Exemple de cas d'utilisation :**
>
> * **Réservations de chambres d'hôtel** : Affiche d'abord **les réservations de la Chambre 101**, puis passe à **la Chambre 102**, puis **la Chambre 103**, etc.
> * **Rapports de ventes d'entreprise** : Affiche les données de ventes **regroupées par département**, garantissant que les données de chaque département sont affichées séparément.

***

## **Remarques importantes**

* **Chaque élément d'une ligne doit avoir le défilement des données activé** pour qu'il puisse défiler dans le tableau.
* **Le tri doit être configuré dans les paramètres du Fournisseur de données**—l'élément de tableau ne **trie pas** les données automatiquement.
