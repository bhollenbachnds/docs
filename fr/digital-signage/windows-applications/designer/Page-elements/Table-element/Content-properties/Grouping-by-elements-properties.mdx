Le **regroupement par propriétés** vous permet d'organiser les lignes du tableau en **masquant les valeurs en double** dans les colonnes sélectionnées. Lorsque **deux lignes consécutives ou plus** contiennent la même valeur pour un **élément groupé**, cette valeur n'est **affichée que dans la première ligne**.

⚠ **Important:** Cette fonctionnalité **ne** regroupe pas réellement les données dans le tableau comme l'option **"Scroll Data, Grouped By"**. Au lieu de cela, elle **masque simplement les valeurs en double** pour améliorer la lisibilité.

***

## **Définir les éléments Group By dans un tableau**

1. Ouvrez la fenêtre **Table Properties**.
2. Accédez à la section **Row Properties**.
3. Cliquez sur **Edit** pour modifier la **Row Definition**.
4. Dans l'**onglet Home** de la barre de ruban, cliquez sur **Group By**.
5. Sélectionnez les **éléments** par lesquels vous souhaitez regrouper.
6. (Facultatif) Utilisez **Move Up** ou **Move Down** pour ajuster la **priorité de regroupement**.
7. Cliquez sur **OK** pour enregistrer les **paramètres de groupe**.
8. Dans le **menu File**, cliquez sur **Close**.
9. Cliquez sur **Yes** pour confirmer les modifications.
10. Cliquez sur **OK** ou **Apply** pour mettre à jour la présentation.

***

## **Options de regroupement supplémentaires**

* Si vous avez besoin d'**afficher les données regroupées ensemble** plutôt que de simplement **masquer les valeurs en double**, utilisez l'option **"Scroll Data, Grouped By"** dans **Scroll Options**.

***

## **Remarques et bonnes pratiques**

* **Tous les éléments** de la définition de ligne **peuvent être utilisés pour le regroupement**, mais **les éléments texte et les éléments image** sont les plus **couramment utilisés**.
* **Lorsque plusieurs éléments sont regroupés**, le système les traite dans l'**ordre de priorité** :
  * Il vérifie d'abord le **premier élément group by**.
  * Si sa valeur **correspond à la ligne précédente**, elle est **masquée**.
  * Le système vérifie ensuite l'**élément group by suivant**, en continuant jusqu'à ce que :
    * Une **valeur différente soit trouvée**, ou
    * Tous les **éléments group by** sélectionnés aient été traités.
* **Il n'y a pas de limite** au nombre d'**éléments group by** que vous pouvez définir.
