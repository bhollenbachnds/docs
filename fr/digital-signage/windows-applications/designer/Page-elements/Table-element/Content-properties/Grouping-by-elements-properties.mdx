Le **regroupement par propriétés** vous permet d'organiser les lignes du tableau en **masquant les valeurs en double** dans les colonnes sélectionnées. Lorsque **deux lignes consécutives ou plus** contiennent la même valeur pour un **élément groupé**, cette valeur est **affichée uniquement dans la première ligne**.

⚠ **Important:** Cette fonctionnalité **ne** regroupe pas réellement les données dans le tableau comme l'option **"Scroll Data, Grouped By"**. Au lieu de cela, elle **masque simplement les valeurs en double** pour améliorer la lisibilité.

***

## **Définition des éléments Group By dans un tableau**

1. Ouvrez la fenêtre **Propriétés du tableau**.
2. Accédez à la section **Propriétés de ligne**.
3. Cliquez sur **Modifier** pour modifier la **définition de ligne**.
4. Dans l'**onglet Accueil** du ruban, cliquez sur **Group By**.
5. Sélectionnez les **éléments** par lesquels vous souhaitez regrouper.
6. (Facultatif) Utilisez **Monter** ou **Descendre** pour ajuster la **priorité de regroupement**.
7. Cliquez sur **OK** pour enregistrer les **paramètres de regroupement**.
8. Dans le **menu Fichier**, cliquez sur **Fermer**.
9. Cliquez sur **Oui** pour confirmer les modifications.
10. Cliquez sur **OK** ou **Appliquer** pour mettre à jour la présentation.

***

## **Options de regroupement supplémentaires**

* Si vous avez besoin d'**afficher les données regroupées ensemble** plutôt que de simplement **masquer les valeurs en double**, utilisez l'option **"Scroll Data, Grouped By"** dans **Options de défilement**.

***

## **Remarques et bonnes pratiques**

* **Tous les éléments** de la définition de ligne **peuvent être utilisés pour le regroupement**, mais **les éléments texte et les éléments image** sont les plus **couramment utilisés**.
* **Lorsque plusieurs éléments sont regroupés**, le système les traite dans l'**ordre de priorité** :
  * Il vérifie d'abord le **premier élément group by**.
  * Si sa valeur **correspond à la ligne précédente**, elle est **masquée**.
  * Le système vérifie ensuite l'**élément group by suivant**, en continuant jusqu'à ce que :
    * Une **valeur différente soit trouvée**, ou
    * Tous les **éléments group by** sélectionnés aient été traités.
* **Il n'y a pas de limite** au nombre d'**regrouper par éléments** vous pouvez définir.
