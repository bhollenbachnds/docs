Ces propriétés déterminent comment votre élément **gère le texte multiligne** lorsqu'il est connecté à une **source de données externe**.

***

## **Options de gestion du texte**

### **Diviser le texte multiligne en éléments de liste de lecture**

* **Activé** – Chaque **ligne de texte** de la source de données est traitée comme un **élément de liste de lecture distinct**, permettant un **affichage alterné**.
* **Désactivé** – La **valeur textuelle entière** est traitée comme **un seul élément de liste de lecture**, affichant toutes les lignes ensemble.

***

## **Fonctionnement**

Si une **valeur de texte multiligne** est reçue d'une **source de données**, ce paramètre contrôle si :

1. **Chaque ligne est affichée séparément** (comme un bandeau d'actualités défilant ou des annonces en rotation).
2. **Toutes les lignes sont affichées ensemble** dans un seul bloc de texte.

***

## **Exemple d'utilisation**

### **Scénario 1 : Annonces de vol (Activé)**

**Entrée de la source de données :**

```
Boarding  
Einsteigen
```

* **Avec la division activée :** Chaque mot apparaît **individuellement** dans la liste de lecture (par ex., "Boarding" → "Einsteigen").
* **Avec la division désactivée :** Les deux lignes sont affichées **ensemble** dans un seul bloc de texte.

***

### **Scénario 2 : Liste de lecture d'images (Activé)**

**Entrée de la source de données :**

```
c:\folder\image1.jpg  
c:\folder\image2.jpg  
```

* **Sans division :** L'élément image **ne peut pas lire** l'intégralité de l'entrée comme un chemin valide.
* **Avec division :** L'élément **traite chaque ligne séparément**, permettant à une **liste de lecture d'images** de s'afficher en séquence.

***

## **Remarques importantes**

* Le **séparateur entre les lignes de texte** est le **caractère de retour chariot et saut de ligne** (`<CR> <LF>`).
* Cette fonctionnalité est utile lorsque vous travaillez avec des **listes d'images, de messages ou de flux de données en temps réel**.
* Fonctionne en combinaison avec les **options de liste de lecture** pour définir la façon dont les éléments sont affichés.
