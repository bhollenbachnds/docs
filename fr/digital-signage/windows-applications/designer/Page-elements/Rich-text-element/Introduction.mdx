L'élément **Rich Text** permet d'afficher du texte formaté, y compris du texte en gras, italique et souligné, ainsi que des styles avancés tels que **les ombres, les rotations et la transparence**. Cet élément peut s'intégrer à des **sources de données externes** pour des mises à jour de contenu en temps réel et prend en charge le collage direct de texte depuis **Microsoft Word**.

***

### **Aperçu des propriétés**

| **Section de propriété**                     | **Description**                                                                                                                                                         |
| -------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Options alternatives**                     | Définit comment l'élément bascule entre les éléments de la liste de lecture.                                                                                            |
| **Couleur d'arrière-plan**                   | Définit la couleur d'arrière-plan de l'élément.                                                                                                                         |
| **Clignotement**                             | Configure un effet de clignotement pour l'élément.                                                                                                                      |
| **Bordure**                                  | Ajuste les paramètres de bordure de l'élément.                                                                                                                          |
| **Fournisseur de données**                   | Connecte l'élément à une source de données externe pour du contenu en temps réel.                                                                                       |
| **Défilement des données**                   | Définit comment l'élément fait défiler les données en temps réel.                                                                                                       |
| **Conversion de texte de source de données** | Spécifie comment le texte multiligne provenant de sources externes est traité.                                                                                          |
| **Durée par défaut**                         | Définit la durée d'affichage des éléments de la liste de lecture.                                                                                                       |
| **Effet de transition par défaut**           | Définit l'effet de transition par défaut pour les changements de contenu.                                                                                               |
| **Retournement**                             | Retourne le contenu horizontalement ou verticalement.                                                                                                                   |
| **Police**                                   | Configure les styles, tailles et couleurs de police pour l'affichage du texte.                                                                                          |
| **Identification**                           | Attribue un identifiant unique à l'élément dans la page.                                                                                                                |
| **Liste de lecture**                         | Gère une liste d'éléments de texte enrichi à afficher.                                                                                                                  |
| **Options de liste de lecture**              | Définit comment les éléments de la liste de lecture sont lus.                                                                                                           |
| **Éditeur avancé**                           | Fournit des options supplémentaires telles que **les connexions aux fournisseurs de données, les variables de visualisation et l'automatisation basée sur des règles**. |
| **Position**                                 | Spécifie l'emplacement de l'élément sur la page.                                                                                                                        |
| **Proof of Play**                            | Enregistre les données de lecture pour l'analyse et la vérification.                                                                                                    |
| **Property Bindings**                        | Permet aux sources de données externes de mettre à jour des propriétés spécifiques en temps réel.                                                                       |
| **Reflection**                               | Affiche le texte avec un effet de réflexion.                                                                                                                            |
| **Rotation**                                 | Fait pivoter l'élément selon un angle spécifié.                                                                                                                         |
| **Rules**                                    | Automatise le comportement des éléments selon des conditions prédéfinies.                                                                                               |
| **Shadow**                                   | Applique des effets d'ombre au texte.                                                                                                                                   |
| **Size**                                     | Définit la largeur et la hauteur de l'élément.                                                                                                                          |
| **Skew**                                     | Incline le texte selon un angle.                                                                                                                                        |
| **Text Alignment**                           | Définit l'alignement horizontal et vertical du texte.                                                                                                                   |
| **Transparency**                             | Ajuste le niveau de transparence de l'élément.                                                                                                                          |
| **Visibility**                               | Détermine si et quand l'élément doit être visible.                                                                                                                      |

***

### **Remarques**

* Cette section couvre uniquement **Rich Text-specific** propriétés. Pour les **propriétés générales des éléments**, référez-vous à la section **Page Element Properties**.
* **Les mises à jour de contenu en temps réel** sont prises en charge via **l'intégration de fournisseurs de données**.
* L'**Advanced Editor** permet aux utilisateurs d'appliquer **un formatage de texte complexe, des mises à jour de texte pilotées par les données et des comportements basés sur des règles**.
