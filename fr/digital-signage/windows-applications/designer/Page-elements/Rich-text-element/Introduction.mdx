L'**Rich Text** permet l'affichage de texte formaté, notamment en gras, en italique et souligné, ainsi que des styles avancés tels que **ombres, rotations et transparence**. Cet élément peut s'intégrer à des **sources de données externes** pour des mises à jour de contenu en temps réel et prend en charge le collage direct de texte depuis **Microsoft Word**.

***

### **Aperçu des propriétés**

| **Section de propriété**        | **Description**                                                                                                                                        |
| ------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Alternate Options**           | Définit comment l'élément bascule entre les éléments de la playlist.                                                                                   |
| **Background Color**            | Définit la couleur d'arrière-plan de l'élément.                                                                                                        |
| **Blinking**                    | Configure un effet de clignotement pour l'élément.                                                                                                     |
| **Border**                      | Ajuste les paramètres de bordure de l'élément.                                                                                                         |
| **Data Provider**               | Connecte l'élément à une source de données externe pour du contenu en temps réel.                                                                      |
| **Data Scrolling**              | Définit comment l'élément fait défiler les données en temps réel.                                                                                      |
| **Data Source Text Conversion** | Spécifie comment le texte multiligne provenant de sources externes est traité.                                                                         |
| **Default Duration**            | Définit la durée d'affichage des éléments de la playlist.                                                                                              |
| **Default Transition Effect**   | Définit l'effet de transition par défaut pour les changements de contenu.                                                                              |
| **Flip**                        | Retourne le contenu horizontalement ou verticalement.                                                                                                  |
| **Font**                        | Configure les styles de police, les tailles et les couleurs pour l'affichage du texte.                                                                 |
| **Identification**              | Attribue un identifiant unique à l'élément dans la page.                                                                                               |
| **Play List**                   | Gère une liste d'éléments de texte enrichi à afficher.                                                                                                 |
| **Play List Options**           | Définit comment les éléments de la playlist sont lus.                                                                                                  |
| **Advanced Editor**             | Fournit des options supplémentaires telles que **connexions aux fournisseurs de données, variables de viewer et automatisation basée sur des règles**. |
| **Position**                    | Spécifie l'emplacement de l'élément sur la page.                                                                                                       |
| **Proof of Play**               | Enregistre les données de lecture à des fins d'analyse et de vérification.                                                                             |
| **Property Bindings**           | Permet aux sources de données externes de mettre à jour des propriétés spécifiques en temps réel.                                                      |
| **Reflection**                  | Affiche le texte avec un effet de reflet.                                                                                                              |
| **Rotation**                    | Fait pivoter l'élément selon un angle spécifié.                                                                                                        |
| **Rules**                       | Automatise le comportement des éléments en fonction de conditions prédéfinies.                                                                         |
| **Shadow**                      | Applique des effets d'ombre au texte.                                                                                                                  |
| **Size**                        | Définit la largeur et la hauteur de l'élément.                                                                                                         |
| **Skew**                        | Incline le texte selon un angle.                                                                                                                       |
| **Text Alignment**              | Définit l'alignement horizontal et vertical du texte.                                                                                                  |
| **Transparency**                | Ajuste le niveau de transparence de l'élément.                                                                                                         |
| **Visibility**                  | Détermine si et quand l'élément doit être visible.                                                                                                     |

***

### **Remarques**

* Cette section couvre uniquement les **propriétés spécifiques au Rich Text**. Pour les **propriétés générales des éléments**, référez-vous à la section **Page Element Properties**.
* **Mises à jour de contenu en temps réel** sont prises en charge via l'**intégration de fournisseur de données**.
* L'**Advanced Editor** permet aux utilisateurs d'appliquer des **mises en forme de texte complexes, des mises à jour de texte pilotées par les données et des comportements basés sur des règles**.
