The **Font Dialog Box** allows users to select a font and adjust text presentation properties, including size, spacing, and various text styles.

***

### **Search Font**

* **Search Bar** → Enter a font name to quickly locate it.
* **Fonts List** → Highlights the first matching font as you type.

***

### **Font Selection & Installation**

| **Option**                  | **Description**                                                                                                         |
| --------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| **PADS4 Fonts**             | Displays all installed fonts. Fonts with a special indicator are installed via **PADS4 System Management**.             |
| **PADS4 System Management** | Allows additional fonts to be installed and distributed across **HTML5 Viewers**, ensuring availability on all devices. |

***

### **Font Customization**

| **Property**                                             | **Description**                                                              |
| -------------------------------------------------------- | ---------------------------------------------------------------------------- |
| **PADS4 Spacing**                                        | Adjusts vertical **line spacing** between text lines.                        |
| **PADS4 Size**                                           | Defines the **font size**.                                                   |
| **PADS4 Typeface**                                       | Selects the **font style** (e.g., Regular, Bold, Italic).                    |
| **PADS4 Underscore, Overline, Strikethrough, Underline** | Adds text decorations such as **underlines, overlines, and strikethroughs**. |

***

### **Preview**

| **Option**         | **Description**                                             |
| ------------------ | ----------------------------------------------------------- |
| **PADS4 New Font** | Displays a preview of the font with selected modifications. |
| **Current Font**   | Shows the **existing font settings** before any changes.    |

***

### **Remarks**

* **Font Availability:** Only fonts installed on **Viewers** will be displayed correctly during playback.
* **Automatic Distribution:** Fonts installed via **PADS4 System Management** are synced to all connected devices.
* **Dynamic Adjustments:** Custom spacing and decorations enhance text readability and presentation consistency.
