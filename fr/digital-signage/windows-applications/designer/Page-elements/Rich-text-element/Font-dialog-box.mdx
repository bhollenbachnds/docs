La **boîte de dialogue Police** permet aux utilisateurs de sélectionner une police et d'ajuster les propriétés de présentation du texte, notamment la taille, l'espacement et divers styles de texte.

***

### **Rechercher une police**

* **Barre de recherche** → Saisissez un nom de police pour la localiser rapidement.
* **Liste des polices** → Met en surbrillance la première police correspondante au fur et à mesure de la saisie.

***

### **Sélection et installation de polices**

| **Option**                  | **Description**                                                                                                                                        |
| --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **PADS4 Fonts**             | Affiche toutes les polices installées. Les polices avec un indicateur spécial sont installées via **PADS4 System Management**.                         |
| **PADS4 System Management** | Permet d'installer des polices supplémentaires et de les distribuer sur les **HTML5 Viewers**, garantissant leur disponibilité sur tous les appareils. |

***

### **Personnalisation des polices**

| **Propriété**                                            | **Description**                                                                        |
| -------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| **PADS4 Spacing**                                        | Ajuste l'espacement vertical **interligne** entre les lignes de texte.                 |
| **PADS4 Size**                                           | Définit la **taille de police**.                                                       |
| **PADS4 Typeface**                                       | Sélectionne le **style de police** (par ex., Normal, Gras, Italique).                  |
| **PADS4 Underscore, Overline, Strikethrough, Underline** | Ajoute des décorations de texte telles que **soulignements, surlignements et barrés**. |

***

### **Aperçu**

| **Option**          | **Description**                                                          |
| ------------------- | ------------------------------------------------------------------------ |
| **PADS4 New Font**  | Affiche un aperçu de la police avec les modifications sélectionnées.     |
| **Police actuelle** | Affiche les **paramètres de police existants** avant toute modification. |

***

### **Remarques**

* **Disponibilité des polices :** Seules les polices installées sur les **Viewers** s'afficheront correctement lors de la lecture.
* **Distribution automatique :** Les polices installées via **PADS4 System Management** sont synchronisées sur tous les appareils connectés.
* **Ajustements dynamiques :** Les espacements et décorations personnalisés améliorent la lisibilité du texte et la cohérence de la présentation.
