L'élément **PowerPoint element** vous permet d'afficher des **Microsoft PowerPoint presentations** dans votre présentation PADS4. Cet élément est **toujours au premier plan**, ce qui signifie que **d'autres éléments ne peuvent pas être superposés par-dessus**.

***

## **Exigences importantes**

* **Microsoft PowerPoint doit être installé** sur le PC Viewer.
* Seul **PowerPoint 2007 ou supérieur** est pris en charge.
* Cet élément **n'utilise pas PowerPoint Viewer (pptview\.exe).**

***

## **Aperçu des propriétés**

| **Section de propriété**                     | **Description**                                                                                        |
| -------------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| **Alignement**                               | Définit **l'alignement horizontal et vertical** de l'affichage PowerPoint.                             |
| **Options alternatives**                     | Détermine comment les **éléments de la playlist** changent.                                            |
| **Couleur d'arrière-plan**                   | Définit la **couleur d'arrière-plan** de l'élément PowerPoint.                                         |
| **Bordure**                                  | Définit les **paramètres de bordure** pour l'élément.                                                  |
| **Fournisseur de données**                   | Connecte l'élément à une **source de données externe** pour des mises à jour de contenu en temps réel. |
| **Conversion de texte de source de données** | Définit comment l'élément **gère le texte multiligne.**                                                |
| **Durée par défaut**                         | Spécifie **combien de temps chaque diapositive PowerPoint** est affichée.                              |
| **Identification**                           | Attribue un **nom unique** à l'élément pour référence.                                                 |
| **Liste de lecture**                         | Définit **quelles présentations PowerPoint** seront affichées.                                         |
| **Options de liste de lecture**              | Détermine **comment les éléments de la playlist sont lus.**                                            |
| **Position**                                 | Spécifie la **position de l'élément** sur la page.                                                     |
| **Proof of Play**                            | Enregistre des informations pour **suivre l'historique de lecture.**                                   |
| **Liaisons de propriétés**                   | Lie des **propriétés** spécifiques de l'élément à des **sources de données externes.**                 |
| **Règles**                                   | Permet à l'élément de **répondre automatiquement** à des conditions prédéfinies.                       |
| **Taille**                                   | Définit la **largeur et la hauteur** de l'élément PowerPoint.                                          |
| **Visibilité**                               | Contrôle si **l'élément est visible** pendant la lecture.                                              |

***

### **Remarques importantes**

* L'élément **PowerPoint ne peut pas être placé derrière d'autres éléments** (il restera toujours au-dessus).
* Si **Microsoft PowerPoint n'est pas installé**, l'élément **ne fonctionnera pas**.
* **Le suivi de lecture** est disponible via la fonctionnalité **Proof of Play**.
