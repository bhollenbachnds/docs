Le **PowerPoint element** vous permet d'afficher **Microsoft PowerPoint presentations** dans votre présentation PADS4. Cet élément est **toujours au premier plan**, ce qui signifie que **d'autres éléments ne peuvent pas être superposés par-dessus**.

***

## **Exigences importantes**

* **Microsoft PowerPoint doit être installé** sur le PC Viewer.
* Seul **PowerPoint 2007 ou supérieur** est pris en charge.
* Cet élément **n'utilise pas PowerPoint Viewer (pptview\.exe).**

***

## **Aperçu des propriétés**

| **Section de propriété**        | **Description**                                                                                        |
| ------------------------------- | ------------------------------------------------------------------------------------------------------ |
| **Alignment**                   | Définit **l'alignement horizontal et vertical** de l'affichage PowerPoint.                             |
| **Alternate Options**           | Détermine comment les **éléments de la playlist** changent.                                            |
| **Background Color**            | Définit la **couleur d'arrière-plan** de l'élément PowerPoint.                                         |
| **Border**                      | Définit les **paramètres de bordure** pour l'élément.                                                  |
| **Data Provider**               | Connecte l'élément à une **source de données externe** pour des mises à jour de contenu en temps réel. |
| **Data Source Text Conversion** | Définit comment l'élément **gère le texte multi-lignes.**                                              |
| **Default Duration**            | Spécifie **combien de temps chaque diapositive PowerPoint** est affichée.                              |
| **Identification**              | Attribue un **nom unique** à l'élément pour référence.                                                 |
| **Play List**                   | Définit **quelles présentations PowerPoint** seront affichées.                                         |
| **Play List Options**           | Détermine **comment les éléments de la playlist sont lus.**                                            |
| **Position**                    | Spécifie la **position de l'élément** sur la page.                                                     |
| **Proof of Play**               | Enregistre des informations pour **suivre l'historique de lecture.**                                   |
| **Property Bindings**           | Lie des **propriétés** spécifiques de l'élément à des **sources de données externes.**                 |
| **Rules**                       | Permet à l'élément de **répondre automatiquement** à des conditions prédéfinies.                       |
| **Size**                        | Définit la **largeur et la hauteur** de l'élément PowerPoint.                                          |
| **Visibility**                  | Contrôle si **l'élément est visible** pendant la lecture.                                              |

***

### **Remarques importantes**

* L'**élément PowerPoint ne peut pas être placé derrière d'autres éléments** (il restera toujours au-dessus).
* Si **Microsoft PowerPoint n'est pas installé**, l'élément **ne fonctionnera pas**.
* **Le suivi de lecture** est disponible via la fonctionnalité **Proof of Play**.
