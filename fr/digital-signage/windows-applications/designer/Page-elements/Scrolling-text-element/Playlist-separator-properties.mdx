Ces propriétés définissent le **séparateur d'image** placé entre les éléments de texte défilant.

***

## **Options de séparateur**

### **Content Library**

* Utilise une image de la **PADS4 content library** (recommandé).
* Les fichiers sont **automatiquement synchronisés** sur tous les PC connectés.
* Les références relatives garantissent que les fichiers restent accessibles **même si la bibliothèque est déplacée**.

### **File Name**

* Utilise une image stockée **en dehors** de la PADS4 content library.
* Les fichiers sont **non synchronisés automatiquement**, nécessitant une **distribution manuelle**.
* Les références de fichiers absolues signifient que l'image **doit exister** sur le PC de lecture.

**Sélection de fichier :**

* Cliquez sur **Browse** pour sélectionner une image depuis la **Content Library** ou le **système de fichiers** local.
* Si vous utilisez l'option **File Name**, le chemin du fichier doit être **entièrement spécifié**.

***

## **Conseils et bonnes pratiques**

* **Les séparateurs indiquent visuellement** le début d'un **nouvel élément de texte défilant**.
* Si l'image du séparateur est **mise à jour**, la présentation **utilisera automatiquement** la dernière version.
