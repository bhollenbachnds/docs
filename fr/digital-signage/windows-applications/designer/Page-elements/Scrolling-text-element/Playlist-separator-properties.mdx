Ces propriétés définissent le **séparateur d'image** placé entre les éléments de texte défilant.

***

## **Options de séparateur**

### **Bibliothèque de contenu**

* Utilise une image de la **bibliothèque de contenu PADS4** (recommandé).
* Les fichiers sont **automatiquement synchronisés** sur tous les PC connectés.
* Les références relatives garantissent que les fichiers restent accessibles **même si la bibliothèque est déplacée**.

### **Nom de fichier**

* Utilise une image stockée **en dehors** de la bibliothèque de contenu PADS4.
* Les fichiers **ne sont pas synchronisés automatiquement**, nécessitant une **distribution manuelle**.
* Les références de fichier absolues signifient que l'image **doit exister** sur le PC de lecture.

**Sélection de fichier :**

* Cliquez sur **Parcourir** pour sélectionner une image depuis la **Bibliothèque de contenu** ou le **système de fichiers** local.
* Si vous utilisez l'option **Nom de fichier**, le chemin du fichier doit être **entièrement spécifié**.

***

## **Conseils et bonnes pratiques**

* **Les séparateurs indiquent visuellement** le début d'un **nouvel élément de texte défilant**.
* Si l'image du séparateur est **mise à jour**, la présentation **utilisera automatiquement** la dernière version.
