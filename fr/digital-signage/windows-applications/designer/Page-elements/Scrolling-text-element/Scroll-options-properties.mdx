Ces propriétés contrôlent comment les **éléments de texte** apparaissent et se déplacent dans l'**élément de texte défilant**.

***

## **Options de défilement**

### **Options de défilement du texte**

Déterminez comment le nouveau texte est introduit dans l'élément de texte défilant :

| **Option**                             | **Description**                                                                                                                              |
| -------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| **Line by Line**                       | Affiche une ligne à la fois. Chaque ligne est affichée séparément, et la ligne suivante n'apparaît qu'après la disparition de la précédente. |
| **Directly After Previous Text**       | Le nouveau texte est placé **immédiatement après** le texte précédent, assurant un effet de défilement continu sans espaces.                 |
| **When Previous Text is Scrolled Out** | Le nouveau texte commence **seulement après** que le texte précédent ait **complètement disparu** de la vue.                                 |

***

### **Paramètres supplémentaires**

#### **Concatenate Multiple Text Lines to One Single Line**

* Si activé, les **retours à la ligne (sauts de ligne)** à l'intérieur d'une seule entrée de texte sont **supprimés**.
* Le texte entier est traité comme **une seule ligne continue** pendant le défilement.
* Ne peut pas être utilisé en combinaison avec le mode **Line by Line**.

#### **Do Not Scroll When the Text Fits**

* Lorsque cette option est activée, PADS4 calcule si le **texte entier (y compris les séparateurs)** tient dans la **zone de l'élément**.
* Si le texte tient, il restera **statique** au lieu de défiler.
* Si le texte **dépasse** la zone de l'élément, le défilement **démarrera automatiquement**.

#### **Show Changes in Text Immediately**

* Lorsqu'il est connecté à une **source de données**, l'activation de ce paramètre garantit que **toutes les mises à jour de texte sont reflétées immédiatement**.
* Le processus de défilement est **interrompu**, et le **le nouveau texte commence à défiler immédiatement**.

***

## **Remarques**

* **Mode "Ligne par Ligne"** ne peut pas être utilisé avec **"Concaténer plusieurs lignes de texte en une seule ligne"**, car ils **sont en conflit de comportement**.
* Assurer un **défilement fluide** nécessite une configuration soignée de **la longueur du texte, l'espacement des séparateurs et la fréquence de mise à jour**.
