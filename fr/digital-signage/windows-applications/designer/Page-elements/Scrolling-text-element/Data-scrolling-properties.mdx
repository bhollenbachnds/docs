Lorsqu'un **élément de texte défilant** est connecté à une **source de données**, ces propriétés définissent comment **les données en temps réel** défilent. **Le défilement des données** parcourt les lignes de données en boucle, tandis que **le défilement visuel** détermine comment le texte se déplace sur l'écran.

***

## **Paramètres de défilement des données**

### **Utiliser le défilement des données**

* Active ou désactive **le défilement des données en temps réel** dans l'élément.

### **Taille du pas de défilement**

* Définit le nombre de **lignes** dont l'élément avance à chaque actualisation des données.
* Example:
  * Si **Ligne = 2** et **Scrolling Step Size = 3**, l'élément affichera **Ligne 2, Ligne 5, Ligne 8**, et ainsi de suite.
  * Une fois toutes les lignes disponibles affichées, l'élément **revient au début** à la Ligne 2.

**Important:**

* Lorsque plusieurs éléments avec **défilement de données** sont placés dans une **ligne de tableau**, ils **doivent** avoir le **même Scrolling Step Size** pour rester synchronisés.

***

## **Défilement de données alternatif (pour les éléments de tableau uniquement)**

Lorsqu'un **élément de texte défilant** fait partie d'une **ligne de tableau**, des options supplémentaires sont disponibles :

### **Number of Rows**

* Limite le défilement aux **N premières lignes** de la source de données.
* Utile pour afficher uniquement les **actualités les plus récentes** depuis un **flux RSS** ou une base de données.

**Tip:**

* Si vous avez besoin d'afficher uniquement les **principales actualités**, utilisez **Number of Rows** pour limiter le défilement aux dernières entrées.

***

## **Notes supplémentaires**

* **Défilement de données** Les propriétés ne sont disponibles que si l'élément est **connecté à une source de données**.
* Les données affichées sont déterminées par :
  * **Propriété Ligne** (position de départ).
  * **Scroll Step Size** (nombre de lignes à ignorer par actualisation).
* Le processus de défilement **boucle** à travers toutes les lignes disponibles et **redémarre** en atteignant la fin.
