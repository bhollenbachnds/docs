Ces propriétés définissent **les éléments multimédias (images, vidéos, flux)** qui seront affichés dans votre élément Mixed Media. Vous pouvez créer un **affichage d'élément unique** ou une **liste de lecture** qui exécute les fichiers multimédias de manière séquentielle.

***

### **Gestion de la liste de lecture**

* **Ajouter**, **modifier**, **réorganiser** ou **supprimer** des éléments de votre liste de lecture.
* Les éléments de la liste de lecture peuvent être gérés en **vue détaillée** ou en **vue liste**.
* **Les boutons de navigation** en bas de la fenêtre de liste de lecture vous permettent de parcourir la liste.
* **Durée totale** affiche la durée cumulée de tous les éléments de la liste de lecture.

> **Important:**\
> Pour des conseils approfondis sur la gestion des listes de lecture, voir **Travailler avec les listes de lecture**.

***

### **Élément de liste de lecture : Options de type**

| **Type**                                    | **Description**                                                                                                                                                                                |
| ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Média depuis la bibliothèque de contenu** | Référence un **fichier image ou vidéo** existant stocké dans la **bibliothèque de contenu PADS4** (recommandé). Les fichiers sont **automatiquement synchronisés** sur tous les PC du système. |
| **Média depuis une URL**                    | Une **référence URL directe** vers une source multimédia (par ex., image ou vidéo hébergée sur le web).                                                                                        |
| **Flux vidéo**                              | Une **URL de flux** de référence (par ex., caméra IP ou service de diffusion en direct).                                                                                                       |
| **Séparateur**                              | Un élément de liste de lecture **vide**, utilisé pour créer une **pause** dans la séquence.                                                                                                    |

***

### **Propriétés de chaque élément de liste de lecture**

| **Propriété**                               | **Description**                                                                                                                 |
| ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **Média depuis la bibliothèque de contenu** | Contient la **référence de fichier relative** vers un élément de la bibliothèque de contenu.                                    |
| **Média depuis une URL**                    | Contient l'**URL** pointant vers la source multimédia.                                                                          |
| **Flux vidéo**                              | Contient la **référence d'URL de flux**.                                                                                        |
| **Durée**                                   | Durée (en secondes) pendant laquelle l'élément sera affiché. S'il s'agit d'une **vidéo**, sa durée naturelle est utilisée.      |
| **Effet**                                   | **Effet de transition** appliqué lorsque l'élément apparaît (par ex., fondu, glissement) et **durée de l'effet** (en secondes). |

***

### **Remarques clés**

* **Seules les références** aux fichiers multimédias sont incluses, **pas** le média lui-même. Cela permet des **mises à jour automatiques** pendant la lecture si le fichier source change.
* **Propriétés de la liste de lecture** sont **désactivées** lorsque l'élément est lié à une **source de données**. Dans ce cas, le contenu multimédia est contrôlé par des **données en temps réel**.
* **Les séparateurs** sont un moyen utile d'**insérer des pauses** sans média entre les éléments d'une séquence.
