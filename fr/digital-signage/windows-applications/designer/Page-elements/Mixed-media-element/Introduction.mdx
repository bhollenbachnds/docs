Les éléments Mixed Media sont utilisés pour afficher une **liste de lecture d'images et de vidéos**. Ils permettent des combinaisons flexibles de types de médias et offrent des options pour contrôler la lecture, l'apparence et l'interaction.

***

### **Aperçu des propriétés**

| **Section de propriété**           | **Description**                                                                                                                       |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| **Alignement**                     | Définissez l'**alignement horizontal et vertical** du contenu dans l'élément.                                                         |
| **Options alternatives**           | Définissez comment l'élément va **basculer entre les éléments de la liste de lecture** (par exemple, boucle, lecture aléatoire).      |
| **Couleur d'arrière-plan**         | Définissez la **couleur d'arrière-plan** de l'élément, y compris les dégradés et les motifs.                                          |
| **Bordure**                        | Définissez la **bordure** autour de l'élément, y compris la couleur et la largeur.                                                    |
| **Fournisseur de données**         | Connectez-vous à une **source de données externe** pour alimenter du contenu en temps réel (images/vidéos).                           |
| **Durée par défaut**               | Définissez la **durée d'affichage par défaut** pour les éléments de la liste de lecture si elle n'est pas spécifiée individuellement. |
| **Effet de transition par défaut** | Définissez les **effets de transition** pour la façon dont le nouveau contenu entre dans l'élément (par exemple, fondu, glissement).  |
| **Identification**                 | Identifiez de manière unique l'élément dans la page à des fins de référencement et d'automatisation.                                  |
| **Liste de lecture**               | Créez et gérez une **liste d'éléments multimédias** (images/vidéos) à lire.                                                           |
| **Options de liste de lecture**    | Contrôler **la façon dont la liste de lecture est lue** (par ex., séquence, aléatoire, boucle).                                       |
| **Position**                       | Définir la **position** de l'élément sur la page (gauche, haut).                                                                      |
| **Preuve de lecture**              | **Enregistrer les détails de** lecture pour les besoins de rapport et de vérification.                                                |
| **Rotation**                       | Afficher l'élément à un **angle spécifique**.                                                                                         |
| **Règles**                         | Définir des **règles d'élément** pour un comportement dynamique basé sur des conditions.                                              |
| **Taille**                         | Définir la **largeur et la hauteur** de l'élément.                                                                                    |
| **Transparence**                   | Ajuster le **niveau d'opacité** de l'élément.                                                                                         |
| **Visibilité**                     | Définir si l'élément ou son contenu est **visible pendant la lecture**.                                                               |
| **Volume**                         | Ajuster le **niveau de volume** pour la lecture vidéo et sonore.                                                                      |

***

### **Remarques clés**

* **Les éléments Mixed Media** peuvent **combiner** à la fois des vidéos et des images dans une seule liste de lecture pour une **lecture fluide**.
* Vous pouvez **contrôler le comportement de lecture** via les options de liste de lecture, notamment **la mise en boucle, la lecture aléatoire et la synchronisation**.
* En utilisant des **fournisseurs de données**, les éléments Mixed Media peuvent afficher dynamiquement **du contenu multimédia à jour** sans mises à jour manuelles.
* Pour les **environnements audités**, la Preuve de lecture garantit que la lecture du contenu est enregistrée et vérifiable.
* **Les présentations interactives** peuvent bénéficier des transitions de liste de lecture et des options alternatives pour créer des séquences de contenu dynamiques et engageantes.
