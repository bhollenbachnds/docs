Les éléments Mixed Media sont utilisés pour afficher une **playlist of images and videos**. Ils permettent des combinaisons flexibles de types de médias et offrent des options pour contrôler la lecture, l'apparence et l'interaction.

***

### **Aperçu des propriétés**

| **Section de propriété**      | **Description**                                                                                                                 |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **Alignment**                 | Définir **horizontal and vertical alignment** du contenu dans l'élément.                                                        |
| **Alternate options**         | Définir comment l'élément va **switch between playlist items** (par exemple, boucle, aléatoire).                                |
| **Background color**          | Définir la **background color** de l'élément, y compris les dégradés et les motifs.                                             |
| **Border**                    | Définir la **border** autour de l'élément, y compris la couleur et la largeur.                                                  |
| **Data provider**             | Se connecter à une **external data source** pour alimenter du contenu en temps réel (images/vidéos).                            |
| **Default duration**          | Définir la **default display duration** pour les éléments de la playlist si non spécifiée individuellement.                     |
| **Default transition effect** | Définir les **transition effects** pour la façon dont le nouveau contenu entre dans l'élément (par exemple, fondu, glissement). |
| **Identification**            | Identifier de manière unique l'élément dans la page pour référence et automatisation.                                           |
| **Play list**                 | Créer et gérer une **list of media items** (images/vidéos) à lire.                                                              |
| **Play list options**         | Contrôler **how the playlist is played** (par exemple, séquence, aléatoire, boucle).                                            |
| **Position**                  | Définir la **position** de l'élément sur la page (gauche, haut).                                                                |
| **Proof of play**             | **Log playback** détails pour les besoins de rapports et de vérification.                                                       |
| **Rotation**                  | Afficher l'élément à un **specific angle**.                                                                                     |
| **Rules**                     | Définir les **element rules** pour un comportement dynamique basé sur des conditions.                                           |
| **Taille**                    | Définir **la largeur et la hauteur** de l'élément.                                                                              |
| **Transparence**              | Ajuster le **niveau d'opacité** de l'élément.                                                                                   |
| **Visibilité**                | Définir si l'élément ou son contenu est **visible pendant la lecture**.                                                         |
| **Volume**                    | Ajuster le **niveau de volume** pour la lecture de vidéos et de sons.                                                           |

***

### **Remarques clés**

* **Mixed Media elements** peuvent **combiner** à la fois des vidéos et des images dans une seule liste de lecture pour une **lecture fluide**.
* Vous pouvez **contrôler le comportement de lecture** grâce aux options de liste de lecture, notamment **la lecture en boucle, la lecture aléatoire et le minutage**.
* En utilisant des **data providers**, les Mixed Media elements peuvent afficher dynamiquement du **contenu média à jour** sans mises à jour manuelles.
* Pour les **environnements audités**, Proof of Play garantit que la lecture du contenu est enregistrée et vérifiable.
* **Interactive presentations** peuvent bénéficier de transitions de liste de lecture et d'options alternatives pour créer des séquences de contenu dynamiques et engageantes.
