Les éléments Mixed Media sont utilisés pour afficher une **playlist d'images et de vidéos**. Ils permettent des combinaisons flexibles de types de médias et offrent des options pour contrôler la lecture, l'apparence et l'interaction.

***

### **Aperçu des propriétés**

| **Section de propriété**           | **Description**                                                                                                               |
| ---------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| **Alignement**                     | Définir l'**alignement horizontal et vertical** du contenu dans l'élément.                                                    |
| **Options alternatives**           | Définir comment l'élément va **passer d'un élément de playlist à l'autre** (par ex., boucle, aléatoire).                      |
| **Couleur d'arrière-plan**         | Définir la **couleur d'arrière-plan** de l'élément, y compris les dégradés et les motifs.                                     |
| **Bordure**                        | Définir la **bordure** autour de l'élément, y compris la couleur et la largeur.                                               |
| **Fournisseur de données**         | Se connecter à une **source de données externe** pour alimenter du contenu en temps réel (images/vidéos).                     |
| **Durée par défaut**               | Définir la **durée d'affichage par défaut** pour les éléments de la playlist si elle n'est pas spécifiée individuellement.    |
| **Effet de transition par défaut** | Définir les **effets de transition** pour la façon dont le nouveau contenu entre dans l'élément (par ex., fondu, glissement). |
| **Identification**                 | Identifier de manière unique l'élément dans la page pour le référencement et l'automatisation.                                |
| **Playlist**                       | Créer et gérer une **liste d'éléments médias** (images/vidéos) à lire.                                                        |
| **Options de playlist**            | Contrôler **comment la playlist est lue** (par ex., séquence, aléatoire, boucle).                                             |
| **Position**                       | Définir la **position** de l'élément sur la page (gauche, haut).                                                              |
| **Preuve de lecture**              | **Enregistrer les détails de lecture** à des fins de rapport et de vérification.                                              |
| **Rotation**                       | Afficher l'élément à un **angle spécifique**.                                                                                 |
| **Règles**                         | Définir les **règles d'élément** pour un comportement dynamique basé sur des conditions.                                      |
| **Taille**                         | Définir la **largeur et la hauteur** de l'élément.                                                                            |
| **Transparence**                   | Ajuster le **niveau d'opacité** de l'élément.                                                                                 |
| **Visibilité**                     | Définir si l'élément ou son contenu est **visible pendant la lecture**.                                                       |
| **Volume**                         | Ajuster le **niveau de volume** pour la lecture de vidéo et de son.                                                           |

***

### **Remarques clés**

* **Les éléments Mixed Media** peuvent **combiner** à la fois des vidéos et des images dans une seule playlist pour une **lecture fluide**.
* Vous pouvez **contrôler le comportement de lecture** grâce aux options de playlist, notamment **la lecture en boucle, la lecture aléatoire et le minutage**.
* En utilisant des **fournisseurs de données**, les éléments Mixed Media peuvent afficher dynamiquement du **contenu média à jour** sans mises à jour manuelles.
* Pour les **environnements audités**, la Preuve de lecture garantit que la lecture du contenu est enregistrée et vérifiable.
* **Les présentations interactives** peuvent bénéficier des transitions de playlist et des options alternatives pour créer des séquences de contenu dynamiques et engageantes.
