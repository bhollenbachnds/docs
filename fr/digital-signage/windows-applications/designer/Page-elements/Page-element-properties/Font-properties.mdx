Ces propriétés déterminent le **font style, color, and text wrapping** pour le contenu textuel de votre élément.

***

### Font

* **Style**\
  Affiche la **font and font style** actuellement sélectionnée.
  * Pour changer la police, cliquez sur le bouton **\[...]** de navigation à côté de la zone de texte pour ouvrir la **font dialog box**.
  * Vous pouvez également utiliser les **typeface buttons** rapides (Gras, Italique, Souligné, etc.) pour ajuster le style.
  * Cliquez sur le bouton **Clear** pour réinitialiser et désactiver tous les paramètres de police.

**Note importante sur les polices :**

* Assurez-vous que toute police que vous choisissez est **available on all PADS4 Viewers**.
* Si une police sélectionnée n'est **not available during playback**, une **default font** sera utilisée à la place.
* Les polices ajoutées via **PADS4 System Management** sont **automatically installed** sur tous les Viewers (HTML5) et affichées **at the top of the font dialog list**, marquées d'une icône spéciale.
* **Recommendation** : Utilisez toujours les polices distribuées par PADS4 pour assurer la cohérence entre les appareils.

***

### Color

* Sélectionnez la **font color** en utilisant le **drop-down color picker**.
* Vous pouvez choisir :
  * **Fixed color**
  * **Gradient (two or three colors)**
  * **Texture or pattern**
* Voir **Utilisation du sélecteur de couleur** pour des instructions d'utilisation détaillées.

***

### Retour à la ligne du texte

Choisissez comment le texte se comportera lorsqu'il dépasse la largeur de l'élément :

| Option                       | Description                                                                                |
| ---------------------------- | ------------------------------------------------------------------------------------------ |
| **Pas de retour à la ligne** | Le texte restera sur une seule ligne. S'il est trop long, il sera coupé.                   |
| **Retour à la ligne**        | Le texte passera automatiquement à la ligne suivante s'il dépasse la largeur de l'élément. |

***

### Remarques

* **Sauts de ligne** (par ex., `<CR>`, `<CR><LF>`) dans votre texte **créeront toujours une nouvelle ligne**, quelle que soit l'option de retour à la ligne sélectionnée.
