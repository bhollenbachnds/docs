Ces propriétés déterminent le **style de police, la couleur et le retour à la ligne du texte** pour le contenu textuel de votre élément.

***

### Police

* **Style**\
  Affiche la **police et le style de police** actuellement sélectionnés.
  * Pour changer la police, cliquez sur le bouton **\[...]** parcourir à côté de la zone de texte pour ouvrir la **boîte de dialogue de police**.
  * Vous pouvez également utiliser les boutons rapides de **style de caractère** (Gras, Italique, Souligné, etc.) pour ajuster le style.
  * Cliquez sur le **Effacer** bouton pour réinitialiser et désactiver tous les paramètres de police.

**Remarque importante sur les polices :**

* Assurez-vous que toute police que vous choisissez est **disponible sur tous les PADS4 Viewers**.
* Si une police sélectionnée n'est **pas disponible lors de la lecture**, une **police par défaut** sera utilisée à la place.
* Les polices ajoutées via **PADS4 System Management** sont **automatiquement installées** sur tous les Viewers (HTML5) et affichées **en haut de la liste de la boîte de dialogue des polices**, marquées d'une icône spéciale.
* **Recommandation** : Utilisez toujours les polices distribuées par PADS4 pour garantir la cohérence entre les appareils.

***

### Couleur

* Sélectionnez la **couleur de police** à l'aide du **sélecteur de couleur déroulant**.
* Vous pouvez choisir :
  * **Couleur fixe**
  * **Dégradé (deux ou trois couleurs)**
  * **Texture ou motif**
* Voir **Utilisation du sélecteur de couleur** pour des instructions d'utilisation détaillées.

***

### Retour à la ligne du texte

Choisissez comment le texte se comportera lorsqu'il dépasse la largeur de l'élément :

| Option                            | Description                                                                                |
| --------------------------------- | ------------------------------------------------------------------------------------------ |
| **Pas de retour à la ligne**      | Le texte restera sur une seule ligne. S'il est trop long, il sera coupé.                   |
| **Retour à la ligne automatique** | Le texte passera automatiquement à la ligne suivante s'il dépasse la largeur de l'élément. |

***

### Remarques

* **Les sauts de ligne** (par ex., `<CR>`, `<CR><LF>`) dans votre texte **créeront toujours une nouvelle ligne**, quelle que soit l'option de retour à la ligne sélectionnée.
