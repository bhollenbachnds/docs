Ces propriétés déterminent **l'alignement horizontal et vertical** du texte à l'intérieur de votre élément.

### **Alignement**

* **Horizontal**\
  Options: **Gauche**, **Milieu**, **Droite**, **Justifié**

* **Vertical**\
  Options: **Haut**, **Milieu**, **Bas**

***

### **Remarques**

* **Les options d'alignement** peuvent être **indisponibles** (invisibles) dans certaines situations selon le type d'élément et la configuration.

* Si **le dimensionnement automatique** est activé pour l'élément, les paramètres d'alignement influencent la façon dont le redimensionnement se produit :
  * **Comportement par défaut** : l'élément se redimensionne sur le côté droit.
  * Si **l'alignement horizontal** est défini sur **Droite**, l'élément se redimensionnera à partir du côté **gauche**.
  * Si **l'alignement horizontal** est défini sur **Milieu**, l'élément se redimensionnera de manière égale des deux côtés.
  * **L'alignement vertical** se comporte de manière similaire, affectant le redimensionnement haut/bas.

* Pour les **Éléments de texte défilant**, le comportement d'alignement dépend de la **direction de défilement** :
  * Si **De droite à gauche** ou **De gauche à droite** est choisi, **l'alignement horizontal** est ignoré.
  * Si **De haut en bas** ou **De bas en haut** est choisi, **l'alignement vertical** est ignoré.
