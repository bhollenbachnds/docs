Ces propriétés déterminent l'**alignement horizontal et vertical** du texte à l'intérieur de votre élément.

### **Alignement**

* **Horizontal**\
  Options: **Gauche**, **Milieu**, **Droite**, **Justifié**

* **Vertical**\
  Options: **Haut**, **Milieu**, **Bas**

***

### **Remarques**

* **Les options d'alignement** peuvent être **indisponibles** (invisibles) dans certaines situations selon le type et la configuration de l'élément.

* Si **le dimensionnement automatique** est activé pour l'élément, les paramètres d'alignement influencent la façon dont le redimensionnement s'effectue :
  * **Comportement par défaut** : l'élément se redimensionne sur le côté droit.
  * Si **l'alignement horizontal** est défini sur **Droite**, l'élément se redimensionnera depuis le côté **gauche**.
  * Si **l'alignement horizontal** est défini sur **Milieu**, l'élément se redimensionnera de manière égale des deux côtés.
  * **L'alignement vertical** se comporte de manière similaire, affectant le redimensionnement en haut/bas.

* Pour les **éléments de texte défilant**, le comportement de l'alignement dépend de la **direction de défilement** :
  * Si **Droite vers Gauche** ou **Gauche vers Droite** est choisi, **l'alignement horizontal** est ignoré.
  * Si **Haut vers Bas** ou **Bas vers Haut** est choisi, **l'alignement vertical** est ignoré.
