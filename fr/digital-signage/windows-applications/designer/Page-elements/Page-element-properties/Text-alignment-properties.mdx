Ces propriétés déterminent **l'alignement horizontal et vertical** du texte à l'intérieur de votre élément.

### **Alignement**

* **Horizontal**\
  Options: **Left**, **Middle**, **Right**, **Justified**

* **Vertical**\
  Options: **Top**, **Middle**, **Bottom**

***

### **Remarques**

* **Les options d'alignement** peuvent être **indisponibles** (invisibles) dans certaines situations selon le type d'élément et sa configuration.

* Si **automatic sizing** est activé pour l'élément, les paramètres d'alignement influencent la façon dont le redimensionnement se produit :
  * **Comportement par défaut** : l'élément se redimensionne sur le côté droit.
  * Si **Horizontal alignment** est défini sur **Right**, l'élément se redimensionnera à partir du **côté gauche**.
  * Si **Alignement horizontal** est défini sur **Milieu**, l'élément se redimensionnera de manière égale des deux côtés.
  * **Alignement vertical** se comporte de manière similaire, affectant le redimensionnement haut/bas.

* Pour les **Scrolling Text Elements**, le comportement d'alignement dépend de la **direction de défilement** :
  * Si **Right to Left** ou **Left to Right** est choisi, **Alignement horizontal** est ignoré.
  * Si **Top to Bottom** ou **Bottom to Top** est choisi, **Alignement vertical** est ignoré.
