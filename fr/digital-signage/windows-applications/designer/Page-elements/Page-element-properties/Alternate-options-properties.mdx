Ces propriétés contrôlent la façon dont un élément passe d'un élément à l'autre dans sa liste de lecture pendant la lecture.

### Play Entire Play List

* **Si désactivé** :\
  Seul le **premier élément de la liste de lecture** sera lu.

* **Si activé** :\
  Tous les éléments de la liste de lecture seront lus selon l'une des méthodes suivantes :

#### Méthodes de lecture (lorsque "Play Entire Play List" est activé)

1. **Play using Play List Durations** :\
   Chaque élément de la liste de lecture sera lu pendant la durée **définie dans la liste de lecture** elle-même.

2. **Play using Page Alternate Interval** :\
   Tous les éléments de la liste de lecture seront lus pendant la **même durée fixe**, comme spécifié dans l'**Alternate Interval** des propriétés de la page.

   **Remarque importante** :

   * Lorsque cette option est utilisée, **plusieurs éléments** l'utilisant vont **synchroniser** le changement de leur liste de lecture.
   * Si les éléments ont un nombre différent d'éléments dans leur liste de lecture, le système calculera quel élément doit s'afficher à chaque moment, afin qu'ils restent alignés.

***

### Exemple de synchronisation

Considérons **deux éléments** avec des longueurs de liste de lecture différentes :

| **Élément 1 (4 éléments)** | **Élément 2 (2 éléments)** |
| -------------------------- | -------------------------- |
| Élément de liste 1         | Élément de liste 1         |
| Élément de liste 2         | Élément de liste 2         |
| Élément de liste 3         | Élément de liste 1         |
| Élément de liste 4         | Élément de liste 2         |
| Élément de liste 1         | Élément de liste 1         |
| Élément de liste 2         | Élément de liste 2         |

Dans cet exemple :

* **L'élément de liste 1** des deux éléments sera **toujours affiché en même temps**.
* Le système aligne automatiquement les cycles, même avec des longueurs de liste différentes.

***

### Lecture synchronisée sur tous les Viewers

Lorsque **Play using Page Alternate Interval** est actif, l'alternance des éléments de la liste de lecture est **synchronisée sur tous les Viewers** (écrans/lecteurs).

* Cela assure une **alternance simultanée** sur tous les écrans connectés.
* Assurez-vous que les paramètres de synchronisation du système sont correctement configurés pour prendre en charge ce comportement.

***

### Résumé

* **Play Entire Play List** : Permet de parcourir tous les éléments de la liste de lecture.
* Choisissez entre **les durées individuelles des éléments** ou un **intervalle d'alternance uniforme**.
* La synchronisation maintient plusieurs éléments et appareils en phase.
* Utile pour l'affichage coordonné de contenu sur plusieurs zones ou écrans.
