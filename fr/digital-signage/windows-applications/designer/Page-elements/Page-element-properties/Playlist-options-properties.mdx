Ces propriétés définissent **comment les éléments de votre play list** seront lus pendant la présentation.

***

### Play List

Vous pouvez choisir parmi les options suivantes :

| **Option**                        | **Description**                                                                                                                                                                                            |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Auto Repeat**                   | L'élément va **redémarrer automatiquement** la lecture depuis le premier élément lorsque tous les éléments de la play list sont terminés. Cela crée une **boucle continue**.                               |
| **Play Once**                     | L'élément lira tous les éléments **une seule fois**. Après avoir terminé le dernier élément, l'élément sera **rendu invisible**.                                                                           |
| **Play Once and Go to Next Page** | L'élément lira tous les éléments **une fois** et, à la fin, **passera automatiquement à la page suivante** de votre présentation.                                                                          |
| **Sync Repeat**                   | Active la **lecture synchronisée** de l'élément sur **plusieurs écrans** pour un flux de contenu multi-écrans fluide. **Remarque** : Cette fonctionnalité fonctionne uniquement sur les **HTML5 Viewers**. |

***

### Remarques

* **Important** : Ces options de play list s'appliquent **uniquement lorsque** l'option **"Play entire Play List"** est activée dans la section **Alternate Options**.
* Utilisez **Sync Repeat** pour du contenu synchronisé sur des configurations multi-écrans, garantissant un **flux de présentation cohérent et coordonné**.
