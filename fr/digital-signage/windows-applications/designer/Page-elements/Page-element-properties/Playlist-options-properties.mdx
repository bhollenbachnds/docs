Ces propriétés définissent **comment les éléments de votre liste de lecture** seront lus pendant la présentation.

***

### Liste de lecture

Vous pouvez choisir parmi les options suivantes :

| **Option**                        | **Description**                                                                                                                                                                                     |
| --------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Auto Repeat**                   | L'élément **redémarrera automatiquement** la lecture à partir du premier élément lorsque tous les éléments de la liste de lecture sont terminés. Cela crée une **boucle continue**.                 |
| **Play Once**                     | L'élément lira tous les éléments **une seule fois**. Après avoir terminé le dernier élément, l'élément sera **rendu invisible**.                                                                    |
| **Play Once and Go to Next Page** | L'élément lira tous les éléments **une fois** et, une fois terminé, **passera automatiquement à la page suivante** de votre présentation.                                                           |
| **Sync Repeat**                   | Permet une **lecture synchronisée** de l'élément sur **plusieurs écrans** pour un flux de contenu multi-écrans fluide. **Note** : Cette fonctionnalité ne fonctionne que sur les **HTML5 Viewers**. |

***

### Remarques

* **Important** : Ces options de liste de lecture s'appliquent **uniquement lorsque** l'option **"Play entire Play List"** est activée dans la section **Alternate Options**.
* Utilisez **Sync Repeat** pour un contenu synchronisé sur des configurations multi-écrans, assurant un **flux de présentation cohésif et coordonné**.
