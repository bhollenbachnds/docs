Ces propriétés définissent **comment les éléments de votre liste de lecture** seront lus pendant la présentation.

***

### Liste de lecture

Vous pouvez choisir parmi les options suivantes :

| **Option**                                     | **Description**                                                                                                                                                                                                |
| ---------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Répétition automatique**                     | L'élément **redémarrera automatiquement** la lecture à partir du premier élément lorsque tous les éléments de la liste de lecture sont terminés. Cela crée une **boucle continue**.                            |
| **Lire une fois**                              | L'élément lira tous les éléments **une seule fois**. Après avoir terminé le dernier élément, l'élément sera **rendu invisible**.                                                                               |
| **Lire une fois et passer à la page suivante** | L'élément lira tous les éléments **une fois** et, une fois terminé, **passera automatiquement à la page suivante** de votre présentation.                                                                      |
| **Sync Repeat**                                | Active **la lecture synchronisée** de l'élément sur **plusieurs écrans** pour un flux de contenu fluide sur plusieurs écrans. **Remarque** : Cette fonctionnalité ne fonctionne que sur les **HTML5 Viewers**. |

***

### Remarques

* **Important** : Ces options de liste de lecture s'appliquent **uniquement lorsque** l'**"Play entire Play List"** option est activée dans la section **Alternate Options**.
* Utilisez **Sync Repeat** pour synchroniser le contenu sur des configurations multi-écrans, assurant un **flux de présentation cohésif et coordonné**.
