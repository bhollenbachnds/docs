Ces propriétés définissent **comment les éléments de votre liste de lecture** seront lus pendant la présentation.

***

### Liste de lecture

Vous pouvez choisir parmi les options suivantes :

| **Option**                                     | **Description**                                                                                                                                                                                            |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Répétition automatique**                     | L'élément va **redémarrer automatiquement** la lecture depuis le premier élément lorsque tous les éléments de la liste de lecture sont terminés. Cela crée une **boucle continue**.                        |
| **Lire une fois**                              | L'élément lira tous les éléments **une seule fois**. Après avoir terminé le dernier élément, l'élément sera **rendu invisible**.                                                                           |
| **Lire une fois et passer à la page suivante** | L'élément lira tous les éléments **une fois** et, à la fin, **passera automatiquement à la page suivante** de votre présentation.                                                                          |
| **Sync Repeat**                                | Active **la lecture synchronisée** de l'élément sur **plusieurs écrans** pour un flux de contenu multi-écrans fluide. **Remarque** : Cette fonctionnalité fonctionne uniquement sur les **HTML5 Viewers**. |

***

### Remarques

* **Important** : Ces options de liste de lecture s'appliquent **uniquement lorsque** l'option **"Lire toute la liste de lecture"** est activée dans la section **Options alternatives**.
* Utilisez **Sync Repeat** pour un contenu synchronisé sur des configurations multi-écrans, garantissant un **flux de présentation cohérent et coordonné**.
