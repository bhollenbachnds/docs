Les propriétés **Proof of Play** vous permettent de **consigner des informations détaillées** sur la lecture d'éléments spécifiques dans votre présentation. Ce journal peut être utilisé, par exemple, pour **démontrer aux annonceurs** que leur contenu a été affiché, y compris quand, où et pendant combien de temps.

### Proof of Play

**Enregistrer les statistiques des éléments**\
Lorsque cette option est activée, le PADS4 Viewer enregistrera des informations détaillées chaque fois que l'élément est lu. Ces informations sont automatiquement envoyées au **PADS4 Server** pour une collecte et un stockage centralisés.

### Remarques

* Les informations de lecture sont stockées dans une **base de données dédiée proof of play** sur le **PADS4 Server PC**.
* Cette base de données peut être consultée et examinée à l'aide de :
  * **PADS4 Agent**, qui permet également d'exporter le journal au format XML pour une analyse plus approfondie (par exemple, dans Microsoft Excel).
  * **PADS4 Software Development Kit (SDK)** pour créer des outils personnalisés afin de traiter et d'analyser les données enregistrées.

### Important

Si vous activez l'enregistrement **Proof of Play**, il est essentiel d'établir une **procédure pour gérer et nettoyer régulièrement** la ou les bases de données proof of play.\
Sans entretien approprié, la base de données peut **croître indéfiniment**, finissant par consommer tout l'espace disque disponible, car **la suppression automatique des anciens enregistrements n'est pas effectuée par le système**.
