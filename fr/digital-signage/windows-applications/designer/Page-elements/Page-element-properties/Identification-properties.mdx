Ces propriétés définissent l'**identification unique** de votre élément au sein d'une page.

***

### Identification

* **Nom**\
  Un **nom unique** qui identifie l'élément dans la page de votre présentation.
  * Ce nom est essentiel pour gérer et organiser les éléments de votre page.
  * Pour vérifier les noms disponibles ou éviter les doublons, **cliquez sur le bouton** situé à côté du champ de saisie Nom — cela permet de garantir l'unicité.

***

### Remarques

* Le **nom de l'élément** spécifié ici apparaîtra également dans la **fenêtre Éléments**, où vous pouvez gérer les éléments et ajuster leur **ordre d'affichage**.
* Par défaut, le système **génère automatiquement** un **nom unique et localisé** pour chaque élément lors de sa création. Vous pouvez **modifier ce nom** selon vos besoins pour une meilleure organisation et référence au sein de votre projet.
