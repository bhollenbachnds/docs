Ces propriétés définissent **l'identification unique** de votre élément dans une page.

***

### Identification

* **Nom**\
  Un **nom unique** qui identifie l'élément dans la page de votre présentation.
  * Ce nom est essentiel pour gérer et organiser les éléments sur votre page.
  * Pour vérifier les noms disponibles ou éviter les doublons, **cliquez sur le bouton** à côté du champ de saisie Nom — cela aide à garantir l'unicité.

***

### Remarques

* Le **nom de l'élément** spécifié ici apparaîtra également dans la **fenêtre Elements**, où vous pouvez gérer les éléments et ajuster leur **ordre d'affichage**.
* Par défaut, le système **générera automatiquement** un **nom unique et localisé** pour chaque élément lors de sa création. Vous pouvez **modifier ce nom** selon vos besoins pour une meilleure organisation et référence dans votre projet.
