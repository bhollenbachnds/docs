Ces propriétés vous permettent de **créer une version miroir (inversée)** du contenu de votre élément, soit horizontalement, verticalement, ou les deux.

***

### Options de retournement

* **Flip Horizontal**\
  Inverse le contenu **horizontalement** (effet miroir de gauche à droite).

* **Flip Vertical**\
  Inverse le contenu **verticalement** (effet miroir de haut en bas).

***

### Remarques

* Vous pouvez **combiner** les deux options pour retourner le contenu **dans les deux directions**.
* Utile pour **des effets de design** ou pour ajuster l'alignement du contenu selon les besoins de mises en page spécifiques.
