Ces propriétés vous permettent de **créer une version en miroir (inversée)** du contenu de votre élément, horizontalement, verticalement, ou dans les deux sens.

***

### Options de retournement

* **Flip Horizontal**\
  Inverse le contenu **horizontalement** (effet miroir de gauche à droite).

* **Flip Vertical**\
  Inverse le contenu **verticalement** (effet miroir de haut en bas).

***

### Remarques

* Vous pouvez **combiner** les deux options pour retourner le contenu **dans les deux sens**.
* Utile pour les **effets de conception** ou pour ajuster l'alignement du contenu selon les besoins de mises en page spécifiques.
