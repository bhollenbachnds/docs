Ces propriétés contrôlent **l'alignement horizontal et vertical** du contenu dans votre élément.

***

### **Paramètres d'alignement**

| **Propriété**  | **Options**         | **Description**                                   |
| -------------- | ------------------- | ------------------------------------------------- |
| **Horizontal** | Left, Middle, Right | Aligne le contenu horizontalement dans l'élément. |
| **Vertical**   | Top, Middle, Bottom | Aligne le contenu verticalement dans l'élément.   |

***

### **Conseil**

Vous pouvez **définir rapidement** l'alignement en cliquant sur l'un des boutons d'option d'alignement, qui affichent toutes les combinaisons d'alignement horizontal et vertical.

***

### **Remarques**

* Si **automatic sizing** est activé pour votre élément, les paramètres d'alignement affecteront **comment l'élément se redimensionne**:
  * **Alignement horizontal** affecte la direction du redimensionnement :
    * **Right**: L'élément se redimensionne vers la gauche.
    * **Middle**: L'élément se redimensionne de manière égale des deux côtés.
    * **Left**: L'élément se redimensionne vers la droite (par défaut).
  * **Alignement vertical** affecte la direction du redimensionnement :
    * **Bottom**: L'élément se redimensionne vers le haut.
    * **Middle**: L'élément se redimensionne de manière égale en haut et en bas.
    * **Top**: L'élément se redimensionne vers le bas (par défaut).
