Cette section décrit les **propriétés communes** disponibles pour tous les éléments de page dans PADS4 Designer.\
Pour les **propriétés spécifiques aux éléments**, référez-vous à la section dédiée de chaque type d'élément de page.

***

### **Propriétés communes et leurs descriptions**

| **Section de propriété**           | **Description**                                                                                                                             |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| **Alignment**                      | Définir l'alignement horizontal et vertical de votre contenu dans l'élément.                                                                |
| **Alternate options**              | Définissez comment votre élément passe d'un élément de playlist à l'autre.                                                                  |
| **Couleur d'arrière-plan**         | Définissez une couleur d'arrière-plan pour votre élément.                                                                                   |
| **Image d'arrière-plan**           | Définissez une image d'arrière-plan pour l'élément (ou la page entière).                                                                    |
| **Clignotement**                   | Ajoutez des effets de clignotement à l'élément pour attirer l'attention.                                                                    |
| **Bordure**                        | Configurez la largeur, le style et la couleur de la bordure autour de votre élément.                                                        |
| **Luminosité/contraste**           | Ajustez les niveaux de luminosité et de contraste du contenu de l'élément.                                                                  |
| **Fournisseur de données**         | Connectez-vous à une source de données externe pour fournir du contenu en temps réel à l'élément.                                           |
| **Défilement des données**         | Définissez si et comment l'élément doit faire défiler les lignes de données pendant la lecture.                                             |
| **Durée par défaut**               | Définissez la durée par défaut (en secondes) pour les éléments de la playlist de l'élément (le cas échéant).                                |
| **Effet de transition par défaut** | Définissez une animation de transition par défaut lors des changements de contenu.                                                          |
| **Retournement**                   | Retournez le contenu de l'élément horizontalement ou verticalement.                                                                         |
| **Police**                         | Spécifiez le style et la taille de police pour l'affichage du contenu textuel.                                                              |
| **Identification**                 | Attribuez un nom ou un identifiant unique à l'élément dans la page.                                                                         |
| **Options de playlist**            | Contrôlez la façon dont les éléments de la playlist sont parcourus ou lus en séquence.                                                      |
| **Position**                       | Définissez la position exacte (haut, gauche) de l'élément sur la page.                                                                      |
| **Preuve de lecture**              | Activez la journalisation de la preuve de lecture pour suivre quand et à quelle fréquence l'élément est affiché.                            |
| **Liaisons de propriétés**         | Liez les propriétés de l'élément à des sources de données externes pour un comportement dynamique.                                          |
| **Reflet**                         | Ajoutez un effet de reflet au contenu de l'élément.                                                                                         |
| **Rotation**                       | Faites pivoter l'élément selon un angle spécifié.                                                                                           |
| **Règles**                         | Définissez un comportement automatique via des règles, basées sur des conditions prédéfinies (par ex., masquer/afficher selon les données). |
| **Ombre**                          | Ajoutez un effet d'ombre au contenu de l'élément.                                                                                           |
| **Taille**                         | Définissez la largeur et la hauteur de l'élément.                                                                                           |
| **Inclinaison**                    | Inclinez le contenu de l'élément selon des angles spécifiques.                                                                              |
| **Alignement du texte**            | Définissez l'alignement horizontal et vertical du texte dans l'élément.                                                                     |
| **Effet de transition**            | Définissez une animation d'entrée ou un effet visuel lorsque l'élément apparaît.                                                            |
| **Transparence**                   | Définissez le niveau de transparence (opacité) de l'élément.                                                                                |
| **Visibilité**                     | Contrôlez si l'élément (ou son contenu) est affiché ou masqué pendant la lecture.                                                           |

***

### **Notes:**

* Ces propriétés permettent **un contrôle précis** sur l'apparence et le comportement des éléments de page.
* **Liaisons de propriétés** et **connexions au fournisseur de données** permettent des mises à jour dynamiques en temps réel.
* Pour les éléments utilisant des playlists (images, vidéos), **les options de playlist** et **la durée par défaut** définissent la façon dont les éléments se succèdent.
