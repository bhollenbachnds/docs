Cette section décrit les **propriétés communes** disponibles pour tous les éléments de page dans PADS4 Designer.\
Pour les **propriétés spécifiques aux éléments**, référez-vous à la section dédiée de chaque type d'élément de page.

***

### **Propriétés Communes et Leurs Descriptions**

| **Section de Propriété**      | **Description**                                                                                                                              |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| **Alignment**                 | Définir l'alignement horizontal et vertical de votre contenu dans l'élément.                                                                 |
| **Alternate options**         | Définir comment votre élément bascule entre les éléments de la liste de lecture.                                                             |
| **Background color**          | Définir une couleur d'arrière-plan pour votre élément.                                                                                       |
| **Background Picture**        | Définir une image d'arrière-plan pour l'élément (ou la page entière).                                                                        |
| **Blinking**                  | Ajouter des effets de clignotement à l'élément pour attirer l'attention.                                                                     |
| **Border**                    | Configurer la largeur, le style et la couleur de la bordure autour de votre élément.                                                         |
| **Brightness/contrast**       | Ajuster les niveaux de luminosité et de contraste du contenu de l'élément.                                                                   |
| **Data provider**             | Se connecter à une source de données externe pour fournir du contenu en temps réel à l'élément.                                              |
| **Data scrolling**            | Définir si et comment l'élément doit faire défiler les lignes de données pendant la lecture.                                                 |
| **Default duration**          | Définir la durée par défaut (en secondes) pour les éléments de la liste de lecture de l'élément (le cas échéant).                            |
| **Default transition effect** | Définir une animation de transition par défaut lors des changements de contenu.                                                              |
| **Flip**                      | Retourner le contenu de l'élément horizontalement ou verticalement.                                                                          |
| **Font**                      | Spécifier le style et la taille de police pour l'affichage du contenu textuel.                                                               |
| **Identification**            | Attribuer un nom ou un identifiant unique à l'élément dans la page.                                                                          |
| **Play list options**         | Contrôler la façon dont les éléments de la liste de lecture sont parcourus ou lus en séquence.                                               |
| **Position**                  | Définir la position exacte (haut, gauche) de l'élément sur la page.                                                                          |
| **Proof of play**             | Activer la journalisation pour la preuve de diffusion afin de suivre quand et à quelle fréquence l'élément est affiché.                      |
| **Property bindings**         | Lier les propriétés de l'élément à des sources de données externes pour un comportement dynamique.                                           |
| **Reflection**                | Ajouter un effet de reflet au contenu de l'élément.                                                                                          |
| **Rotation**                  | Faire pivoter l'élément selon un angle spécifié.                                                                                             |
| **Règles**                    | Définir un comportement automatique via des règles, basé sur des conditions prédéfinies (par ex., masquer/afficher en fonction des données). |
| **Ombre**                     | Ajouter un effet d'ombre au contenu de l'élément.                                                                                            |
| **Taille**                    | Définir la largeur et la hauteur de l'élément.                                                                                               |
| **Inclinaison**               | Incliner le contenu de l'élément selon des angles spécifiques.                                                                               |
| **Alignement du texte**       | Définir l'alignement horizontal et vertical du texte dans l'élément.                                                                         |
| **Effet de transition**       | Définir une animation d'entrée ou un effet visuel lorsque l'élément apparaît.                                                                |
| **Transparence**              | Définir le niveau de transparence (opacité) de l'élément.                                                                                    |
| **Visibilité**                | Contrôler si l'élément (ou son contenu) est affiché ou masqué pendant la lecture.                                                            |

***

### **Notes:**

* Ces propriétés permettent **un contrôle précis** sur l'apparence et le comportement des éléments de page.
* **Liaisons de propriétés** et **connexions aux fournisseurs de données** permettent des mises à jour dynamiques en temps réel.
* Pour les éléments utilisant des playlists (images, vidéos), **les options de playlist** et **la durée par défaut** définissent la façon dont les éléments se succèdent.
