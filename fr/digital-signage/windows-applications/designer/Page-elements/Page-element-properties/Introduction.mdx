Cette section décrit les **propriétés communes** disponibles pour tous les éléments de page dans PADS4 Designer.\
Pour les **propriétés spécifiques aux éléments**, reportez-vous à la section dédiée de chaque type d'élément de page.

***

### **Propriétés communes et leurs descriptions**

| **Section de propriété**           | **Description**                                                                                                                                  |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Alignement**                     | Définir l'alignement horizontal et vertical de votre contenu dans l'élément.                                                                     |
| **Options alternatives**           | Définir comment votre élément bascule entre les éléments de la playlist.                                                                         |
| **Couleur d'arrière-plan**         | Définir une couleur d'arrière-plan pour votre élément.                                                                                           |
| **Image d'arrière-plan**           | Définir une image d'arrière-plan pour l'élément (ou la page entière).                                                                            |
| **Clignotement**                   | Ajouter des effets de clignotement à l'élément pour attirer l'attention.                                                                         |
| **Bordure**                        | Configurer la largeur, le style et la couleur de la bordure autour de votre élément.                                                             |
| **Luminosité/contraste**           | Ajuster les niveaux de luminosité et de contraste du contenu de l'élément.                                                                       |
| **Fournisseur de données**         | Se connecter à une source de données externe pour fournir du contenu en temps réel à l'élément.                                                  |
| **Défilement des données**         | Définir si et comment l'élément doit faire défiler les lignes de données pendant la lecture.                                                     |
| **Durée par défaut**               | Définir la durée par défaut (en secondes) pour les éléments de la playlist de l'élément (le cas échéant).                                        |
| **Effet de transition par défaut** | Définir une animation de transition par défaut lorsque le contenu change.                                                                        |
| **Retournement**                   | Retourner le contenu de l'élément horizontalement ou verticalement.                                                                              |
| **Police**                         | Spécifier le style et la taille de police pour afficher le contenu textuel.                                                                      |
| **Identification**                 | Attribuer un nom ou un identifiant unique à l'élément dans la page.                                                                              |
| **Options de playlist**            | Contrôler comment les éléments de la playlist sont cyclés ou lus en séquence.                                                                    |
| **Position**                       | Définir la position exacte (haut, gauche) de l'élément sur la page.                                                                              |
| **Preuve de diffusion**            | Activer la journalisation pour la preuve de diffusion afin de suivre quand et à quelle fréquence l'élément est affiché.                          |
| **Liaisons de propriétés**         | Lier les propriétés de l'élément à des sources de données externes pour un comportement dynamique.                                               |
| **Réflexion**                      | Ajouter un effet de réflexion au contenu de l'élément.                                                                                           |
| **Rotation**                       | Faire pivoter l'élément à un angle spécifié.                                                                                                     |
| **Règles**                         | Définir un comportement automatique via des règles, basé sur des conditions prédéfinies (par exemple, masquer/afficher en fonction des données). |
| **Ombre**                          | Ajouter un effet d'ombre au contenu de l'élément.                                                                                                |
| **Taille**                         | Définir la largeur et la hauteur de l'élément.                                                                                                   |
| **Inclinaison**                    | Incliner le contenu de l'élément à des angles spécifiques.                                                                                       |
| **Alignement du texte**            | Définir l'alignement horizontal et vertical du texte dans l'élément.                                                                             |
| **Effet de transition**            | Définir une animation entrante ou un effet visuel lorsque l'élément apparaît.                                                                    |
| **Transparence**                   | Définir le niveau de transparence (opacité) de l'élément.                                                                                        |
| **Visibilité**                     | Contrôler si l'élément (ou son contenu) est affiché ou masqué pendant la lecture.                                                                |

***

### **Notes:**

* Ces propriétés permettent un **contrôle précis** sur l'apparence et le comportement des éléments de page.
* **Liaisons de propriétés** et **connexions aux fournisseurs de données** permettent des mises à jour dynamiques en temps réel.
* Pour les éléments utilisant des playlists (images, vidéos), **options de playlist** et **durée par défaut** définissent comment les éléments tournent.
