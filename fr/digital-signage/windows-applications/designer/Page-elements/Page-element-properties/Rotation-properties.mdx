Ces propriétés déterminent l' **angle de rotation** auquel le contenu de votre élément est affiché.

***

### **Rotation**

| **Propriété** | **Description**                                                                                                                                          |
| ------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Angle**     | Définir l' **angle de rotation** (en degrés). <br /> Vous pouvez **sélectionner un angle prédéfini** ou **saisir manuellement** n'importe quelle valeur. |

***

### **Remarques**

* Les angles de rotation sont mesurés **dans le sens des aiguilles d'une montre**.
* Par défaut, le **centre de l'élément** sert de **point de pivot** pour la rotation.
* **Remarque** : Si **le dimensionnement automatique** est appliqué à l'élément, le point de pivot peut s'ajuster en fonction des dimensions redimensionnées.
