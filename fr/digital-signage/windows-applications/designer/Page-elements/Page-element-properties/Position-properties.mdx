Ces propriétés définissent la **position of your element** sur la page.

***

### **Position**

| **Propriété** | **Description**                                    |
| ------------- | -------------------------------------------------- |
| **Left**      | Position horizontale (en pixels) de votre élément. |
| **Top**       | Position verticale (en pixels) de votre élément.   |

***

### **Remarques**

* Si **Snap to Grid** est activé, vous pouvez **manually enter positions** qui **not align to the grid**, permettant un placement précis.
* Les **position properties update automatically** when:
  * Vous déplacez l'élément en utilisant la **mouse** ou les **arrow keys**.
  * Vous **paste** l'élément.
  * Vous appliquez des **formatting functions** à l'élément.
* **Note** : Si votre élément est **rotated**, les propriétés de position reflètent la position de l'**élément non pivoté (original)**.
