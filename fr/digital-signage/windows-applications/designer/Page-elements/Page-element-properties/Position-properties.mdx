Ces propriétés définissent la **position de votre élément** sur la page.

***

### **Position**

| **Propriété** | **Description**                                    |
| ------------- | -------------------------------------------------- |
| **Gauche**    | Position horizontale (en pixels) de votre élément. |
| **Haut**      | Position verticale (en pixels) de votre élément.   |

***

### **Remarques**

* Si **Snap to Grid** est activé, vous pouvez **saisir manuellement des positions** qui ne **s'alignent pas sur la grille**, permettant un placement précis.
* Les **propriétés de position se mettent à jour automatiquement** when:
  * Vous déplacez l'élément à l'aide de la **souris** ou des **touches fléchées**.
  * Vous **collez** l'élément.
  * Vous appliquez des **fonctions de formatage** à l'élément.
* **Remarque** : Si votre élément est **pivoté**, les propriétés de position reflètent la position de l'**élément non pivoté (original)**.
