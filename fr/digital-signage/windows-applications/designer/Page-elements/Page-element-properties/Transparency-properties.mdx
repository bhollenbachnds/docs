Ces propriétés déterminent le **level of transparency** de votre élément.

***

### **Transparence de l'élément**

* **Niveau de transparence**\
  Utilisez le **slider** ou **manually enter** une valeur pour définir le niveau de transparence.\
  **Valeurs possibles :**
  * **0%** – Pas de transparence (complètement opaque).
  * **100%** – Complètement transparent (invisible).
