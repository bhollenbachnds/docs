Ces propriétés déterminent le **niveau de transparence** de votre élément.

***

### **Transparence de l'élément**

* **Niveau de transparence**\
  Utilisez le **curseur** ou **saisissez manuellement** une valeur pour définir le niveau de transparence.\
  **Valeurs possibles :**
  * **0%** – Pas de transparence (complètement opaque).
  * **100%** – Complètement transparent (invisible).
