Les propriétés **Default Duration** définissent la durée d'affichage de chaque élément dans la **play list** de votre élément lorsqu'aucune durée spécifique n'est attribuée aux éléments individuels.

***

### Default Duration

* **Duration**\
  La **durée d'affichage par défaut** (en secondes) pour chaque élément de la play list.
  * Cette valeur s'applique lorsqu'aucune durée personnalisée n'est définie pour les éléments individuels de la play list.

***

### Remarques

* Lorsque le **défilement des données** est activé et que vous choisissez de **Scroll using play list durations**, cette **Default Duration** déterminera automatiquement **combien de temps chaque ligne de données est affichée** avant de défiler vers la ligne suivante.
* La **Default Duration** assure un timing cohérent pour les éléments lorsque les durées individuelles ne sont **pas spécifiées**.
* Si vous **définissez manuellement des durées** pour des éléments spécifiques de la play list, ces paramètres spécifiques **remplaceront** la durée par défaut pour ces éléments.
