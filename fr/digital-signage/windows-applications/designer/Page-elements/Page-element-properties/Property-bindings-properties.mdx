En définissant **des liaisons de propriétés**, vous pouvez lier des propriétés spécifiques d'un élément directement à une **source de données externe**. Une fois liée, la propriété de l'élément sera **mise à jour automatiquement** en temps réel lorsque la valeur dans la source de données change.

Par exemple, si vous liez la propriété **Visible** d'un élément à une valeur dans un fichier XML, l'élément s'affichera ou se masquera automatiquement lorsque cette valeur change.

***

### **Aperçu des liaisons de propriétés**

La **section Liaisons de propriétés** répertorie toutes les propriétés d'éléments pouvant être liées dynamiquement à une source de données. Pour configurer une liaison, vous devez spécifier **au moins** les éléments suivants :

* **Fournisseur de données** (la source externe)
* **Colonne** (le champ de données spécifique)
* **Ligne** (l'enregistrement de données)

***

### **Champs et définitions**

| **Champ**                  | **Description**                                                                                                                                                 |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Propriété**              | La propriété de l'élément à lier (définie par le système et ne peut pas être modifiée).                                                                         |
| **Type**                   | Le type de données de la propriété à lier (défini par le système et ne peut pas être modifié).                                                                  |
| **Fournisseur de données** | Le fournisseur de données représentant la source de données externe à lier (doit être ajouté au préalable).                                                     |
| **Colonne**                | Le champ de données (colonne) de la source de données à lier.                                                                                                   |
| **Ligne**                  | L'enregistrement spécifique (ligne) de la source de données à lier.                                                                                             |
| **Défilement**             | Activez cette option pour appliquer le défilement des données si souhaité. Si activé, **les propriétés de défilement des données** s'appliqueront à la liaison. |

***

### **Remarques importantes et limitations :**

* **Les options de liaison de propriétés varient selon le type d'élément.** Toutes les propriétés ne peuvent pas être liées pour chaque type d'élément.
* **Limitation sur le défilement de la durée de la liste de lecture :** Les liaisons de propriétés **ne** détectent pas et ne suivent pas le défilement basé sur la **durée de la liste de lecture locale**. Ce type de défilement est spécifique à l'élément et ne peut pas se synchroniser avec les liaisons de propriétés.
* **Limitation dans les éléments de tableau :** Lorsqu'un élément est placé dans un **élément de tableau**, les actualisations des données se produisent en synchronisation avec le **cycle d'actualisation du tableau**, et non de manière indépendante.

***

### **Tip:**

Pour plus d'informations sur la façon d'écrire des valeurs de liaison correctes, consultez la page **[Valeurs des liaisons de propriétés](#)** qui détaille les formats exacts et la notation pour les valeurs de liaison (par exemple, les booléens, les couleurs, les tailles).
