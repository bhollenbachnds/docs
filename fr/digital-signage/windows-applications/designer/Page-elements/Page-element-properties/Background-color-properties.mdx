Ces propriétés définissent la couleur d'arrière-plan d'un élément.

### Couleur d'arrière-plan

* **Color** :\
  Vous pouvez sélectionner une couleur d'arrière-plan pour votre élément.\
  Les options incluent :
  * **Fixed color** : Une couleur unique et solide.
  * **Gradient** : Une combinaison de deux ou trois couleurs mélangées ensemble.
  * **Texture or pattern** : Une texture graphique ou un motif répétitif.

Référez-vous à **"Using the color picker"** pour des instructions détaillées sur la sélection et la personnalisation de ces options.

***

### Remarques

* Si l'élément **supports transparency**, vous pouvez également choisir de rendre la couleur d'arrière-plan **fully transparent**.
* Dans certains cas, la **background color** ne sera visible que lorsque :
  * Il n'y a **no content** affiché dans l'élément.
  * Le contenu ne **ne couvre pas entièrement** la zone d'arrière-plan de l'élément.
