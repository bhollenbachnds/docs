Ces propriétés définissent la couleur d'arrière-plan d'un élément.

### Couleur d'arrière-plan

* **Couleur** :\
  Vous pouvez sélectionner une couleur d'arrière-plan pour votre élément.\
  Les options incluent :
  * **Couleur fixe** : Une couleur unique et unie.
  * **Dégradé** : Une combinaison de deux ou trois couleurs mélangées.
  * **Texture ou motif** : Une texture graphique ou un motif répétitif.

Reportez-vous à **"Utiliser le sélecteur de couleurs"** pour des instructions détaillées sur la sélection et la personnalisation de ces options.

***

### Remarques

* Si l'élément **prend en charge la transparence**, vous pouvez également choisir de rendre la couleur d'arrière-plan **entièrement transparente**.
* Dans certains cas, la **couleur d'arrière-plan** ne sera visible que lorsque :
  * Il n'y a **aucun contenu** affiché dans l'élément.
  * Le contenu ne **couvre pas entièrement** la zone d'arrière-plan de l'élément.
