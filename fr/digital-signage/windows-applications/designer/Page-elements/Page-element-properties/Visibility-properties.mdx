Ces propriétés déterminent la **visibility** de votre élément pendant la lecture.

***

### **Visibility**

* **Visible**\
  Lorsqu'il est **switched off**, l'élément **will not be displayed** pendant la lecture de votre présentation.

* **Hide when there is no Content**\
  Si **enabled**, l'élément sera **hidden automatically when there is no content** à afficher.

***

### **Remarques**

* **Not all elements** prennent en charge toutes ces propriétés.
* La propriété **Visible** est souvent utilisée avec des **element rules**, vous permettant de **show or hide** un élément selon certaines conditions.
* La propriété **Hide when there is no content** est utile pour les **picture elements** lors de la superposition de plusieurs images. Si l'image du dessus est vide, elle sera masquée, révélant les images en dessous. C'est utile pour les zones de contenu dynamique (par exemple, changer de logos selon le contexte).
* **Note:** Si un **video element** devient invisible, son **audio will also stop** de jouer.
