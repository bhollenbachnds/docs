Ces propriétés déterminent la **visibilité** de votre élément pendant la lecture.

***

### **Visibilité**

* **Visible**\
  Lorsque cette option est **désactivée**, l'élément **ne sera pas affiché** pendant la lecture de votre présentation.

* **Hide when there is no Content**\
  Si cette option est **activée**, l'élément sera **automatiquement masqué lorsqu'il n'y a pas de contenu** à afficher.

***

### **Remarques**

* **Tous les éléments ne** prennent pas en charge toutes ces propriétés.
* La propriété **Visible** est souvent utilisée avec les **element rules**, vous permettant de **montrer ou cacher** un élément en fonction de certaines conditions.
* La propriété **Hide when there is no content** est utile pour les **picture elements** lors de la superposition de plusieurs images. Si l'image du dessus est vide, elle sera masquée, révélant les images en dessous. Ceci est utile pour les zones de contenu dynamique (par exemple, changer de logos selon le contexte).
* **Note:** Si un **video element** devient invisible, son **audio s'arrêtera également** de jouer.
