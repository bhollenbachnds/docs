Ces propriétés déterminent la **visibilité** de votre élément pendant la lecture.

***

### **Visibilité**

* **Visible**\
  Lorsqu'il est **désactivé**, l'élément **ne sera pas affiché** pendant la lecture de votre présentation.

* **Masquer lorsqu'il n'y a pas de contenu**\
  Si **activé**, l'élément sera **masqué automatiquement lorsqu'il n'y a pas de contenu** à afficher.

***

### **Remarques**

* **Tous les éléments** ne prennent pas en charge toutes ces propriétés.
* La propriété **Visible** est souvent utilisée conjointement avec les **règles d'élément**, vous permettant d'**afficher ou masquer** un élément en fonction de certaines conditions.
* La propriété **Masquer lorsqu'il n'y a pas de contenu** est utile pour les **éléments image** lors de la superposition de plusieurs images. Si l'image du dessus est vide, elle sera masquée, révélant les images en dessous. Cela est utile pour les zones de contenu dynamique (par exemple, changer de logo selon le contexte).
* **Note:** Si un **élément vidéo** devient invisible, son **audio s'arrêtera également** de jouer.
