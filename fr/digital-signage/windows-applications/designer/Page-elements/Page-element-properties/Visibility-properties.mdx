Ces propriétés déterminent la **visibilité** de votre élément pendant la lecture.

***

### **Visibilité**

* **Visible**\
  Lorsque **désactivé**, l'élément **ne sera pas affiché** pendant la lecture de votre présentation.

* **Masquer lorsqu'il n'y a pas de contenu**\
  Si **activé**, l'élément sera **masqué automatiquement lorsqu'il n'y a pas de contenu** à afficher.

***

### **Remarques**

* **Tous les éléments ne** prennent pas en charge toutes ces propriétés.
* La **Visible** propriété est souvent utilisée avec les **element rules**, vous permettant de **montrer ou masquer** un élément selon certaines conditions.
* La **Hide when there is no content** propriété est utile pour les **picture elements** lors de la superposition de plusieurs images. Si l'image du dessus est vide, elle sera masquée, révélant les images en dessous. Ceci est utile pour les zones de contenu dynamique (par exemple, changer de logos selon le contexte).
* **Note:** Si un **video element** devient invisible, son **audio s'arrêtera également** de jouer.
