Ces propriétés définissent comment l'arrière-plan de l'élément de présentation apparaîtra.

#### **Couleur d'Arrière-plan**

* **Utiliser la Couleur d'Arrière-plan**\
  Si activé, la couleur spécifiée sera utilisée comme arrière-plan lorsqu'aucune présentation n'est affichée. Si des présentations sont affichées, leurs couleurs d'arrière-plan prévaudront.

* **Couleur**\
  Cliquez sur le bouton déroulant pour choisir une couleur d'arrière-plan. Vous pouvez sélectionner :

  * Une couleur unie
  * Un dégradé avec deux ou trois couleurs
  * Une texture ou un motif

  *(Voir "Utilisation du Sélecteur de Couleur" pour une description complète des options.)*

* **Pas de Couleur d'Arrière-plan**\
  Sélectionner cette option rend l'arrière-plan complètement transparent pour l'élément de présentation et toutes les présentations affichées.

#### **Remarques**

* Lorsque "Pas de Couleur d'Arrière-plan" est sélectionné, l'arrière-plan de la page où l'élément de présentation est placé sera toujours visible, même lorsque des présentations sont en cours de lecture.
* Si une présentation a sa propre couleur d'arrière-plan, l'activation de **Pas de Couleur d'Arrière-plan** la remplacera.

#### **Conseil**

Pour un effet partiellement transparent, définissez la propriété **Couleur** avec une valeur semi-transparente. Cela permet à l'arrière-plan de rester visible lorsqu'aucune présentation n'est en cours, tout en montrant les couleurs d'arrière-plan des présentations lorsqu'elles sont affichées.
