Ces propriétés définissent l'apparence de l'arrière-plan de l'élément de présentation.

#### **Couleur d'arrière-plan**

* **Utiliser la couleur d'arrière-plan**\
  Si activée, la couleur spécifiée sera utilisée comme arrière-plan lorsqu'aucune présentation n'est affichée. Si des présentations sont affichées, leurs couleurs d'arrière-plan seront prioritaires.

* **Couleur**\
  Cliquez sur le bouton déroulant pour choisir une couleur d'arrière-plan. Vous pouvez sélectionner :

  * Une couleur unie
  * Un dégradé avec deux ou trois couleurs
  * Une texture ou un motif

  *(Voir « Utilisation du sélecteur de couleurs » pour une description complète des options.)*

* **Aucune couleur d'arrière-plan**\
  La sélection de cette option rend l'arrière-plan complètement transparent pour l'élément de présentation et toutes les présentations affichées.

#### **Remarques**

* Lorsque « Aucune couleur d'arrière-plan » est sélectionné, l'arrière-plan de la page où l'élément de présentation est placé sera toujours visible, même lorsque des présentations sont en cours de lecture.
* Si une présentation possède sa propre couleur d'arrière-plan, l'activation de **Aucune couleur d'arrière-plan** la remplacera.

#### **Conseil**

Pour un effet partiellement transparent, définissez la propriété **Couleur** sur une valeur semi-transparente. Cela permet à l'arrière-plan de rester visible lorsqu'aucune présentation n'est en cours de lecture, tout en affichant les couleurs d'arrière-plan des présentations lorsqu'elles sont affichées.
