L'élément **Presentation in Page** vous permet d'intégrer une **présentation PADS4** existante dans une autre présentation. Cette fonctionnalité est utile pour **afficher des informations statiques** à côté de contenu dynamique.

Par exemple :

* Diffuser un **diaporama publicitaire** en haut tandis qu'un **ticker boursier** défile en bas.
* Jouer de la **musique de fond** pendant qu'une autre présentation s'exécute.

***

## **Aperçu des Propriétés**

| **Section de Propriété**                     | **Description**                                                                                       |
| -------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| **Alignement**                               | Définit l'alignement **horizontal et vertical** de la présentation intégrée.                          |
| **Options Alternatives**                     | Détermine comment l'élément bascule entre les éléments de la playlist.                                |
| **Couleur d'Arrière-plan**                   | Définit la couleur d'arrière-plan de l'élément.                                                       |
| **Bordure**                                  | Définit les propriétés de la bordure.                                                                 |
| **Fournisseur de Données**                   | Connecte l'élément à une **source de données externe** pour du contenu en temps réel.                 |
| **Conversion de Texte de Source de Données** | Spécifie comment l'élément **gère le texte multi-lignes**.                                            |
| **Durée par Défaut**                         | Définit la durée par défaut pour les éléments de la playlist.                                         |
| **Retournement**                             | Retourne la présentation intégrée **horizontalement ou verticalement**.                               |
| **Identification**                           | Identifie de manière unique l'élément dans la page.                                                   |
| **Liste de Lecture**                         | Définit **quelles présentations** seront affichées par cet élément.                                   |
| **Options de Liste de Lecture**              | Détermine **comment les présentations dans la liste de lecture sont jouées**.                         |
| **Position**                                 | Définit les **coordonnées X et Y** dans la page.                                                      |
| **Proof of Play**                            | Crée un **journal indiquant quand et où l'élément a été diffusé**.                                    |
| **Liaisons de propriétés**                   | Permet **la mise à jour dynamique de certaines propriétés** à partir d'une source de données externe. |
| **Réflexion**                                | Affiche la présentation avec un **effet de réflexion**.                                               |
| **Rotation**                                 | Fait pivoter l'élément selon un angle spécifié.                                                       |
| **Règles**                                   | Définit des **conditions** pour l'apparition ou la disparition de la présentation.                    |
| **Taille**                                   | Définit la **largeur et la hauteur** de l'élément.                                                    |
| **Inclinaison**                              | Ajuste l'**angle d'affichage**.                                                                       |
| **Transparence**                             | Contrôle **le niveau de transparence** de l'élément.                                                  |
| **Visibilité**                               | Détermine si l'élément est **visible pendant la lecture**.                                            |

***

### **Remarques importantes**

* L'élément **Presentation in Page** s'exécute **inside the PADS4 Viewer** mais ne charge pas les fournisseurs de données **lors de l'édition** dans le PADS4 Designer. Cela garantit une meilleure **performance pendant la conception**.
* Les présentations intégrées **n'interfèrent pas** avec les autres éléments de la page et peuvent s'exécuter **simultanément**.
* **Plusieurs présentations intégrées** peuvent être utilisées ensemble pour des **mises en page complexes**.
