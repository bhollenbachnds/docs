L'élément **Presentation in Page** vous permet d'intégrer une **présentation PADS4** existante dans une autre présentation. Cette fonctionnalité est utile pour **afficher des informations statiques** en parallèle avec du contenu dynamique.

Par exemple :

* Exécuter un **diaporama publicitaire** en haut pendant qu'un **ticker boursier** défile en bas.
* Jouer de la **musique de fond** pendant qu'une autre présentation s'exécute.

***

## **Aperçu des propriétés**

| **Section de propriété**                     | **Description**                                                                                           |
| -------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| **Alignement**                               | Définit l' **alignement horizontal et vertical** de la présentation intégrée.                             |
| **Options alternatives**                     | Détermine comment l'élément bascule entre les éléments de la liste de lecture.                            |
| **Couleur d'arrière-plan**                   | Définit la couleur d'arrière-plan de l'élément.                                                           |
| **Bordure**                                  | Définit les propriétés de la bordure.                                                                     |
| **Fournisseur de données**                   | Connecte l'élément à une **source de données externe** pour du contenu en temps réel.                     |
| **Conversion de texte de source de données** | Spécifie comment l'élément **gère le texte multiligne**.                                                  |
| **Durée par défaut**                         | Définit la durée par défaut des éléments dans la liste de lecture.                                        |
| **Retournement**                             | Retourne la présentation intégrée **horizontalement ou verticalement**.                                   |
| **Identification**                           | Identifie de manière unique l'élément dans la page.                                                       |
| **Liste de lecture**                         | Définit **quelles présentations** seront affichées par cet élément.                                       |
| **Options de liste de lecture**              | Détermine **comment les présentations dans la liste de lecture sont lues**.                               |
| **Position**                                 | Définit les **coordonnées X et Y** dans la page.                                                          |
| **Preuve de lecture**                        | Crée un **journal indiquant quand et où l'élément a été lu**.                                             |
| **Liaisons de propriétés**                   | Permet **à certaines propriétés d'être mises à jour dynamiquement** depuis une source de données externe. |
| **Reflet**                                   | Affiche la présentation avec un **effet de reflet**.                                                      |
| **Rotation**                                 | Fait pivoter l'élément selon un angle spécifié.                                                           |
| **Règles**                                   | Définit des **conditions** pour l'apparition ou la disparition de la présentation.                        |
| **Taille**                                   | Définit la **largeur et la hauteur** de l'élément.                                                        |
| **Inclinaison**                              | Ajuste l' **angle d'affichage**.                                                                          |
| **Transparence**                             | Contrôle **le niveau de transparence** de l'élément.                                                      |
| **Visibilité**                               | Détermine si l'élément est **visible pendant la lecture**.                                                |

***

### **Remarques importantes**

* L'élément **Presentation in Page** s'exécute **dans le PADS4 Viewer** mais ne charge pas les fournisseurs de données **lors de l'édition** dans le PADS4 Designer. Cela garantit de meilleures **performances lors de la conception**.
* Les présentations intégrées **n'interfèrent pas** avec les autres éléments de la page et peuvent s'exécuter **simultanément**.
* **Plusieurs présentations intégrées** peuvent être utilisées ensemble pour **mises en page complexes**.
