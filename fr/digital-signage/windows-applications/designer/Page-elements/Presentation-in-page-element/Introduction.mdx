L'élément **Presentation in Page** vous permet d'intégrer une **PADS4 presentation** existante dans une autre présentation. Cette fonctionnalité est utile pour **afficher des informations statiques** à côté de contenu dynamique.

Par exemple :

* Diffuser un **diaporama publicitaire** en haut pendant qu'un **ticker boursier** défile en bas.
* Jouer de la **musique de fond** pendant qu'une autre présentation s'exécute.

***

## **Aperçu des propriétés**

| **Section de propriété**        | **Description**                                                                                               |
| ------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| **Alignment**                   | Définit l'alignement **horizontal et vertical** de la présentation intégrée.                                  |
| **Alternate Options**           | Détermine comment l'élément bascule entre les éléments de la playlist.                                        |
| **Background Color**            | Définit la couleur d'arrière-plan de l'élément.                                                               |
| **Border**                      | Définit les propriétés de la bordure.                                                                         |
| **Data Provider**               | Connecte l'élément à une **source de données externe** pour du contenu en temps réel.                         |
| **Data Source Text Conversion** | Spécifie comment l'élément **gère le texte multi-lignes**.                                                    |
| **Default Duration**            | Définit la durée par défaut des éléments dans la playlist.                                                    |
| **Flip**                        | Retourne la présentation intégrée **horizontalement ou verticalement**.                                       |
| **Identification**              | Identifie de manière unique l'élément dans la page.                                                           |
| **Play List**                   | Définit **quelles présentations** seront affichées par cet élément.                                           |
| **Play List Options**           | Détermine **comment les présentations dans la playlist sont lues**.                                           |
| **Position**                    | Définit les **coordonnées X et Y** dans la page.                                                              |
| **Proof of Play**               | Crée un **journal indiquant quand et où l'élément a été diffusé**.                                            |
| **Property Bindings**           | Permet à **certaines propriétés d'être mises à jour dynamiquement** à partir d'une source de données externe. |
| **Reflection**                  | Affiche la présentation avec un **effet de réflexion**.                                                       |
| **Rotation**                    | Fait pivoter l'élément selon un angle spécifié.                                                               |
| **Rules**                       | Définit des **conditions** pour l'apparition ou la disparition de la présentation.                            |
| **Size**                        | Définit la **largeur et la hauteur** de l'élément.                                                            |
| **Skew**                        | Ajuste l'**angle d'affichage**.                                                                               |
| **Transparency**                | Contrôle **le niveau de transparence** de l'élément.                                                          |
| **Visibility**                  | Détermine si l'élément est **visible pendant la lecture**.                                                    |

***

### **Remarques importantes**

* L'élément **Presentation in Page** s'exécute **dans le PADS4 Viewer** mais ne charge pas les fournisseurs de données **lors de l'édition** dans le PADS4 Designer. Cela assure une meilleure **performance pendant la conception**.
* Les présentations intégrées **n'interfèrent pas** avec d'autres éléments de la page et peuvent s'exécuter **simultanément**.
* **Plusieurs présentations intégrées** peuvent être utilisées ensemble pour des **mises en page complexes**.
