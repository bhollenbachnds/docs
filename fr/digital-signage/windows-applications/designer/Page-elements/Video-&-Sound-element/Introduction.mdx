L'élément **Video/Sound** permet la lecture de **fichiers vidéo et/ou audio** dans une présentation multimédia. Il prend en charge les **formats de fichiers standard de l'industrie** et comprend diverses options pour **la personnalisation de la lecture**, les **effets** et **l'intégration de données en temps réel**.

***

## **Aperçu des propriétés**

| **Section de propriété**        | **Description**                                                                                        |
| ------------------------------- | ------------------------------------------------------------------------------------------------------ |
| **Alignment**                   | Définit l'alignement **horizontal et vertical** des vidéos.                                            |
| **Alternate Options**           | Définit comment l'élément **bascule entre les éléments de la playlist**.                               |
| **Background Color**            | Spécifie la **couleur d'arrière-plan** de l'élément.                                                   |
| **Border**                      | Définit la **couleur, la largeur et le style de la bordure**.                                          |
| **Data Provider**               | Connecte l'élément à une **source de données externe** pour les mises à jour de contenu en temps réel. |
| **Data Source Text Conversion** | Définit comment l'élément **traite le texte multi-lignes** provenant des sources de données.           |
| **Default Duration**            | Définit la **durée de lecture par défaut** pour les éléments de la playlist.                           |
| **Default Transition Effect**   | Détermine **l'effet de transition d'entrée** pour le contenu.                                          |
| **Fondu**                       | Active**l'apparition et la disparition en fondu** des effets sonores.                                  |
| **Retournement**                | Retourne la vidéo **horizontalement ou verticalement**.                                                |
| **Identification**              | Attribue un **identifiant unique** à l'élément dans la page.                                           |
| **Play List**                   | Spécifie les **vidéos et sons** à lire.                                                                |
| **Play List Options**           | Contrôle **comment les éléments de la playlist sont lus**.                                             |
| **Position**                    | Détermine le **placement exact** de l'élément dans la page.                                            |
| **Proof of Play**               | Enregistre les informations de lecture pour **des fins de suivi**.                                     |
| **Property Bindings**           | Lie les propriétés à une **source de données externe** pour des mises à jour en temps réel.            |
| **Réflexion**                   | Affiche l'élément **avec un effet de réflexion**.                                                      |
| **Rotation**                    | Définit un **angle de rotation** pour l'élément.                                                       |
| **Rules**                       | Définit des **comportements automatisés** basés sur des conditions.                                    |
| **Taille**                      | Détermine la **largeur et la hauteur** de l'élément.                                                   |
| **Inclinaison**                 | Permet un **placement en angle** du contenu.                                                           |
| **Slide Show Options**          | Active la **fonction pause et reprise** pour les vidéos.                                               |
| **Speed Ratio**                 | Ajuste la **vitesse de lecture** des vidéos.                                                           |
| **Streaming Video**             | Active **la lecture vidéo en streaming**.                                                              |
| **Transparence**                | Ajuste le **niveau de transparence** de l'élément.                                                     |
| **Visibilité**                  | Contrôle si **l'élément est visible pendant la lecture**.                                              |
| **Performance Settings**        | Fournit des **options d'optimisation** pour les performances vidéo.                                    |
| **Volume**                      | Définit le **niveau audio** pour les vidéos et les sons.                                               |

***

## **Remarques**

* Cette section **couvre uniquement les propriétés spécifiques aux vidéos/sons**.
* Une description détaillée des **propriétés communes** (par exemple, positionnement, visibilité, transitions) se trouve dans la section **Page Element Properties**.
