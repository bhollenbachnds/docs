L'élément **Video/Sound** permet la lecture de **fichiers vidéo et/ou audio** dans une présentation multimédia. Il prend en charge les **formats de fichiers standard de l'industrie** et comprend diverses options pour **la personnalisation de la lecture**, **les effets**, et **l'intégration de données en temps réel**.

***

## **Aperçu des propriétés**

| **Section de propriété**                     | **Description**                                                                                        |
| -------------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| **Alignement**                               | Définit l'alignement **horizontal et vertical** des vidéos.                                            |
| **Options alternatives**                     | Définit comment l'élément **bascule entre les éléments de la playlist**.                               |
| **Couleur d'arrière-plan**                   | Spécifie la **couleur d'arrière-plan** de l'élément.                                                   |
| **Bordure**                                  | Définit la **couleur, la largeur et le style de la bordure**.                                          |
| **Fournisseur de données**                   | Connecte l'élément à une **source de données externe** pour les mises à jour de contenu en temps réel. |
| **Conversion de texte de source de données** | Définit comment l'élément **traite le texte multi-lignes** provenant des sources de données.           |
| **Durée par défaut**                         | Définit la **durée de lecture par défaut** pour les éléments de la playlist.                           |
| **Effet de transition par défaut**           | Détermine l'**effet de transition d'entrée** pour le contenu.                                          |
| **Fondu**                                    | Active les effets de **fondu d'entrée et de sortie** pour le son.                                      |
| **Retournement**                             | Retourne la vidéo **horizontalement ou verticalement**.                                                |
| **Identification**                           | Attribue un **identifiant unique** à l'élément dans la page.                                           |
| **Liste de lecture**                         | Spécifie les **vidéos et sons** à lire.                                                                |
| **Options de liste de lecture**              | Contrôle **comment les éléments de la liste de lecture sont lus**.                                     |
| **Position**                                 | Détermine le **placement exact** de l'élément dans la page.                                            |
| **Proof of Play**                            | Enregistre les informations de lecture pour **des fins de suivi**.                                     |
| **Liaisons de propriétés**                   | Lie les propriétés à une **source de données externe** pour des mises à jour en temps réel.            |
| **Réflexion**                                | Affiche l'élément **avec un effet de réflexion**.                                                      |
| **Rotation**                                 | Définit un **angle de rotation** pour l'élément.                                                       |
| **Règles**                                   | Définit **des comportements automatisés** basés sur des conditions.                                    |
| **Taille**                                   | Détermine la **largeur et la hauteur** de l'élément.                                                   |
| **Inclinaison**                              | Permet un **placement incliné** du contenu.                                                            |
| **Options de diaporama**                     | Active la **fonction pause et reprise** pour les vidéos.                                               |
| **Ratio de vitesse**                         | Ajuste la **vitesse de lecture** des vidéos.                                                           |
| **Vidéo en streaming**                       | Active **la lecture vidéo en streaming**.                                                              |
| **Transparence**                             | Ajuste le **niveau de transparence** de l'élément.                                                     |
| **Visibilité**                               | Contrôle si **l'élément est visible pendant la lecture**.                                              |
| **Paramètres de performance**                | Fournit **des options d'optimisation** pour les performances vidéo.                                    |
| **Volume**                                   | Définit le **niveau audio** pour les vidéos et les sons.                                               |

***

## **Remarques**

* Cette section **couvre uniquement les propriétés spécifiques aux vidéos/sons**.
* Une description détaillée des **propriétés communes** (par exemple, positionnement, visibilité, transitions) se trouve dans la section **Page Element Properties**.
