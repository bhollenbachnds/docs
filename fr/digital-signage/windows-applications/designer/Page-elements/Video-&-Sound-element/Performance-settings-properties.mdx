Ces propriétés permettent d' **optimiser les performances de lecture vidéo** lors de l'utilisation de **lecteurs Microsoft Windows**, particulièrement utile pour les **matériels peu performants** ou les **configurations multi-écrans**.

***

## **Paramètres d'optimisation des performances**

### **Limit Video Element Functionalities**

* **Améliore la vitesse de lecture vidéo** en désactivant certains **effets graphiques**.
* Recommandé lors de l'utilisation de **plusieurs écrans** ou d'un **matériel basse consommation**.
* **L'activation de cette option désactivera :**

| **Fonctionnalité**           | **Effet**                                                                                                                                                               |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Effets vidéo**             | Aucun effet de transition ne peut être appliqué.                                                                                                                        |
| **Étirement**                | Le ratio d'aspect est toujours préservé. Si la vidéo ne s'adapte pas à la zone, des bordures noires apparaîtront. Utilisez **Uniform** pour éviter les bordures noires. |
| **Always on Top**            | La vidéo reste **au-dessus de tous les autres éléments**.                                                                                                               |
| **Transparence**             | Non prise en charge.                                                                                                                                                    |
| **Rotation**                 | La vidéo **ne peut pas être pivotée** librement. Elle suivra uniquement la rotation de l'affichage.                                                                     |
| **Retournement/Inclinaison** | Le retournement et l'inclinaison sont **impossibles**.                                                                                                                  |
| **Reflet**                   | Les effets de reflet **ne peuvent pas être appliqués**.                                                                                                                 |

***

## **Optimisation pour mur vidéo et multi-écrans**

* Lors de la création d'un **mur vidéo** ou de l'utilisation de **plusieurs écrans** sur un **lecteur unique**, Windows peut provoquer des **baisses de performances** lors du franchissement des limites d'écran.
* **Solution:**
  * Créez un **grand canevas virtuel** (par ex., **2× la largeur et 2× la hauteur** d'un seul écran).
  * Divisez l'affichage en **régions séparées**.
  * **Activez « Limit Video Element Functionalities »** pour maintenir l'**accélération GPU** et éviter les pertes d'images.

***

## **Bonnes pratiques pour les configurations multi-écrans**

* Si un **seul PC pilote plusieurs écrans**, activer **« Limit Video Element Functionalities »** améliore considérablement les **performances**.
* Évitez de franchir les **limites d'affichage** dans les **configurations multi-moniteurs Windows** lors de la lecture de vidéos.
