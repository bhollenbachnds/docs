Ces propriétés vous permettent de **zoomer** et de **rogner** le contenu dans votre élément Flash player, offrant un meilleur contrôle sur la façon dont le contenu Flash apparaît dans l'élément.

***

### **Niveau de zoom**

#### **Zoom**

* Ajustez le **niveau de zoom** du contenu Flash.
* Vous pouvez **saisir manuellement un pourcentage** ou utiliser le **curseur** pour contrôler le zoom.
* Cela vous permet de **mettre à l'échelle** le contenu Flash pour l'adapter ou mettre en évidence des parties spécifiques du contenu.

***

#### **Décalage**

* Les **Décalage** paramètres vous permettent de **rogner** et de positionner le contenu Flash dans l'élément player.
* **Gauche** : Définit le décalage horizontal (en pixels) — déplace le contenu vers la gauche ou la droite dans l'élément.
* **Haut** : Définit le décalage vertical (en pixels) — déplace le contenu vers le haut ou vers le bas dans l'élément.
* Ces décalages déterminent **quelle partie** du contenu Flash sera affichée, à partir de la position spécifiée.

***

### **Conseil d'utilisation**

* **Zoom** et **Décalage** peuvent être combinés pour **se concentrer** sur des zones spécifiques du contenu Flash ou pour ajuster la façon dont il s'intègre dans le cadre de l'élément.
