**Les éléments Flash player** sont utilisés pour inclure du **contenu Flash** dans votre présentation.\
Ces éléments offrent plusieurs options pour contrôler comment le contenu Flash doit être lu.\
De plus, les éléments Flash peuvent être **connectés à des sources de données externes**, permettant des **mises à jour en temps réel** des fichiers Flash à lire.

***

### **Aperçu des propriétés**

| **Section de propriété**                     | **Description**                                                                                                     |
| -------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Options alternatives**                     | Définir comment l'élément bascule entre les éléments de la liste de lecture.                                        |
| **Couleur d'arrière-plan**                   | Définir la couleur d'arrière-plan de l'élément Flash.                                                               |
| **Fournisseur de données**                   | Connecter l'élément à une **source de données en temps réel** pour alimenter dynamiquement le contenu Flash.        |
| **Conversion de texte de source de données** | Définir comment le texte multiligne est traité lors de l'utilisation de sources de données.                         |
| **Identification**                           | Identifier de manière unique l'élément dans la page.                                                                |
| **Interactivité**                            | Activer ou désactiver l'**interaction utilisateur** avec le contenu Flash.                                          |
| **Liste de lecture**                         | Définir les fichiers Flash qui seront affichés par l'élément.                                                       |
| **Options de liste de lecture**              | Déterminer comment les éléments de la liste de lecture sont lus (par exemple, en boucle, ordre aléatoire).          |
| **Position**                                 | Définir la position de l'élément dans la mise en page.                                                              |
| **Preuve de lecture**                        | Activer la journalisation pour **prouver la lecture** du contenu Flash à des fins d'audit.                          |
| **Liaisons de propriétés**                   | Lier les propriétés de l'élément à des sources de données externes pour un contrôle dynamique.                      |
| **Règles**                                   | Définir des règles spécifiques à l'élément pour ajuster automatiquement le comportement en fonction des conditions. |
| **Taille**                                   | Définir la largeur et la hauteur de l'élément Flash.                                                                |
| **Visibilité**                               | Contrôler si l'élément et son contenu sont visibles pendant la lecture.                                             |
| **Niveau de zoom**                           | Définir les options de zoom et de recadrage pour l'affichage du contenu Flash.                                      |

***

### **Remarque**

* Cette section décrit **uniquement** les **propriétés spécifiques à Flash**.
* Pour les **propriétés communes** qui s'appliquent à tous les éléments de page (comme la position, la taille, la visibilité, les règles, etc.), consultez la section **Propriétés des éléments de page**.
