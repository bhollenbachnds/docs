**Flash player elements** sont utilisés pour inclure du **Flash content** dans votre présentation.\
Ces éléments offrent plusieurs options pour contrôler comment le contenu Flash doit être lu.\
De plus, les éléments Flash peuvent être **connected to external data sources**, permettant des **real-time updates** des fichiers Flash à lire.

***

### **Aperçu des propriétés**

| **Section de propriété**        | **Description**                                                                                                     |
| ------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Alternate options**           | Définir comment l'élément bascule entre les éléments de la liste de lecture.                                        |
| **Background color**            | Définir la couleur d'arrière-plan de l'élément Flash.                                                               |
| **Data provider**               | Connecter l'élément à une **real-time data source** pour alimenter dynamiquement le contenu Flash.                  |
| **Data source text conversion** | Définir comment le texte multi-lignes est géré lors de l'utilisation de sources de données.                         |
| **Identification**              | Identifier de manière unique l'élément dans la page.                                                                |
| **Interactivity**               | Activer ou désactiver **user interaction** avec le contenu Flash.                                                   |
| **Play list**                   | Définir les fichiers Flash qui seront affichés par l'élément.                                                       |
| **Play list options**           | Déterminer comment les éléments de la liste de lecture sont lus (par exemple, en boucle, ordre aléatoire).          |
| **Position**                    | Définir la position de l'élément dans la mise en page.                                                              |
| **Proof of play**               | Activer la journalisation pour **prove playback** du contenu Flash à des fins d'audit.                              |
| **Property bindings**           | Lier les propriétés de l'élément à des sources de données externes pour un contrôle dynamique.                      |
| **Rules**                       | Définir des règles spécifiques à l'élément pour ajuster automatiquement le comportement en fonction des conditions. |
| **Size**                        | Définir la largeur et la hauteur de l'élément Flash.                                                                |
| **Visibility**                  | Contrôler si l'élément et son contenu sont visibles pendant la lecture.                                             |
| **Zoom level**                  | Définir les options de zoom et de recadrage pour l'affichage du contenu Flash.                                      |

***

### **Remarque**

* Cette section **only** décrit les **Flash-specific properties**.
* Pour les **common properties** qui s'appliquent à tous les éléments de page (comme la position, la taille, la visibilité, les règles, etc.), consultez la section **Page Element Properties**.
