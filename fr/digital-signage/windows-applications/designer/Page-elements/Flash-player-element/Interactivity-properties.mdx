Ces propriétés contrôlent si les utilisateurs peuvent **interagir** avec le contenu Flash affiché dans l'élément lecteur Flash.

### **Interactivité utilisateur**

* **Autoriser l'interactivité utilisateur**:
  * **Activer** cette option pour permettre aux utilisateurs d'interagir avec le contenu Flash.
  * Cela peut inclure des clics sur des **hyperliens**, **boutons**, ou d'autres composants interactifs faisant partie du fichier Flash.
  * **Désactiver** cette option si vous souhaitez que le contenu Flash soit lu sans autoriser aucune saisie ou interaction de la part de l'utilisateur.
