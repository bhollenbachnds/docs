Ces propriétés contrôlent si les utilisateurs peuvent **interact** avec le contenu Flash affiché dans l'élément Flash player.

### **User Interactivity**

* **Allow User Interactivity**:
  * **Enable** cette option pour permettre aux utilisateurs d'interagir avec le contenu Flash.
  * Cela peut inclure le clic sur des **hyperlinks**, **buttons**, ou d'autres composants interactifs qui font partie du fichier Flash.
  * **Disable** cette option si vous souhaitez que le contenu Flash soit lu sans permettre aucune entrée ou interaction de l'utilisateur.
