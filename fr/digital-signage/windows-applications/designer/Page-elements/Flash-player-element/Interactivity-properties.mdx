Ces propriétés contrôlent si les utilisateurs peuvent **interagir** avec le contenu Flash affiché dans l'élément lecteur Flash.

### **Interactivité utilisateur**

* **Autoriser l'interactivité utilisateur** :
  * **Activez** cette option pour permettre aux utilisateurs d'interagir avec le contenu Flash.
  * Cela peut inclure le clic sur des **hyperliens**, **boutons**, ou d'autres composants interactifs qui font partie du fichier Flash.
  * **Désactivez** cette option si vous souhaitez que le contenu Flash soit lu sans permettre aucune entrée ou interaction de l'utilisateur.
