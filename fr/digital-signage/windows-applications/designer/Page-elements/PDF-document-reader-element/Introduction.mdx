Le **PDF Document Reader** vous permet d'**intégrer et d'afficher des fichiers PDF** dans votre présentation. Les PDF peuvent être affichés en **taille originale** ou **redimensionnés** pour s'adapter à votre mise en page. De plus, les pages peuvent être **défilées automatiquement** avec une durée définie.

***

## **Fonctionnalités clés et limitations**

* **Prise en charge de la plupart des formats PDF** (sauf les **modèles 3D**).
* **Prise en charge du contenu dynamique** en se connectant à une **source de données externe**.
* **Effets et transitions** peuvent être appliqués pour des changements visuels plus fluides.

***

## **Aperçu des propriétés**

| **Section de propriété**                     | **Description**                                                                             |
| -------------------------------------------- | ------------------------------------------------------------------------------------------- |
| **Alignement**                               | Ajustez l'**alignement horizontal et vertical** du contenu.                                 |
| **Options alternatives**                     | Configurez la façon dont l'élément bascule entre les **éléments de la liste de lecture**.   |
| **Couleur d'arrière-plan**                   | Définissez une **couleur d'arrière-plan** pour l'élément PDF.                               |
| **Bordure**                                  | Définissez une **bordure** autour de l'élément.                                             |
| **Fournisseur de données**                   | Connectez-vous à une **source de données externe** pour des mises à jour PDF en temps réel. |
| **Conversion de texte de source de données** | Gérez la façon dont l'**élément gère le texte multiligne**.                                 |
| **Durée par défaut**                         | Définissez une **durée d'affichage par défaut** pour les éléments de la liste de lecture.   |
| **Effet de transition par défaut**           | Appliquez des **effets de transition entrants** au contenu.                                 |
| **Retournement**                             | Retournez le **contenu horizontalement ou verticalement**.                                  |
| **Identification**                           | Attribuez un **identifiant unique** à l'élément.                                            |
| **Liste de lecture**                         | Définissez **quels fichiers PDF** seront affichés.                                          |
| **Options de liste de lecture**              | Définissez **comment les éléments de la liste de lecture** sont lus.                        |
| **Position**                                 | Ajustez le **placement** sur la page.                                                       |
| **Preuve de diffusion**                      | Enregistrez la **preuve de diffusion** pour les rapports.                                   |
| **Liaisons de propriétés**                   | Liez des propriétés à une **source de données externe**.                                    |
| **Reflet**                                   | Ajoutez un **effet de reflet** au contenu.                                                  |
| **Rotation**                                 | Faites pivoter l'**élément PDF selon un angle**.                                            |
| **Règles**                                   | Définissez des **règles pour la visibilité automatisée**.                                   |
| **Taille**                                   | Définir la **largeur et la hauteur** de l'élément.                                          |
| **Inclinaison**                              | Appliquer des **effets d'inclinaison** au contenu.                                          |
| **Transparence**                             | Ajuster le **niveau de transparence** de l'élément.                                         |
| **Visibilité**                               | Contrôler si l'**élément est visible pendant la lecture**.                                  |

***

## **Notes supplémentaires**

* **Les PDF peuvent être mis à jour dynamiquement** à l'aide d'une source de données externe.
* **Lecture et défilement** peuvent être automatisés à l'aide des **options de playlist**.
* **Les transitions et effets de contenu** peuvent améliorer l'expérience d'affichage.

En utilisant le **PDF Document Reader**, vous pouvez intégrer de manière transparente **du contenu basé sur des documents** dans votre présentation avec **des mises à jour dynamiques et des transitions fluides**.
