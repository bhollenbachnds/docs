Le **PDF Document Reader** vous permet d'**intégrer et d'afficher des fichiers PDF** dans votre présentation. Les PDF peuvent être affichés en **taille originale** ou **redimensionnés** pour s'adapter à votre mise en page. De plus, les pages peuvent être **automatiquement défilées** avec une durée définie.

***

## **Fonctionnalités clés et limitations**

* **Prend en charge la plupart des formats PDF** (sauf **modèles 3D**).
* **Prise en charge du contenu dynamique** en se connectant à une **source de données externe**.
* **Effets et transitions** peuvent être appliqués pour des changements visuels plus fluides.

***

## **Aperçu des propriétés**

| **Section de propriété**                     | **Description**                                                                           |
| -------------------------------------------- | ----------------------------------------------------------------------------------------- |
| **Alignement**                               | Ajuster **l'alignement horizontal et vertical** du contenu.                               |
| **Options alternatives**                     | Configurer comment l'élément bascule entre **les éléments de la playlist**.               |
| **Couleur d'arrière-plan**                   | Définir une **couleur d'arrière-plan** pour l'élément PDF.                                |
| **Bordure**                                  | Définir une **bordure** autour de l'élément.                                              |
| **Fournisseur de données**                   | Se connecter à une **source de données externe** pour des mises à jour PDF en temps réel. |
| **Conversion de texte de source de données** | Gérer comment **l'élément gère le texte multi-lignes**.                                   |
| **Durée par défaut**                         | Définir un **temps d'affichage par défaut** pour les éléments de la playlist.             |
| **Effet de transition par défaut**           | Appliquer **des effets de transition entrants** au contenu.                               |
| **Retournement**                             | Retourner le **contenu horizontalement ou verticalement**.                                |
| **Identification**                           | Attribuer un **identifiant unique** à l'élément.                                          |
| **Liste de lecture**                         | Définir **quels fichiers PDF** seront affichés.                                           |
| **Options de liste de lecture**              | Définir **comment les éléments de la playlist** sont lus.                                 |
| **Position**                                 | Ajuster **le placement** sur la page.                                                     |
| **Preuve de diffusion**                      | Enregistrer **la preuve de diffusion** pour les rapports.                                 |
| **Liaisons de propriétés**                   | Lier les propriétés à une **source de données externe**.                                  |
| **Réflexion**                                | Ajouter un **effet de réflexion** au contenu.                                             |
| **Rotation**                                 | Faire pivoter **l'élément PDF selon un angle**.                                           |
| **Règles**                                   | Définir **des règles pour la visibilité automatisée**.                                    |
| **Taille**                                   | Définir la **largeur et la hauteur** de l'élément.                                        |
| **Inclinaison**                              | Appliquer **des effets d'inclinaison** au contenu.                                        |
| **Transparence**                             | Ajuster le **niveau de transparence** de l'élément.                                       |
| **Visibilité**                               | Contrôler si **l'élément est visible pendant la lecture**.                                |

***

## **Notes supplémentaires**

* **Les PDF peuvent être mis à jour dynamiquement** à l'aide d'une source de données externe.
* **La lecture et le défilement** peuvent être automatisés en utilisant **les options de playlist**.
* **Les transitions et effets de contenu** peuvent améliorer l'expérience d'affichage.

En utilisant le **PDF Document Reader**, vous pouvez intégrer de manière transparente **du contenu basé sur des documents** dans votre présentation avec **des mises à jour dynamiques et des transitions fluides**.
