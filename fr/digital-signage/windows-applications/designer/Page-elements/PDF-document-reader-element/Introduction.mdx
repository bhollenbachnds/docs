Le **PDF Document Reader** vous permet d'**intégrer et afficher des fichiers PDF** dans votre présentation. Les PDF peuvent être affichés en **taille originale** ou **redimensionnés** pour s'adapter à votre mise en page. De plus, les pages peuvent être **défilées automatiquement** avec une durée définie.

***

## **Fonctionnalités clés et limitations**

* **Prend en charge la plupart des formats PDF** (sauf les **modèles 3D**).
* **Prise en charge du contenu dynamique** en se connectant à une **source de données externe**.
* **Effets et transitions** peuvent être appliqués pour des changements visuels plus fluides.

***

## **Aperçu des propriétés**

| **Section de propriété**        | **Description**                                                                           |
| ------------------------------- | ----------------------------------------------------------------------------------------- |
| **Alignment**                   | Ajuster l'**alignement horizontal et vertical** du contenu.                               |
| **Alternate Options**           | Configurez comment l'élément bascule entre les **éléments de la playlist**.               |
| **Background Color**            | Définir une **couleur d'arrière-plan** pour l'élément PDF.                                |
| **Border**                      | Définir une **bordure** autour de l'élément.                                              |
| **Data Provider**               | Se connecter à une **source de données externe** pour des mises à jour PDF en temps réel. |
| **Data Source Text Conversion** | Gérer comment l'**élément gère le texte multi-lignes**.                                   |
| **Default Duration**            | Définir un **temps d'affichage par défaut** pour les éléments de la playlist.             |
| **Default Transition Effect**   | Appliquer des **effets de transition entrants** au contenu.                               |
| **Flip**                        | Retourner le **contenu horizontalement ou verticalement**.                                |
| **Identification**              | Attribuer un **identifiant unique** à l'élément.                                          |
| **Play List**                   | Définir **quels fichiers PDF** seront affichés.                                           |
| **Play List Options**           | Définir **comment les éléments de la playlist** sont lus.                                 |
| **Position**                    | Ajuster le **placement** sur la page.                                                     |
| **Proof of Play**               | Enregistrer la **preuve de diffusion** pour les rapports.                                 |
| **Property Bindings**           | Lier les propriétés à une **source de données externe**.                                  |
| **Reflection**                  | Ajouter un **effet de réflexion** au contenu.                                             |
| **Rotation**                    | Faire pivoter l'**élément PDF selon un angle**.                                           |
| **Rules**                       | Définir des **règles pour la visibilité automatisée**.                                    |
| **Size**                        | Définir la **largeur et la hauteur** de l'élément.                                        |
| **Skew**                        | Appliquer des **effets d'inclinaison** au contenu.                                        |
| **Transparency**                | Ajuster le **niveau de transparence** de l'élément.                                       |
| **Visibility**                  | Contrôler si l'**élément est visible pendant la lecture**.                                |

***

## **Notes supplémentaires**

* **Les PDF peuvent être mis à jour dynamiquement** à l'aide d'une source de données externe.
* **La lecture et le défilement** peuvent être automatisés en utilisant les **playlist options**.
* **Les transitions et effets de contenu** peuvent améliorer l'expérience d'affichage.

En utilisant le **PDF Document Reader**, vous pouvez intégrer parfaitement du **contenu basé sur des documents** dans votre présentation avec des **mises à jour dynamiques et des transitions fluides**.
