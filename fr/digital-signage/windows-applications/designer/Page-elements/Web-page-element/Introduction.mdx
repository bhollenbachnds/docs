L'élément **Web Page Element** vous permet d'afficher du **Internet or local web content** dans une présentation. Il prend en charge **Microsoft Internet Explorer** pour le rendu des pages web. Cet élément peut être utilisé pour du contenu web **interactive** ou **static**, avec la possibilité de restreindre l'accès des utilisateurs, d'actualiser périodiquement le contenu ou de découper des zones spécifiques d'une page web. Il peut également être connecté à une **external data source** pour déterminer le contenu affiché en temps réel.

***

## **Aperçu des propriétés**

### **Paramètres généraux**

* **Options alternatives** – Détermine comment l'élément effectue la transition entre les éléments de la liste de lecture.
* **Background Color** – Définit la couleur d'arrière-plan pour l'élément web.
* **Border** – Définit le style et la couleur de la bordure.
* **Identification** – Attribue un identifiant unique à l'élément dans la page.
* **Position** – Détermine le placement de l'élément dans la présentation.
* **Size** – Définit la largeur et la hauteur de l'élément.
* **Visibility** – Contrôle si l'élément est affiché pendant la lecture.

### **Web-Specific Settings**

* **Behavior** – Spécifie si la page web est **interactive** ou **static**.
* **Play List** – Définit les pages web à afficher.
* **Play List Options** – Détermine comment les pages web listées sont affichées.
* **Refresh Content** – Définit la fréquence de rechargement du contenu web.
* **Web Page Restrictions** – Restreint l'accès utilisateur à certaines pages web en **interactive mode**.

### **Authentication & Network Settings**

* **Proxy Settings** – Définit les identifiants d'authentification du proxy pour accéder au contenu web.
* **Authorization** – Permet à l'utilisateur de spécifier un **username** et **password** pour l'accès au site web.

### **Additional Features**

* **Data Provider** – Connecte l'élément à une source de données externe pour des mises à jour de contenu en temps réel.
* **Data Source Text Conversion** – Définit comment le texte multi-lignes provenant de la source de données est traité.
* **Proof of Play** – Active l'enregistrement de quand et où l'élément a été affiché.
* **Property Bindings** – Permet à des propriétés spécifiques d'être contrôlées dynamiquement par une source de données externe.
* **Reflection** – Affiche la page web avec un effet de réflexion.
* **Rules** – Définit les conditions qui ajustent automatiquement les propriétés de l'élément.

***

### **Remarks**

* Le **web page element** affiche le contenu en utilisant **Internet Explorer**, ce qui signifie que certains sites web modernes peuvent ne pas s'afficher correctement.
* Si **interactivity** est activée, les utilisateurs peuvent interagir avec la page web, mais l'accès peut être restreint en utilisant **Web Page Restrictions**.
* **Clipping options** permettent d'afficher seulement une partie de la page web si nécessaire.
* Si l'élément est connecté à un **data provider**, le contenu affiché est mis à jour en temps réel.

Cette section décrit uniquement les **web-specific properties**. Pour les propriétés générales qui s'appliquent à tous les éléments, référez-vous à la **Page Element Properties section**.
