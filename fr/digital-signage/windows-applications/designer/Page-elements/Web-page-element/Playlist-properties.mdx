Ces propriétés définissent le **contenu web** affiché par l'**Web Page Element**. Vous pouvez sélectionner une **page web unique** ou créer une **playlist** de plusieurs pages à afficher séquentiellement.

***

## **Gestion de la playlist**

* Ajouter, modifier, réorganiser ou supprimer des éléments de la playlist.
* Utiliser les contrôles de navigation pour parcourir les éléments de la playlist.
* La **durée totale** de tous les éléments est affichée.

***

## **Éléments de la playlist de pages web**

Le champ **Type** spécifie la source du contenu web.

| **Option**                    | **Description**                                                                                                                                                                                                                                                              |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **HTML**                      | Affiche du **contenu HTML intégré** au lieu d'un site web complet.                                                                                                                                                                                                           |
| **HTML from Content Library** | Charge un fichier HTML **stocké dans la PADS4 Content Library** (recommandé). Les fichiers dans la bibliothèque **se synchronisent sur tous les systèmes**.                                                                                                                  |
| **Page Web**                  | Affiche un **site web depuis Internet ou Intranet**. Nécessite **un accès internet**. Peut également référencer **des fichiers HTML stockés en dehors de la bibliothèque de contenu**. Ces fichiers doivent être **distribués manuellement** à tous les systèmes de lecture. |
| **Séparateur**                | Crée une **pause** dans la playlist sans afficher de contenu.                                                                                                                                                                                                                |

***

## **Configuration du contenu**

* Saisissez **le code HTML, une URL de site web ou un chemin de fichier**, en fonction du **Type**.
* Utilisez le bouton **parcourir** pour sélectionner un fichier depuis la **Content Library** ou votre système local.
* **Les fichiers HTML externes** doivent inclure le **chemin complet du fichier**.

**Tip:**\
L'élément Page Web prend en charge **tout contenu** qu'un navigateur web peut afficher, y compris **des applications tierces, des éléments interactifs et des médias en streaming**.

***

## **Durée**

* Définit **combien de temps (en secondes)** chaque élément reste à l'écran avant de passer au suivant.
* **Les séparateurs** introduisent **des pauses** dans la playlist.

***

## **Recadrage et ajustements d'affichage**

* **Recadrage** : Découpe une portion de la page web pour l'affichage.
* **Décalage (Gauche, Haut)** : Définit la **position de départ en pixels** dans l'élément.
* **Recadrage interactif** : Utilisez l'**outil de recadrage** pour ajuster manuellement la portion visible.
* **Taille du conteneur du navigateur web** :
  * Empêche le **réalignement** indésirable des éléments web.
  * Définissez une **taille fixe** plus grande que la zone visible pour maintenir **la cohérence de la mise en page**.

***

## **Considérations supplémentaires**

* **Fuites de mémoire** : Certaines pages web contiennent des éléments qui **provoquent des fuites de mémoire**, pouvant entraîner **des problèmes de performance** pendant la lecture.
* **Mises à jour en direct** :
  * Les pages web et les fichiers HTML sont **référencés**, non intégrés.
  * Si le **fichier source ou le site web** est mis à jour, les changements apparaissent **automatiquement** pendant la lecture.
* **Connexion à la source de données** : Si l'élément Page Web est **lié à une source de données**, les paramètres de la playlist sont **désactivés**, et le contenu affiché est déterminé **en temps réel**.
