Ces propriétés définissent le **contenu web** affiché par l'**élément Page Web**. Vous pouvez sélectionner une **page web unique** ou créer une **liste de lecture** de plusieurs pages à afficher séquentiellement.

***

## **Gestion de la liste de lecture**

* Ajoutez, modifiez, réorganisez ou supprimez des éléments de la liste de lecture.
* Utilisez les contrôles de navigation pour parcourir les éléments de la liste de lecture.
* La **durée totale** de tous les éléments est affichée.

***

## **Éléments de la liste de lecture de pages web**

Le **Type** spécifie la source du contenu web.

| **Option**                                 | **Description**                                                                                                                                                                                                                                                                     |
| ------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **HTML**                                   | Affiche du **contenu HTML intégré** au lieu d'un site web complet.                                                                                                                                                                                                                  |
| **HTML depuis la bibliothèque de contenu** | Charge un fichier HTML **stocké dans la bibliothèque de contenu PADS4** (recommandé). Les fichiers de la bibliothèque **se synchronisent sur tous les systèmes**.                                                                                                                   |
| **Page Web**                               | Affiche un **site web depuis Internet ou un Intranet**. Nécessite un **accès à Internet**. Peut également référencer des **fichiers HTML stockés en dehors de la bibliothèque de contenu**. Ces fichiers doivent être **distribués manuellement** sur tous les systèmes de lecture. |
| **Séparateur**                             | Crée une **pause** dans la liste de lecture sans affichage de contenu.                                                                                                                                                                                                              |

***

## **Configuration du contenu**

* Saisissez **du code HTML, une URL de site web ou un chemin de fichier**, en fonction du **Type** sélectionné.
* Utilisez le bouton **parcourir** pour sélectionner un fichier depuis la **bibliothèque de contenu** ou votre système local.
* **Les fichiers HTML externes** doivent inclure le **chemin de fichier complet**.

**Tip:**\
L'élément Page Web prend en charge **tout contenu** qu'un navigateur web peut afficher, y compris **des applications tierces, des éléments interactifs et des médias en streaming**.

***

## **Durée**

* Définit **la durée (en secondes)** pendant laquelle chaque élément reste à l'écran avant de passer au suivant.
* **Les séparateurs** introduisent des **pauses** dans la liste de lecture.

***

## **Recadrage et ajustements d'affichage**

* **Recadrage**: Découpe une partie de la page web pour l'affichage.
* **Décalage (Gauche, Haut)**: Définit la **position de départ en pixels** dans l'élément.
* **Recadrage interactif**: Utilisez l'**outil de recadrage** pour ajuster manuellement la portion visible.
* **Taille du conteneur du navigateur web**:
  * Empêche le **réalignement** indésirable des éléments web.
  * Définissez une **taille fixe** plus grande que la zone visible pour maintenir la **cohérence de la mise en page**.

***

## **Considérations supplémentaires**

* **Fuites mémoire**: Certaines pages web contiennent des éléments qui **provoquent des fuites mémoire**, pouvant entraîner des **problèmes de performances** pendant la lecture.
* **Mises à jour en direct**:
  * Les pages web et les fichiers HTML sont **référencé**, pas intégré.
  * Si le **fichier source ou le site web** est mis à jour, les modifications apparaissent **automatiquement** pendant la lecture.
* **Connexion à la source de données** : Si l'élément Page Web est **lié à une source de données**, les paramètres de playlist sont **désactivés**, et le contenu affiché est déterminé **en temps réel**.
