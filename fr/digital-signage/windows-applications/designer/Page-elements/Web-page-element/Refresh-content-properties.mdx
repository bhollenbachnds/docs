Ces propriétés contrôlent la fréquence à laquelle le contenu d'un **élément de page web** est actualisé pendant la lecture.

***

## **Options d'actualisation du contenu**

### **Ne pas actualiser la page web**

* La page web **se charge une seule fois** et reste statique.
* Aucune mise à jour automatique ne se produit, sauf si la présentation se recharge.

### **Actualiser le contenu toutes les X secondes**

* La page web est **rechargée à intervalles réguliers** (en secondes).
* Aide à maintenir le contenu dynamique à jour.

***

## **Considérations importantes**

* Si **Screenshot Mode** est activé, **le contenu ne sera pas actualisé** du tout.
* L'actualisation d'une page web réinitialise tous les **processus en cours** (par ex., vidéos, animations).
* **Une actualisation fréquente** peut entraîner des **problèmes de performances**, surtout si le site présente des fuites mémoire.

***

## **Avertissement important**

* Certains sites web contiennent des **éléments qui provoquent des fuites mémoire**.
* Un rechargement fréquent peut **aggraver le problème**, entraînant une **instabilité** pendant la lecture.
* **Net Display Systems n'est pas responsable** du contenu web tiers affectant les performances.

***

Définir un **intervalle d'actualisation approprié** garantit que la page web se met à jour **sans perte de performances inutile**.
