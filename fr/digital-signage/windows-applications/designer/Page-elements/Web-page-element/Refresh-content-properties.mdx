Ces propriétés contrôlent la fréquence à laquelle le contenu d'un **élément de page web** est actualisé pendant la lecture.

***

## **Options d'actualisation du contenu**

### **Do Not Refresh the Web Page**

* La page web **se charge une seule fois** et reste statique.
* Aucune mise à jour automatique ne se produit à moins que la présentation ne soit rechargée.

### **Refresh Content Every X Seconds**

* La page web est **rechargée à intervalles réguliers** (en secondes).
* Aide à maintenir le contenu dynamique à jour.

***

## **Considérations importantes**

* Si **Screenshot Mode** est activé, **le contenu ne s'actualisera pas** du tout.
* L'actualisation d'une page web réinitialise tous les **processus en cours** (par ex., vidéos, animations).
* **Des actualisations fréquentes** peuvent causer des **problèmes de performance**, surtout si le site présente des fuites de mémoire.

***

## **Avertissement important**

* Certains sites web contiennent des **éléments qui provoquent des fuites de mémoire**.
* Des rechargements fréquents peuvent **aggraver le problème**, entraînant une **instabilité** pendant la lecture.
* **Net Display Systems n'est pas responsable** des contenus web tiers affectant les performances.

***

Définir un **intervalle d'actualisation approprié** garantit que la page web se met à jour **sans perte de performance inutile**.
