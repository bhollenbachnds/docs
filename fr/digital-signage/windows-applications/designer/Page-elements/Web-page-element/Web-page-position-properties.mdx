Ces propriétés définissent la **taille de conteneur par défaut** pour un élément de page web dans la présentation, assurant une cohérence dans l'affichage du contenu web.

***

## **Conteneur de navigateur Web**

### **Objectif**

Lors du recadrage d'une page web (affichage d'une partie seulement), des **problèmes de mise à l'échelle** peuvent survenir si la page web ajuste dynamiquement son contenu en fonction de la taille du conteneur. Pour **éviter un redimensionnement inattendu**, une **taille de conteneur de navigateur web** fixe peut être définie.

Cela garantit que la page web se charge avec une **taille cohérente**, et que seule la **section recadrée** définie est affichée.

***

## **Paramètres de taille du conteneur**

### **Activer la taille fixe**

* Active le **mode fixe** pour le conteneur du navigateur web, en utilisant les dimensions spécifiées.

### **Largeur**

* Définit la **largeur** du conteneur en **pixels**.

### **Hauteur**

* Définit la **hauteur** du conteneur en **pixels**.

***

## **Considérations importantes**

* Si la **taille fixe** n'est **pas activée**, la page web peut se redimensionner dynamiquement, ce qui pourrait **modifier le positionnement** du contenu recadré.
* Définir une **largeur et une hauteur fixes** garantit que la page se charge à la taille spécifiée, maintenant une **mise en page cohérente** même lorsqu'elle est recadrée.
* Si la **taille du conteneur est plus petite que la zone recadrée**, **le contenu peut être coupé**. Assurez-vous que la taille fixe est suffisamment grande pour accueillir la section recadrée.
