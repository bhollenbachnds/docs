Ces propriétés définissent la **taille de conteneur par défaut** pour un élément de page web dans la présentation, assurant la cohérence dans l'affichage du contenu web.

***

## **Conteneur de navigateur Web**

### **Objectif**

Lors du recadrage d'une page web (affichage d'une partie seulement), des **problèmes de mise à l'échelle** peuvent survenir si la page web ajuste dynamiquement son contenu en fonction de la taille du conteneur. Pour **éviter un redimensionnement inattendu**, une **taille de conteneur de navigateur web** fixe peut être définie.

Cela garantit que la page web se charge avec une **taille cohérente**, et seule la **section recadrée** définie est affichée.

***

## **Paramètres de taille du conteneur**

### **Activer la taille fixe**

* Active le **mode fixe** pour le conteneur du navigateur web, en utilisant les dimensions spécifiées.

### **Width**

* Définit la **largeur** du conteneur en **pixels**.

### **Height**

* Définit le **height** du conteneur en **pixels**.

***

## **Considérations clés**

* Si la **fixed size** est **not enabled**, la page web peut se redimensionner dynamiquement, ce qui pourrait **alter the positioning** du contenu recadré.
* Définir une **fixed width and height** garantit que la page se charge à la taille spécifiée, maintenant une **consistent layout** même lorsqu'elle est recadrée.
* Si la **container size is smaller than the cropped area**, **content may be cut off**. Assurez-vous que la taille fixe est suffisamment grande pour accueillir la section recadrée.
