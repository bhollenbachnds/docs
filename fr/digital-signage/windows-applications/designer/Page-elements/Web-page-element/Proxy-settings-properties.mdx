Ces propriétés définissent les **proxy server settings** pour accéder aux pages web. Par défaut, **no proxy server** n'est utilisé, et le système tentera une **direct connection** à Internet.

***

## **Configuration du serveur proxy**

### **Enable Proxy Server**

* **Activer/désactiver** l'utilisation d'un **proxy server** pour récupérer du contenu web.

### **Server Address**

* Entrez le **computer name, DNS name, or IP address** du **proxy server**.

### **Port Number**

* Définit le **port number** utilisé pour la communication avec le proxy.
* Default: **80** (courant pour le trafic HTTP).

### **Identifiants d'authentification (Facultatif)**

* **User ID** : Nom d'utilisateur requis pour s'authentifier auprès du serveur proxy.
* **Password** : Mot de passe associé au nom d'utilisateur fourni.

***

## **Notes supplémentaires**

* Si aucun **proxy server** n'est configuré, le système tentera une **direct internet connection**.
* Les détails d'authentification sont **optional** et ne sont requis que si le serveur proxy exige des **login credentials**.
