Ces propriétés définissent les **paramètres du serveur proxy** pour l'accès aux pages web. Par défaut, **aucun serveur proxy** n'est utilisé, et le système tentera une **connexion directe** à Internet.

***

## **Configuration du serveur proxy**

### **Activer le serveur proxy**

* **Activer/désactiver** l'utilisation d'un **serveur proxy** pour récupérer le contenu web.

### **Adresse du serveur**

* Entrez le **nom de l'ordinateur, le nom DNS ou l'adresse IP** du **serveur proxy**.

### **Numéro de port**

* Définit le **numéro de port** utilisé pour la communication avec le proxy.
* Default: **80** (courant pour le trafic HTTP).

### **Identifiants d'authentification (facultatif)**

* **Identifiant utilisateur** : Nom d'utilisateur requis pour s'authentifier auprès du serveur proxy.
* **Mot de passe** : Mot de passe associé au nom d'utilisateur fourni.

***

## **Notes supplémentaires**

* Si aucun **serveur proxy** n'est configuré, le système tentera une **connexion internet directe**.
* Les informations d'authentification sont **facultatives** et ne sont requises que si le serveur proxy impose des **identifiants de connexion**.
