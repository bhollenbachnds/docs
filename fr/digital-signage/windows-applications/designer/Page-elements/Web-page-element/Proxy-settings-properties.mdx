Ces propriétés définissent les **paramètres du serveur proxy** pour accéder aux pages web. Par défaut, **aucun serveur proxy** n'est utilisé, et le système tentera une **connexion directe** à Internet.

***

## **Configuration du serveur proxy**

### **Activer le serveur proxy**

* **Activer/désactiver** l'utilisation d'un **serveur proxy** pour récupérer du contenu web.

### **Adresse du serveur**

* Entrez le **nom d'ordinateur, nom DNS ou adresse IP** du **serveur proxy**.

### **Numéro de port**

* Définit le **numéro de port** utilisé pour la communication avec le proxy.
* Default: **80** (courant pour le trafic HTTP).

### **Identifiants d'authentification (Optionnel)**

* **User ID** : Nom d'utilisateur requis pour s'authentifier auprès du serveur proxy.
* **Password** : Mot de passe associé au nom d'utilisateur fourni.

***

## **Notes supplémentaires**

* Si aucun **serveur proxy** n'est configuré, le système tentera une **connexion internet directe**.
* Les détails d'authentification sont **optionnels** et uniquement requis si le serveur proxy impose des **identifiants de connexion**.
