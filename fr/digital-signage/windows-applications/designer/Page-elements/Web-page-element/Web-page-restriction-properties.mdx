Ces propriétés vous permettent de **limiter l'accès** à des pages web spécifiques lors de l'utilisation d'un **élément de page web interactive**. Cela empêche les utilisateurs de naviguer librement sur **Internet** et les limite à **du contenu approuvé**.

***

## **Paramètres de Restrictions des Pages Web**

### **Empêcher les Affichages de Pages Web Indésirables**

* **Activer ou désactiver** les restrictions de pages web.
* Lorsqu'elles sont activées, seuls les **sites web approuvés** seront accessibles.

### **Pages Web Autorisées**

* Définir une **liste de sites web autorisés** pour les utilisateurs.

***

## **Gestion des Sites Web Autorisés**

### **Pour Ajouter un Site Web :**

1. Cliquez sur **Add**.
2. Entrez l'**URL** du site web à autoriser.
3. Cliquez sur **OK**.

### **Pour Modifier un Site Web :**

1. Sélectionnez le **site web existant** dans la liste.
2. Cliquez sur **Edit**.
3. Effectuez les modifications nécessaires et **cliquez sur OK**.

### **Pour Supprimer un Site Web :**

1. Sélectionnez le **site web** dans la liste.
2. Cliquez sur **Remove** pour supprimer l'entrée.

***

## **Meilleures Pratiques et Conseils**

* **Utiliser des URL Génériques** :
  * Example: Adding **"nds.eu"** permet toutes les pages sous ce domaine (par exemple, **nds.eu/products, nds.eu/about**).
  * Ajouter **"nds.eu/products"** permet uniquement l'accès à la page **products**.

* **Supprimer "[www](http://www)." dans les URL** :
  * Certains sites web **redirigent automatiquement** (par exemple, de **[www.example.com](http://www.example.com)** vers **example.com**).
  * Entrer simplement **"example.com"** garantit que l'accès n'est pas bloqué en raison de **non-correspondances de redirection**.

***

En définissant des **restrictions de pages web**, vous pouvez contrôler **quels sites sont accessibles**, assurant une **navigation sûre et ciblée** lors de votre **présentation interactive**.
