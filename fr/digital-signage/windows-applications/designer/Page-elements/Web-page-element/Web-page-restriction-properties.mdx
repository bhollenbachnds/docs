Ces propriétés vous permettent de **limiter l'accès** à des pages web spécifiques lors de l'utilisation d'un **élément de page web interactif**. Cela empêche les utilisateurs de naviguer librement sur **Internet** et les limite à **du contenu approuvé**.

***

## **Paramètres de restrictions des pages web**

### **Empêcher les affichages de pages web indésirables**

* **Activer ou désactiver** les restrictions de pages web.
* Lorsque cette option est activée, seuls **les sites web approuvés** seront accessibles.

### **Pages web autorisées**

* Définir une **liste de sites web autorisés** pour les utilisateurs.

***

## **Gestion des sites web autorisés**

### **Pour ajouter un site web :**

1. Cliquez sur **Ajouter**.
2. Saisissez l'**URL** du site web à autoriser.
3. Cliquez sur **OK**.

### **Pour modifier un site web :**

1. Sélectionnez le **site web existant** dans la liste.
2. Cliquez sur **Modifier**.
3. Effectuez les modifications nécessaires et **cliquez sur OK**.

### **Pour supprimer un site web :**

1. Sélectionnez le **site web** dans la liste.
2. Cliquez sur **Supprimer** pour effacer l'entrée.

***

## **Bonnes pratiques et conseils**

* **Utiliser des URL génériques** :
  * Example: Adding **"nds.eu"** permet d'accéder à toutes les pages sous ce domaine (par exemple, **nds.eu/products, nds.eu/about**).
  * L'ajout de **"nds.eu/products"** permet uniquement l'accès à la page **products**.

* **Supprimer "[www](http://www)." dans les URL** :
  * Certains sites web **redirigent automatiquement** (par exemple, de **[www.example.com](http://www.example.com)** vers **example.com**).
  * Saisir simplement **"example.com"** garantit que l'accès n'est pas bloqué en raison de **non-correspondances de redirection**.

***

En définissant des **restrictions de pages web**, vous pouvez contrôler **quels sites sont accessibles**, assurant une expérience de navigation **sûre et ciblée** au sein de votre **présentation interactive**.
