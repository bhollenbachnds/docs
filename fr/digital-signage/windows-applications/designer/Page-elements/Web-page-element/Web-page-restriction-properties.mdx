Ces propriétés vous permettent de **limiter l'accès** à des pages web spécifiques lors de l'utilisation d'un **élément de page web interactive**. Cela empêche les utilisateurs de naviguer librement sur **Internet** et les restreint au **contenu approuvé**.

***

## **Paramètres des restrictions de pages web**

### **Empêcher les affichages de pages web non souhaités**

* **Activer ou désactiver** les restrictions de pages web.
* Lorsqu'elles sont activées, seuls les **sites web approuvés** seront accessibles.

### **Pages web autorisées**

* Définissez une **liste de sites web autorisés** pour les utilisateurs.

***

## **Gestion des sites web autorisés**

### **Pour ajouter un site web :**

1. Cliquez sur **Ajouter**.
2. Saisissez l'**URL** du site web à autoriser.
3. Cliquez sur **OK**.

### **Pour modifier un site web :**

1. Sélectionnez le **site web existant** dans la liste.
2. Cliquez sur **Modifier**.
3. Effectuez les modifications nécessaires et **cliquez sur OK**.

### **Pour supprimer un site web :**

1. Sélectionnez le **site web** dans la liste.
2. Cliquez sur **Supprimer** pour supprimer l'entrée.

***

## **Bonnes pratiques et conseils**

* **Utiliser des URL génériques**:
  * Example: Adding **"nds.eu"** autorise toutes les pages sous ce domaine (par ex., **nds.eu/products, nds.eu/about**).
  * Ajouter **"nds.eu/products"** n'autorise l'accès qu'à la page **products** uniquement.

* **Supprimer "[www](http://www)." dans les URL**:
  * Certains sites web **redirigent automatiquement** (par ex., de **[www.example.com](http://www.example.com)** vers **example.com**).
  * Saisir uniquement **"example.com"** garantit que l'accès n'est pas bloqué en raison de **redirections non concordantes**.

***

En définissant des **restrictions de pages web**, vous pouvez contrôler **quels sites sont accessibles**, garantissant une **expérience de navigation sûre et ciblée** au sein de votre **présentation interactive**.
