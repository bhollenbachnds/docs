Ces propriétés définissent le **comportement interactif** de l'**Web Page Element** et la façon dont il s'affiche et interagit avec les utilisateurs.

***

## **Behavior Settings**

### **Scrollbars**

* **Show Scrollbars**: Active ou désactive les barres de défilement lorsque le contenu dépasse les limites de l'élément.

### **Mouse Interaction**

* **Show Mouse Pointer**: Lorsqu'il est activé, le **pointeur de la souris** est visible, permettant aux utilisateurs de faire défiler et de cliquer sur des hyperliens.

### **Screenshot Mode**

* Lorsqu'il est activé, l'élément **capture une capture d'écran** de la page web après le chargement et affiche l'**image statique** à la place de la page en direct.
* **Advantages**: Améliore les performances, empêche les interactions non souhaitées et permet une mise à l'échelle correcte lorsque les présentations sont redimensionnées automatiquement.
* **Disadvantages**: **Le contenu dynamique (par ex., animations, vidéos, éléments interactifs) ne fonctionnera pas.**

### **User Interactivity**

* **Allow User Interactivity**: Permet aux utilisateurs de cliquer sur des hyperliens dans l'élément de page web.
  * **Note**: Désactivé si le **Screenshot Mode** est activé.
  * **Limitation**: Les hyperliens qui ouvrent **de nouvelles fenêtres/popups** ne sont **pas pris en charge**.

### **Address Bar**

* **Show Address Bar**: Affiche une **barre d'adresse** permettant aux utilisateurs de saisir des URLs.
  * **Disponible uniquement en mode interactif.**
  * **Security Tip**: Combinez ceci avec **Web Page Restrictions** pour empêcher la navigation non autorisée.

### **Scaling Options**

* **Ignore Scaling**: Garantit que l'élément **ignore** les paramètres de **mise à l'échelle de l'affichage** Windows et s'affiche à **100% de zoom**.

***

## **Web Browser Engine Options**

Sélectionnez le **moteur de navigateur** utilisé pour afficher la page web. Différentes pages web **peuvent se comporter différemment** selon le moteur :

| **Engine**            | **Description**                                                      | **Current Version** |
| --------------------- | -------------------------------------------------------------------- | ------------------- |
| **Chromium**          | Basé sur le projet open-source **Chrome**.                           | **Chrome 100**      |
| **Edge**              | Utilise **Microsoft Edge WebView**.                                  | **Edge 108**        |
| **Internet Explorer** | Utilise **IE11**, qui peut présenter des problèmes de compatibilité. | **IE11**            |

***

## **Cache & Performance Settings**

### **Cache Control**

* **Clear Cache on Start**: Efface automatiquement **les données en cache** au démarrage de l'élément web.

### **Performance Optimization**

* **Frame Rate**: Ajuste le **taux de rafraîchissement** (par défaut : **30 FPS**).
* **Render Mode**:
  * **Always on Top**: Garantit que l'**élément web est affiché au-dessus des autres éléments** dans la présentation.

***

## **Remarks**

* **User Interactivity** permet uniquement la **navigation dans la même fenêtre**. **Popups** et **liens vers de nouvelles fenêtres** ne sont pas pris en charge.
* Si **le contenu dynamique provoque des fuites mémoire**, l'activation du **Screenshot Mode** peut aider **améliorer la stabilité**.
* **Conseil de sécurité** : Si vous autorisez l'interaction utilisateur, activez **Web Page Restrictions** pour **contrôler les sites accessibles**.
