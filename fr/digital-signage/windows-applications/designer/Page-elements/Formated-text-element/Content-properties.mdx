Ces propriétés définissent la **valeur calculée** qui sera affichée par l'**élément de texte formaté**. Vous pouvez choisir parmi une variété de valeurs dynamiques telles que la date, l'heure, les numéros de page, les compteurs de données ou les valeurs numériques.

***

### **Types de contenu disponibles**

| **Option**                                           | **Description**                                                                                   |
| ---------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| **Date/Heure**                                       | Affiche la date et/ou l'heure actuelle.                                                           |
| **Compte à rebours**                                 | Affiche un compte à rebours jusqu'à un moment spécifié. Changement automatique de page en option. |
| **Chronomètre**                                      | Compte à partir de 0 ou à partir d'une valeur de départ spécifiée.                                |
| **Numérique**                                        | Affiche une valeur numérique avec un formatage personnalisé.                                      |
| **Numéro de Page**                                   | Affiche le numéro de page actuel de votre présentation.                                           |
| **Nombre Total de Pages**                            | Affiche le nombre total de pages dans votre présentation.                                         |
| **Numéro de Page sur Nombre Total de Pages**         | Affiche à la fois le numéro de page actuel et le nombre total de pages.                           |
| **Nombre Total de Lignes du Fournisseur de Données** | Affiche le nombre total de lignes renvoyées par un fournisseur de données connecté.               |

***

### **Options de Formatage par Type de Contenu**

#### **1. Formatage Date/Heure**

* **Préfixe**: Texte affiché avant la valeur de date/heure.
* **Format de Date**:
  * None: No date shown.
  * Date Longue : Utilise le format de date longue de Windows.
  * Date Courte : Utilise le format de date courte de Windows.
  * Custom: Use custom formats (see [Standard Date and Time Format Strings](https://learn.microsoft.com/en-us/dotnet/standard/base-types/standard-date-and-time-format-strings)).
* **Insertion**: Texte entre la date et l'heure (par défaut, un espace).
* **Format d'Heure**:
  * None: No time shown.
  * Heure Longue : Format d'heure longue de Windows.
  * Heure Courte : Format d'heure courte de Windows.
  * Custom: Custom time format (see link above).
* **Suffixe**: Texte affiché après la valeur de date/heure.
* **Culture**: Définir la culture (par ex., en-US, de-DE) pour ajuster le formatage.
* **Fuseau horaire**: Ajuster l'heure à afficher pour un fuseau horaire différent.

***

#### **2. Formatage du Numéro de Page et du Nombre Total de Pages**

* **Préfixe**: Texte avant la valeur.
* **Format**: Spécifier la longueur/le format des caractères pour l'affichage.
* **Suffixe**: Texte après la valeur.

***

#### **3. Formatage du Numéro de Page sur Nombre Total de Pages**

* **Préfixe**: Texte avant le numéro de page.
* **Numéro de Page**: Longueur/format des caractères.
* **Entre**: Texte entre le numéro de page et le nombre total de pages.
* **Nombre Total de Pages**: Longueur/format des caractères.
* **Suffixe**: Texte après la valeur du nombre total de pages.

***

#### **4. Formatage du Nombre Total de Lignes du Fournisseur de Données**

* Identique au **Formatage du Numéro de Page**.
* **Remarque**: Nécessite une connexion à une source de données et au moins un autre élément utilisant cette source de données avec le défilement activé.

***

#### **5. Formatage du Compte à Rebours**

* **Préfixe**: Texte avant la valeur du compte à rebours.
* **Format d'Heure**:
  * Heure Longue : Format d'heure longue de Windows.
  * Heure Courte : Format d'heure courte de Windows.
  * Custom: Custom format (see [Standard Date and Time Format Strings](https://learn.microsoft.com/en-us/dotnet/standard/base-types/standard-date-and-time-format-strings)).
* **Suffixe**: Texte après la valeur du compte à rebours.
* **Durée**: Heure de départ (par ex., `1:00:00` pour 1 heure).
* **Afficher la page suivante lorsque le temps atteint zéro**: Naviguer automatiquement vers la page suivante lorsque le compte à rebours se termine.

***

#### **6. Formatage du Chronomètre**

* **Valeur de Départ**: Définir la valeur initiale ou commencer à partir de zéro.
* **Format**: Utiliser le formatage de l'heure pour l'affichage.

***

#### **7. Formatage Numérique**

* Formats numériques personnalisés pris en charge (voir [Standard Numeric Format Strings](https://learn.microsoft.com/en-us/dotnet/standard/base-types/standard-numeric-format-strings)).

***

### **Remarques Importantes**

* Les valeurs calculées sont mises à jour **en temps réel pendant la lecture**.
* Si connecté à une source de données, les formats comme DateTime ou Timespan suivent le **type de données de la colonne**.
* Le système va automatiquement **synchroniser l'heure** avec le PADS4 Server, sauf indication contraire.
* Les comptes à rebours peuvent déclencher **des changements automatiques de page** lorsque le compte à rebours atteint zéro.
