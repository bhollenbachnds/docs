Les éléments de texte formaté sont utilisés pour **afficher des valeurs dynamiques calculées**, telles que :

* **Date et heure actuelles**
* **Numéro de page actuel**
* **Nombre total de pages**
* **Nombre total de lignes**
* Autres **données calculées personnalisées**

En plus d'afficher des valeurs calculées, les éléments de texte formaté offrent de nombreuses options de personnalisation identiques à celles des éléments de texte standard, notamment le style de police, l'alignement et les effets.

***

### **Aperçu des propriétés**

| **Section de propriété**           | **Description**                                                                                            |
| ---------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| **Couleur d'arrière-plan**         | Définissez la couleur d'arrière-plan de votre élément.                                                     |
| **Clignotement**                   | Définissez un effet de clignotement pour attirer l'attention sur le texte.                                 |
| **Bordure**                        | Définissez la couleur, la largeur et le style de bordure de votre élément.                                 |
| **Contenu**                        | Définissez quelle **valeur calculée** (par ex., date/heure, numéro de page) sera affichée.                 |
| **Fournisseur de données**         | Connectez-vous à une **source de données** externe pour du contenu en temps réel.                          |
| **Défilement des données**         | Activez et configurez le défilement des données en temps réel.                                             |
| **Effet de transition par défaut** | Définissez l'**effet de transition** par défaut pour le contenu entrant.                                   |
| **Retournement**                   | Retournez le contenu horizontalement ou verticalement.                                                     |
| **Police**                         | Définissez le **type, la taille et la couleur de la police**, ainsi que le style du texte.                 |
| **Identification**                 | Attribuez un **identifiant unique** à l'élément dans la page.                                              |
| **Position**                       | Spécifiez la **position** de l'élément sur la page.                                                        |
| **Preuve de diffusion**            | Activez la journalisation de la preuve de diffusion pour les rapports et la validation.                    |
| **Liaisons de propriétés**         | Liez des propriétés à une **source de données externe** pour les mises à jour dynamiques.                  |
| **Reflection**                     | Ajouter un **effet de reflet** au contenu affiché.                                                         |
| **Rotation**                       | Faire pivoter l'élément selon un angle spécifique.                                                         |
| **Rules**                          | Définir des **rules** pour automatiser le comportement de l'élément en fonction de conditions spécifiques. |
| **Shadow**                         | Ajouter un **effet d'ombre** au contenu pour un effet visuel accentué.                                     |
| **Size**                           | Définir la **largeur et la hauteur** de l'élément.                                                         |
| **Skew**                           | Appliquer une transformation d'inclinaison pour incliner le contenu.                                       |
| **Alignement du texte**            | Ajuster l'**alignement horizontal et vertical** du texte.                                                  |
| **Transparency**                   | Définir le niveau de transparence de l'élément.                                                            |
| **Visibility**                     | Contrôler la **visibility** de l'élément et de son contenu pendant la lecture.                             |

***

### **Note importante**

> Cette section se concentre sur les **propriétés spécifiques au texte formaté**. Pour des explications détaillées sur les propriétés communes à tous les éléments de page (par ex., bordures, taille, position), voir **Page Element Properties**.
