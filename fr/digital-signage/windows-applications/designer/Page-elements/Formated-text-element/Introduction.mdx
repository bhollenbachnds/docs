Les éléments de texte formaté sont utilisés pour **afficher des valeurs dynamiques et calculées**, telles que :

* **Date et heure actuelles**
* **Numéro de page actuel**
* **Nombre total de pages**
* **Nombre total de lignes**
* Autres **données calculées personnalisées**

En plus d'afficher des valeurs calculées, les éléments de texte formaté offrent de nombreuses options de personnalisation similaires aux éléments de texte standard, notamment le style de police, l'alignement et les effets.

***

### **Aperçu des propriétés**

| **Section de propriété**           | **Description**                                                                                             |
| ---------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| **Couleur d'arrière-plan**         | Définir la couleur d'arrière-plan de votre élément.                                                         |
| **Clignotement**                   | Définir un effet de clignotement pour attirer l'attention sur le texte.                                     |
| **Bordure**                        | Définir la couleur, la largeur et le style de bordure pour votre élément.                                   |
| **Contenu**                        | Définir quelle **valeur calculée** (par exemple, date/heure, numéro de page) sera affichée.                 |
| **Fournisseur de données**         | Se connecter à une **source de données** externe pour du contenu en temps réel.                             |
| **Défilement des données**         | Activer et configurer le défilement à travers les données en temps réel.                                    |
| **Effet de transition par défaut** | Définir l'**effet de transition** par défaut pour le contenu entrant.                                       |
| **Retournement**                   | Retourner le contenu horizontalement ou verticalement.                                                      |
| **Police**                         | Définir le **type, la taille, la couleur de police** et le style du texte.                                  |
| **Identification**                 | Attribuer un **identifiant unique** à l'élément dans la page.                                               |
| **Position**                       | Spécifier la **position** de l'élément sur la page.                                                         |
| **Preuve de diffusion**            | Activer l'enregistrement de preuve de diffusion pour les rapports et la validation.                         |
| **Liaisons de propriétés**         | Lier les propriétés à une **source de données externe** pour des mises à jour dynamiques.                   |
| **Réflexion**                      | Ajouter un **effet de réflexion** au contenu affiché.                                                       |
| **Rotation**                       | Faire pivoter l'élément à un angle spécifique.                                                              |
| **Règles**                         | Définir des **règles** pour automatiser le comportement de l'élément en fonction de conditions spécifiques. |
| **Ombre**                          | Ajouter un **effet d'ombre** au contenu pour un accent visuel.                                              |
| **Taille**                         | Définir la **largeur et la hauteur** de l'élément.                                                          |
| **Inclinaison**                    | Appliquer une transformation d'inclinaison pour donner un angle au contenu.                                 |
| **Alignement du texte**            | Ajuster l'**alignement horizontal et vertical** du texte.                                                   |
| **Transparence**                   | Définissez le niveau de transparence de l'élément.                                                          |
| **Visibilité**                     | Contrôlez la **visibilité** de l'élément et de son contenu pendant la lecture.                              |

***

### **Note Importante**

> Cette section se concentre sur les **propriétés spécifiques au texte formaté**. Pour des explications détaillées sur les propriétés communes partagées par tous les éléments de page (par exemple, les bordures, la taille, la position), voir **Page Element Properties**.
