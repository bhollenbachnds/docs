Ces propriétés déterminent l'**alignement horizontal** du texte affiché sur un bouton interactif.

***

### **Alignement**

* **Horizontal**:\
  Choisissez comment le texte est aligné **à l'intérieur du bouton**. Options:
  * **Gauche**: Aligner le texte sur le bord gauche.
  * **Milieu**: Centrer le texte horizontalement.
  * **Droite**: Aligner le texte sur le bord droit.
  * **Justifié**: Distribuer le texte uniformément sur toute la largeur.

***

### **Remarques**

* Si **le dimensionnement automatique** est activé pour le bouton, l'alignement affecte **comment le bouton se redimensionne**:
  * **Comportement par défaut**: Le bouton se redimensionne vers la **droite**.
  * **Alignement à droite**: Le bouton se redimensionne vers la **gauche**.
  * **Alignement au milieu**: Le bouton se redimensionne **de manière égale des deux côtés** (gauche et droite).

Cela permet des ajustements de conception flexibles lors de l'utilisation de contenu dynamique ou variable à l'intérieur du bouton.
