**Les éléments de boutons interactifs** vous permettent d'inclure des **boutons texte** ou des **boutons image** dans vos présentations interactives. Ces boutons peuvent déclencher la navigation entre les pages, faire défiler des données ou activer des actions prédéfinies. Plusieurs effets visuels et options de style sont disponibles pour personnaliser l'apparence de ces boutons.

***

### **Aperçu des propriétés**

| **Section de propriété**   | **Description**                                                                                     |
| -------------------------- | --------------------------------------------------------------------------------------------------- |
| **Style du bouton**        | Définissez le style général du bouton, y compris les états normal, survol et appuyé.                |
| **Contenu (Image)**        | Définissez la mise en page et l'image utilisée pour un bouton image.                                |
| **Contenu (Texte)**        | Définissez la mise en page et les paramètres de texte pour un bouton texte.                         |
| **Retournement**           | Retournez le contenu du bouton horizontalement ou verticalement.                                    |
| **Police**                 | Définissez la police pour afficher le texte sur le bouton.                                          |
| **Identification**         | Identifiez de manière unique l'élément bouton dans la page.                                         |
| **Interactivité**          | Définissez l'action déclenchée lorsque le bouton est appuyé (par ex., aller à une autre page).      |
| **Position**               | Définissez la position du bouton dans la page.                                                      |
| **Liaisons de propriétés** | Liez des propriétés spécifiques du bouton à des sources de données externes.                        |
| **Reflet**                 | Ajoutez des effets de reflet au contenu du bouton.                                                  |
| **Rotation**               | Faites pivoter le bouton selon un angle spécifié.                                                   |
| **Règles**                 | Appliquez des règles qui déclenchent des actions spécifiques en fonction de conditions prédéfinies. |
| **Ombre**                  | Ajoutez des effets d'ombre au contenu du bouton.                                                    |
| **Taille**                 | Définissez la largeur et la hauteur du bouton.                                                      |
| **Inclinaison**            | Inclinez le bouton pour créer des effets angulaires ou obliques.                                    |
| **Alignement du texte**    | Définissez l'alignement horizontal du texte à l'intérieur du bouton.                                |
| **Effet de transition**    | Définissez un effet de transition d'entrée pour l'apparition du bouton.                             |
| **Transparence**           | Définissez le niveau de transparence du bouton.                                                     |
| **Visibilité**             | Contrôlez si et quand le bouton et son contenu sont visibles pendant la lecture.                    |

***

### **Fonctionnalités clés des boutons interactifs**

* **Navigation** : Permettez aux utilisateurs de se déplacer entre différentes pages de la présentation.
* **Interaction avec les données** : Faites défiler des données en temps réel lorsqu'elles sont connectées à des sources de données.
* **Déclenchement d'actions** : Exécutez des actions prédéfinies telles que la lecture de médias, le lancement d'applications ou l'affichage de contenu masqué.
* **Visuels personnalisables** : Appliquez différents styles, effets et alignements pour une conception de bouton personnalisée.
* **Retour interactif** : Utilisez des retours visuels (effets de survol et d'appui) pour améliorer l'expérience utilisateur.

***

### **Notes importantes**

* Cette section se concentre sur **propriétés spécifiques aux boutons interactifs**.
* Pour les propriétés communes (partagées avec d'autres éléments tels que le texte, les images, les vidéos), reportez-vous à **Page Element Properties** section.
* Les boutons peuvent être stylisés pour les états **normal**, **hover** et **pressed**, permettant un retour interactif.
