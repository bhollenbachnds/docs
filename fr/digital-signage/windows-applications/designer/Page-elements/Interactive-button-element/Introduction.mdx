**Les éléments de boutons interactifs** vous permettent d'inclure des **boutons texte** ou des **boutons image** dans vos présentations interactives. Ces boutons peuvent déclencher la navigation entre les pages, faire défiler des données ou activer des actions prédéfinies. Plusieurs effets visuels et options de style sont disponibles pour personnaliser l'apparence de ces boutons.

***

### **Aperçu des propriétés**

| **Section de propriété** | **Description**                                                                                     |
| ------------------------ | --------------------------------------------------------------------------------------------------- |
| **Style du bouton**      | Définissez le style général du bouton, y compris les états normal, survol et appuyé.                |
| **Contenu (Image)**      | Définissez la mise en page et l'image utilisée pour un bouton image.                                |
| **Contenu (Texte)**      | Définir la mise en page et les paramètres de texte pour un bouton texte.                            |
| **Flip**                 | Retourner le contenu du bouton horizontalement ou verticalement.                                    |
| **Font**                 | Définir la police pour l'affichage du texte sur le bouton.                                          |
| **Identification**       | Identifier de manière unique l'élément bouton dans la page.                                         |
| **Interactivité**        | Définir l'action déclenchée lorsque le bouton est pressé (par exemple, aller à une autre page).     |
| **Position**             | Définir la position du bouton dans la page.                                                         |
| **Property Bindings**    | Lier des propriétés spécifiques du bouton à des sources de données externes.                        |
| **Reflection**           | Ajouter des effets de reflet au contenu du bouton.                                                  |
| **Rotation**             | Faire pivoter le bouton selon un angle spécifié.                                                    |
| **Rules**                | Appliquer des règles qui déclenchent des actions spécifiques en fonction de conditions prédéfinies. |
| **Shadow**               | Ajouter des effets d'ombre au contenu du bouton.                                                    |
| **Size**                 | Définir la largeur et la hauteur du bouton.                                                         |
| **Skew**                 | Incliner le bouton pour créer des effets angulaires ou obliques.                                    |
| **Text Alignment**       | Définir l'alignement horizontal du texte à l'intérieur du bouton.                                   |
| **Transition Effect**    | Définir un effet de transition d'entrée pour l'apparition du bouton.                                |
| **Transparency**         | Définir le niveau de transparence du bouton.                                                        |
| **Visibility**           | Contrôler si et quand le bouton et son contenu sont visibles pendant la lecture.                    |

***

### **Fonctionnalités clés des boutons interactifs**

* **Navigation**: Permettre aux utilisateurs de naviguer entre différentes pages de la présentation.
* **Data Interaction**: Faire défiler des données en temps réel lorsqu'elles sont connectées à des sources de données.
* **Action Triggering**: Exécuter des actions prédéfinies telles que la lecture de médias, le lancement d'applications ou l'affichage de contenu masqué.
* **Customizable Visuals**: Appliquer différents styles, effets et alignements pour une conception de bouton personnalisée.
* **Interactive Feedback**: Utiliser des retours visuels (effets de survol et de pression) pour améliorer l'expérience utilisateur.

***

### **Remarques importantes**

* Cette section se concentre sur les **propriétés spécifiques aux boutons interactifs**.
* Pour les propriétés communes (partagées avec d'autres éléments tels que le texte, les images, les vidéos), consultez la section **Page Element Properties**.
* Les boutons peuvent être stylisés pour les états **normal**, **survol**, et **pressé**, permettant un retour interactif.
