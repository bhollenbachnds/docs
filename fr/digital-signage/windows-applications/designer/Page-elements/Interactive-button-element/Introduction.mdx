**Éléments de bouton interactifs** vous permettent d'inclure **du texte** ou **des boutons d'image** dans vos présentations interactives. Ces boutons peuvent déclencher la navigation entre les pages, faire défiler des données ou activer des actions prédéfinies. De multiples effets visuels et options de style sont disponibles pour personnaliser l'apparence de ces boutons.

***

### **Aperçu des propriétés**

| **Section de propriété**   | **Description**                                                                                     |
| -------------------------- | --------------------------------------------------------------------------------------------------- |
| **Style de bouton**        | Définir le style général du bouton, y compris les états normal, survol et enfoncé.                  |
| **Contenu (Image)**        | Définir la mise en page et l'image utilisée pour un bouton d'image.                                 |
| **Contenu (Texte)**        | Définir la mise en page et les paramètres de texte pour un bouton de texte.                         |
| **Retournement**           | Retourner le contenu du bouton horizontalement ou verticalement.                                    |
| **Police**                 | Définir la police pour afficher le texte sur le bouton.                                             |
| **Identification**         | Identifier de manière unique l'élément bouton dans la page.                                         |
| **Interactivité**          | Définir l'action déclenchée lorsque le bouton est pressé (par exemple, aller à une autre page).     |
| **Position**               | Définir la position du bouton dans la page.                                                         |
| **Liaisons de propriétés** | Lier des propriétés spécifiques du bouton à des sources de données externes.                        |
| **Réflexion**              | Ajouter des effets de réflexion au contenu du bouton.                                               |
| **Rotation**               | Faire pivoter le bouton à un angle spécifié.                                                        |
| **Règles**                 | Appliquer des règles qui déclenchent des actions spécifiques basées sur des conditions prédéfinies. |
| **Ombre**                  | Ajouter des effets d'ombre au contenu du bouton.                                                    |
| **Taille**                 | Définir la largeur et la hauteur du bouton.                                                         |
| **Inclinaison**            | Incliner le bouton pour créer des effets angulaires ou obliques.                                    |
| **Alignement du texte**    | Définir l'alignement horizontal du texte à l'intérieur du bouton.                                   |
| **Effet de transition**    | Définir un effet de transition d'entrée pour l'apparition du bouton.                                |
| **Transparence**           | Définir le niveau de transparence du bouton.                                                        |
| **Visibilité**             | Contrôler si et quand le bouton et son contenu sont visibles pendant la lecture.                    |

***

### **Fonctionnalités clés des boutons interactifs**

* **Navigation** : Permet aux utilisateurs de se déplacer entre différentes pages au sein de la présentation.
* **Interaction avec les données** : Faire défiler des données en temps réel lorsqu'elles sont connectées à des sources de données.
* **Déclenchement d'actions** : Exécuter des actions prédéfinies telles que la lecture de médias, le lancement d'applications ou l'affichage de contenu caché.
* **Visuels personnalisables** : Appliquer différents styles, effets et alignements pour une conception de bouton personnalisée.
* **Retour interactif** : Utiliser un retour visuel (effets de survol et de pression) pour améliorer l'expérience utilisateur.

***

### **Remarques importantes**

* Cette section se concentre sur **les propriétés spécifiques aux boutons interactifs**.
* Pour les propriétés communes (partagées avec d'autres éléments comme le texte, les images, les vidéos), consultez la section **Page Element Properties**.
* Les boutons peuvent être stylisés pour les états **normal**, **hover**, et **pressed**, permettant un retour interactif.
