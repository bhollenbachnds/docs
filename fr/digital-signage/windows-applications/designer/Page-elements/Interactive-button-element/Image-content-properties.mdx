Ces propriétés définissent l'**image affichée** sur la surface du bouton d'un élément de bouton interactif.

***

### **Options de source d'image**

Pour spécifier l'**origine** ou le **type d'image**, choisissez l'une des options suivantes :

| **Option**                                        | **Description**                                                                                                                                                                                                                     |
| ------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Bibliothèque de contenu**                       | Référencer un fichier image existant depuis la **Bibliothèque de contenu PADS4** (recommandé). Les éléments de la bibliothèque de contenu sont **synchronisés automatiquement sur tous les PC**, avec des **références relatives**. |
| **Nom de fichier**                                | Référencer un fichier image existant situé **en dehors de la bibliothèque de contenu**. Les fichiers doivent être distribués manuellement et référencés avec des **chemins de fichiers absolus**.                                   |
| **Données externes depuis une source de données** | Connectez l'élément à un **Fournisseur de données** et spécifiez quelle ligne et colonne de données fourniront l'image de manière dynamique.                                                                                        |

***

### **Paramètres du fournisseur de données** (lors de l'utilisation de données externes)

| **Propriété**              | **Description**                                                                                                                                                                                                                                      |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Fournisseur de données** | Connectez-vous à une source telle qu'une table, une requête, une procédure stockée, un fichier XML, un flux RSS, etc. Vous pouvez créer de nouveaux fournisseurs à la volée. Voir *Travailler avec les fournisseurs de données* pour en savoir plus. |
| **Colonne**                | Sélectionnez la colonne contenant la référence de l'image.                                                                                                                                                                                           |
| **Type de contenu**        | Spécifiez comment le contenu doit être interprété (par exemple, chemin de fichier image, URL, etc.).                                                                                                                                                 |
| **Ligne**                  | Spécifiez quelle ligne de la source de données doit être liée à l'élément.                                                                                                                                                                           |

***

### **Options d'affichage de l'image**

| **Propriété**                     | **Description**                                                                                     |
| --------------------------------- | --------------------------------------------------------------------------------------------------- |
| **Étirement**                     | Si activé, l'image sera automatiquement redimensionnée pour **s'adapter aux limites de l'élément**. |
| **Conserver le rapport d'aspect** | Si activé, l'image **conservera son rapport largeur/hauteur d'origine** lors de l'étirement.        |

***

### **Remarques**

* Seules les **références** aux fichiers image sont stockées dans la présentation, **pas les images elles-mêmes**. Si le fichier image est mis à jour, **la nouvelle version sera utilisée automatiquement** lors de la lecture.
* Si la **ligne** spécifiée n'existe pas dans la source de données, **aucune image** ne sera affichée.
* **CONSEIL** : Pour une meilleure qualité visuelle, essayez d'afficher les images à leur **taille d'origine** dans la mesure du possible.
