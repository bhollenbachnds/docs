Ces propriétés définissent l'**image displayed** sur la surface du bouton d'un élément de bouton interactif.

***

### **Options de source d'image**

Pour spécifier l'**origin** ou le **type of picture**, choisissez l'une des options suivantes :

| **Option**                         | **Description**                                                                                                                                                                                  |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Content Library**                | Référencer un fichier image existant depuis la **PADS4 Content Library** (recommandé). Les éléments de la Content Library sont **auto-synced across all PCs**, avec des **relative references**. |
| **File Name**                      | Référencer un fichier image existant situé **outside the Content Library**. Les fichiers doivent être distribués manuellement et référencés avec des **absolute file paths**.                    |
| **External Data from Data Source** | Connecter l'élément à un **Data Provider** et spécifier quelle ligne et colonne de données fourniront l'image dynamiquement.                                                                     |

***

### **Paramètres du fournisseur de données** (Lors de l'utilisation de données externes)

| **Propriété**     | **Description**                                                                                                                                                                                                                         |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Data Provider** | Se connecter à une source telle qu'une table, une requête, une procédure stockée, un fichier XML, un flux RSS, etc. Vous pouvez créer de nouveaux fournisseurs à la volée. Voir *Working with Data Providers* pour plus d'informations. |
| **Column**        | Sélectionnez la colonne contenant la référence de l'image.                                                                                                                                                                              |
| **Content Type**  | Spécifiez comment le contenu doit être interprété (par exemple, chemin du fichier image, URL, etc.).                                                                                                                                    |
| **Row**           | Spécifiez quelle ligne de la source de données lier à l'élément.                                                                                                                                                                        |

***

### **Options d'affichage d'image**

| **Propriété**             | **Description**                                                                             |
| ------------------------- | ------------------------------------------------------------------------------------------- |
| **Stretch**               | Si activé, l'image sera automatiquement redimensionnée pour **fit the element boundaries**. |
| **Maintain Aspect Ratio** | Si activé, l'image **retain its original width-to-height ratio** lorsqu'elle est étirée.    |

***

### **Remarques**

* Seulement **références** aux fichiers image sont stockées dans la présentation, **pas les images elles-mêmes**. Si le fichier image est mis à jour, **la nouvelle version sera utilisée automatiquement** pendant la lecture.
* Si la **ligne** spécifiée n'existe pas dans la source de données, **aucune image** ne sera affichée.
* **CONSEIL** : Pour une meilleure qualité visuelle, essayez d'afficher les images à leur **taille d'origine** lorsque c'est possible.
