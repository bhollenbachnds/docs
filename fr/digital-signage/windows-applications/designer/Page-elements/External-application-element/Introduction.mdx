Les éléments d'application externe sont utilisés pour **démarrer et intégrer d'autres applications** pendant la lecture de votre présentation.

Cet élément offre diverses options pour identifier, démarrer et arrêter des applications externes. Il peut également être connecté à une **external data source** pour déterminer dynamiquement quelle application lancer lors de l'exécution.

> **Remarque** : L'élément External Application **remplace** l'élément Plug-in des versions antérieures de PADS.

***

### **Aperçu des propriétés**

| Section de propriété   | Description                                                                                                         |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Application window** | Propriétés pour détecter et intégrer la fenêtre de l'application externe dans la présentation.                      |
| **Background color**   | Définir la couleur d'arrière-plan de la zone de l'élément.                                                          |
| **Behaviour**          | Activer ou désactiver l'interactivité sur le contenu externe.                                                       |
| **Border**             | Définir la bordure autour de l'élément.                                                                             |
| **Identification**     | Attribuer un identifiant unique à l'élément dans la page.                                                           |
| **Position**           | Définir la position de l'élément sur la page.                                                                       |
| **Property bindings**  | Connecter les propriétés de l'élément à des sources de données externes.                                            |
| **Rules**              | Définir des règles pour contrôler dynamiquement le comportement de l'élément en fonction de conditions prédéfinies. |
| **Size**               | Spécifier la largeur et la hauteur de l'élément.                                                                    |
| **Start application**  | Définir quelle application externe démarrer et comment la démarrer.                                                 |
| **Stop application**   | Spécifier si et quand l'application externe doit être arrêtée.                                                      |
| **Visibility**         | Contrôlez si l'élément ou son contenu est visible pendant la lecture.                                               |

***

### **Remarque**

Cette section couvre uniquement les **propriétés spécifiques à l'application externe**.\
Pour les propriétés générales des éléments partagées entre différents éléments de page (comme l'arrière-plan, la position, la taille et la visibilité), consultez la section **Page Element Properties**.
