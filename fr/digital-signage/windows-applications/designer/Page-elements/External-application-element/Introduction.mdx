Les éléments d'application externe sont utilisés pour **démarrer et intégrer d'autres applications** pendant la lecture de votre présentation.

Cet élément fournit diverses options pour identifier, démarrer et arrêter des applications externes. Il peut également être connecté à une **source de données externe** pour déterminer dynamiquement quelle application lancer lors de l'exécution.

> **Remarque** : L'élément Application Externe **remplace** l'élément Plug-in des versions antérieures de PADS.

***

### **Aperçu des propriétés**

| Section de propriété       | Description                                                                                                         |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **Fenêtre d'application**  | Propriétés pour détecter et intégrer la fenêtre de l'application externe dans la présentation.                      |
| **Couleur d'arrière-plan** | Définir la couleur d'arrière-plan de la zone de l'élément.                                                          |
| **Comportement**           | Activer ou désactiver l'interactivité sur le contenu externe.                                                       |
| **Bordure**                | Définir la bordure autour de l'élément.                                                                             |
| **Identification**         | Attribuer un identifiant unique à l'élément dans la page.                                                           |
| **Position**               | Définir la position de l'élément sur la page.                                                                       |
| **Liaisons de propriétés** | Connecter les propriétés de l'élément à des sources de données externes.                                            |
| **Règles**                 | Définir des règles pour contrôler dynamiquement le comportement de l'élément en fonction de conditions prédéfinies. |
| **Taille**                 | Spécifier la largeur et la hauteur de l'élément.                                                                    |
| **Démarrer l'application** | Définir quelle application externe démarrer et comment la démarrer.                                                 |
| **Arrêter l'application**  | Spécifier si et quand l'application externe doit être arrêtée.                                                      |
| **Visibilité**             | Contrôler si l'élément ou son contenu est visible pendant la lecture.                                               |

***

### **Remarque**

Cette section ne couvre que les **propriétés spécifiques à l'application externe**.\
Pour les propriétés générales des éléments partagées entre différents éléments de page (comme l'arrière-plan, la position, la taille et la visibilité), consultez la section **Propriétés des éléments de page**.
