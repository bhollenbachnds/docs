Ces propriétés définissent **si et quand** l'application externe lancée par l'élément doit être **arrêtée automatiquement**.

***

### **Options d'arrêt de l'application**

| Option                                      | Description                                                                                                                                                                                                                  |
| ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Lors du passage à une autre page**        | Arrêter automatiquement l'application externe **lors du passage à la page suivante** dans la présentation.                                                                                                                   |
| **Lors de la fermeture de la présentation** | Arrêter automatiquement l'application externe **lorsque la présentation se termine**.                                                                                                                                        |
| **Ne jamais arrêter cette application**     | L'application **ne sera pas** arrêtée automatiquement. Cela n'est recommandé que lorsqu'elle est combinée avec **« Do not start this application when already running »** pour éviter de la redémarrer à plusieurs reprises. |

***

### **Remarques et notes importantes**

* **Mécanisme d'arrêt** :\
  PADS4 va **tuer** le processus en cours de l'application externe pour l'arrêter.\
  Cependant, **tous les processus enfants** ou processus supplémentaires **démarrés par cette application NE seront PAS arrêtés automatiquement**. Vous devrez les arrêter manuellement si nécessaire.

* **Avertissement concernant les performances du système** :
  > **Important** : **Assurez-vous toujours** que les applications externes sont correctement arrêtées.\
  > Ne pas le faire peut entraîner des **fuites mémoire**, une dégradation des performances ou une **instabilité du système** au fil du temps.

* **Utilisation recommandée** :
  * Si vous définissez **« Never stop this application »**, assurez-vous que :
    * L'application est correctement gérée ailleurs.
    * Vous utilisez également **« Do not start this application when already running »** pour éviter plusieurs instances.
