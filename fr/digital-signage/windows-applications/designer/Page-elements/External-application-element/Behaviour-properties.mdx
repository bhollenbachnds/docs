Ces propriétés contrôlent si les utilisateurs peuvent **interagir** avec le contenu de l'application externe intégrée dans votre présentation.

***

### **Comportement**

* **Autoriser l'Interactivité Utilisateur**\
  Activer ou désactiver l'**interaction utilisateur** avec le contenu de l'application externe.\
  Lorsque cette option est activée, les utilisateurs peuvent interagir avec les éléments à l'intérieur de l'application externe, tels que :
  * Cliquer sur des **hyperliens**
  * Appuyer sur des **boutons interactifs**
  * Naviguer dans un logiciel tiers lancé par l'élément Application Externe

***

### **Remarques**

* La désactivation de cette option rend le contenu externe **en lecture seule**, empêchant toute interaction.
* L'activation de cette option est utile lorsque vous souhaitez autoriser **l'utilisation interactive** de logiciels intégrés ou d'applications web dans le cadre de votre présentation.
