Ces propriétés définissent les **commandes RS-232** qui seront envoyées par votre **élément de communication RS-232**. Vous pouvez entrer une **commande unique** ou créer une **liste de lecture** de commandes qui seront exécutées séquentiellement.

Vous pouvez **ajouter, modifier, réorganiser ou supprimer** des commandes de la liste de lecture. Les boutons de navigation vous permettent de parcourir les éléments de la liste de lecture. La **durée totale** affiche le temps d'exécution combiné de toutes les commandes.

***

## **Élément de liste de lecture de commande RS-232**

### **Type**

Définit l'**origine** ou le **format** de la commande RS-232.

| **Option**                              | **Description**                                                                                                                                                                           |
| --------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **RS-232 Command (Text)**               | Commande statique saisie en texte brut.                                                                                                                                                   |
| **RS-232 Command (Hexadecimal)**        | Commande saisie en **format hexadécimal**.                                                                                                                                                |
| **RS-232 Command from Content Library** | Référence à un **fichier texte prédéfini** stocké dans la **bibliothèque de contenu PADS4** (recommandé). Les éléments sont **automatiquement synchronisés** sur tous les PC.             |
| **RS-232 Command from File**            | Référence à un fichier texte **en dehors de la bibliothèque de contenu PADS4**. Ces fichiers ne sont **pas synchronisés** automatiquement, une distribution manuelle est donc nécessaire. |
| **No RS-232 Command**                   | Aucune commande n'est incluse. Ceci est utilisé pour **faire une pause** entre les commandes.                                                                                             |

***

### **Contenu**

Définit la **commande** à exécuter.

* Si vous utilisez la **saisie manuelle**, tapez la **commande en texte ou en hexadécimal** directement.
* Si vous utilisez une **référence de fichier**, naviguez pour **sélectionner le fichier** contenant les commandes.
* Les références de fichiers **doivent inclure le chemin complet du fichier** si elles sont stockées en dehors de la bibliothèque de contenu.

***

### **Durée**

Spécifie la durée (en **secondes**) à **attendre** avant d'envoyer la commande suivante ou de **mettre en pause** la lecture.

***

### **Effet**

Définit l'**effet de transition entrant** pour l'élément de la liste de lecture et la **durée** (en secondes).

***

## **Remarques**

* **Seules les références de fichiers sont stockées**, pas le contenu réel de la commande. Si la **fichier texte référencé est mis à jour**, les dernières commandes seront utilisées automatiquement.
* Les **propriétés de la liste de lecture** seront **désactivées** si l'**élément est connecté à une source de données**. Dans ce cas, la **source de données en temps réel** détermine quelles commandes sont exécutées.
