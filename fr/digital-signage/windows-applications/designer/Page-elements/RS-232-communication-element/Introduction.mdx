L'**élément de communication RS-232** vous permet d'envoyer des **commandes RS-232** depuis votre présentation pour contrôler des périphériques externes, tels que des écrans, des projecteurs ou d'autres équipements contrôlés en série. Ces commandes peuvent ajuster la luminosité, changer les canaux d'entrée ou basculer les états d'alimentation.

Pour une utilisation correcte, consultez la **documentation RS-232** de votre périphérique externe pour déterminer les commandes et configurations prises en charge.

Comme l'élément de communication RS-232 **ne nécessite pas de présence visuelle**, il est généralement défini comme **invisible** dans les présentations. Cependant, il peut être connecté à une **source de données externe** pour déterminer, en temps réel, quelles commandes doivent être envoyées.

***

## **Aperçu des propriétés**

| **Section de propriété**           | **Description**                                                                                                                      |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| **Options alternatives**           | Définit comment l'élément basculera entre les éléments de la liste de lecture.                                                       |
| **Couleur d'arrière-plan**         | Définit la couleur d'arrière-plan de l'élément.                                                                                      |
| **Bordure**                        | Ajuste la couleur, la largeur et le style de la bordure de l'élément.                                                                |
| **Fournisseur de données**         | Connecte l'élément à une **source de données** externe pour les mises à jour de commandes en temps réel.                             |
| **Défilement des données**         | Détermine si et comment l'élément fait défiler les données en temps réel.                                                            |
| **Durée par défaut**               | Spécifie la durée par défaut pour les éléments de la **liste de lecture**.                                                           |
| **Effet de transition par défaut** | Définit l'**effet de transition** lors de la mise à jour du contenu.                                                                 |
| **Retournement**                   | Permet le **retournement horizontal ou vertical** du contenu.                                                                        |
| **Police**                         | Définit les paramètres de police pour les affichages de commandes RS-232 **basés sur du texte**.                                     |
| **Identification**                 | Attribue un nom unique pour identifier l'élément dans la page.                                                                       |
| **Liste de lecture**               | Définit une **séquence de commandes RS-232** à envoyer.                                                                              |
| **Options de liste de lecture**    | Détermine comment les éléments de la liste de lecture sont exécutés.                                                                 |
| **Paramètres du port**             | Définit les **paramètres du port RS-232**, y compris la vitesse de transmission, la parité, les bits de données et les bits d'arrêt. |
| **Position**                       | Détermine la position de l'élément dans la page.                                                                                     |
| **Preuve de lecture**              | Enregistre les données de lecture à des fins de **rapport et de vérification**.                                                      |
| **Liaisons de propriétés**         | Lie des propriétés spécifiques de l'élément à une source de données externe.                                                         |
| **Réflexion**                      | Active un **effet de réflexion** sur l'élément.                                                                                      |
| **Rotation**                       | Vous permet de **faire pivoter** l'élément selon un certain angle.                                                                   |
| **Règles**                         | Définit les **règles d'élément** pour un comportement dynamique.                                                                     |
| **Ombre**                          | Ajoute un effet d'ombre à l'élément.                                                                                                 |
| **Taille**                         | Définit la largeur et la hauteur de l'élément.                                                                                       |
| **Alignement du texte**            | Ajuste l'alignement **horizontal et vertical** des commandes basées sur du texte.                                                    |
| **Transparence**                   | Définit le niveau de transparence de l'élément.                                                                                      |
| **Visibilité**                     | Contrôle si l'élément est visible pendant la lecture.                                                                                |

***

### **Remarques**

* **Assurez-vous que les paramètres RS-232 corrects** (vitesse de transmission, parité, bits de données, bits d'arrêt) correspondent aux spécifications de votre **périphérique externe**.
* **Les éléments RS-232 restent généralement invisibles**, car ils jouent un rôle fonctionnel plutôt que visuel.
* Si connecté à une **source de données externe**, les commandes peuvent changer dynamiquement en temps réel.
* **L'enregistrement de preuve de lecture** peut être utile pour surveiller quand et où les commandes RS-232 ont été exécutées.
