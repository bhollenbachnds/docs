L'**RS-232 communication element** vous permet d'envoyer des **RS-232 commands** depuis votre présentation pour contrôler des périphériques externes, tels que des écrans, des projecteurs ou d'autres équipements contrôlés en série. Ces commandes peuvent ajuster la luminosité, changer les canaux d'entrée ou basculer les états d'alimentation.

Pour une utilisation correcte, consultez la **RS-232 documentation** de votre périphérique externe pour déterminer les commandes et configurations prises en charge.

Comme l'élément de communication RS-232 **does not require a visual presence**, il est généralement défini comme **invisible** dans les présentations. Cependant, il peut être connecté à une **external data source** pour déterminer, en temps réel, quelles commandes doivent être envoyées.

***

## **Aperçu des propriétés**

| **Section de propriété**      | **Description**                                                                                                        |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| **Alternate Options**         | Définit comment l'élément basculera entre les éléments de la liste de lecture.                                         |
| **Background Color**          | Définit la couleur d'arrière-plan de l'élément.                                                                        |
| **Border**                    | Ajuste la couleur, la largeur et le style de la bordure de l'élément.                                                  |
| **Data Provider**             | Connecte l'élément à une **data source** externe pour les mises à jour de commandes en temps réel.                     |
| **Data Scrolling**            | Détermine si et comment l'élément fait défiler les données en temps réel.                                              |
| **Default Duration**          | Spécifie la durée par défaut pour les éléments de la **play list**.                                                    |
| **Default Transition Effect** | Définit l'**transition effect** lors de la mise à jour du contenu.                                                     |
| **Flip**                      | Permet le **horizontal or vertical flipping** du contenu.                                                              |
| **Font**                      | Définit les paramètres de police pour les affichages de commandes RS-232 **text-based**.                               |
| **Identification**            | Attribue un nom unique pour identifier l'élément dans la page.                                                         |
| **Play List**                 | Définit une **sequence of RS-232 commands** à envoyer.                                                                 |
| **Play List Options**         | Détermine comment les éléments de la liste de lecture sont exécutés.                                                   |
| **Port Settings**             | Définit les **RS-232 port settings**, y compris le débit en bauds, la parité, les bits de données et les bits d'arrêt. |
| **Position**                  | Détermine la position de l'élément dans la page.                                                                       |
| **Proof of Play**             | Enregistre les données de lecture à des fins de **reporting and verification**.                                        |
| **Property Bindings**         | Lie des propriétés spécifiques de l'élément à une source de données externe.                                           |
| **Reflection**                | Active un **reflection effect** sur l'élément.                                                                         |
| **Rotation**                  | Vous permet de **rotate** l'élément à un certain angle.                                                                |
| **Rules**                     | Définit les **element rules** pour un comportement dynamique.                                                          |
| **Shadow**                    | Ajoute un effet d'ombre à l'élément.                                                                                   |
| **Taille**                    | Définit la largeur et la hauteur de l'élément.                                                                         |
| **Alignement du texte**       | Ajuste le **horizontal et vertical** alignement des commandes basées sur le texte.                                     |
| **Transparence**              | Définit le niveau de transparence de l'élément.                                                                        |
| **Visibilité**                | Contrôle si l'élément est visible pendant la lecture.                                                                  |

***

### **Remarques**

* **Assurez-vous que les paramètres RS-232 corrects** (débit en bauds, parité, bits de données, bits d'arrêt) correspondent aux spécifications de votre **périphérique externe**.
* **Les éléments RS-232 restent généralement invisibles**, car ils jouent un rôle fonctionnel plutôt que visuel.
* Si connecté à une **source de données externe**, les commandes peuvent changer dynamiquement en temps réel.
* **L'enregistrement Proof of Play** peut être utile pour surveiller quand et où les commandes RS-232 ont été exécutées.
