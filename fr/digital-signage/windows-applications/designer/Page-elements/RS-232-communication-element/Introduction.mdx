L'**élément de communication RS-232** vous permet d'envoyer des **commandes RS-232** depuis votre présentation pour contrôler des périphériques externes, tels que des écrans, des projecteurs ou d'autres équipements contrôlés en série. Ces commandes peuvent ajuster la luminosité, changer les canaux d'entrée ou basculer les états d'alimentation.

Pour une utilisation correcte, consultez la **documentation RS-232** de votre périphérique externe pour déterminer les commandes et configurations prises en charge.

Comme l'élément de communication RS-232 **ne nécessite pas de présence visuelle**, il est généralement défini comme **invisible** dans les présentations. Cependant, il peut être connecté à une **source de données externe** pour déterminer, en temps réel, quelles commandes doivent être envoyées.

***

## **Aperçu des Propriétés**

| **Section de Propriété**      | **Description**                                                                                                        |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| **Options Alternatives**      | Définit comment l'élément basculera entre les éléments de la liste de lecture.                                         |
| **Couleur d'Arrière-plan**    | Définit la couleur d'arrière-plan de l'élément.                                                                        |
| **Bordure**                   | Ajuste la couleur, la largeur et le style de la bordure de l'élément.                                                  |
| **Fournisseur de Données**    | Connecte l'élément à une **source de données** externe pour les mises à jour de commandes en temps réel.               |
| **Défilement des Données**    | Détermine si et comment l'élément fait défiler les données en temps réel.                                              |
| **Durée par Défaut**          | Spécifie la durée par défaut pour **play list** items.                                                                 |
| **Default Transition Effect** | Définit le **transition effect** lors de la mise à jour du contenu.                                                    |
| **Flip**                      | Permet **horizontal or vertical flipping** du contenu.                                                                 |
| **Font**                      | Définit les paramètres de police pour **text-based** affichages de commandes RS-232.                                   |
| **Identification**            | Attribue un nom unique pour identifier l'élément dans la page.                                                         |
| **Play List**                 | Définit une **sequence of RS-232 commands** à envoyer.                                                                 |
| **Play List Options**         | Détermine comment les éléments de la play list sont exécutés.                                                          |
| **Port Settings**             | Définit les **RS-232 port settings**, y compris le débit en bauds, la parité, les bits de données et les bits d'arrêt. |
| **Position**                  | Détermine la position de l'élément dans la page.                                                                       |
| **Proof of Play**             | Enregistre les données de lecture pour **reporting and verification** des fins.                                        |
| **Property Bindings**         | Lie des propriétés spécifiques de l'élément à une source de données externe.                                           |
| **Reflection**                | Active un **reflection effect** sur l'élément.                                                                         |
| **Rotation**                  | Vous permet de **rotate** l'élément selon un certain angle.                                                            |
| **Rules**                     | Définit **element rules** pour un comportement dynamique.                                                              |
| **Shadow**                    | Ajoute un effet d'ombre à l'élément.                                                                                   |
| **Size**                      | Définit la largeur et la hauteur de l'élément.                                                                         |
| **Text Alignment**            | Ajuste l'**horizontal and vertical** alignement des commandes basées sur du texte.                                     |
| **Transparency**              | Définit le niveau de transparence de l'élément.                                                                        |
| **Visibility**                | Contrôle si l'élément est visible pendant la lecture.                                                                  |

***

### **Remarks**

* **Ensure the correct RS-232 settings** (débit en bauds, parité, bits de données, bits d'arrêt) correspondent aux spécifications de votre **external device**.
* **RS-232 elements typically remain invisible**, car ils jouent un rôle fonctionnel plutôt que visuel.
* Si connecté à une **external data source**, les commandes peuvent changer dynamiquement en temps réel.
* **Proof of Play logging** peut être utile pour surveiller quand et où les commandes RS-232 ont été exécutées.
