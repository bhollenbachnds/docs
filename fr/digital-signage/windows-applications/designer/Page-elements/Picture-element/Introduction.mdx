**Les éléments image** sont utilisés pour afficher **des logos, des photos ou d'autres fichiers graphiques**.\
Tous les **formats d'image standard du secteur** sont pris en charge.

Les images peuvent être affichées à leur **taille originale** ou **redimensionnées automatiquement** pour s'adapter à la conception de votre page.\
Vous pouvez appliquer **divers effets** pour ajuster l'apparence ou créer des **transitions fluides** lors des changements d'images.\
Les éléments image peuvent également être **connectés à des sources de données externes** pour afficher dynamiquement du contenu en temps réel.

***

### **Aperçu des propriétés**

| **Section de propriété**                        | **Description**                                                                                    |
| ----------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| **Alignement**                                  | Définir l'**alignement horizontal et vertical** des images.                                        |
| **Options alternatives**                        | Contrôler **la façon dont l'élément bascule entre les éléments de la liste de lecture**.           |
| **Couleur d'arrière-plan**                      | Définir la **couleur d'arrière-plan** de votre élément.                                            |
| **Clignotement**                                | Définir un **effet de clignotement** pour votre élément.                                           |
| **Bordure**                                     | Définir **la couleur, la largeur et le style de la bordure** pour votre élément.                   |
| **Luminosité/contraste**                        | Ajuster la **luminosité et le contraste** des images.                                              |
| **Fournisseur de données**                      | Se connecter à une **source de données** pour du contenu en temps réel.                            |
| **Conversion de texte de la source de données** | Gérer le **texte multiligne** pour les sources de données liées.                                   |
| **Durée par défaut**                            | Définir la **durée par défaut** pour les éléments de la liste de lecture.                          |
| **Effet de transition par défaut**              | Définir un **effet de transition par défaut** pour le contenu entrant.                             |
| **Retournement**                                | Retourner le contenu **horizontalement ou verticalement**.                                         |
| **Identification**                              | Définir un **nom unique** pour identifier votre élément sur la page.                               |
| **Liste de lecture**                            | Définir les **images à afficher**.                                                                 |
| **Options de liste de lecture**                 | Définir la façon dont les éléments de la liste de lecture seront lus (par ex., répéter, une fois). |
| **Position**                                    | Définir la **position** de votre élément sur la page.                                              |
| **Preuve de diffusion**                         | **Enregistrer les détails de lecture** à des fins de suivi et de rapport.                          |
| **Liaisons de propriétés**                      | Lier les propriétés de l'élément à des **sources de données externes**.                            |
| **Reflet**                                      | Afficher le contenu avec un **effet de reflet**.                                                   |
| **Rotation**                                    | Faire pivoter l'élément à un **angle**.                                                            |
| **Règles**                                      | Créer des **règles** pour ajuster automatiquement le comportement des éléments.                    |
| **Ombre**                                       | Ajouter un **effet d'ombre** à votre élément.                                                      |
| **Taille**                                      | Définir la **largeur et la hauteur** de l'élément.                                                 |
| **Inclinaison**                                 | Appliquer un **angle d'inclinaison** à l'élément.                                                  |
| **Transparence**                                | Définir le **niveau de transparence** de l'élément.                                                |
| **Visibilité**                                  | Contrôler si l'élément est **visible pendant la lecture**.                                         |

***

### **Remarque**

* Cette section décrit **uniquement les propriétés spécifiques aux images**.
* Pour les propriétés communes partagées entre les éléments de page, reportez-vous à la section **Propriétés des éléments de page**.
