**Picture elements** sont utilisés pour afficher **logos, photos, or other graphic files**.\
Tous les **industry-standard image formats** sont pris en charge.

Les images peuvent être affichées à leur **original size** ou **automatically resized** pour s'adapter à votre conception de page.\
Vous pouvez appliquer **various effects** pour ajuster l'apparence ou créer des **smooth transitions** lorsque les images changent.\
Les éléments d'image peuvent également être **connected to external data sources** pour afficher dynamiquement du contenu en temps réel.

***

### **Aperçu des propriétés**

| **Section de propriété**        | **Description**                                                                          |
| ------------------------------- | ---------------------------------------------------------------------------------------- |
| **Alignment**                   | Définir **horizontal and vertical alignment** des images.                                |
| **Alternate options**           | Contrôler **how the element switches between playlist items**.                           |
| **Background color**            | Définir la **background color** de votre élément.                                        |
| **Blinking**                    | Définir un **blinking effect** pour votre élément.                                       |
| **Border**                      | Définir **border color, width, and style** pour votre élément.                           |
| **Brightness/contrast**         | Ajuster la **brightness and contrast** des images.                                       |
| **Data provider**               | Se connecter à une **data source** pour du contenu en temps réel.                        |
| **Data source text conversion** | Gérer le **multi-line text** pour les sources de données liées.                          |
| **Default duration**            | Définir la **default duration** pour les éléments de la playlist.                        |
| **Default transition effect**   | Définir un **default transition effect** pour le contenu entrant.                        |
| **Flip**                        | Retourner le contenu **horizontally or vertically**.                                     |
| **Identification**              | Définir un **unique name** pour identifier votre élément sur la page.                    |
| **Play list**                   | Définir les **pictures to be displayed**.                                                |
| **Play list options**           | Définir comment les éléments de la playlist seront lus (par exemple, répéter, une fois). |
| **Position**                    | Définir la **position** de votre élément sur la page.                                    |
| **Proof of play**               | **Log playback details** à des fins de suivi et de rapports.                             |
| **Property bindings**           | Lier les propriétés de l'élément à des **external data sources**.                        |
| **Reflection**                  | Afficher le contenu avec un **reflection effect**.                                       |
| **Rotation**                    | Faire pivoter l'élément selon un **angle** spécifié.                                     |
| **Rules**                       | Créer des **rules** pour ajuster automatiquement le comportement de l'élément.           |
| **Shadow**                      | Ajouter un **shadow effect** à votre élément.                                            |
| **Size**                        | Définir la **width and height** de l'élément.                                            |
| **Skew**                        | Appliquer un **skew angle** à l'élément.                                                 |
| **Transparency**                | Définir le **transparency level** de l'élément.                                          |
| **Visibility**                  | Contrôler si l'élément est **visible during playback**.                                  |

***

### **Remarque**

* Cette section décrit **propriétés spécifiques aux images uniquement**.
* Pour les propriétés communes partagées entre les éléments de page, consultez la section **Propriétés des éléments de page**.
