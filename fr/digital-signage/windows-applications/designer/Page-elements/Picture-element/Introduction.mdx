**Picture elements** sont utilisés pour afficher **des logos, des photos ou d'autres fichiers graphiques**.\
Tous les **formats d'image standard de l'industrie** sont pris en charge.

Les images peuvent être affichées à leur **taille originale** ou **redimensionnées automatiquement** pour s'adapter à la conception de votre page.\
Vous pouvez appliquer **divers effets** pour ajuster l'apparence ou créer des **transitions fluides** lors des changements d'images.\
Les Picture elements peuvent également être **connectés à des sources de données externes** pour afficher dynamiquement du contenu en temps réel.

***

### **Aperçu des propriétés**

| **Section de propriété**        | **Description**                                                                      |
| ------------------------------- | ------------------------------------------------------------------------------------ |
| **Alignment**                   | Définir l' **alignement horizontal et vertical** des images.                         |
| **Alternate options**           | Contrôler **comment l'élément bascule entre les éléments de la playlist**.           |
| **Background color**            | Définir la **couleur d'arrière-plan** de votre élément.                              |
| **Blinking**                    | Définir un **effet de clignotement** pour votre élément.                             |
| **Border**                      | Définir **la couleur, la largeur et le style de la bordure** pour votre élément.     |
| **Brightness/contrast**         | Ajuster la **luminosité et le contraste** des images.                                |
| **Data provider**               | Se connecter à une **source de données** pour du contenu en temps réel.              |
| **Data source text conversion** | Gérer le **texte multiligne** pour les sources de données liées.                     |
| **Default duration**            | Définir la **durée par défaut** pour les éléments de la playlist.                    |
| **Default transition effect**   | Définir un **effet de transition par défaut** pour le contenu entrant.               |
| **Flip**                        | Retourner le contenu **horizontalement ou verticalement**.                           |
| **Identification**              | Définir un **nom unique** pour identifier votre élément sur la page.                 |
| **Play list**                   | Définir les **images à afficher**.                                                   |
| **Play list options**           | Définir comment les éléments de la playlist seront lus (par ex., répéter, une fois). |
| **Position**                    | Définir la **position** de votre élément sur la page.                                |
| **Proof of play**               | **Enregistrer les détails de lecture** à des fins de suivi et de rapport.            |
| **Property bindings**           | Lier les propriétés de l'élément à des **sources de données externes**.              |
| **Reflection**                  | Afficher le contenu avec un **effet de reflet**.                                     |
| **Rotation**                    | Faire pivoter l'élément à un **angle** spécifié.                                     |
| **Rules**                       | Créer des **règles** pour ajuster automatiquement le comportement de l'élément.      |
| **Shadow**                      | Ajouter un **effet d'ombre** à votre élément.                                        |
| **Size**                        | Définir la **largeur et la hauteur** de l'élément.                                   |
| **Skew**                        | Appliquer un **angle d'inclinaison** à l'élément.                                    |
| **Transparency**                | Définir le **niveau de transparence** de l'élément.                                  |
| **Visibility**                  | Contrôler si l'élément est **visible pendant la lecture**.                           |

***

### **Remarque**

* Cette section décrit **uniquement les propriétés spécifiques aux images**.
* Pour les propriétés communes partagées entre les éléments de page, référez-vous à la section **Page Element Properties**.
