**Les éléments image** sont utilisés pour afficher des **logos, photos ou autres fichiers graphiques**.\
Tous les **formats d'image standard du secteur** sont pris en charge.

Les images peuvent être affichées à leur **taille originale** ou **redimensionnées automatiquement** pour s'adapter à la conception de votre page.\
Vous pouvez appliquer **divers effets** pour ajuster l'apparence ou créer des **transitions fluides** lors des changements d'images.\
Les éléments image peuvent également être **connectés à des sources de données externes** pour afficher dynamiquement du contenu en temps réel.

***

### **Aperçu des propriétés**

| **Section de propriété**                        | **Description**                                                                                     |
| ----------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| **Alignement**                                  | Définissez l'**alignement horizontal et vertical** des images.                                      |
| **Options alternatives**                        | Contrôlez **la façon dont l'élément bascule entre les éléments de la liste de lecture**.            |
| **Couleur d'arrière-plan**                      | Définissez la **couleur d'arrière-plan** de votre élément.                                          |
| **Clignotement**                                | Définissez un **effet de clignotement** pour votre élément.                                         |
| **Bordure**                                     | Définissez la **couleur, la largeur et le style de la bordure** de votre élément.                   |
| **Luminosité/contraste**                        | Ajustez la **luminosité et le contraste** des images.                                               |
| **Fournisseur de données**                      | Connectez-vous à une **source de données** pour du contenu en temps réel.                           |
| **Conversion de texte de la source de données** | Gérez le **texte multiligne** pour les sources de données liées.                                    |
| **Durée par défaut**                            | Définissez la **durée par défaut** des éléments de la liste de lecture.                             |
| **Effet de transition par défaut**              | Définissez un **effet de transition par défaut** pour le contenu entrant.                           |
| **Retournement**                                | Retournez le contenu **horizontalement ou verticalement**.                                          |
| **Identification**                              | Définissez un **nom unique** pour identifier votre élément sur la page.                             |
| **Liste de lecture**                            | Définir les **images à afficher**.                                                                  |
| **Options de liste de lecture**                 | Définir comment les éléments de la liste de lecture seront lus (par exemple, répétition, une fois). |
| **Position**                                    | Définir la **position** de votre élément sur la page.                                               |
| **Preuve de lecture**                           | **Enregistrer les détails de lecture** à des fins de suivi et de rapport.                           |
| **Liaisons de propriétés**                      | Lier les propriétés des éléments à **des sources de données externes**.                             |
| **Reflet**                                      | Afficher le contenu avec un **effet de reflet**.                                                    |
| **Rotation**                                    | Faire pivoter l'élément à un **angle** spécifié.                                                    |
| **Règles**                                      | Créer des **règles** pour ajuster automatiquement le comportement des éléments.                     |
| **Ombre**                                       | Ajouter un **effet d'ombre** à votre élément.                                                       |
| **Taille**                                      | Définir la **largeur et la hauteur** de l'élément.                                                  |
| **Inclinaison**                                 | Appliquer un **angle d'inclinaison** à l'élément.                                                   |
| **Transparence**                                | Définir le **niveau de transparence** de l'élément.                                                 |
| **Visibilité**                                  | Contrôler si l'élément est **visible pendant la lecture**.                                          |

***

### **Remarque**

* Cette section décrit **uniquement les propriétés spécifiques aux images**.
* Pour les propriétés communes partagées entre les éléments de page, référez-vous à la section **Propriétés des éléments de page**.
