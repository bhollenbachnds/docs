**Picture elements** sont utilisés pour afficher des **logos, photos ou autres fichiers graphiques**.\
Tous les **formats d'image standard de l'industrie** sont pris en charge.

Les images peuvent être affichées à leur **taille d'origine** ou **redimensionnées automatiquement** pour s'adapter à votre conception de page.\
Vous pouvez appliquer **divers effets** pour ajuster l'apparence ou créer des **transitions fluides** lorsque les images changent.\
Les éléments d'image peuvent également être **connectés à des sources de données externes** pour afficher dynamiquement du contenu en temps réel.

***

### **Aperçu des propriétés**

| **Section de propriété**        | **Description**                                                                                     |
| ------------------------------- | --------------------------------------------------------------------------------------------------- |
| **Alignment**                   | Définir l'**alignement horizontal et vertical** des images.                                         |
| **Alternate options**           | Contrôler **comment l'élément bascule entre les éléments de la playlist**.                          |
| **Background color**            | Définir la **couleur d'arrière-plan** de votre élément.                                             |
| **Blinking**                    | Définir un **effet de clignotement** pour votre élément.                                            |
| **Border**                      | Définir la **couleur, la largeur et le style de bordure** pour votre élément.                       |
| **Brightness/contrast**         | Ajuster la **luminosité et le contraste** des images.                                               |
| **Data provider**               | Se connecter à une **source de données** pour du contenu en temps réel.                             |
| **Data source text conversion** | Gérer le **texte multi-lignes** pour les sources de données liées.                                  |
| **Default duration**            | Définir la **durée par défaut** pour les éléments de la playlist.                                   |
| **Default transition effect**   | Définir un **effet de transition par défaut** pour le contenu entrant.                              |
| **Flip**                        | Retourner le contenu **horizontalement ou verticalement**.                                          |
| **Identification**              | Définir un **nom unique** pour identifier votre élément sur la page.                                |
| **Play list**                   | Définir les **images à afficher**.                                                                  |
| **Options de liste de lecture** | Définir comment les éléments de la liste de lecture seront lus (par exemple, répétition, une fois). |
| **Position**                    | Définir la **position** de votre élément sur la page.                                               |
| **Preuve de diffusion**         | **Enregistrer les détails de lecture** à des fins de suivi et de rapport.                           |
| **Liaisons de propriétés**      | Lier les propriétés des éléments à **des sources de données externes**.                             |
| **Réflexion**                   | Afficher le contenu avec un **effet de réflexion**.                                                 |
| **Rotation**                    | Faire pivoter l'élément à un **angle** spécifié.                                                    |
| **Règles**                      | Créer des **règles** pour ajuster automatiquement le comportement des éléments.                     |
| **Ombre**                       | Ajouter un **effet d'ombre** à votre élément.                                                       |
| **Taille**                      | Définir la **largeur et la hauteur** de l'élément.                                                  |
| **Inclinaison**                 | Appliquer un **angle d'inclinaison** à l'élément.                                                   |
| **Transparence**                | Définir le **niveau de transparence** de l'élément.                                                 |
| **Visibilité**                  | Contrôler si l'élément est **visible pendant la lecture**.                                          |

***

### **Remarque**

* Cette section décrit **uniquement les propriétés spécifiques aux images**.
* Pour les propriétés communes partagées entre les éléments de page, consultez la section **Propriétés des éléments de page**.
