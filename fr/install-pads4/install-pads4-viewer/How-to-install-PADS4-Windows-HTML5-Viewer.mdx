## Exigences

* Windows 10 build 1909 ou supérieur
* PADS4 Server version 2025.1 ou supérieur

## Aperçu

Le Windows HTML5 Viewer offre plusieurs avantages par rapport au Windows Viewer classique :

* Prend en charge la fonctionnalité de répétition synchronisée
* Permet les changements d'orientation sans modifier les paramètres Windows
* Fournit une expérience cohérente sur tous les visualiseurs HTML
* Peut être utilisé comme économiseur d'écran
* Support plus facile pour les nouvelles fonctionnalités
* Performances améliorées
* Sécurité renforcée via le mode Kiosque
* Mises à jour automatiques depuis le PADS4 Server (aucune mise à jour manuelle requise)

> Note: This viewer shares the same limitations as other HTML viewers. Refer to the [Limitations connues du HTML5 Viewer](https://support.pads4.com/hc/en-us/articles/207590696-Known-limitations-for-HTML5-Viewer) pour plus d'informations.

## Instructions d'installation

1. Exécuter `PADS4WindowsHTMLviewer.exe` en tant qu'**Administrateur**.
2. Choisir le répertoire d'installation (ou utiliser celui par défaut).
3. **Optionnel** : Décocher *"Configure Windows to launch the PADS4 HTML5 Viewer in Kiosk Mode"* if:
   * Vous voulez l'utiliser comme économiseur d'écran
   * Vous voulez l'exécuter sous un utilisateur Windows standard
4. Accepter le contrat de licence et cliquer sur **Install**.

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/installer.png" />

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/installer-progress.png" />

5. Une fois l'installation terminée, cliquer sur **Reboot** pour redémarrer la machine.

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/installation-succesful.png" />

## Configuration initiale

Après le redémarrage, le visualiseur se lancera en **Kiosk Mode**.

1. Au premier lancement, saisissez votre **PADS4 Server Address** et **Viewer Name**.

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/server-config.png" />

2. Cliquez sur **Start** pour connecter le visualiseur.

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/viewer-started.png" />

Pour accéder aux paramètres du visualiseur ou revenir aux paramètres du lanceur, faites un clic droit n'importe où sur l'écran.

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/viewer-settings.png" />

***

## Passer à un compte utilisateur standard

Vous pouvez quitter le mode Kiosque et passer à un compte Windows standard de deux façons :

* **Méthode 1** : Redémarrer la machine tout en maintenant la touche **Shift** enfoncée. Cela affiche l'écran de connexion Windows.
* **Méthode 2** : Utiliser **Remote Desktop** (si configuré) pour se connecter directement avec un compte utilisateur différent.

***

## Utiliser le visualiseur comme économiseur d'écran

1. Pendant l'installation, assurez-vous que le mode Kiosque **n'est pas** activé.
2. Ouvrir **Screensaver Settings** dans Windows.
3. Sélectionner **Pads4viewer** comme économiseur d'écran.

<img height="200" src="/images/install-pads4/install-pads4-viewer/intall-pads4-html5-windows-viewer/screensaver.png" />

4. Cliquez **Appliquer** et **OK**.

Le visualiseur démarrera maintenant en fonction de la minuterie d'économiseur d'écran configurée.

***

## Référence du script du mode Kiosk

Pour examiner la configuration du mode Kiosk du système, consultez le script PowerShell :

```plaintext
C:\Program Files\PADS4\Launcher Service\configure-kioskmode
```

## Fonctionnalités prises en charge et limitations

| Fonctionnalité                          | Description                                                     |
| --------------------------------------- | --------------------------------------------------------------- |
| ✅ Mises à jour automatiques du logiciel | Mises à jour automatiques depuis le serveur PADS4               |
| ✅ Mode hors ligne                       | Exécute des présentations jusqu'à 7 jours sans connexion réseau |
| ✅ Prise en charge HTTP & HTTPS          | Les deux protocoles sont pris en charge                         |
| ✅ Max 4 vidéos par page                 | Jusqu'à 4 vidéos peuvent être affichées sur une seule page      |
| ❌ Rotation vidéo / Seamless / Flip      | Ces améliorations vidéo ne sont pas prises en charge            |
| ✅ Prise en charge du streaming          | Streaming via HTTP & HTTPS                                      |
| ✅ Mode Cluster Viewer                   | Pris en charge                                                  |
| ✅ Éléments interactifs                  | Boutons, texte, texte défilant, image, tableaux                 |
| ✅ PowerPoint & PDF                      | PowerPoint nécessite une installation sur le serveur PADS4      |
| ✅ Gestion des appareils à distance      | Gérée via PADS4 Agent                                           |
| ✅ Wake-on-LAN                           | Les appareils peuvent être allumés à distance                   |
| ❌ Éteindre l'écran                      | Non pris en charge                                              |
