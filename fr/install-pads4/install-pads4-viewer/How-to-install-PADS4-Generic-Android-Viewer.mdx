## **Installation de PADS4 Launcher à partir d'une clé USB**

<Steps>
  <Step title="Première étape">
    Copiez le **PADS4 Launcher APK** sur une **clé USB**.
  </Step>

  <Step title="Deuxième étape">
    Insérez la **clé USB** dans l'appareil Android.
  </Step>

  <Step title="Troisième étape">
    Ouvrir **APK Installer** sur l'appareil.
  </Step>

  <Step title="Étape Quatre">
    Sélectionnez **Install** et naviguez jusqu'au fichier APK sur la **clé USB**.
  </Step>

  <Step title="Étape Cinq">
    Lorsque vous êtes invité avec :\
    *"Voulez-vous installer cette application ?"*

    * Appuyez sur **Next** jusqu'à ce que le bouton **Install** apparaisse.
    * Appuyez sur **Install** pour continuer.
  </Step>

  <Step title="Étape Six">
    Après l'installation, appuyez sur **Open** pour lancer le **PADS4 Launcher**.
  </Step>

  <Step title="Étape Sept">
    Entrez l'**PADS4 Server location**, en spécifiant :

    * **Domain name**
    * **IP address**
    * **PC name** du serveur exécutant **PADS4 Server**
  </Step>

  <Step title="Étape Huit - *Optionnel*">
    Si les **server settings** par défaut ont été modifiés, allez dans **Advanced Settings** et entrez :

    * Le **port number**
    * Le **host/root path**
  </Step>

  <Step title="Étape Neuf">
    Appuyez sur **Start**.

    * La première fois que l'application s'exécute, elle vous invitera à sélectionner votre **home app**.
    * Sélectionnez **PADS4 Launcher** et choisissez **Always** pour éviter les invites futures.
  </Step>

  <Step title="Étape Dix">
    Assurez-vous d'avoir une **valid connection** au **PADS4 Server** lors du lancement du **PADS4 Launcher** pour la première fois.

    * Cela permet à l'appareil de **download and start PADS4 Viewer**.
    * Une fois téléchargé, **PADS4 Viewer** peut fonctionner **offline**.
  </Step>
</Steps>

## **Mise à jour de PADS4 Launcher à partir d'une clé USB**

<Steps>
  <Step title="Étape Un">
    Copiez le dernier **PADS4 Launcher APK** sur une **clé USB**.
  </Step>

  <Step title="Étape Deux">
    Insérez la **clé USB** dans l'appareil Android.
  </Step>

  <Step title="Étape Trois">
    Ouvrez **Viewer Settings**.
  </Step>

  <Step title="Étape Quatre">
    Naviguez vers **Launcher Settings**.
  </Step>

  <Step title="Étape Cinq">
    Appuyez sur l'**icône de dossier** dans le **coin inférieur droit**.
  </Step>

  <Step title="Étape Six">
    Naviguez vers :

    ```
    mnt -> usb_storage  
    ```

    Sélectionnez le **fichier APK**.
  </Step>

  <Step title="Étape Sept">
    Lorsque vous êtes invité avec :\
    *"Voulez-vous installer une mise à jour de cette application existante ?"*

    * Appuyez sur **Install**.
  </Step>

  <Step title="Étape Huit">
    Une fois l'installation terminée, le **PADS4 Launcher** redémarrera automatiquement.
  </Step>
</Steps>
