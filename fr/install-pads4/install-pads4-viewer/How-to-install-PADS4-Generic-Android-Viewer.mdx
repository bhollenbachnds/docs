Ce guide explique comment installer **PADS4 Launcher APK** pour démarrer le **PADS4 Viewer** sur **du matériel Android certifié NDS**.

<Note>
  Certains appareils sont livrés avec un **PADS4 Launcher dédié**. Pour plus d'informations, consultez la page [**PADS4 Certified Devices**](#).
</Note>

***

## **Téléchargement du PADS4 Launcher APK**

Téléchargez le dernier **PADS4 Launcher APK** from:\
[**PADS4 Launcher Download**](http://go.pads4.com/?fwlink=1004)

<Warning>
  Le PADS4 Generic Android Launcher est uniquement pris en charge sur la version PADS4 Server **2025.1** et supérieure. Veuillez vous assurer que votre serveur est mis à jour en conséquence.
</Warning>

### **Versionnement**

Pour les détails de version et la compatibilité, visitez la [**PADS4 Launcher Download Page**](http://go.pads4.com/?fwlink=1004).

## **Installation de PADS4 Launcher à partir d'une clé USB**

<Steps>
  <Step title="Étape Un">
    Copiez le **PADS4 Launcher APK** sur une **clé USB**.
  </Step>

  <Step title="Étape Deux">
    Insérez la **clé USB** dans l'appareil Android.
  </Step>

  <Step title="Étape Trois">
    Ouvrez **APK Installer** sur l'appareil.
  </Step>

  <Step title="Étape Quatre">
    Sélectionnez **Install** et naviguez jusqu'au fichier APK sur la **clé USB**.
  </Step>

  <Step title="Étape Cinq">
    Lorsque vous êtes invité avec :\
    *"Voulez-vous installer cette application ?"*

    * Appuyez sur **Next** jusqu'à ce que le bouton **Install** apparaisse.
    * Appuyez sur **Install** pour continuer.
  </Step>

  <Step title="Étape Six">
    Après l'installation, appuyez sur **Open** pour lancer le **PADS4 Launcher**.
  </Step>

  <Step title="Étape Sept">
    Entrez l'**emplacement du serveur PADS4**, en spécifiant :

    * **Nom de domaine**
    * **Adresse IP**
    * **Nom du PC** du serveur exécutant **PADS4 Server**
  </Step>

  <Step title="Étape Huit - *Optionnel*">
    Si les **paramètres par défaut du serveur** ont été modifiés, allez dans **Advanced Settings** et entrez :

    * Le **numéro de port**
    * Le **chemin d'hôte/racine**
  </Step>

  <Step title="Étape Neuf">
    Appuyez sur **Start**.

    * La première fois que l'application s'exécute, elle vous demandera de sélectionner votre **application d'accueil**.
    * Sélectionnez **PADS4 Launcher** et choisissez **Always** pour éviter les invites futures.
  </Step>

  <Step title="Étape Dix">
    Assurez-vous d'avoir une **connexion valide** au **PADS4 Server** lors du lancement du **PADS4 Launcher** pour la première fois.

    * Cela permet à l'appareil de **télécharger et démarrer PADS4 Viewer**.
    * Une fois téléchargé, **PADS4 Viewer** peut fonctionner **hors ligne**.
  </Step>
</Steps>

## **Mise à jour de PADS4 Launcher à partir d'une clé USB**

<Steps>
  <Step title="Étape Une">
    Copiez le dernier **PADS4 Launcher APK** sur une **clé USB**.
  </Step>

  <Step title="Étape Deux">
    Insérez la **clé USB** dans l'appareil Android.
  </Step>

  <Step title="Étape Trois">
    Ouvrez **Viewer Settings**.
  </Step>

  <Step title="Étape Quatre">
    Naviguez vers **Launcher Settings**.
  </Step>

  <Step title="Étape Cinq">
    Appuyez sur **l'icône de dossier** dans le **coin inférieur droit**.
  </Step>

  <Step title="Étape Six">
    Naviguez vers :

    ```
    mnt -> usb_storage  
    ```

    Sélectionnez le **fichier APK**.
  </Step>

  <Step title="Étape Sept">
    Lorsque vous êtes invité avec :\
    *"Voulez-vous installer une mise à jour de cette application existante ?"*

    * Appuyez sur **Installer**.
  </Step>

  <Step title="Étape Huit">
    Une fois l'installation terminée, le **PADS4 Launcher** redémarrera automatiquement.
  </Step>
</Steps>
