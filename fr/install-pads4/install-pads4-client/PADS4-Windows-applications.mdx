---
- title: Applications Windows PADS4
- description: Installez le package PADS4 Clients (Designer, Scheduler, Agent et
    System Management) sur une machine Windows.
---

Le programme d'installation de PADS4 Clients regroupe les applications Windows — Designer, Scheduler, Agent et System Management — en un seul package. Suivez les étapes ci-dessous pour les installer sur une machine Windows et connecter le Viewer à votre PADS4 Server.

## Avant de commencer

Assurez-vous que :

* Vous disposez de **droits d'administrateur local** sur la machine Windows cible.
* Vous connaissez l'**emplacement du PADS4 Server** (nom DNS ou adresse IP) et le **nom du Viewer** à utiliser.
* Vous installez **PADS4 version 2025.1** ou une version ultérieure.

## Étape 1 : Télécharger PADS4 Clients

Téléchargez le dernier programme d'installation depuis le lien officiel : [Téléchargement de PADS4 Clients](http://go.pads4.com/?fwlink=1011).

## Étape 2 : Démarrer l'installation

1. Faites un clic droit sur **PADS4 Clients.exe** et choisissez **Exécuter en tant qu'administrateur**.
2. Sélectionnez votre **langue** préférée et continuez.
3. Confirmez que **PADS4 version 2025.1** ou une version ultérieure est en cours d'installation.
4. Lisez le **Contrat de licence**, acceptez les conditions et sélectionnez **Suivant**.

## Étape 3 : Choisir les utilisateurs pour lesquels effectuer l'installation

Sélectionnez les utilisateurs Windows pour lesquels PADS4 Clients doit être installé, puis sélectionnez **Suivant**.

## Étape 4 : Installation Windows de PADS4

Sélectionnez la **configuration du Viewer** souhaitée dans le menu déroulant, puis sélectionnez **Suivant**.

## Étape 5 : Documentation d'aide (facultatif)

Choisissez si vous souhaitez installer le **fichier d'aide en ligne**. Une fois installé, les utilisateurs peuvent appuyer sur **F1** dans n'importe quelle application PADS4 pour ouvrir la rubrique d'aide correspondante.

<Tip>
  L'installation du fichier d'aide est recommandée — elle rend les conseils intégrés au produit disponibles hors ligne.
</Tip>

## Étape 6 : Configurer les paramètres du Viewer

Saisissez le **nom du Viewer** et l'**emplacement du PADS4 Server**, puis sélectionnez **Suivant**.

## Étape 7 : Mode Bureau et Contrôle de compte d'utilisateur (UAC)

Configurez la façon dont le Viewer s'exécute sur cette machine :

* **Mode Bureau** — choisissez si PADS4 Clients doit s'exécuter en mode Bureau.
* **Contrôle de compte d'utilisateur (UAC)** :
  * La désactivation de l'UAC permet à PADS4 Clients de s'exécuter avec des **privilèges d'administrateur** sans invites supplémentaires (recommandé pour les machines d'affichage dynamique dédiées).
  * Si l'utilisateur connecté n'est **pas** membre du groupe Administrateurs local, spécifiez un compte administrateur qui exécutera le Viewer.

## Étape 8 : Raccourcis et dossier d'installation

* Sélectionnez les **raccourcis** à créer.
* Choisissez le **dossier de destination de l'installation**.
* Sélectionnez **Installer** pour commencer.

## Étape 9 : Terminer l'installation

Attendez que le programme d'installation se termine. Lorsqu'il affiche **"PADS4 Clients a été installé avec succès"**, les applications Windows sont prêtes à être utilisées.
