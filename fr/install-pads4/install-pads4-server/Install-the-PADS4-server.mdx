## **Télécharger le programme d'installation PADS4**

Téléchargez le programme d'installation PADS4 depuis [ici](/fr/install-pads4/Downloads).​

**Vérifications préalables à l'installation**

* Avant d'installer le produit PADS4, vérifiez les prérequis en fonction de votre configuration. Si vous utilisez Workspace et/ou CMS, assurez-vous qu'Internet Information Services (IIS) est installé sur le serveur.
* Pour stocker les données Workspace et/ou CMS dans une base de données SQL Server, confirmez que SQL Server est installé sur le serveur.

<Note>
  Veuillez désactiver les antivirus avant de démarrer le processus d'installation ou de mise à jour. Il est possible que vous receviez un message « could not lock file » pendant la mise à jour en raison d'un antivirus.
</Note>

## Processus d'installation

Installez PADS4 en suivant ces étapes numérotées, en vous assurant de l'exécuter en tant qu'Administrateur.​

1. Faites un clic droit sur le fichier d'installation et sélectionnez « Exécuter en tant qu'administrateur ».
2. Modifiez le dossier racine d'installation si nécessaire (par défaut : C:\Program Files\PADS4).​
3. Lisez le document d'installation et le contrat de licence, puis cochez les cases.​
4. Cochez les cases pour confirmer et appuyez sur Installer pour commencer.​

Une fois l'installation de PADS4 terminée, l'application PADS4 Configuration s'ouvre automatiquement. Utilisez-la pour définir les ports d'hébergement, les paramètres du serveur et les options de base de données.

1. Définir les ports HTTP et HTTPS : Entrez le port HTTP et/ou HTTPS pour l'accès à PADS4.
2. Activer HSTS (Optionnel) : Activez pour rediriger tout le trafic HTTP vers HTTPS, en imposant un transport sécurisé.
3. Définir la version TLS minimale : Sélectionnez TLS 1.2 ou 1.3 (recommandé) dans le menu déroulant pour le chiffrement SSL.
4. Message Broker et Reverse Proxy (Optionnel) : Utilisez les valeurs par défaut (NDS Message Broker, PADS4 Reverse Proxy) sauf en cas de besoins personnalisés.
5. Mot de passe du compte Root : Entrez un mot de passe fort pour l'utilisateur su\root ; [requis pour la connexion au système](https://docs.pads4.com/ctrl-centre/account-creation).

Cliquez sur **Appliquer** pour enregistrer les paramètres et redémarrer automatiquement les services (environ 30 secondes).

Pour plus de détails sur PADS4 Configuration, consultez la [page PADS4 Configuration](/fr/install-pads4/install-pads4-server/PADS4-Configuration) dans la documentation.

Vous pouvez maintenant vous connecter à PADS4 sur `http(s)://localhost`, ce qui charge la page de connexion.

### Erreur lors de l'installation

Si une erreur survient lors de l'installation de PADS4, une fenêtre contextuelle apparaît avec un lien « cliquez ICI » vers le fichier journal généré. Ce journal enregistre chaque étape du processus, facilitant le diagnostic des problèmes. Partagez-le avec le support si les problèmes persistent.

<Warning>
  N'installez jamais PADS4 Foundation sur un poste de travail ou un PC Viewer destiné uniquement aux applications clientes telles que Designer, Scheduler ou Messenger. Cela ajouterait des fonctionnalités serveur, le reconfigurant en tant que machine serveur et perturbant potentiellement les opérations clientes. Après l'installation de Foundation, vous pouvez ajouter le logiciel client PADS4 sur ce PC classifié comme serveur.
</Warning>
