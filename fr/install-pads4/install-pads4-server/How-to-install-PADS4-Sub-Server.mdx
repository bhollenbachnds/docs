---
- title: Installer un PADS4 Sub Server
---

Le processus d'installation correspond à la configuration du PADS4 Server. Voir [Installer PADS4 Server](/fr/install-pads4/install-pads4-server/Install-the-PADS4-server) et exécutez le programme d'installation.

## Configuration d'un sous-serveur

Pour configurer un sous-serveur, effectuez les étapes suivantes dans PADS4 System Management sur le serveur principal :​

1. Ouvrez **PADS4 System Management** sur le serveur principal.
2. Dans le menu de gauche, sélectionnez **Servers**.​
3. Dans la barre de ruban, cliquez sur **Add Sub Server**.
4. Entrez un **nom d'affichage** pour le sous-serveur tel qu'il doit apparaître dans PADS4.
5. Vérifiez qu'une **licence valide** est attribué au Sub Server.

## Configuration des ports réseau

PADS4 utilise des ports réseau spécifiques pour sa communication interne et ses services. Référez-vous à la [**Configuration des ports réseau** ](/fr/network-requirements)page pour la liste complète des ports requis et des paramètres.

Pour plus de détails sur la façon dont les données sont synchronisées entre les Main, Sub et Backup Servers, consultez la [**Réplication avec les Sub et Backup Servers**](/fr/network-requirements) page.

## Exigences d'installation du serveur

<Warning>
  N'installez pas **PADS4 Foundation** sur un poste de travail ou un PC Viewer. Il doit fonctionner sur un **serveur dédié**.
</Warning>
