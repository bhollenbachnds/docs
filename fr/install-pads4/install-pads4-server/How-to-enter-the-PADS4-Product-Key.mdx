---
- title: Activation de PADS4 avec une clé de licence
---

Pour activer votre **produit PADS4**, suivez les étapes ci-dessous en utilisant la **clé de licence PADS4 fournie.**

## Étape 1 : Ouvrir PADS4 CTRL Center

* Après avoir installé **PADS4**, accédez au **Portail Web PADS4.**

<Tip>
  Si vous souhaitez (ré-)activer une installation PADS4 déjà en cours d'utilisation, l'utilisation de su\root n'est pas obligatoire ; tout utilisateur disposant de la licence **System Management + License Management** peut réactiver l'installation en accédant à **Licences** dans le menu de gauche.
</Tip>

* Connectez-vous avec l'utilisateur **su\root**.
* En haut de l'écran, choisissez le domaine "**pads**".

![Root Domain](/images/install-pads4/activating-pads4/root-domain.png)

* Maintenant, dans le menu de gauche, "**Licences**" apparaîtra.

<img src="/images/install-pads4/activating-pads4/root-licenses.png" alt="Root Licenses" title="Licences racine" style={{ width:"100%" }} />

* Cliquez sur licences et appuyez sur "**Activer"**

![Root License2](/images/install-pads4/activating-pads4/root-license2.png)

* Si une connexion Internet n'est pas disponible, une **activation hors ligne** méthode sera affichée.
  * Suivez les instructions affichées à l'écran.
  * Après avoir reçu le **Fichier de licence**, cliquez sur **"Télécharger le fichier de licence"** et sélectionnez le fichier.
  * Complétez le processus d'activation en suivant l'assistant.

Une fois l'activation terminée, votre **installation PADS4 est prête à l'emploi.**
