## **Activation de PADS4 avec une clé produit**

Pour activer votre **produit PADS4**, suivez les étapes ci-dessous en utilisant la **clé produit PADS4** fournie.

## Étape 1 : Ouvrir PADS4 CTRL Center

* Après avoir installé **PADS4**, naviguez vers le **portail web PADS4.**

<Tip>
  Si vous allez (ré)activer une installation PADS4 qui est déjà en cours d'utilisation, l'utilisation de su\root n'est pas obligatoire, tout utilisateur disposant de la licence **System Management + License Management** peut réactiver l'installation en allant dans \*\*Licenses\*\* dans le menu de gauche.
</Tip>

* Connectez-vous avec l'utilisateur **su\root**.
* En haut de l'écran, choisissez le domaine "**pads**".

![Root Domain](/images/install-pads4/activating-pads4/root-domain.png)

* Maintenant, dans le menu de gauche, "**Licenses**" apparaîtra.

<img src="/images/install-pads4/activating-pads4/root-licenses.png" alt="Root Licenses" title="Licences racines" style={{ width:"100%" }} />

* Cliquez sur licences et appuyez sur "**Activate"**

![Root License2](/images/install-pads4/activating-pads4/root-license2.png)

* Si une connexion Internet n'est pas disponible, une méthode d'**activation hors ligne** sera affichée.
  * Suivez les instructions fournies à l'écran.
  * Après avoir reçu le **fichier de licence**, cliquez sur **"Upload License File"** et sélectionnez le fichier.
  * Complétez le processus d'activation en suivant l'assistant.

Une fois l'activation terminée, votre **installation PADS4 est prête à être utilisée.**
