Ceci explique l'étape de vérification MFA lors de la connexion à PADS4. Une fois l'application d'authentification configurée, PADS4 demande à l'utilisateur de saisir le code d'authentification à 6 chiffres généré par l'application avant que l'accès ne soit accordé.

Cet écran apparaît lors de la connexion après que la MFA a été activée pour le compte utilisateur. Il s'agit de l'étape de vérification où l'utilisateur doit confirmer son identité avec le code temporel généré par l'application d'authentification.

## **Comment se connecter avec la MFA**

1. Connectez-vous à PADS4 avec le nom d'utilisateur et le mot de passe habituels.
2. Lorsque l'écran **Authentification à deux facteurs** apparaît, ouvrez l'application d'authentification sur l'appareil mobile.
3. Trouvez le code d'authentification à 6 chiffres actuel pour le compte PADS4 dans l'application.
4. Saisissez le code dans le champ **Code de vérification**&#x20;
   <Frame>
     ![Verification Code](/images/CTRLCenter/Config/MFA/Verification-code.png)
   </Frame>
5. Poursuivez le processus de connexion pour accéder à PADS4.

<Note>
  Le code d'authentification change régulièrement dans l'application d'authentification, il convient donc de saisir le code actuel avant qu'il n'expire. Si le code n'est pas accepté, l'utilisateur doit vérifier que le bon compte est sélectionné dans l'application, puis saisir le nouveau code à 6 chiffres.
</Note>
