- title: "Conditions "
---

<div style={{ width: "100%", overflowX: "auto" }}>
  | Option                           | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
  | :------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
  | Constante                        | Utiliser une constante statique.<br />Spécifier manuellement la **Valeur** à utiliser comme constante.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
  | Variable de visionneuse          | Les variables de visionneuse seront remplacées par la valeur réelle au moment de l'exécution.<br />Consultez la [page des variables de visionneuse](https://pads4documentation.blob.core.windows.net/designer/Content/Working%20with%20play%20lists/Viewer%20Variables.htm) pour plus d'informations sur l'utilisation des variables de visionneuse.                                                                                                                                                                                                                                                                                                                                                           |
  | Date système                     | Utiliser la date système du PC de lecture.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
  | Heure longue système             | Utiliser l'heure système du PC de lecture au format heure longue.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
  | Heure longue système (UTC)       | Utiliser l'heure système du PC de lecture au format heure longue et fuseau horaire UTC.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
  | Heure système                    | Utiliser l'heure système du PC de lecture.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
  | Heure système (UTC)              | Utiliser l'heure système du PC de lecture au format fuseau horaire UTC.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
  | Données actuelles                | Utiliser la colonne actuelle et la ligne actuelle du fournisseur de données actif.<br />Cette option s'applique uniquement aux éléments de page. Le fournisseur de données actif est déterminé par les [propriétés du fournisseur de données](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20provider%20properties%20\(element\).htm) de l'élément de page.                                                                                                                                                                                                                                                                              |
  | Données précédentes              | Utiliser la colonne actuelle et la ligne précédente du fournisseur de données actif.<br />Cette option s'applique uniquement aux éléments de page ayant activé le [défilement des données](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20scrolling%20properties%20\(element\).htm). Elle vous permet de comparer deux valeurs de données consécutives. Le fournisseur de données actif est déterminé par les [propriétés du fournisseur de données](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20provider%20properties%20\(element\).htm) de l'élément de page. |
  | Autre colonne                    | Utiliser la ligne actuelle, mais une colonne différente du fournisseur de données actif.<br />Cette option s'applique uniquement aux éléments de page. Le fournisseur de données actif est déterminé par les [propriétés du fournisseur de données](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20provider%20properties%20\(element\).htm) de l'élément de page. Sélectionner manuellement la **Colonne** à utiliser.                                                                                                                                                                                                                   |
  | Autres données                   | Utiliser un fournisseur de données, une colonne et une ligne spécifiques.<br />Sélectionner manuellement le **Fournisseur de données**, **Colonne** et **Ligne** à utiliser.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
  | Défilement d'autres données      | Utiliser un fournisseur de données et une colonne spécifiques, mais garder la ligne synchronisée avec la ligne de votre fournisseur de données actif.<br />Cette option s'applique uniquement aux éléments de page ayant activé le [défilement des données](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20scrolling%20properties%20\(element\).htm). Le fournisseur de données actif est déterminé par les [propriétés du fournisseur de données](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20provider%20properties%20\(element\).htm) de l'élément de page.   |
  | Nombre de lignes                 | Utiliser le nombre de lignes d'un fournisseur de données spécifique.<br />Sélectionner manuellement le **Fournisseur de données** à utiliser.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
  | Numéro de ligne actuel           | Utiliser le numéro de ligne actuel de votre fournisseur de données actif.<br />Cette option s'applique uniquement aux éléments de page. Le fournisseur de données actif est déterminé par les [propriétés du fournisseur de données](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20provider%20properties%20\(element\).htm) de l'élément de page.                                                                                                                                                                                                                                                                                       |
  | Nom de la page                   | Utiliser le nom de la page actuelle dans votre présentation.<br />Cette option s'applique uniquement aux pages et aux éléments de page.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
  | Numéro de page                   | Utiliser le numéro de séquence de la page actuelle dans votre présentation.<br />Cette option s'applique uniquement aux pages et aux éléments de page.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
  | Nom de la visionneuse            | Utiliser le nom unique de la visionneuse qui lit la présentation.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
  | Nom de l'ordinateur              | Utiliser le nom de l'ordinateur sur lequel la présentation est lue.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
  | Nom de la langue de l'ordinateur | Utiliser le nom de la langue Microsoft Windows du PC sur lequel la présentation est lue.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
</div>
