Un appareil Viewer également appelé Player est un matériel sur lequel les présentations d'affichage dynamique peuvent être affichées.

Cette section contient tous les appareils PADS4 Viewer certifiés disponibles (triés par ordre alphabétique).

Ces appareils sont testés avec chaque nouvelle version de PADS4.
