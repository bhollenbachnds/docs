## Qu'est-ce que PADS4 ?

PADS4 est une plateforme logicielle pour améliorer les expériences utilisateur dans un bâtiment numérique intelligent. Concevez le meilleur pour votre bâtiment intelligent grâce à une combinaison sur mesure des modules PADS4 comme Digital Signage, Workspace, FIDS et plus encore.

## Avantages de la plateforme PADS4

* ### Flexible

  Utilisez la puissance de PADS4 comme plateforme pour construire l'ensemble du flux de travail des utilisateurs et évitez les coûts inattendus provenant de problèmes d'interfaces multiples, d'intégrations et de configurations.
* ### Évolutif

  De la communication interne à l'espace de travail intelligent, de l'information sur les vols à la gestion des portes d'embarquement, de la publicité à l'orientation, PADS4 est une plateforme logicielle haut de gamme pour construire votre expérience de bâtiment numérique intelligent.
* ### Ouvert

  La plateforme PADS4 se compose de quatre modules différents vous permettant de moduler votre projet selon vos besoins. Réduisez les coûts de mise en œuvre et les courbes d'apprentissage en utilisant la même plateforme.

## Modules

<AccordionGroup>
  <Accordion title="PADS4 CTRL Centre">
    Portail web PADS4 pour gérer et maintenir votre système PADS4. En plus du web, nous disposons également d'applications Windows pour gérer votre système PADS4.
  </Accordion>

  <Accordion title="PADS4 Digital Signage">
    Logiciel dédié appelé Viewers qui est le point final sur lequel le contenu est affiché. La présentation est créée à l'aide d'un puissant Designer. Des serveurs secondaires et de sauvegarde peuvent être utilisés pour configurer des serveurs supplémentaires lorsque nécessaire, où PADS4 synchronisera automatiquement tout entre ces serveurs.
  </Accordion>

  <Accordion title="Applications client PADS4">
    Applications PADS4 basées sur Microsoft Windows qui peuvent être installées séparément sur les PC clients. Ces applications peuvent se connecter à distance à n'importe quel serveur PADS4.

    La plupart des fonctionnalités présentes dans l'application client peuvent également être disponibles dans le portail web PADS4 (\*nous travaillons actuellement à la migration de toutes les fonctionnalités vers le portail web).

    | Application             | Description                                                                      |
    | ----------------------- | -------------------------------------------------------------------------------- |
    | PADS4 Designer          | Application pour créer des designs de présentation à l'aide d'un éditeur WYSIWYG |
    | PADS4 Scheduler         | Application pour planifier des présentations pour les Viewers                    |
    | PADS4 Messenger         | Application pour planifier des messages pour les Desktop Viewers                 |
    | PADS4 Agent             | Application pour surveiller le système PADS4                                     |
    | PADS4 System Management | Application pour gérer le système PADS4                                          |
    | PADS4 Desktop Viewer    | Application qui affiche les messages planifiés                                   |
    | PADS4 Viewer            | Application qui affiche les présentations planifiées                             |
  </Accordion>

  <Accordion title="PADS4 FIDS">
    Signalisation aéroportuaire numérique tout-en-un. PADS4 FIDS Operator est un système de gestion d'informations de vol pour gérer vos vols quotidiens pour les petits et moyens aéroports. Les systèmes d'affichage d'informations de vol constituent le cœur de tout aéroport, opérant dans des fonctions cruciales tant pour les salles de contrôle que pour l'information des passagers.
  </Accordion>

  <Accordion title="PADS4 CMS">
    Créez des messages à l'aide d'une application web et affichez-les sur les Viewers du module PADS4 Digital Signage
  </Accordion>

  <Accordion title="PADS4 Workspace">
    Gestion des bureaux et des salles.
  </Accordion>

  <Accordion title="PADS4 Wayfinding">
    Présentation de cartes interactives et guidage des personnes. Les cartes peuvent être affichées sur des bornes ou d'autres appareils interactifs.
  </Accordion>

  <Accordion title="PADS4 VMS">
    Préenregistrement, surveillance et rapports des visiteurs dans votre bâtiment numérique intelligent.
  </Accordion>
</AccordionGroup>
