## Qu'est-ce que PADS4 ?

PADS4 est une plateforme logicielle qui améliore l'expérience utilisateur dans un bâtiment numérique intelligent en connectant les données, les personnes et les espaces. Vous concevez votre solution idéale en combinant des modules PADS4 tels que Digital Signage, Workspace, FIDS et d'autres pour répondre aux besoins spécifiques de votre bâtiment.

## Avantages de la plateforme PADS4

* **Flexible**\
  Utilisez la puissance de PADS4 en tant que plateforme unique pour concevoir des flux de travail utilisateur complets, en minimisant les coûts imprévus causés par de multiples interfaces, des intégrations séparées et des configurations complexes.
* **Évolutif**\
  De la communication interne aux espaces de travail intelligents, des informations de vol à la gestion des portes d'embarquement, de la publicité au wayfinding, PADS4 fournit une plateforme logicielle de haute qualité pour construire votre expérience de bâtiment numérique intelligent de bout en bout.
* **Ouvert**\
  La plateforme PADS4 est composée de plusieurs modules, vous permettant d'adapter chaque projet à ses exigences tout en travaillant dans un environnement cohérent. Cette approche modulaire contribue à réduire l'effort de mise en œuvre et le temps de formation, car les utilisateurs et les administrateurs travaillent avec la même plateforme pour différents cas d'usage.

## Modules

<AccordionGroup>
  <Accordion title="PADS4 CTRL Centre">
    Portail PADS4 basé sur le web pour gérer et maintenir votre système PADS4. En plus de la version web, nous disposons également d'applications Windows pour gérer votre système PADS4.
  </Accordion>

  <Accordion title="PADS4 Digital Signage">
    Un logiciel dédié appelé Viewers qui constitue le point de terminaison sur lequel le contenu est affiché. La présentation est créée à l'aide d'un puissant Designer. Des Sub et Backup Servers peuvent être utilisés pour configurer des Servers supplémentaires si nécessaire, PADS4 synchronisant automatiquement tout entre ces serveurs.
  </Accordion>

  <Accordion title="Applications client PADS4">
    Applications PADS4 basées sur Microsoft Windows pouvant être installées séparément sur des PC clients. Ces applications peuvent se connecter à distance à n'importe quel PADS4 Server.

    La plupart des fonctionnalités présentes dans l'application client peuvent également être disponibles dans le portail web PADS4 (\*nous travaillons actuellement à la migration de toutes les fonctionnalités vers le portail web).

    | Application             | Description                                                                      |
    | ----------------------- | -------------------------------------------------------------------------------- |
    | PADS4 Designer          | Application pour créer des designs de présentation à l'aide d'un éditeur WYSIWYG |
    | PADS4 Scheduler         | Application pour planifier des présentations pour les Viewers                    |
    | PADS4 Messenger         | Application pour planifier des messages vers les Desktop Viewers                 |
    | PADS4 Agent             | Application pour surveiller le système PADS4                                     |
    | PADS4 System Management | Application pour gérer le système PADS4                                          |
    | PADS4 Desktop Viewer    | Application qui affiche les messages planifiés                                   |
    | PADS4 Viewer            | Application qui affiche les présentations planifiées                             |
  </Accordion>

  <Accordion title="PADS4 FIDS">
    Signalétique numérique aéroportuaire tout inclus. PADS4 FIDS Operator est un système de gestion des informations de vol pour gérer vos vols quotidiens dans les aéroports de petite et moyenne taille. Les systèmes d'affichage des informations de vol constituent le cœur de tout aéroport, remplissant des fonctions essentielles aussi bien pour les salles de contrôle que pour l'information des passagers.
  </Accordion>

  <Accordion title="PADS4 CMS">
    Créez des messages à l'aide d'une application web et affichez-les sur les Viewers du module PADS4 Digital Signage
  </Accordion>

  <Accordion title="PADS4 Workspace">
    Gestion des bureaux et des salles.
  </Accordion>
</AccordionGroup>
