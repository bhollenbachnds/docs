L'onglet **Rooms** fournit une vue d'ensemble de toutes les salles qui ont été ajoutées au **PADS4 Workspace Module**. Depuis cet onglet, vous pouvez ajouter, supprimer et modifier des salles selon vos besoins.

## Ajouter une salle

Pour ajouter une salle au module PADS4 Workspace, vous pouvez choisir parmi deux options :

### Import Rooms

Cette option interroge le calendrier connecté (par exemple, Microsoft Exchange) pour récupérer les informations sur les ressources de salle. Une fois la requête terminée, une liste des salles disponibles s'affiche, vous permettant de sélectionner les salles à ajouter au système.

### Création manuelle de salle

Vous pouvez également créer manuellement une salle qui ne fait pas partie du calendrier connecté en sélectionnant l'option de création manuelle.

## Détails de la salle

Lors de la création d'une salle, vous devrez saisir les informations suivantes sous différents onglets :

### Général

* Nom de la salle
* Description
* Type de salle (profil de salle à utiliser)
* Adresse e-mail de la salle
* Capacité (occupation maximale)
* Image (utilisée dans l'interface de réservation)
* Emplacement
* Fuseau horaire
* Inactif (cochez cette case si la salle ne doit pas être réservable)
* Définir une capacité réduite (autoriser les réservations chevauchantes pour l'espace)

### Équipement

* Attribuer l'équipement disponible pour cette salle (par exemple, projecteur, écran, système de sonorisation)

### Service supplémentaire

* Ajouter des services tels que la restauration, le nettoyage ou d'autres options d'assistance disponibles avec la salle

### Approbation

* Indiquer si les réservations pour cette salle nécessitent une approbation

### Propriété étendue

* Définir des propriétés supplémentaires pour la salle, telles que l'accessibilité en fauteuil roulant ou des étiquettes d'utilisation spéciale

### Licence

* Configurer si la salle apparaît dans le Outlook Add-in ou génère un QR Code pour l'accès
