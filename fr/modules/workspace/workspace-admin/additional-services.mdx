L'onglet **Services supplémentaires** permet aux administrateurs d'ajouter et de gérer des services supplémentaires — tels que la restauration, la livraison de matériel ou la préparation de salle — dans le **PADS4 Workspace Module**.

![Additional Services Screen](/mnt/data/image.png)

## Ajout de services supplémentaires

Pour ajouter un nouveau service, sélectionnez le bouton **Ajouter un service supplémentaire**.

### Informations requises

Lors de la création d'un service, renseignez les champs suivants :

* **Nom**\
  Le nom du service (par ex. Restauration, Support informatique).

* **Image**\
  Une image facultative pour aider les utilisateurs à identifier visuellement le service dans l'interface.

* **Type**\
  Choisissez entre :
  * **Par défaut** – Pour les salles et les espaces de réunion.
  * **Zones communes** – Pour les services fournis dans des zones telles que les cantines ou les espaces partagés.

* **Description**\
  Une brève explication du service.

* **Autoriser les commentaires**\
  Activez cette option si l'utilisateur doit pouvoir ajouter un message lors d'une demande (par ex., des notes alimentaires pour la restauration).

* **E-mail de destination**\
  L'adresse e-mail à laquelle les demandes de service seront envoyées.

### Liste des services supplémentaires

Cette section vous permet de spécifier des éléments de service individuels (par ex., des options de menu ou des formules de configuration). Pour chaque élément, vous pouvez définir :

* **Nom**
* **Informations/Description**
* **Temps de préparation**
* **Prix**
* **Autoriser les commentaires**
