L'onglet **Additional Services** permet aux administrateurs d'ajouter et de gérer des services supplémentaires—tels que la restauration, la livraison de matériel ou la préparation de salle—au sein du **PADS4 Workspace Module**.

## Ajout de services supplémentaires

Pour ajouter un nouveau service, sélectionnez le bouton **Add Additional Service**.

### Informations requises

Lors de la création d'un service, renseignez les champs suivants :

* **Nom**\
  Le nom du service (ex. : Restauration, Support informatique).

* **Image**\
  Une image facultative pour aider les utilisateurs à identifier visuellement le service dans l'interface.

* **Type**\
  Choisissez entre :
  * **Default** – Pour les salles et les espaces de réunion.
  * **Common Areas** – Pour les services fournis dans des zones telles que les cantines ou les espaces partagés.

* **Description**\
  Une brève explication du service.

* **Allow Comment**\
  Activez cette option si l'utilisateur doit pouvoir ajouter un message lors d'une demande (ex. : remarques alimentaires pour la restauration).

* **E-mail de destination**\
  L'adresse e-mail à laquelle les demandes de service seront envoyées.

### Liste des services supplémentaires

Cette section vous permet de spécifier des éléments de service individuels (ex. : options de menu ou forfaits de configuration). Pour chaque élément, vous pouvez définir :

* **Nom**
* **Informations/Description**
* **Temps de préparation**
* **Prix**
* **Allow Comment**
