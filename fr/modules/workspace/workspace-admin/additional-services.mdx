Les **Services Additionnels** l'onglet permet aux administrateurs d'ajouter et de gérer des services supplémentaires—tels que la restauration, la livraison d'équipement ou l'aménagement de salle—dans le **PADS4 Workspace Module**.

![Additional Services Screen](/mnt/data/image.png)

## Ajout de services supplémentaires

Pour ajouter un nouveau service, sélectionnez le bouton **Add Additional Service**.

### Informations requises

Lors de la création d'un service, remplissez les champs suivants :

* **Name**\
  Le nom du service (par exemple Restauration, Support informatique).

* **Image**\
  Une image facultative pour aider les utilisateurs à identifier visuellement le service dans l'interface.

* **Type**\
  Choisissez entre :
  * **Default** – Pour les salles et espaces de réunion.
  * **Common Areas** – Pour les services fournis dans des zones comme les cantines ou les espaces partagés.

* **Description**\
  Une brève explication du service.

* **Allow Comment**\
  Activez cette option si l'utilisateur doit pouvoir ajouter un message lors d'une demande (par exemple, des notes diététiques pour la restauration).

* **Destination Email**\
  L'adresse e-mail où les demandes de service seront envoyées.

### Additional Service List

Cette section vous permet de spécifier des éléments de service individuels (par exemple, options de menu ou forfaits d'installation). Pour chaque élément, vous pouvez définir :

* **Name**
* **Information/Description**
* **Preparation Time**
* **Price**
* **Allow Comment**
