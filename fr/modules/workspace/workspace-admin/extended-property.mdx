L'onglet **Extended Property** permet aux administrateurs de créer des attributs personnalisés pour les salles et les espaces, tels que **l'accessibilité en fauteuil roulant**, **les types de sièges**, ou d'autres caractéristiques spécialisées de la salle.

![Extended Property Configuration Screen](/mnt/data/image.png)

## Ajout d'une Extended Property

Pour ajouter une nouvelle propriété, cliquez sur le bouton **Add Extended Property**.

### Champs obligatoires

Lors de la création d'une propriété, fournissez les informations suivantes :

* **Nom**\
  Le nom de la propriété étendue (par ex., Accessible en fauteuil roulant, Disposition en U).

* **Type**\
  Définit où la propriété s'applique :
  * **Default** – Pour les salles et les espaces de travail.
  * **Common Areas** – Pour les restaurants, les halls d'entrée ou autres espaces partagés.

* **Valeur**\
  La valeur ou caractéristique spécifique de la propriété (par ex., "Oui" pour l'accessibilité, "10 chaises" pour les sièges).

* **Ajouter à tous les profils de salle**\
  Activez cette option si la propriété doit être disponible dans tous les profils de salle du système.
