L'onglet **Propriété étendue** permet aux administrateurs de créer des attributs personnalisés pour les salles et les espaces, tels que **l'accessibilité aux fauteuils roulants**, **les types de sièges**, ou d'autres caractéristiques spécialisées des salles.

![Extended Property Configuration Screen](/mnt/data/image.png)

## Ajout d'une propriété étendue

Pour ajouter une nouvelle propriété, cliquez sur le bouton **Ajouter une propriété étendue**.

### Champs obligatoires

Lors de la création d'une propriété, fournissez les informations suivantes :

* **Nom**\
  Le nom de la propriété étendue (par exemple, Accessible aux fauteuils roulants, Disposition en U).

* **Type**\
  Définit où la propriété s'applique :
  * **Par défaut** – Pour les salles et les espaces de travail.
  * **Espaces communs** – Pour les restaurants, halls ou autres espaces partagés.

* **Valeur**\
  La valeur ou caractéristique spécifique de la propriété (par exemple, "Oui" pour l'accessibilité, "10 chaises" pour les sièges).

* **Ajouter à tous les profils de salle**\
  Activez cette option si la propriété doit être disponible pour tous les profils de salle dans le système.
