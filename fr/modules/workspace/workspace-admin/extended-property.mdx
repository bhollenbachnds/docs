L'**Propriété Étendue** onglet permet aux administrateurs de créer des attributs personnalisés pour les salles et les espaces, tels que **l'accessibilité en fauteuil roulant**, **les types de sièges**, ou d'autres caractéristiques spécialisées de la salle.

![Extended Property Configuration Screen](/mnt/data/image.png)

## Ajouter une Propriété Étendue

Pour ajouter une nouvelle propriété, cliquez sur le **Ajouter une Propriété Étendue** bouton.

### Champs Obligatoires

Lors de la création d'une propriété, fournissez les informations suivantes :

* **Nom**\
  Le nom de la propriété étendue (ex. : Accessible en Fauteuil Roulant, Sièges en U).

* **Type**\
  Définit où la propriété s'applique :
  * **Par Défaut** – Pour les salles et les espaces de travail.
  * **Espaces Communs** – Pour les restaurants, les halls ou autres espaces partagés.

* **Valeur**\
  La valeur ou caractéristique spécifique de la propriété (ex. : "Oui" pour l'accessibilité, "10 chaises" pour les sièges).

* **Ajouter à Tous les Profils de Salle**\
  Activez cette option si la propriété doit être disponible pour tous les profils de salle dans le système.
