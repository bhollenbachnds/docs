Ce manuel fournit des conseils techniques pour le **PADS4 Workspace** et **CMS Admin Interface**. Il est destiné à être utilisé par le personnel technique responsable de la **configuration de base**, **configuration**, et du **dépannage** des modules PADS4 Workspace et CMS.

## Accès à l'interface d'administration

Pour accéder à l'interface d'administration :

1. Ouvrez un navigateur web.

2. Entrez l'adresse suivante :

   ```
   http://serverip:port
   ```

3. Les identifiants de connexion ont été fournis à l'équipe technique.

> Dans la plupart des cas, les administrateurs peuvent utiliser **Single Sign-On (SSO)** pour se connecter en utilisant leurs identifiants Windows.

![PADS4 Login Screen](/mnt/data/image.png)

Une fois connecté, en fonction de vos droits d'accès, vous serez invité à sélectionner le module approprié à gérer.

![PADS4 Module Selection Screen](/mnt/data/image.png)
