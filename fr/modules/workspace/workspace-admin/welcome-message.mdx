## Mise à jour de la page d'accueil

Pour personnaliser la page d'accueil :

1. Sélectionnez **Ajouter une image** pour télécharger ou choisir un nouveau logo.
2. Saisissez un **message de bienvenue** personnalisé dans le champ prévu à cet effet.
3. Une fois toutes les modifications effectuées, cliquez sur **Enregistrer** pour les appliquer.

Ces mises à jour seront reflétées sur l'écran d'accueil destiné aux utilisateurs de PADS4 Workspace.
