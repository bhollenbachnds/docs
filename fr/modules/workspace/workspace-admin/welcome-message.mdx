L'onglet **Welcome Message** permet aux administrateurs de personnaliser la page d'accueil en mettant à jour le logo et le message de bienvenue affichés aux utilisateurs.

![Home Page Setup Screen](/mnt/data/image.png)

## Mise à jour de la page d'accueil

Pour personnaliser la page d'accueil :

1. Sélectionnez **Add an Image** pour télécharger ou choisir un nouveau logo.
2. Saisissez un **welcome message** personnalisé dans le champ prévu.
3. Une fois toutes les modifications effectuées, cliquez sur **Save** pour les appliquer.

Ces mises à jour seront reflétées sur l'écran d'accueil destiné aux utilisateurs de PADS4 Workspace.
