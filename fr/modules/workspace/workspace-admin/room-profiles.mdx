L'onglet **Room Profiles** vous permet de définir des règles opérationnelles pour les salles. Chaque profil définit des paramètres de comportement qui peuvent être réutilisés dans plusieurs salles, simplifiant la configuration et garantissant la cohérence.

## Créer un profil de salle

Pour créer un nouveau profil de salle, cliquez sur le bouton **Create Room Profile**.

### Onglet General

Dans l'onglet General, renseignez les informations suivantes :

* **Name**\
  Nom du profil de salle.

* **Description**\
  Une brève explication de l'objectif du profil.

* **Use External Booking System**\
  Active l'intégration avec des systèmes externes tels que Microsoft Outlook.

* **Is a Sharing Desk Profile**\
  Utilisez cette option pour les profils destinés aux bureaux partagés.

* **Is Default Assigned to a User**\
  Indique si ce bureau est normalement attribué à un utilisateur spécifique.

* **Inactive**\
  Marque le profil comme non réservable (par exemple, temporairement indisponible).

* **Limit Capacity**\
  Utilisé dans les scénarios de bureau partagé pour limiter le nombre de places disponibles.

* **Enable Cleanup**\
  Ajoute un délai tampon de nettoyage après une réservation. Pendant cette période, la salle ne peut pas être réservée.

* **App Booking**\
  Permet les réservations via l'application web PADS4 ou les panneaux de salle physiques.

* **Sensor Booking**\
  Active la réservation basée sur des capteurs d'occupation (par ex., réservation automatique lorsqu'une présence est détectée).

### Onglets supplémentaires

* **Service supplémentaire**\
  Attribuez des services supplémentaires tels que la restauration ou l'installation audiovisuelle aux salles utilisant ce profil.

* **Approbation**\
  Exiger l'approbation d'un utilisateur désigné avant qu'une réservation soit finalisée.

* **Propriété étendue**\
  Ajoutez des caractéristiques personnalisées à la salle, comme l'accessibilité en fauteuil roulant ou des notes sur l'équipement.

* **Licence**\
  Activez cette salle pour qu'elle apparaisse dans le complément Outlook ou attribuez-lui un QR Code pour l'accès des utilisateurs.
