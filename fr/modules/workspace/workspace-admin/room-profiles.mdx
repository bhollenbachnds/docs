L'onglet **Room Profiles** vous permet de définir des règles opérationnelles pour les salles. Chaque profil établit des paramètres de comportement qui peuvent être réutilisés dans plusieurs salles, simplifiant la configuration et assurant la cohérence.

![Room Profiles Screen](/mnt/data/image.png)

## Création d'un profil de salle

Pour créer un nouveau profil de salle, cliquez sur le bouton **Create Room Profile**.

### Onglet Général

Dans l'onglet Général, remplissez les détails suivants :

* **Name**\
  Nom du profil de salle.

* **Description**\
  Une brève explication de l'objectif du profil.

* **Use External Booking System**\
  Permet l'intégration avec des systèmes externes tels que Microsoft Outlook.

* **Is a Sharing Desk Profile**\
  Utilisez cette option pour les profils destinés aux bureaux partagés.

* **Is Default Assigned to a User**\
  Indique si ce bureau est normalement attribué à un utilisateur spécifique.

* **Inactive**\
  Marque le profil comme non réservable (par exemple, temporairement indisponible).

* **Limit Capacity**\
  Utilisé dans les scénarios de bureau partagé pour limiter le nombre de places disponibles.

* **Enable Cleanup**\
  Ajoute un temps tampon de nettoyage après une réservation. Pendant cette période, la salle ne peut pas être réservée.

* **App Booking**\
  Permet les réservations via l'application web PADS4 ou les panneaux physiques des salles.

* **Sensor Booking**\
  Active la réservation basée sur des capteurs d'occupation (par exemple, réservation automatique lorsqu'une présence est détectée).

### Onglets supplémentaires

* **Additional Service**\
  Attribuez des services supplémentaires tels que la restauration ou la configuration audiovisuelle aux salles utilisant ce profil.

* **Approval**\
  Exiger l'approbation d'un utilisateur désigné avant qu'une réservation ne soit finalisée.

* **Extended Property**\
  Ajoutez des caractéristiques personnalisées comme l'accessibilité aux fauteuils roulants ou des notes sur l'équipement.

* **License**\
  Permettez à cette salle d'apparaître dans le module complémentaire Outlook ou attribuez-lui un code QR pour l'accès des utilisateurs.
