L'onglet **Capteurs d'occupation** permet aux administrateurs d'afficher et de gérer toutes les passerelles, tous les capteurs et tous les dispositifs LED connectés utilisés dans le **module PADS4 Workspace**.

## Gateways

Affiche tous les **passerelles de capteurs** ainsi que leur statut de connexion et les détails d'activité.

## Sensors

Affiche tous les **capteurs d'occupation** et permet aux administrateurs de les affecter à des salles spécifiques dans le système.

## LEDs

Répertorie tous les **appareils LED** gérés par le PADS4 Workspace Module, généralement utilisés pour signaler le statut ou la disponibilité d'une salle.
