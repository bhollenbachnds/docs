L'onglet **Occupancy Sensors** permet aux administrateurs de visualiser et de gérer toutes les passerelles connectées, les capteurs et les appareils LED utilisés dans le **PADS4 Workspace Module**.

![Occupancy Sensors Overview](/mnt/data/image.png)

## Passerelles

Affiche toutes les **passerelles de capteurs** connectées ainsi que leur état de connexion et les détails d'activité.

## Capteurs

Affiche tous les **capteurs d'occupation** connectés et permet aux administrateurs de les attribuer à des salles spécifiques dans le système.

## LEDs

Liste tous les **appareils LED** gérés par le PADS4 Workspace Module, généralement utilisés pour signaler l'état ou la disponibilité d'une salle.
