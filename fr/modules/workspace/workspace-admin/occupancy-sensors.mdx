L'onglet **Occupancy Sensors** permet aux administrateurs de visualiser et de gérer toutes les passerelles connectées, les capteurs et les dispositifs LED utilisés dans le **PADS4 Workspace Module**.

## Gateways

Affiche toutes les **sensor gateways** connectées ainsi que leur état de connexion et les détails d'activité.

## Sensors

Montre tous les **occupancy sensors** connectés et permet aux administrateurs de les assigner à des salles spécifiques dans le système.

## LEDs

Liste tous les **LED devices** connectés gérés par le PADS4 Workspace Module, généralement utilisés pour signaler l'état ou la disponibilité d'une salle.
