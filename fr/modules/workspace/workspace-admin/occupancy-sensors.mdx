Le **Occupancy Sensors** permet aux administrateurs de visualiser et de gérer toutes les passerelles, capteurs et appareils LED connectés utilisés dans le **PADS4 Workspace Module**.

![Occupancy Sensors Overview](/mnt/data/image.png)

## Gateways

Affiche toutes les **passerelles de capteurs** ainsi que leur statut de connexion et les détails d'activité.

## Sensors

Affiche tous les **capteurs de présence** connectés et permet aux administrateurs de les affecter à des salles spécifiques dans le système.

## LEDs

Répertorie tous les **appareils LED** gérés par le PADS4 Workspace Module, généralement utilisés pour signaler le statut ou la disponibilité d'une salle.
