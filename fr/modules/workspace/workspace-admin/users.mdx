L'onglet **Room Users** permet aux administrateurs de créer ou d'importer des utilisateurs qui auront accès à l'**PADS4 Workspace User Interface**.

## Ajout d'un Room User

Pour ajouter un nouvel utilisateur, cliquez sur le bouton **Add Room User**.

### Création manuelle d'un utilisateur

Pour ajouter manuellement un utilisateur, sélectionnez **Add a Room User** et remplissez les champs requis.

### Informations générales

* **Login**\
  Le nom d'utilisateur du compte.

* **Mot de passe / Confirmer le mot de passe**\
  Requis pour sécuriser le compte.

* **Profile**\
  Attribuez l'utilisateur à un Room User Profile spécifique.

* **Email**\
  Adresse e-mail de l'utilisateur.

* **Nom / Prénom**\
  Le nom de famille et le prénom de l'utilisateur.

* **NFC Number**\
  Le cas échéant, saisissez l'identifiant NFC de l'utilisateur (utilisé pour les enregistrements en salle via NFC).

* **Télétravail**\
  Activez cette option si l'utilisateur travaille principalement depuis chez lui.

* **Empêcher la réservation de salle**\
  Si cette option est sélectionnée, l'utilisateur pourra consulter les salles mais pas les réserver.

### Accès aux salles

Spécifiez les salles que cet utilisateur peut **consulter** et/ou **réserver** dans l'interface PADS4 Workspace.

### Délégation

Désignez les utilisateurs qui doivent approuver les réservations de salle effectuées par cet utilisateur, si nécessaire.
