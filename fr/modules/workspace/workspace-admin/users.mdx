L'onglet **Room Users** permet aux administrateurs de créer ou d'importer des utilisateurs qui auront accès à **PADS4 Workspace User Interface**.

![Room Users Management Screen](/mnt/data/image.png)

## Ajout d'un utilisateur de salle

Pour ajouter un nouvel utilisateur, cliquez sur le bouton **Add Room User**.

### Création manuelle d'utilisateur

Pour ajouter manuellement un utilisateur, sélectionnez **Add a Room User** et remplissez les champs requis.

### Informations générales

* **Login**\
  Le nom d'utilisateur pour le compte.

* **Password / Confirm Password**\
  Requis pour sécuriser le compte.

* **Profile**\
  Attribuer l'utilisateur à un profil d'utilisateur de salle spécifique.

* **Email**\
  Adresse e-mail de l'utilisateur.

* **Name / First Name**\
  Le nom de famille et le prénom de l'utilisateur.

* **NFC Number**\
  Le cas échéant, saisissez l'identifiant NFC de l'utilisateur (utilisé pour les enregistrements de salle via NFC).

* **Teleworking**\
  Activez cette option si l'utilisateur travaille principalement depuis son domicile.

* **Prevent Room Booking**\
  Si cette option est sélectionnée, l'utilisateur pourra voir les salles mais ne pourra pas les réserver.

### Accès aux salles

Spécifiez quelles salles cet utilisateur peut **voir** et/ou **réserver** dans l'interface PADS4 Workspace.

### Délégation

Attribuez des utilisateurs qui doivent approuver les réservations de salle effectuées par cet utilisateur, si nécessaire.
