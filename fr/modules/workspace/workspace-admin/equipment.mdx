L'onglet **Equipment** permet aux administrateurs de définir des équipements pouvant être associés à des salles dans le **PADS4 Workspace Module**. Cela aide les utilisateurs finaux à comprendre quels outils ou appareils sont disponibles dans chaque espace.

![Equipment Management Screen](/mnt/data/image.png)

## Ajout d'équipement

Pour ajouter un nouvel équipement :

1. Cliquez sur le bouton **Add Equipment**.

2. Dans la fenêtre **Create Equipment**, renseignez les détails suivants :

   * **Nom**\
     Entrez le nom de l'équipement (par ex., Projecteur, Tableau blanc, Système de vidéoconférence).

   * **Image**\
     Téléchargez une image qui représentera visuellement l'équipement dans l'interface utilisateur.

![Create Equipment Screen](/mnt/data/image.png)

3. Cliquez sur **Save** pour terminer le processus.

Une fois ajouté, l'équipement peut être assigné à des salles individuelles via le panneau de configuration des salles.
