L'onglet **Equipment** permet aux administrateurs de définir des équipements pouvant être associés à des salles dans le **PADS4 Workspace Module**. Cela aide les utilisateurs finaux à comprendre quels outils ou appareils sont disponibles dans chaque espace.

## Ajout d'équipements

Pour ajouter un nouvel équipement :

1. Cliquez sur le bouton **Add Equipment**.

2. Dans la fenêtre **Create Equipment**, renseignez les informations suivantes :

   * **Nom**\
     Saisissez le nom de l'équipement (par exemple, Projecteur, Tableau blanc, Système de vidéoconférence).

   * **Image**\
     Téléchargez une image qui représentera visuellement l'équipement dans l'interface utilisateur.

{/* TODO: missing image — original placeholder referenced /mnt/data/image.png with alt: "Create Equipment Screen" */}

3. Cliquez sur **Enregistrer** pour finaliser le processus.

Une fois ajouté, l'équipement peut être attribué à des salles individuelles via le panneau de configuration des salles.
