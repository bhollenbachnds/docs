L'onglet **Room User Profiles** permet aux administrateurs de créer et de gérer des profils d'utilisateurs qui définissent les niveaux d'accès et les permissions dans le système.

![Room User Profiles Screen](/mnt/data/image.png)

## Ajout d'un Room User Profile

Pour créer un nouveau Room User Profile, cliquez sur le bouton **Add Room User Profile**.

### Informations générales

Lors de la création d'un nouveau profil, fournissez les détails suivants :

* **Name**\
  Le nom du profil.

* **Description**\
  Une brève description de l'objectif du profil ou du groupe d'utilisateurs visé.

* **Teleworking**\
  Activez cette option si les utilisateurs assignés à ce profil travaillent principalement depuis leur domicile.

* **Prevent Room Booking**\
  Activez cette option pour permettre aux utilisateurs de voir les salles mais pas de les réserver.

![Create Room User Profile Form](/mnt/data/image.png)

### Onglet Rooms

Spécifiez quelles salles les utilisateurs de ce profil sont autorisés à **voir** et/ou à **réserver**.

## Gestion des Room User Profiles

* **Modifier un profil**\
  Pour apporter des modifications à un profil existant, sélectionnez l'option **edit** et mettez à jour les champs nécessaires.

* **Supprimer un profil**\
  Pour supprimer un profil, sélectionnez l'option **edit** et choisissez **delete**.
