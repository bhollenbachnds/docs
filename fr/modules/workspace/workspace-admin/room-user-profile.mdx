L'onglet **Room User Profiles** permet aux administrateurs de créer et de gérer des profils d'utilisateurs qui définissent les niveaux d'accès et les permissions dans le système.

## Ajout d'un profil d'utilisateur de salle

Pour créer un nouveau profil d'utilisateur de salle, cliquez sur le bouton **Add Room User Profile**.

### Informations générales

Lors de la création d'un nouveau profil, fournissez les détails suivants :

* **Nom**\
  Le nom du profil.

* **Description**\
  Une brève description de l'objectif du profil ou du groupe d'utilisateurs visé.

* **Télétravail**\
  Activez cette option si les utilisateurs assignés à ce profil travaillent principalement depuis leur domicile.

* **Empêcher la réservation de salles**\
  Activez cette option pour permettre aux utilisateurs de voir les salles mais pas de les réserver.

### Onglet Salles

Spécifiez quelles salles les utilisateurs de ce profil sont autorisés à **voir** et/ou **réserver**.

## Gestion des profils d'utilisateurs de salles

* **Modifier un profil**\
  Pour apporter des modifications à un profil existant, sélectionnez l'option **modifier** et mettez à jour les champs nécessaires.

* **Supprimer un profil**\
  Pour supprimer un profil, sélectionnez l'option **modifier** et choisissez **supprimer**.
