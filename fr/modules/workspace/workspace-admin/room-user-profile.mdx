L'onglet **Room User Profiles** permet aux administrateurs de créer et de gérer des profils utilisateur qui définissent les niveaux d'accès et les autorisations au sein du système.

![Room User Profiles Screen](/mnt/data/image.png)

## Ajout d'un Room User Profile

Pour créer un nouveau Room User Profile, cliquez sur le bouton **Add Room User Profile**.

### Informations Générales

Lors de la création d'un nouveau profil, fournissez les informations suivantes :

* **Nom**\
  Le nom du profil.

* **Description**\
  Une brève description de l'objectif du profil ou du groupe d'utilisateurs concerné.

* **Télétravail**\
  Activez cette option si les utilisateurs assignés à ce profil travaillent principalement depuis leur domicile.

* **Empêcher la Réservation de Salle**\
  Activez cette option pour permettre aux utilisateurs de consulter les salles sans pouvoir les réserver.

![Create Room User Profile Form](/mnt/data/image.png)

### Onglet Salles

Spécifiez les salles que les utilisateurs de ce profil sont autorisés à **consulter** et/ou **réserver**.

## Gestion des Room User Profiles

* **Modifier un Profil**\
  Pour apporter des modifications à un profil existant, sélectionnez l'option **modifier** et mettez à jour les champs nécessaires.

* **Supprimer un Profil**\
  Pour supprimer un profil, sélectionnez l'option **modifier** et choisissez **supprimer**.
