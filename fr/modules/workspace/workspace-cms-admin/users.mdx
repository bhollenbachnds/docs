L'onglet **Utilisateurs** vous permet d'ajouter et de gérer des comptes administrateur au sein du système. Les administrateurs peuvent être ajoutés manuellement ou en les important depuis **Active Directory**, selon la configuration de votre système.

Pour ajouter un utilisateur via Active Directory, veuillez vous référer à la section **Active Directory** sous **Plugins**.

## Modification des comptes administrateur

En sélectionnant l'icône **Modifier** à côté d'un utilisateur, vous pouvez accéder à plusieurs options de gestion :

{/* TODO: missing image — original placeholder referenced /mnt/data/image.png with alt: "Edit Options for User" */}

* **Changer le mot de passe** : Vous permet de réinitialiser le mot de passe d'un compte administrateur — applicable uniquement si l'utilisateur a été ajouté manuellement.
  > *Note: This option is not available for Active Directory users.*

* **Modifier** : Vous permet de mettre à jour le nom d'utilisateur et les informations de contact du compte administrateur.

* **Approbation** : Attribue à ce compte le statut de compte d'**Approbation**, lui permettant d'être utilisé dans les flux de travail d'approbation.
