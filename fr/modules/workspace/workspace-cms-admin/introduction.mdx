# Accès à l'espace de travail PADS4 et à l'interface d'administration CMS

Ce manuel fournit des informations sur l'accès et la navigation dans l'espace de travail PADS4 et l'interface d'administration CMS. Il est destiné à l'équipe technique et couvre la configuration de base et le dépannage des modules PADS4 Workspace et CMS.

## Accès à l'interface d'administration

Pour ouvrir l'interface d'administration :

1. Ouvrez un navigateur web.

2. Saisissez l'adresse suivante dans la barre d'adresse :

   ```
   http://serverip:port
   ```

3. Les identifiants de connexion ont été fournis à l'équipe technique.

> **Note:** Dans la plupart des cas, les administrateurs peuvent utiliser l'option **Single Sign-On (SSO)** pour se connecter en utilisant leurs identifiants Windows.

![PADS4 Login Screen](/mnt/data/image.png)

## Navigation après connexion

Une fois connecté, et en fonction de vos droits d'accès, vous verrez une sélection de modules ou de zones à gérer. Choisissez le module souhaité pour continuer.

![PADS4 Module Selection Screen](/mnt/data/image.png)
