# Accéder à l'interface PADS4 Workspace & CMS Admin

Ce manuel fournit des informations sur l'accès et la navigation dans l'interface PADS4 Workspace et CMS Admin. Il est destiné à l'équipe technique et couvre la configuration de base et le dépannage des modules PADS4 Workspace et CMS.

## Accéder à l'interface Admin

Pour ouvrir l'interface Admin :

1. Ouvrez un navigateur web.

2. Saisissez l'adresse suivante dans la barre d'adresse :

   ```
   http://serverip:port
   ```

3. Les identifiants de connexion ont été fournis à l'équipe technique.

> **Note:** Dans la plupart des cas, les administrateurs peuvent utiliser l'option **Single Sign-On (SSO)** pour se connecter avec leurs identifiants Windows.

![PADS4 Login Screen](/mnt/data/image.png)

## Navigation après la connexion

Une fois connecté, et en fonction de vos droits d'accès, une sélection de modules ou de zones à gérer vous sera présentée. Choisissez le module souhaité pour continuer.
