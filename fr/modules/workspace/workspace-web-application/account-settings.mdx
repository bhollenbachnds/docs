La **Account Settings** section vous permet de gérer vos préférences personnelles et les informations de base liées à votre compte PADS4 Workspace.

**Accès aux Account Settings**

Depuis le **main menu**, cliquez sur **Settings** pour ouvrir vos préférences de compte.

### **Ce que vous pouvez mettre à jour**

Dans l'écran des paramètres, vous pouvez consulter ou modifier les éléments suivants :

* **Login Information**\
  (Visible, mais non modifiable si vous utilisez SSO)
* **Password**\
  (Modifiable uniquement si vous n'utilisez pas Single Sign-On)
* **Language**\
  Choisissez votre langue d'interface préférée
* **Notifications**\
  Activer ou désactiver les notifications dans l'application et par e-mail
* **Reminders**\
  Configurez vos paramètres de rappel par défaut pour votre compte.

> **Note:** Si vous êtes connecté via **SSO**, les champs d'identifiants de connexion et de mot de passe seront désactivés pour des raisons de sécurité et de conformité.
