Pour accéder à l'interface Crystal Administrator, ouvrez un navigateur web pris en charge comme Firefox ou Chrome. Ensuite, saisissez l'URL fournie par votre organisation dans la barre d'adresse.

Il est recommandé de mettre cette URL en favori pour un accès futur pratique. Vous pouvez étiqueter le favori "Crystal" ou utiliser un nom qui a du sens pour votre flux de travail.

Sur la page de connexion :

* Saisissez votre nom d'utilisateur et votre mot de passe dans les champs appropriés.
* Cliquez sur le bouton **Login** pour accéder à l'environnement administrateur.

Si vous ne connaissez pas vos identifiants de connexion ou si vous rencontrez des problèmes d'accès, contactez votre administrateur système pour obtenir de l'aide.
