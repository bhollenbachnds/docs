Après avoir créé un ou plusieurs profils, vous pouvez ajouter des utilisateurs et leur attribuer le profil approprié en fonction de leur rôle dans le système.

### Étapes pour créer un utilisateur

1. Dans l'interface Administrateur, accédez à la section **Utilisateurs**.

2. Cliquez sur l'option **Créer un nouvel utilisateur**.

3. Complétez les informations de l'utilisateur :
   * **Nom d'utilisateur** : Le nom de connexion que l'utilisateur utilisera.
   * **Mot de passe** : Créez un mot de passe sécurisé pour l'utilisateur.
   * **E-mail** (facultatif) : Utile pour les notifications ou la récupération de mot de passe, si pris en charge.

4. Attribuez un **Profil** à l'utilisateur. Cela déterminera ses permissions au sein du système.

5. Spécifiez des paramètres supplémentaires, si disponibles :
   * Préférence de langue
   * Fuseau horaire
   * Statut du compte (actif/inactif)

6. Cliquez sur **Enregistrer** pour ajouter l'utilisateur au système.

### Remarques

* Les utilisateurs ne peuvent pas accéder à la plateforme sans profil attribué.
* Les comptes utilisateurs peuvent être modifiés ou désactivés à tout moment par un administrateur.
* Les politiques de mot de passe doivent être respectées conformément aux exigences organisationnelles.
