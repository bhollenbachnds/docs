Une fois que vous avez terminé vos tâches dans l'interface Crystal Administrator, il est important de vous déconnecter pour garantir la sécurité de votre session—surtout si vous travaillez sur un appareil partagé ou public.

### Étapes pour se déconnecter

1. Accédez au coin supérieur droit de l'interface, où votre nom d'utilisateur ou les options de compte sont généralement affichés.

2. Sélectionnez l'option **Logout** dans le menu déroulant ou le menu du compte.

3. Le système mettra fin à votre session et vous redirigera vers l'écran de connexion.

### Remarques

* La déconnexion aide à prévenir les accès non autorisés.
* Les modifications non enregistrées peuvent être perdues si vous vous déconnectez avant d'avoir terminé certaines tâches.
* Déconnectez-vous toujours avant de fermer votre navigateur pour mettre fin complètement à votre session.
