Pour accéder à la plateforme Crystal :

1. **Ouvrez votre navigateur**\
   Crystal prend en charge les navigateurs web modernes tels que Firefox et Chrome. Nous vous recommandons d'enregistrer la page de connexion dans vos favoris pour un accès rapide à l'avenir.

2. **Saisissez l'URL fournie**\
   Celle-ci sera partagée par votre administrateur. Tapez-la directement dans la barre d'adresse du navigateur.

3. **Connectez-vous avec vos identifiants**\
   Utilisez l'identifiant et le mot de passe qui vous ont été donnés. Ces identifiants sont généralement fournis lors de l'intégration. Si vous n'êtes pas sûr de vos informations de connexion, veuillez contacter votre administrateur.

4. **Sélectionnez le module 'Affichage Dynamique'**\
   Une fois connecté, la page d'accueil affichera les modules accessibles. Cliquez sur "Affichage Dynamique" pour commencer à gérer ou créer votre contenu.
