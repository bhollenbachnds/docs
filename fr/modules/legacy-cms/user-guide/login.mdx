Pour accéder à la plateforme Crystal :

1. **Ouvrez votre navigateur**\
   Crystal prend en charge les navigateurs web modernes tels que Firefox et Chrome. Nous vous recommandons d'enregistrer la page de connexion dans vos favoris pour un accès rapide à l'avenir.

2. **Saisissez l'URL fournie**\
   Celle-ci sera communiquée par votre administrateur. Saisissez-la directement dans la barre d'adresse du navigateur.

3. **Connectez-vous avec vos identifiants**\
   Utilisez l'identifiant et le mot de passe qui vous ont été attribués. Ces identifiants sont généralement fournis lors de l'intégration. Si vous n'êtes pas sûr de vos informations de connexion, veuillez contacter votre administrateur.

4. **Sélectionnez le module « Digital Signage »**\
   Une fois connecté, la page d'accueil affiche les modules accessibles. Cliquez sur « Digital Signage » pour commencer à gérer ou créer votre contenu.
