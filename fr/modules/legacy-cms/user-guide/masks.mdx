**Masks** sont des modèles utilisés pour définir la mise en page et la structure des messages d'affichage dynamique. Ils déterminent où le contenu apparaît sur l'écran et comment il est formaté—comme un cadre de conception qui peut être réutilisé pour différents messages.

### Création d'un nouveau Mask

Pour créer un mask :

1. **Naviguez vers la section 'Masks'**\
   Sélectionnez **Masks** dans le menu de navigation.

2. **Cliquez sur 'Create Mask' ou Add New**\
   Cela ouvrira l'interface de l'éditeur de mask.

3. **Définissez les paramètres du mask** :
   * **Name** : Attribuez un nom qui reflète son objectif (par ex., "Welcome Layout" ou "News Bar").
   * **Resolution/Size** : Choisissez la résolution d'écran appropriée (par ex., 1920x1080 pour la HD complète).
   * **Orientation** : Sélectionnez **Portrait** ou **Landscape** selon votre configuration d'affichage.

4. **Concevez la mise en page du mask** :\
   Utilisez les outils de l'éditeur pour ajouter et positionner des zones de contenu telles que :
   * **Blocs de texte**
   * **Zones d'image**
   * **Zones vidéo**
   * **Bandeaux défilants**
   * **Champs de données dynamiques (par ex., date, météo, RSS)**

5. **Enregistrez votre mask**\
   Une fois la mise en page définie, enregistrez le mask pour qu'il puisse être réutilisé lors de la création de messages.

### Gestion des Masks existants

* **Edit** : Ouvrez n'importe quel mask pour mettre à jour sa mise en page ou ajuster les champs.
* **Duplicate** : Créez une copie d'un mask existant pour des modifications rapides.
* **Delete** : Supprimez les masks inutilisés (notez que les masks utilisés par des messages peuvent être verrouillés contre la suppression).

Les Masks vous permettent de maintenir une identité visuelle cohérente tout en donnant aux utilisateurs la flexibilité de mettre à jour le contenu des messages dans une structure prédéfinie.
