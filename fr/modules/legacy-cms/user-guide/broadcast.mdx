Le module **Broadcast** est utilisé pour distribuer des messages vers des écrans désignés ou des groupes d'écrans. C'est ici que vous finalisez et envoyez votre contenu afin qu'il apparaisse sur le réseau d'affichage dynamique.

### Accéder au module Broadcast

1. **Naviguez vers la section 'Broadcast'**\
   Sélectionnez **Broadcast** dans le menu principal.

2. **Afficher les messages en attente ou programmés**\
   Cette section affiche tous les messages qui sont soit en attente de publication, programmés pour une diffusion future, ou actuellement actifs.

### Diffuser un message

Pour publier un message sur vos écrans :

1. **Sélectionnez le message à diffuser**\
   Choisissez parmi la liste des messages enregistrés dans votre bibliothèque.

2. **Configurez les options de diffusion** :
   * **Start and End Date/Time** : Définissez quand le message doit commencer et arrêter de s'afficher.
   * **Recurrence** (si nécessaire) : Choisissez les paramètres de répétition—quotidien, hebdomadaire ou des modèles personnalisés.
   * **Target Viewers** : Sélectionnez un ou plusieurs écrans (viewers) ou groupes d'écrans pour recevoir le message.

3. **Vérifiez et confirmez**\
   Assurez-vous que tous les paramètres sont corrects, et cliquez sur **Publish** ou **Confirm Broadcast**.

### Gestion des diffusions

* **Edit** : Ajustez la programmation ou les cibles pour les diffusions déjà programmées.
* **Cancel or Delete** : Supprimez une diffusion avant ou pendant sa période active.
* **Filter** : Triez les diffusions par statut, date ou écrans assignés pour gérer plus efficacement plusieurs campagnes.

La diffusion est l'étape finale du processus de livraison de contenu et garantit que vos messages soigneusement créés atteignent les bons écrans au bon moment.
