Le **Broadcast** module est utilisé pour distribuer des messages vers des écrans désignés ou des groupes d'écrans. C'est ici que vous finalisez et envoyez votre contenu afin qu'il apparaisse sur le réseau d'affichage dynamique.

### Accéder au module Broadcast

1. **Accédez à la section 'Broadcast'**\
   Sélectionnez **Broadcast** depuis le menu principal.

2. **Afficher les messages en attente ou planifiés**\
   Cette section affiche tous les messages qui sont soit en attente de publication, planifiés pour une diffusion future, ou actuellement actifs.

### Diffuser un message

Pour publier un message sur vos écrans :

1. **Sélectionnez le message à diffuser**\
   Choisissez parmi la liste des messages enregistrés dans votre bibliothèque.

2. **Configurer les options de diffusion** :
   * **Date/Heure de début et de fin** : Définissez quand le message doit commencer et arrêter de s'afficher.
   * **Récurrence** (si nécessaire) : Choisissez les paramètres de répétition—quotidien, hebdomadaire ou personnalisé.
   * **Viewers cibles** : Sélectionnez un ou plusieurs écrans (viewers) ou groupes d'écrans pour recevoir le message.

3. **Vérifier et confirmer**\
   Assurez-vous que tous les paramètres sont corrects, puis cliquez sur **Publish** ou **Confirm Broadcast**.

### Gérer les diffusions

* **Modifier** : Ajustez la planification ou les viewers cibles pour les diffusions déjà planifiées.
* **Annuler ou Supprimer** : Supprimez une diffusion avant ou pendant sa fenêtre active.
* **Filtrer** : Triez les diffusions par statut, date ou viewers assignés pour gérer plusieurs campagnes plus efficacement.

La diffusion est la dernière étape du processus de livraison de contenu et garantit que vos messages soigneusement créés atteignent les bons écrans au bon moment.
