Les messages sont les éléments de contenu réels qui s'affichent sur vos écrans. Ils sont créés en utilisant une combinaison de catégories et de masques, et incluent généralement du texte, des images ou des éléments multimédias selon la mise en page définie dans le masque.

### Création d'un nouveau message

Pour créer un message :

1. **Accédez à la section 'Messages'**\
   Sélectionnez **Messages** dans le menu de navigation principal.

2. **Cliquez sur 'Créer un message' ou sur le bouton Ajouter (+)**\
   Cela ouvrira la page de création de message.

3. **Sélectionnez une Catégorie et un Masque**
   * Choisissez la **Catégorie** appropriée pour organiser votre message.
   * Sélectionnez un **Masque** qui détermine la mise en page du message.

4. **Remplissez les champs du message**\
   Les champs disponibles dépendront du masque que vous avez sélectionné. Les types d'entrée courants comprennent :
   * **Zones de texte**
   * **Téléchargements d'images**
   * **Liens ou téléchargements de vidéos**
   * **Texte défilant**
   * **Éléments dynamiques** (par exemple, heure, date, météo)

5. **Personnalisez chaque zone de contenu**\
   Saisissez ou téléchargez les médias et textes requis dans les zones prédéfinies.

6. **Enregistrez votre message**\
   Le message apparaîtra maintenant dans la liste et pourra être prévisualisé, modifié ou publié.

### Gestion des messages

* **Modifier** : Ouvrez n'importe quel message pour mettre à jour son contenu.
* **Supprimer** : Supprimez les messages qui ne sont plus nécessaires.
* **Rechercher / Filtrer** : Utilisez les filtres pour localiser rapidement les messages par catégorie, masque ou date.
* **Trier** : Vous pouvez trier la liste des messages par date, titre ou autres colonnes disponibles.

Les messages peuvent être prévisualisés avant publication pour s'assurer qu'ils apparaissent comme prévu.
