Les messages sont les éléments de contenu réels qui s'affichent sur vos écrans. Ils sont créés à l'aide d'une combinaison de catégories et de masks, et comprennent généralement du texte, des images ou des éléments multimédias selon la mise en page définie dans le mask.

### Créer un nouveau message

Pour créer un message :

1. **Accéder à la section 'Messages'**\
   Sélectionnez **Messages** dans le menu de navigation principal.

2. **Cliquez sur 'Create Message' ou sur le bouton Ajouter (+)**\
   Cela ouvrira la page de création de message.

3. **Sélectionner une Category et un Mask**
   * Choisissez la **Category** appropriée pour organiser votre message.
   * Sélectionnez un **Mask** qui détermine la mise en page du message.

4. **Remplir les champs du message**\
   Les champs disponibles dépendront du mask que vous avez sélectionné. Les types de saisie courants incluent :
   * **Zones de texte**
   * **Téléchargements d'images**
   * **Liens ou téléchargements de vidéos**
   * **Texte défilant**
   * **Éléments dynamiques** (par ex., heure, date, météo)

5. **Personnaliser chaque zone de contenu**\
   Saisissez ou téléchargez les médias et le texte requis dans les zones prédéfinies.

6. **Enregistrer votre message**\
   Le message apparaîtra désormais dans la liste et pourra être prévisualisé, modifié ou publié.

### Gérer les messages

* **Modifier** : Ouvrez n'importe quel message pour mettre à jour son contenu.
* **Supprimer** : Supprimez les messages qui ne sont plus nécessaires.
* **Rechercher / Filtrer** : Utilisez les filtres pour localiser rapidement les messages par catégorie, mask ou date.
* **Trier** : Vous pouvez trier la liste des messages par date, titre ou autres colonnes disponibles.

Les messages peuvent être prévisualisés avant publication pour s'assurer qu'ils s'affichent comme prévu.
