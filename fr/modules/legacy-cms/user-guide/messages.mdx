Les Messages sont les éléments de contenu réels qui s'affichent sur vos écrans. Ils sont créés en utilisant une combinaison de catégories et de masks, et incluent généralement du texte, des images ou des éléments multimédias selon la mise en page définie dans le mask.

### Création d'un nouveau Message

Pour créer un message :

1. **Allez à la section 'Messages'**\
   Sélectionnez **Messages** dans le menu de navigation principal.

2. **Cliquez sur 'Create Message' ou le bouton Ajouter (+)**\
   Cela ouvrira la page de création de message.

3. **Sélectionnez une Catégorie et un Mask**
   * Choisissez la **Category** appropriée pour organiser votre message.
   * Sélectionnez un **Mask** qui détermine la mise en page du message.

4. **Remplissez les champs du message**\
   Les champs disponibles dépendront du mask que vous avez sélectionné. Les types d'entrée courants comprennent :
   * **Zones de texte**
   * **Téléchargements d'images**
   * **Liens ou téléchargements de vidéos**
   * **Texte défilant**
   * **Éléments dynamiques** (par ex., heure, date, météo)

5. **Personnalisez chaque zone de contenu**\
   Saisissez ou téléchargez les médias et textes requis dans les zones prédéfinies.

6. **Enregistrez votre message**\
   Le message apparaîtra maintenant dans la liste et pourra être prévisualisé, modifié ou publié.

### Gestion des Messages

* **Edit** : Ouvrez n'importe quel message pour mettre à jour son contenu.
* **Delete** : Supprimez les messages qui ne sont plus nécessaires.
* **Search / Filter** : Utilisez les filtres pour localiser rapidement les messages par catégorie, mask ou date.
* **Sort** : Vous pouvez trier la liste des messages par date, titre ou autres colonnes disponibles.

Les messages peuvent être prévisualisés avant publication pour s'assurer qu'ils apparaissent comme prévu.
