L'onglet **Settings** dans le PADS4 Visitor Management System vous permet de personnaliser l'expérience utilisateur, l'apparence des badges, le comportement des bornes et les intégrations système. C'est l'emplacement central pour configurer les aspects opérationnels et visuels de votre processus d'enregistrement des visiteurs.

***

## Conception de badge

Vous pouvez concevoir un badge d'identification personnalisé qui correspond à l'image de marque de votre organisation.

### Options de personnalisation du badge :

* **Company Logo**\
  Téléchargez un logo (taille recommandée : **300x300 px**) pour apparaître sur le badge.

* **Brand Colors**\
  Sélectionnez vos **primary and secondary colors** pour refléter votre identité d'entreprise.

* **Font Selection**\
  Choisissez parmi les **Google Fonts** standard pour styliser le texte de votre badge.

### Aperçu et impression du badge :

* Utilisez le bouton **Preview** pour visualiser votre conception en temps réel.
* Vous pouvez **download the badge as an image** ou **print it as a PDF**.
* Le badge s'adaptera automatiquement en fonction de vos paramètres d'imprimante.
  > Assurez-vous que votre **printer or badge printer** est correctement configurée via les paramètres d'impression Windows.

***

## Configuration de la borne et du flux de travail

Dans l'onglet Settings, vous pouvez également définir comment la **VMS Kiosk** se comporte pendant l'enregistrement.

### Options disponibles :

* **Two-Step Verification**\
  Exige que les visiteurs s'enregistrent à la borne même après une pré-inscription.

* **Mandatory Company Documents**\
  Joignez des documents que les visiteurs doivent **read and agree to** avant d'entrer (par exemple, NDA, politiques de sécurité).

* **Decline Terms Message**\
  Ajoutez un **custom message** pour les utilisateurs qui refusent les conditions générales.

* **Auto-Archive**\
  Archivez automatiquement un visiteur après un nombre sélectionné de jours après sa sortie.

* **Include Terms in Email**\
  Intégrez les documents de l'entreprise directement dans l'**pre-registration email** pour examen à l'avance.

* **Show Virtual Keyboard**\
  Activez un **on-screen keyboard** pendant l'enregistrement à la borne (recommandé pour les écrans tactiles).

* **Time Zone Selection**\
  Choisissez le **default time zone** pour le système VMS et toutes les communications par e-mail.

## Clés API et intégration

Un **clé d'authentification API** est générée ici pour une utilisation dans **PADS4 Designer**.

### Pour se connecter aux données VMS via le JSON Dataprovider:

1. Ouvrez les **Advanced Connection Settings**
2. Naviguez vers **Headers > Add**
3. Configurez l'en-tête suivant:

* **Custom Header**:
  ```
  x-apikey
  ```

* **Value**:
  ```
  #your API Key#
  ```

> Pour une documentation API détaillée, accédez à la page Swagger de votre système:

```
http://<servername>:<portnumber>/rdx/NDS.Visitor.Management/swagger/index.html
```
