L'onglet **Settings** dans le PADS4 Visitor Management System vous permet de personnaliser l'expérience utilisateur, l'apparence des badges, le comportement des bornes et les intégrations système. C'est l'emplacement central pour configurer les aspects opérationnels et visuels de votre processus d'enregistrement des visiteurs.

***

## Conception de badge

Vous pouvez concevoir un badge d'identification personnalisé qui correspond à l'image de marque de votre organisation.

### Options de personnalisation du badge :

* **Logo de l'entreprise**\
  Téléchargez un logo (taille recommandée : **300x300 px**) pour apparaître sur le badge.

* **Couleurs de la marque**\
  Sélectionnez vos **couleurs primaires et secondaires** pour refléter l'identité de votre entreprise.

* **Sélection de police**\
  Choisissez parmi les **Google Fonts** standard pour styliser le texte de votre badge.

### Aperçu et impression du badge :

* Utilisez le bouton **Preview** pour visualiser votre conception en temps réel.
* Vous pouvez **télécharger le badge sous forme d'image** ou **l'imprimer en PDF**.
* Le badge s'adaptera automatiquement en fonction de vos paramètres d'impression.
  > Assurez-vous que votre **imprimante ou imprimante de badges** est correctement configurée via les paramètres d'impression Windows.

***

## Configuration de la borne et du flux de travail

Dans l'onglet Settings, vous pouvez également définir comment la **VMS Kiosk** se comporte pendant l'enregistrement.

### Options disponibles :

* **Vérification en deux étapes**\
  Oblige les visiteurs à s'enregistrer au kiosque même après une pré-inscription.

* **Documents d'entreprise obligatoires**\
  Joindre des documents que les visiteurs doivent **lire et accepter** avant d'entrer (par exemple, NDA, politiques de sécurité).

* **Message de refus des conditions**\
  Ajouter un **message personnalisé** pour les utilisateurs qui refusent les termes et conditions.

* **Archivage automatique**\
  Archiver automatiquement un visiteur après un nombre sélectionné de jours après sa sortie.

* **Inclure les conditions dans l'e-mail**\
  Intégrer les documents de l'entreprise directement dans l'**e-mail de pré-inscription** pour examen préalable.

* **Afficher le clavier virtuel**\
  Activer un **clavier à l'écran** pendant l'enregistrement au kiosque (recommandé pour les écrans tactiles).

* **Sélection du fuseau horaire**\
  Choisir le **fuseau horaire par défaut** pour le système VMS et toutes les communications par e-mail.

## Clés API et intégration

Une **clé d'authentification API** est générée ici pour une utilisation dans **PADS4 Designer**.

### Pour se connecter aux données VMS via le JSON Dataprovider :

1. Ouvrez les **Advanced Connection Settings**
2. Naviguez vers **Headers > Add**
3. Configurez l'en-tête suivant :

* **Custom Header**:
  ```
  x-apikey
  ```

* **Value**:
  ```
  #your API Key#
  ```

> Pour une documentation API détaillée, accédez à la page Swagger de votre système :

```
http://<servername>:<portnumber>/rdx/NDS.Visitor.Management/swagger/index.html
```
