L'onglet **Settings** dans le PADS4 Visitor Management System vous permet de personnaliser l'expérience utilisateur, l'apparence des badges, le comportement du kiosque et les intégrations système. C'est l'emplacement central pour configurer les aspects opérationnels et visuels de votre processus d'enregistrement des visiteurs.

***

## Badge Design

Vous pouvez concevoir un badge d'identification personnalisé correspondant à l'image de marque de votre organisation.

### Options de personnalisation du badge :

* **Logo de l'entreprise**\
  Téléchargez un logo (taille recommandée : **300x300 px**) pour qu'il apparaisse sur le badge.

* **Couleurs de marque**\
  Sélectionnez vos **couleurs primaires et secondaires** pour refléter votre identité d'entreprise.

* **Sélection de la police**\
  Choisissez parmi les **Google Fonts** standard pour styliser le texte de votre badge.

### Aperçu et impression du badge :

* Utilisez le bouton **Preview** pour visualiser votre design en temps réel.
* Vous pouvez **télécharger le badge en tant qu'image** ou **l'imprimer en PDF**.
* Le badge sera automatiquement mis à l'échelle en fonction des paramètres de votre imprimante.
  > Assurez-vous que votre **imprimante ou imprimante de badges** est correctement configurée via les paramètres d'impression Windows.

***

## Configuration du kiosque et du flux de travail

Dans l'onglet Settings, vous pouvez également définir le comportement du **VMS Kiosk** lors de l'enregistrement.

### Options disponibles :

* **Vérification en deux étapes**\
  Oblige les visiteurs à s'enregistrer au kiosque même après une pré-inscription.

* **Documents d'entreprise obligatoires**\
  Joignez des documents que les visiteurs doivent **lire et accepter** avant d'entrer (par ex., NDA, politiques de sécurité).

* **Message de refus des conditions**\
  Ajoutez un **message personnalisé** pour les utilisateurs qui refusent les conditions générales.

* **Auto-Archive**\
  Archivez automatiquement un visiteur après un nombre de jours sélectionné après son départ.

* **Inclure les conditions dans l'e-mail**\
  Intégrez les documents de l'entreprise directement dans l'**e-mail de pré-inscription** pour consultation préalable.

* **Afficher le clavier virtuel**\
  Activez un **clavier à l'écran** lors de l'enregistrement au kiosque (recommandé pour les écrans tactiles).

* **Sélection du fuseau horaire**\
  Choisissez le **fuseau horaire par défaut** pour le système VMS et toutes les communications par e-mail.

## Clés API et intégration

Une **clé d'authentification API** est générée ici pour être utilisée dans **PADS4 Designer**.

### Pour se connecter aux données VMS via le JSON Dataprovider :

1. Ouvrez les **Advanced Connection Settings**
2. Accédez à **Headers > Add**
3. Configurez l'en-tête suivant :

* **Custom Header**:
  ```
  x-apikey
  ```

* **Value**:
  ```
  #your API Key#
  ```

> Pour une documentation API détaillée, accédez à la page Swagger de votre système :

```
http://<servername>:<portnumber>/rdx/NDS.Visitor.Management/swagger/index.html
```
