La **page d'accueil** du PADS4 Visitor Management System (VMS) fournit une vue d'ensemble des visiteurs actifs et pré-enregistrés, des actions système et de l'accès aux fonctionnalités principales telles que **la création d'utilisateurs**, **les archives** et **les paramètres**.

Ce tableau de bord central constitue votre point de départ pour gérer les entrées des invités, configurer l'environnement et gérer les enregistrements du jour.

***

## Premiers pas

Depuis la page d'accueil, vous pouvez :

* **Créer une nouvelle entrée de visiteur**
* **Consulter les dossiers d'invités archivés**
* **Accéder aux paramètres système**

Nous allons commencer par parcourir le processus de création d'un nouvel utilisateur.

***

## Options du tableau de bord en haut à droite

* **Sélection de la langue** : PADS4 VMS prend en charge **l'anglais** et **le français**.
* **Déconnexion** : Utilisez l'**icône utilisateur** dans le coin supérieur droit pour vous déconnecter du système en toute sécurité.

***

## Création d'une nouvelle entrée de visiteur

Pour créer un nouveau visiteur, les champs suivants sont **obligatoires** :

* **Adresse e-mail**
* **Hôte assigné** (depuis Workspace)
* **Bâtiment assigné** (depuis Workspace)

### Champs facultatifs :

* Prénom / Nom
* Nom de l'entreprise
* Objet de la visite
* Heure d'arrivée et de départ prévues

### Notification par e-mail et auto-enregistrement

Une fois le visiteur créé :

* Il reçoit un **e-mail automatisé** avec les instructions d'enregistrement.
* L'e-mail contient un **lien de pré-inscription**, leur permettant de :
  * Modifier leurs informations personnelles (nom, entreprise, etc.)
  * Confirmer leur présence
* Les visiteurs pré-inscrits peuvent **se valider eux-mêmes** sur place à l'aide d'un QR code ou d'un code PIN.

***

## Inscription sur site (lien VMS Kiosk)

Pour l'auto-enregistrement sur site :

Déployer **des tablettes interactives, des ordinateurs portables ou des PADS4 Viewers** avec des navigateurs configurés pour ouvrir le **VMS Kiosk URL**:

```
https://yourpads4domain.com/VMS/Kiosk/welcome
```

ou

```
http://<public-ip>/VMS/Kiosk/welcome
```

> **Important (PADS4 version 2022.3 et ultérieure) :**\
> Utiliser la **structure d'URL spécifique au domaine correcte** pour éviter les problèmes de licence dans les environnements multi-domaines :

```
https://serverlocation/VMS/Kiosk/domain
```

Pour les environnements à domaine unique :

```
https://serverlocation/VMS/Kiosk/pads
```

***

## Enregistrement par QR Code ou PIN

Les visiteurs peuvent s'enregistrer en utilisant :

* Un **QR Code** (appareils avec caméra requis)
* Un **PIN attribué aléatoirement**

> Les navigateurs demanderont l'autorisation d'accéder à la caméra si la lecture de QR code est utilisée. Navigateurs pris en charge : Chrome, Edge, Firefox, Internet Explorer.

***

## Boutons d'action du tableau de bord

Une fois qu'un utilisateur est inscrit, les actions suivantes sont disponibles via le **Bouton d'action** :

### Pour tous les utilisateurs inscrits :

* **Enregistrement manuel** – Enregistrer directement l'utilisateur depuis le tableau de bord.
* **Modifier l'utilisateur** – Mettre à jour les informations personnelles et les coordonnées.
* **Supprimer l'utilisateur** – Supprimer l'utilisateur du système.

### Pour les utilisateurs enregistrés :

* **Imprimer le badge d'identité** – Télécharger ou imprimer un badge visiteur personnalisé. Le style du badge peut être configuré dans l'onglet **Paramètres**.
* **Départ manuel** – Déconnecter l'utilisateur lorsque sa visite est terminée.
