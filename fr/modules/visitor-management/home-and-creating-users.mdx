La **homepage** du système de gestion des visiteurs PADS4 (VMS) fournit une vue d'ensemble des visiteurs actifs et préenregistrés, des actions système et un accès aux fonctionnalités principales telles que la **user creation**, les **archives** et les **settings**.

Ce tableau de bord central sert de point de départ pour gérer les entrées des invités, configurer l'environnement et gérer les enregistrements le jour même.

***

## Premiers pas

Depuis la page d'accueil, vous pouvez :

* **Créer une nouvelle entrée de visiteur**
* **Consulter les enregistrements archivés des invités**
* **Accéder aux paramètres du système**

Nous allons commencer par vous guider à travers le processus de création d'un nouvel utilisateur.

***

## Options du tableau de bord en haut à droite

* **Sélection de la langue** : PADS4 VMS prend en charge **English** et **French**.
* **Logout** : Utilisez **user icon** dans le coin supérieur droit pour vous déconnecter en toute sécurité du système.

***

## Création d'une nouvelle entrée de visiteur

Pour créer un nouveau visiteur, les champs suivants sont **required** :

* **Email Address**
* **Assigned Host** (de Workspace)
* **Assigned Building** (de Workspace)

### Champs optionnels :

* First Name / Last Name
* Company Name
* Visit Purpose
* Expected Arrival and Departure Time

### Notification par e-mail et auto-enregistrement

Une fois le visiteur créé :

* Il reçoit un **automated email** avec les instructions d'enregistrement.
* L'e-mail inclut un **pre-registration link**, leur permettant de :
  * Modifier leurs informations personnelles (nom, entreprise, etc.)
  * Confirmer leur présence
* Les visiteurs préenregistrés peuvent **validate themselves** sur place en utilisant un code QR ou un code PIN.

***

## Enregistrement sur site (Lien Kiosque VMS)

Pour l'auto-enregistrement sur site :

Déployez **des tablettes interactives, des ordinateurs portables ou des PADS4 Viewers** avec des navigateurs configurés pour ouvrir l'**URL du Kiosque VMS**:

```
https://yourpads4domain.com/VMS/Kiosk/welcome
```

ou

```
http://<public-ip>/VMS/Kiosk/welcome
```

> **Important (PADS4 version 2022.3 et plus récente) :**\
> Utilisez la **structure d'URL spécifique au domaine correcte** pour éviter les problèmes de licence dans les environnements multi-domaines :

```
https://serverlocation/VMS/Kiosk/domain
```

Pour les environnements à domaine unique :

```
https://serverlocation/VMS/Kiosk/pads
```

***

## Enregistrement par QR Code ou PIN

Les visiteurs peuvent s'enregistrer en utilisant soit :

* Un **QR Code** (appareils équipés d'une caméra requis)
* Un **PIN attribué aléatoirement**

> Les navigateurs demanderont l'autorisation d'accéder à la caméra si le scan de QR code est utilisé. Navigateurs pris en charge : Chrome, Edge, Firefox, Internet Explorer.

***

## Boutons d'action du tableau de bord

Une fois qu'un utilisateur est enregistré, les actions suivantes sont disponibles via le **Bouton d'Action**:

### Pour tous les utilisateurs enregistrés :

* **Enregistrement manuel** – Enregistrer directement l'utilisateur depuis le tableau de bord.
* **Modifier l'utilisateur** – Mettre à jour les informations personnelles et les coordonnées.
* **Supprimer l'utilisateur** – Retirer l'utilisateur du système.

### Pour les utilisateurs enregistrés :

* **Imprimer un badge d'identification** – Télécharger ou imprimer un badge visiteur personnalisé. Le style du badge peut être configuré sous l'onglet **Paramètres**.
* **Départ manuel** – Déconnecter l'utilisateur lorsque sa visite est terminée.
