Un Data Vault vous permet de réutiliser du contenu géré de manière centralisée dans plusieurs présentations. Suivez les étapes ci-dessous pour ajouter un Data Vault existant depuis PADS4 Designer.

<Info>
  Les Data Vaults sont créés et gérés dans PADS4 System Management. Pour plus de détails sur la création ou la configuration d'un Data Vault, consultez le manuel PADS4 System Management.
</Info>
