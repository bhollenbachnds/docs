La création d'un message dans PADS4 CMS Designer est le processus de combinaison d'un modèle préconçu avec du contenu dynamique ou statique. Les messages sont ce qui est finalement publié sur les écrans.

***

## Catégories et modèles

1. Naviguez vers la section**Messages** depuis le menu de navigation principal.
2. Cliquez sur**Create New Message**.
3. Vous serez invité à :
   * **Select a Category** : Choisissez la catégorie appropriée pour organiser le message.
   * **Select a Template** : Sélectionnez parmi les modèles existants disponibles dans cette catégorie.

Une fois sélectionné, la page de création de message se chargera avec le modèle choisi.

***

## Page de création de message

Cette page est l'endroit où vous remplissez le contenu qui sera affiché dans votre modèle sélectionné. Les zones modifiables du modèle sont définies lors de la création du modèle et seront clairement marquées.

***

## Champs à remplir

Chaque zone modifiable du modèle affichera des champs de saisie. Selon la conception, ces champs peuvent inclure :

* **Text fields** (par exemple, titre, sous-titre, description)
* **Image or video placeholders**
* **Date/time selectors**
* **Dropdowns or checkboxes** pour des options personnalisées

Entrez simplement ou sélectionnez le contenu requis dans chaque champ.

***

## Changer un média

Pour changer une image ou une vidéo dans un emplacement média :

1. Cliquez sur la zone média ou sur son bouton associé**Change Media**.
2. Choisissez parmi :
   * **Upload a new file**
   * **Select from the Library**

Le fichier sélectionné sera inséré dans le modèle et reflété immédiatement dans l'aperçu.

***

## Ajouter plusieurs médias (glisser-déposer)

Vous pouvez ajouter plusieurs fichiers médias simultanément en utilisant le glisser-déposer :

1. Faites glisser des images ou des vidéos depuis votre bureau ou dossier directement dans l'éditeur de message.
2. Les fichiers seront téléchargés et rempliront les champs respectifs dans votre message.
3. Vous pouvez les réaffecter ou les remplacer selon vos besoins.

Ceci est particulièrement utile lorsque vous travaillez avec des modèles qui comportent plusieurs éléments visuels.

***

## Aperçu

Avant de publier :

1. Cliquez sur le bouton**Preview** pour voir votre message en temps réel.
2. L'aperçu reflétera votre saisie et mise en page actuelles.
3. Profitez de cette occasion pour confirmer l'alignement, le formatage et l'exactitude du contenu.

Une fois satisfait de l'aperçu, vous êtes prêt à passer aux étapes de publication.
