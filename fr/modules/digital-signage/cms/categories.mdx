---
- title: Categories
---

Les catégories dans PADS4 CMS Designer sont utilisées pour organiser les modèles et les messages. Elles contribuent à maintenir un environnement structuré, notamment lors de la gestion du contenu entre plusieurs départements, emplacements ou cas d'utilisation.

## Créer une nouvelle catégorie

Pour créer une nouvelle catégorie :

* Accédez à la section **Nouveau message** dans le menu de navigation principal.

![Cms1](/images/digital-signage/cms/cms1.png)

* Cliquez sur le bouton **Créer une nouvelle catégorie**.

<img src="/images/digital-signage/cms/cms-new-category.png" alt="Cms New Category" style={{ width:"41%" }} />

* Dans le formulaire qui apparaît, saisissez les informations suivantes :
  * **Nom de la catégorie** : Le nom qui sera utilisé pour identifier cette catégorie.
  * **Description** *(facultatif)*: Une brève explication de l'objet de cette catégorie.
* Sélectionnez le viewer avec lequel ce modèle pourra être utilisé.

<img src="/images/digital-signage/cms/cms-new-category2.png" alt="Cms New Category2" style={{ width:"44%" }} />

* Cliquez sur **Create** pour créer la catégorie.

Une fois créée, la catégorie apparaîtra dans la liste et pourra être utilisée pour organiser les modèles et les messages. Les catégories peuvent également aider à définir les autorisations des utilisateurs et à simplifier la création de messages en regroupant les contenus associés.

![Create A Template1](/images/digital-signage/cms/create-a-template1.png)
