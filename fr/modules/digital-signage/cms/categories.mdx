---
- title: Catégories
---

Les catégories dans PADS4 CMS Designer sont utilisées pour organiser les modèles et les messages. Elles aident à maintenir un environnement structuré, en particulier lors de la gestion de contenu à travers plusieurs départements, emplacements ou cas d'utilisation.

## Créer une nouvelle catégorie

Pour créer une nouvelle catégorie :

* Naviguez vers la section **Nouveau message** dans le menu de navigation principal.

![Cms1](/images/digital-signage/cms/cms1.png)

* Cliquez sur le bouton **Créer une nouvelle catégorie**.

<img src="/images/digital-signage/cms/cms-new-category.png" alt="Cms New Category" style={{ width:"41%" }} />

* Dans le formulaire qui apparaît, entrez les détails suivants :
  * **Nom de la catégorie** : Le nom qui sera utilisé pour identifier cette catégorie.
  * **Description** *(facultatif)* : Une brève explication de l'utilité de cette catégorie.
* Sélectionnez le viewer avec lequel ce modèle sera autorisé à être utilisé.

<img src="/images/digital-signage/cms/cms-new-category2.png" alt="Cms New Category2" style={{ width:"44%" }} />

* Cliquez sur **Créer** pour créer la catégorie.

Une fois créée, la catégorie apparaîtra dans la liste et pourra être utilisée pour organiser les modèles et les messages. Les catégories peuvent également aider à définir les permissions des utilisateurs et à rationaliser la création de messages en regroupant le contenu connexe.
