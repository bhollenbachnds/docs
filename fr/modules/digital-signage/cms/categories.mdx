Les catégories dans PADS4 CMS Designer sont utilisées pour organiser les modèles et les messages. Elles aident à maintenir un environnement structuré, particulièrement lors de la gestion de contenu à travers plusieurs départements, emplacements ou cas d'utilisation.

## Créer une nouvelle catégorie

Pour créer une nouvelle catégorie :

1. Naviguez vers la section **Categories** dans le menu de navigation principal.
2. Cliquez sur le bouton **Create a new category**.
3. Dans le formulaire qui apparaît, saisissez les détails suivants :
   * **Category Name** : Le nom qui sera utilisé pour identifier cette catégorie.
   * **Description** *(facultatif)* : Une brève explication de l'utilité de cette catégorie.
4. Cliquez sur **Save** pour créer la catégorie.

Une fois créée, la catégorie apparaîtra dans la liste et pourra être utilisée pour organiser les modèles et les messages. Les catégories peuvent également aider à définir les permissions des utilisateurs et à simplifier la création de messages en regroupant le contenu associé.
