---
- title: Categories
---

Les Categories dans PADS4 CMS Designer sont utilisées pour organiser les templates et les messages. Elles permettent de maintenir un environnement structuré, notamment lors de la gestion de contenu entre plusieurs départements, emplacements ou cas d'utilisation.

## Créer une nouvelle Category

Pour créer une nouvelle category :

* Accédez à la section **New message** dans le menu de navigation principal.

![Cms1](/images/digital-signage/cms/cms1.png)

* Cliquez sur le bouton **Create a new category**.

<img src="/images/digital-signage/cms/cms-new-category.png" alt="Cms New Category" style={{ width:"41%" }} />

* Dans le formulaire qui apparaît, saisissez les informations suivantes :
  * **Category Name** : Le nom qui sera utilisé pour identifier cette category.
  * **Description** *(optionnel)* : Une brève explication de l'objet de cette category.
* Sélectionnez le viewer avec lequel ce template pourra être utilisé.

<img src="/images/digital-signage/cms/cms-new-category2.png" alt="Cms New Category2" style={{ width:"44%" }} />

* Cliquez sur **Create** pour créer la category.

Une fois créée, la category apparaîtra dans la liste et pourra être utilisée pour organiser les templates et les messages. Les categories peuvent également aider à définir les permissions des utilisateurs et à simplifier la création de messages en regroupant les contenus associés.

![Create A Template1](/images/digital-signage/cms/create-a-template1.png)
