---
- title: Categories
---

Les catégories dans PADS4 CMS Designer sont utilisées pour organiser les modèles et les messages. Elles aident à maintenir un environnement structuré, particulièrement lors de la gestion de contenu à travers plusieurs départements, emplacements ou cas d'utilisation.

## Créer une nouvelle catégorie

Pour créer une nouvelle catégorie :

* Naviguez vers la section **New message** dans le menu de navigation principal.

![Cms1](/images/digital-signage/cms/cms1.png)

* Cliquez sur le bouton **Create a new category**.

<img src="/images/digital-signage/cms/cms-new-category.png" alt="Cms New Category" style={{ width:"41%" }} />

* Dans le formulaire qui apparaît, saisissez les détails suivants :
  * **Category Name** : Le nom qui sera utilisé pour identifier cette catégorie.
  * **Description** *(facultatif)* : Une brève explication de l'utilité de cette catégorie.
* Sélectionnez le visualiseur avec lequel ce modèle pourra être utilisé.

<img src="/images/digital-signage/cms/cms-new-category2.png" alt="Cms New Category2" style={{ width:"44%" }} />

* Cliquez sur **Create** pour créer la catégorie.

Une fois créée, la catégorie apparaîtra dans la liste et pourra être utilisée pour organiser les modèles et les messages. Les catégories peuvent également aider à définir les permissions des utilisateurs et à rationaliser la création de messages en regroupant le contenu connexe.

![Create A Template1](/images/digital-signage/cms/create-a-template1.png)
