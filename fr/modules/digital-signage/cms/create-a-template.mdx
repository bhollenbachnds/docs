Les modèles servent de base pour les messages créés dans PADS4 CMS. Ils définissent la mise en page, la conception et les zones modifiables du contenu que les utilisateurs finaux utiliseront pour créer et publier des messages.

***

## Ouvrez le Web Designer

1. Naviguez vers la section **Templates** depuis le menu de navigation.
2. Cliquez sur **Create a new template** ou sélectionnez un modèle existant à modifier.
3. Le système ouvrira le **Web Designer**, l'éditeur visuel intégré pour créer et gérer les modèles.

***

## Définir le modèle (Nom, Taille, Orientation)

Lors de la création d'un nouveau modèle, vous serez invité à configurer ses propriétés de base :

* **Name** : Attribuez un nom significatif à votre modèle.
* **Size** : Définissez la résolution (par exemple, 1920x1080 pour les écrans Full HD).
* **Orientation** : Choisissez entre **Landscape** ou **Portrait**, selon la façon dont vos écrans sont montés.

Cliquez sur **Next** ou **Save** pour continuer vers l'interface de conception.

***

## Présentation de l'interface

Une fois dans le Web Designer, l'interface est divisée en zones clés :

* **Canvas Area** : L'espace de conception principal où vous placez et organisez les éléments.
* **Toolbar** : Donne accès aux outils de dessin, d'insertion de médias, d'alignement et de gestion des calques.
* **Property Panel** : Situé à droite, il affiche les options de configuration pour l'élément sélectionné (par exemple, taille, couleur, alignement).
* **Layer Panel** : Liste tous les objets du modèle, vous permettant de réorganiser, regrouper ou supprimer des éléments.

***

## Agrandir la page principale

Pour zoomer ou maximiser l'espace de travail :

* Utilisez le curseur de zoom ou les icônes de zoom généralement situées en bas à droite du canevas.
* Vous pouvez également appuyer sur **Ctrl + Scroll** sur votre souris pour zoomer et dézoomer.

Cela permet un placement plus précis des éléments de conception.

***

## Ajouter une forme

1. Dans la barre d'outils, sélectionnez l'outil **Shape**.
2. Choisissez un type de forme (par exemple, rectangle, cercle).
3. Cliquez et faites glisser sur le canevas pour dessiner la forme.
4. Utilisez le Property Panel pour personnaliser sa couleur, sa taille, sa bordure et sa position.

***

## Ajouter une image

1. Sélectionnez l'outil **Image** dans la barre d'outils.
2. Cliquez sur le canevas pour placer le bloc d'image.
3. Dans le Property Panel, cliquez sur **Upload** ou **Select from Library** pour ajouter un fichier image.
4. Redimensionnez et positionnez l'image selon vos besoins.

***

## Ajouter du texte

1. Cliquez sur l'outil **Text** dans la barre d'outils.
2. Cliquez sur le canevas pour créer une zone de texte.
3. Tapez le texte souhaité.
4. Dans le Property Panel, ajustez la police, la taille, la couleur, l'alignement et les effets.

***

## Enregistrer votre modèle

Lorsque votre mise en page est terminée :

1. Cliquez sur le bouton **Save** dans la barre d'outils supérieure.
2. Votre modèle sera enregistré et deviendra disponible dans la section **Templates**.
3. Il peut maintenant être utilisé comme base pour créer des messages.
