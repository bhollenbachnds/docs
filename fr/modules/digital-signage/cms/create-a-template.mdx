## Interface Overview

Once inside the Web Designer, the interface is divided into key areas:

- **Canvas Area**: The main design space where you place and arrange elements.

![Create A Template3](/images/digital-signage/cms/create-a-template3.png)

- **Toolbar**: Provides access to drawing tools, media insertion, alignment, and layer management.

  <img
    src="/images/digital-signage/WebDesigner/WebDesignerElementToolBar.png"
    alt="Web Designer Element Tool Bar"
    title="Web Designer Element Tool Bar"
    style={{ width:"64%" }}
  />
- **Property Panel**: Located on the right, it displays configuration options for the selected element (e.g., size, color, alignment).

<img
  src="/images/digital-signage/cms/create-a-template5.png"
  alt="Create A Template5"
  style={{ width:"53%" }}
/>

- **Layer Panel**: Lists all objects on the template, allowing you to reorder, group, or delete elements.

## Agrandir la page principale

Pour zoomer ou maximiser l'espace de travail :

* Utilisez le curseur de zoom ou les icônes de zoom situées en bas à droite du canevas.

<img src="/images/digital-signage/cms/create-a-template8.png" alt="Create A Template8" style={{ width:"42%" }} />

* Vous pouvez également appuyer sur **Ctrl + Scroll** sur votre souris pour zoomer et dézoomer.

Cela permet un placement plus précis des éléments de conception.

## Ajouter une forme

1. Dans la barre d'outils, sélectionnez l'outil **Shape**.

<img src="/images/digital-signage/cms/create-a-template9.png" alt="Create A Template9" style={{ width:"32%" }} />

2. Choisissez un type de forme (par exemple, rectangle, cercle).
3. Cliquez et faites glisser sur le canevas pour dessiner la forme.
4. Utilisez le panneau des propriétés pour personnaliser sa couleur, sa taille, sa bordure et sa position.

![Create A Template10](/images/digital-signage/cms/create-a-template10.png)

## Ajouter une image / vidéo

1. Sélectionnez l'outil **Image** dans la barre d'outils.
2. Cliquez sur le canevas pour placer le bloc d'image.
3. Dans le panneau des propriétés, vous aurez trois options : 4. **Empty item** : Si vous souhaitez que l'utilisateur final ajoute le contenu et que vous fournissez simplement l'élément lui-même. 4. **Content item** : Vous sélectionnez le contenu dans la bibliothèque de contenu. 4. **External item** : Image ou vidéo hébergée ailleurs.

![Create A Template11](/images/digital-signage/cms/create-a-template11.png)

* Lorsque vous utilisez **content item** une fenêtre contextuelle de votre bibliothèque de contenu PADS4 s'affichera et vous permettra de choisir le média souhaité.

![Create A Template12](/images/digital-signage/cms/create-a-template12.png)

* Redimensionnez et positionnez l'image selon vos besoins.

## Ajouter du texte

1. Cliquez sur l'outil **Text** dans la barre d'outils et choisissez un champ de texte normal ou un élément de date et heure.

![Create A Template14](/images/digital-signage/cms/create-a-template14.png)

2. Cliquez sur le canevas pour créer une zone de texte.
3. Tapez le texte souhaité.
4. Dans le panneau des propriétés, ajustez la police, la taille, la couleur, l'alignement et les effets.

![Create A Template13](/images/digital-signage/cms/create-a-template13.png)

## Enregistrer votre modèle

Lorsque votre mise en page est terminée :

1. Cliquez sur le bouton **Save** dans la barre d'outils supérieure.

![Create A Template15](/images/digital-signage/cms/create-a-template15.png)

2. Votre modèle sera enregistré et deviendra disponible dans la section **Templates**.
3. Il peut maintenant être utilisé comme base pour créer des messages.

![Create A Template16](/images/digital-signage/cms/create-a-template16.png)
