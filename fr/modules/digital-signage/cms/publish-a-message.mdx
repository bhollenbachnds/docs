---
- title: Publier un message
---

Une fois votre message terminé et prévisualisé, la dernière étape consiste à le publier. La publication implique de définir quand et où le message doit apparaître.

Le processus de publication est divisé en trois étapes principales :

## Étape 01 : Titre

Dans cette étape, vous fournissez un titre pour votre message.

* **Titre** : Saisissez un nom descriptif qui aide à identifier le message dans le CMS.
* Ce titre est à usage interne et ne sera pas affiché à l'écran.
* Assurez-vous que le nom est significatif afin que les autres utilisateurs ou administrateurs puissent le reconnaître facilement.

Une fois terminé, cliquez sur **Suivant** pour continuer.

## Étape 02 : Date et récurrences

Ici, vous définissez quand le message doit être affiché.

* **Date et heure de début** : Quand le message doit commencer à s'afficher.
* **Date et heure de fin** : Quand le message doit cesser de s'afficher.
* **Options de récurrence** *(le cas échéant)* :
  * **Quotidien, hebdomadaire ou personnalisé** des modèles de récurrence peuvent être configurés pour automatiser la répétition des messages.

Cette étape garantit que votre contenu s'affiche au bon moment sans intervention manuelle.

## Étape 03 : Viewers

Dans cette dernière étape, vous sélectionnez où le message sera affiché.

* **Viewers ou groupes de Viewers** : Choisissez un ou plusieurs écrans (ou groupes d'écrans) où le message apparaîtra.
* Une liste des Viewers disponibles est affichée ; sélectionnez la ou les destinations appropriées en fonction de votre audience ou emplacement cible.

Après avoir sélectionné vos Viewers, cliquez sur **Publish** pour terminer le processus.

Votre message est maintenant planifié et s'affichera selon les paramètres que vous avez définis.
