Cette section décrit les étapes nécessaires pour installer et activer le **PADS4FIDS Module** dans le cadre de votre configuration PADS4 Foundation. Une fois PADS4 Foundation installé, le produit PADS4FIDS est automatiquement inclus sur le même serveur.

## Installation étape par étape

***

### **Étape 1 : Activer votre licence produit**

Après avoir terminé l'installation :

1. Lancez le **PADS4 CTRL Center**.
2. Naviguez vers le menu **Licenses**.
3. Entrez la clé produit fournie avec votre package d'installation.

> Besoin d'aide ? Consultez le [PADS4 CTRL Center Aide en ligne](#) pour des instructions détaillées sur la saisie d'une nouvelle clé de produit.

***

### **Étape 2 : Se connecter à PADS4 System Management**

1. Ouvrez l'**PADS4 System Management** application.
2. Connectez-vous avec vos identifiants.
   > **Identifiants de connexion par défaut après installation :**\
   > Username: `user1`\
   > Password: `user1`

***

### **Étape 3 : Configurer les paramètres de la base de données SQL**

1. Dans l'**System Management** application, accédez à **PADS Server Settings**.
2. Modifiez les paramètres de **PADS4FIDS** pour le connecter à votre serveur SQL.

> PADS4FIDS nécessite SQL Server pour le stockage de ses données.

***

### **Étape 4 : Attribuer une licence utilisateur pour PADS4FIDS Operator**

Pour permettre aux utilisateurs d'accéder à l'interface FIDS Operator :

1. Ouvrez le **PADS4 CTRL Center**.
2. Accédez à la section **Licenses**.
3. Attribuez une **licence PADS4FIDS Operator** au compte utilisateur requis.

***

### **Étape 5 : Lancer le PADS4FIDS Operator**

Utilisez un navigateur web tel que **Google Chrome** ou **Microsoft Edge** (anciennement Internet Explorer) pour démarrer l'interface **PADS4FIDS Operator**.

Une fois lancé, vous pouvez commencer à configurer et à gérer le contenu de votre affichage FIDS.
