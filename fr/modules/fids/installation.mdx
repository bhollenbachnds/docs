Cette section décrit les étapes nécessaires pour installer et activer le **PADS4FIDS Module** dans le cadre de votre configuration PADS4 Foundation. Une fois que PADS4 Foundation a été installé, le produit PADS4FIDS est automatiquement inclus sur le même serveur.

## Installation étape par étape

***

### **Étape 1 : Activer votre licence de produit**

Après avoir terminé l'installation :

1. Lancez le **PADS4 CTRL Center**.
2. Accédez au menu **Licenses**.
3. Saisissez la clé de produit fournie avec votre package d'installation.

> Besoin d'aide ? Consultez l'[Aide en ligne du PADS4 CTRL Center](#) pour des instructions détaillées sur la saisie d'une nouvelle clé de produit.

***

### **Étape 2 : Connectez-vous à PADS4 System Management**

1. Ouvrez le **PADS4 System Management** application.
2. Connectez-vous en utilisant vos identifiants.
   > **Connexion par défaut après l'installation :**\
   > Username: `user1`\
   > Password: `user1`

***

### **Étape 3 : Configurer les paramètres de la base de données SQL**

1. Dans l' **System Management** application, allez à **PADS Server Settings**.
2. Modifiez les paramètres pour **PADS4FIDS** pour le connecter à votre serveur SQL.

> PADS4FIDS nécessite SQL Server pour son stockage de données.

***

### **Étape 4 : Attribuer une licence utilisateur pour l'opérateur PADS4FIDS**

Pour permettre aux utilisateurs d'accéder à l'interface de l'opérateur FIDS :

1. Ouvrez le **PADS4 CTRL Center**.
2. Naviguez vers la section **Licenses**.
3. Attribuez une **PADS4FIDS Operator license** au compte utilisateur requis.

***

### **Étape 5 : Lancer l'opérateur PADS4FIDS**

Utilisez un navigateur web tel que **Google Chrome** ou **Microsoft Edge** (anciennement Internet Explorer) pour démarrer l'interface **PADS4FIDS Operator**.

Une fois lancé, vous pouvez commencer à configurer et à gérer le contenu de votre affichage FIDS.
