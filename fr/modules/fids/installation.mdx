Cette section décrit les étapes nécessaires pour installer et activer le **PADS4FIDS Module** dans le cadre de votre configuration PADS4 Foundation. Une fois que PADS4 Foundation a été installé, le produit PADS4FIDS est automatiquement inclus sur le même serveur.

## Installation étape par étape

***

### **Étape 1 : Activez votre licence de produit**

Après avoir terminé l'installation:

1. Lancez le **PADS4 CTRL Center**.
2. Naviguez vers le menu **Licenses** menu.
3. Saisissez la clé produit fournie avec votre package d'installation.

> Besoin d'aide ? Consultez l'[PADS4 CTRL Center Online Help](#) pour des instructions détaillées sur la saisie d'une nouvelle clé produit.

***

### **Étape 2 : Connectez-vous à PADS4 System Management**

1. Ouvrez l'application **PADS4 System Management**.
2. Connectez-vous en utilisant vos identifiants.
   > **Connexion par défaut après l'installation :**\
   > Username: `user1`\
   > Password: `user1`

***

### **Étape 3 : Configurer les paramètres de la base de données SQL**

1. Dans l'application **System Management**, allez à **PADS Server Settings**.
2. Modifiez les paramètres pour **PADS4FIDS** pour le connecter à votre serveur SQL.

> PADS4FIDS nécessite SQL Server pour son stockage de données.

***

### **Étape 4 : Attribuer une licence utilisateur pour PADS4FIDS Operator**

Pour permettre aux utilisateurs d'accéder à l'interface FIDS Operator :

1. Ouvrez le **PADS4 CTRL Center**.
2. Naviguez vers la section **Licenses**.
3. Attribuez une **PADS4FIDS Operator license** au compte utilisateur requis.

***

### **Étape 5 : Lancer le PADS4FIDS Operator**

Utilisez un navigateur web comme **Google Chrome** ou **Microsoft Edge** (anciennement Internet Explorer) pour démarrer l'interface **PADS4FIDS Operator**.

Une fois lancé, vous pouvez commencer à configurer et gérer le contenu de votre affichage FIDS.
