Pour commencer à utiliser le **PADS4FIDS Operator**, vous devez y accéder via votre navigateur web en utilisant l'URL correcte et vous connecter avec votre compte PADS4.

***

## Connexion au PADS4FIDS Operator

Dans la barre d'adresse de votre navigateur, saisissez l'URL suivante :

```
http://<Server Name>/PADS4/FIDS/Operator
```

Remplacez `<Server Name>` par le **hostname** ou **IP address** du serveur où PADS4FIDS est installé.

> **Optional:** Si votre serveur utilise un port spécifique, incluez-le dans l'adresse :

```
http://<Server Name>:<Port>/PADS4/FIDS/Operator
```

***

## Connexion

Une fois l'URL correcte saisie, la page **Sign In** apparaîtra.

1. Saisissez votre **Username** et **Password**.
2. (Facultatif) Activez **Keep me signed in** pour rester connecté lors des sessions futures.
3. Cliquez sur **Sign In** pour accéder à l'application.

> Si vous cliquez sur **"Can't access your account"**, une page d'aide s'ouvrira dans une fenêtre séparée pour vous aider avec les problèmes de connexion.

Après une connexion réussie, l'interface principale du **PADS4FIDS Operator** se lancera.

***

## Dépannage

### Pas d'identifiants ?

Si vous n'avez pas d'identifiants de connexion :

* Contactez votre administrateur.
* Assurez-vous qu'une **PADS4FIDS Operator license** a été attribuée à votre compte en utilisant l'application **PADS4 System Management**.

### Page introuvable ?

Si vous recevez un message **"Page Not Found"** error:

* Vérifiez que l'installation de PADS4FIDS est complète et fonctionne sur le serveur spécifié.
* Vérifiez les fautes de frappe dans l'URL ou un numéro de port incorrect.
