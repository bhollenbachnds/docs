Pour commencer à utiliser le **PADS4FIDS Operator**, vous devez y accéder via votre navigateur web en utilisant l'URL correcte et vous connecter avec votre compte PADS4.

***

## Connexion au PADS4FIDS Operator

Dans la barre d'adresse de votre navigateur, entrez l'URL suivante :

```
http://<Server Name>/PADS4/FIDS/Operator
```

Remplacez `<Server Name>` par le **nom d'hôte** ou l'**adresse IP** du serveur sur lequel PADS4FIDS est installé.

> **Optional:** Si votre serveur utilise un port spécifique, incluez-le dans l'adresse :

```
http://<Server Name>:<Port>/PADS4/FIDS/Operator
```

***

## Connexion

Une fois l'URL correcte saisie, la page de **Connexion** s'affichera.

1. Entrez votre **Nom d'utilisateur** et votre **Mot de passe**.
2. (Facultatif) Activez **Rester connecté** pour rester connecté lors des sessions futures.
3. Cliquez sur **Connexion** pour accéder à l'application.

> Si vous cliquez sur **"Vous ne pouvez pas accéder à votre compte"**, une page d'aide s'ouvrira dans une fenêtre séparée pour vous aider à résoudre les problèmes de connexion.

Une fois la connexion réussie, l'interface principale du **PADS4FIDS Operator** se lancera.

***

## Dépannage

### Pas d'identifiants ?

Si vous n'avez pas d'identifiants de connexion :

* Contactez votre administrateur.
* Assurez-vous qu'une **PADS4FIDS Operator license** a été attribuée à votre compte à l'aide de l'application **PADS4 System Management**.

### Page introuvable ?

Si vous recevez un **"Page Not Found"** error:

* Vérifiez que l'installation de PADS4FIDS est complète et en cours d'exécution sur le serveur spécifié.
* Vérifiez les fautes de frappe dans l'URL ou le numéro de port incorrect.
