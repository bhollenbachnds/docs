Pour commencer à utiliser **PADS4FIDS Operator**, vous devez y accéder via votre navigateur web en utilisant l'URL correcte et vous connecter avec votre compte PADS4.

***

## Connexion au PADS4FIDS Operator

Dans la barre d'adresse de votre navigateur, saisissez l'URL suivante :

```
http://<Server Name>/PADS4/FIDS/Operator
```

Remplacez `<Server Name>` par le **nom d'hôte** ou **l'adresse IP** du serveur où PADS4FIDS est installé.

> **Optional:** Si votre serveur utilise un port spécifique, incluez-le dans l'adresse :

```
http://<Server Name>:<Port>/PADS4/FIDS/Operator
```

***

## Connexion

Une fois l'URL correcte saisie, la page de **Connexion** apparaîtra.

1. Saisissez votre **Nom d'utilisateur** et votre **Mot de passe**.
2. (Facultatif) Activez **Rester connecté** pour rester connecté lors des sessions futures.
3. Cliquez sur **Se connecter** pour accéder à l'application.

> Si vous cliquez sur **"Impossible d'accéder à votre compte"**, une page d'aide s'ouvrira dans une fenêtre séparée pour vous aider avec les problèmes de connexion.

Après une connexion réussie, le **PADS4FIDS Operator** l'interface principale se lancera.

***

## Dépannage

### Pas d'identifiants ?

Si vous n'avez pas d'identifiants de connexion :

* Contactez votre administrateur.
* Assurez-vous qu'une **PADS4FIDS Operator license** a été attribuée à votre compte en utilisant l'application **PADS4 System Management**.

### Page introuvable ?

Si vous recevez un message **"Page Not Found"** error:

* Vérifiez que l'installation de PADS4FIDS est complète et fonctionne sur le serveur spécifié.
* Vérifiez les fautes de frappe dans l'URL ou un numéro de port incorrect.
