The **Welcome Message** tab allows administrators to customize the home page by updating the logo and welcome message displayed to users.

![Home Page Setup Screen](/mnt/data/image.png)

## Updating the Home Page

To personalize the home page:

1. Select **Add an Image** to upload or choose a new logo.
2. Enter a custom **welcome message** in the provided field.
3. Once all changes are made, click **Save** to apply them.

These updates will be reflected on the PADS4 Workspace user-facing home screen.
