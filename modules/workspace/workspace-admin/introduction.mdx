This manual provides technical guidance for the **PADS4 Workspace** and **CMS Admin Interface**. It is intended for use by technical staff responsible for the **basic setup**, **configuration**, and **troubleshooting** of the PADS4 Workspace and CMS modules.

## Accessing the Admin Interface

To access the Admin Interface:

1. Open a web browser.
2. Enter the following address:

   ```
   http://serverip:port
   ```

3. Login credentials have been provided to the technical team.

> In most cases, administrators can use **Single Sign-On (SSO)** to log in using their Windows credentials.

![PADS4 Login Screen](/mnt/data/image.png)

Once logged in, based on your access rights, you will be prompted to select the appropriate module to manage.

![PADS4 Module Selection Screen](/mnt/data/image.png)
