The **Additional Services** tab allows administrators to add and manage extra services—such as catering, equipment delivery, or room setup—within the **PADS4 Workspace Module**.

![Additional Services Screen](/mnt/data/image.png)

## Adding Additional Services

To add a new service, select the **Add Additional Service** button.

### Required Information

When creating a service, fill in the following fields:

- **Name**  
  The name of the service (e.g. Catering, IT Support).

- **Image**  
  An optional image to help users visually identify the service in the interface.

- **Type**  
  Choose between:
  - **Default** – For rooms and meeting spaces.
  - **Common Areas** – For services provided in areas like canteens or shared zones.

- **Description**  
  A short explanation of the service.

- **Allow Comment**  
  Enable this option if the user should be able to add a message when placing a request (e.g., dietary notes for catering).

- **Destination Email**  
  The email address where service requests will be sent.

### Additional Service List

This section allows you to specify individual service items (e.g., menu options or setup packages). For each item, you can define:

- **Name**
- **Information/Description**
- **Preparation Time**
- **Price**
- **Allow Comment**
