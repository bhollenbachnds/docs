# Profiles

**Profiles** define the rights and permissions that can be assigned to administrator and user accounts within the system.

## Creating a New Profile

To add a new profile:

1. Navigate to the **Profiles** section.
2. Click the **âž• (Add)** icon.
3. A pop-up window will appear prompting you to enter a name for the new profile.

![Create a Profile Dialog](/mnt/data/image.png)

4. Enter a name and click **Save**.

## Editing Profile Rights

After saving the profile name, you will be able to configure the associated rights.

![Profile Rights Settings](/mnt/data/image.png)

### Available Rights

- **Module Access**:
  - Digital Signage  
  - Workspace  
  - Central Administration (Admin Interface)

Depending on the module access granted to the profile, additional module-specific settings will become available for configuration.

At the bottom of the profile settings, you will find the **Relative Path** field. This defines the content library location specific to the selected profile.
