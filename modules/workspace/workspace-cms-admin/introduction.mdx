# Accessing the PADS4 Workspace & CMS Admin Interface

This manual provides information on accessing and navigating the PADS4 Workspace and CMS Admin Interface. It is intended for the technical team and covers the basic setup and troubleshooting of the PADS4 Workspace and CMS Modules.

## Accessing the Admin Interface

To open the Admin Interface:

1. Open a web browser.
2. Enter the following address in the address bar:

   ```
   http://serverip:port
   ```

3. Login credentials have been provided to the technical team.

> **Note:** In most cases, administrators can use the **Single Sign-On (SSO)** option to log in using their Windows credentials.

![PADS4 Login Screen](/mnt/data/image.png)

## Navigation After Login

Once logged in, and based on your access rights, you will be presented with a selection of modules or areas to manage. Choose the desired module to continue.
