title: "Introduction"
---

PADS4 Web Designer and CMS is a web-based solution for creating, managing, and publishing digital signage content. It is a core component of the PADS4 platform, designed to help users build visually engaging, responsive templates and messages that can be easily distributed across digital signage networks.

With the PADS4 Web Designer and CMS, users can create templates using videos, images, text, and more, then publish customized messages based on those templates. It supports streamlined workflows for both designers and end users by clearly separating layout design from content editing, providing maximum flexibility.

This manual offers a step-by-step guide to using PADS4 Web Designer and CMS for creating and publishing messages. It also explains how to manage categories, work with the media library, and use advanced features such as scheduling and message ordering.
