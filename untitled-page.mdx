  | Option                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
  | :--------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
  | Constant               | Use a static constant.<br />Manually specify the **Value** to be used as constant.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
  | Viewer variable        | Viewer variables will be replaced by the actual value at runtime.<br />Check out the [Viewer Variables page](https://pads4documentation.blob.core.windows.net/designer/Content/Working%20with%20play%20lists/Viewer%20Variables.htm) for more info about using Viewer Variables.                                                                                                                                                                                                                                                                                                                                                                 |
  | System Date            | Use the system data of the playback PC.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
  | System Long Time       | Use the system time of the playback PC in long time format.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
  | System Long Time (UTC) | Use the system time of the playback PC in long time and UTC time zone format.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
  | System Time            | Use the system time of the playback PC.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
  | System Time (UTC)      | Use the system time of the playback PC in UTC time zone format.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
  | Current Data           | Use the current column and current row of the active data provider.<br />This option is only applicable to page elements. The active data provider is determined by the [data provider properties](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20provider%20properties%20\(element\).htm) of the page element.                                                                                                                                                                                                                                                            |
  | Previous Data          | Use the current column and previous row of the active data provider.<br />This option is only applicable to page elements having switched on [data scrolling](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20scrolling%20properties%20\(element\).htm). It enables you to compare two consecutive data values. The active data provider is determined by the [data provider properties](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20provider%20properties%20\(element\).htm) of the page element. |
  | Other Column           | Use the current row, but a different column of the active data provider.<br />This option is only applicable to page elements. The active data provider is determined by the [data provider properties](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20provider%20properties%20\(element\).htm) of the page element. Manually select the **Column** to be used.                                                                                                                                                                                                            |
  | Other Data             | Use a specific data provider, column and row.<br />Manually select the **Data Provider**, **Column** and **Row** to be used.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
  | Other Data Scroll      | Use a specific data provider and column, but keep the row in sync with the row of your active data provider.<br />This option is only applicable to page elements having switched on [data scrolling](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20scrolling%20properties%20\(element\).htm). The active data provider is determined by the [data provider properties](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20provider%20properties%20\(element\).htm) of the page element.                |
  | Row Count              | Use the number of rows of a specific data provider.<br />Manually select the **Data Provider** to be used.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
  | Current Row Number     | Use the current row number of your active data provider.<br />This option is only applicable to page elements. The active data provider is determined by the [data provider properties](https://pads4documentation.blob.core.windows.net/designer/Content/Page%20elements/Page%20element%20properties/Data%20provider%20properties%20\(element\).htm) of the page element.                                                                                                                                                                                                                                                                       |
  | Page Name              | Use the name of the current page in your presentation.<br />This option is only applicable to pages and page elements.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
  | Page Number            | Use the sequence number of the current page in your presentation.<br />This option is only applicable to pages and page elements.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
  | Viewer Name            | Use the unique viewer name of the viewer that plays the presentation.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
  | Computer Name          | Use the computer name of the PC on which the presentation plays.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
  | Computer Language Name | Use the Microsoft Windows language name of the PC on which the presentation plays.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
</div>
