title: "BONVOY VMS Installation"
---

Download the BonVoy VMS Installer package from Finessetech or receive it from the authorized implementation partner. Run the installer as Administrator.

## **Installer Screen – Select Destination Location**

This screen allows the user to select the installation directory for BonVoy VMS.

<Frame>
  ![File Location](/images/Addons/Finessetech/BonvoyVMS/FileLocation.jpeg)
</Frame>

- Installation Path –Specifies the location where the BonVoy VMS files will be installed.

<Check>
  Keep the default installation path unless the customer has a dedicated application drive.
</Check>

Click Next to continue installation.

## Installer Screen – SQL Server Setup

**This screen configures the SQL Server connection required for BonVoy VMS**

<Frame>
  ![Installer SQL Server](/images/Addons/Finessetech/BonvoyVMS/Installer-SQLServer.png)
</Frame>

- **SQL Server**: Specifies the SQL Server instance name, for example `localhost`, `localhost\SQLEXPRESS`, or `SERVERNAME\INSTANCE`
- **Username**: Specifies the SQL authentication username. 
- **Password**: Specifies the SQL authentication password.\

<Check>
  The SQL user should have permission to create or update the BonVoy database.
</Check>

Click Test Connection and ensure the connection succeeds before proceeding.

Click on Next 

## Installer Screen – PADS4 / VMS Settings

This screen is used to configure the integration between BonVoy and the PADS4 CMS.

<Frame>
  ![Installer PADS4 VMS Setting](/images/Addons/Finessetech/BonVoyVMS/Installer-PADS4-VMSSetting.png)
</Frame>

- **PADS API Root URL**: The URL of the PADS4 server.
- **Username**: A PADS4 user account with **Admin** and **VMS** access.
- **Password**: The password for the PADS4 user account.
- **BonVoy Server URL**: The URL where the BonVoy web application is hosted.
- **BonVoy Server Port**: The IIS application port used by the BonVoy server.
