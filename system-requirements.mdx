title: "System requirements"
description: "PADS4 System Requirements from version 2021.1 and above"
---

## Minimum requirements PADS4 Server

- CPU: 4 cores
- RAM: 16 GB
- Full installation size 1.32 GB
- 64 bit OS: Windows 10, Windows 11, Windows Server 2016, Windows Server 2019, Windows Server 2022, Windows Server 2025

<Warning>
  We do not recommend to run PADS4 on Windows Server with the Datacenter edition
</Warning>

## Recommended requirements PADS4 Server

- CPU: 8 cores
- RAM: 24 GB
- Full installation size 1.32 GB
- 64 bit OS: Windows 10, Windows 11, Windows Server 2016, Windows Server 2019, Windows Server 2022, Windows Server 2025

<Warning>
  We do not recommend to run PADS4 on Windows Server with the Datacenter edition
</Warning>
