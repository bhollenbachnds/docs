As a CMS Designer, you can create categories to organize your templates. This helps users navigate the interface and quickly find relevant templates.

In this example, three categories already exist.

![Web Designer Categories Overview 1](/images/digital-signage/WebDesigner/WebDesignerCategoriesOverview-1.png)

## Create a New Category

1. Click **Create category**.
2. Enter a name for your category.
3. Click **Create**.

![Web Designer Categories Create](/images/digital-signage/WebDesigner/WebDesignerCategoriesCreate.png)

Your new category is now ready with three default items to help you get started:

- **Fullscreen media**: Publish full-screen messages.
- **Create template**: Build a new template from scratch for CMS users.
- **Select template**: Add an existing template to the category.
