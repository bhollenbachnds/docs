## Interface Overview

Once inside the Web Designer, the interface is divided into key areas: 

- **Canvas Area**: The main design space where you place and arrange elements.

  ![Create A Template3](/images/digital-signage/cms/create-a-template3.png)
- **Toolbar**: Provides access to drawing tools, media insertion, alignment, and layer management.

  <img
    src="/images/digital-signage/WebDesigner/WebDesignerElementToolBar.png"
    alt="Web Designer Element Tool Bar"
    title="Web Designer Element Tool Bar"
    style={{ width:"45%" }}
  />
- **Property Panel**: Located on the right, it displays configuration options for the selected element (e.g., size, color, alignment).

  <img
    src="/images/digital-signage/cms/create-a-template5.png"
    alt="Create A Template5"
    title="Create A Template5"
    style={{ width:"53%" }}
  />
- **Layer Panel**: Lists all objects on the template, allowing you to reorder, group, or delete elements.

  <img
    src="/images/digital-signage/cms/create-a-template4.png"
    alt="Create A Template4"
    title="Create A Template4"

  />

## Enlarge the Main Page

To zoom or maximize the workspace:

- Use the zoom slider or zoom icons located at the bottom-right of the canvas.

  <img
    src="/images/digital-signage/cms/create-a-template8.png"
    alt="Create A Template8"
    title="Create A Template8"
    style={{ width:"46%" }}
  />
- You can also press **Ctrl + Scroll** on your mouse to zoom in and out.

This allows for more precise placement of design elements.

## Add a Shape

1. From the toolbar, select the **Shape** tool.

   <img
     src="/images/digital-signage/cms/create-a-template9.png"
     alt="Create A Template9"
     title="Create A Template9"
     style={{ width:"30%" }}
   />
2. Choose a shape type (e.g., rectangle, circle).
3. Click and drag on the canvas to draw the shape.
4. Use the Property Panel to customize its color, size, border, and position.

   ![Create A Template10](/images/digital-signage/cms/create-a-template10.png)

## Add an Image / video

1. Select the **Image** tool from the toolbar.
2. Click on the canvas to place the image block.
3. In the Property Panel you will have three options: 4. **Empty item**: If you want the end user to add the content and you just provide the element itself. 4. **Content item**: You select the content from the content library.  **External item**: Image or video hosted somwhere else.

   ![Create A Template11](/images/digital-signage/cms/create-a-template11.png)

- When using **content item** an pop-up of your PADS4 content library will show up and allows you to pick the media you want.

  ![Create A Template12](/images/digital-signage/cms/create-a-template12.png)
- Resize and position the image as needed.

## Add Text

1. Click the **Text** tool in the toolbar and choose a regular text field or a date & time element.

   <img
     src="/images/digital-signage/cms/create-a-template14.png"
     alt="Create A Template14"
     title="Create A Template14"
     style={{ width:"30%" }}
   />
2. Click on the canvas to create a text box.
3. Type your desired text.
4. In the Property Panel, adjust the font, size, color, alignment, and effects.

   ![Create A Template13](/images/digital-signage/cms/create-a-template13.png)

## Save Your Template

When your layout is complete:

1. Click the **Save** button in the top toolbar.

   <img
     src="/images/digital-signage/cms/create-a-template15.png"
     alt="Create A Template15"
     title="Create A Template15"
     style={{ width:"56%" }}
   />
2. Your template will be saved and become available under the **Templates** section.
3. It can now be used as a base for creating messages.

   ![Create A Template16](/images/digital-signage/cms/create-a-template16.png)
