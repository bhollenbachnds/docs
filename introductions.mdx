PADS4 Designer provides a broad set of **page elements** that you combine on the canvas to build presentations. Each element type is optimized for a specific kind of content — text, media, live web data, PDFs, and more — so you can compose engaging signage without leaving Designer.

<Columns cols={2}>
  <Column>
    - **Move** — Reposition elements freely on the canvas.
    - **Text** — The fundamental building block of a presentation. Designer offers a wide range of text options for clear, concise messaging.
    - **Date and Time** — Add the current date or time to a presentation, with support for multiple time zones and cultures.
    - **Media** — Play dynamic multimedia content such as images, video, and audio.
    - **Adobe PDF** — Embed PDF documents so you can share detailed information directly on screen.
    - **Microsoft PowerPoint** — Import PowerPoint presentations into your PADS4 Designer projects for full compatibility and flexibility.
    - **Web element** — Embed live web content directly in a presentation for real-time information and dynamic updates.
    - **Shape** — Use rectangles, circles, arrows, and other shapes as visual accents or to organize content.
    - **Line** — Connect elements, create borders, or guide the viewer's attention.
  </Column>
