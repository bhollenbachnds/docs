---
title: "PADS4 Windows applications"
description: "Install the PADS4 Clients package (Designer, Scheduler, Agent, and System Management) on a Windows machine."
---

The PADS4 Clients installer bundles the Windows applications — Designer, Scheduler, Agent, and System Management — into a single package. Follow the steps below to install them on a Windows machine and connect the Viewer to your PADS4 Server.

## Before you begin

Make sure that:

- You have **local administrator rights** on the target Windows machine.
- You know the **PADS4 Server location** (DNS name or IP address) and the **Viewer name** to use.
- You're installing **PADS4 version 2025.1** or later.

## Step 1: Download PADS4 Clients

Download the latest installer from the official link: [PADS4 Clients download](http://go.pads4.com/?fwlink=1011).

## Step 2: Start the installation

1. Right-click **PADS4 Clients.exe** and choose **Run as administrator**.
2. Select your preferred **language** and continue.
3. Confirm that **PADS4 version 2025.1** or later is being installed.
4. Read the **License Agreement**, accept the terms, and select **Next**.

## Step 3: Choose which users to install for

Select which Windows users PADS4 Clients should be installed for, then select **Next**.

## Step 4: PADS4 Windows installation

Select the desired **Viewer configuration** from the dropdown, then select **Next**.

## Step 5: Help documentation (optional)

Choose whether to install the **online help file**. When installed, users can press **F1** in any PADS4 application to open the relevant help topic.

<Tip>
  Installing the help file is recommended — it makes in-product guidance available offline.
</Tip>

## Step 6: Configure Viewer settings

Enter the **Viewer name** and the **PADS4 Server location**, then select **Next**.

## Step 7: Desktop mode and User Account Control (UAC)

Configure how the Viewer runs on this machine:

- **Desktop mode** — choose whether PADS4 Clients should run in Desktop mode.
- **User Account Control (UAC)**:
  - Disabling UAC lets PADS4 Clients run with **administrator privileges** without extra prompts (recommended for dedicated signage machines).
  - If the signed-in user is **not** a member of the local Administrators group, specify an administrator account that will run the Viewer.

## Step 8: Shortcuts and installation folder

- Select which **shortcuts** to create.
- Choose the **installation destination folder**.
- Select **Install** to begin.

## Step 9: Complete the installation

Wait for the installer to finish. When it displays **"PADS4 Clients has been successfully installed"**, the Windows applications are ready to use.
