## **Activating PADS4 with a Product Key**

To activate your **PADS4 product**, follow the steps below using the provided **PADS4 Product Key**.

## Step 1: Open PADS4 CTRL Center

- After installing **PADS4**, browse to the **PADS4 Web Portal.**

<Tip>
  If you are going to (re-)activate a PADS4 installation which is already in use, using the su\\root is not mandatory, any user with the **System Management + License Management** license can re-activate the installation by going to **Licenses** at the left hand menu.
</Tip>

- Login with the **su\\root** user.
- At the top of the screen choose the "**pads**" domain.

![Root Domain](/images/install-pads4/activating-pads4/root-domain.png)

- Now on the left hand menu, "**Licenses**" will appear.

<img
  src="/images/install-pads4/activating-pads4/root-licenses.png"
  alt="Root Licenses"
  title="Root Licenses"
  style={{ width:"100%" }}
/>

- Click on licenses and press "**Activate"**

![Root License2](/images/install-pads4/activating-pads4/root-license2.png)

- If an internet connection is unavailable, an **offline activation** method will be displayed.
  - Follow the instructions provided on-screen.
  - After receiving the **License File**, click **"Upload License File"** and select the file.
  - Complete the activation process by following the wizard.

Once activation is complete, your **PADS4 installation is ready for use.**
