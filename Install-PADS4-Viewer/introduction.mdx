The PADS4 Viewer runs on a wide range of media players and displays. To help you deploy with confidence, PADS4 **certifies** specific hardware devices — each one is tested end-to-end with the current PADS4 version, so you know exactly which features are supported and what to expect during setup.

This section is your reference for every certified device. For each device you'll find:

- **Minimum PADS4 version** required.
- **Setup steps** to install and configure the Viewer.
- **Supported features** specific to that hardware.

## Choose a device

Use the pages in this section to find the model you're deploying. Each page walks you through:

1. **Prerequisites** — server-side configuration and any additional software you need.
2. **Device configuration** — first-time access, licensing, and required settings.
3. **Viewer installation** — how the device downloads and connects to your PADS4 Server.
4. **Supported features** — what works out of the box on that device.

<Tip>
  If your device isn't listed here, it may still work with the PADS4 Android Launcher, ChromeOS Flex or Windows Viewer, but only certified devices are covered by the tested configurations in this section.
</Tip>
