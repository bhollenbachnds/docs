
The **Flickr feed provider** in **PADS4** allows you to **retrieve images** from **Flickr**, an online photo-sharing platform.

---

## **Steps to Add/Modify a Flickr Feed Connection in PADS4**

### **1. Start the Data Provider Wizard**
- Open the **Data Provider Wizard** by following the instructions in **Adding a Data Provider** or **Editing a Data Provider**.
- Select **Flickr Feed** as the **provider type**.

### **2. Specify Flickr Search Arguments**
- Enter the **Flickr search terms** to **retrieve images** matching specific keywords.
- You can filter images based on:
  - **Tags** (keywords to find relevant photos).
  - **User ID** (retrieve images from a specific Flickr user).
  - **Date range** (filter photos by upload date).
  - **Sort order** (most recent, most relevant, etc.).
- Click **Next** to continue.

### **3. Complete the Wizard**
- Click **Finish** to finalize the **Flickr feed data provider**.
- The **PADS4 Viewer** will retrieve and display **Flickr images** in real-time.

---

## **Important Considerations**
- **Data Server Access:**  
  - If using a **PADS4 Data Server**, ensure that **Flickr is accessible** from the **server's point of view**.  
  - If **not using a Data Server**, verify that each **PADS4 Designer and Viewer** has an **active internet connection**.

By following these steps, you can successfully **connect PADS4 to Flickr** and **display dynamic image content** in your **digital signage presentations**.
