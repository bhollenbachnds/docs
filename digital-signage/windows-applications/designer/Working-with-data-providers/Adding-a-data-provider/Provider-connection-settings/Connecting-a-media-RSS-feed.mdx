
Media RSS (**Rich Site Summary**) extends the standard **RSS web feed format** to support multimedia content, such as **videos and images**.

---

### **Steps to Add or Modify a Media RSS Feed Connection**  

#### **1. Start the Data Provider Wizard**  
- Open the **Data Provider Wizard** following the instructions in **Adding a Data Provider** or **Editing a Data Provider**.  
- Select **Media RSS Feed** as the provider type.  

---

#### **2. Specify the Media RSS Feed URL**  
- Enter the **URL (URI)** of the Media RSS feed.  
- If required, click **Preview** to verify the feed’s output.  

---

#### **3. Configure Authentication (If Required)**  
- If the Media RSS feed **requires login credentials**, enable **Authorization Required**.  
- Enter the **User ID, Password, and (if necessary) a Domain**.  

---

#### **4. Configure User-Agent (If Required)**  
- Some servers may require a **specific User-Agent string** to allow access.  
- If needed, enter the required **User-Agent identification string**.  

---

#### **5. Finalize and Save the Connection**  
- Click **Next** to proceed.  
- Complete the setup by finishing the **Data Provider Wizard**.  

---

### **Important Considerations**  
- **Using a PADS4 Data Server**  
  - If a **PADS4 Data Server** is in use, ensure that the **Media RSS feed is accessible** from the **server’s point of view**.  
  - If **not using a Data Server**, confirm that **PADS4 Designer and PADS4 Viewers** have direct access to the Media RSS feed via the internet.  

By following these steps, PADS4 will successfully retrieve and display **multimedia content from a Media RSS feed**.
