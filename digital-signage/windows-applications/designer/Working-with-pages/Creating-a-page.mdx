
You can easily add a **new blank page** to your presentation when you want to start designing fresh content.

---

## **To Add a New Blank Page to Your Presentation**

### Option 1: Using the Ribbon Menu
1. Go to the **Page** menu.
2. Click on **New Page**.

### Option 2: Using the Thumbnails Window
1. **Right-click** on any existing page thumbnail in the **Thumbnails window**.
2. Select **New Page** from the context menu.

---

## **Remarks**
- New pages are **always added to the end** of your presentation.
- Once added, you can rearrange the page order if needed by moving pages within the Thumbnails window.
