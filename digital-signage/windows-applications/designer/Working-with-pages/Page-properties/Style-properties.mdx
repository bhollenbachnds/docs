
The **Style Properties** define how a page will **visually appear and behave** during playback.

---

## **Background Color**

### **Color**
- Determines the **background color** of the page.
- You can specify:
  - A **fixed color**
  - A **gradient** (two or three colors)
  - A **texture or pattern**  
- For full options, see **Using the Color Picker**.
- **Note**: Transparency is **not supported** for page background colors.
- **Important**: If a **master page** is applied to this page, **the background color is inherited from the master page** and **cannot be set individually**.

---

## **Optional Device LED Light Bar**

### **LED Color**
- Allows setting a **LED bar color** when this page becomes visible.
- **Note**: This works only on displays with an integrated **LED bar**.
- Supported colors depend on the **device type** â€” refer to device specifications and **PADS4 certified devices documentation**.
- When enabled, the color will **activate** on the display when the page is shown.

---

## **Master Page Visibility**

### **Show the Master Page on the Background of this Page**
- When **enabled**, the **master page elements** are displayed **behind** the current page's content.
- **Only available** if a **master page** exists in the presentation.
- If disabled, this page can have a **unique layout**, different from the master page.
- **Note**: Master page elements (like videos) will **pause** when hidden and **resume** when shown again.

> **Tip**: Disabling the master page for specific pages lets you design **custom layouts** where needed.

---

## **Page Background Image**

### **Picture File**
- Choose a background image:
  - **Picture from Content Library** (recommended, for automatic synchronization across the system).
  - **Picture from File** (external file, requires manual distribution to all viewers).

> **Tip**: Always prefer using the **Content Library** to ensure consistency and synchronization across all PADS4 devices.

---

### **Picture Size**
- Controls how the background image fits within the page.

| Value            | Description                                                                                                                                      |
|------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| **Original Size** | Displays the image at its **native resolution**. If the page is smaller, only part of the image will show; if larger, the image won't stretch. **Best quality and performance**. |
| **Stretched**     | Resizes the image to **fully cover** the page, regardless of the original image size.                                                             |
