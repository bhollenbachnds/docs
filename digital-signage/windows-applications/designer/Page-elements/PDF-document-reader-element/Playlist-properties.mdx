
These properties define the **PDF files** displayed by the **PDF Document Reader**. You can **select a single PDF** or create a **playlist** of multiple PDFs that will be displayed sequentially.

---

## **Managing the Play List**  

- **Add** new PDF files.  
- **Edit** or rearrange play list items.  
- **Change the sequence** order of files.  
- **Remove** unwanted files.  
- **Navigate** through play list items using navigation buttons.  

The **total duration** represents the **sum of all individual durations** for PDF items.

---

## **PDF Play List Item Options**  

### **Type**  

Defines the **source** of the PDF file.  

| **Option** | **Description** |
|------------|----------------|
| **PDF from Content Library** | Uses a reference to a **PDF stored in the PADS4 content library** (recommended). Automatically synchronized across all PCs. |
| **PDF from File** | Uses a **PDF stored outside the PADS4 content library**. Not automatically synchronized—must be manually distributed. |
| **No PDF** | Creates a **pause** in the playlist (no content displayed). |

### **Content**  

- Defines the **PDF file path or reference** (depending on the selected type).  
- Browse the **Content Library** or **local storage** to select a file.  

### **Duration**  

- Sets the **display time (in seconds) per page** of the PDF.  
- **Total duration** = (Duration per page) × (Number of pages displayed).  

### **Effect**  

- Determines the **transition effect** applied when a new PDF appears.  
- Set the **duration of the effect (in seconds)**.  

---

## **Clipping & Page Selection**  

### **Clip Document**  

- Enables **partial PDF display** by selecting **specific pages**.  

### **First Page**  

- Sets the **first page** of the PDF to be displayed.  

### **Last Page**  

- Defines the **last page** to be displayed.  

---

## **Additional Notes**  

- **Only references** to PDF files are stored in the presentation, **not the actual files**.  
- **If a PDF is updated**, the latest version will be displayed automatically.  
- **Play list properties are disabled** if the element is connected to a **real-time data source**.  

By using these **playlist settings**, you can create a **dynamic PDF-based presentation** with **custom durations, effects, and page selection**.
