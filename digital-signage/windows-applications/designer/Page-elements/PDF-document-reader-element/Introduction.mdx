
The **PDF Document Reader** allows you to **embed and display PDF files** within your presentation. PDFs can be shown in **original size** or **resized** to fit your layout. Additionally, pages can be **automatically scrolled** through with a set duration.  

---

## **Key Features & Limitations**  

- **Supports most PDF formats** (except **3D models**).  
- **Dynamic content support** by connecting to an **external data source**.  
- **Effects and transitions** can be applied for smoother visual changes.  

---

## **Properties Overview**  

| **Property Section**       | **Description**  |
|-----------------------------|------------------|
| **Alignment**               | Adjust **horizontal and vertical alignment** of content. |
| **Alternate Options**       | Configure how the element switches between **playlist items**. |
| **Background Color**        | Set a **background color** for the PDF element. |
| **Border**                  | Define a **border** around the element. |
| **Data Provider**           | Connect to an **external data source** for real-time PDF updates. |
| **Data Source Text Conversion** | Manage how the **element handles multi-line text**. |
| **Default Duration**        | Set a **default display time** for playlist items. |
| **Default Transition Effect** | Apply **incoming transition effects** to content. |
| **Flip**                    | Flip the **content horizontally or vertically**. |
| **Identification**          | Assign a **unique identifier** to the element. |
| **Play List**               | Define **which PDF files** will be displayed. |
| **Play List Options**       | Set **how playlist items** are played. |
| **Position**                | Adjust **placement** on the page. |
| **Proof of Play**           | Log **proof of play** for reporting. |
| **Property Bindings**       | Bind properties to an **external data source**. |
| **Reflection**              | Add a **reflection effect** to the content. |
| **Rotation**                | Rotate the **PDF element at an angle**. |
| **Rules**                   | Define **rules for automated visibility**. |
| **Size**                    | Set the **width and height** of the element. |
| **Skew**                    | Apply **skew effects** to the content. |
| **Transparency**            | Adjust the **transparency level** of the element. |
| **Visibility**              | Control if the **element is visible during playback**. |

---

## **Additional Notes**  

- **PDFs can be dynamically updated** using an external data source.  
- **Playback and scrolling** can be automated using **playlist options**.  
- **Content transitions and effects** can enhance the display experience.  

By using the **PDF Document Reader**, you can seamlessly integrate **document-based content** into your presentation with **dynamic updates and smooth transitions**.
