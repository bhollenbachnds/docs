
The **Advanced Rich Text Editor** allows users to format, edit, and enhance text content with various tools, including data-driven content, rules, and visual elements.

---

### **Editor Window**  
- **Save & Close** → Saves changes and closes the editor.  

---

### **Clipboard Functions**  
- **Cut / Copy / Paste / Delete** → Standard clipboard operations.  
- **Undo / Redo** → Allows reverting or restoring previous actions.  

---

### **Font Formatting**  
- Change **font type, size, style** (bold, italic, underline, strikethrough).  
- Adjust **text color, background color, and highlight text**.  

---

### **Paragraph Formatting**  
- Adjust **text alignment** (left, center, right, justified).  
- Set **line spacing** and indentation.  
- Add **bullet points or numbered lists**.  

---

### **Insert Options**  

| **Option**  | **Description**  |
|-------------|----------------|
| **Symbol**  | Inserts special characters or symbols. |
| **Picture**  | Inserts an image from a **file** or **PADS4 Content Library**. (**Note:** Files from external locations are **not synchronized**.) |
| **Table**  | Inserts a table and allows specifying the number of **rows** and **columns**. |
| **Formatted Text**  | Inserts dynamic **date/time** values that update automatically during playback. |
| **Data Provider**  | Inserts real-time **text from a PADS4 Data Provider**. The text updates when the source data changes. |
| **Viewer Variable**  | Inserts text linked to a **Viewer Variable**, updating automatically when the variable value changes. |
| **Rule**  | Defines **conditional behavior** for text. If linked to a **Data Provider**, it allows dynamic formatting (e.g., hiding text, changing fonts, etc.). |

---

### **Find & Replace**  
- **Find** → Search for specific text within the document.  
- **Replace** → Replace text with new content.  

---

### **Remarks**  
- **Data-Driven Content:** When using a **Data Provider** or **Viewer Variable**, the inserted content updates dynamically.  
- **Rules & Formatting:** Allows text styling based on **real-time conditions**.  
- **Tables & Images:** Supports structured content and media embedding.
