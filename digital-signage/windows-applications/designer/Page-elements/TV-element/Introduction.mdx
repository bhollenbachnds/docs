
The **Television element** enables the display of **live TV video and audio** from a **TV card** installed in the **Viewer PC**.  

---

## **Requirements**  
- A **TV card** must be installed on the **Viewer PC** to use this feature.  
- The TV signal source must be **properly configured** for playback.  

---

## **Properties Overview**  

### **Alignment**  
- Defines **horizontal and vertical alignment** of the TV content.  

### **Alternate Options**  
- Determines how the element **switches between playlist items**.  

### **Background Color**  
- Sets the **background color** of the TV element when no content is displayed.  

### **Border**  
- Configures the **border settings** for the element.  

### **Data Provider**  
- Connects the element to an **external data source** for real-time TV channel selection.  

### **Data Source Text Conversion**  
- Defines how **multi-line text** is processed if connected to an external source.  

### **Default Duration**  
- Specifies the **default duration** for each channel in the playlist.  

### **Identification**  
- Assigns a **unique name** to the element for reference.  

### **Playlist**  
- Defines the **TV channels** to be displayed.  

### **Playlist Options**  
- Determines how the **playlist of channels** is played and switched.  

### **Position**  
- Sets the **position** of the element within the page layout.  

### **Proof of Play**  
- Logs playback information for **performance tracking and reporting**.  

### **Property Bindings**  
- Binds the element’s properties to an **external data source** for dynamic adjustments.  

### **Reflection**  
- Applies a **reflection effect** to the TV content.  

### **Rotation**  
- Allows **rotation** of the TV element to a specified angle.  

### **Rules**  
- Defines **conditions** to **automatically modify** the TV element’s behavior.  

### **Shadow**  
- Adds a **shadow effect** to the TV display.  

### **Size**  
- Defines the **width and height** of the TV element.  

### **Skew**  
- Adjusts the **skew angle** for a stylized appearance.  

### **Transparency**  
- Sets the **level of transparency** for the element.  

### **Visibility**  
- Determines whether the TV element **remains visible** during playback.  

---

## **Remarks**  
- This section **covers only TV-specific properties**.
