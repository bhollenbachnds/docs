
These properties define the **image separator** placed between scrolling text items.

---

## **Separator Options**  

### **Content Library**  
- Uses an image from the **PADS4 content library** (recommended).  
- Files are **automatically synchronized** across all connected PCs.  
- Relative references ensure files remain accessible **even if the library is moved**.

### **File Name**  
- Uses an image stored **outside** the PADS4 content library.  
- Files are **not synchronized automatically**, requiring **manual distribution**.  
- Absolute file references mean the image **must exist** on the playback PC.  

**File Selection:**  
- Click **Browse** to select an image from the **Content Library** or local **file system**.  
- If using the **File Name** option, the file path must be **fully specified**.

---

## **Tips & Best Practices**  
- **Separators visually indicate** the start of a **new scrolling text item**.  
- If the separator image is **updated**, the presentation will **automatically use** the latest version.
