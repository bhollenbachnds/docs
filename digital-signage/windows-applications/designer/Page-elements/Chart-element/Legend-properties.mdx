
These properties define how the legend of a chart is displayed. A legend explains, for each data series, how it can be identified in the chart.

---

#### **Legend**

**Show Legend**  
Enable or disable the display of a legend in the chart.

**Position**  
Select a predefined position for the legend within the chart area.

**Show Backplane**  
Enable or disable a background frame for the legend.

**Background Color**  
Select a background color for the legend frame. This can be a fixed color, gradient, texture, or pattern. (See *Using the color picker* for details.)

**Border Color**  
Select a border color for the legend frame. Supports fixed colors, gradients, textures, or patterns.

**Border Width**  
Set the thickness of the border in pixels.

---

#### **Font (Legend Items)**

**Style**  
Shows the current font and font style used for legend items. Click to open the font dialog and change the font.

**Color**  
Select a font color for the legend items. Supports fixed colors, gradients, textures, or patterns.

**Shadow**  
Select a shadow color for the legend items. Supports fixed colors, gradients, textures, or patterns.

**Shadow Depth**  
Set the distance (in pixels) between legend items and their shadows.

---

#### **Header**

**Text**  
The title displayed in the legend frame.

**Font**  
Shows the current font and font style used for the legend title. Click to open the font dialog and change the font.

**Color**  
Select a font color for the legend title. Supports fixed colors, gradients, textures, or patterns.
