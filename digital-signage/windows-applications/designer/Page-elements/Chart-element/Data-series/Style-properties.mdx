
The **Style properties** in the **Data Series Wizard** determine the **appearance and visual formatting** of your data series, including **lines, markers, and custom images**.

---

## **Marker Options**

Depending on the **Chart Type** selected, you can configure various marker-related style settings:

### **Line Thickness (for Line Charts only)**

| **Property**            | **Description**                                                       |
|------------------------|-----------------------------------------------------------------------|
| **Line Thickness**      | Set the **width of the line** (in pixels) used to represent the data. |

---

## **Marker Settings**

### **Predefined Marker**

| **Property**            | **Description**                                                                                             |
|------------------------|-------------------------------------------------------------------------------------------------------------|
| **Use Predefined Marker** | Enable this option to use standard markers as configured below.                                           |
| **Marker**              | Choose a **marker shape** (e.g., square, circle, triangle, diamond) from the available set.                 |
| **Marker Size**         | Set the **size of the markers** (in pixels).                                                                |
| **Border Width**        | Set the **border thickness** (in pixels) around the markers.                                                |
| **Marker Color**        | Choose a color for the markers. You can use solid colors, gradients, patterns, or textures.                 |
| **Marker Border Color** | Choose a color for the marker border. Options include solid colors, gradients, patterns, or textures.       |

---

### **Image as Marker**

| **Property**              | **Description**                                                                                          |
|--------------------------|----------------------------------------------------------------------------------------------------------|
| **Use Image as Marker**   | Enable this option to **use an image as a marker** instead of a shape.                                    |
| **Image Source**          | Specify the image file. You can **retrieve images from the PADS4 Content Library** or specify a **local disk file** (provide the full path). |

---

## **Remarks**

- **Markers** help **highlight individual data points** within the series.
- **Only references** to image files are stored in the presentation — **not the actual image content**.  
  ➤ **If the source image is updated**, the updated version will be used during playback automatically.
- The **color picker** allows you to choose from **fixed colors, gradients, textures, or patterns** for both markers and their borders.
- If you use **image markers**, ensure that the source image file remains available and synchronized for consistent presentation playback.
