
The **Options properties** in the **Data Series Wizard** allow you to set **chart-type-specific settings** for the visual behavior and styling of your data series. These options depend on the **Chart Type** selected for the data series.

---

## **Options Overview by Chart Type**

### **Bar Chart Options**

| **Property**           | **Description**                                                                                             |
|-----------------------|-------------------------------------------------------------------------------------------------------------|
| **Style**              | Choose one of the available **bar styles** (e.g., flat, 3D, stacked).                                       |
| **Bar Width**          | Define the **width of the bars** as a percentage relative to the chart's width.                           |
| **Group with Serie**   | Enable this option to **group or combine data** when working with **multiple data series** in the same chart.|

---

### **Pie Chart Options**

| **Property**           | **Description**                                                                                             |
|-----------------------|-------------------------------------------------------------------------------------------------------------|
| **Style**              | Choose one of the available **pie chart styles** (e.g., standard, exploded).                                |
| **Doughnut Size**      | Define the **size of the doughnut hole** as a percentage relative to the chart, when using doughnut-style pies.|

---

### **Bubble Chart Options**

| **Property**                | **Description**                                                                                       |
|----------------------------|-----------------------------------------------------------------------------------------------------|
| **Maximum Bubble Size**     | Define the **maximum size of bubbles** as a percentage relative to the chart area.                   |

---

### **Stock Chart Options**

| **Property**          | **Description**                                                                                                                      |
|----------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| **Price Up Color**    | Set the **color to represent an increase** in stock value. You can choose solid colors, gradients, textures, or patterns.             |
| **Price Down Color**  | Set the **color to represent a decrease** in stock value. You can choose solid colors, gradients, textures, or patterns.             |

---

## **Notes**
- The available options **depend on the selected chart type**.
- Colors can be customized extensively using gradients, patterns, or solid colors via the **color picker**.
- **Grouping** series (for bar charts) is useful when displaying multiple series in a compact, comparative layout.
- **Pie chart doughnut size** controls how much "empty space" is inside a doughnut-style pie chart.
