
These properties control whether users can **interact** with the content of the external application embedded in your presentation.

---

### **Behaviour**

- **Allow User Interactivity**  
  Enable or disable **user interaction** with the external application's content.  
  When enabled, users can interact with elements inside the external application, such as:
  - Clicking on **hyperlinks**
  - Pressing **interactive buttons**
  - Navigating within third-party software launched by the External Application element  

---

### **Remarks**

- Disabling this option makes the external content **view-only**, preventing any interaction.
- Enabling this option is useful when you want to allow **interactive use** of embedded software or web apps as part of your presentation.
