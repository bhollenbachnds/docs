
These properties control whether users can **interact** with the Flash content displayed in the Flash player element.

### **User Interactivity**

- **Allow User Interactivity**:  
  - **Enable** this option to allow users to interact with the Flash content.  
  - This can include clicking on **hyperlinks**, **buttons**, or other interactive components that are part of the Flash file.  
  - **Disable** this option if you want the Flash content to be played without allowing any user input or interaction.
