
The **Font Dialog Box** allows users to **select a font** and configure various **text presentation properties**.

---

## **Search**  
- Enter a **font name** to find a specific font.  
- The **Fonts list** highlights the first matching font.  

---

## **Installed Fonts**  
- Displays a **list of available fonts**.  
- Fonts installed through **PADS4 System Management** are available on all **PADS4 HTML5 Viewers**.  
- Additional fonts can be added and automatically distributed across **PADS4-managed systems**.  

---

## **Font Customization Options**  

### **Spacing**  
- Adjust the **vertical line spacing** between text lines.  

### **Size**  
- Define the **font size** for display.  

### **Typeface**  
- Choose the **desired font style**.  

### **Text Formatting Options**  
- Apply **underscore, overline, strikethrough, or underline** styles to text.  

### **Preview**  
- Displays a comparison of the **current font settings** and the **newly selected font properties**.
