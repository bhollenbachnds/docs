
The **Default Duration** properties define how long each item in the **play list** of your element will be displayed when no specific duration is assigned to individual items.

---

### Default Duration

- **Duration**  
  The **default display time** (in seconds) for each play list item.  
  - This value applies when no custom duration is set for individual play list items.

---

### Remarks

- When **data scrolling** is enabled and you choose to **Scroll using play list durations**, this **Default Duration** will automatically determine **how long each data row is displayed** before scrolling to the next row.
- The **Default Duration** ensures a consistent timing for items when individual durations are **not specified**.
- If you **manually define durations** for specific play list items, those specific settings will **override** the default duration for those items.
