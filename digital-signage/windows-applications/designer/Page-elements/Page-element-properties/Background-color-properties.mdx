
These properties define the background color of an element.  

### Background Color

- **Color**:  
  You can select a background color for your element.  
  Options include:  
  - **Fixed color**: A single, solid color.  
  - **Gradient**: A combination of two or three colors blended together.  
  - **Texture or pattern**: A graphical texture or repeating pattern.  

Refer to **"Using the color picker"** for detailed instructions on selecting and customizing these options.  

---

### Remarks

- If the element **supports transparency**, you may also choose to make the background color **fully transparent**.
- In some cases, the **background color** will only be visible when:
  - There is **no content** displayed in the element.
  - The content does **not fully cover** the background area of the element.
