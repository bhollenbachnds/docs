
These properties allow you to display the content of your element **with a reflection effect**, adding visual depth and professionalism to your design.

---

### **Reflection Settings**

| **Property**                 | **Description**                                                                                       |
|----------------------------|---------------------------------------------------------------------------------------------------|
| **Reflection**              | Select from predefined reflection styles or manually adjust the reflection settings below.          |
| **Depth**                   | Size of the reflection **relative to the element size**. **100%** means same size as the element. |
| **Offset**                  | **Distance** (in pixels) between the element and its reflection.                                    |
| **Transparency Level**      | Defines **how transparent** the reflection is: Specify **Start** (near element) and **End** (farther from element). Values range from **100%** (opaque) to **0%** (fully transparent). |

---

### **Tip**

For a smooth **dissolve effect**, set:
- **Start Transparency Level**: 100%
- **End Transparency Level**: 0%

This will make the reflection fade out gradually from the element downward.
