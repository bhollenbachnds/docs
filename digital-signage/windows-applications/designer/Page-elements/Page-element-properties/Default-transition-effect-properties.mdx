
These properties define the **default incoming transition effect** that will be applied to items in the **play list** of your element when they appear.

---

### Default Transition Effect

- **Effect**  
  Select the type of **incoming transition effect** from the drop-down list.  
  - This effect will be applied automatically when each play list item is displayed.

- **Duration**  
  Choose one of the predefined durations or manually enter the duration (in seconds) for how long the transition effect should last.

---

### Remarks

- The **Default Transition Effect** applies to all play list items **unless a specific effect is assigned to an individual item**.
- **Caution**: It is **not recommended** to use transition effects in combination with **blinking**, as this may result in **conflicting behaviors** and **unpredictable visual results**.
