
These properties enable you to place the content of your element at a certain angle, creating a skewed or slanted effect.

### **Skew**

- **Skew**  
  Click the drop-down button to select a predefined skew style or manually set custom skew angles.

- **Horizontal**  
  The horizontal skew angle, in degrees.

- **Vertical**  
  The vertical skew angle, in degrees.

---

### **Remarks**

- Skew angles are specified in degrees and counted **clockwise**.
- Skewing an element distorts its content along the specified axes, allowing creative visual effects or perspective-like appearances.
