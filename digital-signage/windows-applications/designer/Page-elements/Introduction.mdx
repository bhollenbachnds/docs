
PADS4 Designer provides a **wide range of built-in page elements** to create professional presentations. Each element type has **specific properties**, along with **common properties** shared by all elements.

In addition to built-in elements, **Add-On elements** can be installed and used just like standard elements, offering extended functionality.

---

## **Page Elements Overview**

| **Page Element**                  | **Description**                                                                                      |
|----------------------------------|------------------------------------------------------------------------------------------------------|
| **Picture**                      | Display logos, photos, or other images.                                                              |
| **Mixed Media**                  | Display a playlist of videos and images.                                                             |
| **Text**                        | Add plain text.                                                                                      |
| **PowerPoint**                   | Embed Microsoft PowerPoint presentations.                                                            |
| **Rich Text**                   | Add formatted text.                                                                                  |
| **Video/Sound**                  | Playback of video and/or sound files.                                                                |
| **Web Page**                    | Display web content or Internet pages.                                                               |
| **Scrolling Text**               | Add ticker tapes or continuously moving text.                                                        |
| **Formatted Text**               | Show dynamic values like date, time, or page numbers.                                                |
| **Shape**                        | Add shapes such as squares, rectangles, circles, or ovals.                                            |
| **Line**                         | Draw lines.                                                                                          |
| **Chart**                        | Display real-time 2D or 3D charts and graphs.                                                        |
| **Table**                        | Present real-time data in a tabular format.                                                         |
| **TV Element**                   | Show TV channels.                                                                                    |
| **Flash Player**                 | Play Flash content.                                                                                  |
| **PDF Document Reader**          | Display PDF files directly on the page.                                                              |
| **Zone Viewer**                  | Create a separate content zone controlled by its own scheduler.                                      |
| **Presentation in Page**         | Embed another PADS4 presentation within the current page.                                            |
| **Interactive Button**           | Add buttons for interactive navigation within presentations.                                         |
| **External Application**         | Launch external software directly from the presentation.                                             |
| **RS-232 Communication**         | Control external devices via RS-232 commands.                                                        |

---

## **Add-On Elements Overview**

The following add-ons are included in a **standard PADS4 installation**:

| **Add-On Element**               | **Description**                                                                                     |
|----------------------------------|-----------------------------------------------------------------------------------------------------|
| **Progress Bar**                  | Displays a bar indicating the remaining time before the next page is shown.                         |
| **DVD Media Player**              | Plays DVD and other media content.                                                                 |
| **Advanced TV**                   | Display live TV channels with advanced control.                                                    |
| **QR Code**                       | Generate and display QR codes (for scanning with smartphones).                                     |
| **Page Selector**                 | Show a specific page based on the content of a text file (e.g., "10" to show page 10).             |
| **Custom (SDK)**                  | Create custom elements via the **PADS4 SDK** for specific functionality and appearance.            |

---
