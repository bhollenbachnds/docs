
These properties control the **volume level and balance** for media playback.  

---

## **Volume Settings**  

### **Volume Level**  
- Adjust the **volume** using the **slider** or manually **enter a percentage value**.  
- **100% = Default Full Volume**  
- **Lower values reduce sound output**.  

### **Balance**  
- If **two speakers** are connected, this setting determines how the **audio is distributed** between the left and right speakers.  
- Adjust to shift sound towards the **left** or **right** speaker as needed.  

---

### **Remarks**  
- The **volume and balance settings** apply to **all media files** played by this element.  
- These settings ensure **consistent audio levels** across multiple files in a **play list**.
