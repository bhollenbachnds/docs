
These properties define the **proxy server settings** for accessing web pages. By default, **no proxy server** is used, and the system will attempt a **direct connection** to the Internet.  

---

## **Proxy Server Configuration**  

### **Enable Proxy Server**  
- **Turn on/off** the use of a **proxy server** for retrieving web content.  

### **Server Address**  
- Enter the **computer name, DNS name, or IP address** of the **proxy server**.

### **Port Number**  
- Defines the **port number** used for communication with the proxy.  
- Default: **80** (common for HTTP traffic).

### **Authentication Credentials (Optional)**  
- **User ID**: Username required to authenticate with the proxy server.  
- **Password**: Password associated with the provided username.  

---

## **Additional Notes**  

- If no **proxy server** is configured, the system will attempt a **direct internet connection**.  
- Authentication details are **optional** and only required if the proxy server enforces **login credentials**.
