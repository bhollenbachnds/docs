
With rotation functions, you can **place page elements at any desired angle** to enhance the design of your presentation.

---

## How to Rotate One or More Page Elements

### Option 1: Using the Elements Menu

1. **Select** the page elements you want to rotate. (See *Selecting Page Elements* for guidance.)
2. Go to the **Elements menu** and click **Rotate**.
3. Choose one of the following options:

| Menu Item                  | Function                                          |
|---------------------------|---------------------------------------------------|
| **Rotate Right 90°**        | Rotates all selected elements 90° clockwise.      |
| **Rotate Left 90°**         | Rotates all selected elements 90° counterclockwise.|
| **Rotate 180°**             | Rotates all selected elements 180°.               |
| **More Rotation Options…**  | Opens a detailed rotation options dialog for the primary selected item. |

---

### Option 2: Using the Rotation Grab Handle

1. **Select** the page elements you want to rotate.
2. Place the mouse pointer over the **rotation grab handle** (usually above or near the element).
3. **Press and hold** the left mouse button.
4. **Drag** the mouse to set the desired rotation angle.
5. **Release** the mouse button to apply the rotation.

---

### Tip
- To **abort** rotation while dragging, press the **Esc** key on your keyboard.

---

## Remarks

- If **Lock Elements** is enabled on the **Home menu**, rotation via the grab handle is **disabled**.
- **Flash** and **Web Page elements** **cannot be rotated**.
- Use **More Rotation Options…** to set an exact angle or to fine-tune rotation beyond 90° increments.
