The **Table View** displays all scheduled presentations as structured records in a tabular format. This view is especially useful for quickly reviewing detailed scheduling data and comparing multiple entries.

---

## Accessing Table View

To switch to Table View:

1. Navigate to the **View** tab in the ribbon bar.  
2. In the **Schedules View** section, click on **Table**.

The schedules for the selected destination and date(s) will now appear in a **table layout**, showing details such as:

- Presentation name  
- Start and end time  
- Duration  
- Assigned destination  
- Additional properties (such as repeat intervals or priority)
