These options represent global settings for the PADS4 Agent application.

#### **To open the General settings of the Agent application**

1. Click on the **Tools** tab in the ribbon bar.

   ![EN Agent Toolstab Pn](/images/digital-signage/windows-applications/agent/tools/EN-Agent-Toolstab.png)
2. Click **Options** in the list of available options.
3. The options menu will open, and the **General** page will be displayed.

![EN Agent Options General Pn](/images/digital-signage/windows-applications/agent/tools/EN-Agent-Options-General.png)

#### **Language**

**Language**\
\
The language in which the user interface of the PADS4 Agent application will be displayed.

When the Language setting is changed, the new setting will only take effect after a restart of the application.
