#### **To close the PADS4 Agent application**

1. Select **File** in the ribbon bar.
2. Click **Exit**.

Or press the close button to close the application

![EN Agent Close Pn](/images/digital-signage/windows-applications/agent/remote-management-and-monitoring/EN-Agent-close.png)

Alternatively, you can use the key combination [Alt] \+ F4 to leave the application. If you have opened additional windows, then you must use this key combination several times to quit.

**NOTE**

When closing the application, you will automatically be logged off and disconnected from all connections that you have active.
