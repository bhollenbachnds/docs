This guide provides information on installing and configuring PADS4 Viewer on a Hisence DM66D

## Compatibility

- Supported Devices: Hisense DM66D

## Installation Instructions

To install PADS4 Viewer on a HiSense DM66D device, follow the standard Android installation procedure.

<Note>
  Support for the Hisense DM66D has been added to our PADS4 Generic Launcher solution in **PADS4 version 2025.2**, kindly refer to the PADS4 Generic Android Viewer manual for further instructions.
</Note>
