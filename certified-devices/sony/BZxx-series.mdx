## **Installing PADS4Sony from USB Stick**

**You will install PADS4Sony from a USB flash drive.**

**Steps to Install PADS4Sony:**

<Steps>
  <Step title="Step One">
    Download the **PADS4Sony .pkg file** from [**here at PADS4 Github.**](https://github.com/PADS4/PADS4-Viewer-Sony/releases/latest)
  </Step>
  <Step title="Step Two">
    Copy the **.pkg file** to a **USB flash drive** and insert the drive into the Sony display.
  </Step>
  <Step title="Step Three">
    Perform a **Factory Reset** on the display:

    - **Go to** `Settings -> System -> About -> Reset -> Factory Data Reset`
    - **Choose** "Erase everything"
    - **(Skip this step if setting up a brand-new display)**
  </Step>
  <Step title="Step Four">
    The display will **begin a factory reset within 15 seconds** and **reboot**.
  </Step>
  <Step title="Step Five">
    **When the initial setup screen appears (language selection), do not start the setup process.**
  </Step>
  <Step title="Step Six">
    The display will **detect** the `bpk` file on the **USB drive** and present the **package installation screen**.
  </Step>
  <Step title="Step Seven">
    **Choose "Install"** and follow the on-screen instructions.
  </Step>
  <Step title="Step Eight">
    After installation, the TV will open an **activation screen**.
  </Step>
  <Step title="Step Nine">
    Scroll to the bottom and **select "Activate this device admin app."**
  </Step>
  <Step title="Step Ten">
    An **"Install is complete"** message will appear. **Remove the USB drive** and select **OK**.
  </Step>
  <Step title="Step Eleven">
    The display will return to the **initial setup screen**. **Complete the setup** but **skip the Google account sign-in step**.
  </Step>
</Steps>

**Allow all permission requests during or before installation.**

## **Connecting to PADS4 Server**

**You must configure the PADS4 Server URL to establish a connection.**

**PADS4 Server URL Format:**

```
<protocol>://<host>:<port>/<hostpath>/
```

**Protocol:** `http` or `https` (based on SSL certificate)\
**Host:** `DNS name` or `IP address` of the PADS4 Server\
**Port:** **Optional** (use only if PADS4 Server is not on the default HTTP/HTTPS ports)\
**Default Hostpath:** `PADS4`

**Example Configuration:**

```
http://localhost/pads4
```

---

### **Steps to Connect to the PADS4 Server**

<Steps>
  <Step title="Step One">
    Enter the **PADS4 Server Location**.
  </Step>
  <Step title="Step Two">
    (Optional) Enter a **Viewer Name** (or let the PADS4 Server generate one).
  </Step>
  <Step title="Step Three">
    Press **Start**.
  </Step>
  <Step title="Step Four">
    Wait for the **Viewer Download to complete**, then the Viewer will start.
  </Step>
  <Step title="Step Five">
    When prompted, **grant permission** to permanently store content on the device.
  </Step>
</Steps>

**Now the PADS4 Viewer will start and connect to the PADS4 Server.**
