title: "DM Series (Windows)"
---

This manual provides step-by-step instructions for configuring the Giada DM-series. It covers initial setup and viewer setup.

**Compatible Devices:**

- **DM-series** 

**Minimum requirements**

PADS4 Version Required: **2022.1**\
Operating System: **Windows 10, Windows 11**

## **Installation Instructions**

To install **PADS4 Viewer** on a **Giada DM-Series** device, follow the standard **Windows installation process**.

<Note>
  For **detailed step-by-step instructions**, refer to the [**Installing PADS4 on Windows**](/certified-devices/windows/Windows-Viewer) guide.
</Note>

##  **Additional Configuration**

After installation, configure **PADS4 Viewer** using **PADS4 CTRL Center** or **PADS4 Agent** for:

- **Network Configuration**
- **Content Library Setup**
- **Viewer Settings Adjustments**

For remote management, use **PADS4 CTRL Center** to:

- **Monitor Viewer Status**
- **Manage Proof of Play Logs**
- **Perform Software Updates**
