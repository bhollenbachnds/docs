title: "5050D-Series"
---

This guide gives step‑by‑step instructions for installing and configuring PADS4 Viewer on Philips 5050D series displays.

## **Prepare the 5050D-series**

Before installation, ensure that the **PADS4 Server is set up and configured properly**.

<Note>
  The minimum **PADS4 Server version** required is **PADS4 Foundation 2025.1**.
</Note>

When turning on the device for the first time, the **Configuration Wizard** will appear. If not, **reset the device** as described in **Resetting the Device**.

### **Resetting the 5050D-series**

If the device has been used before, perform a factory reset to clear the storage and avoid software conflicts.

1. On the Philips remote control, press **Home**, then **1 8 8 8** to open the Android Menu.
2. Open **Settings**, then select **System Tools → Factory Reset**.
3. Confirm by clicking **Factory Reset** and pressing **OK**.

After the reset, the **Welcome Screen** appears. Continue with the steps in **Configuring the Device**.

### **Configuring the 5050D-series**

1. On the **Welcome Screen**, select your preferred **language** and click **Start**.
2. On the **Security Settings** screen, ensure **SICP Network Port** is **enabled**. _(Other settings can be enabled or disabled as required.)_
3. On the **Network Settings** screen, select and configure the **network type**.
4. On the **Signage Display Settings** screen, press **Skip**.
5. On the **Clone Media Files** screen, press **Skip**.
6. Press **Finish** to complete the setup.

### **Checking Device Firmware**

Ensure the **correct firmware version** is installed for full **PADS4 Viewer** functionality.

1. Open the Android Menu by pressing Home 1 8 8 8.
2. Open Settings → Select About.
3. Ensure the Build Number is FB01.05 _(required for PADS4)_.

## **Installing PADS4 Viewer**

Follow these steps to install PADS4 Viewer on Philips D-line (5050D) displays.

### **Downloading the PADS4 Viewer**

The **PADS4Philips.apk** file is available on the PADS4 Server in,

```text
https://[PADS4Server]/pads4/apps
```

where \[PADS4Server\] should be replaced with the name of your server.\
Locate the **PADS4Philips.apk** file and copy it to a USB stick

### **Installation Steps**

1. Insert the USB stick into the Philips 5050D-series.
2. Open the Android Menu by pressing **Home 1 8 8 8**.
3. Go to **Apps → Settings → Security → More Security Setting**.
4. Enable **Unknown Source**.
5. Return to the app screen.
6. Open **File Manager** and select the USB stick.
7. Open **PADS4Philips.apk** and tap **Install**.
8. Tap **Done**

### **Configure automatic launch of PADS4**

1. Go to **Settings → Signage Display → Source Settings → Custom Apps**.
2. Select **PADS4** from the list of available applications.
3. Enable **Force Restart Custom App**.
4. Restart the screen to apply the changes.

### **Connect the PADS4 Viewer**

After the device restarts, the **PADS4 Viewer** will appear.

1. Enter the **Server URL** in the following format: `https://[PADS4server]/[Domain]`.
2. Replace `[PADS4server]` with the name of your server, and replace `[Domain]` with your domain. The default domain is **PADS4**.
3. Fill in the **ViewerName** if required.
4. Click **Start** to connect.

The **PADS4 Viewer** is now connected and ready for use.

<Note>
  You may see a message indicating that PADS4 is incompatible with this version of Android. This is due to the screen running a newer Android version. You can safely acknowledge and ignore this message. PADS4 is compatible with Android 14, and this warning will be resolved in a future release.
</Note>
