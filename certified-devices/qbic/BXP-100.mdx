This guide provides **step-by-step instructions** for installing and configuring **PADS4 Viewer** on **Qbic BXP-100 devices** running **Android 10 and Android 12**.  

---

## **Before You Start**  

### **Supported Qbic BXP-100 Devices**  

The following **Qbic BXP-100** devices are supported:  

- **BXP-100 (Android 10)**  
- **BXP-100 (Android 12)**  

**Important:** Ensure that your **PADS4 Server** is installed and configured before setting up your Qbic device.  

---

## **Minimum Requirements**  

**PADS4 Server Version:** **2023.1 or later**  

**Firmware Version:**  

| Android Version | Minimum Firmware Version |
|:--------------:|:------------------------:|
| **Android 10** | **1.0.4** |
| **Android 12** | **2.12** |

<Note> Ensure your **Qbic device firmware** meets the **minimum requirements** before proceeding with installation.</Note>  

---

## **Bulk Deployment of Qbic Devices**  

To deploy **PADS4 Viewer** across multiple **Qbic devices simultaneously**, refer to the **Bulk Deployment of PADS4 Viewer for Qbic Devices** guide.  

**Once PADS4 is installed, proceed to configuring the BXP-100.**

## **Installing PADS4 Viewer via USB Stick**  

This method allows **manual installation** of the **PADS4 Viewer** application using a **USB flash drive**.  

### **Step 1: Download the PADS4 Viewer APK**  

**Download Locations:**  

**PADS4 Server Directory:**  
```
https://[YourPADS4Server]/PADS4/apps
```  
**Official Download Link:**  
[Download PADS4Qbic APK](https://install.appcenter.ms/orgs/nds/apps/nds.droid.qbic/distribution_groups/public)  

**Transfer to USB Stick:**  

<Steps>  
  <Step title="Step One">  
    Copy the **PADS4Qbic.apk** file to a **USB flash drive**.  
  </Step>  
  <Step title="Step Two">  
    Insert the **USB flash drive** into the **Qbic BXP-100 device**.  
  </Step>  
</Steps>  

---

### **Step 2: Enable Native Android Mode**  

To install the application, the device must be switched to **Native Android Mode**.  

<Steps>  
  <Step title="Step One">  
    Power on the **Qbic BXP-100** device.  
  </Step>  
  <Step title="Step Two">  
    Navigate to **Settings → Player Settings**.  
  </Step>  
  <Step title="Step Three">  
    Under **Android System**, enable **Switch to Native Android Mode**.  
  </Step>  
  <Step title="Step Four">  
    Exit **Settings**. You should now see the **Android home screen**.  
  </Step>  
</Steps>  

---

### **Step 3: Install PADS4 Viewer APK**  

<Steps>  
  <Step title="Step One">  
    Open the **Files App** on your **Qbic BXP-100**.  
  </Step>  
  <Step title="Step Two">  
    Navigate to the **USB Storage** location.  
  </Step>  
  <Step title="Step Three">  
    Locate and **tap on** the **PADS4Qbic.apk** file.  
  </Step>  
  <Step title="Step Four">  
    If prompted, **allow installation from unknown sources**.  
  </Step>  
  <Step title="Step Five">  
    Click **Install**, then **Done** once the installation completes.  
  </Step>  
</Steps>  

---

### **Step 4: Disable Native Android Mode**  

Once installation is complete, **switch back to Qbic Configuration Mode**.  

<Steps>  
  <Step title="Step One">  
    Open **Settings → Player Settings**.  
  </Step>  
  <Step title="Step Two">  
    Under **Android System**, disable **Switch to Native Android Mode**.  
  </Step>  
  <Step title="Step Three">  
    Exit **Settings**. The **Qbic Configuration screen** should appear.  
  </Step>  
</Steps>  

---

## **Configuring the BXP-100**  

This section guides you in **configuring the Qbic BXP-100** to **automatically launch PADS4 Viewer** on startup.  

### **Step 1: Set Content URL (USB Installation Only)**  

If PADS4 Viewer was installed via USB:  

<Steps>  
  <Step title="Step One">  
    Locate the **Content URL** field in **Settings**.  
  </Step>  
  <Step title="Step Two">  
    Enter the following value:  
    ```
    com.pads4.viewer.qbic/.MainActivity
    ```  
  </Step>  
</Steps>  

---

### **Step 2: Configure Browser View and Screen Settings**  

<Steps>  
  <Step title="Step One">  
    Open **Settings → Player Settings**.  
  </Step>  
  <Step title="Step Two">  
    Adjust **Browser Settings**:  
    - **Request Desktop Site:** On  
    - **Show Security Warnings:** Off  
    - **Block Pop-ups:** On  
    - **Keep Cookies:** Off  
  </Step>  
  <Step title="Step Three">  
    Adjust **Screen Settings**:  
    - **Full Screen:** On  
    - **Adjust window for input method:** On  
  </Step>  
</Steps>  

**The PADS4 Viewer will automatically start after 20 seconds.**  

---

## **Connecting to the PADS4 Server**  

### **Step 1: Configure Server Connection**  

**Format:**  
```
https://[YourPADS4Server]/PADS4/
```  

<Steps>  
  <Step title="Step One">  
    Open the **PADS4 Viewer** app.  
  </Step>  
  <Step title="Step Two">  
    Enter the **PADS4 Server Address**.  
  </Step>  
  <Step title="Step Three">  
    (Optional) Enter a **custom Viewer Name**, or leave blank to auto-generate.  
  </Step>  
  <Step title="Step Four">  
    Click **Start** to establish the connection.  
  </Step>  
</Steps>  

**If prompted, grant permissions for the PADS4 Viewer app.**  

**Once connected, the PADS4 Splash Screen should appear.**  

---

## **Troubleshooting**  

**Issue:** PADS4 Viewer does not launch automatically.  
**Solution:** Ensure **PADS4 Viewer** is set as the **Content URL** in **Settings**.  

**Issue:** Device fails to connect to the PADS4 Server.  
**Solution:** Verify the **server URL** and ensure the **PADS4 Server is running**.  

**Issue:** App installation is blocked.  
**Solution:** Enable **Install from Unknown Sources** in **Settings**.
