This guide explains how to install and configure PADS4 Viewer on AOPEN ChromeBox-Series devices running ChromeOS.

## Compatibility

- Supported Operating System: ChromeOS
- Supported Devices: AOPEN ChromeBox-Series computers

## Installation Instructions

To install PADS4 Viewer on an AOPEN ChromeBox-Series device, follow the standard ChromeOS installation procedure.

For detailed, step-by-step guidance, see the “Installing PADS4 on ChromeOS” guide.
