This guide provides information on installing and configuring PADS4 Viewer on Aopen DE-Series computers running Windows 10 or Windows 11.

## Compatibility

- Supported Operating Systems: Windows 10 or Windows 11
- Supported Devices: AOPEN DE-Series Computers

## Installation Instructions

To install PADS4 Viewer on an AOPEN DE-Series device, follow the standard Windows installation procedure.

For detailed, step-by-step guidance, see the “Installing PADS4 on Windows” guide.
