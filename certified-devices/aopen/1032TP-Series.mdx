## **First-Time Use**

During installation, you will need to configure the **PADS4 Server location**. It is recommended to **complete the installation of the PADS4 Server first** before configuring the device.

<Note>
  The minimum **PADS4 Server version** required for the **PADS4 AOpen App** is **2024**.
</Note>

---

## **Supported models**

- AOpen AT1032TP
- AOpen eTILE-X1032TP

<Note>
  Support for the X1032TP has been added in PADS4 version 2025.1, in case support is required for this particular model on older PADS4 versions then replace the APK on your server with the latest available download of the PADS4AOpen.apk.
</Note>

## **Installing the PADS4 AOpen Viewer**

The **PADS4AOpen.apk** file is available on the **PADS4 Server** in the **Apps** folder of the installation directory:

```
C:\Program Files\PADS4\Foundation\Apps  
```

Alternatively, download the latest version from [**here**](#).

<Note>
   If downloading the latest version, ensure that the APK is also placed in the **Apps** directory on the **PADS4 Server**. If the server contains a different version, it will install that version when connected.
</Note>

### **Installation Steps**

<Steps>
  <Step title="Step One">
    Copy the **PADS4AOpen.apk** onto a **USB stick**.
  </Step>
  <Step title="Step Two">
    Insert the **USB stick** into the device.
  </Step>
  <Step title="Step Three">
    Open the **File Manager app** on the device.
  </Step>
  <Step title="Step Four">
    Browse to the **USB stick** and open **PADS4AOpen.apk**.
  </Step>
  <Step title="Step Five">
    A warning will appear stating that installation from unknown sources is not allowed.
  </Step>
  <Step title="Step Six">
    Go to **Settings** and allow installation from this source.
  </Step>
  <Step title="Step Seven">
    Press the **Back** button and install the application.
  </Step>
  <Step title="Step Eight">
    After installation, open the **PADS4 AOpen** application.
  </Step>
</Steps>

## **Connecting to the PADS4 Server**

Once installed, the **PADS4 Viewer** needs to be connected to the **PADS4 Server**.

### **PADS4 Server Address Format:**

```
protocol://host:port/hostpath/  
```

- **Protocol:** Use `http` or `https` depending on whether an SSL certificate is applied.
- **Host:** The DNS name or IP address of the **PADS4 Server**.
- **Port:** Specify only if the **PADS4 Server** is not using the default HTTP/HTTPS ports.
- **Host Path:** Configured in **System Management** (default: `PADS4`).

**Example:**

```
http://localhost/pads4  
```

### **Connection Steps**

<Steps>
  <Step title="Step One">
    Enter the **PADS4 Server Location**.
  </Step>
  <Step title="Step Two">
    (Optional) Specify a **Viewer Name**. If left blank, the **PADS4 Server** will generate one.
  </Step>
  <Step title="Step Three">
    Set **PADS4 AOpen Viewer** as the **Home app** and select **Always**.
  </Step>
  <Step title="Step Four">
    Press **Start**.
  </Step>
  <Step title="Step Five">
    Allow the **PADS4 AOpen application** to modify system settings.
  </Step>
  <Step title="Step Six">
    Enable **"Allow to modify system settings"** and press the **Back** button.
  </Step>
  <Step title="Step Seven">
    Press **Start** again.
  </Step>
  <Step title="Step Eight">
    Grant **permission to store content** on the device when prompted.
  </Step>
</Steps>

Once completed, the **PADS4 Viewer** should connect to the **PADS4 Server**. Additional settings such as **screen orientation** and **remote control restrictions** can be configured using **PADS4 Agent** or **Viewer Settings**.

## **Adjusting PADS4 Viewer Options**

- **Long-press the screen** to open the menu.
- Alternatively, options can be changed remotely via **PADS4 CTRL Center** or **PADS4 Agent**.

---

## **Accessing the Android Menu**

1. Open **Viewer Menu**.
2. Click **Launcher Settings**.
3. Click the **Gear Icon** in the bottom-right corner.
4. The **Android Settings** menu will open.

<Note>
  If **"Hide Status Bar"** is enabled, the only way to exit Android Settings is to **restart the device** by disconnecting the power.
</Note>

---

## **Disabling the Android Navigation Bar**

(Optional) The navigation bar can be disabled to prevent unauthorized interaction.

### **Steps:**

1. Open **Viewer Options**.
2. Click **Launcher Settings**.
3. Click the **Gear Icon** in the bottom-right corner.
4. Select **System Tools**.
5. Click **Disable Navigation Bar**.

<Note>
  If disabled, the only way to exit the settings is to **restart the device** by disconnecting the power.
</Note>

---

## **Performing a Factory Reset**

To restore the device to factory settings:

1. Open **Android Settings**.
2. Navigate to **Backup & Reset**.
3. Select **Factory Data Reset**.
4. (Optional) Select **Erase SD Card**.
5. Tap **Reset Tablet**.
6. Confirm with **Erase Everything**.

## **Important Notes**

<Warning>
   **Content File Integrity** – The **PADS4 Content Library** verifies file integrity using **MD5 checksum** during synchronization. 
</Warning>

<Warning>
   **Proof of Play Limitations** – A maximum of **50,000 records** can be stored locally during offline periods. When full, the oldest records are deleted first. 
</Warning>
