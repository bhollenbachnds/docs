This guide explains how to install the **PADS4 Launcher APK** to start the **PADS4 Viewer** on **NDS-certified Android hardware**.

<Note>
  Some devices come with a **dedicated PADS4 Launcher**. For more information, refer to the [**PADS4 Certified Devices**](#) page.
</Note>

## **Installing PADS4 Launcher from a USB Stick**

<Steps>
  <Step title="Step One">
    Copy the **PADS4 Launcher APK** to a **USB stick**.
  </Step>
  <Step title="Step Two">
    Insert the **USB stick** into the Android device.
  </Step>
  <Step title="Step Three">
    Open **APK Installer** on the device.
  </Step>
  <Step title="Step Four">
    Select **Install** and browse to the APK file on the **USB stick**.
  </Step>
  <Step title="Step Five">
    When prompted with:\
    _"Do you want to install this application?"_

    - Press **Next** until the **Install** button appears.
    - Press **Install** to proceed.
  </Step>
  <Step title="Step Six">
    After installation, press **Open** to launch the **PADS4 Launcher**.
  </Step>
  <Step title="Step Seven">
    Enter the **PADS4 Server location**, specifying:

    - **Domain name**
    - **IP address**
    - **PC name** of the server running **PADS4 Server**
  </Step>
  <Step title="Step Eight - *Optional*">
    If the default **server settings** were modified, go to **Advanced Settings** and enter:

    - The correct **port number**
    - The **host/root path**
  </Step>
  <Step title="Step Nine">
    Press **Start**.

    - The first time the app runs, it will prompt you to select your **home app**.
    - Select **PADS4 Launcher** and choose **Always** to prevent future prompts.
  </Step>
  <Step title="Step Ten">
    Ensure a **valid connection** to the **PADS4 Server** when launching the **PADS4 Launcher** for the first time.

    - This allows the device to **download and start PADS4 Viewer**.
    - Once downloaded, **PADS4 Viewer** can run **offline**.
  </Step>
</Steps>

## **Updating PADS4 Launcher from a USB Stick**

<Steps>
  <Step title="Step One">
    Copy the latest **PADS4 Launcher APK** to a **USB stick**.
  </Step>
  <Step title="Step Two">
    Insert the **USB stick** into the Android device.
  </Step>
  <Step title="Step Three">
    Open **Viewer Settings**.
  </Step>
  <Step title="Step Four">
    Navigate to **Launcher Settings**.
  </Step>
  <Step title="Step Five">
    Tap the **folder icon** in the **bottom-right corner**.
  </Step>
  <Step title="Step Six">
    Browse to:

    ```
    mnt -> usb_storage  
    ```

    Select the **APK file**.
  </Step>
  <Step title="Step Seven">
    When prompted with:\
    _"Do you want to install an update to this existing application?"_

    - Press **Install**.
  </Step>
  <Step title="Step Eight">
    Once the installation is complete, the **PADS4 Launcher** will automatically restart.
  </Step>
</Steps>
