This guide provides step-by-step instructions for **installing and configuring PADS4 Viewer** on **ProDVX APPC-Series (RK3288) devices** running **Android 9**.

## **Before You Start**

### **Supported ProDVX APPC-Series Devices**

The following **ProDVX APPC-Series (RK3288)** devices are supported:

- **APPC-7XPL**
- **APPC-7XPLN**
- **APPC-10X**
- **APPC-10XP**
- **APPC-10XPL**
- **APPC-10XPLN**
- **APPC-10SLB** (Requires **10X Firmware**)
- **APPC-10SLBN**
- **APPC-15XP**
- **APPC-22X**
- **APPC-22XP**
- **APPC-24**
- **APPC-27X**
- **APPC-32**

## Reset the device (recommended)

If the device was used before, perform a **factory reset** to clear storage and avoid software conflicts.

1. Open **Android Settings**.
2. Go to **System \> Reset options**.
3. Select **Erase all data (factory reset)**.
4. Select **Reset tablet**, then confirm with **Erase everything**.
5. Wait for the device to restart automatically.
