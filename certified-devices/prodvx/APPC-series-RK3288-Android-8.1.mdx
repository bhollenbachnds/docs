## Reset the device (recommended)

If the device was used before, perform a **factory reset** to clear storage and avoid software conflicts.

1. Open **Android Settings**.
2. Go to **System \> Reset options**.
3. Select **Erase all data (factory reset)**.
4. Select **Reset tablet**, then confirm with **Erase everything**.
5. Wait for the device to restart automatically.

## Install the PADS4 Viewer

### Step 1: Download and transfer the APK

Download the APK from either location:

- **PADS4 Server directory:** `C:\Program Files\PADS4\Clients\Apps`
- **Official download:** [PADS4ProDVX APK on GitHub](https://github.com/PADS4/PADS4-Viewer-ProDVX-x-series/releases/latest)

Then transfer it to the device:

1. Copy **PADS4ProDVX.apk** to a USB stick.
2. Insert the USB stick into the ProDVX APPC-Series device.

### Step 2: Install the APK

1. Open the **APK Installer** app on the device. Allow access to external storage if prompted.
2. Select **Install** and browse the USB stick for **PADS4ProDVX.apk**.
3. Select the APK file to start installation.
4. If prompted, enable **Allow install from unknown sources**, then retry the installation.
5. Select **Install**, and choose **Open** when the installation completes.

## Configure the PADS4 Viewer

### Step 1: Validate date and time

Incorrect date and time settings can cause certificate errors and connection issues.

1. Open **PADS4ProDVX** and go to **System settings**.
2. Open **Date & time**.
3. Set the correct date and time, or enable **Auto sync date & time** if supported.
4. Return to the **PADS4ProDVX launcher**.

### Step 2: Connect to the PADS4 Server

Use the following URL format:

```text
<protocol>://<host>:<port>/<domain>/
```

**Example**

```text
https://example.pads365.com/pads4
```

1. Open **PADS4ProDVX** and enter the **PADS4 Server address**.
2. Optionally, enter a custom **Viewer name**.
3. Select **Start** to connect.
4. If the **Android System WebView** is outdated, it updates automatically.
5. Once updated, the PADS4 Viewer restarts — select **Start** again.

## Updating the PADS4 Viewer

After installation, the PADS4 Viewer updates automatically whenever the PADS4 Server is updated.

## Troubleshooting

| Issue | Resolution |
| :-- | :-- |
| Video transition effects not working | Device hardware limitation. |
| Device fails to connect to the network | Ensure Wi-Fi or Ethernet is enabled in Android Settings. |
| PADS4 Viewer restarts frequently | Verify the PADS4 Server connection and that Android System WebView is up to date. |
