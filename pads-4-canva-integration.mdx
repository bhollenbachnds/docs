---
title: "PADS4 Canva integration"
---

1. Open PADS4 Control Center and select **Credentials**.
   <Frame>
     <img
       src="/images/CTRL-Center/IconCredentials.png"
       alt="Icon Credentials"
       title="Icon Credentials"
       style={{ width:"67%" }}
     />
   </Frame>
2. In the Credentials overview, select Providers.
3. Select New and choose Canva. This opens the Create a new Canva app registration window.
   <Frame>
     <img
       src="/images/DigitalSignage/Designer/NewCanvaRegistration.png"
       alt="Icon Credentials"
       title="Icon Credentials"
       style={{ width:"100%" }}
     />
   </Frame>
4. Enter a clear and recognizable name for the integration to identify it later if needed.
5. Enter the **App (Client) ID** and **Client Secret** that were generated earlier in the Canva Developer Portal.
6. Click **Create** to register the integration within PADS4.

Your registration is done. You can now open Designer and start using Canva.
