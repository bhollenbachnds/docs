## What is PADS4?

PADS4 is a software platform to enhance user experiences in a smart digital building. Design the best for your smart building through a tailor-made combination of PADS4 modules like Digital Signage, Workspace, FIDS and more.


## Benefits of the PADS4 platform

- ### Flexible

  Use the power of PADS4 as a platform to build the entire user workflow and avoid unexpected costs coming from multiple interface issues, integrations and configurations.

- ### Scalable

  From internal communication to smart workspace, from flight information to boarding gate management, from advertisements to wayfinding, PADS4 is a high-class software platform to build your smart digital building experience.

- ### Open

  The PADS4 platform consists of four different modules allowing you to modulate your project according to your needs. Reduce implementation costs and learning curves by using the same platform.


## Modules

<AccordionGroup>
  <Accordion title="PADS4 CTRL Centre">
    Web based PADS4 portal to manage and maintain your PADS4 system.
    Besides web based we also have Windows applications to manage your PADS4 system.
  </Accordion>
  <Accordion title="PADS4 Digital Signage">
    Dedicated software called Viewers which are the endpoint on which content is shown. The presentation is created using a powerful Designer.
    Sub and Backup Servers can be used to setup additional Servers when required where PADS4 will automatically synchronize everything between those servers.
  </Accordion>
  <Accordion title="PADS4 Client applications">
    Microsoft Windows based PADS4 applications that can be installed separately on client PC's.
    These application can connect remotely to any PADS4 Server.

    Most of the functionality that is in the client application may also be available in the PADS4 Web based portal (*currently we are working on migrating all functionality to the web portal).
    | Application | Description |
    | ----------- | ----------- |
    | PADS4 Designer | Application to create presentation designs using a WYSIWYG editor|
    | PADS4 Scheduler	| Application to schedule presentations for Viewers|
    | PADS4 Messenger	| Application to schedule messages to Desktop Viewers|
    | PADS4 Agent	| Application to monitor the PADS4 system|
    | PADS4 System Management	| Application to manage the PADS4 system|
    | PADS4 Desktop Viewer | Application which shows the scheduled messages|
    | PADS4 Viewer | Application which shows the scheduled presentations|
  </Accordion>
  <Accordion title="PADS4 FIDS">
    All-inclusive digital airport signage. PADS4 FIDS Operator is a Flight Information Management System to manage your daily flights for small and medium airports.
    Flight Information Display Systems make up the heart of any airport, operating in crucial functions for both control rooms and passenger information.
  </Accordion>
  <Accordion title="PADS4 CMS">
    Create messages using a web application and display these on the Viewers from the PADS4 Digital Signage module
  </Accordion>
  <Accordion title="PADS4 Workspace">
    Desk and Room management.
  </Accordion>
  <Accordion title="PADS4 Wayfinding">
    Presenting interactive maps and guiding people. Maps can be shown on Kiosk or other interactive devices.
  </Accordion>
  <Accordion title="PADS4 VMS">
    Preregistration, Monitoring and reporting visitors in your smart digital building.
  </Accordion>
</AccordionGroup>
