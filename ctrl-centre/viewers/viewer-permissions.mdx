<Warning>
  This article applies to PADS4 version 2025.2 and above.

  All versions before that have the viewer permission administration in the PADS4 System Management application.
</Warning>

The **Viewer permissions** in **PADS4** allows you to have granular control over which \*\*viewer group (s) \*\*and **viewer(s)** are visible for each specific user. This way you can precisly configure what an end user is able to see and control.
