Viewer Variables, also known as **Dynamic Filtering**, allow you to **customize content dynamically** across multiple Viewers **without creating multiple versions** of the same presentation. This feature is ideal for **data-driven digital signage networks** where minor variations in displayed content are required.

For a **detailed introduction to Viewer Variables**, [click here](#).

---

## **Creating Viewer Variables**
- Click **New** to manually define a **new Viewer Variable**.
- Assign a **default value** that will apply to all Viewers unless otherwise specified.

---

## **Editing Viewer Variables**
1. **Edit Variable Name & Default Value**  
   - Hover over the **column header** and click the **pencil icon** ✏️.
   - Modify the **Viewer Variable name** or **default value**.

2. **Edit Variable Value for a Specific Viewer**  
   - Hover over a **specific cell** where the value needs adjustment.
   - Click the **pencil icon** ✏️ and **update the value**.

---

## **Bulk Editing via Excel**
For large-scale updates, you can use **Microsoft Excel** to manage Viewer Variables efficiently.

### **Steps to Export & Import Viewer Variables**
1. **Export**
   - Click **Export** to download the current Viewer Variables in an Excel sheet.
   
2. **Modify in Excel**
   - **Row 1**: Viewer names (top headers).
   - **Row 2**: Default values (applied if no specific value is set).
   - **Rows 4 and above**: Individual Viewer-specific values.
   - **Leaving a cell empty** means that Viewer will use the **default value**.

3. **Import**
   - Once changes are made, use the **Import** function to upload the updated Excel file.

---

## **Why Use Viewer Variables?**
✔ **Save time** by using a single presentation across multiple Viewers.  
✔ **Easily customize content** per Viewer without modifying individual presentations.  
✔ **Manage large networks efficiently** with bulk editing via Excel.
