PADS4 CTRL Center is the central hub for managing and optimizing your PADS4 environment, providing an intuitive web interface to monitor, configure, and control your system.

## PADS4 CTRL Center overview

PADS4 CTRL Center is a web-based management console used to administer and supervise your entire PADS4 deployment. Through this interface, you can monitor system status, adjust configuration settings, and control connected components from a single location.

## Starting PADS4 CTRL Center

To open PADS4 CTRL Center, enter the following URL in your browser:

`http(s)://servername:portnumber`

Use `https` in secured environments; in all other cases you can use `http`. The `servername` specifies the server where PADS4 CTRL Center is installed, and the `portnumber` is optional when the application is hosted on port 80.

<Card title="Supported web browsers" type="note">
  PADS4 CTRL Center officially supports the following web browsers:

  - Google Chrome
  - Microsoft Edge

  Other browsers are not supported and may not function as expected
</Card>

## **Logging in to PADS4 CTRL Center**

To log in to PADS4 CTRL Center, follow these steps:

- Open a supported web browser and navigate to the PADS4 CTRL Center URL.
- The PADS4 Login page is displayed.
- Enter your Username and Password, then click Log In.

<img
  height="200"
  src="/images/ctrl-centre/login.png"
/>

- When valid credentials are entered, the PADS4 CTRL Center homepage is displayed.
- If you have permissions for more than one application, a list of available applications is shown. Select the application you want to open.
